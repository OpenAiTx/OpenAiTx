# mciwb

Minecraft インタラクティブ ワールド ビルダー
===================================

このプロジェクトは、Pythonプログラミング言語を楽しく学ぶためのものです。

経験豊富なPythonプログラマーも、このライブラリを使用して
インタラクティブなPython機能を持つMinecraftの世界を作成できます。

目標
-----

Minecraft Interactive World Builder の目標は次のとおりです:

 - Python を使って Minecraft の世界内に何でも作成すること。
 - 世界内でイベントが発生したときに Python コードを呼び出すこと。
 - iPython プロンプトを使用して Python コマンドを入力し、世界と対話すること。
 - プレイヤーがコマンドサインを設置したり、スイッチやレバーを操作して Python アクションを実行すること。

Minecraft サーバーと Python 開発環境のセットアップ方法のすべての手順が含まれています。  
Python プログラミングと Minecraft Interactive World Builder の機能を紹介するステップバイステップのチュートリアルもあります。

.. note::

    これは進行中の作業です。Python の基礎を学ぶのに十分なチュートリアルと、
    Python コードから Minecraft と楽しくやり取りできる十分な機能があります。

    チュートリアルを拡充し、ライブラリにさらに多くの機能を追加していく予定です。
