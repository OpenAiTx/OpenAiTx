# mciwb

Minecraft 交互式世界构建器
==========================

本项目旨在作为一种有趣的方式来学习 Python 编程语言。

有经验的 Python 程序员也可以使用此库来创建
具有交互式 Python 功能的 Minecraft 世界。

目标
-----

Minecraft 交互式世界构建器的目标是：

 - 使用 Python 在 Minecraft 世界中创建任何内容。
 - 当世界内发生事件时调用 Python 代码。
 - 使用 iPython 提示符通过输入 Python 命令与世界交互。
 - 让你的玩家通过放置命令牌或激活带有 Python 动作的开关/杠杆来执行 Python 代码。

所有关于如何设置 Minecraft 服务器和 Python 开发环境的说明均包含在内。还有逐步教程介绍 Python 编程和 Minecraft 交互式世界构建器的功能。

.. note::

    这是一个进行中的项目。现在已有足够的教程来学习 Python 基础，并且具备了一些功能，可以通过 Python 代码与 Minecraft 进行有趣的交互。

    我将继续扩展教程并添加更多功能到该库中。
