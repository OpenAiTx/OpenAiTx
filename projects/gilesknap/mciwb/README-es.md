# mciwb

Constructor de Mundos Interactivos de Minecraft
===============================================

Este proyecto está pensado como una forma divertida de aprender el Lenguaje de Programación Python.

Los programadores experimentados en Python también pueden usar esta biblioteca para crear
mundos de Minecraft con características interactivas en Python.

Objetivos
---------

Los objetivos de Minecraft Interactive World Builder son:

 - Usar Python para crear cualquier cosa dentro de los mundos de Minecraft.
 - Ejecutar código Python cuando ocurran eventos dentro del mundo.
 - Usar un prompt de iPython para interactuar con el mundo escribiendo comandos Python
 - Hacer que tu jugador dentro del mundo ejecute código Python colocando
   carteles de comando o activando interruptores/palancas con acciones Python.

Todas las instrucciones sobre cómo configurar un Servidor de Minecraft y el entorno
de desarrollo Python están incluidas. Hay tutoriales paso a paso para
introducir la programación en Python y las características de Minecraft Interactive World
Builder.

.. nota::

    Este es un trabajo en progreso. Ahora hay suficientes tutoriales para aprender
    los fundamentos de Python y suficiente funcionalidad para divertirse
    interactuando con Minecraft desde código Python.

    Continuaré ampliando los tutoriales y añadiendo más funciones a la
    biblioteca.
