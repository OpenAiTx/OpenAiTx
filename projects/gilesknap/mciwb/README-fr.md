# mciwb

Constructeur de Monde Interactif Minecraft
=========================================

Ce projet est destiné à être un moyen ludique d'apprendre le langage de programmation Python.

Les programmeurs Python expérimentés peuvent également utiliser cette bibliothèque pour créer
des mondes Minecraft avec des fonctionnalités Python interactives.

Objectifs
---------

Les objectifs de Minecraft Interactive World Builder sont :

 - Utiliser Python pour créer n'importe quoi à l'intérieur des mondes Minecraft.
 - Appeler du code Python lorsque des événements se produisent dans le monde.
 - Utiliser une invite iPython pour interagir avec le monde en tapant des commandes Python
 - Faire exécuter du code Python à votre joueur dans le monde en plaçant des panneaux de commande
   ou en activant des interrupteurs/leviers avec des actions Python.

Toutes les instructions pour configurer un serveur Minecraft et l'environnement de développement Python
sont incluses. Il y a des tutoriels étape par étape pour
initier à la programmation Python et aux fonctionnalités de Minecraft Interactive World
Builder.

.. note::

    Ceci est un travail en cours. Il y a maintenant assez de tutoriels pour apprendre les
    bases de Python et suffisamment de fonctionnalités pour s'amuser
    à interagir avec Minecraft depuis du code Python.

    Je continuerai à développer les tutoriels et à ajouter plus de fonctionnalités à la
    bibliothèque.
