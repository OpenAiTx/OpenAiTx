## 这是什么

jerry-list 是本仓库中的主词表，涵盖了广泛的路径、参数、端点和模式，这些内容在实际漏洞赏金狩猎中被证明有效。列表中的每一项都源自真实的实战经验，而非自动生成或重复使用公开资源。正是这个列表帮助我在多个项目中获得了多项P1和P2级别的发现。

一个私有的DNS词表也在制作中。该词表是我依赖的重要工具，用于发现大多数传统工具和公开词表完全遗漏的独特子域名。整理完成并准备好后，将在此发布。

## 用法

兼容任何标准的模糊测试工具。以下是几个快速示例：

**ffuf**
```bash
ffuf -u https://target.com/FUZZ -w jerry-list.txt -mc 200,301,302,403,500
```

**gobuster**
```bash
gobuster dir -u https://target.com -w jerry-list.txt
```

**subfinder / DNS爆破（DNS列表一旦生成后）**
```bash
puredns brute target.com -w jerry-dns.txt
```
