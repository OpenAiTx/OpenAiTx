# AI Partner Chat

โครงการ Claude Skills ที่ทำให้ AI เป็นคู่สนทนาเฉพาะตัวของคุณ

## แนะนำโครงการ

AI Partner Chat ผสานรวมโปรไฟล์ผู้ใช้, โปรไฟล์ AI และโน้ตส่วนตัวที่ถูกแปลงเป็นเวกเตอร์ เพื่อมอบประสบการณ์สนทนาแบบเฉพาะบุคคลที่เข้าใจบริบท ทักษะนี้ช่วยให้ AI จดจำและอ้างอิงความคิด ความชอบ และคลังความรู้ของคุณในอดีต สร้างประสบการณ์โต้ตอบที่ต่อเนื่องและเหมาะสมกับแต่ละบุคคลมากยิ่งขึ้น

## ฟังก์ชันหลัก

### 🎭 ระบบภาพลักษณ์คู่
- **ภาพลักษณ์ผู้ใช้**: กำหนดภูมิหลัง ความเชี่ยวชาญ ความสนใจ และรูปแบบการสื่อสารของคุณ
- **ภาพลักษณ์ AI**: ปรับแต่งบทบาท ลักษณะการสื่อสาร และรูปแบบการโต้ตอบของ AI

### 📝 การค้นหาบันทึกอัจฉริยะ
- ทำดัชนีบันทึก Markdown ของคุณโดยอัตโนมัติ
- ค้นหาประวัติที่เกี่ยวข้องตามเนื้อหาการสนทนาอย่างชาญฉลาด
- อ้างอิงความคิดหรือบันทึกเก่าของคุณอย่างเป็นธรรมชาติระหว่างการสนทนา

### 💬 การสนทนาแบบปรับเฉพาะบุคคล
- สร้างคำตอบเฉพาะบุคคลจากภาพลักษณ์และบันทึกของคุณ
- รักษาความต่อเนื่องของบริบทการสนทนา
- อ้างอิงความคิดของคุณอย่างเป็นมิตร ไม่ใช่แบบ "ตามบันทึก"

## สถานการณ์การใช้งาน

เมื่อคุณต้องการ:
- การสื่อสารที่ปรับเฉพาะบุคคล ไม่ใช่คำตอบทั่วไป
- คำตอบที่รับรู้บริบท และ AI จดจำภูมิหลังของคุณได้
- ให้ AI จดจำและอ้างอิงความคิดหรือบันทึกก่อนหน้า
- ประสบการณ์สนทนาอย่างต่อเนื่อง ไม่ใช่เริ่มใหม่ทุกครั้ง

## การติดตั้งและการใช้งาน

### การติดตั้งทักษะ

คัดลอกโปรเจกต์นี้ไปยังโฟลเดอร์ `.claude/skills/` ในไดเร็กทอรีงานของคุณ:

```
<你的项目根目录>/
└── .claude/
    └── skills/
        └── ai-partner-chat/    # 本技能包
            ├── assets/
            ├── scripts/
            ├── SKILL.md
            └── README.md
```

### การใช้ทักษะ

ใน Claude Code เพียงส่งคำสั่งต่อไปนี้เพื่อเปิดใช้งานทักษะนี้:

```
遵循 ai-partner-chat 对话
```

AI agent จะทำงานอัตโนมัติ:
- อ่านการตั้งค่าทักษะและคำแนะนำ
- สร้างโครงสร้างไดเรกทอรีที่จำเป็น (`notes/`, `config/`, `vector_db/` เป็นต้น)
- ดำเนินการเริ่มต้นตามความต้องการของคุณ

### ขั้นตอนการเริ่มต้น

#### วิธีที่หนึ่ง: ให้ AI สร้างและตั้งค่าโดยอัตโนมัติ

เมื่อใช้ครั้งแรก บอก AI โดยตรงว่า:

```
我刚刚在 notes 里放入了对应的笔记，请根据笔记内容，进行向量化；并基于笔记内容，推测并更新 user-persona.md，以及最适合我的 ai-persona.md
```

AI agent จะดำเนินการดังนี้:
1. วิเคราะห์เนื้อหาในไดเรกทอรี `notes/`
2. แบ่งเนื้อหาออกเป็นส่วนย่อยอย่างชาญฉลาดและสร้างฐานข้อมูลเวกเตอร์ตามรูปแบบของโน้ต
3. คาดเดาพื้นหลังและความชอบของคุณจากเนื้อหาบันทึก
4. สร้างและอัปเดตไฟล์ `config/user-persona.md` โดยอัตโนมัติ
5. แนะนำและสร้างไฟล์ `config/ai-persona.md` ตามลักษณะของคุณ

#### วิธีที่สอง: กำหนดโปรไฟล์ด้วยตนเอง

หากคุณต้องการกำหนดโปรไฟล์เอง:
1. AI agent จะสร้างไฟล์โปรไฟล์จากเทมเพลตไปยังไดเรกทอรี `config/` โดยอัตโนมัติ
2. คุณสามารถแก้ไขไฟล์เหล่านี้ด้วยตนเองเพื่อปรับแต่งโปรไฟล์
3. จากนั้นแจ้งให้ AI ทำการเวกเตอร์ไลซ์

### เริ่มต้นสนทนา

หลังจากตั้งค่าเสร็จ ในแต่ละครั้งที่ใช้ คุณเพียงแค่ส่ง:

```
遵循 ai-partner-chat 对话
```
AI จะ:
- อ่านโปรไฟล์ของคุณเพื่อทำความเข้าใจภูมิหลังของคุณ
- ค้นหาบันทึกประวัติที่เกี่ยวข้อง
- สร้างคำตอบที่ปรับให้เหมาะสมและรับรู้บริบทเฉพาะบุคคล

## โครงสร้างโปรเจกต์

### โครงสร้างชุดทักษะ (อยู่ที่ `.claude/skills/ai-partner-chat/`)


```
ai-partner-chat/
├── assets/              # 画像模板
│   ├── user-persona-template.md
│   └── ai-persona-template.md
├── scripts/             # 核心脚本
│   ├── chunk_schema.py
│   ├── vector_indexer.py
│   ├── vector_utils.py
│   └── requirements.txt
├── SKILL.md            # 技能详细文档（AI agent 会读取此文件）
└── README.md           # 本文件
```

### ไดเรกทอรีข้อมูลผู้ใช้ (อยู่ในไดเรกทอรีรากของโปรเจกต์)

AI agent จะสร้างโครงสร้างต่อไปนี้ในไดเรกทอรีรากของโปรเจกต์ของคุณ:

```
<项目根目录>/
├── notes/              # 你的笔记（由你或 AI agent 创建）
├── config/             # 画像配置（由 AI agent 创建）
│   ├── user-persona.md
│   └── ai-persona.md
├── vector_db/          # 向量数据库（由 AI agent 创建）
└── venv/               # Python 虚拟环境（由 AI agent 创建）
```

**สำคัญ**: ข้อมูลผู้ใช้ถูกแยกออกจากแพ็กเกจทักษะ เพื่อความสะดวกในการสำรองข้อมูลและการย้ายถ่าย

## กระบวนการทำงาน

1. **โหลดโปรไฟล์**: อ่านโปรไฟล์ผู้ใช้และ AI เพื่อเข้าใจบริบทของการโต้ตอบ
2. **ค้นหาบันทึก**: ดึงบันทึกที่เกี่ยวข้องมากที่สุดจากฐานข้อมูลเวกเตอร์ตามคำค้นของผู้ใช้
3. **สร้างบริบท**: ผสานข้อมูลโปรไฟล์ บันทึกที่เกี่ยวข้อง และประวัติการสนทนา
4. **สร้างการตอบกลับ**: สร้างการตอบกลับที่เป็นธรรมชาติและปรับแต่งเฉพาะบุคคลโดยอิงบริบท

## จุดเด่น

### 🤖 AI Agent การแบ่งบล็อกอัจฉริยะ
ระบบจะวิเคราะห์รูปแบบจริงของแต่ละบันทึกและสร้างกลยุทธ์แบ่งบล็อกที่เหมาะสมที่สุดแบบไดนามิก ไม่ใช้แม่แบบที่ตั้งไว้ล่วงหน้า หมายความว่าไม่ว่าบันทึกของคุณจะมีรูปแบบใด ก็จะได้รับการประมวลผลอย่างเหมาะสมที่สุด

### 🎯 การอ้างอิงที่เป็นธรรมชาติ
AI จะอ้างอิงข้อมูลเก่าของคุณอย่างเป็นธรรมชาติราวกับการนึกถึงความทรงจำ ไม่พูดตรงๆ ว่า "ตามบันทึก" แต่จะผสมผสานเข้าไปในบทสนทนาอย่างลื่นไหล

### 📦 ข้อมูลแยกเป็นอิสระ
ข้อมูลทั้งหมดของคุณ (บันทึก โปรไฟล์ ฐานข้อมูลเวกเตอร์) จะถูกจัดเก็บไว้ที่โฟลเดอร์รากของโปรเจกต์ ง่ายต่อการสำรอง ย้าย หรือแชร์ระหว่างทักษะต่างๆ

## แนวทางปฏิบัติที่ดีที่สุด

### การออกแบบโปรไฟล์
- **เฉพาะเจาะจง**: โปรไฟล์ที่คลุมเครือจะนำไปสู่การตอบกลับแบบทั่วไป
- **มีตัวอย่างประกอบ**: ใส่รูปแบบการโต้ตอบที่คาดหวังไว้ในโปรไฟล์ AI
- **อัปเดตเป็นประจำ**: ปรับปรุงโปรไฟล์ตามคุณภาพการสนทนา

### การจัดการบันทึก
- **รูปแบบอิสระ**: ระบบสามารถปรับใช้กับโครงสร้างบันทึกได้ทุกรูปแบบ
- **เนื้อหาหลากหลาย**: บันทึกที่ลึกซึ้งจะให้ผลการค้นหาที่ดีกว่า
- **อัปเดตทันเวลา**: อย่าลืมเพิ่มบันทึกใหม่ลงในดัชนี

### ประสบการณ์สนทนา
- **อ้างอิงอย่างเป็นธรรมชาติ**: อ้างอิงบันทึกเมื่อเกี่ยวข้องจริงเท่านั้น
- **รักษาความลื่นไหล**: อย่าให้อ้างอิงขัดจังหวะจังหวะของบทสนทนา
- **ใส่ใจคุณภาพ**: ให้ความสำคัญกับการเชื่อมโยงที่มีความหมายมากกว่าปริมาณ

## การบำรุงรักษาและอัปเดต

### เพิ่มบันทึกใหม่
หลังจากวางบันทึกใหม่ไว้ในไดเรกทอรี `notes/` แล้ว ให้บอก AI ว่า:

```
我刚刚在 notes 里添加了新笔记，请更新向量数据库
```

AI agent จะวิเคราะห์โน้ตใหม่โดยอัตโนมัติและอัปเดตดัชนี

### อัปเดตโปรไฟล์
คุณสามารถแก้ไขไฟล์โปรไฟล์ในไดเรกทอรี `config/` ได้โดยตรง หรือบอก AI:

```
请根据我最近的笔记内容，更新 user-persona.md 和 ai-persona.md
```

### สร้างดัชนีใหม่
เมื่อโครงสร้างโน้ตมีการเปลี่ยนแปลงครั้งใหญ่ ให้แจ้ง AI:

```
请重新初始化向量数据库
```

AI agent จะวิเคราะห์บันทึกทั้งหมดใหม่และสร้างดัชนีขึ้นมาใหม่

## ข้อควรระวัง

- **การรันครั้งแรก**: AI agent จะดาวน์โหลดโมเดลฝังตัวโดยอัตโนมัติ (ประมาณ 4.3GB) ขณะสร้างฐานข้อมูลเวกเตอร์ครั้งแรก โปรดรออย่างใจเย็น
- **สภาพแวดล้อม Python**: AI agent จะสร้างสภาพแวดล้อมเสมือนและติดตั้งไลบรารีที่จำเป็นโดยอัตโนมัติ
- **การจัดเก็บข้อมูล**: ข้อมูลทั้งหมด (บันทึก, โปรไฟล์, ฐานข้อมูลเวกเตอร์) จะถูกจัดเก็บไว้ที่ root directory ของโปรเจกต์ ไม่ใช่ในไดเรกทอรีของ skill package
- **ตำแหน่งของสกิล**: โปรดตรวจสอบให้แน่ใจว่า skill package อยู่ในไดเรกทอรี `.claude/skills/ai-partner-chat/`
