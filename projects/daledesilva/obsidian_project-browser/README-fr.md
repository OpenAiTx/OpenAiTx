# Navigateur de Projet Obsidian
Un plugin pour [Obsidian](https://obsidian.md) qui remplace votre nouvelle fenêtre d’onglet par une présentation en cartes des fichiers dans chaque dossier, organisés par état. La vue est filtrable grâce à un champ de recherche intégré (commencez simplement à taper !), et les fichiers peuvent se voir attribuer un état via un menu visible dans la note.

## 💾 Installation
Vous pouvez trouver ce plugin dans le répertoire des plugins d’Obsidian.
<details>
<summary>Cliquez pour de l’aide à l’installation des plugins</summary>

1. Ouvrez votre coffre Obsidian et allez dans **Paramètres**.

2. Cliquez sur **Plugins Communautaires** dans la barre latérale.

3. Si ce n’est pas déjà fait, vous devrez activer les plugins communautaires.

4. Recherchez 'Dale de Silva' et installez **Project Browser**.
</details>

Si vous souhaitez installer de nouvelles versions du plugin encore en phase de test, vous pouvez l’installer via BRAT à la place.
BRAT est un autre plugin communautaire qui permet d’installer une version Beta. Les nouvelles fonctionnalités sont d’abord publiées en version Beta et peuvent prendre de quelques jours à un mois avant d’arriver dans la version standard.
<details>
<summary>Cliquez pour les instructions d'installation de la version Beta</summary>

1. Ouvrez votre coffre Obsidian et allez dans **Paramètres**.
2. Cliquez sur **Plugins communautaires** dans la barre latérale.
3. Activez les plugins communautaires et cliquez sur **Parcourir**.
4. Recherchez et installez **BRAT**.
5. Faites défiler vers le bas et **activez** BRAT.
6. Dans le menu BRAT dans le panneau latéral, sélectionnez **Ajouter un plugin Beta**.
7. Suivez les instructions présentées.
8. Lorsqu'une URL est demandée, utilisez : `https://github.com/daledesilva/obsidian_project-browser/`

</details>
<details>
<summary>Cliquez pour les instructions de mise à jour de la version Beta</summary>

- BRAT est configuré pour mettre à jour les plugins Beta par défaut au démarrage, cependant, cela peut parfois prendre un certain temps.
- Pour forcer une mise à jour, exécutez la commande Obsidian de BRAT `Choisir un seul plugin à mettre à jour` et choisissez Project Browser.
</details>
