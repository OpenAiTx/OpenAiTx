# VLM-Gemini
基于Google的Gemini模型API开发VLM

根目录下建立一个.env的环境文件，里面配置好APIkey（GOOGLE_API_KEY = 自己的KEY）

根目录下运行时先创建虚拟环境，之后激活虚拟环境，运行Python app.py
