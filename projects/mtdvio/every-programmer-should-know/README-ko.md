매우 주관적입니다 :bomb:. 과학적으로 뒷받침되지 않습니다.
특별한 순서 없이 나열되어 있습니다 :recycle:

맘에 드시나요? :star: 를 누르고 [공유](https://twitter.com/mr_mig_by/status/900735231552098306)하여 개발자 친구들과 나누세요!
