# Dartotsu

**Dartotsu** は [Dantotsu](https://github.com/rebelonion/Dantotsu/) を Flutter で完全に書き直したものです。これは [**AniList**](https://anilist.co/) と [**MyAnimeList**](https://myanimelist.net/) のハイブリッドトラッキングクライアントで、[**Simkl**](https://simkl.com/) のサポートもあります！  
> [!WARNING]
> Dartotsu は視聴管理・トラッキング専用です。
>
> ストリーミングコンテンツや拡張機能のホスト、提供、管理は**行っていません**。
>
> 利用者の行為はすべて自己責任です。開発者は誤用や法的問題について一切の責任を負いません。法的な懸念は第三者へお問い合わせください。Dartotsu では対応できません。
