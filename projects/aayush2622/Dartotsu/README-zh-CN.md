# Dartotsu

**Dartotsu** 是对 [Dantotsu](https://github.com/rebelonion/Dantotsu/) 的完整重写，采用 Flutter 开发。它是一个混合的 [**AniList**](https://anilist.co/) 和 [**MyAnimeList**](https://myanimelist.net/) 追踪客户端，同时支持 [**Simkl**](https://simkl.com/)！  
> [!WARNING]
> Dartotsu 仅用于追踪和管理。
>
> 它**不**托管、提供或维护任何流媒体内容或扩展。
>
> 用户需对自己的行为全权负责。开发者对误用或法律问题不承担任何责任。法律问题应向第三方来源咨询，而非 Dartotsu。
