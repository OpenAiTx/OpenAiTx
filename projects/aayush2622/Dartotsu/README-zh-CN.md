# Dartotsu

**Dartotsu** 是 [Dantotsu](https://github.com/rebelonion/Dantotsu/) 的完全重写版，使用 Flutter 实现。它是一个混合型 [**AniList**](https://anilist.co/) 和 [**MyAnimeList**](https://myanimelist.net/) 追番客户端，同时也支持 [**Simkl**](https://simkl.com/)!  
> [!WARNING]
> Dartotsu 仅用于追番与管理。
>
> 它**不**托管、提供或维护任何流媒体内容或扩展。
>
> 用户需对自己的行为承担全部责任。开发者对任何滥用或法律问题免责。法律相关问题请联系第三方资源，非 Dartotsu。
