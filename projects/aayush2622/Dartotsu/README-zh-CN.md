# Dartotsu

**Dartotsu** 是基于 Flutter 对 [Dantotsu](https://github.com/rebelonion/Dantotsu/) 的完全重写。它是一个混合型 [**AniList**](https://anilist.co/) 和 [**MyAnimeList**](https://myanimelist.net/) 跟踪客户端，并支持 [**Simkl**](https://simkl.com/)！  
> [!WARNING]
> Dartotsu 仅用于跟踪和管理。
>
> 它**不**托管、提供或维护任何流媒体内容或扩展。
>
> 用户需对其行为承担全部责任。开发者对任何滥用或法律问题概不负责。法律相关问题请联系第三方资源，勿向 Dartotsu 询问。
