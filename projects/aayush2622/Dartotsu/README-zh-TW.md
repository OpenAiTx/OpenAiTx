# Dartotsu

**Dartotsu** 是以 Flutter 重新開發的 [Dantotsu](https://github.com/rebelonion/Dantotsu/) 完整版本。這是一個混合型 [**AniList**](https://anilist.co/) 與 [**MyAnimeList**](https://myanimelist.net/) 追番客戶端，同時支援 [**Simkl**](https://simkl.com/)!  
> [!WARNING]
> Dartotsu 僅用於追蹤與管理。
>
> 它**不**提供、主辦或維護串流內容或擴充功能。
>
> 用戶對其行為負全責。開發者對於任何濫用或法律問題不承擔責任。法律相關疑慮請聯絡第三方來源，非 Dartotsu。
