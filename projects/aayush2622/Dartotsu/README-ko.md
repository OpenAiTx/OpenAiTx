# Dartotsu

**Dartotsu**는 [Dantotsu](https://github.com/rebelonion/Dantotsu/)를 Flutter로 완전히 새롭게 작성한 버전입니다. [**AniList**](https://anilist.co/)와 [**MyAnimeList**](https://myanimelist.net/)를 동시에 지원하는 하이브리드 추적 클라이언트이며, [**Simkl**](https://simkl.com/)도 지원합니다!  
> [!WARNING]
> Dartotsu는 추적 및 관리 용도로만 사용됩니다.
>
> 스트리밍 콘텐츠나 확장 기능을 호스팅, 제공, 유지하지 **않습니다**.
>
> 사용자는 자신의 행동에 대해 전적으로 책임을 집니다. 개발자는 오용이나 법적 문제에 대해 모든 책임을 부인합니다. 법적 문제는 Dartotsu가 아닌 제3자 소스에 문의하시기 바랍니다.
