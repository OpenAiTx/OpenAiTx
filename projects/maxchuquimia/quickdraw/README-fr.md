# Quick Draw

![capture d’écran](https://raw.githubusercontent.com/Jugale/quickdraw/master/AppStore/Screenshot%201.png)

Avez-vous besoin d’un moyen rapide, simple et discret pour mettre en valeur du contenu lors de screencasts ?
Plutôt que de faire tourner votre souris de manière désordonnée, essayez QuickDraw !

Lorsque QuickDraw est actif, vous pouvez tracer des lignes, des flèches, des rectangles et des cercles par-dessus d’autres fenêtres à l’écran. Appuyez sur la touche Échap pour effacer le dessin, puis appuyez de nouveau dessus pour revenir immédiatement à votre point d’origine sans perdre le rythme !

### Utilisation
Raccourcis clavier
```
Key/s       Function
~~~~~       ~~~~~~~~
12345       Select colours 1 to 5
L           Line mode - hold Shift while drawing for straight lines
A           Arrow mode - hold Shift while drawing for straight arrows
R           Rectangle mode - hold Shift while drawing to fill
C           Circle mode - hold Shift while drawing to fill
⌘C          Copy screenshot to Clipboard (permissions may be requested the first time)
⌘S          Save a screenshot to Finder
⌘Z          Undo
Space       Hold whilst drawing to adjust object position
⌫           Backspace to delete all drawings below the mouse
/           Show or hide the startup message (persisted)
⌥/          Show or hide the toolbar (persisted)
ESC         Clear all drawings. Press again to hide Quick Draw
```

## Motivation
1. Je mentor un cours en ligne de Swift et je souhaite un moyen rapide, facile et discret de mettre en évidence et de décrire du code lors des appels vidéo
2. Je voudrais augmenter mon expérience en développement macOS (je suis développeur iOS)
3. Je voudrais expérimenter la création d’une architecture simple sans dépendances tierces
