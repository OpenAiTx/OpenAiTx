## このカードデッキは何ですか？  
WCAG 2.2カードデッキは、週末のプロジェクトとして複数の優れたアクセシビリティリソースを単一のツールキットにまとめるために始まりました：  
- WCAG 2.2の全成功基準の要約  
- 理解しやすい説明で  
- 実用的なテーマごとにテスト用にグループ化（'[WCAG 2.2 map by theme](https://andrewhick.com/accessibility/wcag-map/)', '[WCAG 2.2 map by theme - AAA version](https://andrewhick.com/aaa/)', '[WCAG decision tree](https://github.com/andrewhick/accessibility/wiki/WCAG-decision-tree)' Andrew Hickによる）  
- 各基準を満たすことで恩恵を受ける人を示す指標を含む（Charlie Triplettの[Atomic A11Y](https://www.atomica11y.com/)、[Appt.org](https://appt.org/en/)、[IBMアクセシビリティ要件](https://www.ibm.com/able/requirements/requirements/)に基づく）  
- 主に責任を持つ役割を強調（GOV.UKの[WCAG Primer](https://alphagov.github.io/wcag-primer/)、メルボルン大学の'[Accessibility Responsibilities by Project Role](https://www.unimelb.edu.au/accessibility/project-management-old/responsibility-breakdown/print-version)'チェックリストに基づく）  
- 一緒に機能する成功基準の提案  
  
  
## 使い方は？  
WCAG 2.2カードデッキを使って、ワークショップ、アクセシビリティ監査、トレーニング、その他ご自由にご利用ください。  
  
  
## 今後の予定は？  
LinkedInでの初回発表後の素晴らしいフィードバックのおかげで、将来のバージョンに追加予定の機能リストがあります：  
- 多言語対応（既に計画中：日本語、アラビア語、スウェーデン語、ノルウェー語、カタルーニャ語 - 今後も増加予定）  
- 方法の説明追加（カードデッキを最大限活用する方法 - 例：トレーニング/学習やアクセシビリティ監査向け）  
- 各成功基準のテストに使えるツールの追加（関連外部ソースも含む）  
- カードデッキと一緒に使えるツールの参照を含む  
- WCAG 2.2カードデッキのアクセシブルなHTML版（公開GitHubリポジトリ上；CC BY-SA 4.0ライセンス）  
- WCAGカードデッキのポスター  
- 'WCAG 2.2 map by theme'ポスター（地下鉄路線図のようにカードを可視化）  
- 各成功基準の良い例/悪い例（カードを使った自主改善や支援付きトレーニング向け）  
- 無視した場合に生じる障壁についての情報  
- …その他多数
