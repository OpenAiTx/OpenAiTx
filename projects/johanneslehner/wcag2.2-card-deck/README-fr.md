## De Quoi S’agit-il dans Ce Jeu de Cartes ?  
Le jeu de cartes WCAG 2.2 a démarré comme un projet de week-end pour combiner plusieurs excellentes ressources sur l’accessibilité en un seul kit :  
- un résumé de tous les critères de succès WCAG 2.2  
- en descriptions faciles à comprendre  
- groupés par thème pratique pour les tests (basé sur la « [carte WCAG 2.2 par thème](https://andrewhick.com/accessibility/wcag-map/) », la « [carte WCAG 2.2 par thème - version AAA](https://andrewhick.com/aaa/) » et l'« [arbre de décision WCAG](https://github.com/andrewhick/accessibility/wiki/WCAG-decision-tree) » par Andrew Hick)  
- incluant des indicateurs de qui bénéficie de la satisfaction de chaque critère (basé sur [Atomic A11Y](https://www.atomica11y.com/) par Charlie Triplett, [Appt.org](https://appt.org/en/) et [exigences d’accessibilité IBM](https://www.ibm.com/able/requirements/requirements/))  
- mettant en lumière les rôles principalement responsables (basé sur le [WCAG Primer](https://alphagov.github.io/wcag-primer/) du GOV.UK et la checklist « [Responsabilités en accessibilité par rôle de projet](https://www.unimelb.edu.au/accessibility/project-management-old/responsibility-breakdown/print-version) » de l’Université de Melbourne)  
- suggérant quels critères de succès fonctionnent le mieux ensemble  
  
  
## Comment l’Utiliser ?  
Utilisez le jeu de cartes WCAG 2.2 pour animer des ateliers, des audits d’accessibilité, des formations, ou pour tout autre usage.  
  
  
## Quelles Sont les Prochaines Étapes ?  
Grâce à des retours incroyables après mon annonce initiale sur LinkedIn, j’ai déjà une liste de fonctionnalités que je prévois d’ajouter dans les prochaines versions :  
- Plus de langues (déjà prévues : japonais, arabe, suédois, norvégien, catalan - d’autres suivront)  
- Ajouter des descriptions de méthodes (comment tirer le meilleur parti du jeu de cartes - par ex. pour la formation/l’apprentissage ou les audits d’accessibilité)  
- Ajouter des outils pouvant être utilisés pour tester chaque critère de succès (y compris des sources externes pertinentes)  
- Inclure des références à des outils pouvant être utilisés avec le jeu de cartes  
- Version HTML accessible du jeu de cartes WCAG 2.2 (sur un dépôt GitHub public ; sous licence CC BY-SA 4.0)  
- Affiches du jeu de cartes WCAG  
- Une affiche « Carte WCAG 2.2 par thème » (visualiser les cartes comme une ligne de métro)  
- Exemples bons/mauvais pour chaque critère de succès (pour s’améliorer de façon autonome ou pour des formations facilitées avec les cartes)  
- Informations sur les barrières créées si ignorées  
- ... et plus encore
