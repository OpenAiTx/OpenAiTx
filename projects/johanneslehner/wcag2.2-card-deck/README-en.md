## What Is This Card Deck About?  
The WCAG 2.2 Card Deck started as a weekend project to combine multiple great accessibility resources into a single toolkit:  
- a summary of all WCAG 2.2 success criteria  
- in easy to understand descriptions  
- grouped by practical theme for testing (based on '[WCAG 2.2 map by theme](https://andrewhick.com/accessibility/wcag-map/)', '[WCAG 2.2 map by theme - AAA version](https://andrewhick.com/aaa/)' and '[WCAG decision tree](https://github.com/andrewhick/accessibility/wiki/WCAG-decision-tree)' by Andrew Hick)  
- including indicators who benefits from meeting each criterion (based on [Atomic A11Y](https://www.atomica11y.com/) by Charlie Triplett, [Appt.org](https://appt.org/en/), and [IBM accessibility requirements](https://www.ibm.com/able/requirements/requirements/))  
- highlighting which roles are primarily responsible (based on the GOV.UK [WCAG Primer](https://alphagov.github.io/wcag-primer/), and the '[Accessibility Responsibilities by Project Role](https://www.unimelb.edu.au/accessibility/project-management-old/responsibility-breakdown/print-version)' checklist from the University of Melbourne)  
- suggesting which success criteria work best together  
  
  
## How To Use?  
Use the WCAG 2.2 Deck to run workshops, accessibility audits, training, or whatever you can use it for.  
  
  
## What's Up Next?  
Thanks to amazing feedback after my initial announcement on LinkedIn, I already have a list of features that I am planning to add in future versions:  
- More languages (already planned: Japanese, Arabian, Swedish, Norwegian, Catalan - more to come)  
- Add method descriptions (how to get most out of the card deck - e.g. for training/learning, or accessibility audits)  
- Add tools that can be used to test for each success criteria (including relevant external sources)  
- Include references to tools that can be used together with the Card Deck  
- Accessible HTML version of the WCAG 2.2 Card Deck (on a public GitHub repository; licensed under CC BY-SA 4.0)  
- WCAG Card Deck posters  
- A 'WCAG 2.2 map by theme' poster (visualize the cards as underground line)  
- Good/bad examples for each success criteria (to improve independently or facilitated training with the cards)  
- Information about what barriers are created if ignored  
- ... and more
