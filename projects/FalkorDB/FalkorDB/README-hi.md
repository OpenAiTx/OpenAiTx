### MotoGP ‡§≤‡•Ä‡§ó ‡§â‡§¶‡§æ‡§π‡§∞‡§£

‡§á‡§∏ ‡§â‡§¶‡§æ‡§π‡§∞‡§£ ‡§Æ‡•á‡§Ç, ‡§π‡§Æ [FalkorDB Python client](https://pypi.org/project/FalkorDB/) ‡§ï‡§æ ‡§â‡§™‡§Ø‡•ã‡§ó ‡§ï‡§∞‡§ï‡•á ‡§è‡§ï ‡§õ‡•ã‡§ü‡§æ ‡§ó‡•ç‡§∞‡§æ‡§´ ‡§¨‡§®‡§æ‡§è‡§Ç‡§ó‡•á, ‡§ú‡§ø‡§∏‡§Æ‡•á‡§Ç MotoGP ‡§≤‡•Ä‡§ó ‡§Æ‡•á‡§Ç ‡§≠‡§æ‡§ó ‡§≤‡•á‡§®‡•á ‡§µ‡§æ‡§≤‡•á ‡§ï‡•Å‡§õ ‡§Æ‡•ã‡§ü‡§∞‡§∏‡§æ‡§á‡§ï‡§ø‡§≤ ‡§∞‡§æ‡§á‡§°‡§∞‡•ç‡§∏ ‡§î‡§∞ ‡§ü‡•Ä‡§Æ‡•ã‡§Ç ‡§ï‡§æ ‡§â‡§™‡§∏‡§Æ‡•Å‡§ö‡•ç‡§ö‡§Ø ‡§¶‡§∞‡•ç‡§∂‡§æ‡§Ø‡§æ ‡§ó‡§Ø‡§æ ‡§π‡•à‡•§ ‡§ó‡•ç‡§∞‡§æ‡§´ ‡§¨‡§®‡§æ‡§®‡•á ‡§ï‡•á ‡§¨‡§æ‡§¶, ‡§π‡§Æ ‡§â‡§∏‡§ï‡•á ‡§∏‡§Ç‡§∞‡§ö‡§®‡§æ ‡§î‡§∞ ‡§∏‡§Ç‡§¨‡§Ç‡§ß‡•ã‡§Ç ‡§ï‡§æ ‡§™‡§§‡§æ ‡§≤‡§ó‡§æ‡§®‡•á ‡§ï‡•á ‡§≤‡§ø‡§è ‡§°‡•á‡§ü‡§æ ‡§ï‡•ç‡§µ‡•á‡§∞‡•Ä ‡§ï‡§∞‡•á‡§Ç‡§ó‡•á‡•§

res = g.query("""MATCH (r:Rider)-[:rides]->(t:Team {name:'Ducati'})
