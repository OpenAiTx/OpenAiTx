# AnymeX: マルチサービストラッキングクライアント

**AnymeX** は、[**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/)、[**Simkl**](https://simkl.com/) でのアニメおよびマンガの進捗管理・トラッキング専用に設計されたマルチサービストラッキングクライアントです。

> [!重要]  
> **AnymeXはトラッキングツールのみです。** ストリーミングコンテンツ、メディア、またはサードパーティ拡張機能のホスト、提供、配布、または推奨は行いません。  
>  
> **ユーザーの責任:** アプリやサードパーティサービス・拡張機能の利用方法はユーザー自身の責任です。ユーザーはすべての適用法、著作権、知的財産権を遵守しなければなりません。  
>  
> **免責事項:** AnymeXの開発者は、ユーザーの行為による不正使用、法的問題、違反について一切の責任を負いません。法的な問題（DMCA請求など）は、各サードパーティサービスに直接お問い合わせください。アプリは「現状のまま」保証なしで提供されます。  
>  
> **サービス:** AnymeXは、対応サービス（AniList、MyAnimeList、Simkl）の公式APIのみと連携します。サードパーティ拡張機能はその作成者の責任であり、AnymeX開発者は関与しません。  
>  
> **AnymeXを利用することで、[利用規約](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)の遵守に同意したことになります。DMCA準拠のトラッキング機能や、AniList・MyAnimeList・Simkl以外のコンテンツ・サービスへの非関与についてご確認ください。**
