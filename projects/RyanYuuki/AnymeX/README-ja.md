# AnymeX

**AnymeX** は、[**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/)、[**Simkl**](https://simkl.com/) でアニメとマンガの進捗を管理・追跡するために設計されたマルチサービス対応のトラッキングクライアントです。

> [!IMPORTANT]  
> **AnymeX はトラッキング専用ツールです。** 本アプリは **いかなるストリーミングコンテンツ、メディア、またはサードパーティ拡張機能のホスティング、提供、配信、推奨も行いません。**  
> 本アプリはサポートされているサービス（AniList, MyAnimeList, Simkl）の公式APIのみと連携し、トラッキング目的で動作します。  
> 利用者は本アプリおよび任意で利用するサードパーティサービスや拡張機能の使用に関する一切の責任を負います。AnymeXの開発者は、利用者の行為による誤用・法的問題・違反に関して一切の責任を明確に否認します。法的な問題については、それぞれのサードパーティサービスへお問い合わせください。AnymeXは関与しません。  
>  
> **本アプリを利用することで、[利用規約](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md) に同意したものとみなします。**
