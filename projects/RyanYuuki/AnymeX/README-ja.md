# AnymeX: マルチサービストラッキングクライアント

**AnymeX** は、[**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/)、[**Simkl**](https://simkl.com/) のアニメおよびマンガの進捗管理・追跡専用に設計されたマルチサービストラッキングクライアントです。

> [!IMPORTANT]  
> **AnymeXはトラッキングツール専用です。** ストリーミングコンテンツ、メディア、またはサードパーティ拡張機能のホスト、提供、配布、推奨は行いません。  
>  
> **ユーザー責任:** アプリおよびサードパーティサービス・拡張機能の利用は、ユーザー自身の責任です。利用者は全ての適用法令、著作権および知的財産権を順守しなければなりません。  
>  
> **免責事項:** AnymeXの開発者は、誤用、法的問題、ユーザーによる違反に関する一切の責任を負いません。法的懸念は各サードパーティサービスにご相談ください。アプリは「現状のまま」保証なしで提供されます。  
>  
> **サービス:** AnymeXは、対応サービス（AniList、MyAnimeList、Simkl）の公式APIのみと連携します。サードパーティ拡張機能は、その作成者の責任であり、AnymeXの開発者は一切責任を負いません。  
>  
> **本アプリの利用により、[利用規約](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)に同意したものとみなします。**
