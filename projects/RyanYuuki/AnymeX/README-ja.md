# AnymeX

**AnymeX** は、[**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/)、および [**Simkl**](https://simkl.com/) に対応したマルチサービス追跡クライアントです！

> [!NOTE]
> AnymeX は追跡目的のみを意図しています。
>
> ストリーミングコンテンツや拡張機能をホスト、提供、維持するものではありません。
>
> ユーザーはこのアプリの使用方法に全責任を負います。開発者は誤用や法的問題に対して一切の責任を負いません。法的な問題はAnymeXではなく第三者にお問い合わせください。
