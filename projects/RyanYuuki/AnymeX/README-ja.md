# AnymeX: 複数サービス対応トラッキングクライアント

**AnymeX** は、[**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/)、[**Simkl**](https://simkl.com/) でアニメとマンガの進捗を一元管理・トラッキングするために設計された複数サービス対応のトラッキングクライアントです。

> [!重要]  
> **AnymeX はトラッキングツールのみです。** 本アプリは**ストリーミングコンテンツ、メディア、サードパーティ拡張機能のホスティング、提供、配布、推奨は一切行いません。**  
> 本アプリはトラッキング目的でサポート対象サービス（AniList、MyAnimeList、Simkl）の公式APIのみと連携します。  
> アプリやサードパーティサービス・拡張機能の利用はすべてユーザーの責任となります。AnymeX 開発者は、ユーザーの行動による不正利用や法的問題、違反等一切の責任を明確に否認します。法的な懸念については、必ず各サードパーティサービスにお問い合わせください。AnymeX では対応できません。  
>  
> **本アプリを利用することで、[利用規約](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)に同意したものとみなされます。**
