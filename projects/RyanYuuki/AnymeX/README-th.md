# AnymeX: ลูกค้าติดตามหลายบริการ

**AnymeX** เป็นโปรแกรมติดตามหลายบริการที่ออกแบบมาโดยเฉพาะสำหรับการจัดการและติดตามความคืบหน้าอะนิเมะและมังงะผ่าน [**AniList**](https://anilist.co/), [**MyAnimeList**](https://myanimelist.net/) และ [**Simkl**](https://simkl.com/)

> [!IMPORTANT]  
> **AnymeX เป็นเครื่องมือสำหรับติดตามเท่านั้น** ไม่ได้โฮสต์ ให้บริการ แจกจ่าย หรือสนับสนุนเนื้อหาสตรีมหรือส่วนขยายของบุคคลที่สามใด ๆ  
>  
> **ความรับผิดชอบของผู้ใช้:** ผู้ใช้ต้องรับผิดชอบต่อการใช้งานแอปและบริการ/ส่วนขยายของบุคคลที่สามที่เลือกใช้งานเอง ผู้ใช้ต้องปฏิบัติตามกฎหมาย ลิขสิทธิ์ และสิทธิ์ในทรัพย์สินทางปัญญาทั้งหมดที่เกี่ยวข้อง  
>  
> **ไม่มีความรับผิดชอบ:** ทีมพัฒนา AnymeX ปฏิเสธความรับผิดชอบต่อการใช้งานผิดกฎหมายหรือปัญหาทางกฎหมายที่เกิดจากการกระทำของผู้ใช้ หากมีปัญหาทางกฎหมาย รวมถึง DMCA ให้ติดต่อกับบริการบุคคลที่สามที่เกี่ยวข้อง ไม่ใช่ AnymeX แอปให้บริการแบบ "ตามสภาพ" โดยไม่มีการรับประกันใด ๆ  
>  
> **บริการ:** AnymeX เชื่อมต่อเฉพาะกับ API อย่างเป็นทางการของบริการที่รองรับ (AniList, MyAnimeList, Simkl) ส่วนขยายของบุคคลที่สามเป็นความรับผิดชอบของผู้สร้าง ไม่ใช่ทีมงาน AnymeX  
>  
> **โดยการใช้ AnymeX คุณตกลงที่จะปฏิบัติตาม [ข้อกำหนดการให้บริการ](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md) ของเรา โปรดอ่านข้อกำหนดเพื่อเข้าใจฟังก์ชันการติดตามที่สอดคล้องกับ DMCA และการไม่เกี่ยวข้องกับเนื้อหาหรือบริการอื่นนอกเหนือจาก AniList, MyAnimeList และ Simkl**
