# AnymeX

**AnymeX** 是一款多服務追蹤客戶端，專為在 [**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/) 及 [**Simkl**](https://simkl.com/) 管理與追蹤動畫與漫畫進度而設計。

> [!IMPORTANT]  
> **AnymeX 僅為追蹤工具。** 它**不會**託管、提供、分發或支持任何串流內容、媒體或第三方擴充功能。  
> 本應用僅與所支援服務（AniList、MyAnimeList、Simkl）之官方 API 整合，用於進度追蹤。  
> 用戶需自行負責其對本應用及任何第三方服務或擴充功能的使用。AnymeX 開發者明確聲明對任何濫用、法律問題或因用戶行為產生之違規不承擔任何責任。所有法律問題應直接聯繫相關第三方服務，而非 AnymeX。  
>  
> **使用本應用即表示您同意遵守我們的[服務條款](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)。**
