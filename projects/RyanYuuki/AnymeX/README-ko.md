# AnymeX

**AnymeX**는 [**AniList**](https://anilist.co/), [**MyAnimeList**](https://myanimelist.net/), 그리고 [**Simkl**](https://simkl.com/)에서 애니메이션 및 만화 진척도를 관리하고 추적할 수 있도록 설계된 멀티서비스 트래킹 클라이언트입니다.

> [!IMPORTANT]  
> **AnymeX는 오직 트래킹 도구입니다.** 이 앱은 **어떠한** 스트리밍 콘텐츠, 미디어, 또는 서드파티 확장 기능을 호스팅, 제공, 배포 또는 지지하지 **않습니다**.  
> 앱은 트래킹 목적으로 지원 서비스(AniList, MyAnimeList, Simkl)의 공식 API만을 통합합니다.  
> 사용자는 앱 및 사용자가 선택한 서드파티 서비스나 확장 기능 사용에 대해 전적으로 책임집니다. AnymeX 개발자는 사용자 행동으로 인한 모든 오용, 법적 문제, 혹은 위반에 대해 명시적으로 모든 책임을 부인합니다. 법적 문제가 발생할 경우 해당 서드파티 서비스에 문의해야 하며, AnymeX에는 문의하지 마십시오.  
>  
> **이 앱을 사용함으로써, 귀하는 우리의 [서비스 이용약관](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)에 동의하게 됩니다.**
