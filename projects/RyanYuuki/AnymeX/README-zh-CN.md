# AnymeX

**AnymeX** 是一款多服务追踪客户端，支持 [**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/) 和 [**Simkl**](https://simkl.com/)！

> [!NOTE]
> AnymeX 仅用于追踪用途。
>
> 它**不**托管、提供或维护任何流媒体内容或扩展。
>
> 用户须自行对如何使用本应用负责。开发者对任何误用或法律问题不承担任何责任。法律问题应向第三方来源咨询，而非 AnymeX。
