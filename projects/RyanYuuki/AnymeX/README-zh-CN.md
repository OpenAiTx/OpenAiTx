# AnymeX：多服务追踪客户端

**AnymeX** 是一款专为管理和追踪动画及漫画进度而设计的多服务追踪客户端，支持 [**AniList**](https://anilist.co/)、[**MyAnimeList**](https://myanimelist.net/) 和 [**Simkl**](https://simkl.com/)。

> [!重要]  
> **AnymeX 仅为追踪工具。** 它不托管、提供、分发或认可任何流媒体内容、媒体或第三方扩展。  
>  
> **用户责任：** 用户对如何使用本应用及选择使用的任何第三方服务或扩展负全部责任。用户必须遵守所有相关法律、版权及知识产权。  
>  
> **免责声明：** AnymeX 开发者不对因用户行为而产生的误用、法律问题或违规行为承担任何责任。法律相关问题应由第三方服务解决，而非 AnymeX。该应用按“原样”提供，不附带任何保证。  
>  
> **服务说明：** AnymeX 仅集成受支持服务（AniList、MyAnimeList、Simkl）的官方 API。第三方扩展由其开发者负责，与 AnymeX 开发者无关。  
>  
> **使用本应用即表示您同意遵守我们的[服务条款](https://raw.githubusercontent.com/RyanYuuki/AnymeX/main/./TOS.md)。**
