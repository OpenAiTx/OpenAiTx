# PieJam
**PieJam** 是一款灵活的、以触摸屏为主的 Linux 音频混音器。

它专为 7 英寸 Raspberry Pi 触摸屏设计，目标是在配备 USB 音频接口的 Raspberry Pi 上作为独立混音器运行。虽然这是主要目标，但 PieJam 并不限于 Raspberry Pi，也可以运行在其他 Linux 系统上。

PieJam 支持动态音频路由、每通道处理、MIDI 控制和会话录制，适合现场演出、排练设置和嵌入式音频系统。

有关构建完整 Raspberry Pi 系统映像，请参阅配套的 [PieJam OS](https://github.com/nooploop/piejam_os) 仓库。

更多[文档](https://piejam.readthedocs.io/en/latest/)

## 功能

### 混音与路由
* 动态输入/输出配置
* 灵活的通道路由
* 声像、立体声平衡、音量
* 静音和独奏

### 效果与处理
* 每通道效果链
* 内置模块：双声像、滤波器、实用工具
* 分析工具：示波器、频谱分析仪、调音器
* LADSPA 插件支持

### 控制与自动化
* MIDI CC/弯音控制参数

### 会话
* 会话录制
* 会话管理
