# PieJam
**PieJam**은 Linux용 유연한 터치스크린 우선 오디오 믹서입니다.

7인치 Raspberry Pi 터치스크린을 위해 특별히 설계되었으며, USB 오디오 인터페이스와 함께 Raspberry Pi에서 독립형 믹서로 실행하는 것을 목표로 합니다. 이것이 주요 대상이지만, PieJam은 Raspberry Pi에 국한되지 않고 다른 Linux 시스템에서도 실행할 수 있습니다.

PieJam은 동적 오디오 라우팅, 채널별 처리, MIDI 제어 및 세션 녹음을 지원하여 라이브 공연, 리허설 설정 및 임베디드 오디오 시스템에 적합합니다.

완전한 Raspberry Pi 기반 시스템 이미지를 빌드하려면, 함께 제공되는 [PieJam OS](https://github.com/nooploop/piejam_os) 저장소를 참조하십시오.

더 많은 [문서](https://piejam.readthedocs.io/en/latest/)

## 기능

### 믹싱 및 라우팅
* 동적 입력/출력 구성
* 유연한 채널 라우팅
* 패닝, 스테레오 밸런스, 볼륨
* 음소거 및 솔로

### 효과 및 처리
* 채널별 FX 체인
* 내장 모듈: 듀얼 패닝, 필터, 유틸리티
* 분석 도구: 오실로스코프, 스펙트럼 분석기, 튜너
* LADSPA 플러그인 지원

### 제어 및 자동화
* MIDI CC/피치벤드 파라미터 제어

### 세션
* 세션 녹음
* 세션 관리
