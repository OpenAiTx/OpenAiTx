# PieJam
**PieJam** は、Linux向けの柔軟でタッチスクリーン優先のオーディオミキサーです。

これは特に7インチのRaspberry Piタッチスクリーン用に設計されており、USBオーディオインターフェースと組み合わせたRaspberry Pi上でスタンドアロンのミキサーとして動作することを目的としています。これが主な対象ですが、PieJamはRaspberry Piに限定されず、他のLinuxシステム上でも動作します。

PieJamは動的なオーディオルーティング、チャンネルごとの処理、MIDIコントロール、セッション録音をサポートしており、ライブパフォーマンス、リハーサルセットアップ、組み込みオーディオシステムに適しています。

完全なRaspberry Piベースのシステムイメージの構築については、付属の[PieJam OS](https://github.com/nooploop/piejam_os)リポジトリを参照してください。

詳細な[ドキュメント](https://piejam.readthedocs.io/en/latest/)

## 機能

### ミキシング＆ルーティング
* 動的な入力/出力設定
* 柔軟なチャンネルルーティング
* パンニング、ステレオバランス、ボリューム
* ミュートとソロ

### エフェクト＆プロセッシング
* チャンネルごとのFXチェーン
* 内蔵モジュール：デュアルパン、フィルター、ユーティリティ
* 分析ツール：オシロスコープ、スペクトラムアナライザー、チューナー
* LADSPAプラグイン対応

### コントロール＆オートメーション
* MIDI CC/ピッチベンドパラメーターコントロール

### セッション
* セッション録音
* セッション管理
