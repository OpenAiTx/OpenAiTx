# PieJam
**PieJam** es un mezclador de audio flexible, diseñado principalmente para pantallas táctiles en Linux.

Fue diseñado específicamente para la pantalla táctil Raspberry Pi de 7", con el objetivo de funcionar como un mezclador independiente en un Raspberry Pi junto con una interfaz de audio USB. Aunque este es el objetivo principal, PieJam no está limitado al Raspberry Pi y también puede ejecutarse en otros sistemas Linux.

PieJam soporta enrutamiento dinámico de audio, procesamiento por canal, control MIDI y grabación de sesiones, haciéndolo adecuado para actuaciones en vivo, configuraciones de ensayo y sistemas de audio embebidos.

Para construir una imagen completa del sistema basada en Raspberry Pi, consulte el repositorio adjunto [PieJam OS](https://github.com/nooploop/piejam_os).

Más [documentación](https://piejam.readthedocs.io/en/latest/)

## Características

### Mezcla y enrutamiento
* Configuración dinámica de entrada/salida
* Enrutamiento flexible de canales
* Paneo, balance estéreo, volumen
* Silenciar y solo

### Efectos y procesamiento
* Cadenas de efectos por canal
* Módulos incorporados: Paneo dual, Filtro, Utilidad
* Herramientas de análisis: Osciloscopio, Analizador de espectro, Afinador
* Soporte para plugins LADSPA

### Control y automatización
* Control de parámetros MIDI CC/Pitchbend

### Sesiones
* Grabación de sesiones
* Gestión de sesiones
