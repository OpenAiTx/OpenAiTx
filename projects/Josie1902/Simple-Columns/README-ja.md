## 📝 説明
**Simple Columns** プラグインは、Obsidianのノート内で簡単にリサイズ可能かつカスタマイズ可能なカラムを作成できます。タスクを整理したり、コンテンツを分割したり、アイデアを視覚化したりする際に、このプラグインはシンプルなマークダウン構文で素早くレイアウトの調整と構造化を可能にします。

## ⚙️ 使い方 
シンプルなカラムレイアウトを作成するには、次のいずれかを行います:
- ノート内で右クリックし、サブメニューからオプションを選択します。

![サブメニュー](https://drive.google.com/uc?export=view&id=1Nu2tmvENJFtsnbDIqnk4XqT05WSJexN_)

- コマンドパレットを使ってカラムを直接挿入します。例えば、「2カラム追加」と入力すると、ノートに2カラムのレイアウトが挿入されます。

カラムを作成すると、プラグインは各カラムにユニークIDを付けた以下のマークダウンを生成します:

```columns
id: b52707b0-ce15-6458-825d-32615b4b7h85
===
Column 1

===
Column 2
```

### 例：コードブロックを使ったカラムの作成
コードブロックをカラム内に配置するレイアウトを作成するには、次の構文を使用します（4つのバッククォート ```` で開始および終了）：

````columns

id: b52707b0-ce15-6458-825d-32615b4b7h85
===
カラム 1
```javascript
console.log("Hello world")
```
===
カラム 2
````
`````

### 例：YAMLによる列幅の指定

特定の列に対してカスタム幅をYAMLで設定できます。比率が指定された列が優先され、比率が定義されていない列は残りのスペースを均等に分け合います。

```
id: b52707b0-ce15-6458-825d-32615b4b7h85
column-1-ratio: 60%
===
Column 1
hello
===
Column 2
```
説明：
- column-1-ratio: 60% → カラム1はコンテナ幅の60%を占めます。
- カラム2（指定なし）は自動的に残りの40%を埋めます。
- 複数のカラムに対して比率を定義できます（column-2-ratio、column-3-ratioなど）、最大4カラムまで。

## ✨ 機能
- **ユニークなカラムID**：各カラムはユニークなIDで生成され、カラムを個別にスタイル設定や参照できます。
- **簡単にサイズ変更可能なカラム**：カラムは完全にリサイズ可能で、コンテンツのレイアウトを自由にコントロールできます。
- **レスポンシブレイアウト**：画面サイズに応じてカラムが自動調整され、クリーンで適応性の高いレイアウトを提供します。
- **高いカスタマイズ性**：カラムは完全に調整可能で、サイズ、間隔、配置を簡単にコントロールしてニーズに合わせられます。

## 🔧 グローバル設定

**グローバル設定**では、Obsidianボールト全体のすべてのカラムに対するデフォルトのスタイルや動作を定義できます。これらの設定はプラグイン設定内でアクセス・構成可能です。

![グローバル設定](https://drive.google.com/uc?export=view&id=1WPBg0Y3pffL5pw1dVRUC81TMMlaod-eK)

### コンテナのボーダー
カラムコンテナの外観をボーダーでカスタマイズし、視認性を向上させられます。

- **コンテナボーダーを表示**：この設定を切り替えて各カラムコンテナの周囲にボーダーを表示します。
- **ボーダー幅**：カラムコンテナのボーダー幅を設定します。
- **ボーダーカラー**：ボーダーの色を選択します。
- **ボーダーの透明度**：RGBAカラーコードを使ってボーダーの透明度を調整します。アルファ値（最後の数字）が透明度を制御します。

### リサイズバーの設定
カラムのリサイズバーの外観と動作もカスタマイズできます。

- **リサイズバーを表示**：この設定を切り替えてカラム間のリサイズバーを表示または非表示にします。リサイズバーはカラムの区切りにカーソルを合わせた時のみ表示されます。
- **リサイズバー幅**：リサイズバーの幅を設定します。
- **リサイズバーカラー**：リサイズバーの色を選択します。
- **リサイズバーの透明度**：RGBAカラーコードを使ってリサイズバーの透明度を調整します。

### 詳細設定
- **スタイルのリセット**：ボーダー設定やリサイズバーのスタイルを含むすべてのカラムスタイルをデフォルト値に戻します。
- **ローカルストレージのクリア**：すべてのカスタム設定を削除し、プラグインを初期状態に戻します。問題が発生した場合や設定をやり直したい場合に使用してください。

## ⚡️ ローカル設定


**ローカル設定**では、ノート内の個々の列に対してより**詳細なカスタマイズ**を適用できます。

![ローカル設定](https://drive.google.com/uc?export=view&id=1q11NA49CeizEAozavC1YAiiSI7J7oe5U)

### 利用可能なローカル設定：
- **すべてのスタイルをリセット**：列のすべてのカスタムスタイルをデフォルト設定に戻します。
- **境界線を表示**
- **境界線の色と透明度を設定**
- **リサイズハンドルを表示**：列のリサイズハンドルの表示を切り替えます。リサイズハンドルはホバー時のみ表示されます。
- **リサイズハンドルの色と透明度を設定**
- **テキストの色を設定**
- **背景色と透明度を設定**
- **テキストの配置を設定**：列内のテキストの配置を設定します（例：`左寄せ`、`中央揃え`、`右寄せ`）。

## 📦 インストール
このプラグインは現在Obsidianコミュニティプラグインで利用可能です！ 🎉

1. Obsidianで、設定 > コミュニティプラグイン に移動します。
2. ブラウズをクリックし、Simple Columnsを検索します。
3. インストールをクリックし、プラグインを有効にします。
