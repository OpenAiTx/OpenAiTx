# 설명
**CKPE**는 공유 소스의 강력한 플랫폼으로, 베데스다의 Creation Kit 편집기에 수많은 수정 및 개선을 제공하며, **Skyrim Special Edition**, **Fallout 4**, **Starfield**와 같은 게임 편집기 지원을 포함하고, [SSE CKFixes](https://github.com/Nukem9/skyrimse-test), [SSE CKFixes Update](https://github.com/Perchik71/SkyrimSETest), [FO4 CKFixes](https://github.com/Perchik71/Fallout4Test) 프로젝트의 후속작이며, 작성자 **perchik71**가 시작한 [SSE CK 1.5.73용 유니코드 패치](https://github.com/Perchik71/usse_test)도 포함합니다.  

[위키](https://github.com/Perchik71/Creation-Kit-Platform-Extended/wiki) 페이지

# 설치
### 새로 설치
지원되는 [게임](https://github.com/Perchik71/Creation-Kit-Platform-Extended/wiki#brief) 중 하나를 다운로드하고 Creation Kit를 설치하세요.  
게임에 맞는 **CKPE**의 [최신 릴리스](https://github.com/Perchik71/Creation-Kit-Platform-Extended/releases)를 다운로드하세요.  
모든 내용을 게임 디렉터리에 압축 해제하여 **ckpe_loader.exe**가 **CreationKit.exe**와 동일 폴더에 있도록 하세요.
### 업데이트
게임에 맞는 **CKPE**의 [최신 릴리스](https://github.com/Perchik71/Creation-Kit-Platform-Extended/releases)를 다운로드하세요.  
모든 내용을 게임 디렉터리에 압축 해제하여 **ckpe_loader.exe**가 **CreationKit.exe**와 동일 폴더에 있도록 하세요.  
모든 파일 교체에 동의하세요.
### 컴파일
일반 사용자는 아직 릴리스되지 않은 최신 기능에 접근하기 위해 이 방법만 필요합니다.  
**`Visual Studio 2022`** 이상에서 프로젝트를 열고, **`Release`** 모드 또는 **`Release-NoAVX2`**로 빌드하세요.  
그런 다음 빌드 출력물을 게임 디렉터리에 이동하여 **ckpe_loader.exe**가 **CreationKit.exe**와 동일 폴더에 있도록 하세요.  
이전 파일 교체 요청 시 모두 교체에 동의하세요.
### 자동 빌드 (베타)
게임에 맞는 **CKPE**의 [최신 커밋 빌드](https://github.com/Perchik71/Creation-Kit-Platform-Extended/actions)를 다운로드하세요.  
모든 내용을 게임 디렉터리에 압축 해제하여 **ckpe_loader.exe**가 **CreationKit.exe**와 동일 폴더에 있도록 하세요.  
가능한 오류들.
