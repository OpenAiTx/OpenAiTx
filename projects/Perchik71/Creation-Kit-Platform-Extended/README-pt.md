# Descrição
**CKPE** é uma poderosa plataforma de código compartilhado que realiza inúmeras correções e melhorias no editor Creation Kit da **Bethesda**, incluindo suporte ao editor para jogos como **Skyrim Special Edition**, **Fallout 4**, **Starfield**, sendo um sucessor dos projetos [SSE CKFixes](https://github.com/Nukem9/skyrimse-test), [SSE CKFixes Update](https://github.com/Perchik71/SkyrimSETest), [FO4 CKFixes](https://github.com/Perchik71/Fallout4Test), além do [Patch Unicode para SSE CK 1.5.73](https://github.com/Perchik71/usse_test) que iniciou tudo para o autor **perchik71**.  

Página da [Wiki](https://github.com/Perchik71/Creation-Kit-Platform-Extended/wiki)

# Instalação
>[!IMPORTANTE]
>Lançamento regular ou noavx2?  
>Verifique com a Steam se seu processador suporta avx2. Abra a Steam, no canto esquerdo passe o mouse sobre o menu ajuda e selecione Informações do Sistema, depois procure por AVX2.  
>Se for suportado, utilize a versão regular  
>Se não for suportado, utilize a versão noavx2  
### Instalação Nova
Baixe qualquer [jogo suportado](https://github.com/Perchik71/Creation-Kit-Platform-Extended/wiki#brief) e instale o Creation Kit.  
Baixe a [última versão](https://github.com/Perchik71/Creation-Kit-Platform-Extended/releases) do **CKPE** para o seu jogo.  
Extraia todos os conteúdos no diretório do jogo, de modo que o **ckpe_loader.exe** esteja na mesma pasta que o **CreationKit.exe**.
### Atualização
Baixe a [última versão](https://github.com/Perchik71/Creation-Kit-Platform-Extended/releases) do **CKPE** para o seu jogo.  
Extraia todos os conteúdos no diretório do jogo, de modo que o **ckpe_loader.exe** esteja na mesma pasta que o **CreationKit.exe**.  
Concorde em substituir todos os arquivos.
### Compilação
Usuários comuns só precisam deste método para acessar recursos mais recentes que ainda não foram lançados.  
Abra o projeto no **`Visual Studio 2022`** ou superior. Compile o projeto no modo **`Release`** ou **`Release-NoAVX2`**.  
Depois, mova o resultado da compilação para o diretório do jogo, de modo que o **ckpe_loader.exe** esteja na mesma pasta que o **CreationKit.exe**.  
Se solicitado para substituir arquivos antigos, concorde em substituir todos.
### Automatizar builds (Beta)
Baixe a [build mais recente do commit](https://github.com/Perchik71/Creation-Kit-Platform-Extended/actions) do **CKPE** para o seu jogo.  
Extraia todos os conteúdos no diretório do jogo, de modo que o **ckpe_loader.exe** esteja na mesma pasta que o **CreationKit.exe**.  
Possíveis erros.
