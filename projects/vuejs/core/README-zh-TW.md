## 問題諮詢

如需諮詢或支援，請使用[官方論壇](https://forum.vuejs.org)或[社群聊天室](https://chat.vuejs.org/)。本倉庫的 issue 清單**僅限**用於錯誤回報和功能請求。

## 問題回報

請務必遵守問題提交要求，並在提交問題時使用[新問題助手](https://new-issue.vuejs.org/)。不符合指南的問題可能會被立即關閉。
