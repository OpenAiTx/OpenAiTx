### Introducción：

#### 🔥🔥🔥Uso de MyBatisPlus para operar MongoDB con elegancia
  
* #### Como el viento que se cuela en la noche
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Puede combinarse con frameworks Mongo existentes, sin intrusiones
* #### Simplificación del desarrollo
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Sólo requiere una configuración sencilla para realizar operaciones CRUD rápidas, llamada encadenada más conveniente

### Ventajas：

* Sin intrusión: solo mejora sin cambiar, introducirlo no afecta el proyecto existente, tan suave como la seda
* Bajo consumo: inyecta automáticamente CRUD básico al iniciar, rendimiento prácticamente sin pérdidas, operación directamente orientada a objetos
* Poderosas operaciones CRUD: Servicio genérico, con solo una pequeña configuración se pueden realizar la mayoría de operaciones CRUD de una sola tabla, además de un potente constructor de condiciones que satisface diversas necesidades
* Soporte para llamadas en forma Lambda: mediante expresiones Lambda, se facilita escribir todo tipo de condiciones de consulta, sin preocuparse por errores en los nombres de campos
* Soporta generación automática de claves primarias: hasta 5 estrategias de clave primaria (incluye generador de ID único distribuido - Sequence), configurable libremente, solución perfecta para claves primarias
* Soporte para operaciones sin necesidad de clases de entidad
* Soporte para fuentes de datos dinámicas
* Soporte para eliminación lógica, prevención de actualización y eliminación de colecciones completas, relleno automático, entre otras funciones
