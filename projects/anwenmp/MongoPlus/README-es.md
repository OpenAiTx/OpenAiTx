### Introducción：

#### 🔥🔥🔥Usa MongoDB de forma elegante al estilo de MyBatisPlus
  
* #### Se integra sutilmente
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Puede combinarse con frameworks MongoDB existentes, sin ser invasivo
* #### Simplifica el desarrollo
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Solo requiere una configuración sencilla para realizar operaciones CRUD rápidamente, con llamadas encadenadas más cómodas

### Ventajas：

* No invasivo: solo agrega mejoras, sin cambios, integrarlo no afecta el proyecto existente, funcionamiento suave como la seda
* Bajo consumo: inyecta automáticamente operaciones básicas CURD al iniciar, sin afectar el rendimiento, operaciones orientadas a objetos directamente
* Potente CRUD: Service universal, con poca configuración permite la mayoría de operaciones CRUD sobre una sola tabla, además de potentes constructores de condiciones para todo tipo de necesidades
* Soporte para llamadas en Lambda: expresiones Lambda para construir consultas fácilmente, sin preocuparse por errores de campo
* Soporte para generación automática de claves primarias: hasta 5 estrategias de clave primaria (incluye generador distribuido de IDs únicos - Sequence), configurable, solución perfecta para claves primarias
* Soporta operaciones sin clase de entidad
* Soporta fuente de datos dinámica
* Soporta borrado lógico, previene actualizaciones/eliminaciones masivas, autocompletado y más
