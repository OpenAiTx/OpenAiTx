### Описание:

#### 🔥🔥🔥Используйте MongoDB так же элегантно, как MyBatisPlus
  
* #### Проникает, как ветер ночью
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Может использоваться совместно с существующими Mongo-фреймворками, неинвазивен
* #### Упрощение разработки
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Требуется лишь простая настройка для быстрой работы с CRUD-операциями, удобные цепочные вызовы

### Преимущества:

* Неинвазивен: только расширяет, ничего не меняет, интеграция не влияет на существующий проект, работает плавно как шелк
* Минимальные издержки: базовые CURD внедряются автоматически при запуске, практически не влияет на производительность, работа напрямую с объектами
* Мощные CRUD-операции: универсальный сервис, большинство CRUD-операций для одной таблицы доступны после минимальной настройки, мощный конструктор условий для любых задач
* Поддержка вызова в стиле Lambda: благодаря лямбда-выражениям легко формировать запросы, не нужно беспокоиться об ошибках в названиях полей
* Поддержка автогенерации первичного ключа: до 5 стратегий генерации ключей (включая распределённый уникальный генератор ID - Sequence), настраивается, полностью решает вопросы с ключами
* Поддержка операций без сущностных классов
* Поддержка динамического источника данных
* Поддержка логического удаления, предотвращения обновления/удаления всей коллекции, авто-заполнение и другие функции

MyBatisPlus позволяет не писать избыточные SQL-запросы, и Mongo-Plus тоже!
