### 소개:

#### 🔥🔥🔥MyBatisPlus 방식으로 MongoDB를 우아하게 조작

* #### 밤에 몰래 스며들다
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;기존 Mongo 프레임워크와 결합 사용 가능, 비침투성
* #### 개발 간소화
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;간단한 설정만으로 빠르게 CRUD 작업 가능, 체인 호출이 더욱 편리함

### 장점:

* 비침투성: 강화만 하고 변경은 하지 않으므로, 도입해도 기존 프로젝트에 영향 없음, 실크처럼 매끄러움
* 낮은 오버헤드: 시작 시 자동으로 기본 CURD 주입, 성능 손실 거의 없음, 객체 지향 직접 조작 가능
* 강력한 CRUD 작업: 범용 Service, 소량의 설정만으로 단일 테이블 대부분의 CRUD 작업 가능, 강력한 조건 생성기로 다양한 요구 충족
* Lambda 형태 호출 지원: Lambda 표현식으로 다양한 조회 조건을 손쉽게 작성, 필드 오타 걱정 없음
* 자동 주키 생성 지원: 5가지 이상의 주키 전략(분산 고유 ID 생성기 - Sequence 포함) 지원, 자유롭게 설정 가능, 주키 문제 완벽 해결
* 엔티티 클래스 없이도 작업 지원
* 동적 데이터 소스 지원
* 논리 삭제, 전체 컬렉션 업데이트/삭제 방지, 자동 채우기 등 다양한 기능 지원
