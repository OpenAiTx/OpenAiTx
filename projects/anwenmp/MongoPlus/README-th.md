### แนะนำ:

#### 🔥🔥🔥ใช้งาน MongoDB ได้อย่างสง่างามในสไตล์ของ MyBatisPlus
  
* #### ซึมซับเข้าไปอย่างแนบเนียน
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;สามารถใช้งานร่วมกับเฟรมเวิร์ก MongoDB เดิมได้โดยไม่รุกล้ำ
* #### ลดความซับซ้อนในการพัฒนา
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;เพียงแค่ตั้งค่าบางอย่าง ก็สามารถทำ CRUD ได้อย่างรวดเร็ว เรียกใช้งานแบบ chain ได้ง่ายยิ่งขึ้น

### ข้อดี:

* ไม่มีการรุกล้ำ: เสริมความสามารถโดยไม่เปลี่ยนแปลงโครงสร้างเดิม การนำเข้าจะไม่ส่งผลกระทบต่อโปรเจกต์ที่มีอยู่ ใช้งานได้อย่างลื่นไหล
* ประสิทธิภาพสูง: เริ่มต้นใช้งานจะ inject CURD พื้นฐานให้อัตโนมัติ ประสิทธิภาพไม่ลดลง สามารถใช้งานแบบอิงอ็อบเจกต์ได้โดยตรง
* การทำ CRUD ที่ทรงพลัง: Service ทั่วไป สามารถทำ CRUD ส่วนใหญ่ในตารางเดียวได้ด้วยการตั้งค่าน้อยมาก พร้อมด้วยตัวสร้างเงื่อนไขที่ทรงพลัง รองรับทุกความต้องการในการใช้งาน
* รองรับการเรียกแบบ Lambda: ใช้ Lambda expressions เพื่อเขียนเงื่อนไขการค้นหาได้ง่าย ไม่ต้องกังวลเรื่องการเขียนชื่อฟิลด์ผิด
* รองรับการสร้าง Primary Key อัตโนมัติ: รองรับกลยุทธ์ primary key ถึง 5 แบบ (มีตัวสร้าง ID แบบแจกจ่าย - Sequence) ตั้งค่าได้อย่างอิสระ แก้ปัญหา primary key ได้อย่างสมบูรณ์
* รองรับการใช้งานโดยไม่ต้องมี Entity class
* รองรับ Data Source แบบไดนามิก
* รองรับการลบแบบตรรกะ, ป้องกันการอัปเดตและลบทั้งหมด, การเติมข้อมูลอัตโนมัติ และฟีเจอร์อื่น ๆ
