### Introdução:

#### 🔥🔥🔥Use MongoDB de forma elegante, como o MyBatisPlus
  
* #### Silencioso como a noite
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Pode ser usado junto com frameworks Mongo já existentes, sem invasão
* #### Simplifica o desenvolvimento
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Com uma configuração simples, é possível realizar operações CRUD rapidamente, chamadas em cadeia são ainda mais fáceis

### Vantagens:

* Não intrusivo: apenas faz melhorias, sem alterar nada, sua inclusão não afeta projetos existentes, funcionamento suave como seda
* Baixo consumo: injeta automaticamente as operações CURD básicas ao iniciar, desempenho praticamente sem perdas, operação orientada a objetos direta
* Operações CRUD poderosas: Service genérico, com pouca configuração realiza a maioria das operações CRUD em tabela única, além de um poderoso construtor de condições, atendendo diversas necessidades
* Suporte para chamadas em Lambda: usando expressões Lambda, facilita a escrita de condições de consulta, sem se preocupar em errar nomes de campos
* Suporte para geração automática de chave primária: até 5 estratégias de chave primária (inclui gerador de ID único distribuído - Sequence), configuração livre, solução perfeita para chave primária
* Suporte para operações sem classe de entidade
* Suporte para fonte de dados dinâmica
* Suporte para exclusão lógica, prevenção de atualizações e exclusões em toda a coleção, preenchimento automático e muito mais

### Desenvolvimento rápido

MyBatisPlus permite não precisar escrever muitos comandos SQL como no MyBatis, e o Mongo-Plus também!
