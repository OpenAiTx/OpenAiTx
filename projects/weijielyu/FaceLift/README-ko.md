> *FaceLift*는 단일 얼굴 이미지를 고품질 3D 가우시안 헤드 표현으로 변환하며, 실제 인간 이미지에 대해 놀랄 만큼 잘 일반화됩니다.

이것은 *FaceLift*의 자체 재구현입니다. 코드는 곧 공개될 예정입니다.

그동안, *FaceLift*를 직접 사용해 보고 싶으시면, wlyu3@ucmerced.edu로 이미지를 보내주십시오. 저희가 처리하여 결과를 보내드리겠습니다.
