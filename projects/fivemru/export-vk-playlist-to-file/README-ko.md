# ВКонтакте(ВК)에서 Яндекс 음악으로 음악 옮기는 방법

내 플레이리스트를 VK에서 텍스트 파일로 내보내고 야ндекс 음악으로 가져오기.
스크립트는 자신의 플레이리스트가 아니어도 작동할 수 있습니다.

```quote
Страница импорта в яндекс музыке поменялась на https://music.yandex.ru/newimport
Теперь там вроде можно импортировать через ВК сразу, не проверял.
```
## VK 음악을 텍스트 파일로 내보내기

VK에서 자신의 음악 페이지를 열고 `F12`를 누릅니다.

https://raw.githubusercontent.com/fivemru/export-vk-playlist-to-file/main/chrome-snippet.js
스크립트를 복사하여 VK 페이지 콘솔에 붙여넣고 `Enter`를 누릅니다.

*스크립트가 페이지를 끝까지 스크롤하여 전체 목록을 불러오고 `vk-playlist.txt` 파일로 저장합니다.*
