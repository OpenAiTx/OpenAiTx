# Cómo transferir música de VKontakte (VK) a Yandex Music

Exportar tu lista de reproducción de VK a un archivo de texto e importarla a Yandex Music.  
El script también puede funcionar en listas de reproducción que no sean propias.  

```quote
Страница импорта в яндекс музыке поменялась на https://music.yandex.ru/newimport
Теперь там вроде можно импортировать через ВК сразу, не проверял.
```

## Exportar música de VK a un archivo de texto

Abrir en VK la página con tu música, presionar `F12`.

Copiar el script de la página
https://raw.githubusercontent.com/fivemru/export-vk-playlist-to-file/main/chrome-snippet.js
pegarlo en la consola de la página de VK, y presionar `Enter`.

*El script desplazará la página hasta el final para cargar toda la lista y la guardará en el archivo `vk-playlist.txt`*
