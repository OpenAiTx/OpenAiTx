# How to transfer music from VKontakte (VK) to Yandex Music

Export your playlist from VK to a text file and import it into Yandex Music.
The script may also work with playlists that are not your own.

```quote
Страница импорта в яндекс музыке поменялась на https://music.yandex.ru/newimport
Теперь там вроде можно импортировать через ВК сразу, не проверял.
```

## Export VK Music to a Text File

Open your VK music page, press `F12`.

Copy the script from the page
https://raw.githubusercontent.com/fivemru/export-vk-playlist-to-file/main/chrome-snippet.js
paste it into the console on the VK page, and press `Enter`.

*The script will scroll the page to the end to load the entire list and save it to the file `vk-playlist.txt`*
