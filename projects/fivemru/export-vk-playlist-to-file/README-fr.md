# Comment transférer de la musique de VKontakte (VK) vers Yandex Musique

Exporter sa playlist de VK dans un fichier texte et importer dans Yandex Musique.  
Le script peut aussi fonctionner avec des playlists qui ne sont pas les vôtres.

```quote
Страница импорта в яндекс музыке поменялась на https://music.yandex.ru/newimport
Теперь там вроде можно импортировать через ВК сразу, не проверял.
```

## Exporter la musique VK dans un fichier texte

Ouvrez la page VK avec votre musique, appuyez sur `F12`.

Copiez le script depuis la page
https://raw.githubusercontent.com/fivemru/export-vk-playlist-to-file/main/chrome-snippet.js
collez-le dans la console de la page VK, puis appuyez sur `Entrée`.

*Le script fera défiler la page jusqu'en bas pour charger toute la liste et enregistrera dans le fichier `vk-playlist.txt`*
