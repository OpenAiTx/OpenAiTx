# Florida

Podążaj za [FRIDA](https://github.com/frida/frida) upstream, aby automatycznie łatać i budować wersję frida-server dla Androida odporną na wykrywanie.

跟随 FRIDA 上游自动修补程序，并为 Android 构建反检测版本的 frida-server。

**Podpowiedź: Nie forkuj tego repozytorium**
