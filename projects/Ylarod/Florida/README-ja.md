# Florida

[FRIDA](https://github.com/frida/frida) の上流に従い、Android 用の frida-server のアンチディテクト版を自動パッチ・ビルドします。

FRIDA の上流の自動パッチプログラムを追跡し、Android 用のアンチディテクトバージョンの frida-server を構築します。
