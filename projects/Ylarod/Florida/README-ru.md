# Florida

Следите за [FRIDA](https://github.com/frida/frida) upstream для автоматического патчинга и сборки антидетектированной версии frida-server для Android.

跟随 FRIDA 上游自动修补程序，并为 Android 构建反检测版本的 frida-server。
