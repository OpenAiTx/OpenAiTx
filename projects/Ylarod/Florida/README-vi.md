# Florida

Theo dõi [FRIDA](https://github.com/frida/frida) upstream để tự động vá lỗi và xây dựng phiên bản frida-server chống phát hiện cho Android.

跟随 FRIDA 上游自动修补程序，并为 Android 构建反检测版本的 frida-server。

**Gợi ý: Không fork kho lưu trữ này**
