# Florida

ติดตาม [FRIDA](https://github.com/frida/frida) อัปสตรีมเพื่อทำการแพตช์และสร้าง frida-server เวอร์ชันป้องกันการตรวจจับสำหรับ Android โดยอัตโนมัติ

跟随 FRIDA 上游自动修补程序，并为 Android 构建反检测版本的 frida-server。
