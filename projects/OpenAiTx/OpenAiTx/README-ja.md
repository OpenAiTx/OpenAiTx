## はじめに

**OpenAiTx** は、GitHub プロジェクトの readme や wiki 向けの20言語自動翻訳ツールです。

## なぜ？

1. プレミアムグレードの LLM を用いた一度きりの翻訳で、オープンソースコミュニティに結果を無料で公開できます。
2. Google/Bingなど複数言語のSEO検索に対応。クライアント翻訳ツールでは実現できません。
3. 永久に無料＆オープンソース
4. 一度提出すれば自動で将来の更新も反映。プロジェクト管理者の時間を大幅に節約します。


## はじめ方

1. URL の `GitHub` を `OpenAiTx` に置き換えます。例: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 好きなスタイルバッジやテキストをクリックしてコピーします。

または

1. https://openaitx.github.io/ にアクセス
2. プロジェクトリンクを提出

## 計算リソースの貢献

このプロジェクトに貢献したい場合、必要な手順は以下の通りです：
プロジェクトをフォーク → プロジェクトをクローン → 自分の言語のスクリプトを選択 → AIトークンを入力 → 実行 → コミット & プッシュ & PR作成

> 注意: **トークンは絶対にアップロードしないでください！**

## 注意事項

- GitHubマークダウンのみサポートしています。
- プロジェクトオーナーまたはコア開発者の場合、GitHubアカウントを認証することでインデックス後に自動でPRを作成できます。

## その他

- 1プロジェクトあたり10～30分
- 1コンソールにつき1日200～400プロジェクト
- 既存プロジェクトは3～7日ごとに更新されます（将来の計算リソースにより変動）
