# Open Ai Tx

![](https://openaitx.github.io/logo_crop.png)

**OpenAiTx** は、あなたのGitHubプロジェクトのために20言語の翻訳済みREADMEを自動作成します。プロジェクトのURLの `GitHub` を `OpenAiTx` に置き換えるだけで利用できます。例: https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx 。

## サポート/貢献

プロジェクトに貢献したい場合、必要な手順は以下の通りです：
Fork プロジェクト => プロジェクトをクローン => お好きな言語のスクリプトを選択 => ご自身のAIトークンを入力 => 実行 => コミット＆プッシュ＆PR作成

注意：**絶対にトークンをアップロードしたり漏洩しないでください！**
