## はじめに

**OpenAiTx** は、あなたのGitHubプロジェクトのREADMEやWiki向けの20言語自動翻訳ツールです。

## なぜ？

1. 一度の翻訳で高品質なLLMを利用し、その結果をオープンソースコミュニティに無料で公開できます。
2. Google/Bingの多言語SEO検索をサポートし、クライアント側の翻訳ツールではできないことが可能です。
3. 永久に無料＆オープンソース
4. 一度提出すれば自動で将来の更新も行われるため、プロジェクト管理者の時間を節約できます。


## はじめ方

1. URLの`GitHub`を`OpenAiTx`に置き換えます。例: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  

2. 好きなスタイルのバッジやテキストをクリックしてコピーします。

## 計算リソースへの貢献

プロジェクトへの貢献を希望される場合は、以下の手順を行ってください：
プロジェクトをフォーク → クローン → 自分の言語のスクリプトを選ぶ → AIトークンを入力 → 実行 → コミット＆プッシュ＆PR作成

> 注意: **トークンをアップロードしないでください！**

## 注意事項

- GithubのMarkdownのみ対応しています。
- プロジェクトオーナーやコア開発者の場合、Githubアカウントで認証しインデックス後に自動でPRを作成できます。

## その他

- 1プロジェクトあたり10～30分
- 1コンソールあたり1日200～400プロジェクト
- 3～7日ごとに既存プロジェクトを更新（将来の計算リソースにより変動）
