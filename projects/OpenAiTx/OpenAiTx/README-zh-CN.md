# Open Ai Tx

**OpenAiTx** 会自动为你的 GitHub 项目创建 20 种语言的翻译版 README，你只需要将项目 URL 中的 `GitHub` 替换为 `OpenAiTx` 即可。例如：https://github.com/OpenAiTx/OpenAiTx ：替换为 https://openaitx.com/OpenAiTx/OpenAiTx

## 支持/贡献

**本项目不需要你的资金支持，但需要你的计算机算力、PR 贡献和 Token 支持。**

如果你想为项目做贡献，只需要：
Fork 项目 => Clone 项目 => 选择你的语言脚本 => 填入你的 AI token  => 运行脚本 => 提交 & 推送 & 创建 PR

注意：**请不要上传或泄露你的 Token！**



## 其他

- 如果你希望获得高优先级和持续更新，请创建 PR 并将你的项目 github 链接添加到 `priority-list.json`。
- 你可以在 `projects-data.json` 获取索引项目。
- 50 KB 的 readme 使用 GPT4.1 翻译，每种语言约需 `2 分钟`。
