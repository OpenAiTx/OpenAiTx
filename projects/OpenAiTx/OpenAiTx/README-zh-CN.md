## 介绍

**OpenAiTx** 20 种语言自动翻译工具，适用于您的 GitHub 项目 readme 和 wiki。

## 为什么？

1. 一次性翻译，采用高品质 LLM，结果免费开放给开源社区。
2. 支持 Google/Bing 多语言 SEO 搜索，客户端翻译工具无法做到。
3. 永久免费和开源
4. 一次提交，自动未来更新，如果你是项目维护者，可以节省你的时间。


## 入门指南

1. 将 URL 中的 `GitHub` 替换为 `OpenAiTx`，例如 https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 提交您的项目。
3. 点击并复制你喜欢的样式徽章或文本。
4. 更新到你的 readme 文件。

或

1. 访问 https://openaitx.com
2. 提交您的项目链接。
3. 点击并复制你喜欢的样式徽章或文本。
4. 更新到你的 readme 文件。
