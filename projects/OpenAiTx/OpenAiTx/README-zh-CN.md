## 简介

**OpenAiTx** 会为你的 GitHub 项目自动生成 20 种语言的翻译版 README，你只需要将项目 URL 的 `GitHub` 替换为 `OpenAiTx`，例如 https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx 。

## 支持/贡献

如果你想为本项目做出贡献，你需要做的只是：
Fork 项目 => Clone 项目 => 选择你语言的脚本 => 填写你的 AI token => 运行脚本 => Commit & push & 创建 PR

注意：**请不要上传或泄露你的 token！**

### 注意事项

- 目前仅支持 markdown 格式的 readme。
- 推荐图片链接使用完整 URL 路径。

## 其他

- 50 KB 的 readme，GPT4.1 需要 `每种语言 2 分钟`。
- 每天每个控制台可处理 200-400 个项目。
- 每 3-N 天会更新已存在的项目（取决于未来的计算资源）
