## 介绍

**OpenAiTx** 自动为您的 GitHub 项目创建 20 种语言的翻译版自述文件，您只需将项目 URL 中的 `GitHub` 替换为 `OpenAiTx`，例如 https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx 。

## 支持/贡献

如果您想为本项目做出贡献，只需：
Fork 项目 => 克隆项目 => 选择您语言的脚本 => 填入您的 AI 令牌 => 运行 => 提交 & 推送 & 创建 PR

注意：**请勿上传或泄露您的令牌！**

### 说明

- 目前仅支持 markdown 格式的自述文件。
- 推荐图片链接使用完整的 URL 路径。

## 其他

- 50 KB 的自述文件需要 GPT4.1 `每种语言约 2 分钟`。
- 每台控制台每天可处理 200-400 个项目。
- 每 3-N 天会更新已有项目（取决于未来的计算资源😎）。
