## 简介

**OpenAiTx** 是一个为您的 GitHub 项目 readme 和 wiki 提供 20 种语言自动翻译的工具。

## 为什么选择我们？

1. 一次性使用高端 LLM 翻译，并让结果免费服务于开源社区。
2. 支持 Google/Bing 多语言 SEO 搜索，客户端翻译工具无法做到这一点。
3. 永久免费 & 开源
4. 只需提交一次并自动更新后续内容，如果您是项目维护者，这将节省您的时间。


## 开始使用

1. 将 URL 中的 `GitHub` 替换为 `OpenAiTx`，例如 https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 点击并复制您喜欢的样式徽章或文本。

或者

1. 访问 https://openaitx.github.io/ 
2. 提交您的项目链接。

## 计算贡献

如果你希望为本项目做出贡献，只需按照以下步骤操作：
Fork 项目 → Clone 项目 → 选择你的语言脚本 → 填写你的 AI Token → 运行脚本 → Commit & Push & 创建 PR

> 注意：**请不要上传你的 Token！**

## 注意事项

- 仅支持 GitHub Markdown。
- 如果你是项目所有者或核心开发者，可以授权 GitHub 账号，在索引后自动创建 PR。

- 每个项目耗时约 10~30 分钟
- 每台控制台每天处理 200~400 个项目
- 每 3~7 天会更新已存在项目（视未来计算资源而定）
