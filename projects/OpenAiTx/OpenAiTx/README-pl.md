## Wprowadzenie

**OpenAiTx** to narzędzie do automatycznego tłumaczenia na 20 języków dla plików readme w projektach GitHub.



### Funkcje

- Wsparcie dla SEO w wielu językach Google/Bing.
- Darmowe i na zawsze otwarte oprogramowanie 
- Jednorazowe zgłoszenie i automatyczna aktualizacja w przyszłości.   



## Jak zacząć

Zamień URL `GitHub` na `OpenAiTx`, np. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx .

## Współdzielenie wkładu

Jeśli możesz pomóc, zgłaszając PR do swojego ulubionego projektu, bardzo pomoże to w promocji naszego projektu.

Przykład:

```
Tytuł:
docs: dodaj 20 linków językowych do README

Treść:
PR dodaje 20 linków językowych do README i użytkownik może łatwo uzyskać dostęp do przetłumaczonego README, wspiera SEO w wielu językach Google/Bing.

Demo strony https://openaitx.github.io/view.html?user=CherryHQ&project=cherry-studio&lang=ja

> OpenAiTx jest darmowy i open-source : https://github.com/OpenAiTx/OpenAiTx
```



## Wkład obliczeniowy 

Jeśli chcesz mieć wkład w projekt, wystarczy:
Fork projektu → Klonowanie projektu → Wybierz skrypt w swoim języku → Wstaw swój token AI  → Uruchom go → Commit & push & utwórz PR

> Uwaga: **Prosimy nie przesyłać swoich tokenów!**

### Uwagi

- Wspierane tylko markdown readme z githuba.

## Inne

- 10~30 min/projekt
- 200~400 projektów/konsola dziennie.
- Co 3~7 dni aktualizowane są istniejące projekty (w zależności od przyszłych zasobów obliczeniowych)

- Zespół Microsoft MVP udostępnia 150USD/miesiąc budżetu na platformę azure dla społeczności open source.
