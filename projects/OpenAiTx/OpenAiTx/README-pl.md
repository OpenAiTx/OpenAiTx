## Wprowadzenie

**OpenAiTx** narzędzie do automatycznego tłumaczenia na 20 języków dla plików readme Twojego projektu GitHub.



### Funkcje

- Obsługa wyszukiwania w wielu językach Google/Bing.
- Darmowe i otwarte źródło na zawsze
- Jedno zgłoszenie i automatyczna aktualizacja w przyszłości.   



## Jak zacząć

Zamień URL `GitHub` na `OpenAiTx`, np. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx .

## Wsparcie/Wkład 

Jeśli chcesz wnieść wkład do projektu, wystarczy:
Fork projektu → Sklonuj projekt → Wybierz skrypt w swoim języku → Wstaw swój token AI  → Uruchom go → Commit & push & stwórz PR

> Uwaga: **Nie przesyłaj swoich tokenów!**

### Uwagi

- Obsługiwane są tylko pliki readme markdown z githuba.

## Inne

- 10~30 min/ na projekt
- 200~400 projektów/na jedną konsolę dziennie.
- Co 3~7 dni aktualizowane będą istniejące projekty (w zależności od przyszłych zasobów obliczeniowych)


### Podziękowania

- Zespół Microsoft MVP zapewnia 150USD/miesiąc budżetu na otwartą platformę społecznościową Azure.
