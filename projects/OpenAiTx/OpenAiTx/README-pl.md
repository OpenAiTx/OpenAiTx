## Wsparcie/Wkład

**Ten projekt nie potrzebuje Twoich pieniędzy, ale potrzebuje mocy obliczeniowej Twojego komputera, wkładów PR oraz Tokenów.**

Jeśli chcesz wnieść swój wkład do projektu, wystarczy:
Fork projektu => Sklonuj projekt => Wybierz skrypt w swoim języku => Wpisz swój token AI => Uruchom go => Commit & push & utwórz PR

Uwaga: **Nie przesyłaj ani nie udostępniaj swoich tokenów!**

## Inne

- Jeśli chcesz uzyskać wysoką priorytetowość i ciągłe aktualizacje, utwórz PR i dodaj link do swojego projektu github w `priority-list.json`.
- Możesz uzyskać indeks projektów w `projects-data.json`.
- Plik readme 50 KB wymaga `2 min/język` przez GPT4.1.
