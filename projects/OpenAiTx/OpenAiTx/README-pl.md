## Wprowadzenie

**OpenAiTx** narzędzie do automatycznego tłumaczenia readme projektów GitHub na 20 języków.

## Cel

Jednorazowe tłumaczenie przy użyciu najwyższej klasy LLM i udostępnienie wyniku społeczności open-source za darmo.

## Funkcje

- Wsparcie Google/Bing SEO dla wielu języków.
- Całkowicie darmowy i otwartoźródłowy
- Jednorazowe zgłoszenie i automatyczna aktualizacja w przyszłości.   



## Pierwsze kroki

Zamień URL `GitHub` na `OpenAiTx`, np. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx .

## Współdzielenie Wkładu

Jeśli możesz pomóc, przesyłając PR do swojego ulubionego projektu, znacznie pomoże to w promowaniu naszego projektu.

Przykład:

```
Tytuł:
docs: dodano 20 linków językowych do README

Treść:
PR dodaje 20 linków językowych do README, dzięki czemu użytkownik może łatwo uzyskać dostęp do przetłumaczonego READEME, obsługuje SEO wielu języków Google/Bing.

Demo strony https://openaitx.github.io/view.html?user=CherryHQ&project=cherry-studio&lang=ja

> OpenAiTx jest darmowy i open-source : https://github.com/OpenAiTx/OpenAiTx , Cel: Jednorazowe tłumaczenie przy użyciu wysokiej klasy LLM i udostępnienie wyniku społeczności open-source za darmo.
```



## Wkład Obliczeniowy

Jeśli chcesz mieć swój wkład w projekcie, wystarczy:
Fork projektu → Sklonuj projekt → Wybierz skrypt w swoim języku → Wstaw swój token AI  → Uruchom → Zatwierdź & wypchnij & utwórz PR

> Uwaga: **Proszę nie przesyłać swoich tokenów!**

### Uwagi

- Obsługiwane są tylko pliki markdown README z GitHub.

## Inne

- 10~30 min/projekt
- 200~400 projektów/jedna konsola dziennie.
- Co 3~7 dni aktualizowane będą istniejące projekty (w zależności od przyszłych zasobów obliczeniowych)
