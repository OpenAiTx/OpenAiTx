Jeśli chcesz wnieść swój wkład w projekt, wystarczy:
Fork projektu => Sklonuj projekt => Wybierz skrypt w swoim języku => Wprowadź swój token AI => Uruchom go => Commit & push & utwórz PR

Uwaga: **Nie przesyłaj ani nie ujawniaj swoich tokenów!**

### Uwagi

- Obecnie obsługiwane są tylko pliki readme w formacie markdown.
- Zalecane jest używanie pełnych ścieżek URL do obrazów.

## Inne

- Plik readme o rozmiarze 50 KB wymaga `2 min/język` przy użyciu GPT4.1.
- 200-400 projektów/jedna konsola dziennie.
- Co 3-N dni aktualizowane są istniejące projekty (zależnie od przyszłych zasobów obliczeniowych)
