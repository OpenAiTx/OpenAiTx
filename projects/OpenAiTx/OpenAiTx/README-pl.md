## Wsparcie/Wkład

Jeśli chciałbyś wnieść wkład w projekt, wystarczy:
Fork projektu => Sklonuj projekt => Wybierz skrypt w swoim języku => Wprowadź swój token AI => Uruchom => Commit & push & stwórz PR

Uwaga: **Prosimy nie przesyłać ani nie ujawniać swoich tokenów!**

## Inne

- Readme o wielkości 50 KB wymaga `2 minuty/na język` przez GPT4.1. 
- 200-400 projektów/na jedno konsolę dziennie.
- Co 3-N dni następuje aktualizacja istniejących projektów (w zależności od przyszłych zasobów obliczeniowych😎)
