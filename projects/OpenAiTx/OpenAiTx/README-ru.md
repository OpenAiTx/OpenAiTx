# Open Ai Tx

![](https://openaitx.github.io/logo_crop.png)

**OpenAiTx** автоматически создает readme-файлы на 20 языках для вашего проекта на GitHub. Вам нужно лишь заменить URL проекта `GitHub` на `OpenAiTx`, например: https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

## Поддержка/Вклад

Если вы хотите внести вклад в проект, все, что вам нужно сделать:
Сделать форк проекта => Клонировать проект => Выбрать скрипт на вашем языке => Вставить ваш AI токен => Запустить => Сделать commit, push и создать PR

Примечание: **Пожалуйста, не загружайте и не раскрывайте свои токены!**



## Другое

- Для readme размером 50 КБ требуется `2 минуты/на язык` с помощью GPT4.1.
- 200-400 проектов/одна консоль в день.
