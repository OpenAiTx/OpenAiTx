# Open Ai Tx

**OpenAiTx** автоматически создает README-файлы с переводом на 20 языков для вашего проекта на GitHub, вам просто нужно заменить URL проекта `GitHub` на `OpenAiTx`. Например, https://github.com/OpenAiTx/OpenAiTx : заменить на https://openaitx.com/OpenAiTx/OpenAiTx

## Поддержка/Вклад

**Этому проекту не нужны ваши деньги, но ему нужна вычислительная мощность вашего компьютера, PR-вклады и токены.**

Если вы хотите внести вклад в проект, все, что вам нужно сделать:

Fork проекта => Клонировать проект => Выбрать скрипт на вашем языке => Вставить ваш AI-токен => Запустить его => Commit & push & создать PR

Примечание: **Пожалуйста, не загружайте и не раскрывайте ваши токены!**


## Другое

- Если вы хотите получить высокий приоритет и постоянные обновления, создайте PR и добавьте ссылку на ваш проект в github в `priority-list.json`.
- Вы можете получить индекс проектов в `projects-data.json`.
- README размером 50 КБ требует `2 минуты/на язык` с помощью GPT4.1.
