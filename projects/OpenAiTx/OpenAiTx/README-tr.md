# Open Ai Tx

**OpenAiTx**, GitHub projeniz için otomatik olarak 20 dilde çevirilmiş readme dosyası oluşturur, yapmanız gereken tek şey proje URL'sindeki `GitHub` ifadesini `OpenAiTx` ile değiştirmektir. Örneğin: https://github.com/OpenAiTx/OpenAiTx adresini https://openaitx.com/OpenAiTx/OpenAiTx ile değiştirin.

## Destek/Katkı

**Bu projenin paraya ihtiyacı yok, ancak bilgisayarınızın işlem gücüne, PR katkılarına ve Token'lara ihtiyacı var.**

Projeye katkıda bulunmak isterseniz yapmanız gerekenler:
Projeyi çatallayın => Projeyi klonlayın => Dilinizde bir script seçin => AI tokenınızı doldurun => Çalıştırın => Commit & push & PR oluşturun

Not: **Lütfen tokenlarınızı yüklemeyin veya sızdırmayın!**



## Diğer

- Daha yüksek öncelik ve sürekli güncelleme almak isterseniz, lütfen bir PR oluşturun ve proje github bağlantınızı `priority-list.json` dosyasına ekleyin.
- `projects-data.json` dosyasından indeks projeleri alabilirsiniz.
- 50 KB'lık bir readme, GPT4.1 ile `her dil için 2 dakika` sürer.
