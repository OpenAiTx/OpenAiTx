## Giriş

**OpenAiTx**, GitHub projeniz için otomatik olarak 20 dilde çevrilmiş readme dosyaları oluşturur, yapmanız gereken tek şey proje URL'sindeki `GitHub` kelimesini `OpenAiTx` ile değiştirmektir, örn. https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

Projeye katkıda bulunmak isterseniz yapmanız gerekenler:
Projeyi çatallayın => Projeyi klonlayın => Dilinize uygun bir script seçin => AI tokenınızı girin => Çalıştırın => Commit & push yapın & PR oluşturun

## Diğer

- 50 KB'lık bir readme için GPT4.1 ile `dil başına 2 dakika` gerekir.
- Günde 200-400 proje/tek konsol.
- Her 3-N günde bir mevcut projeler güncellenir (gelecekteki hesaplama kaynaklarına bağlı olarak😎)
