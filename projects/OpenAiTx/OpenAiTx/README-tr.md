## Destek/Katkı

Projeyi çatallayın (Fork) => Projeyi klonlayın => Kendi dilinizde bir betik seçin => AI tokeninizi girin => Çalıştırın => Commit & push & PR oluşturun

Not: **Lütfen tokenlerinizi yüklemeyin veya sızdırmayın!**
