## Giriş

**OpenAiTx** GitHub projenizin readme & wiki'si için 20 dilde otomatik çeviri aracı.

## Neden?

1. Üst düzey LLM ile tek seferlik çeviri ve sonucu açık kaynak topluluğuna ücretsiz sunma.
2. Google/Bing çoklu dil SEO arama desteği, istemci çeviri aracı bunu yapamaz.
3. Sonsuza dek Ücretsiz & Açık Kaynak 
4. Bir kez gönderin ve otomatik güncelleyin, bir proje yöneticisiyseniz zaman kazandırır.


## Başlarken

1. URL'de `GitHub` yerine `OpenAiTx` yazın, ör. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Projenizi gönderin.
3. Beğendiğiniz stil rozetlerini veya metni tıklayıp kopyalayın.
4. Readme dosyanızı güncelleyin.

veya

1. https://openaitx.com adresine erişin
2. Proje bağlantınızı gönderin.
3. Beğendiğiniz stil rozetlerini veya metni tıklayıp kopyalayın.
4. Readme dosyanızı güncelleyin.
