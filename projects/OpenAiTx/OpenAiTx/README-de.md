## Einführung

**OpenAiTx** 20-Sprachen-Auto-Übersetzungstool für das README deines GitHub-Projekts.

## Warum?

1. Einmalige Übersetzung mit Premium-LLM und mache das Ergebnis der Open-Source-Community frei zugänglich.
2. Unterstützt Google/Bing SEO-Suchanfragen in mehreren Sprachen, was Client-Übersetzungstools nicht können.
3. Für immer kostenlos & Open Source 
4. Einmal einreichen und automatische Updates in der Zukunft, das spart Zeit für Maintainer von Projekten.


## Erste Schritte

1. Ersetze die URL `GitHub` durch `OpenAiTx`, z.B. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Klicke und kopiere das Stil-Badge oder den Text, der dir gefällt; das System erstellt nach der Indexierung automatisch einen PR für das Projekt.

Wenn du im Projekt beitragen möchtest, musst du nur Folgendes tun:
Projekt forken → Projekt klonen → Wähle ein Skript in deiner Sprache → Trage deinen AI-Token ein → Führe es aus → Commit & Push & PR erstellen
