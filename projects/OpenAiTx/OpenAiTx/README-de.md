# Open Ai Tx

**OpenAiTx** erstellt automatisch eine README-Datei mit Übersetzungen in 20 Sprachen für Ihr GitHub-Projekt. Sie müssen nur die Projekt-URL `GitHub` durch `OpenAiTx` ersetzen. Zum Beispiel: https://github.com/OpenAiTx/OpenAiTx : zu https://openaitx.com/OpenAiTx/OpenAiTx

## Unterstützung/Beitrag

**Dieses Projekt benötigt kein Geld von Ihnen, aber es benötigt die Rechenleistung Ihres Computers, PR-Beiträge und Tokens.**

Wenn Sie zum Projekt beitragen möchten, müssen Sie nur Folgendes tun:  
Projekt forken => Projekt klonen => Ein Skript in Ihrer Sprache auswählen => Ihren AI-Token eintragen => Ausführen => Commit & Push & PR erstellen

Hinweis: **Bitte laden Sie Ihre Tokens nicht hoch und geben Sie diese nicht weiter!**

## Sonstiges

- Wenn Sie hohe Priorität und kontinuierliche Updates erhalten möchten, erstellen Sie bitte einen PR und fügen Sie Ihren Projekt-GitHub-Link in `priority-list.json` hinzu.
- Sie finden die Index-Projekte in `projects-data.json`.
- Ein 50 KB großes README benötigt `2 Minuten/Sprache` mit GPT4.1.
