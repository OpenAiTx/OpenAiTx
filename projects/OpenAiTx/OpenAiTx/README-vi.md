## Hỗ trợ/Đóng góp

**Dự án này không cần tiền của bạn, nhưng cần sức mạnh tính toán của máy tính bạn, đóng góp PR, và Tokens.**

Nếu bạn muốn đóng góp cho dự án, bạn chỉ cần:
Fork dự án => Clone dự án => Chọn script theo ngôn ngữ của bạn => Điền token AI của bạn => Chạy script => Commit & push & tạo PR

Lưu ý: **Vui lòng không tải lên hoặc để lộ token của bạn!**

## Khác

- Nếu bạn muốn được ưu tiên cao và cập nhật liên tục, hãy tạo PR và thêm link dự án github của bạn vào `priority-list.json`.
- Bạn có thể lấy danh sách dự án trong `projects-data.json`.
- File README 50 KB cần `2 phút/mỗi ngôn ngữ` với GPT4.1.
