## Giới thiệu

**OpenAiTx** là công cụ tự động dịch README dự án GitHub sang 20 ngôn ngữ.

## Tại sao?

1. Dịch một lần bằng LLM cao cấp và công khai kết quả miễn phí cho cộng đồng mã nguồn mở.
2. Hỗ trợ SEO tìm kiếm Google/Bing đa ngôn ngữ, công cụ dịch phía client không làm được điều này.
3. Miễn phí & Mã nguồn mở vĩnh viễn
4. Gửi một lần và tự động cập nhật về sau, tiết kiệm thời gian cho bạn nếu bạn là maintainer dự án.


## Bắt đầu

1. Thay thế URL `GitHub` bằng `OpenAiTx`, ví dụ https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Nhấn và sao chép style badge hoặc văn bản bạn thích, hệ thống sẽ tự động tạo PR cho dự án sau khi đã được index.

```



## Đóng góp tính toán 

Nếu bạn muốn đóng góp cho dự án, tất cả những gì bạn cần làm là:
Fork dự án → Clone dự án → Chọn script ngôn ngữ của bạn → Điền token AI của bạn  → Chạy script → Commit & push & tạo PR

> Lưu ý: **Vui lòng không tải lên token của bạn!**

### Ghi chú

- Chỉ hỗ trợ readme markdown trên github.

## Khác

- 10~30 phút/ mỗi dự án
- 200~400 dự án/mỗi console mỗi ngày.
- Cứ mỗi 3~7 ngày sẽ cập nhật các dự án đã tồn tại (tùy thuộc vào tài nguyên tính toán trong tương lai)
```
