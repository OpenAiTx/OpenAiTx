## Giới thiệu

**OpenAiTx** là công cụ tự động dịch 20 ngôn ngữ cho readme & wiki dự án GitHub của bạn.

## Tại sao?

1. Dịch một lần duy nhất bằng LLM cao cấp và cung cấp kết quả hoàn toàn miễn phí cho cộng đồng mã nguồn mở.
2. Hỗ trợ SEO tìm kiếm đa ngôn ngữ Google/Bing, các công cụ dịch trên trình duyệt không làm được.
3. Miễn phí & Mã nguồn mở mãi mãi
4. Gửi một lần và tự động cập nhật về sau, giúp bạn tiết kiệm thời gian nếu bạn là người duy trì dự án.


## Bắt đầu

1. Thay đường dẫn URL `GitHub` bằng `OpenAiTx`, ví dụ: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Nhấp và sao chép badge hoặc đoạn văn bản kiểu bạn thích.

hoặc

1. Truy cập https://openaitx.github.io/ 
2. Gửi liên kết dự án của bạn.

## Đóng Góp Tính Toán

Nếu bạn muốn đóng góp vào dự án, tất cả những gì bạn cần làm là:
Fork project → Clone project → Chọn một script bằng ngôn ngữ của bạn → Điền token AI của bạn  → Chạy nó → Commit & push & tạo PR

> Lưu ý: **Vui lòng không tải lên token của bạn!**

## Ghi chú

- Chỉ hỗ trợ markdown trên github.
- Nếu bạn là chủ sở hữu dự án hoặc nhà phát triển cốt lõi, bạn có thể xác thực tài khoản github để tự động tạo PR sau khi đã được lập chỉ mục.

## Khác

- 10~30 phút/ mỗi dự án
- 200~400 dự án/mỗi console mỗi ngày.
- Cứ mỗi 3~7 ngày sẽ cập nhật lại các dự án đã tồn tại (tùy theo tài nguyên tính toán trong tương lai)
