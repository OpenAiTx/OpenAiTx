## Hỗ trợ/Đóng góp 

Nếu bạn muốn đóng góp cho dự án, bạn chỉ cần:
Fork dự án → Clone dự án → Chọn script ngôn ngữ của bạn → Điền token AI của bạn  → Chạy script → Commit & push & tạo PR

> Lưu ý: **Vui lòng không tải lên token của bạn!**

### Lưu ý

- Chỉ hỗ trợ file readme markdown trên github.

## Khác

- 10~30 phút/ một dự án
- 200~400 dự án/một console mỗi ngày.
- Mỗi 3~7 ngày sẽ cập nhật các dự án đã tồn tại (tùy theo tài nguyên tính toán trong tương lai)
