## معرفی

**OpenAiTx** ابزاری برای ترجمه خودکار ۲۰ زبان برای فایل readme پروژه GitHub شما.

## چرا؟

1. ترجمه یک‌باره با استفاده از LLM سطح پریمیوم و فراهم کردن دسترسی رایگان نتیجه برای جامعه متن‌باز.
2. پشتیبانی از جستجوی چندزبانه Google/Bing SEO، که ابزارهای ترجمه سمت کاربر قادر به انجام آن نیستند.
3. کاملاً رایگان و متن‌باز برای همیشه
4. یک‌بار ارسال و بروزرسانی خودکار در آینده، اگر نگهدارنده پروژه هستید، زمان شما را ذخیره می‌کند.


## شروع کنید

1. نشانی `GitHub` را با `OpenAiTx` جایگزین کنید، مثال: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. سبک نشان (badge) یا متنی که می‌پسندید را کلیک و کپی کنید، سیستم پس از ایندکس به طور خودکار برای پروژه PR ایجاد خواهد کرد.

## مشارکت در اشتراک‌گذاری

اگر بتوانید با ارسال یک PR به پروژه مورد علاقه خود کمک کنید، این کار به ترویج پروژه ما کمک بزرگی خواهد کرد.

نمونه:

```
عنوان:
docs: افزودن ۲۰ لینک زبان مختلف به README

محتوا:
این PR لینک ۲۰ زبان مختلف را به README اضافه می‌کند و کاربر می‌تواند به راحتی به READEME ترجمه‌شده دسترسی داشته باشد، از جستجوی چندزبانه Google/Bing برای SEO پشتیبانی می‌کند.

نمونه صفحه https://openaitx.github.io/view.html?user=CherryHQ&project=cherry-studio&lang=ja

> OpenAiTx https://github.com/OpenAiTx/OpenAiTx 
> چرا؟
> ۱. یک‌بار ترجمه با استفاده از LLM درجه ممتاز و ارائه نتیجه به‌صورت رایگان به جامعه اپن‌سورس.
> ۲. پشتیبانی از جستجوی چندزبانه Google/Bing برای SEO، ابزار ترجمه کلاینت این قابلیت را ندارد.
> ۳. همیشه رایگان و اپن‌سورس 
> ۴. با یک بار ارسال، به صورت خودکار در آینده به‌روزرسانی می‌شود، اگر نگهدارنده پروژه هستید در وقت شما صرفه‌جویی می‌شود.
```

## مشارکت محاسباتی

اگر مایل به مشارکت در پروژه هستید، کافیست:
فورک پروژه → کلون پروژه → انتخاب اسکریپت به زبان خود → وارد کردن توکن AI خود → اجرا → کامیت و پوش و ساخت یک PR
