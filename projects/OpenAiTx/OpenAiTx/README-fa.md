## معرفی

**OpenAiTx** ابزاری برای ترجمه خودکار ۲۰ زبان برای فایل readme پروژه GitHub شما.

## چرا؟

1. ترجمه یک‌باره با استفاده از LLM سطح پریمیوم و فراهم کردن دسترسی رایگان نتیجه برای جامعه متن‌باز.
2. پشتیبانی از جستجوی چندزبانه Google/Bing SEO، که ابزارهای ترجمه سمت کاربر قادر به انجام آن نیستند.
3. کاملاً رایگان و متن‌باز برای همیشه
4. یک‌بار ارسال و بروزرسانی خودکار در آینده، اگر نگهدارنده پروژه هستید، زمان شما را ذخیره می‌کند.


## شروع کنید

1. نشانی `GitHub` را با `OpenAiTx` جایگزین کنید، مثال: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. سبک نشان (badge) یا متنی که می‌پسندید را کلیک و کپی کنید، سیستم پس از ایندکس به طور خودکار برای پروژه PR ایجاد خواهد کرد.

## اشتراک‌گذاری مشارکت

اگر بتوانید با ارسال یک PR به پروژه مورد علاقه‌تان کمک کنید، این کار به ترویج پروژه ما بسیار کمک خواهد کرد.

مثال:

```
عنوان:
docs: افزودن ۲۰ لینک زبان به README

محتوا:
این PR، لینک ۲۰ زبان را به README اضافه می‌کند و کاربران به راحتی می‌توانند به README ترجمه‌شده دسترسی پیدا کنند. از سئو چندزبانه گوگل/بینگ پشتیبانی می‌کند.

دمو صفحه https://openaitx.github.io/view.html?user=CherryHQ&project=cherry-studio&lang=ja

> OpenAiTx https://github.com/OpenAiTx/OpenAiTx 
> چرا؟
> ۱. ترجمه یک‌باره با استفاده از LLM با کیفیت بالا و ارائه نتیجه به طور رایگان به جامعه متن‌باز.
> ۲. پشتیبانی از جستجوی سئو چندزبانه گوگل/بینگ، ابزارهای ترجمه سمت کلاینت این کار را انجام نمی‌دهند.
> ۳. همیشه رایگان و متن‌باز 
> ۴. تنها یک بار ارسال و به‌روزرسانی خودکار در آینده، اگر نگهدارنده پروژه هستید، در زمان شما صرفه‌جویی می‌شود.
```



## مشارکت محاسباتی

اگر مایل به مشارکت در پروژه هستید، فقط کافیست:
فورک پروژه → کلون پروژه → انتخاب یک اسکریپت به زبان خود → وارد کردن توکن AI خود → اجرای آن → کامیت و پوش و ایجاد PR

> توجه: **لطفاً توکن‌های خود را بارگذاری نکنید!**
