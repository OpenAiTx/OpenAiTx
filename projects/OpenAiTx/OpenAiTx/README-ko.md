## 소개

**OpenAiTx**는 GitHub 프로젝트의 readme 및 wiki를 위한 20개 언어 자동 번역 도구입니다.

## 왜 사용해야 할까요?

1. 프리미엄 등급 LLM을 사용한 한 번의 번역으로 오픈소스 커뮤니티에 결과를 무료로 제공합니다.
2. Google/Bing 다국어 SEO 검색을 지원하며, 클라이언트 번역 도구로는 불가능합니다.
3. 영원히 무료 & 오픈소스
4. 한 번 제출하면 이후 자동 업데이트, 프로젝트 관리자의 시간을 절약할 수 있습니다.


## 시작하기

1. URL의 `GitHub`를 `OpenAiTx`로 교체하세요. 예: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 원하는 스타일의 배지 또는 텍스트를 클릭 후 복사하세요.

또는

1. https://openaitx.github.io/ 에 접속하세요.
2. 프로젝트 링크를 제출하세요.

## 컴퓨팅 기여

프로젝트에 기여하고 싶으시다면, 아래의 절차만 따라주시면 됩니다:
프로젝트 포크 → 프로젝트 클론 → 원하는 언어의 스크립트 선택 → AI 토큰 입력 → 실행 → 커밋 & 푸시 & PR 생성

> 참고: **토큰을 업로드하지 마세요!**

## 참고 사항

- Github 마크다운만 지원합니다.
- 프로젝트 소유자이거나 핵심 개발자라면, Github 계정 인증을 통해 인덱싱 후 자동으로 PR을 생성할 수 있습니다.

## 기타

- 프로젝트당 10~30분 소요
- 콘솔 1개당 1일 최대 200~400개 프로젝트
- 3~7일마다 기존 프로젝트 업데이트 (향후 컴퓨팅 리소스에 따라 변경 가능)
