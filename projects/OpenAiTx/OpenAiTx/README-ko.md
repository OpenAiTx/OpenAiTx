## 소개

**OpenAiTx**는 GitHub 프로젝트 README를 위한 20개 언어 자동 번역 도구입니다.

## 목표

프리미엄 등급의 LLM을 이용한 일회성 번역과, 그 결과를 오픈소스 커뮤니티에 무료로 공개합니다.

## 특징

- Google/Bing 다국어 SEO 검색 지원
- 영원히 무료 & 오픈소스 
- 한 번 제출하면, 이후 자동 업데이트

```

## 컴퓨팅 기여

프로젝트에 기여하고 싶으시다면, 아래 단계만 따라 주시면 됩니다:
프로젝트 포크 → 프로젝트 클론 → 원하는 언어의 스크립트 선택 → AI 토큰 입력 → 실행 → 커밋 & 푸시 & PR 생성

> 참고: **토큰은 절대 업로드하지 마세요!**
```
