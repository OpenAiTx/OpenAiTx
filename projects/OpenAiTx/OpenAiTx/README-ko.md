## 소개

**OpenAiTx** GitHub 프로젝트 readme를 위한 20개 언어 자동 번역 도구.

## 왜 사용해야 하나요?

1. 프리미엄급 LLM을 사용한 1회 번역 결과를 오픈소스 커뮤니티에 자유롭게 공개합니다.
2. Google/Bing 다국어 SEO 검색 지원, 클라이언트 번역 도구는 불가능합니다.
3. 영구적으로 무료 & 오픈소스 
4. 한 번 제출하면 자동으로 미래 업데이트, 프로젝트 유지자라면 시간을 절약할 수 있습니다.


## 시작하기

1. URL의 `GitHub`를 `OpenAiTx`로 변경하세요. 예: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 원하는 스타일 배지나 텍스트를 클릭 및 복사하세요. 시스템이 색인 후 자동으로 프로젝트에 PR을 생성합니다.
