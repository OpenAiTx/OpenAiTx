## 지원/기여 

프로젝트에 기여하고 싶으시다면, 다음과 같이 하시면 됩니다:
프로젝트 포크 → 프로젝트 클론 → 원하는 언어의 스크립트 선택 → AI 토큰 입력 → 실행 → 커밋 & 푸시 & PR 생성

> 참고: **절대 토큰을 업로드하지 마세요!**

### 참고 사항

- github 마크다운 readme만 지원합니다.

## 기타

- 프로젝트당 10~30분 소요
- 콘솔 1개당 하루 200~400개 프로젝트 처리
- 3~7일마다 기존 프로젝트 업데이트(향후 컴퓨팅 리소스에 따라 다름)
