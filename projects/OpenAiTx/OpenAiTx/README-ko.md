# Open Ai Tx

**OpenAiTx**는 GitHub 프로젝트의 README 파일을 20개 언어로 자동 번역해 생성해 줍니다. 프로젝트 URL의 `GitHub` 부분을 `OpenAiTx`로 바꿔주기만 하면 됩니다. 예: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx

## 지원/기여

**이 프로젝트는 여러분의 금전적 지원이 아니라, 컴퓨터의 연산 자원, PR(풀 리퀘스트) 기여, 그리고 토큰이 필요합니다.**

프로젝트에 기여하고 싶다면, 아래의 절차만 따라주세요:  
포크(Fork) → 클론(Clone) → 원하는 언어의 스크립트 선택 → 본인 AI 토큰 입력 → 실행 → 커밋 & 푸시 & PR 생성

참고: **토큰을 업로드하거나 유출하지 마세요!**



## 기타

- 높은 우선순위와 지속적인 업데이트를 원한다면, PR을 생성하고 `priority-list.json`에 본인 프로젝트 GitHub 링크를 추가해 주세요.
- `projects-data.json`에서 인덱스 프로젝트를 확인할 수 있습니다.
- 50KB README의 경우 GPT4.1 기준으로 `언어당 2분`이 소요됩니다.
