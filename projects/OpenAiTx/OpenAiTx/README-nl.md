## Introductie

**OpenAiTx** maakt automatisch README-bestanden in 20 talen aan voor jouw GitHub-project. Je hoeft alleen maar de project-URL `GitHub` te vervangen door `OpenAiTx`, bijvoorbeeld https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

## Ondersteuning/Bijdrage

Als je wilt bijdragen aan het project, hoef je alleen het volgende te doen:
Fork het project => Clone het project => Kies een script in jouw taal => Vul je AI-token in => Voer het uit => Commit & push & maak een PR aan

Opmerking: **Upload of lek je tokens niet!**

### Opmerkingen

- Ondersteunt momenteel alleen markdown readme.
- Aanbevolen wordt om voor afbeeldingslinks een volledig URL-pad te gebruiken.

## Overig

- 50 KB readme kost `2 minuten/per taal` met GPT4.1. 
- 200-400 projecten/één console per dag.
- Elke 3-N dagen worden bestaande projecten bijgewerkt (afhankelijk van toekomstige rekenkracht)
