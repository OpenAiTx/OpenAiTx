## Introductie

**OpenAiTx** 20-talen automatische vertaaltool voor de readme van jouw GitHub-project.

## Waarom?

1. Eenmalige vertaling met premium LLM en maak het resultaat vrij toegankelijk voor de open source community.
2. Ondersteuning voor Google/Bing meertalige SEO-zoekopdrachten, wat met een client vertaaltool niet mogelijk is.
3. Gratis & Open Source voor altijd 
4. Eenmalig indienen en automatische toekomstige updates, het bespaart tijd als je projectbeheerder bent.


## Aan de slag

1. Vervang de URL `GitHub` door `OpenAiTx`, bijv. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  

2. Klik en kopieer de stijlbadges of tekst die je wilt.

## Bijdrage aan computing

Als je wilt bijdragen aan het project, hoef je alleen het volgende te doen:
Fork het project → Clone het project → Kies een script in jouw taal → Vul je AI-token in → Voer het uit → Commit & push & maak een PR aan

> Opmerking: **Upload je tokens niet!**

## Notities

- Ondersteunt alleen github markdown readme.
- Als je project eigenaar of kernontwikkelaar bent, kun je je github-account autoriseren om automatisch een PR aan te maken nadat het is geïndexeerd.

## Overig

- 10~30 minuten/per project
- 200~400 projecten/één console per dag.
- Elke 3~7 dagen worden bestaande projecten bijgewerkt (afhankelijk van toekomstige computerbronnen)
