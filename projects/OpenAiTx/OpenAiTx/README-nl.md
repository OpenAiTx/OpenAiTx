## Introductie

**OpenAiTx** 20-talen automatische vertaalsysteem voor de readme & wiki van je GitHub-project.

## Waarom?

1. Eenmalige vertaling met premium LLM en het resultaat gratis beschikbaar maken voor de open-source community.
2. Ondersteunt Google/Bing meertalige SEO-zoekopdrachten, client-vertaaltools kunnen dit niet.
3. Altijd gratis & open source 
4. Eén keer indienen en automatische toekomstige updates, dat bespaart tijd voor projectbeheerders.


## Aan de slag

1. Vervang in de URL `GitHub` door `OpenAiTx`, bijvoorbeeld https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Dien je project in.
3. Klik en kopieer de stijlbadges of tekst die je wilt.
4. Werk je readme-bestand bij.

of

1. Ga naar https://openaitx.com
2. Dien je projectlink in.
3. Klik en kopieer de stijlbadges of tekst die je wilt.
4. Werk je readme-bestand bij.
