# Open Ai Tx

**OpenAiTx** cria automaticamente um README traduzido para 20 idiomas para o seu projeto no GitHub. Você só precisa substituir a URL do projeto `GitHub` por `OpenAiTx`. Exemplo: https://github.com/OpenAiTx/OpenAiTx : para https://openaitx.com/OpenAiTx/OpenAiTx

## Suporte/Contribuição

**Este projeto não precisa do seu dinheiro, mas precisa do poder de processamento do seu computador, contribuições via PR e Tokens.**

Se você quiser contribuir para o projeto, tudo o que precisa fazer é:
Fork no projeto => Clone o projeto => Escolha um script no seu idioma => Preencha o seu token de IA => Execute => Commit & push & crie um PR

Nota: **Por favor, não faça upload nem compartilhe seus tokens!**

## Outros

- Se você deseja obter alta prioridade e atualizações contínuas, crie um PR e adicione o link do seu projeto github em `priority-list.json`.
- Você pode obter os projetos indexados em `projects-data.json`.
- Um README de 50 KB precisa de `2 min/por idioma` usando o GPT4.1.
