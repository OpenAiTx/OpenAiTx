## Supporto/Contributo

**Questo progetto non ha bisogno del tuo denaro, ma ha bisogno della potenza di calcolo del tuo computer, dei contributi PR e dei Token.**

Se vuoi contribuire al progetto, tutto ciò che devi fare è:
Fork del progetto => Clona il progetto => Scegli uno script nella tua lingua => Inserisci il tuo token AI => Esegui => Commit & push & crea una PR

Nota: **Non caricare o divulgare i tuoi token!**

## Altro

- Se desideri ottenere aggiornamenti ad alta priorità e continui, crea una PR e aggiungi il link del tuo progetto github in `priority-list.json`.
- Puoi ottenere l'indice dei progetti in `projects-data.json`.
- Un readme da 50 KB richiede `2 minuti/lingua` con GPT4.1.
