## सहायता/योगदान

**इस प्रोजेक्ट को आपके पैसे की आवश्यकता नहीं है, लेकिन इसे आपके कंप्यूटर की कम्प्यूटिंग पावर, PR योगदान, और टोकन की आवश्यकता है।**

यदि आप इस प्रोजेक्ट में योगदान देना चाहते हैं, तो आपको बस इतना करना है:
Fork प्रोजेक्ट => Clone प्रोजेक्ट => अपनी भाषा में स्क्रिप्ट चुनें => अपना AI टोकन भरें => रन करें => Commit & push & PR बनाएं

नोट: **कृपया अपने टोकन अपलोड या लीक न करें!**



## अन्य

- यदि आप उच्च प्राथमिकता और लगातार अपडेट चाहते हैं, तो कृपया एक PR बनाएं और अपने प्रोजेक्ट का GitHub लिंक `priority-list.json` में जोड़ें।
- आप `projects-data.json` में इंडेक्स प्रोजेक्ट्स प्राप्त कर सकते हैं।
