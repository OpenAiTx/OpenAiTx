## Introducción

**OpenAiTx** crea automáticamente archivos README traducidos a 20 idiomas para tu proyecto de GitHub; solo necesitas reemplazar la URL del proyecto `GitHub` por `OpenAiTx`, por ejemplo: https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

## Soporte/Contribución

Si deseas contribuir al proyecto, solo tienes que:
Bifurcar el proyecto => Clonar el proyecto => Elegir un script en tu idioma => Rellenar tu token de IA => Ejecutarlo => Hacer commit, push y crear un PR

Nota: **¡Por favor, no subas ni compartas tus tokens!**

## Otros

- 200-400 proyectos/una consola por día.
- Cada 3-N días se actualizarán los proyectos existentes (dependiendo de los recursos de cómputo futuros)
