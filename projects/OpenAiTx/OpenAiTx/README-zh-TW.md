## 介紹

**OpenAiTx** 是一款支援 20 種語言自動翻譯的 GitHub 專案 Readme 工具。



### 特色

- 支援 Google/Bing 多語言 SEO 搜尋。
- 永久免費且開源 
- 一次提交，自動未來更新。
