# Open Ai Tx

**OpenAiTx** 會自動為你的 GitHub 專案建立 20 種語言的 README 翻譯，你只需要將專案的 URL 從 `GitHub` 替換為 `OpenAiTx` 即可。例如：https://github.com/OpenAiTx/OpenAiTx 變為 https://openaitx.com/OpenAiTx/OpenAiTx

## 支持／貢獻

**本專案不需要你的資金，但需要你的電腦運算能力、PR 貢獻以及 Tokens。**

如果你想為本專案作出貢獻，只需要：
Fork 專案 => Clone 專案 => 選擇你的語言腳本 => 填入你的 AI token  => 執行腳本 => Commit & push & 建立 PR

注意：**請不要上傳或洩漏你的 token！**



## 其他

- 如果你希望獲得高優先權和持續更新，請建立 PR 並將你的專案 github 連結加入到 `priority-list.json` 中。
- 你可以在 `projects-data.json` 中獲取索引專案列表。
- 50 KB 的 readme，GPT4.1 翻譯每種語言約需 `2 分鐘`。
