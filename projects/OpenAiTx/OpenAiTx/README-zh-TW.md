## 介紹

**OpenAiTx** 為您的 GitHub 專案 README 及 Wiki 提供 20 種語言自動翻譯工具。

## 為什麼選擇？

1. 一次性翻譯，採用高級 LLM，讓開源社群能免費存取翻譯成果。
2. 支援 Google/Bing 多語言 SEO 搜尋，客戶端翻譯工具無法做到。
3. 永久免費與開源
4. 只需提交一次並自動更新，專案維護者可大幅節省時間。


## 開始使用

1. 將網址中的 `GitHub` 替換為 `OpenAiTx`，如 https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 提交您的專案。
3. 點擊並複製您喜歡的樣式徽章或文字。
4. 更新到您的 readme 檔案。

或

1. 訪問 https://openaitx.com
2. 提交您的專案連結。
3. 點擊並複製您喜歡的樣式徽章或文字。
4. 更新到您的 readme 檔案。
