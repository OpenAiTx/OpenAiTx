## 介紹

**OpenAiTx** 支援 20 種語言的自動翻譯工具，適用於您的 GitHub 專案 readme。



### 功能

- 支援 Google/Bing 多語言搜尋。
- 永久免費且開源
- 僅需提交一次，未來自動更新。
