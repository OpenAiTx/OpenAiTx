## 支援/貢獻

如果您想要為本專案貢獻，只需進行以下步驟：
Fork 專案 => Clone 專案 => 選擇您的語言腳本 => 填入您的 AI Token => 執行腳本 => Commit & Push & 建立 PR

注意：**請勿上傳或洩漏您的 Token！**
