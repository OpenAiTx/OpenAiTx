## 簡介

**OpenAiTx** 是一個支援 20 種語言的 GitHub 專案 readme 自動翻譯工具。

## 為什麼選擇？

1. 一次性翻譯，使用高品質 LLM，讓結果自由開放給開源社群。
2. 支援 Google/Bing 多語言 SEO 搜尋，客戶端翻譯工具無法做到。
3. 永久免費且開源。
4. 只需提交一次，未來自動更新，如果你是專案維護者，能大幅節省時間。

## 快速開始

1. 將網址中的 `GitHub` 替換為 `OpenAiTx`，例如 https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. 點擊並複製你喜歡的樣式徽章或文字，系統在索引後會自動為專案創建 PR。

## 運算貢獻 

如果你想在專案中做出貢獻，只需：
Fork 本專案 → Clone 專案 → 選擇你語言的腳本 → 填入你的 AI Token → 執行 → Commit & Push 並建立 PR

> 注意：**請勿上傳你的 Token！**
