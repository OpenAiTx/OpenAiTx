## 介紹

**OpenAiTx** 自動為您的 GitHub 專案建立 20 種語言的翻譯說明文件，您只需將專案 URL 中的 `GitHub` 替換為 `OpenAiTx`，例如：https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx 。

## 支援/貢獻

如果您想對此專案做出貢獻，只需：
Fork 專案 => Clone 專案 => 選擇您的語言腳本 => 填寫您的 AI 令牌 => 執行腳本 => Commit & push 並建立 PR

注意：**請勿上傳或洩漏您的令牌！**

### 注意事項

- 目前僅支援 markdown 格式的說明文件。
- 建議圖片連結使用完整 URL 路徑。

## 其他

- 50 KB 的說明文件，GPT4.1 需要約 `2 分鐘/每種語言`。
- 每天可處理 200-400 個專案/一個主控台。
- 每 3-N 天會更新現有專案（視未來運算資源而定😎）。
