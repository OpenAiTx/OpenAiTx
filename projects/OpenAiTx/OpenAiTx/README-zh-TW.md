## 支援/貢獻

如果您想參與本專案，只需要：
Fork 專案 => Clone 專案 => 選擇您語言的腳本 => 填入您的 AI token => 執行腳本 => Commit & push 並建立 PR

注意：**請勿上傳或洩漏您的 token！**
