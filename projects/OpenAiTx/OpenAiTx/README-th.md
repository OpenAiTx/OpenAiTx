## การสนับสนุน/การมีส่วนร่วม

**โปรเจกต์นี้ไม่ต้องการเงินของคุณ แต่ต้องการพลังประมวลผลของคอมพิวเตอร์คุณ, การมีส่วนร่วม PR และ Tokens ของคุณ**

ถ้าคุณต้องการมีส่วนร่วมกับโปรเจกต์นี้ สิ่งที่คุณต้องทำคือ:
Fork โปรเจกต์ => Clone โปรเจกต์ => เลือกสคริปต์ภาษาของคุณ => ใส่ AI token ของคุณ => รันสคริปต์ => Commit & push & สร้าง PR

หมายเหตุ: **โปรดอย่าอัปโหลดหรือเปิดเผย token ของคุณ!**
