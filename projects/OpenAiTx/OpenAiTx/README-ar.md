## المقدمة

أداة الترجمة التلقائية **OpenAiTx** لـ 20 لغة لملف readme وwiki الخاص بمشروعك على GitHub.

## لماذا؟

1. ترجمة لمرة واحدة باستخدام نموذج لغوي كبير عالي الجودة وجعل النتيجة متاحة مجانًا لمجتمع المصدر المفتوح.
2. دعم البحث متعدد اللغات في Google/Bing SEO، أداة الترجمة الخاصة بالعميل لا يمكنها فعل ذلك.
3. مجانية ومفتوحة المصدر إلى الأبد
4. إرسال مرة واحدة وتحديث تلقائي مستقبلاً، يمكن أن يوفر وقتك إذا كنت مسؤولاً عن مشروع.

## ابدأ الآن

1. استبدل رابط `GitHub` بـ `OpenAiTx`، مثلاً https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. انقر وانسخ شارات أو نصوص النمط التي تفضلها.

أو

1. ادخل إلى https://openaitx.github.io/
2. قدم رابط مشروعك.

## المساهمة الحاسوبية

إذا كنت ترغب في المساهمة في المشروع، كل ما عليك فعله هو:
تفرع المشروع → استنساخ المشروع → اختيار سكربت بلغتك → إدخال رمز الذكاء الاصطناعي الخاص بك → تشغيله → الالتزام والدفع وإنشاء طلب دمج (PR)

> ملاحظة: **يرجى عدم رفع رموزك!**

## ملاحظات

- يدعم فقط تنسيق ماركداون الخاص بـ Github.
- إذا كنت مالك المشروع أو مطورًا أساسيًا، يمكنك مصادقة حساب Github الخاص بك لإنشاء طلب دمج تلقائي بعد الفهرسة.

## أخرى
- 10~30 دقيقة/ لكل مشروع
- 200~400 مشروع/ وحدة تحكم واحدة في اليوم.
- سيتم تحديث المشاريع الحالية كل 3~7 أيام (حسب موارد الحوسبة المستقبلية)
