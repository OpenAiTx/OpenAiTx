## الدعم/المساهمة

**هذا المشروع لا يحتاج أموالك، ولكنه يحتاج إلى قدرة الحوسبة في جهازك، والمساهمات عبر PR، والرموز (Tokens).**

إذا كنت ترغب في المساهمة في المشروع، كل ما عليك فعله هو:
قم بعمل Fork للمشروع => استنسخ المشروع (Clone) => اختر سكريبت بلغتك => ضع رمز الذكاء الاصطناعي الخاص بك => شغل السكريبت => قم بعمل Commit و Push وأنشئ PR

ملاحظة: **يرجى عدم رفع أو تسريب رموزك الخاصة!**



## أخرى

- إذا كنت ترغب في الحصول على أولوية عالية وتحديثات مستمرة، يرجى إنشاء PR وإضافة رابط مشروعك على GitHub في `priority-list.json`.
- يمكنك الحصول على فهرس المشاريع في `projects-data.json`.
- ملف README بحجم 50 كيلوبايت يحتاج إلى `دقيقتين/لكل لغة` باستخدام GPT4.1.
