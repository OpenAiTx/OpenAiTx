## المقدمة

**OpenAiTx** أداة ترجمة تلقائية لـ 20 لغة لملف README و ويكي مشروع GitHub الخاص بك.

## لماذا؟

1. ترجمة لمرة واحدة باستخدام نموذج لغوي متقدم ونتائج مجانية ومفتوحة لمجتمع المصادر المفتوحة.
2. دعم بحث SEO متعدد اللغات على Google/Bing، لا يمكن لأداة الترجمة على العميل القيام بذلك.
3. مجاني ومفتوح المصدر للأبد 
4. أرسل مرة واحدة وسيتم التحديث تلقائيًا مستقبلاً، سيوفر وقتك إذا كنت مسؤول مشروع.


## بدء الاستخدام

1. استبدل رابط `GitHub` بـ `OpenAiTx`، مثل https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. أرسل مشروعك.
3. اضغط وانسخ شارة النمط أو النص الذي تفضله.
4. حدّث ملف README الخاص بك.
