## المقدمة

**OpenAiTx** أداة ترجمة تلقائية لـ 20 لغة لملف README الخاص بمشروعك على GitHub.

## لماذا؟

1. ترجمة لمرة واحدة باستخدام نموذج لغوي كبير عالي الجودة وجعل النتيجة متاحة بحرية لمجتمع المصادر المفتوحة.
2. دعم البحث في Google/Bing SEO متعدد اللغات، أداة الترجمة العميلة لا يمكنها ذلك.
3. مجاني ومفتوح المصدر للأبد
4. قدم مرة واحدة وسيتم التحديث تلقائيًا مستقبلاً، سيوفر وقتك إذا كنت مشرف مشروع.


## ابدأ الآن

1. استبدل عنوان URL `GitHub` بـ `OpenAiTx`، مثال https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. انقر ونسخ شارة النمط أو النص الذي تفضله، سيقوم النظام تلقائيًا بإنشاء PR للمشروع بعد الفهرسة.

## مشاركة المساهمة

إذا كان بإمكانك المساعدة عن طريق إرسال PR إلى مشروعك المفضل، فسيساعد ذلك بشكل كبير في الترويج لمشروعنا.

مثال:

```
العنوان:
docs: إضافة روابط 20 لغة لملف README

المحتوى:
يضيف PR روابط 20 لغة إلى README ويمكن للمستخدم بسهولة الوصول إلى README المترجم، ويدعم بحث SEO متعدد اللغات من جوجل/بينج.

صفحة العرض التوضيحي https://openaitx.github.io/view.html?user=CherryHQ&project=cherry-studio&lang=ja

> OpenAiTx https://github.com/OpenAiTx/OpenAiTx 
> لماذا؟
> 1. ترجمة لمرة واحدة باستخدام LLM عالية الجودة وجعل النتيجة متاحة مجاناً لمجتمع المصادر المفتوحة.
> 2. دعم بحث SEO متعدد اللغات لجوجل/بينج، أداة الترجمة من العميل لا يمكنها ذلك.
> 3. مجاني ومفتوح المصدر إلى الأبد
> 4. إرسال لمرة واحدة وتحديث تلقائي مستقبلاً، يمكن أن يوفر وقتك إذا كنت من صيانة المشروع.
```



## مساهمة بالحوسبة 

إذا كنت ترغب في المساهمة في المشروع، كل ما عليك فعله هو:
تفرع المشروع → استنساخ المشروع → اختيار سكربت بلغتك → ملء رمز AI الخاص بك → تشغيله → الالتزام & الدفع & إنشاء PR
