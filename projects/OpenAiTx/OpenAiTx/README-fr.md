## Introduction

**OpenAiTx** crée automatiquement des readmes traduits en 20 langues pour votre projet GitHub, vous n’avez qu’à remplacer l’URL du projet `GitHub` par `OpenAiTx`, par exemple https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

## Support/Contribution 

Si vous souhaitez contribuer au projet, il vous suffit de :
Forker le projet => Cloner le projet => Choisir un script dans votre langue => Remplir votre token AI => Lancer => Commit & push & créer une PR

Note : **Merci de ne pas uploader ou divulguer vos tokens !**

### Notes

- Supporte uniquement les readmes en markdown pour l’instant.
- Il est recommandé que les liens d’images utilisent un chemin URL complet.

## Autres

- Un readme de 50 KB nécessite `2 minutes/par langue` avec GPT4.1.
- 200-400 projets / une console par jour.
- Tous les 3-N jours, les projets existants seront mis à jour (en fonction des ressources informatiques futures😎)
