# Open Ai Tx

**OpenAiTx** crée automatiquement un fichier README traduit en 20 langues pour votre projet GitHub. Il vous suffit de remplacer l’URL du projet `GitHub` par `OpenAiTx`. Par exemple : https://github.com/OpenAiTx/OpenAiTx devient https://openaitx.com/OpenAiTx/OpenAiTx

## Support/Contributions

**Ce projet n’a pas besoin de votre argent, mais il a besoin de la puissance de calcul de votre ordinateur, de vos contributions PR, et de vos Tokens.**

Si vous souhaitez contribuer au projet, il vous suffit de :
Forker le projet => Cloner le projet => Choisir un script dans votre langue => Renseigner votre token IA => Lancer le script => Commit & push & créer une PR

Remarque : **Veuillez ne pas télécharger ni divulguer vos tokens !**

## Autres

- Si vous souhaitez obtenir une priorité élevée et des mises à jour continues, veuillez créer une PR et ajouter le lien GitHub de votre projet dans `priority-list.json`.
- Vous pouvez obtenir l’index des projets dans `projects-data.json`.
- Un README de 50 Ko nécessite `2 minutes/par langue` avec GPT4.1.
