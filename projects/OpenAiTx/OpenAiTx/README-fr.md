# Open Ai Tx

![](https://openaitx.github.io/logo_crop.png)

**OpenAiTx** crée automatiquement des fichiers README traduits en 20 langues pour votre projet GitHub. Il vous suffit de remplacer l’URL du projet `GitHub` par `OpenAiTx`, par exemple : https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

## Support/Contribution

Si vous souhaitez contribuer au projet, il vous suffit de :
Forker le projet => Cloner le projet => Choisir un script dans votre langue => Renseigner votre jeton AI => L’exécuter => Commit & push & créer une PR

Remarque : **Merci de ne pas télécharger ni divulguer vos jetons !**



## Autre

- Un README de 50 Ko nécessite `2 minutes/par langue` avec GPT4.1.
- 200-400 projets/une console par jour.
