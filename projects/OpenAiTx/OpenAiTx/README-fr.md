## Introduction

**OpenAiTx** est un outil de traduction automatique en 20 langues pour le readme et le wiki de votre projet GitHub.

## Pourquoi ?

1. Traduction unique utilisant un LLM de qualité premium, et rend le résultat librement accessible à la communauté open source.
2. Prise en charge du référencement multilingue Google/Bing, ce que les outils de traduction côté client ne peuvent pas faire.
3. Gratuit et Open Source pour toujours 
4. Soumettez une fois et mettez à jour automatiquement à l'avenir, cela vous fera gagner du temps si vous êtes mainteneur de projet.


## Commencer

1. Remplacez l’URL `GitHub` par `OpenAiTx`, par ex. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx  
2. Soumettez votre projet.
3. Cliquez et copiez les badges ou le texte de style que vous aimez.
4. Mettez à jour votre fichier readme.

ou

1. Accédez à https://openaitx.com
2. Soumettez le lien de votre projet.
3. Cliquez et copiez les badges ou le texte de style que vous aimez.
4. Mettez à jour votre fichier readme.

## Contribution Informatique

Si vous souhaitez contribuer au projet, tout ce que vous avez à faire est :
Forker le projet → Cloner le projet → Choisir un script dans votre langue → Renseigner votre jeton IA → Lancer le script → Commit & push & créer une PR

> Remarque : **Veuillez ne pas télécharger vos jetons !**
