### 指令行
複製此倉庫後執行以下指令：
```shell
cd <download-directory>
npm install
npm start
```

## 🎉 功能
- 有機成長您的 TikTok 帳號
- 自動追蹤用戶並對影片按讚及留言
- 這些用戶將成為有機粉絲
