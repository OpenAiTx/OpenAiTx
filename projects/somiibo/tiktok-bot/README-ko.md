
## 🎉 기능
- TikTok을 유기적으로 성장시킵니다
- 사용자를 자동으로 팔로우하고 비디오에 좋아요 및 댓글을 자동으로 작성합니다
- 해당 사용자들이 유기적인 팔로워가 됩니다
