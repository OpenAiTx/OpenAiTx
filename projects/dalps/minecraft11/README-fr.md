# Minecraft11

Minecraft11 est une collection amusante et gratuite d’icônes de dossiers pour Windows 11 mettant en scène des mobs et blocs de Minecraft.

<p align="center">
  <img src="https://raw.githubusercontent.com/dalps/minecraft11/main/art/preview.png" width="350" alt="logo"/>
</p>

Les fichiers `.svg` et `.ico` sont fournis. Vous n’avez besoin que du contenu du dossier [ico](ico/) ([dossier icns](icns/) pour les utilisateurs MacOS) si vous souhaitez utiliser les icônes dans votre explorateur de fichiers.

Si vous souhaitez modifier les vecteurs des icônes, assurez-vous d’utiliser [Inkscape](https://inkscape.org/).

Ce projet un peu loufoque a été inspiré par les [icônes de davidvkimball](https://www.deviantart.com/davidvkimball/art/885700560) et la collection [Folder11](https://github.com/icon11-community/Folder11).

## Cool ! Comment les installer ?

Sous Windows, cela se passe un peu comme ceci :
1. Clonez le dépôt ou téléchargez-le en tant qu’archive zip (gros bouton vert vif "Code" en haut de la page > Download ZIP)  
2. Extrayez le dossier `ico` dans un endroit sûr, par exemple votre dossier personnel ou votre bureau. Retenez cet emplacement !  
3. Faites un clic droit sur un dossier que vous souhaitez personnaliser  
4. Propriétés > Personnaliser > Changer d’icône > Parcourir > Sélectionnez un fichier .ico depuis l’emplacement mentionné  
5. Terminé, vous devriez voir l’icône du dossier changée !  

## Créez vos propres icônes !
