# Minecraft11

Minecraft11 是一个有趣且免费的 Windows 11 文件夹图标合集，特色是 Minecraft 的生物和方块。

<p align="center">
  <img src="https://raw.githubusercontent.com/dalps/minecraft11/main/art/preview.png" width="350" alt="logo"/>
</p>

同时提供 `.svg` 和 `.ico` 文件。如果您只想在文件资源管理器中使用这些图标，只需使用 [ico 文件夹](ico/)（MacOS 用户请使用 [icns 文件夹](icns/)）中的内容。

如果您有兴趣编辑图标矢量文件，请务必使用 [Inkscape](https://inkscape.org/)。

这个略显搞笑的项目灵感来自于 [davidvkimball 的图标](https://www.deviantart.com/davidvkimball/art/885700560) 和 [Folder11](https://github.com/icon11-community/Folder11) 收藏。

## 很酷！我该如何安装它们？

在 Windows 上，步骤大致如下：
1. 克隆仓库或下载为zip压缩包（页面顶部大而醒目的绿色“Code”按钮 > Download ZIP）  
2. 将 `ico` 文件夹解压到安全位置，例如你的主目录或桌面。记住此位置！  
3. 右键点击你想要自定义的文件夹  
4. 属性 > 自定义 > 更改图标 > 浏览 > 从上述位置选择一个 .ico 文件  
5. 完成，你应该能看到文件夹图标已经更改！
