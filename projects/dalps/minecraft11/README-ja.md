# Minecraft11

Minecraft11は、MinecraftのモブやブロックをフィーチャーしたWindows 11用の楽しく無料のフォルダアイコンコレクションです。

<p align="center">
  <img src="https://raw.githubusercontent.com/dalps/minecraft11/main/art/preview.png" width="350" alt="logo"/>
</p>

`.svg`ファイルと`.ico`ファイルの両方が提供されています。ファイルエクスプローラーでアイコンを使用したい場合は、[icoフォルダ](ico/)（MacOSユーザーは[icnsフォルダ](icns/)）の内容だけが必要です。

アイコンのベクター編集に興味がある場合は、[Inkscape](https://inkscape.org/)の使用を推奨します。

このややふざけたプロジェクトは、[davidvkimballのアイコン](https://www.deviantart.com/davidvkimball/art/885700560)と[Folder11](https://github.com/icon11-community/Folder11)コレクションに触発されました。

## すごい！どうやってインストールするの？

Windowsでは、だいたいこんな感じです：
1. リポジトリをクローンするか、zipアーカイブとしてダウンロードします（ページ上部の大きな明るい緑色の「Code」ボタン > Download ZIP）  
2. `ico` フォルダを安全な場所に解凍します。例えばホームディレクトリやデスクトップなど。この場所を覚えておいてください！  
3. カスタマイズしたいフォルダを右クリックします  
4. プロパティ > カスタマイズ > アイコンの変更 > ブラウズ > 先ほどの場所から .ico ファイルを選択します  
5. 完了です。フォルダのアイコンが変更されているはずです！  

## 自分だけのアイコンを作ろう！  

ピクセルパーフェクトな[空のフォルダテンプレート](https://raw.githubusercontent.com/dalps/minecraft11/main/template.svg)も作成しました。アイコンのベースとして自由にお使いください。貢献も歓迎します 🙂
