## 比 `about::inspect (chrome://inspect/#devices)` 更简单

NiM 能够管理 Chrome DevTools 窗口/标签页的生命周期，让您专注于调试代码。无需再复制粘贴 DevTools URL 或频繁打开/关闭标签页和窗口。

NiM 会自动检测使用 --inspect 选项运行 node（本地或远程）时生成的 URL，并为您提供自动打开和关闭 Chrome DevTools 的选项（可选标签页或窗口）。只需切换“手动/自动”设置，然后开始调试会话。DevTools 会在点击“打开 DevTools”按钮或等待设定超时后自动打开。如果设置为自动关闭，调试会话结束后 DevTools 标签页/窗口也会自动关闭。

 * 管理和监控本地及远程调试会话
 * 手动或自动控制 DevTools 界面
 * 在新标签页或窗口打开 DevTools
 * 启动时可设置 DevTools 聚焦或非激活
 * 自定义 v8 Inspector 探测间隔时间
 * 自动保存设置
 * 调试由 [VSCode](https://imgur.com/download/PRMn9md) 启动的 node 进程
