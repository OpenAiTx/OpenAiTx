## Lebih Mudah daripada `about::inspect (chrome://inspect/#devices)`

NiM mengelola siklus hidup jendela/tab Chrome DevTools sehingga Anda dapat lebih fokus pada hal yang penting... debugging kode Anda.  Anda tidak perlu lagi copy/paste URL DevTools atau terus membuka/menutup tab/jendela.

NiM secara otomatis mendeteksi URL yang dihasilkan saat menjalankan node (secara lokal atau remote) dengan opsi --inspect. NiM memberikan opsi untuk membuka dan menutup Chrome DevTools secara otomatis di tab atau jendela. Cukup aktifkan pengaturan Manual/Otomatis lalu mulai sesi debugging.  DevTools akan terbuka baik saat mengklik tombol "Open DevTools" atau setelah periode timeout yang ditentukan.  Jika diatur untuk menutup otomatis, setelah Anda mengakhiri sesi debugging, tab/jendela DevTools akan tertutup secara otomatis.
 
 * Kelola dan monitor sesi debugging lokal dan remote
 * Kontrol antarmuka DevTools secara manual atau otomatis
 * Buka DevTools di tab atau jendela baru
 * Jadikan DevTools fokus atau tidak aktif saat mulai
 * Kustomisasi durasi antar probe Inspector v8
 * Pengaturan autosave
 * Debug proses node yang dijalankan oleh [VSCode](https://imgur.com/download/PRMn9md)
