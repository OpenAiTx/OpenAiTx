## Getting started

1. Gets the git project. (Auto run installation)

```
  npx react-vite-ts-boilerplate my-app
```
2. 개발 서버를 시작합니다. 그리고 브라우저에서 http://localhost:3000 을 엽니다.


```
  yarn dev
```
3. 제품을 만드세요.


```
  yarn prod
```
## 종속성

- Vite
- Typescript 
- eslint
