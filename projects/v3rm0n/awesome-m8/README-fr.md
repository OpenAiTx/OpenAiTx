# awesome-m8
Une liste sélectionnée de l'incroyable écosystème [Dirtywave M8](https://dirtywave.com/products/m8-tracker-model-02). Précommandez-le [ici](https://dirtywave.com/products/m8-tracker-model-02), vous ne le regretterez pas !
