# Bert-VITS2

هسته VITS2 با برت چندزبانه

برای راهنمای سریع، لطفاً به `webui_preprocess.py` مراجعه کنید.

简易教程请参见 `webui_preprocess.py`。

## 【معرفی پروژه】
