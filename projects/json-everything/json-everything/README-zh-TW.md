## 什麼是 `json-everything`？

`json-everything` 的主要理念是確保常見的 JSON 功能在 _System.Text.Json_ 領域有良好的支援。

第一步是檢查是否已有其他專案支援某個功能。

如果有的話，這個專案通常不會重複這方面的努力。再往下你會看到一些其他專案，它們在 JSON 領域做了很酷的事情。

否則，則會考慮納入此專案。

## 其他專案對 _System.Text.Json_ 的支援

如果你在這裡找不到想要的功能，請嘗試以下優秀專案：

- [JsonCons.Net](https://github.com/danielaparker/JsonCons.Net) 由 [@danielaparker](https://github.com/danielaparker) 製作

  - JSON 指標
  - JSON 修補
  - JSON 合併修補
  - JSON 路徑
  - JMES 路徑
- [Corvus.JsonSchema](https://github.com/corvus-dotnet/Corvus.JsonSchema) 由 [@mwadams](https://github.com/mwadams) 開發
  - 著重於 C# 程式碼產生的 JSON Schema 驗證
- （未來還會新增）

如果你使用 JSON 做的事情尚未被這個套件組或上述專案的函式庫涵蓋，歡迎[建立功能請求 issue](https://github.com/json-everything/json-everything/issues/new?assignees=&labels=feature&projects=&template=New_function.yml)。

如果你維護或知道其他能擴充 _System.Text.Json_ 以實現酷炫功能的專案，也請[建立一般 issue](https://github.com/json-everything/json-everything/issues/new) 告訴我，我會將它列在這裡。
