## `json-everything` nedir?

`json-everything` arkasındaki temel felsefe, yaygın JSON işlevselliğinin _System.Text.Json_ alanında iyi desteklenmesini sağlamaktır.

Bunun ilk adımı, belirli bir işlevi destekleyen başka projelerin olup olmadığını kontrol etmektir.

Eğer varsa, o çaba muhtemelen burada yinelenmeyecektir. Aşağıda JSON ile ilgili bazı harika şeyler yapan diğer projelerin bir listesini bulacaksınız.

Aksi takdirde, bu proje için değerlendirmeye açıktır.

Yukarıdaki tüm kütüphaneler ilgili spesifikasyonlarına tam destek sunar.

_JsonSchema.Net_ için hata mesajı çevirileri şu dillerde mevcuttur:

- İtalyanca `it`
- Norveççe `nb-NO`
- Lehçe `pl-PL`
- Portekizce `pt`
- Rusça `ru`
- İspanyolca `es`
- İsveççe `sv-SE`
- Türkçe `tr-TR`

***NOT** Her dil paketi kendi Nuget paketiyle sağlanmaktadır.*

Ayrıca JSON Schema Test Suite sonuçlarını [Bowtie](https://bowtie-json-schema.github.io/bowtie) üzerinde görüntüleyebilirsiniz; bu platform test paketini farklı platformlardaki birden çok uygulama üzerinde çalıştırır.

***UYARI** Benim kütüphanem, _JsonSchema.Net_, https://jsonschema.net web sitesiyle ilgili veya ilişkili değildir; sadece her ikisi de mükemmel JSON Schema araçlarıdır.*

## Diğer projelerde _System.Text.Json_ desteği

Burada aradığınızı bulamazsanız, bu harika projelerden birini deneyin:

- [JsonCons.Net](https://github.com/danielaparker/JsonCons.Net) [@danielaparker](https://github.com/danielaparker) tarafından

  - JSON Pointer
  - JSON Patch
  - JSON Merge Patch
  - JSON Path
  - JMES Path
- [Corvus.JsonSchema](https://github.com/corvus-dotnet/Corvus.JsonSchema) [@mwadams](https://github.com/mwadams) tarafından
  - C# kod üretimine odaklanmış JSON Schema doğrulaması
- (daha fazlası gelecek)

JSON ile yukarıdaki projeler veya bu takımda yer alan bir kütüphane tarafından kapsanmayan bir şeyi yapıyorsanız, lütfen [bir özellik isteği oluşturun](https://github.com/json-everything/json-everything/issues/new?assignees=&labels=feature&projects=&template=New_function.yml).

_System.Text.Json_'ı genişleten başka bir projeyi yönetiyorsanız veya biliyorsanız ve harika şeyler yapıyorsa, burada listelemek isterim; bu yüzden lütfen [bir genel sorun oluşturun](https://github.com/json-everything/json-everything/issues/new) ve bana bildirin.
