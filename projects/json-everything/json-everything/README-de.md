## Was ist `json-everything`?

Die Hauptphilosophie hinter `json-everything` besteht darin, sicherzustellen, dass häufige JSON-Funktionalitäten im Bereich _System.Text.Json_ gut unterstützt werden.

Der erste Schritt dazu ist die Überprüfung, ob es bereits andere Projekte gibt, die eine bestimmte Funktion unterstützen.

Falls ja, wird dieser Aufwand hier wahrscheinlich nicht dupliziert. Etwas weiter unten findest du eine Liste anderer Projekte, die interessante Dinge mit JSON machen.

Andernfalls wird die Aufnahme in dieses Projekt in Erwägung gezogen.

Wenn Sie hier nicht finden, wonach Sie suchen, probieren Sie bitte eines dieser exzellenten Projekte:

  - JSON-Schema-Validierung mit Fokus auf C#-Codegenerierung
- (weitere folgen)

Wenn Sie ein weiteres Projekt pflegen oder kennen, das _System.Text.Json_ erweitert, um coole Dinge zu tun, möchte ich es hier auflisten. Bitte [erstellen Sie ein allgemeines Issue](https://github.com/json-everything/json-everything/issues/new), um mich darüber zu informieren.
