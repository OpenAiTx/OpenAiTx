# Obsidian Chronology Plugin

ノートのカレンダー表示を備えたサイドバーを提供します。

![](https://raw.githubusercontent.com/Canna71/obsidian-chronology/master/media/demo.gif)

カレンダーは日次、週次、月次、またはカスタム範囲のノートをナビゲートおよび選択できます。日次および週次範囲はタイムラインで表示されますが、設定の「Display Simple List」を有効にするとノートをシンプルなリスト表示に強制できます。
タイムラインの表示では、「Group Notes in same time slot」を有効にすると同じ時間帯に作成または変更されたノートをグループ化できます。

月名をクリックすると、その月のすべてのノートのリストが表示されます。

表示する日付範囲を選択するには、1つの日付を選択してからシフトクリックで範囲のもう1つの日付を選択します。

タイムライン内のノートは、その時点で**作成**または**変更**されたことを示す色付きのマークで表示されます。

![](https://raw.githubusercontent.com/Canna71/obsidian-chronology/master/media/example.png)

ノート名をクリックするとノートが開きます。Ctrl（またはCommand）クリックすると新しいペインで開きます。

カレンダーでは単一の日または左側の週番号を選択できます。

日付タイルの背景は、その日に作成または編集されたノートの数を示しています。スケールは設定で指定された1日あたりの平均ノート数にも依存します。

# Obsidian Syncの問題
Obsidian Syncを使用している場合、同期されたファイルの作成日時が他のマシンに同期された日時になることに気づいたかもしれません。
バージョン1.1.9では、この問題を回避するために（オプションで）ファイルメタデータから実際のノート作成日時を読み取る設定が追加されました。
すべてのノートがメタデータに作成/変更日時を持つ必要はなく、欠落している場合はプラグインがファイルの作成または変更日時にフォールバックします。
