# 世界還需要另一個 CRM 嗎？

我們建立 Twenty 的三個理由：

**CRM 太昂貴，且用戶被綁架。** 公司會利用被鎖定的客戶資料來哄抬價格。不應該是這樣。

**需要全新出發，打造更好的體驗。** 我們可以從過去錯誤中學習，並融合 Notion、Airtable、Linear 等新工具的 UX 模式，設計出整合性體驗。

**我們相信開源與社群。** 已有數百位開發者一同參與 Twenty 的建設。當我們擁有插件能力後，整個生態圈將圍繞其成長。
