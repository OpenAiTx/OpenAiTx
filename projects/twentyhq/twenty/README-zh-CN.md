# 世界还需要另一个CRM吗？

我们创建Twenty有三个原因：

**CRM太昂贵，用户被困其中。** 企业通过锁定客户数据来抬高价格。事情本不该如此。

**需要全新出发，打造更好的体验。** 我们可以吸取过去的经验，借鉴Notion、Airtable或Linear等工具的新型UX模式，打造更加统一流畅的体验。

**我们相信开源与社区。** 数百名开发者已在共建Twenty。等插件能力上线后，整个生态将围绕其发展壮大。
