# 世界は新たなCRMを必要としているのか？

私たちがTwentyを作った理由は3つあります：

**CRMは高価すぎ、ユーザーは閉じ込められている。** 企業はロックインされた顧客データを利用して価格を引き上げています。本来はそうあるべきではありません。

**より良い体験には新しいスタートが必要です。** 過去の失敗から学び、Notion、Airtable、Linearなどの新しいUXパターンに着想を得て、一貫した体験を作り上げることができます。

**オープンソースとコミュニティを信じています。** すでに何百人もの開発者がTwentyの開発に参加しています。プラグイン機能が追加されれば、まったく新しいエコシステムが成長していくでしょう。
