# 소개

[Svelte](https://svelte.dev)와 [Tailwind](https://www.tailwindcss.com)를 기반으로 한 모바일 웹 컴포넌트 라이브러리입니다.

> **S**imple • **T**iny • **D**esign • **F**ast

# 특징

- 🔥 Svelte v5, Tailwind CSS v4, TypeScript 완벽 지원
- 🚀 런타임 없음, 가상 DOM 없음 — 온라인에서 더욱 빠르게 동작
- 🧰 풍부한 API — 컴포넌트 스타일을 쉽게 설정하여 원하는 대로 커스터마이즈
- 🍭 다크 모드 및 다중 테마 구성 지원
- 📖 중국어 및 영어의 풍부한 문서와 컴포넌트 예제 제공
- 🌍 국제화 지원, 60개 이상의 내장 언어 패키지 제공
- 🫰 사용자 친화적 인터랙션, 애니메이션 파라미터 설정 가능, 커스텀 테마 지원
- 🤝 UnoCSS 등 Tailwind 계열 라이브러리 지원
- 🫡 서드파티 의존성 없음 — 라이브러리 버전이나 보안 문제 걱정 불필요
- 📦 필요할 때만 가져오기(import) 지원으로 번들 크기 최소화
- 🪜 빠른 프로젝트 생성을 위한 스캐폴딩 도구 제공
- 🔗 SVG 파일을 빠르게 병합할 수 있는 SVG 병합 플러그인 기본 제공, Iconify 지원
- 🍺 컴포넌트 API를 빠르게 확인할 수 있는 IDE 플러그인 제공
