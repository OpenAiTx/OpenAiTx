# معرفی

کتابخانه کامپوننت موبایل وب بر پایه [Svelte](https://svelte.dev) و [Tailwind](https://www.tailwindcss.com) .

> **S**imple • **T**iny • **D**esign • **F**ast

# قابلیت‌ها

- 🔥 پشتیبانی کامل از Svelte نسخه ۵، Tailwind CSS نسخه ۴ و TypeScript.
- 🚀 بدون ران‌تایم، بدون DOM مجازی — اجرای سریع‌تر آنلاین.
- 🧰 API غنی — به راحتی استایل کامپوننت‌ها را مطابق نیاز خود پیکربندی کنید.
- 🍭 پشتیبانی از حالت تاریک و پیکربندی تم‌های متعدد.
- 📖 ارائه مستندات و نمونه‌های کامپوننت گسترده به زبان چینی و انگلیسی.
- 🌍 پشتیبانی از بین‌المللی‌سازی، با بیش از ۶۰ بسته زبان داخلی.
- 🫰 تعامل کاربرپسند، پارامترهای انیمیشن قابل پیکربندی و پشتیبانی از تم‌های سفارشی.
- 🤝 پشتیبانی از کتابخانه‌هایی مانند UnoCSS برای Tailwind.
- 🫡 بدون وابستگی به کتابخانه‌های شخص ثالث — بدون نگرانی درباره نسخه یا امنیت کتابخانه‌ها.
- 📦 پشتیبانی از ایمپورت به صورت نیاز، برای کاهش حجم بسته.
- 🪜 ارائه ابزار اسکافولدینگ برای ایجاد سریع پروژه.
- 🔗 ارائه افزونه ادغام SVG برای ادغام سریع فایل‌های SVG، پشتیبانی از Iconify.
- 🍺 افزونه IDE برای مشاهده سریع API کامپوننت‌ها.
