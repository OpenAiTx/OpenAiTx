# 介绍

基于 [Svelte](https://svelte.dev) 和 [Tailwind](https://www.tailwindcss.com) 的移动端 Web 组件库。

> **S**imple • **T**iny • **D**esign • **F**ast

# 特性

- 🔥 完全支持 Svelte v5、Tailwind CSS v4 和 TypeScript。
- 🚀 无运行时，无虚拟 DOM —— 在线运行更快。
- 🧰 丰富的 API —— 轻松配置组件样式以满足你的需求。
- 🍭 支持深色模式和多主题配置。
- 📖 提供丰富的中英文文档和组件示例。
- 🌍 支持国际化，内置 60+ 语言包。
- 🫰 友好的交互体验，动画参数可配置，支持自定义主题。
- 🤝 支持 UnoCSS 等 Tailwind 类库。
- 🫡 无第三方依赖 —— 无需担心库版本或安全问题。
- 📦 支持按需引入，减小包体积。
- 🪜 内置脚手架工具，快速创建项目。
- 🔗 自带 SVG 合并插件，支持快速合并 SVG 文件，支持 Iconify。
- 🍺 提供 IDE 插件，快捷查看组件 API。
