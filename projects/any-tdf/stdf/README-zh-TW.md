# 介紹

基於 [Svelte](https://svelte.dev) 和 [Tailwind](https://www.tailwindcss.com) 的行動端網頁元件庫。

> **S**imple（簡單） • **T**iny（輕量） • **D**esign（設計） • **F**ast（快速）

# 特性

- 🔥 完全支援 Svelte v5、Tailwind CSS v4 及 TypeScript。
- 🚀 無運行時、無虛擬 DOM — 線上運行更快。
- 🧰 豐富的 API — 輕鬆配置元件樣式以滿足您的需求。
- 🍭 支援深色模式與多主題配置。
- 📖 提供詳盡的中英文文件與元件示例。
- 🌍 支援國際化，內建 60+ 語言包。
- 🫰 友善的互動體驗，動畫參數可配置，支援自訂主題。
- 🤝 支援 UnoCSS 等 Tailwind 類庫。
- 🫡 無第三方依賴 — 無需擔心庫版本或安全問題。
- 📦 支援按需引入，減少包體積。
- 🪜 內建腳手架工具，快速創建專案。
- 🔗 內建 SVG 合併插件，快速合併 SVG 檔案，支援 Iconify。
- 🍺 IDE 插件，快速檢視元件 API。

```sh
npx create-stdf@latest
```
<!-- :: -->
