## ℹ プロジェクトについて

[**Note Block World**](https://noteblock.world/) は、ユーザーが [Note Block Studio](https://noteblock.studio/) で作成したノートブロック曲を共有できる、協力型・ソーシャルウェブサイトです。NBSコミュニティが世界中に作品を発信し、新しい音楽を発見し、他のクリエイターと繋がるためのプラットフォームを提供することを目指しています！

## 💬 問題・フィードバック

問題を発見した場合や新機能の提案がある場合は、リポジトリの [新しいイシュー](/issues/new/choose) を作成してください。ご希望の機能が既に計画されているかどうかは [プロジェクトのロードマップ](https://github.com/orgs/OpenNBS/projects/4) をご確認ください。ウェブサイトの改善に努めておりますので、ぜひご意見をお寄せください！

## 🔧 コントリビューション

プロジェクトへの貢献を歓迎します！プロジェクトのセットアップや開始方法については [コントリビューターガイド](https://raw.githubusercontent.com/OpenNBS/NoteBlockWorld/main/CONTRIBUTING.md) をご覧ください。

プルリクエストを送る前に、変更したい内容について [イシューを新規作成](/issues/new/choose) してください。または、[Discordサーバー](https://discord.gg/note-block-world-608692895179997252) に参加し、開発チームと直接ご相談いただくことも可能です！

## ❤ 寄付

プロジェクトを支援したい方は、[Open Collective](https://opencollective.com/opennbs/donate) または [GitHub Sponsors](https://github.com/sponsors/OpenNBS) からご寄付いただけます！皆様からのご支援は、ウェブサイト運営費用の補填や、開発により多くの時間を割くために役立てられます。

Note Block Worldの運営を支えてくださっている、全ての現・過去のスポンサーの皆様に感謝します！
