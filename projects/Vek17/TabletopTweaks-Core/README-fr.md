**Comment installer**

1. Téléchargez et installez [Unity Mod Manager](https://github.com/newman55/unity-mod-manager), assurez-vous qu’il s’agit d’au moins la version 0.23.0
2. Lancez Unity Mod Manager et configurez-le pour détecter Wrath of the Righteous
3. Téléchargez le mod Tabletop Tweaks
4. Installez le mod en glissant le fichier zip de l’étape 3 dans la fenêtre Unity Mod Manager sous l’onglet Mods
