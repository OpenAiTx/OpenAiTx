**Cómo instalar**

1. Descarga e instala [Unity Mod Manager](https://github.com/newman55/unity-mod-manager), asegúrate de que sea al menos la versión 0.23.0
2. Ejecuta Unity Mod Manager y configúralo para encontrar Wrath of the Righteous
3. Descarga el mod Tabletop Tweaks
4. Instala el mod arrastrando el archivo zip del paso 5 a la ventana de Unity Mod Manager en la pestaña Mods
