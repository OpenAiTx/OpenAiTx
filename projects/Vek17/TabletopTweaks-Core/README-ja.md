**インストール方法**

1. [Unity Mod Manager](https://github.com/newman55/unity-mod-manager)をダウンロードしてインストールし、バージョンが0.23.0以上であることを確認してください。
2. Unity Mod Managerを起動し、Wrath of the Righteousを認識させて設定します。
3. Tabletop Tweaksモッドをダウンロードします。
4. 5番目のステップのzipファイルをModsタブのUnity Mod Managerウィンドウにドラッグしてモッドをインストールします。
