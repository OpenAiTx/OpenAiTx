**安装方法**

1. 下载并安装 [Unity Mod Manager](https://github.com/newman55/unity-mod-manager)，确保版本不低于 0.23.0  
2. 运行 Unity Mod Manager 并设置其识别《正义之怒》  
3. 下载 Tabletop Tweaks 模组  
4. 将第 3 步下载的 zip 文件拖入 Unity Mod Manager 窗口的 Mods 标签页中进行安装
