## ✨ 특징

- 🌈 웹 애플리케이션을 위한 엔터프라이즈급 UI 설계
- 📦 바로 사용할 수 있는 고품질 React 컴포넌트 세트
- 🛡 예측 가능한 정적 타입의 TypeScript로 작성됨
- ⚙️ 디자인 리소스와 개발 도구의 전체 패키지 제공
- 🌍 수십 개 언어를 지원하는 국제화
- 🎨 CSS-in-JS 기반의 강력한 테마 커스터마이징

## 🔨 사용법

```tsx
import { Button, DatePicker } from 'antd';

export default () => (
  <>
    <Button type="primary">PRESS ME</Button>
    <DatePicker placeholder="select date" />
  </>
);
```

## ⌨️ 개발

무료 온라인 순수 프론트엔드 개발 환경인 [opensumi.run](https://opensumi.run)을 사용해보세요.

[![opensumi.run](https://custom-icon-badges.demolab.com/badge/opensumi-run-blue.svg?logo=opensumi)](https://opensumi.run/ide/ant-design/ant-design)

또는 로컬로 클론하여 사용하세요:

```bash
$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start
```

브라우저에서 http://127.0.0.1:8001 을 열고, 자세한 내용은 [Development](https://github.com/ant-design/ant-design/wiki/Development)를 참고하세요.
