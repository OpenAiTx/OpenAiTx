## ✨ الميزات

- 🌈 واجهات مستخدم من فئة المؤسسات مصممة لتطبيقات الويب.
- 📦 مجموعة من مكونات React عالية الجودة جاهزة للاستخدام.
- 🛡 مكتوبة بلغة TypeScript مع أنواع ثابتة متوقعة.
- ⚙️ حزمة كاملة من موارد التصميم وأدوات التطوير.
- 🌍 دعم التدويل لعشرات اللغات.
- 🎨 تخصيص قوي للثيمات مبني على CSS-in-JS.

  <>

);

## ⌨️ التطوير

استخدم [opensumi.run](https://opensumi.run)، بيئة تطوير أمامية مجانية بالكامل على الإنترنت.

[![opensumi.run](https://custom-icon-badges.demolab.com/badge/opensumi-run-blue.svg?logo=opensumi)](https://opensumi.run/ide/ant-design/ant-design)

أو استنسخ المشروع محليًا:

```bash
$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start
```

افتح متصفحك وقم بزيارة http://127.0.0.1:8001، للمزيد راجع [التطوير](https://github.com/ant-design/ant-design/wiki/Development).
