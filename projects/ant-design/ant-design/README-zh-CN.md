## ✨ 特性

- 🌈 为 Web 应用而生的企业级 UI 设计。
- 📦 开箱即用的一套高质量 React 组件。
- 🛡 全部使用 TypeScript 编写，类型可预测。
- ⚙️ 完善的设计资源和开发工具。
- 🌍 支持多语言国际化。
- 🎨 强大的基于 CSS-in-JS 的主题定制能力。

## 📦 安装

```bash
npm install antd
```

```bash
yarn add antd
```

```bash
pnpm add antd
```

```bash
bun add antd
```

## 🔨 使用

```tsx
import { Button, DatePicker } from 'antd';

export default () => (
  <>
    <Button type="primary">按我</Button>
    <DatePicker placeholder="请选择日期" />
  </>
);
```

## ⌨️ 开发

使用 [opensumi.run](https://opensumi.run)，一个免费的在线前端开发环境。

[![opensumi.run](https://custom-icon-badges.demolab.com/badge/opensumi-run-blue.svg?logo=opensumi)](https://opensumi.run/ide/ant-design/ant-design)

或本地克隆：

```bash
$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start
```

打开浏览器访问 http://127.0.0.1:8001，更多内容请见 [开发者指引](https://github.com/ant-design/ant-design/wiki/Development)。
