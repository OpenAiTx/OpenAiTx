## ✨ Özellikler

- 🌈 Web uygulamaları için tasarlanmış kurumsal sınıf bir UI.
- 📦 Kutudan çıkar çıkmaz yüksek kaliteli React bileşenleri seti.
- 🛡 Tahmin edilebilir statik tiplerle TypeScript ile yazılmıştır.
- ⚙️ Tasarım kaynakları ve geliştirme araçlarının tam paketi.
- 🌍 Onlarca dil için uluslararasılaştırma desteği.
- 🎨 CSS-in-JS tabanlı güçlü tema özelleştirme.

## ⌨️ Geliştirme

Ücretsiz, çevrimiçi, saf ön uç geliştirme ortamı olan [opensumi.run](https://opensumi.run) kullanın.

[![opensumi.run](https://custom-icon-badges.demolab.com/badge/opensumi-run-blue.svg?logo=opensumi)](https://opensumi.run/ide/ant-design/ant-design)

Veya yerel olarak klonlayın:

```bash
$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start
```

Tarayıcınızı açın ve http://127.0.0.1:8001 adresini ziyaret edin, daha fazla bilgi için [Geliştirme](https://github.com/ant-design/ant-design/wiki/Development) bölümüne bakın.
