InsightCAE
==========

InsightCAE est un banc de travail pour l'ingénierie assistée par ordinateur. Les projets open source individuels répondent souvent à seulement des sous-tâches du processus d'analyse. Pour automatiser les tâches computationnelles répétitives, une combinaison de plusieurs outils logiciels d'ingénierie open source est souvent nécessaire. Afin d'utiliser les logiciels open source de manière productive et efficace pour les tâches quotidiennes, le cadre d'automatisation d'analyse InsightCAE est fourni par silentdynamics.

InsightCAE sert de cadre pour la mise en œuvre des procédures d'analyse. L'objectif est de fournir des interfaces vers les outils et programmes de simulation nécessaires à une tâche informatique spécifique.

Fonctionnalités
--------------

- outils d'automatisation des flux de travail d'analyse (GUI)
- CAO entièrement paramétrique, basée sur des scripts

    - noyau géométrique OpenCASCADE
    - assemblages, esquisses basées sur des contraintes, bibliothèque de pièces, exportation de dessins

- extensions OpenFOAM (schémas, conditions aux limites, modèles, ...)
- outil de pré- & post-traitement (OpenFOAM Case Builder)
