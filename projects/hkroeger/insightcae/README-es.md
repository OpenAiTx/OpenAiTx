InsightCAE
==========

InsightCAE es un banco de trabajo para Ingeniería Asistida por Computadora. Los proyectos individuales de código abierto a menudo solo cubren subtareas en el proceso de análisis. Para automatizar tareas computacionales repetitivas, a menudo se requiere una combinación de varias herramientas de software de ingeniería de código abierto. Para usar el software de código abierto de manera productiva y eficiente en tareas diarias, silentdynamics proporciona el marco de automatización de análisis InsightCAE.

InsightCAE sirve como un marco para la implementación de procedimientos de análisis. El objetivo es proporcionar interfaces a las herramientas y programas de simulación necesarios para una tarea computacional específica.

Características
--------------

- herramientas de automatización de flujo de trabajo de análisis (GUI)
- CAD completamente paramétrico basado en scripts

    - núcleo de geometría OpenCASCADE
    - ensamblajes, bocetos basados en restricciones, biblioteca de piezas, exportación de dibujos

- complementos de OpenFOAM (esquemas, condiciones de frontera, modelos, ...)
- herramienta de preprocesamiento y postprocesamiento (OpenFOAM Case Builder)
