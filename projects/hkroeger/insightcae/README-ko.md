InsightCAE
==========

InsightCAE는 컴퓨터 지원 공학을 위한 워크벤치입니다. 개별 오픈 소스 프로젝트는 종종 분석 과정에서 부분 작업만을 수행합니다. 반복적인 계산 작업을 자동화하기 위해서는 여러 오픈 소스 공학 소프트웨어 도구의 조합이 종종 필요합니다. 일상 업무에 오픈 소스 소프트웨어를 생산적이고 효율적으로 사용하기 위해, 분석 자동화 프레임워크인 InsightCAE가 silentdynamics에 의해 제공됩니다.

InsightCAE는 분석 절차 구현을 위한 프레임워크 역할을 합니다. 목적은 특정 계산 작업에 필요한 도구 및 시뮬레이션 프로그램에 대한 인터페이스를 제공하는 것입니다.

기능
----

- 분석 워크플로우 자동화 도구 (GUI)
- 스크립트 기반, 완전 매개변수화된 CAD

    - OpenCASCADE 기하학 커널
    - 어셈블리, 제약 기반 스케치, 부품 라이브러리, 도면 내보내기

- OpenFOAM 애드온 (스킴, 경계 조건, 모델 등)
- 전처리 및 후처리 도구 (OpenFOAM 케이스 빌더)
