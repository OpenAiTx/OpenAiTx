# Fonctionnalités

Cette application offre des fonctionnalités de base telles que :
* Réglage de la vitesse manuellement ou par "tapotement"
* Si vous utilisez le "tapotement" pendant que le métronome fonctionne, les clics devraient se synchroniser avec vos tapotements.
* Choisir différents sons et adapter individuellement le volume
* Sauvegarder et charger vos réglages actuels
* Choisir entre thème clair et sombre
* Afficher une notification, qui supporte "démarrer" et "arrêter"
