# 功能

此应用提供基础功能，如：
* 手动设置速度或通过“敲击输入”
* 如果在节拍器运行时使用“敲击输入”，点击声应锁定到你的敲击节奏
* 选择不同的声音并单独调整音量
* 保存和加载当前设置
* 选择浅色或深色主题
* 显示通知，支持“开始”和“停止”
