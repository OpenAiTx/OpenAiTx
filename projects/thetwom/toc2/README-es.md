# Funcionalidad

Esta aplicación ofrece funcionalidades básicas como:
* Ajustar la velocidad manualmente o mediante "tocar"
* Si usas "tocar" mientras el metrónomo está funcionando, los clics deberían sincronizarse con tus toques.
* Elegir diferentes sonidos y adaptar individualmente el volumen
* Guardar y cargar tus configuraciones actuales
* Elegir entre tema claro y oscuro
* Mostrar una notificación, que soporta "iniciar" y "detener"
