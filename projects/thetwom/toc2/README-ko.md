# 기능

이 앱은 다음과 같은 기본 기능을 제공합니다:
* 속도를 수동으로 설정하거나 "탭 인"으로 설정
* 메트로놈이 작동 중일 때 "탭 인"을 사용하면, 클릭이 탭에 맞춰 고정됩니다.
* 다양한 소리를 선택하고 개별적으로 볼륨 조절 가능
* 현재 설정을 저장하고 불러오기
* 라이트 및 다크 테마 선택 가능
* "시작"과 "중지"를 지원하는 알림 표시
