# 機能

このアプリは以下の基本機能を提供します:
* 手動または「タップイン」による速度設定
* メトロノームが動作中に「タップイン」を使用すると、クリックがタップに同期します
* 異なる音を選択し、音量を個別に調整可能
* 現在の設定を保存および読み込み
* ライトテーマとダークテーマの選択
* 「開始」と「停止」をサポートする通知の表示
