## 💿 Instalação

Qualquer um dos métodos mencionados abaixo pode ser utilizado

> [!TIP]
> Para habilitar a função de conversão de Markdown para imagem, é necessário instalar `nonebot-plugin-deepseek[image]`

<details open>
<summary>[Recomendado] Instale usando nb-cli</summary>
Abra o terminal na raiz do Bot e digite o comando abaixo para instalar

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>Instalação usando o gerenciador de pacotes</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

Abra o arquivo de configuração no diretório raiz do projeto NoneBot e adicione na seção `[plugin]`

```toml
plugins = ["nonebot_plugin_deepseek"]
```
</details>

## ⚙️ Configuração

Adicione as configurações da tabela abaixo no arquivo de configuração do projeto

> [!note]
> Por favor, obtenha a `api_key` na [DeepSeek Plataforma Aberta](https://platform.deepseek.com/)  
> `enable_models` é um dicionário na estrutura [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34); caso não precise integrar modelos locais, não é necessário modificar  
> Para integrar modelos locais, consulte: 👉 [Wiki - Integrando Modelos Locais](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|           Configuração           | Obrigatório |             Valor Padrão             |                                                                          Descrição                                                                         |
| :------------------------------: | :---------: | :-----------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |     Sim     |                 Nenhum                |                                                                         API Key                                                                             |
|     deepseek__enable_models      |     Não     | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Modelos habilitados [Descrição da Configuração](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |     Não     |                 Nenhum                |                                                                      Pré-definição do modelo                                                                |
|         deepseek__stream         |     Não     |                False                  |                                                               Ativar transmissão em fluxo                                                                  |
|        deepseek__timeout         |     Não     |      {"api_request": 100, "user_input": 60}      |                                                                         Configuração de timeout                                                            |
|       deepseek__md_to_pic        |     Não     |                False                  |                                                        Ativar conversão de Markdown para imagem                                                            |
|  deepseek__enable_send_thinking  |     Não     |                False                  |                                                                  Enviar cadeia de pensamento                                                               |

## 🎉 Uso

> [!note]
> Verifique seu `COMMAND_START` e as configurações acima. Aqui, usamos `/` por padrão

### Ajuda


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Comando rápido: `/ds [conteúdo]` ou responda à mensagem de texto

### Conversa em múltiplas etapas

```bash
/deepseek --with-context [内容]
```

Atalho de comando: `/ds --with-context [conteúdo]` `/diálogo-múltiplo`

### Pensamento Profundo

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Atalho de comando: `/lista-modelos` `/definir-modelo-padrão [nome do modelo]`

### Atalhos de comando personalizados

> Este recurso depende do [Alconna Atalhos de comando](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). Os comandos personalizados não incluem `COMMAND_START`, se necessário deve ser preenchido manualmente

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
