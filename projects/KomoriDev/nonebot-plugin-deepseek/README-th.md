## 💿 การติดตั้ง

เลือกวิธีที่กล่าวถึงด้านล่าง **อย่างใดอย่างหนึ่ง** ก็เพียงพอ

> [!TIP]
> หากต้องการเปิดใช้งานฟังก์ชันแปลง Markdown เป็นรูปภาพ ต้องติดตั้ง `nonebot-plugin-deepseek[image]`

<details open>
<summary>[แนะนำ] ติดตั้งด้วย nb-cli</summary>
เปิดบรรทัดคำสั่งในไดเรกทอรีหลักของ Bot แล้วพิมพ์คำสั่งต่อไปนี้เพื่อทำการติดตั้ง

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>ติดตั้งโดยใช้ตัวจัดการแพ็คเกจ</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

เปิดไฟล์การกำหนดค่าที่อยู่ในไดเรกทอรีหลักของโครงการ NoneBot แล้วเพิ่มเนื้อหาที่ส่วน `[plugin]`

```toml
plugins = ["nonebot_plugin_deepseek"]
```

</details>

## ⚙️ การตั้งค่า

เพิ่มการตั้งค่าตามตารางด้านล่างในไฟล์ตั้งค่าของโปรเจกต์

> [!note]
> `api_key` กรุณารับจาก [DeepSeek Open Platform](https://platform.deepseek.com/)  
> `enable_models` เป็นพจนานุกรมแบบ [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) หากไม่ต้องการเชื่อมต่อโมเดลในเครื่องไม่จำเป็นต้องแก้ไข  
> หากต้องการเชื่อมต่อโมเดลในเครื่อง โปรดดูที่: 👉 [Wiki - เชื่อมต่อโมเดลในเครื่อง](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|              ชื่อการตั้งค่า              | จำเป็น  |                             ค่าเริ่มต้น                             |                                                                          คำอธิบาย                                                                          |
| :------------------------------: | :---: | :------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  ใช่   |                               ไม่มี                               |                                                                        API Key                                                                         |
|     deepseek__enable_models      |  ไม่   | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | โมเดลที่เปิดใช้งาน [คำอธิบายการตั้งค่า](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  ไม่   |                               ไม่มี                               |                                                                        พรีเซ็ตโมเดล                                                                        |
|         deepseek__stream         |  ไม่   |                             False                              |                                                                    เปิดใช้งานการส่งข้อมูลแบบสตรีม                                                                    |
|        deepseek__timeout         |  ไม่   |             {"api_request": 100, "user_input": 60}             |                                                                        ตั้งค่าเวลาใช้งาน                                                                        |
|       deepseek__md_to_pic        |  ไม่   |                             False                              |                                                                เปิดใช้งาน Markdown เป็นรูปภาพ                                                                |
|  deepseek__enable_send_thinking  |  ไม่   |                             False                              |                                                                     ส่ง chain of thought หรือไม่                                                                     |

## 🎉 การใช้งาน

> [!note]
> กรุณาตรวจสอบ `COMMAND_START` และการตั้งค่าข้างต้น โดยค่าเริ่มต้นใช้ `/`

### วิธีใช้

```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

คำสั่งด่วน: `/ds [เนื้อหา]` หรือ ตอบกลับข้อความ

### สนทนาแบบหลายรอบ

```bash
/deepseek --with-context [内容]
```

คำสั่งด่วน: `/ds --with-context [เนื้อหา]` `/สนทนาแบบหลายรอบ`

### การคิดเชิงลึก

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

ทางลัดด่วน: `/รายการโมเดล` `/ตั้งค่าโมเดลเริ่มต้น [ชื่อโมเดล]`

### ทางลัดด่วนที่กำหนดเอง

> ฟีเจอร์นี้ต้องพึ่งพา [Alconna Shortcut Command](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8) ทางลัดที่กำหนดเองจะไม่มี `COMMAND_START` หากจำเป็นต้องกรอกเอง

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
