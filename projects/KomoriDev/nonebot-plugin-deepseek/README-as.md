## 🎉 ব্যৱহাৰ

> [!note]
> অনুগ্ৰহ কৰি আপোনাৰ `COMMAND_START` আৰু ওপৰৰ সংৰূপসমূহ পৰীক্ষা কৰক। ইয়াত ডিফল্ট ৰূপে `/` ব্যৱহাৰ কৰা হৈছে

### সহায়

```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

দ্ৰুত আদেশ: `/ds [সমগ্ৰী]` বা পাঠ্য বার্তা উত্তৰ কৰক

### বহু-পৰ্যায় আলোচনা

```bash
/deepseek --with-context [内容]
```

দ্ৰুত নির্দেশনা: `/ds --with-context [সমগ্ৰী]` `/বহু-চক্ৰ সংলাপ`

### গভীৰ চিন্তা

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

দ্ৰুত নিৰ্দেশনা: `/মডেল তালিকা` `/ডিফল্ট মডেল ছেট কৰক [মডেল নাম]`

### কাষ্টম দ্ৰুত নিৰ্দেশনা

> এই বৈশিষ্ট্যটো [Alconna দ্ৰুত নিৰ্দেশনা](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8)ৰ ওপৰত নিৰ্ভৰ কৰে। কাষ্টম নিৰ্দেশনাত `COMMAND_START` নাথাকে, প্ৰয়োজন হ'লে নিজেৰে যোগ দিব লাগিব

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
