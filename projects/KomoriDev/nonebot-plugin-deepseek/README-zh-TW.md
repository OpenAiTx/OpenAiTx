## 💿 安裝

以下提到的方法任選 **其一** 即可

> [!TIP]
> 想要啟用 Markdown 轉圖片功能，需安裝 `nonebot-plugin-deepseek[image]`

<details open>
<summary>[推薦] 使用 nb-cli 安裝</summary>
在 Bot 的根目錄下打開命令行，輸入以下指令即可安裝

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>使用套件管理器安裝</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

打開 NoneBot 專案根目錄下的設定檔，在 `[plugin]` 部分追加寫入

```toml
plugins = ["nonebot_plugin_deepseek"]
```

</details>

## ⚙️ 配置

在專案的配置檔案中新增下表中的配置

> [!note]
> `api_key` 請從 [DeepSeek 開放平台](https://platform.deepseek.com/) 取得  
> `enable_models` 為 [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) 結構的字典，若無接入本地模型的需求則無需修改  
> 若要接入本地模型，請參見：👉 [Wiki - 接入本地模型](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|              配置項              | 必填  |                             預設值                             |                                                                          說明                                                                          |
| :------------------------------: | :---: | :------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  是   |                               無                               |                                                                        API Key                                                                         |
|     deepseek__enable_models      |  否   | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | 啟用的模型 [配置說明](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  否   |                               無                               |                                                                        模型預設                                                                        |
|         deepseek__stream         |  否   |                             False                              |                                                                    是否啟用串流傳輸                                                                    |
|        deepseek__timeout         |  否   |             {"api_request": 100, "user_input": 60}             |                                                                        超時設定                                                                        |
|       deepseek__md_to_pic        |  否   |                             False                              |                                                                是否啟用 Markdown 轉圖片                                                                |
|  deepseek__enable_send_thinking  |  否   |                             False                              |                                                                     是否傳送思維鏈                                                                     |

## 🎉 使用

> [!note]
> 請檢查你的 `COMMAND_START` 以及上述配置項。此處預設使用 `/`

### 幫助

```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

快捷命令：`/ds [內容]` 或回覆文本訊息

### 多輪對話

```bash
/deepseek --with-context [内容]
```

快捷指令：`/ds --with-context [內容]` `/多輪對話`

### 深度思考

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

快捷指令：`/模型列表` `/設置默認模型 [模型名]`

### 自訂快捷指令

> 此功能依賴於 [Alconna 快捷指令](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8)。自訂指令不帶有 `COMMAND_START`，如有需要請手動填寫

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
