## ⚙️ پیکربندی

در فایل پیکربندی پروژه، تنظیمات زیر را اضافه کنید

> [!note]
> `api_key` را لطفاً از [DeepSeek پلتفرم باز](https://platform.deepseek.com/) دریافت کنید  
> `enable_models` یک دیکشنری به ساختار [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) است؛ اگر نیازی به اتصال مدل محلی ندارید، نیاز به ویرایش نیست  
> برای اتصال مدل محلی، لطفاً ببینید: 👉 [ویکی - اتصال مدل محلی](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|              گزینه پیکربندی              | اجباری |                           مقدار پیش‌فرض                           |                                                                            توضیحات                                                                            |
| :--------------------------------------: | :----: | :---------------------------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  بله   |                              ندارد                               |                                                                        کلید API                                                                              |
|     deepseek__enable_models      |  نه    | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | مدل‌های فعال [توضیحات پیکربندی](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  نه    |                              ندارد                               |                                                                        پیش‌فرض مدل                                                                            |
|         deepseek__stream         |  نه    |                              False                               |                                                                فعال‌سازی انتقال جریانی                                                                       |
|        deepseek__timeout         |  نه    |            {"api_request": 100, "user_input": 60}                |                                                                          تنظیمات تایم‌اوت                                                                      |

## 🎉 استفاده

> [!note]
> لطفاً `COMMAND_START` خود و تنظیمات بالا را بررسی کنید. اینجا به طور پیش‌فرض از `/` استفاده شده است

### راهنما


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

دستور سریع: `/ds [محتوا]` یا پاسخ به پیام متنی

### گفتگو چند مرحله‌ای

```bash
/deepseek --with-context [内容]
```

دستور سریع: `/ds --with-context [محتوا]` `/گفتگوی چندمرحله‌ای`

### تفکر عمیق

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

دستور سریع: `/لیست مدل‌ها` `/تنظیم مدل پیش‌فرض [نام مدل]`

### دستور سریع سفارشی

> این ویژگی وابسته به [دستور سریع Alconna](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8) است. دستور سفارشی بدون `COMMAND_START` بوده و در صورت نیاز باید به صورت دستی وارد شود.

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
