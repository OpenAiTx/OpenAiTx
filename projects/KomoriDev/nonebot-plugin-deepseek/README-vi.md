## ⚙️ Cấu hình

Thêm các cấu hình dưới đây vào file cấu hình của dự án

> [!note]
> `api_key` vui lòng lấy từ [Nền tảng mở DeepSeek](https://platform.deepseek.com/)  
> `enable_models` là một từ điển theo cấu trúc [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34), nếu không cần tích hợp mô hình cục bộ thì không cần chỉnh sửa  
> Nếu muốn tích hợp mô hình cục bộ, vui lòng tham khảo: 👉 [Wiki - Tích hợp mô hình cục bộ](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|           Tên cấu hình            | Bắt buộc |           Giá trị mặc định           |                                                                        Giải thích                                                                         |
| :-------------------------------: | :------: | :----------------------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key          |   Có     |                Không                 |                                                                        API Key                                                                             |
|     deepseek__enable_models       |   Không  | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Các mô hình được kích hoạt [Hướng dẫn cấu hình](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt          |   Không  |                Không                 |                                                                      Thiết lập mặc định cho mô hình                                                        |
|         deepseek__stream          |   Không  |                False                 |                                                                 Có bật truyền dữ liệu dạng stream hay không                                                 |
|        deepseek__timeout          |   Không  |         {"api_request": 100, "user_input": 60}         |                                                                    Thiết lập thời gian chờ                                                                 |
|       deepseek__md_to_pic         |   Không  |                False                 |                                                           Có bật chuyển Markdown sang hình ảnh hay không                                                    |
|  deepseek__enable_send_thinking   |   Không  |                False                 |                                                                  Có gửi chuỗi suy nghĩ hay không                                                           |

## 🎉 Sử dụng

> [!note]
> Vui lòng kiểm tra `COMMAND_START` và các mục cấu hình ở trên. Ở đây mặc định sử dụng `/`

### Trợ giúp


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Lệnh nhanh: `/ds [nội dung]` hoặc trả lời tin nhắn văn bản

### Đối thoại nhiều vòng

```bash
/deepseek --with-context [内容]
```

Phím tắt: `/ds --with-context [nội dung]` `/đa vòng đối thoại`

### Suy nghĩ sâu

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Phím tắt nhanh: `/danh sách mô hình` `/cài đặt mô hình mặc định [tên mô hình]`

### Phím tắt tự định nghĩa

> Tính năng này dựa vào [Phím tắt Alconna](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). Phím tắt tự định nghĩa không có `COMMAND_START`, nếu cần thiết phải điền thủ công

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
