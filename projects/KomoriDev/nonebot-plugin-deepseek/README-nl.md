## 💿 Installatie

Kies **één** van de onderstaande methoden

> [!TIP]
> Wil je Markdown naar afbeelding functionaliteit inschakelen, installeer dan `nonebot-plugin-deepseek[image]`

<details open>
<summary>[Aanbevolen] Installeren met nb-cli</summary>
Open de opdrachtregel in de hoofdmap van de Bot en voer het volgende commando uit om te installeren

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>Installeren met pakketbeheerder</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

Open het configuratiebestand in de hoofdmap van het NoneBot-project en voeg het volgende toe aan het gedeelte `[plugin]`

```toml
plugins = ["nonebot_plugin_deepseek"]
```
</details>

## ⚙️ Configuratie

Voeg de onderstaande configuratie toe aan het configuratiebestand van het project

> [!note]
> `api_key` verkrijg je via het [DeepSeek Open Platform](https://platform.deepseek.com/)  
> `enable_models` is een woordenboek volgens de [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) structuur; als je geen lokale modellen wilt gebruiken, hoef je dit niet aan te passen  
> Voor het aansluiten van lokale modellen, zie: 👉 [Wiki - Lokale modellen aansluiten](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|            Configuratie           | Verplicht |                         Standaardwaarde                         |                                                                        Uitleg                                                                         |
| :------------------------------: | :-------: | :------------------------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |    Ja     |                               Geen                              |                                                                      API-sleutel                                                                      |
|     deepseek__enable_models      |    Nee    | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Ingeschakelde modellen [configuratie-uitleg](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |    Nee    |                               Geen                              |                                                                       Modelpreset                                                                     |
|         deepseek__stream         |    Nee    |                             False                               |                                                                 Streamingsmodus inschakelen                                                          |
|        deepseek__timeout         |    Nee    |             {"api_request": 100, "user_input": 60}             |                                                                      Timeout-instellingen                                                            |
|       deepseek__md_to_pic        |    Nee    |                             False                               |                                                              Markdown-naar-afbeelding inschakelen                                                    |
|  deepseek__enable_send_thinking  |    Nee    |                             False                               |                                                                     Chain-of-thought verzenden inschakelen                                           |

## 🎉 Gebruik

> [!note]
> Controleer je `COMMAND_START` en bovenstaande instellingen. Hier wordt standaard `/` gebruikt

### Help


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Snelle opdracht: `/ds [inhoud]` of antwoord op tekstbericht

### Meervoudige dialoog

```bash
/deepseek --with-context [内容]
```

Sneltoets: `/ds --with-context [inhoud]` `/meerdere-gesprekken`

### Diep nadenken

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```
Sneltoetsen: `/model lijst` `/stel standaardmodel in [modelnaam]`

### Aangepaste sneltoetsen

> Deze functie is afhankelijk van [Alconna Sneltoetsen](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). Aangepaste commando's bevatten geen `COMMAND_START`, indien nodig moet dit handmatig ingevuld worden.


```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
