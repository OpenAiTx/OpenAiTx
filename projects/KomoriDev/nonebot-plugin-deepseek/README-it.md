## 💿 Installazione

È sufficiente scegliere **una** delle seguenti modalità

> [!TIP]
> Per abilitare la funzione di conversione Markdown in immagine, è necessario installare `nonebot-plugin-deepseek[image]`

<details open>
<summary>[Consigliato] Installa tramite nb-cli</summary>
Apri il terminale nella directory principale del Bot e inserisci il seguente comando per installare

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>Installazione tramite gestore di pacchetti</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

Apri il file di configurazione nella directory principale del progetto NoneBot e aggiungi la seguente voce nella sezione `[plugin]`

```toml
plugins = ["nonebot_plugin_deepseek"]
```
</details>

## ⚙️ Configurazione

Aggiungi la seguente configurazione nel file di configurazione del progetto

> [!note]
> `api_key` da ottenere dalla [DeepSeek Piattaforma Aperta](https://platform.deepseek.com/)  
> `enable_models` è un dizionario con la struttura [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34), se non hai bisogno di integrare modelli locali non è necessario modificare  
> Per integrare modelli locali, consulta: 👉 [Wiki - Integrazione Modelli Locali](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|             Configurazione             | Obblig. |                          Valore predefinito                          |                                                                              Descrizione                                                                               |
| :------------------------------------: | :------: | :-------------------------------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key               |   Sì     |                                Nessuno                                |                                                                           API Key                                                                                      |
|     deepseek__enable_models            |   No     | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }]      | Modelli abilitati [Descrizione configurazione](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt               |   No     |                                Nessuno                                |                                                                           Preset del modello                                                                           |
|         deepseek__stream               |   No     |                               False                                   |                                                                  Abilitare o meno la trasmissione in streaming                                                        |
|        deepseek__timeout               |   No     |         {"api_request": 100, "user_input": 60}                        |                                                                           Impostazione timeout                                                                         |
|       deepseek__md_to_pic              |   No     |                               False                                   |                                                             Abilitare la conversione Markdown in immagine                                                             |
|  deepseek__enable_send_thinking        |   No     |                               False                                   |                                                                    Abilitare l'invio della catena di pensiero                                                         |

## 🎉 Utilizzo

> [!note]
> Controlla il tuo `COMMAND_START` e le impostazioni di configurazione sopra. Qui si usa di default `/`

### Aiuto


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Comando rapido: `/ds [contenuto]` oppure rispondi a un messaggio di testo

### Conversazione multi-turno

```bash
/deepseek --with-context [内容]
```

Comando rapido: `/ds --with-context [contenuto]` `/conversazione-multipla`

### Pensiero profondo

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Scorciatoia rapida: `/elenco-modelli` `/imposta-modello-predefinito [nome_modello]`

### Scorciatoie personalizzate

> Questa funzione dipende da [Alconna Comandi Rapidi](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). I comandi personalizzati non includono `COMMAND_START`; se necessario, va inserito manualmente

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
