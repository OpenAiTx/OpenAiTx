## 💿 Installation

Eine der unten genannten Methoden reicht **aus**

> [!TIP]
> Um die Funktion „Markdown zu Bild“ zu aktivieren, muss `nonebot-plugin-deepseek[image]` installiert werden

<details open>
<summary>[Empfohlen] Installation mit nb-cli</summary>
Öffne die Kommandozeile im Hauptverzeichnis des Bots und gib den folgenden Befehl ein, um zu installieren

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>Installation mit Paketmanager</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```

Öffnen Sie die Konfigurationsdatei im Stammverzeichnis des NoneBot-Projekts und fügen Sie im Abschnitt `[plugin]` Folgendes hinzu.

```toml
plugins = ["nonebot_plugin_deepseek"]
```

</details>

## ⚙️ Konfiguration

Fügen Sie die folgende Konfiguration in die Konfigurationsdatei des Projekts ein

> [!note]
> `api_key` erhalten Sie auf der [DeepSeek Open Platform](https://platform.deepseek.com/)  
> `enable_models` ist ein Dictionary der Struktur [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34). Wenn Sie keine lokalen Modelle einbinden möchten, ist keine Änderung notwendig.  
> Wenn Sie ein lokales Modell einbinden möchten, siehe: 👉 [Wiki - Lokales Modell einbinden](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|         Konfigurationspunkt          | Pflicht |             Standardwert              |                                                                        Beschreibung                                                                        |
| :----------------------------------: | :-----: | :-----------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key             |   Ja    |                Kein                   |                                                                          API Key                                                                            |
|     deepseek__enable_models          |   Nein  | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Aktivierte Modelle [Konfigurationsbeschreibung](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt             |   Nein  |                Kein                   |                                                                          Modell-Preset                                                                      |
|         deepseek__stream             |   Nein  |                False                  |                                                                  Aktivierung von Streaming                                                                  |
|        deepseek__timeout             |   Nein  |     {"api_request": 100, "user_input": 60}     |                                                                        Timeout-Einstellung                                                                  |
|       deepseek__md_to_pic            |   Nein  |                False                  |                                                          Aktivierung von Markdown-zu-Bild                                                                  |
|  deepseek__enable_send_thinking      |   Nein  |                False                  |                                                               Senden von Chain-of-Thought aktiviert                                                         |

## 🎉 Nutzung

> [!note]
> Bitte überprüfen Sie Ihren `COMMAND_START` sowie die oben genannten Konfigurationen. Hier wird standardmäßig `/` verwendet.

### Hilfe

```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Schnellbefehl: `/ds [Inhalt]` oder auf Textnachricht antworten

### Mehrstufiger Dialog

```bash
/deepseek --with-context [内容]
```

Schnellbefehl: `/ds --with-context [Inhalt]` `/Mehrfachdialog`

### Tiefgründiges Denken

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Schnellbefehl: `/Modellliste` `/SetzeStandardModell [Modellname]`

### Benutzerdefinierte Schnellbefehle

> Diese Funktion basiert auf [Alconna Schnellbefehle](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). Benutzerdefinierte Befehle enthalten kein `COMMAND_START`, falls nötig muss dies manuell hinzugefügt werden.

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
