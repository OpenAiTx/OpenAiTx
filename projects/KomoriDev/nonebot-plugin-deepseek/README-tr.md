## ⚙️ Yapılandırma

Projede yapılandırma dosyasına aşağıdaki tabloya göre yapılandırma ekleyin

> [!note]
> `api_key` değerini [DeepSeek Açık Platformu](https://platform.deepseek.com/) üzerinden alın  
> `enable_models`, [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) yapısında bir sözlüktür, yerel model eklemesi yapılmayacaksa değiştirmenize gerek yoktur  
> Yerel model eklemek için bkz.: 👉 [Wiki - Yerel Model Entegrasyonu](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|              Yapılandırma Adı              | Zorunlu  |                             Varsayılan Değer                             |                                                                          Açıklama                                                                          |
| :------------------------------: | :---: | :------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  Evet   |                               Yok                               |                                                                        API Anahtarı                                                                         |
|     deepseek__enable_models      |  Hayır   | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Etkin modeller [Yapılandırma Açıklaması](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  Hayır   |                               Yok                               |                                                                        Model ön ayarı                                                                        |
|         deepseek__stream         |  Hayır   |                             False                              |                                                                    Akışlı iletim etkin mi                                                                    |
|        deepseek__timeout         |  Hayır   |             {"api_request": 100, "user_input": 60}             |                                                                        Zaman aşımı ayarı                                                                        |
|       deepseek__md_to_pic        |  Hayır   |                             False                              |                                                                Markdown'dan resme dönüştürme etkin mi                                                                |
|  deepseek__enable_send_thinking  |  Hayır   |                             False                              |                                                                     Düşünce zinciri gönderilsin mi                                                                     |

## 🎉 Kullanım

> [!note]
> Lütfen `COMMAND_START` ve yukarıdaki yapılandırmaları kontrol edin. Burada varsayılan olarak `/` kullanılmıştır.

### Yardım


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Kısa komut: `/ds [içerik]` veya metin mesajına yanıt verin

### Çoklu tur sohbet

```bash
/deepseek --with-context [内容]
```

Kısa yol komutları: `/ds --with-context [içerik]` `/çoklu diyalog`

### Derin Düşünce

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Kısa Komutlar: `/model listesi` `/varsayılan modeli ayarla [model adı]`

### Özelleştirilmiş Kısa Komutlar

> Bu özellik [Alconna Kısa Komutları](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8) özelliğine bağlıdır. Özelleştirilmiş komutlarda `COMMAND_START` bulunmaz, gerekirse manuel olarak eklenmelidir.

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
