> `enable_models` هو قاموس به بنية [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34)، إذا لم تكن بحاجة إلى ربط نموذج محلي فلا حاجة للتعديل  
> إذا أردت ربط نموذج محلي، يرجى مراجعة: 👉 [Wiki - ربط النماذج المحلية](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)

## 🎉 الاستخدام

> [!note]
> يرجى التحقق من قيمة `COMMAND_START` وأيضاً الإعدادات أعلاه. الافتراضي هنا هو `/`

### المساعدة

```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

الأمر السريع: `/ds [المحتوى]` أو الرد على رسالة نصية

### الحوار متعدد الجولات

```bash
/deepseek --with-context [内容]
```

الاختصار السريع: `/ds --with-context [المحتوى]` `/محادثة متعددة الجولات`

### التفكير العميق

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

الاختصار السريع: `/قائمة النماذج` `/تعيين النموذج الافتراضي [اسم النموذج]`

### اختصارات مخصصة

> تعتمد هذه الميزة على [اختصار Alconna](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). لا تحتوي الاختصارات المخصصة على `COMMAND_START`، وإذا لزم الأمر يجب تعبئتها يدويًا

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
