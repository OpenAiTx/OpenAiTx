## 💿 Instalacja

Wybierz **jedną** z poniższych metod

> [!TIP]
> Aby włączyć funkcję konwersji Markdown na obraz, należy zainstalować `nonebot-plugin-deepseek[image]`

<details open>
<summary>[Rekomendowane] Instalacja za pomocą nb-cli</summary>
Otwórz terminal w katalogu głównym Bota i wpisz poniższe polecenie, aby zainstalować

```shell
nb plugin install nonebot-plugin-deepseek
```

</details>
<details>
<summary>Instalacja za pomocą menedżera pakietów</summary>

```shell
pip install nonebot-plugin-deepseek
# or, use poetry
poetry add nonebot-plugin-deepseek
# or, use pdm
pdm add nonebot-plugin-deepseek
```
Otwórz plik konfiguracyjny w katalogu głównym projektu NoneBot i dodaj wpis w sekcji `[plugin]`


```toml
plugins = ["nonebot_plugin_deepseek"]
```
</details>

## ⚙️ Konfiguracja

Dodaj poniższe ustawienia do pliku konfiguracyjnego projektu

> [!note]
> `api_key` pobierz z [DeepSeek Open Platform](https://platform.deepseek.com/)  
> `enable_models` to słownik o strukturze [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34), jeśli nie masz potrzeby podłączenia lokalnego modelu, nie musisz go zmieniać  
> Aby podłączyć lokalny model, zobacz: 👉 [Wiki - Podłączenie lokalnego modelu](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/本地模型)  

|              Nazwa konfiguracji              | Wymagane  |                             Wartość domyślna                             |                                                                          Opis                                                                          |
| :------------------------------: | :---: | :------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  Tak   |                               Brak                              |                                                                        API Key                                                                         |
|     deepseek__enable_models      |  Nie   | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Aktywowane modele [Opis konfiguracji](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  Nie   |                               Brak                              |                                                                        Preset modelu                                                                   |
|         deepseek__stream         |  Nie   |                             False                              |                                                                    Czy włączyć transmisję strumieniową                                                 |
|        deepseek__timeout         |  Nie   |             {"api_request": 100, "user_input": 60}             |                                                                        Ustawienia limitu czasu                                                          |
|       deepseek__md_to_pic        |  Nie   |                             False                              |                                                                Czy włączyć konwersję Markdown na obraz                                                 |
|  deepseek__enable_send_thinking  |  Nie   |                             False                              |                                                                     Czy wysyłać łańcuch myślowy                                                        |

## 🎉 Użycie

> [!note]
> Sprawdź swoje `COMMAND_START` oraz powyższe elementy konfiguracji. Domyślnie używane jest `/`

### Pomoc


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

Skrót polecenia: `/ds [treść]` lub odpowiedz na wiadomość tekstową

### Wieloetapowa rozmowa

```bash
/deepseek --with-context [内容]
```

Skrót polecenia: `/ds --with-context [treść]` `/wielokrotna rozmowa`

### Głębokie myślenie

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

Skróty poleceń: `/model-list` `/set-default-model [nazwa modelu]`

### Niestandardowe skróty poleceń

> Ta funkcja zależy od [Alconna Skróty poleceń](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8). Niestandardowe polecenia nie zawierają `COMMAND_START`, w razie potrzeby należy je wpisać ręcznie

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
