## ⚙️ कॉन्फ़िगरेशन

प्रोजेक्ट की कॉन्फ़िगरेशन फ़ाइल में नीचे दी गई सेटिंग्स जोड़ें

> [!note]
> `api_key` कृपया [DeepSeek ओपन प्लेटफॉर्म](https://platform.deepseek.com/) से प्राप्त करें  
> `enable_models` [`CustomModel`](https://github.com/KomoriDev/nonebot-plugin-deepseek/blob/ee9f0b0f0568eedb3eb87423e6c1bf271787ab76/nonebot_plugin_deepseek/config.py#L34) संरचना की डिक्शनरी है, यदि स्थानीय मॉडल को जोड़ने की आवश्यकता नहीं है तो इसमें बदलाव आवश्यक नहीं  
> यदि स्थानीय मॉडल जोड़ना हो तो देखें: 👉 [Wiki - स्थानीय मॉडल को जोड़ना](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/स्थानीय-मॉडल)  

|             कॉन्फ़िगरेशन           | आवश्यक |                           डिफ़ॉल्ट मान                          |                                                                             विवरण                                                                          |
| :------------------------------: | :---: | :------------------------------------------------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------: |
|        deepseek__api_key         |  हाँ   |                               नहीं                               |                                                                         API कुंजी                                                                          |
|     deepseek__enable_models      |  नहीं  | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | सक्षम किए गए मॉडल [कॉन्फ़िगरेशन विवरण](https://github.com/KomoriDev/nonebot-plugin-deepseek/wiki/%E9%85%8D%E7%BD%AE#enable_models-%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E) |
|         deepseek__prompt         |  नहीं  |                               नहीं                               |                                                                          मॉडल प्रीसेट                                                                       |
|         deepseek__stream         |  नहीं  |                             False                              |                                                                   क्या स्ट्रीमिंग सक्षम करें                                                                |
|        deepseek__timeout         |  नहीं  |             {"api_request": 100, "user_input": 60}             |                                                                          टाइमआउट सेटिंग                                                                    |
|       deepseek__md_to_pic        |  नहीं  |                             False                              |                                                               क्या Markdown को इमेज में बदलें                                                              |
|  deepseek__enable_send_thinking  |  नहीं  |                             False                              |                                                                 क्या Thinking Chain भेजें                                                                  |

## 🎉 उपयोग

> [!note]
> कृपया अपने `COMMAND_START` और ऊपर दिए गए कॉन्फ़िगरेशन की जांच करें। यहां डिफ़ॉल्ट रूप से `/` का उपयोग किया गया है

### सहायता


```bash
deepseek --help
         --balance

         [...content] 
           --use-model [model]
           --with-context
           --render | -r

         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

### 问答

```bash
/deepseek [内容]
```

शॉर्टकट कमांड: `/ds [सामग्री]` या टेक्स्ट संदेश का उत्तर दें

### बहु-चरण संवाद

```bash
/deepseek --with-context [内容]
```

त्वरित निर्देश: `/ds --with-context [सामग्री]` `/बहु-चरण संवाद`

### गहन विचार

```bash
/deepseek [内容] --use-model deepseek-v4-pro
```

快捷指令：`/深度思考 [内容]`

### 余额

> 权限：SUPERUSER

```bash
/deepseek --balance
```

快捷指令：`/ds --balance` `/余额`

### 设置

> 权限：`设置默认模型` 指令仅 SUPERUSER 可用

```bash
# 查看支持的模型列表
/deepseek model -l|--list
# 设置默认模型
/deepseek model --set-default [模型名]
# 设置 Markdown 转图片
/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']
```

शॉर्टकट निर्देश: `/मॉडल सूची` `/डिफ़ॉल्ट मॉडल सेट करें [मॉडल नाम]`

### कस्टम शॉर्टकट निर्देश

> यह सुविधा [Alconna शॉर्टकट निर्देश](https://nonebot.dev/docs/2.3.3/best-practice/alconna/command#command%E7%9A%84%E4%BD%BF%E7%94%A8) पर निर्भर करती है। कस्टम निर्देश में `COMMAND_START` नहीं होता, आवश्यकता होने पर इसे मैन्युअल रूप से भरना होगा

```bash
# 增加
/deepseek --shortcut <自定义指令> /deepseek
# 删除
/deepseek --shortcut delete <自定义指令>
# 列出
/deepseek --shortcut list
```

例子:

```bash
user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
```
