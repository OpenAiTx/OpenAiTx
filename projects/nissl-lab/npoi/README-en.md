Advantage of NPOI
=================
a. It's totally free to use

b. Covers most features of Excel (cell style, data format, formula, and so on)

c. Supported formats: xls, xlsx, docx.

d. Designed to be interface-oriented (take a look at the NPOI.SS namespace)

e. Supports not only export but also import

f. Real successful cases all over the world

g. Works on both Windows and Linux 

System Requirement
===================
