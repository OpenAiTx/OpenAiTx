# 🍔 Foodike
Foodikeはシンプルで使いやすいフードデリバリーアプリです。Android-Jetpack Composeを使用して構築されており、MVVMとモダンAndroidアーキテクチャコンポーネントの原則に基づいています。

## 使用技術 🛠

- [Kotlin](https://kotlinlang.org/) - Android開発のための公式の第一級プログラミング言語。
- [Jetpack Compose](https://developer.android.com/jetpack/compose) - Jetpack ComposeはAndroidの
  最新のネイティブUI構築ツールキットです。
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) - コルーチンは
  Androidで非同期に実行されるコードを簡素化するための並行性設計パターンです。
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flowは、値を遅延生成する
  シーケンスの非同期バージョンです。
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture) -
  頑健でテスト可能かつ保守しやすいアプリ設計を支援するライブラリ群です。
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) - StateFlowは
    現在および新しい状態更新をコレクターへ発行する状態保持型のオブザーバブルFlowです。
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flowは、値を遅延生成する
    シーケンスの非同期バージョンです。
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) - UI関連データを
    UI変更時にも破棄されずに保持します。
  - [Jetpack Compose Navigation](https://developer.android.com/jetpack/compose/navigation) - この
    NavigationコンポーネントはJetpack Composeアプリ向けナビゲーションをサポートします。
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - Jetpack
    DataStoreは、プロトコルバッファを用いてキー・バリューや型付きオブジェクトを保存できる
    データ保存ソリューションです。DataStoreはKotlin CoroutinesおよびFlowを使い、非同期・
    一貫性・トランザクション性のあるデータ保存を実現します。
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) - HiltはAndroid向けの依存性注入ライブラリで、手動依存性注入のボイラープレートを削減します。手動で依存性注入を行う場合、各クラスとその依存関係をすべて手で構築し、依存性を再利用・管理するためのコンテナも使う必要があります。
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) - Android 12以降、SplashScreen APIによりアニメーション付きでアプリ起動、アプリアイコンを表示するスプラッシュ画面、アプリ本体への遷移などが可能です。SplashScreenはウィンドウの一種で、アクティビティ全体を覆います。
- [Material Components for Android](https://github.com/material-components/material-components-android)
  - Android向けのモジュール化可能でカスタマイズ可能なMaterial Design UIコンポーネント。
- [Figma](https://figma.com/) - Figmaは主にウェブベースのベクターグラフィックエディタ兼
  プロトタイピングツールです。

## アーキテクチャ 🗼



このアプリは[***MVVM (Model View
View-Model)***](https://developer.android.com/jetpack/docs/guide#recommended-app-arch)アーキテクチャを使用しています。
