# 🍔 Foodike
Foodike ist eine einfache, benutzerfreundliche Essenslieferungs-App. Sie wurde mit Android-Jetpack Compose entwickelt und basiert auf dem Prinzip von MVVM mit modernen Android Architekturkomponenten.

## Entwickelt mit 🛠

- [Kotlin](https://kotlinlang.org/) – Erstklassige und offizielle Programmiersprache für die Android-
  Entwicklung.
- [Jetpack Compose](https://developer.android.com/jetpack/compose) – Jetpack Compose ist das moderne Toolkit von Android zum Erstellen nativer UIs.
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) – Eine Coroutine ist ein
  Nebenläufigkeits-Designmuster, mit dem Sie auf Android Code vereinfachen können, der asynchron
  ausgeführt wird.
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) – Ein Flow ist eine asynchrone
  Version einer Sequence, eine Sammlung, deren Werte verzögert erzeugt werden.
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture) –
  Sammlung von Bibliotheken, die Ihnen helfen, robuste, testbare und wartbare Apps zu entwickeln.
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) – StateFlow ist ein
    zustandshaltender, beobachtbarer Flow, der aktuelle und neue Statusaktualisierungen an seine Collector sendet.
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) – Ein Flow ist eine asynchrone
    Version einer Sequence, eine Sammlung, deren Werte verzögert erzeugt werden.
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) – Speichert
    UI-bezogene Daten, die bei UI-Änderungen nicht verloren gehen.
  - [Jetpack Compose Navigation](https://developer.android.com/jetpack/compose/navigation) – Die
    Navigationskomponente bietet Unterstützung für Jetpack Compose-Anwendungen.
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) – Jetpack
    DataStore ist eine Datenspeicherlösung, mit der Sie Schlüssel-Wert-Paare oder typisierte Objekte
    mit Protokollpuffern speichern können. DataStore verwendet Kotlin Coroutines und Flow, um Daten asynchron, konsistent und transaktional zu speichern.
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) – Hilt ist eine Dependency-Injection-Bibliothek für Android, die den Boilerplate-Code für manuelle Dependency Injection im Projekt reduziert. Bei manueller Dependency Injection müssen Sie jede Klasse und ihre Abhängigkeiten von Hand konstruieren und Container zur Wiederverwendung und Verwaltung von Abhängigkeiten verwenden.
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) – Ab Android 12 ermöglicht die SplashScreen API das Starten von Apps mit Animation, einschließlich einer Bewegung beim Start, einem Splashscreen mit Ihrem App-Symbol und einer Überleitung zur App selbst. Ein SplashScreen ist ein Fenster und deckt daher eine Aktivität ab.
- [Material Components for Android](https://github.com/material-components/material-components-android)
  – Modulare und anpassbare Material-Design-UI-Komponenten für Android.
- [Figma](https://figma.com/) – Figma ist ein vektorbasiertes Grafik- und Prototyping-Tool, das
  hauptsächlich webbasiert ist.

## Architektur 🗼
