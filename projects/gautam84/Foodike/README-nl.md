# 🍔 Foodike
Foodike is een eenvoudige, gebruiksvriendelijke bezorgapp voor eten. De app is gebouwd met Android-Jetpack Compose en gebaseerd op het MVVM-principe met moderne Android Architectuurcomponenten.

## Gemaakt met 🛠

- [Kotlin](https://kotlinlang.org/) - Eerste klas en officiële programmeertaal voor Android
  ontwikkeling.
- [Jetpack Compose](https://developer.android.com/jetpack/compose) - Jetpack Compose is Android's
  moderne toolkit voor het bouwen van native UI.
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) - Een coroutine is een
  concurrentiepatroon dat je op Android kunt gebruiken om code die asynchroon wordt uitgevoerd te
  vereenvoudigen.
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Een flow is een asynchrone
  versie van een Sequence, een type collectie waarvan de waarden lui worden geproduceerd.
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture) -
  Collectie van bibliotheken die helpen bij het ontwerpen van robuuste, testbare en onderhoudbare apps.
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) - StateFlow is een
    state-holder observeerbare flow die de huidige en nieuwe statusupdates naar zijn collectors uitzendt.
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Een flow is een asynchrone
    versie van een Sequence, een type collectie waarvan de waarden lui worden geproduceerd.
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) - Slaat
    UI-gerelateerde data op die niet wordt vernietigd bij UI-wijzigingen.
  - [Jetpack Compose Navigation](https://developer.android.com/jetpack/compose/navigation) - De
    Navigatiecomponent biedt ondersteuning voor Jetpack Compose-applicaties.
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - Jetpack
    DataStore is een gegevensopslagoplossing waarmee je sleutel-waarde-paren of getypte objecten kunt
    opslaan met protocol buffers. DataStore gebruikt Kotlin coroutines en Flow om gegevens asynchroon,
    consistent en transactioneel op te slaan.
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) - Hilt is een dependency injection library voor Android die het aantal handmatige dependency injection stappen in je project vermindert. Handmatige dependency injection vereist dat je elke klasse en zijn                 afhankelijkheden met de hand construeert, en containers gebruikt om afhankelijkheden te hergebruiken en te beheren.
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) - Vanaf Android 12 stelt de SplashScreen API apps in staat om te starten met animatie, inclusief een in-app beweging bij het starten, een splashscreen met je app-icoon, en een overgang naar de app zelf. Een SplashScreen is een Window en dekt dus een Activity af.
- [Material Components voor Android](https://github.com/material-components/material-components-android)
  - Modulaire en aanpasbare Material Design UI-componenten voor Android.
- [Figma](https://figma.com/) - Figma is een vector grafische editor en prototyping tool die
  voornamelijk web-based is.

## Architectuur 🗼

View-Model)***](https://developer.android.com/jetpack/docs/guide#recommended-app-arch) architectuur.
