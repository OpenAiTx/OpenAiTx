# 🍔 Foodike
Foodike एक सरल, उपयोग में आसान फूड डिलीवरी ऐप है। यह Android-Jetpack Compose का उपयोग करके बनाया गया है और MVVM के सिद्धांत पर आधुनिक Android आर्किटेक्चर कंपोनेंट्स के साथ निर्मित है।

## किससे बनाया गया 🛠

- [Kotlin](https://kotlinlang.org/) - एंड्रॉइड विकास के लिए प्रथम श्रेणी और आधिकारिक प्रोग्रामिंग भाषा।
- [Jetpack Compose](https://developer.android.com/jetpack/compose) - Jetpack Compose एंड्रॉइड के लिए
  मूल UI बनाने का आधुनिक टूलकिट है।
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) - एक coroutine एक
  concurrency डिज़ाइन पैटर्न है जिसका उपयोग आप एंड्रॉइड पर असिंक्रोनस रूप से निष्पादित होने वाले कोड को सरल बनाने के लिए कर सकते हैं।
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flow Sequence का असिंक्रोनस
  संस्करण है, एक प्रकार का संग्रह जिसके मान आलसी तरीके से उत्पन्न होते हैं।
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture) -
  लाइब्रेरी का संग्रह जो आपको मजबूत, परीक्षण योग्य और अनुरक्षित ऐप्स डिज़ाइन करने में मदद करता है।
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) - StateFlow एक
    state-holder observable flow है जो अपने कलेक्टर्स को वर्तमान और नए state अपडेट्स भेजता है।
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flow Sequence का असिंक्रोनस
    संस्करण है, एक प्रकार का संग्रह जिसके मान आलसी तरीके से उत्पन्न होते हैं।
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) - UI से संबंधित डेटा को स्टोर करता है जो UI बदलाव पर नष्ट नहीं होता।
  - [Jetpack Compose Navigation](https://developer.android.com/jetpack/compose/navigation) - Navigation कॉम्पोनेन्ट Jetpack Compose एप्लिकेशन के लिए समर्थन प्रदान करता है।
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - Jetpack
    DataStore एक डेटा स्टोरेज समाधान है जो आपको key-value pairs या टाइप्ड ऑब्जेक्ट्स को स्टोर करने की सुविधा देता है, प्रोटोकॉल बफ़र्स के साथ। DataStore Kotlin coroutines और Flow का उपयोग करता है ताकि डेटा असिंक्रोनस, लगातार और ट्रांज़ेक्शनल रूप से स्टोर हो सके।
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) - Hilt एंड्रॉइड के लिए एक डिपेंडेंसी इंजेक्शन लाइब्रेरी है, जो आपके प्रोजेक्ट में मैनुअल डिपेंडेंसी इंजेक्शन का बोइलरप्लेट कम करती है। मैनुअल डिपेंडेंसी इंजेक्शन में आपको हर क्लास और उसकी डिपेंडेंसियों को हाथ से बनाना पड़ता है, और डिपेंडेंसियों को फिर से उपयोग और प्रबंधित करने के लिए कंटेनर का उपयोग करना पड़ता है।
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) - एंड्रॉइड 12 से शुरू होकर, SplashScreen API ऐप्स को एनिमेशन के साथ लॉन्च करने देती है, जिसमें लॉन्च पर ऐप आइकन दिखाने वाली स्प्लैश स्क्रीन और ऐप में ट्रांजिशन शामिल है। SplashScreen एक विंडो होती है और इसलिए एक एक्टिविटी को कवर करती है।
- [Material Components for Android](https://github.com/material-components/material-components-android)
  - एंड्रॉइड के लिए मॉड्यूलर और अनुकूलन योग्य मटेरियल डिज़ाइन UI कॉम्पोनेन्ट्स।
- [Figma](https://figma.com/) - Figma एक वेक्टर ग्राफिक्स एडिटर और प्रोटोटाइपिंग टूल है जो
  मुख्य रूप से वेब-आधारित है।
