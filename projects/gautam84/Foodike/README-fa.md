# 🍔 فودایک
فودایک یک اپلیکیشن ساده و کاربرپسند برای سفارش آنلاین غذا است. این برنامه با استفاده از Android-Jetpack Compose توسعه یافته و بر پایه اصل MVVM با اجزای معماری مدرن اندروید ساخته شده است.

## ساخته شده با 🛠

- [Kotlin](https://kotlinlang.org/) - زبان برنامه‌نویسی رسمی و سطح اول برای توسعه اندروید.
- [Jetpack Compose](https://developer.android.com/jetpack/compose) - جت‌پک کامپوز ابزار مدرن اندروید برای ساخت رابط کاربری بومی است.
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) - کوروتین یک الگوی طراحی همزمانی است که در اندروید برای ساده‌سازی کدهای اجرای غیرهمزمان استفاده می‌شود.
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - فلو نسخه غیرهمزمانی از Sequence است؛ نوعی مجموعه که مقادیر آن به صورت تنبل تولید می‌شوند.
- [اجزای معماری اندروید](https://developer.android.com/topic/libraries/architecture) -
  مجموعه‌ای از کتابخانه‌ها که به طراحی اپ‌های قوی، قابل تست و قابل نگهداری کمک می‌کنند.
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) - StateFlow یک جریان مشاهده‌پذیر نگهدارنده وضعیت است که وضعیت فعلی و جدید را به جمع‌کننده‌های خود ارسال می‌کند.
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - فلو نسخه غیرهمزمانی از Sequence است؛ نوعی مجموعه که مقادیر آن به صورت تنبل تولید می‌شوند.
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) - داده‌های مرتبط با UI را ذخیره می‌کند که در تغییرات UI از بین نمی‌روند.
  - [ناوبری جت‌پک کامپوز](https://developer.android.com/jetpack/compose/navigation) - کامپوننت ناوبری پشتیبانی از برنامه‌های جت‌پک کامپوز را فراهم می‌کند.
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - جت‌پک دیتاستور راه‌حل ذخیره‌سازی داده است که امکان ذخیره جفت‌های کلید-مقدار یا اشیاء تایپ شده با پروتکل بافرها را فراهم می‌کند. دیتاستور از کوروتین‌های Kotlin و Flow برای ذخیره داده‌ها به صورت غیرهمزمان، پیوسته و تراکنشی استفاده می‌کند.
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) - هیلت یک کتابخانه تزریق وابستگی برای اندروید است که حجم کار تزریق دستی وابستگی‌ها در پروژه را کاهش می‌دهد. تزریق وابستگی دستی نیاز دارد هر کلاس و وابستگی‌های آن را به صورت دستی بسازید و برای مدیریت و استفاده مجدد وابستگی‌ها از کانتینرها بهره بگیرید.
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) - از اندروید ۱۲، SplashScreen API به اپ‌ها اجازه می‌دهد با انیمیشن راه‌اندازی شوند، شامل حرکت ورود به اپ، نمایش اسپلش اسکرین با آیکن اپ و انتقال به خود اپ. اسپلش اسکرین یک Window است و به همین خاطر یک Activity را می‌پوشاند.
- [اجزای متریال برای اندروید](https://github.com/material-components/material-components-android)
  - کامپوننت‌های UI متریال دیزاین ماژولار و قابل سفارشی‌سازی برای اندروید.
- [Figma](https://figma.com/) - فیگما یک ویرایشگر گرافیک برداری و ابزار نمونه‌سازی است که عمدتاً مبتنی بر وب می‌باشد.
