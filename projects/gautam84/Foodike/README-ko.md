# 🍔 Foodike
Foodike는 간단하고 사용하기 쉬운 음식 배달 앱입니다. Android-Jetpack Compose로 제작되었으며, MVVM 원칙과 최신 안드로이드 아키텍처 컴포넌트 기반으로 구축되었습니다.

## 빌드 도구 🛠

- [Kotlin](https://kotlinlang.org/) - Android 개발을 위한 공식 프로그래밍 언어입니다.
- [Jetpack Compose](https://developer.android.com/jetpack/compose) - Jetpack Compose는 네이티브 UI를 구축하기 위한 Android의 최신 툴킷입니다.
- [Coroutines](https://kotlinlang.org/docs/reference/coroutines-overview.html) - 코루틴은 비동기적으로 실행되는 코드를 단순화하기 위해 Android에서 사용할 수 있는 동시성 디자인 패턴입니다.
- [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flow는 값이 지연 생성되는 시퀀스의 비동기 버전입니다.
- [Android Architecture Components](https://developer.android.com/topic/libraries/architecture) - 견고하고 테스트 가능하며 유지 관리가 쉬운 앱을 설계하는 데 도움이 되는 라이브러리 모음입니다.
  - [Stateflow](https://developer.android.com/kotlin/flow/stateflow-and-sharedflow) - StateFlow는 현재 및 새로운 상태 업데이트를 수집자에게 내보내는 상태 보유 관찰 가능한 흐름입니다.
  - [Flow](https://kotlinlang.org/docs/reference/coroutines/flow.html) - Flow는 값이 지연 생성되는 시퀀스의 비동기 버전입니다.
  - [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel) - UI 변경 시 파괴되지 않는 UI 관련 데이터를 저장합니다.
  - [Jetpack Compose Navigation](https://developer.android.com/jetpack/compose/navigation) - Navigation 컴포넌트는 Jetpack Compose 애플리케이션을 지원합니다.
  - [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - Jetpack DataStore는 프로토콜 버퍼를 사용하여 키-값 쌍 또는 타입 객체를 저장할 수 있는 데이터 저장 솔루션입니다. DataStore는 Kotlin 코루틴과 Flow를 사용하여 데이터를 비동기적으로, 일관되게, 트랜잭션 방식으로 저장합니다.
- [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android) - Hilt는 Android를 위한 의존성 주입 라이브러리로, 프로젝트에서 수동 의존성 주입의 반복 코드를 줄여줍니다. 수동 의존성 주입은 모든 클래스와 그 의존성을 직접 생성하고, 컨테이너를 사용해 의존성을 재사용 및 관리해야 합니다.
  
- [Splash API](https://developer.android.com/develop/ui/views/launch/splash-screen) - Android 12부터 SplashScreen API를 통해 앱이 애니메이션과 함께 실행되며, 실행 시 인앱 모션, 앱 아이콘을 보여주는 스플래시 화면, 실제 앱으로의 전환을 지원합니다. SplashScreen은 Window이므로 Activity를 덮습니다.
- [Material Components for Android](https://github.com/material-components/material-components-android)
  - Android를 위한 모듈형 및 사용자 정의 가능한 머티리얼 디자인 UI 컴포넌트입니다.
- [Figma](https://figma.com/) - Figma는 주로 웹 기반의 벡터 그래픽 편집 및 프로토타이핑 도구입니다.
