# BiliAnalysis

[![License](https://img.shields.io/github/license/mmyo456/BiliAnalysis?style=flat-square)](https://github.com/mmyo456/BiliAnalysis/blob/main/LICENSE)

**경량 · 간결 · 오픈소스 · 모듈화**

비리비리 영상과 라이브 스트림 직접 링크를 획득하는 유저스크립트 및 브라우저 확장 프로그램입니다.

## 기능 특징

- 영상과 라이브룸 분석 지원
- 로컬 분석 및 클라우드 분석 제공
- 우클릭 메뉴로 직접 링크 빠른 복사
- 모듈화 설계, 필요에 따라 설치 가능

## 버전 선택

| 버전 | 분석 방식 | 설명 |
| :--- | :--- | :--- |
| **BiliCloudAnalysis** | 클라우드 (추천) | 안정적이며 1080P+ 분석 문제 해결 |
| BiliAnalysis | 로컬 | 속도 빠르나 일부 고화질 영상 실패 가능 |
| BiliRoomAnalysis | 로컬 | 라이브룸 전용 |
| 우클릭 분석 플러그인 | 브라우저 확장 | 클라우드 버전과 동일, 유저스크립트 불필요 |

### 브라우저 확장 프로그램

1. `우클릭 분석 플러그인.zip` 다운로드 후 압축 해제
2. 브라우저 확장 프로그램 페이지에서 "개발자 모드" 활성화
3. 압축 해제한 확장 프로그램 불러오기
4. 자세한 내용은 [Edge 확장 프로그램 불러오기 튜토리얼](https://learn.microsoft.com/zh-cn/microsoft-edge/extensions-chromium/getting-started/extension-sideloading) 참조

## 사용 방법

비리비리 사이트의 동영상 제목에서 우클릭 후 "동영상 분석" 선택하면 직접 링크를 얻을 수 있습니다.

> **중요 알림**: 클라우드 분석은 공용 서버에 의존하므로 과도한 사용을 삼가주세요. VRChat 월드나 웹사이트에서 사용할 경우 원작자 정보를 반드시 남겨주세요.
