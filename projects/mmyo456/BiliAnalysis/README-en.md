**Lightweight · Simple · Open Source · Modular**

Userscript and browser extension for obtaining direct links of Bilibili videos and live streams.

## Features

- Supports video and live room parsing
- Offers local parsing and cloud parsing
- Quick copy direct link from right-click menu
- Modular design, install as needed

## Version Selection

| Version | Parsing Method | Description |
| :--- | :--- | :--- |
| **BiliCloudAnalysis** | Cloud (Recommended) | Stable and reliable, solves 1080P+ parsing issues |
| BiliAnalysis | Local | Fast, but some HD videos may fail |
| BiliRoomAnalysis | Local | For live rooms only |
| Right-click Parsing Plugin | Browser Extension | Same as cloud version, no userscript needed |

## Quick Installation

### Userscript

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. Click the links below to install the script

| Local Parsing | [Install](https://raw.githubusercontent.com/mmyo456/BiliAnalysis/main/BiliAnalysis.user.js) | [Install](https://raw.gitmirror.com/mmyo456/BiliAnalysis/main/BiliAnalysis.user.js) |
| Live Room Parsing | [Install](https://raw.githubusercontent.com/mmyo456/BiliAnalysis/main/BiliRoomAnalysis.user.js) | [Install](https://raw.gitmirror.com/mmyo456/BiliAnalysis/main/BiliRoomAnalysis.user.js) |

### Browser Extension

1. Download `RightClickParsingPlugin.zip` and unzip it  
2. Enable "Developer Mode" on the browser extensions page  
3. Load the unpacked extension  
4. See [Edge Extension Loading Tutorial](https://learn.microsoft.com/en-us/microsoft-edge/extensions-chromium/getting-started/extension-sideloading) for details

## Usage

Right-click on the video title on the Bilibili website and select "Video Parsing" to get the direct link.

> **Important Note**: Cloud parsing depends on public servers; please do not abuse it. When using in VRChat worlds or websites, please keep the original author information.
