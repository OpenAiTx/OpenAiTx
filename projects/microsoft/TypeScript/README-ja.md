# TypeScript

[![CI](https://github.com/microsoft/TypeScript/actions/workflows/ci.yml/badge.svg)](https://github.com/microsoft/TypeScript/actions/workflows/ci.yml)
[![npm version](https://badge.fury.io/js/typescript.svg)](https://www.npmjs.com/package/typescript)
[![Downloads](https://img.shields.io/npm/dm/typescript.svg)](https://www.npmjs.com/package/typescript)
[![OpenSSF Scorecard](https://api.securityscorecards.dev/projects/github.com/microsoft/TypeScript/badge)](https://securityscorecards.dev/viewer/?uri=github.com/microsoft/TypeScript)

[TypeScript](https://www.typescriptlang.org/) は、アプリケーション規模のJavaScript向けの言語です。TypeScriptは、JavaScriptにオプションの型を追加し、あらゆるブラウザ、あらゆるホスト、あらゆるOS上で大規模なJavaScriptアプリケーションのためのツールをサポートします。TypeScriptは、可読性が高く標準に基づいたJavaScriptにコンパイルされます。[プレイグラウンド](https://www.typescriptlang.org/play/)で試したり、[公式ブログ](https://blogs.msdn.microsoft.com/typescript)や[Twitterアカウント](https://twitter.com/typescript)で最新情報を入手してください。

TypeScriptを利用している他のユーザーは、[コミュニティページ](https://www.typescriptlang.org/community/)で見つけることができます。

## インストール

最新の安定版バージョンの場合:

```bash
npm install -D typescript
```

ナイトリービルドの場合:

```bash
npm install -D typescript@next
```

TypeScriptには多くの[貢献方法](https://github.com/microsoft/TypeScript/blob/main/CONTRIBUTING.md)があります。
* [バグを報告](https://github.com/microsoft/TypeScript/issues)し、修正がコミットされた際に検証にご協力ください。
* [ソースコードの変更](https://github.com/microsoft/TypeScript/pulls)をレビューしてください。
* [StackOverflow](https://stackoverflow.com/questions/tagged/typescript)で他のTypeScriptユーザーや開発者と交流してください。

* Twitterの[#typescript](https://twitter.com/search?q=%23TypeScript)ディスカッションに参加しましょう。
