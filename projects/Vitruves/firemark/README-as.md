<h1 align="center">firemark</h1>

<p align="center">এটা দ্ৰুত, একক-বাইনাৰী জলচিহ্ন যোগ কৰাৰ টুল, চিত্ৰ আৰু PDF ৰ বাবে। ৰাষ্টত নিৰ্মিত।</p>


## আপোনাৰ নথিপত্ৰসমূহত ৱাটাৰমাৰ্ক কিয় প্ৰয়োগ কৰিব?

প্ৰত্যেক বছৰে, লাখ লাখ মানুহ পৰিচয় জালিয়াতিৰ শিকার হয় যাৰ আৰম্ভণি হয়
এটা সাধাৰণ নথিপত্ৰ বিনিময়ৰ পৰা। এটা সাধাৰণ পৰিস্থিতি: আপুনি ভাড়াৰ বাবে ঘৰ বিচাৰি আছে।
ঘৰৰ মালিক — অথবা কোনোবাই যিয়ে মালিক বুলি ভাৱ প্ৰকাশ কৰিছে — আপোনাৰ পৰিচয় পত্ৰৰ কপি,
দৰমহাৰ চেক, কৰ নোটিচ বিচাৰে। আপুনি চিহ্ন বিহীনভাৱে পঠিয়াই দিয়ে। "মালিক"টো উধাও হৈ যায়, আৰু আপোনাৰ
নথিপত্ৰসমূহ এতিয়া বেংক একাউণ্ট খোলাত, ঋণ লোৱাত, অথবা আপোনাৰ নামত
পৰিচয় নকল কৰিবলৈ ব্যৱহাৰ কৰা হয়।

আপুনি পঠিওৱা প্ৰতিটো নথিপত্ৰত ৱাটাৰমাৰ্ক প্ৰয়োগ কৰাটো আটাইতকৈ প্ৰভাৱশালী সুৰক্ষা।
এটা দৃশ্যমান আৱৰণ য'ত লিখা থাকে **"XYZ সংস্থালৈ পঠিওৱা — মাৰ্চ ২০২৬ — কেৱল ভাড়াৰ আবেদন"**
নথিপত্ৰখন আন কোনো উদ্দেশ্যৰ বাবে অকামৰ কৰে। যদি ই লিক হয়, আপুনি ঠিক জানিব পাৰে ক'ৰ পৰা আহিছিল।

firemark-এ এইটো সৰল কৰে: এটা কমাণ্ড, যিকোনো চিত্ৰ বা PDF, ১৭টা দৃশ্যমান
শৈলী, সম্পাদনাৰ বিৰুদ্ধে প্ৰতিরোধী ক্ৰিপ্টোগ্ৰাফিক ফিলিগ্ৰেইন আকাৰ, আৰু সম্পূৰ্ণ
ফোল্ডাৰৰ বাবে বেচ প্ৰক্ৰিয়া।

## স্থাপন কৰক

[crates.io](https://crates.io/crates/firemark) ৰ পৰা:


```bash
cargo install firemark
```

উৎসৰ পৰা:

```bash
git clone https://github.com/Vitruves/firemark.git
cd firemark
cargo install --path .
```
এয়ে এটা একক অপ্টিমাইজ কৰা বাইনেৰি (প্ৰায় ৫ এমবি) উত্পাদন কৰে।

## দ্ৰুত আৰম্ভণি


```bash
# Watermark a single image
firemark photo_id.png -m "Flat rental — SCI Dupont — March 2026"

# Watermark a PDF
firemark tax_notice.pdf -m "CONFIDENTIAL" -s "Do not distribute"

# Watermark an entire folder recursively
firemark ./documents/ -R -m "Sent to Agency X" -t stamp

# Preview without writing files
firemark id_card.jpg -m "Draft" -n
```
Outputটো সাধাৰণতে `{name}-watermarked.{ext}` হিচাপে inputৰ লগত সংৰক্ষণ কৰা হয়।
Explicit output path নিৰ্ধাৰণ কৰিবলৈ `-o` ব্যৱহাৰ কৰক, বা custom suffixৰ বাবে `-S`।

## Watermarkৰ ধৰণসমূহ

| Flag | Style | Description |
|---|---|---|
| `diagonal` | Diagonal grid | সম্পূর্ণ পৃষ্ঠাত পুনৰাবৃত্তি diagonal লিখা (default) |
| `stamp` | Rubber stamp | ডাবল বর্ডাৰ থকা ডাঙৰ কেন্দ্ৰিত stamp |
| `stencil` | Stencil | সম্পূর্ণ-প্ৰস্থ মিলিটাৰী stencil অক্ষৰ |
| `typewriter` | Typewriter | Monospaced typewriter লিখা |
| `handwritten` | Signature | Underline সহ হস্তলিখিত signature |
| `redacted` | Redaction | সম্পূর্ণ-প্ৰস্থ ক’লা redaction bar |
| `badge` | Shield | Security shield/badge emblem |
| `ribbon` | Ribbon | Diagonal কোণা ribbon banner |
| `seal` | Seal | Circular notary-style seal |
| `frame` | Frame | সম্পূর্ণ পৃষ্ঠাত decorative border |
| `tile` | Tile | ঘন uniform লিখাৰ grid |
| `mosaic` | Mosaic | Randomised scattered লিখা |
| `weave` | Weave | Interlocking diagonal weave |
| `ghost` | Ghost | অত্যন্ত সূক্ষ্ম embossed লিখা |
| `watercolor` | Watercolour | Soft blurred wash effect |
| `noise` | Noise | Distressed লিখা আৰু pixel noise |
| `halftone` | Halftone | লিখা halftone dot grid হিচাপে |


```bash
firemark doc.pdf -t stamp -m "CONFIDENTIAL" --border --color red
```

## সুৰক্ষা ফিলিগ্ৰেন

firemark ক্রিপ্টোগ্ৰাফিক ফিলিগ্ৰেন পেটাৰ্নসমূহক ব্যাঙ্কনোটৰ
সুৰক্ষা বৈশিষ্ট্যৰ পৰা অনুপ্ৰাণিতকৈ ওপৰত থয়। এই সূক্ষ্ম জ্যামিতিক পেটাৰ্নসমূহ
ছবিৰ সম্পাদনাত অত্যন্ত কঠিনভাৱে আঁতৰাব পৰা যায়।

| শৈলী | বিৱৰণ |
|---|---|
| `guilloche` | সিনুসয়ডেল ৱেভ এনভেল'প বেংড (ডিফল্ট) |
| `rosette` | স্পাইরোগ্ৰাফ + কোণৰ ৰোজ কাৰ্ভ |
| `crosshatch` | সূক্ষ্ম কাষে কাষে হীৰাৰ জাল |
| `border` | ঢৌবিশিষ্ট নেষ্টেড সুৰক্ষা সীমা |
| `lissajous` | পেৰামেট্ৰিক লিসাজউ ফিগাৰ |
| `moire` | কেন্দ্ৰীয় বৃত্তৰ ইণ্টাৰফেৰেন্স |
| `spiral` | আর্কিমিডিয়ান স্পাইৰেল ভৰ্টেক্স |
| `mesh` | ষ্টagonal হানিকম্ব গ্ৰীড |
| `plume` | পৃষ্ঠত ছটিয়াই থকা ফেদাৰ-সদৃশ ক্ৰিভসমূহ |
| `constellation` | তৰা নোডসমূহ সূক্ষ্ম জ্যামিতিক ৱেবে সংযুক্ত |
| `ripple` | অনিয়মিত উৎসৰ পৰা ওভাৰলেপিং এলিপ্টিকেল ৱেভ ফ্ৰণ্ট |
| `full` | সকলো পেটাৰ্ন একত্ৰিত |
| `none` | ফিলিগ্ৰেন বন্ধ কৰক |

```bash
firemark id.png -m "Rental application" --filigrane moire
firemark id.png -m "Rental application" --filigrane none   # disable
```

## AI-অপসাৰণ শক্তিশালীকৰণ

প্ৰত্যেকখন ৰেণ্ডাৰ ডিফ’ল্ট ৰূপে অ-নির্ধাৰিত। firemark-এ বিশ্বজনীন
পোস্ট-ৰেণ্ডাৰ বিঘ্ন (এলফা জিটাৰ, ছাব-পিক্সেল ৰংৰ শব্দ, প্ৰান্তৰ ক্ষুদ্ৰ বিন্দু,
বিৰল ভূত পিক্সেল) আৰু প্ৰতিটো ৰেণ্ডাৰৰ বাবে ৰেণ্ডাৰকাৰী-ভিত্তিক ৰেণ্ডমাইজেচন প্ৰয়োগ কৰে যাতে একে ছেটিংছৰ লগতেও দুইটা আউটপুট কেতিয়াও
পিক্সেল-একেই নোহোৱা হয়। এইটো AI
ভিছন মডেলসমূহে বিয়োগ কৰাৰ বাবে পূৰ্বানুমানযোগ্য পেটাৰ্ন শিকিবলৈ অক্ষম কৰে।

ইয়াৰ উপৰিও, ডিফ’ল্ট ৰূপে প্ৰতিদ্বন্দ্বী প্ৰম্প্ট-ইঞ্জেকশ্যন ষ্ট্ৰিপসমূহ এম্বেড কৰা হয়
AI ৱাটাৰমাৰ্ক অপসাৰণ সঁজুলিক বিভ্ৰান্ত কৰিবলৈ। আপুনি দৃশ্যমান প্ৰম্প্ট টেক্সট নোচাহিলে `--no-anti-ai` ব্যৱহাৰ কৰি
নিষ্ক্ৰিয় কৰিব পাৰে:

```bash
firemark doc.png -m "CONFIDENTIAL" --no-anti-ai
```

## সাধাৰণ বিকল্পসমূহ

```
-m, --main-text           Primary watermark text
-s, --secondary-text      Secondary text line
-t, --type                Watermark style (see table above)
-o, --output              Output file path
-S, --suffix              Custom output suffix (default: "watermarked")
-c, --color               Color — name or #RRGGBB (default: blue)
-O, --opacity             Opacity 0.0–1.0 (default: 0.5)
-r, --rotation            Angle in degrees (default: -45)
-p, --position            center, top-left, top-right, bottom-left, bottom-right, tile
-f, --font                Font name or path to .ttf/.otf
-I, --image               Overlay an image as watermark
    --qr-data             Embed a QR code with custom data
    --qr-code-position    QR code placement (default: center)
    --qr-code-size        QR code size in pixels (default: auto)
    --border              Draw a border around the watermark
    --shadow              Add a drop shadow
    --filigrane           Security filigrane style (default: guilloche)
    --no-anti-ai          Disable adversarial prompt injection (on by default)
```

৭০+ পতাকাৰ সম্পূৰ্ণ তালিকাৰ বাবে, চাওক [`CLI.md`](https://raw.githubusercontent.com/Vitruves/firemark/main/CLI.md)।

## PDF বিকল্পসমূহ

```
    --pages           Pages to watermark (e.g. 1,3-5 or "all")
    --skip-pages      Pages to skip
    --behind          Place watermark behind content
    --no-flatten      Keep layers separate (flattened by default)
    --dpi             Render resolution (default: 150)
```

## বেটচ প্ৰচেছিং

```bash
# Process all images and PDFs in a folder
firemark ./inbox/ -m "INTERNAL" -t tile

# Recursive, 8 threads, custom suffix
firemark ./docs/ -R -j 8 -m "Draft" -S draft

# Dry run — list what would be processed
firemark ./docs/ -R -m "Draft" -n
```

পূৰ্বতে ৱাটাৰমাৰ্ক কৰা ফাইলসমূহ (যিসকলৰ ছাফিক্স মিলিছে) পুনৰ চলোৱাৰ সময়ত স্বয়ংক্ৰিয়ভাৱে এৰাই দিয়া হয়।

## সংৰূপ ফাইল

আপোনাৰ বিকল্পসমূহ এটা TOML ফাইলত সংৰক্ষণ কৰক যাতে পতাকা পুনৰাবৃত্তি নকৰে। চাওক
[`examples/config/firemark.toml`](https://raw.githubusercontent.com/Vitruves/firemark/main/examples/config/firemark.toml) সম্পূৰ্ণ
উদাহৰণৰ বাবে, য'ত দুটা প্ৰিসেট আছে: **অতি-নিরাপদ** (ঘন টাইলিং, সম্পূৰ্ণ ফিলিগ্ৰেন, QR
অনুসৰণযোগ্যতা, মেটাডাটা আঁতৰোৱা) আৰু **হালকা** (সহজ তিৰ্যক পাঠ, অতিৰিক্ত কিছু নাই)।


```toml
# Global defaults
main_text = "CONFIDENTIAL"
secondary_text = "{author} — {date}"
watermark_type = "diagonal"
color = "#1a3c6e"
opacity = 0.45
font_weight = "bold"
filigrane = "guilloche"
border = true

[preset.ultra-secure]
main_text = "CONFIDENTIAL — {author}"
watermark_type = "tile"
color = "#CC0000"
opacity = 0.6
filigrane = "full"
anti_ai = true
qr_data = "firemark://{author}/{timestamp}/{uuid}"
strip_metadata = true

[preset.light]
main_text = "COPY"
watermark_type = "diagonal"
color = "#555555"
opacity = 0.3
filigrane = "none"
anti_ai = false
```

```bash
firemark doc.pdf --config firemark.toml
firemark doc.pdf --config firemark.toml --preset ultra-secure
firemark doc.pdf --config firemark.toml --preset light
firemark doc.pdf --save-preset mypreset    # save current flags
firemark --list-presets                     # list available presets
```
