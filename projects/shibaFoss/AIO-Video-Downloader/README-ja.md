## 🚀 使い方

1. **ダウンロード**: 任意の動画URLをコピーするか内蔵ブラウザを使用
2. **取得**: AIOが自動でダウンロード可能なコンテンツを検出
3. **選択**: お好みの品質とフォーマットを選択
4. **ダウンロード**: 超高速ダウンロードをお楽しみください
5. **再生**: 内蔵プレーヤーで再生、またはプライベートストレージへ移動
