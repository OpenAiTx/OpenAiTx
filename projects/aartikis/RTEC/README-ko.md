# RTEC: 런타임 이벤트 계산법

RTEC는 데이터 스트림 추론에 최적화된 오픈 소스 [이벤트 계산법](https://en.wikipedia.org/wiki/Event_calculus) 방언입니다. Prolog로 작성되었으며 Linux, MacOS, Windows의 [SWI-Prolog](https://www.swi-prolog.org/)에서 테스트되었습니다.

# 라이선스

RTEC는 전혀 보증하지 않습니다. 이 소프트웨어는 자유 소프트웨어이며 특정 조건 하에 재배포할 수 있습니다; 자세한 내용은 [GNU 약소 일반 공중 사용 허가서 v3](http://www.gnu.org/licenses/lgpl-3.0.html)를 참조하십시오.

# 관련 소프트웨어
- [iRTEC](https://github.com/eftsilio/Incremental_RTEC): 증분 RTEC. iRTEC는 증분 추론을 지원하며 데이터 스트림의 지연과 철회를 효율적으로 처리합니다.
- [oPIEC](https://github.com/Periklismant/oPIEC): 온라인 확률적 구간 기반 이벤트 계산법. oPIEC는 불확실성 하의 데이터 스트림에 대한 이벤트 계산법 추론을 지원합니다.
- [Wayeb](https://github.com/ElAlev/Wayeb): Wayeb는 Scala로 작성된 복합 이벤트 처리 및 예측(CEP/F) 엔진입니다. 이는 기호 자동자와 마르코프 모델을 기반으로 합니다.
