# RTEC : Calcul des Événements en Temps Réel

RTEC est un dialecte open-source du [Calcul des Événements](https://fr.wikipedia.org/wiki/Calcul_des_%C3%A9v%C3%A9nements) optimisé pour le raisonnement sur les flux de données. Il est écrit en Prolog et a été testé sous [SWI-Prolog](https://www.swi-prolog.org/) sur Linux, MacOS et Windows.

# Logiciels Connexes
- [iRTEC](https://github.com/eftsilio/Incremental_RTEC) : RTEC Incrémental. iRTEC supporte le raisonnement incrémental, gérant efficacement les délais et rétractations dans les flux de données.
- [oPIEC](https://github.com/Periklismant/oPIEC) : Calcul des Événements en Ligne Probabiliste Basé sur les Intervalles. oPIEC supporte le raisonnement du Calcul des Événements sur des flux de données incertains.
- [Wayeb](https://github.com/ElAlev/Wayeb) : Wayeb est un moteur de Traitement Complexe d'Événements et de Prévision (CEP/F) écrit en Scala. Il est basé sur des automates symboliques et des modèles de Markov.
