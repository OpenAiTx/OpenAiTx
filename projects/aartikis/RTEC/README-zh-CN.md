# RTEC: 运行时事件演算

RTEC 是一个开源的[事件演算](https://en.wikipedia.org/wiki/Event_calculus)方言，针对数据流推理进行了优化。它用 Prolog 编写，已在 Linux、MacOS 和 Windows 下的[SWI-Prolog](https://www.swi-prolog.org/)环境中测试。

# 相关软件
- [iRTEC](https://github.com/eftsilio/Incremental_RTEC)：增量式 RTEC。iRTEC 支持增量推理，有效处理数据流中的延迟和撤回。
- [oPIEC](https://github.com/Periklismant/oPIEC)：在线概率区间事件演算。oPIEC 支持在不确定性条件下对数据流进行事件演算推理。
- [Wayeb](https://github.com/ElAlev/Wayeb)：Wayeb 是用 Scala 编写的复杂事件处理与预测（CEP/F）引擎。基于符号自动机和马尔可夫模型。
