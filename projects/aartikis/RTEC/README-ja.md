# RTEC: ランタイムイベント計算

RTECはデータストリーム推論に最適化されたオープンソースの[イベント計算](https://en.wikipedia.org/wiki/Event_calculus)方言です。Prologで書かれており、Linux、MacOS、Windows上の[SWI-Prolog](https://www.swi-prolog.org/)で動作確認されています。

# 関連ソフトウェア
- [iRTEC](https://github.com/eftsilio/Incremental_RTEC): インクリメンタルRTEC。iRTECはインクリメンタル推論をサポートし、データストリームにおける遅延や取り消しを効率的に処理します。
- [oPIEC](https://github.com/Periklismant/oPIEC): オンライン確率的区間ベースイベント計算。oPIECは不確実性の下でのデータストリームに対するイベント計算推論をサポートします。
- [Wayeb](https://github.com/ElAlev/Wayeb): WayebはScalaで書かれた複雑イベント処理および予測（CEP/F）エンジンです。これは記号的オートマトンとマルコフモデルに基づいています。
