# ExtremeCopy
![](https://raw.githubusercontent.com/kevinwu1024/ExtremeCopy/main/ExetremeCopyUI.jpg)

ExtremeCopy est un utilitaire Windows qui copie et déplace les fichiers extrêmement rapidement, il optimise automatiquement la vitesse et les ressources en fonction de la machine physique cible.

**ExtremeCopy inclut les fonctionnalités suivantes :**

   * Copier ou déplacer des fichiers ou dossiers très rapidement.
   * Augmenter la vitesse de copie des fichiers de 20 % à 120 % par rapport à la fonction de copie Windows par défaut.
   * Gagner beaucoup de temps lors de la sauvegarde de gros fichiers vers un autre disque dur ou une clé USB.
   * Gérer facilement les tâches de copie.
   * Capable de calculer et d'afficher exactement le temps restant.
   * Prise en charge du point de rupture permettant de suspendre en cas de problème, comme un fichier verrouillé, et de reprendre en cliquant sur le bouton « Continuer » une fois le problème résolu.
   * Supporte plusieurs langues.

Pour des performances élevées, tout le code est écrit en C++ et utilise directement l’API Win32.

ExtremeCopy utilise l’email comme moyen de rapport de bugs pour envoyer le fichier de vidage, configurez votre compte email dans **BugReport.cpp** si vous souhaitez l’utiliser.

Pour la version payante héritée, il existe un projet **RegisterCodeGenerator** qui peut générer un code d’enregistrement pour celle-ci.
