## あなたが期待するすべて

### それはただのMarkdownです
ドキュメントの内容に集中し、数分でプロフェッショナルな静的サイトを作成しましょう。HTML、CSS、JavaScriptの知識は不要です – Material for MkDocsがその重労働を代わりに行います。

### すべてのデバイスで動作

自信を持ってドキュメントを公開 – Material for MkDocsは利用可能な画面領域に完璧に適応し、デバイスの種類やサイズに関係なく表示します。デスクトップ、タブレット、モバイル、すべて素晴らしい対応です。

### オーダーメイド

自分仕様にカスタマイズ – 数行の設定で色、フォント、言語、アイコン、ロゴなどを変更可能。Material for MkDocsは簡単に拡張でき、見た目や動作を変更する多くのオプションを提供します。

### 高速かつ軽量

ユーザーを待たせない – 優れたパフォーマンスを持つ最速クラスのテーマの一つを使い、小さなフットプリントで驚異的な価値を実現。最適な検索エンジンランキングとリピーターの満足度をもたらします。

### 所有権を維持

ドキュメントの完全なソースと出力を所有し、整合性とセキュリティを保証 – 製品知識の基盤を第三者プラットフォームに委ねる必要はありません。完全なコントロールを保持できます。

### オープンソース

信頼できる仲間がいます – 先進のオープンソース技術で構築され、成熟し積極的にメンテナンスされているソリューションを選択。50,000を超える個人と組織に信頼されています。MITライセンスの下で提供。

## クイックスタート

Material for MkDocsは`pip`でインストール可能です：














``` sh
pip install mkdocs-material
```

`mkdocs.yml` に次の行を追加してください：

``` yaml
theme:
  name: material
```
詳細なインストール手順、設定オプション、およびデモについては、
[squidfunk.github.io/mkdocs-material][Material for MkDocs] をご覧ください。
