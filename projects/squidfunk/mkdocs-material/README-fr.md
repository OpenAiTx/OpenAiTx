## Tout ce à quoi vous vous attendez

### C'est juste du Markdown

Concentrez-vous sur le contenu de votre documentation et créez un site statique professionnel
en quelques minutes. Pas besoin de connaître HTML, CSS ou JavaScript – laissez Material for MkDocs faire
le travail lourd pour vous.

### Fonctionne sur tous les appareils

Diffusez votre documentation en toute confiance – Material for MkDocs s’adapte automatiquement
pour s’ajuster parfaitement à l’espace écran disponible, quel que soit le type ou la taille
de l’appareil de visualisation. Bureau. Tablette. Mobile. Tout est parfait.

### Sur mesure

Faites-le vôtre – changez les couleurs, polices, langue, icônes, logo, et plus encore avec
quelques lignes de configuration. Material for MkDocs peut être facilement étendu et
offre de nombreuses options pour modifier l’apparence et le comportement.

### Rapide et léger

Ne faites pas attendre vos utilisateurs – obtenez une valeur incroyable avec une faible empreinte
en utilisant l’un des thèmes les plus rapides disponibles avec d’excellentes performances,
offrant un classement optimal dans les moteurs de recherche et des utilisateurs satisfaits qui reviennent.

### Conservez la propriété

Possédez les sources complètes et les résultats de votre documentation, garantissant à la fois
l’intégrité et la sécurité – pas besoin de confier la colonne vertébrale de vos connaissances produit
à des plateformes tierces. Gardez un contrôle total.

### Open Source

Vous êtes en bonne compagnie – choisissez une solution mature et activement maintenue, construite
avec des technologies Open Source de pointe, utilisée par plus de 50 000
individus et organisations. Sous licence MIT.

## Démarrage rapide

Material for MkDocs peut être installé avec `pip` :

``` sh
pip install mkdocs-material
```

Ajoutez les lignes suivantes à `mkdocs.yml` :

``` yaml
theme:
  name: material
```
Pour des instructions d'installation détaillées, des options de configuration et une démo, visitez  
[squidfunk.github.io/mkdocs-material][Material for MkDocs]

  [Material for MkDocs]: https://squidfunk.github.io/mkdocs-material/
