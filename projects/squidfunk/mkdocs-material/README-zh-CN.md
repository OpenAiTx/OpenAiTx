## 你所期望的一切

### 它只是 Markdown
专注于您的文档内容，几分钟内创建一个专业的静态站点。无需了解 HTML、CSS 或 JavaScript —— 让 Material for MkDocs 为您承担繁重的工作。

### 适用于所有设备

自信地发布您的文档 —— Material for MkDocs 会自动适配，完美契合任何类型或尺寸的屏幕空间。桌面。平板。手机。全都支持。

### 量身定制

让它属于您 —— 只需几行配置即可更改颜色、字体、语言、图标、徽标等。Material for MkDocs 易于扩展，提供多种选项以改变外观和行为。

### 快速且轻量

不要让用户等待 —— 使用性能卓越的快速主题之一，以小巧的体积获得惊人的价值，实现最佳搜索引擎排名和让用户满意地回访。

### 保持所有权

拥有您的文档完整的源文件和输出，保证完整性和安全性 —— 无需将产品知识的核心托付给第三方平台。保持完全控制。

### 开源项目

您选择了一个优秀的伙伴 —— 选择一个成熟且积极维护的解决方案，基于最先进的开源技术，受到超过 50,000 个个人和组织的信赖。采用 MIT 许可证。

## 快速开始

Material for MkDocs 可通过 `pip` 安装：














``` sh
pip install mkdocs-material
```

将以下内容添加到 `mkdocs.yml`：

``` yaml
theme:
  name: material
```

有关详细的安装说明、配置选项和演示，请访问
