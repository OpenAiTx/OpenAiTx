## Introducción

> [!IMPORTANTE]
> ¡Este proyecto se está actualizando continuamente!

Este proyecto proporciona muchas implementaciones de técnicas de malware usando Zig, ya que soy un gran fan de este lenguaje. Puedes usar este repositorio para armar Zig. Black-Hat-Zig se actualiza de forma continua para asegurarse de que contiene tanto contenido como sea posible. Será perfecto si quieres crear un PR para este proyecto.

¡Bien, vamos a hackear el planeta!

## ¿Por qué Zig?

- 🤝 Fácil de interactuar con código fuente C/C++
- 🔎 Es más nuevo, por lo que es más difícil de detectar
- 💪 Control de bajo nivel muy fuerte, incluso más bajo que C
- 😱 Es más difícil de revertir debido al compilador moderno
- 🛡️ Dependencias mínimas en tiempo de ejecución - Sin una enorme huella de biblioteca estándar
- 🎯 Sin comportamiento indefinido - El manejo explícito de casos límite previene caídas
