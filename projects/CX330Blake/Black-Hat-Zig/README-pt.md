## Introdução

> [!IMPORTANTE]
> Este projeto está sendo continuamente atualizado!

Este projeto fornece muitas implementações de técnicas de malware utilizando Zig, já que sou um grande fã da linguagem. Você pode usar este repositório para armar o Zig. Black-Hat-Zig está sendo continuamente atualizado para garantir que contenha o máximo de conteúdo possível. Será perfeito se você quiser criar um PR para este projeto.

Ok, vamos hackear o planeta!

## Por que Zig?

- 🤝 Fácil de interagir com código-fonte C/C++
- 🔎 É mais novo, então é mais difícil de ser detectado
- 💪 Forte controle de baixo nível, ainda mais que C
- 😱 É mais difícil de fazer engenharia reversa por causa do compilador moderno
- 🛡️ Dependências mínimas em tempo de execução - Sem grande pegada da biblioteca padrão
- 🎯 Sem comportamento indefinido - Tratamento explícito de casos extremos evita falhas
