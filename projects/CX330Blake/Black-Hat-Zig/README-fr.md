## Intro

> [!IMPORTANT]
> Ce projet est continuellement mis à jour !

Ce projet propose de nombreuses implémentations de techniques de malware en utilisant Zig, car j'en suis un grand fan. Vous pouvez utiliser ce dépôt pour armer Zig. Black-Hat-Zig est mis à jour en continu pour s'assurer qu'il contient le plus de contenu possible. Ce serait parfait si vous souhaitez créer une PR pour ce projet.

Allez, piratons la planète !

## Pourquoi Zig ?

- 🤝 Facile d’interagir avec du code source C/C++
- 🔎 C’est plus récent, donc plus difficile à détecter
- 💪 Contrôle bas niveau puissant, encore plus que le C
- 😱 Plus difficile à rétroconcevoir à cause du compilateur moderne
- 🛡️ Dépendances d’exécution minimales - Pas d’empreinte massive de bibliothèque standard
- 🎯 Aucun comportement indéfini - La gestion explicite des cas limites évite les crashs
