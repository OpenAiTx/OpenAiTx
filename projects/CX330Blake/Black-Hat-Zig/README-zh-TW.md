## 簡介

> [!IMPORTANT]
> 本專案持續更新中！

本專案提供許多使用 Zig 實現的惡意軟體技術，因為我非常熱愛 Zig。你可以利用本倉庫將 Zig 武器化。Black-Hat-Zig 會持續更新，盡可能納入更多內容。如果你願意為這個專案提交 PR 那就太完美了。

好了，一起來 hack the planet 吧！

## 為什麼選擇 Zig？

- 🤝 容易與 C/C++ 原始碼互動
- 🔎 它較新，因此更難被偵測
- 💪 超低層級控制，甚至比 C 還低
- 😱 由於現代編譯器，使逆向工程更加困難
- 🛡️ 最小化運行時依賴——沒有龐大的標準函式庫負擔
- 🎯 無未定義行為——明確處理邊界情況以避免崩潰
