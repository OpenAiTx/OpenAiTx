## ما هو CORE

CORE هو الطبقة التشغيلية مفتوحة المصدر للعمل المعتمد على الذكاء الاصطناعي.

أنت تكتب المطلوب في مساحة العمل. CORE تلتقط المهمة، وتحمل السياق من الذاكرة والتطبيقات المتصلة، وتضع خطة، وتدير الوكيل المناسب عبر البوابة، وتعالج العقبات عندما تستطيع، وتعود عند الحاجة للحكم البشري.

ليست روبوت دردشة تواصل تحفيزه. إنها الطبقة التي تتذكر، وتنسق، وتتصرف، وتتصاعد.

### الهندسة المعمارية

| | |
|---|---|
| **يراقب** | السياق من محادثات الذكاء الاصطناعي عبر MCP، وأكثر من 50 تطبيقاً متصلاً، وعلى ماك، أي تطبيق تمنحه الوصول صراحة. |
| **يتذكر** | رسم بياني معرفي يتتبع ليس فقط ما قلت، بل ما قررت، ومتى ولماذا، عبر كل أداة ومحادثة. |
| **يتصرف** | يتخذ إجراءات مباشرة في تطبيقاتك المتصلة، ويفوض الأعمال الطويلة لوكلاء البرمجة والتصفح عبر البوابة. |
| **السياسات** | تدفقات الموافقة، قواعد التصعيد، الخطط، وسجلات التدقيق حتى تظل الاستقلالية خاضعة للمساءلة. |

CORE يقرر ما يمكنه فعله بأمان، يطلب الإذن قبل الإجراءات الحساسة، ويترك أثرًا يمكنك مراجعته.

## CORE أثناء العمل

### فوّض مهمة برمجية، وارجع إلى طلب دمج (PR).

أخبر CORE بما يجب فعله. يجمع السياق من مستودعك، وتطبيقاتك، وذاكرته، ويعد خطة، ويشغل جلسة Claude Code أو Codex، ويتعامل مع العوائق حيثما أمكن، ويعود بطلب دمج (PR). تراجع العمل عند الانتهاء.

`[ ] أصلح حالة التنافس في تدفق الدفع من المشكلة #312`

### نظف قائمة المهام المتراكمة أثناء نومك.

حدد مهمة متكررة لسحب المهام من قائمتك في وقت محدد. يعمل CORE عليها وأنت غير متصل. العمليات السلسة تنتظر المراجعة في الصباح. الجلسات العالقة تعود بسؤال محدد، وليس بعلامة تبويب متوقفة.

`[ ] نفذ المهام المتراكمة الليلة ابتداءً من الساعة 11 مساءً`

### تحقق من التنبيهات قبل أن تتحول إلى مقاطعات.

حدد مهمة متكررة لمراقبة Sentry أو السجلات أو أي مصدر تنبيهات. عندما يحدث تنبيه، يحقق CORE، ويسحب الآثار والحوادث السابقة ذات الصلة، ويقرر ما إذا كان سيتعامل معها أو يصعّدها.

يصدر تنبيه من Sentry الساعة 2 صباحًا. يحقق CORE، يقترح إصلاحًا، ويراسلك على Slack للمراجعة.

### احصل على ملخص صباحي يعرف عملك.

حدد مهمة متكررة لجمع البيانات من البريد الإلكتروني، GitHub، Linear، وSlack كل صباح. يلخص CORE ما يحتاج انتباهك، ويتجاوز غير المهم، ويحول المتابعات إلى مهام تلقائيًا.

### فوّض من أي مكان كنت فيه.

أنشئ مهامًا من Slack، WhatsApp، Telegram، البريد الإلكتروني، أو الويب. البوابة تعمل دائمًا عبر Docker أو Railway، لذا يمكن لـ CORE التقاط العمل حتى عندما يكون حاسوبك المحمول مغلقًا.

## ما بداخل CORE

| | |
|---|---|
| **الذاكرة** | تتبع تفضيلاتك وقراراتك وأهدافك وتوجيهاتك عبر كل أداة وكل محادثة، بحيث يبدأ كل مهمة مع تحميل السياق. |
| **المهام** | وحدات عمل لمرة واحدة أو متكررة وفق مواصفاتك، خطة CORE، الحالة الحية، وخيط دردشة مخصص. يمكن لكل مهمة أن تنتج جلسات ترميز أو تصفح. |
| **المفكرة** | صفحة يومية للمهام والأفكار والعمل الجاري. اكتب `[ ]` في أي مكان وسيقوم CORE بالتقاطها خلال 3 دقائق. |
| **الموصلات** | أكثر من 50 تطبيقاً عبر نقطة نهاية MCP واحدة، بالإضافة إلى مشغلات webhook للأتمتة الاستباقية. GitHub، Linear، Jira، Slack، Gmail، Calendar، Sentry، Granola، Todoist، وغيرها. |
| **المهارات** | أكثر من 100 تعليمات قابلة لإعادة الاستخدام يتم تطبيقها تلقائياً بناءً على السياق. استخدم المهارات المدمجة أو اكتب مهاراتك الخاصة لتكرار سير العمل. |
| **البوابة** | تقوم بتشغيل Claude Code، Codex، وكلاء المتصفح، والأوامر المحلية على جهازك أو في Docker / Railway، بحيث يواصل CORE العمل حتى عند إغلاق اللابتوب الخاص بك. |
| **محايد للنماذج** | استخدم مزودك الخاص: Anthropic، OpenAI، أو نماذج مفتوحة المصدر. استضف المجموعة الكاملة ذاتياً للعزل. |

## ما ليس عليه CORE

| | |
|---|---|
| **ليس مجرد غلاف RAG.** | الذاكرة ليست مجرد أجزاء مضمنة. إنها رسم بياني معرفي يتتبع ما قررت، ومتى، ولماذا. |
| **ليس منشئ سير عمل.** | لا يوجد سحب وإفلات DAGs. أنت تكتب ما يجب فعله. CORE يكتشف سير العمل ويسأل عندما يحتاج إلى حكم. |
| **ليس نسخة أخرى من Devin.** | CORE يقترح الخطط، وأنت توافق. CORE يطلب فك الحظر، وأنت تقرر. CORE يعيد PRs، وأنت تراجع. الوكلاء لا يدمجون بأنفسهم. |
| **ليس مساعد سحابي مغلق.** | CORE مفتوح المصدر، قابل للاستضافة الذاتية، محايد للنماذج، ومصمم حول بنيتك التحتية. |

## بدء سريع

مفتوح المصدر وقابل للاستضافة الذاتية. بياناتك تبقى في بنيتك التحتية.

**قم بتثبيت وبدء CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

يطلب معالج الإعداد دليل التثبيت، مزود الذكاء الاصطناعي، مفتاح API، ونموذج الدردشة. يقوم بإنشاء الأسرار، ويشغل الحزمة، ويفتح `http://localhost:3033`.

تستغرق معظم عمليات التثبيت المحلية بضع دقائق بمجرد تشغيل Docker.

**أو قم بالنشر على Railway:**

[![النشر على Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**اتصل ببوابة** حتى يتمكن CORE من تشغيل متصفحك، وتشغيل وكلاء البرمجة، والوصول إلى المجلدات المحلية:

```bash
corebrain login
corebrain gateway setup
```

**المتطلبات:** Docker 20.10+، Docker Compose 2.20+، 4 vCPU / 8GB RAM

[دليل الاستضافة الذاتية الكامل](https://docs.getcore.me/self-hosting/setup)

> هل تريد تطبيق ماك؟ انضم إلى قائمة الانتظار على [getcore.me](https://www.getcore.me/).

## ما نؤمن به

- الدردشة هي واجهة. ليست نظام تشغيل.
- الذكاء بلا ذاكرة مجرد معلومات سطحية.
- يجب أن يعرفك الذكاء الاصطناعي عبر كل أداة، وليس فقط التبويب الحالي.
- يجب أن ينتقل العمل من النية إلى التنفيذ دون أن تكون أنت الوسيط.
- الأتمتة دون تحمل المسؤولية فوضى.
