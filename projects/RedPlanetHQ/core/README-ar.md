## تراقب دائمًا. جاهزة دائمًا.

معظم أدوات الذكاء الاصطناعي تنتظر أن تُطلب منها الأوامر. CORE تراقب.

اربطها بتطبيقاتك وستراقب النشاط عبر جميعها. يصل بريد إلكتروني من عميل، يتم تعيين قضية في GitHub، يصدر تنبيه من Sentry، ينتهي اجتماع. CORE تلاحظ ذلك، تراجع ذاكرتك والمهارات التي قمت بتعيينها، إما تتعامل معه أو تعرضه لحكمك. أنت لا تقوم بتشغيلها. هي تلاحظ بنفسها.

قم بتثبيت إضافة CORE في Claude Code أو Codex أو Cursor وستتم مراقبة محادثات وكيلك أيضًا. السياق الذي نوقش، القرارات المتخذة، الكود المكتوب، كل ذلك يُغذي رسم معرفة الذاكرة. المهمة التالية التي تلتقطها CORE تعرف بالفعل ما حدث في جلستك الأخيرة.

عندما تتصرف، تتصرف مباشرة. يمكنها الرد على الرسائل الإلكترونية، تحديث قضايا Linear، تقديم طلبات دمج GitHub، إرسال رسائل Slack، تنفيذ أوامر المحطة الطرفية، تشغيل متصفح، وبدء جلسة Claude Code أو Codex من أي واجهة. أرسل رسالة WhatsApp من المطار ويمكن لـ CORE بدء جلسة ترميز وفتح طلب دمج قبل أن تصعد إلى الطائرة.

أنت من يقرر متى يتصرف بشكل تلقائي ومتى ينتظر توجيهك. عالج تنبيهات Sentry تلقائيًا لكن اسأل دائمًا قبل الدمج. وافق على الخطة قبل بدء جلسة البرمجة. تطلب التأكيد قبل إرسال أي بريد إلكتروني. مستوى الاستقلالية هو من اختيارك، حسب المهمة أو التطبيق أو الإجراء.

## أربع طرق للوصول إليه

الروبوت الحواري له واجهة واحدة. نظام التشغيل له عدة واجهات.

**الصوت.** اضغط Ctrl+Option على الماك وقل ما يجب القيام به. يقوم CORE بتنفيذه في الخلفية دون تعطيل سير عملك.

**المسودة.** افتح صفحتك اليومية واكتب `[ ] إصلاح خطأ التوثيق من المشكلة #47`. يلتقط CORE المهمة خلال 3 دقائق، يحمل السياق من مستودعك وذاكرته، ويضع خطة مبدئية.

**المراسلة.** واتساب، سلاك، تيليجرام. أرسل مهمة من المطار، من هاتفك، أو من السرير. لدى CORE كل السياق الكامل بغض النظر عن مصدر الرسالة.

**الدردشة.** افتح لوحة القيادة وتحدث مباشرة، كما تفعل مع أي مساعد. عندما ترغب في حوار قبل التفويض.

ذكاء اصطناعي واحد، أربع واجهات، نفس الذاكرة والسياق خلفها جميعًا.

## اجعله ملكك

امنحه اسمًا. اختر طريقة حديثه. ضع القواعد التي يتبعها في كل مكان.

اختر من بين خمس شخصيات مدمجة — كفاءة TARS الجافة، رسمية Alfred المخلصة، أو عملية Hudson الدافئة — أو اكتب شخصيتك الخاصة. الشخصية تظهر في كل مهمة ينفذها، كل خطة يصيغها، وكل رسالة يرسلها نيابة عنك.

حدد صوته للتفاعلات المنطوقة. من تلك اللحظة، سيبدو ويتصرف كالذكاء الاصطناعي الذي قمت بإعداده، عبر كل واجهة.

## CORE قيد التنفيذ

### قلها، وارجع إلى طلب الدمج.

*اضغط Ctrl+Option، وتحدث:* "أصلح حالة السباق في سير الدفع من المشكلة #312."

يقوم CORE بتحميل المشكلة، وجلب الالتزامات المتعلقة وسلاسل سلاك، وصياغة خطة، وتشغيل جلسة Claude Code. تعود لتجد فرق الشيفرة. لم تكن على مكتبك أبدًا.

### اكتبها ليلاً، وراجعها صباحًا.

*المذكرة:* `[ ] العمل على تراكم أعمال الليلة بدءًا من الساعة 11 مساءً`

CORE يسحب من Linear وGitHub والذاكرة، ويعطي الأولوية، ويعمل عليها أثناء نومك. الجلسات السلسة تنتظر مراجعتك. الجلسات المتعثرة تعود بسؤال محدد، وليس تبويب متوقف.

### أرسل رسالة إليه من أي مكان.

*واتساب من المطار:* "انشر إعادة هيكلة المصادقة."

CORE يعرف بالفعل الفرع والسياق وتفضيلاتك. يعمل على Railway. يبدأ الجلسة قبل أن تصعد للطائرة.

### تحقق من التنبيهات قبل أن تصبح حوادث.

*Sentry يرسل في الثانية صباحاً.* CORE يحقق، يسحب تتبعات ذات صلة وحوادث سابقة من الذاكرة، يقترح إصلاحاً، وينبهك على Slack: "المشكلة رقم 847، تم اقتراح إصلاح، في انتظار مراجعتك." توافق من هاتفك.

### احصل على ملخص يعرف أسبوعك مسبقًا.

*مهمة متكررة، كل صباح الساعة 8 صباحاً.* CORE يسحب من البريد الإلكتروني وGitHub وLinear وSlack، يعرض ما يحتاج فعلاً الانتباه، ويتجاوز ما لا يحتاج، ويحول المتابعات تلقائياً إلى مهام.

## ما يوجد داخل CORE

| | |
|---|---|
| **الذاكرة** | رسم بياني معرفي زمني عبر كل أداة وكل محادثة. تفضيلات، قرارات، أهداف، وتوجيهات، ليبدأ كل مهمة مع السياق جاهز. |
| **المهام** | وحدات عمل فردية أو متكررة بمواصفاتك وخطة CORE والحالة الحية وسلسلة دردشة مخصصة. كل مهمة يمكن أن تنشئ جلسات برمجة أو متصفح أو طرفية. |
| **الموصلات** | أكثر من 50 تطبيقًا عبر نقطة نهاية MCP واحدة، بالإضافة إلى مشغلات webhook للتشغيل الآلي الاستباقي. GitHub وLinear وJira وSlack وGmail وCalendar وSentry وNotion وTodoist وغيرها. |
| **المهارات** | تعليمات قابلة لإعادة الاستخدام يتم تشغيلها تلقائيًا بناءً على السياق. على سبيل المثال: "دائمًا اسحب مشاكل Linear ذات الصلة قبل التخطيط للإصلاح"، "شغّل الاختبارات قبل فتح PR"، أو "انشر ملخصاً على Slack عند اكتمال المهمة." أكثر من 100 مدمجة، أو اكتب تعليماتك الخاصة. |
| **البوابة** | يشغل Claude Code وCodex ووكلاء المتصفح وأوامر الطرفية على جهازك أو في Docker وRailway، حتى يستمر CORE في العمل عندما يكون حاسوبك مغلقًا. |
| **محايد للنماذج** | استخدم مزودك الخاص: Anthropic أو OpenAI أو نماذج مفتوحة المصدر. استضف المجموعة الكاملة بنفسك لعزل كامل. |

## البداية السريعة

مفتوح المصدر وقابل للاستضافة الذاتية. بياناتك تبقى ضمن بنيتك التحتية.

**اختر مسارك:**

| أريد أن... | كيف |
|---|---|
| تجربته على جهازي | شغل التثبيت بخطوة واحدة أدناه (يتطلب Docker) |
| النشر على خادم أو VPS | نشر بنقرة واحدة عبر Railway |
| استخدام تطبيق ماك | [انضم إلى قائمة الانتظار](https://www.getcore.me/) |

**ثبّت وابدأ CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

يطلب معالج الإعداد تحديد دليل التثبيت، مزود الذكاء الاصطناعي، مفتاح API، ونموذج الدردشة. يقوم بإنشاء بيانات سرية، ويشغل الحزمة، ويفتح `http://localhost:3033`.

تستغرق معظم التثبيتات المحلية عدة دقائق بمجرد تشغيل Docker.

**أو قم بالنشر على Railway:**

[![النشر على Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**قم بتوصيل بوابة** حتى يتمكن CORE من تشغيل وكلاء البرمجة، والتحكم في متصفحك، والوصول إلى المجلدات المحلية:

```bash
corebrain login
corebrain gateway setup
```

**المتطلبات:** Docker 20.10+، Docker Compose 2.20+، معالج رباعي النواة / 8 جيجابايت من الذاكرة العشوائية

[دليل الاستضافة الذاتية الكامل](https://docs.getcore.me/self-hosting/setup)

**مهمتك الأولى (بعد دقيقتين من الإعداد):**

1. افتح **دفتر الملاحظات** (صفحتك اليومية على `http://localhost:3033`)
2. اكتب `[ ] لخص مشكلات GitHub المفتوحة الخاصة بي` أو أي مهمة تقوم بها عادة بنفسك
3. يلتقط CORE المهمة خلال 3 دقائق، ويجمع السياق من التطبيقات المتصلة، ويضع خطة مبدئية
4. وافق على الخطة وسيقوم CORE بتنفيذها وإرجاع النتيجة

[قم بتوصيل تطبيقك الأول](https://docs.getcore.me/connectors)
