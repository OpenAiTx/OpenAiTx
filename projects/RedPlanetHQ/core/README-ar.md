## يراقب دائماً. جاهز دائماً.

معظم أدوات الذكاء الاصطناعي تنتظر أن يُطلب منها العمل. CORE يراقب.

اربطه بتطبيقاتك ليقوم بمراقبة النشاط عبر جميعها. يصل بريد إلكتروني من عميل، يتم تعيين مشكلة في GitHub، يصدر تنبيه من Sentry، ينتهي اجتماع. CORE يلاحظ ذلك، يفحص ذاكرتك والمهارات التي قمت بتحديدها، ويعالج الأمر أو يعرضه ليحكمك عليه. أنت لا تقوم بتشغيله. هو يلاحظ بنفسه.

عند تثبيت إضافة CORE في Claude Code أو Codex أو Cursor، سيتم مراقبة محادثات وكيلك أيضاً. السياق الذي تمت مناقشته، القرارات المتخذة، الكود المكتوب، كل ذلك يغذي رسم معرفة الذاكرة. المهمة التالية التي يلتقطها CORE يعرف بالفعل ما حدث في جلستك الأخيرة.

عندما يتصرف، يتصرف مباشرة. يمكنه الرد على الرسائل الإلكترونية، تحديث مشكلات Linear، إنشاء طلبات سحب في GitHub، إرسال رسائل Slack، تنفيذ أوامر الطرفية، قيادة المتصفح، وبدء جلسة Claude Code أو Codex من أي واجهة. أرسل رسالة WhatsApp من المطار ويمكن لـ CORE بدء جلسة برمجية وفتح طلب سحب قبل صعودك للطائرة.


أنت من يقرر متى يتصرف بشكل تلقائي ومتى ينتظر استدعاءك. عالج تنبيهات Sentry تلقائيًا لكن اسأل دائمًا قبل الدمج. وافق على الخطة قبل بدء جلسة البرمجة. اطلب التأكيد قبل إرسال أي بريد إلكتروني. مستوى الاستقلالية متروك لك لتحدده، حسب المهمة، حسب التطبيق، حسب الإجراء.

## أربع طرق للوصول إليه

للدردشة واجهة واحدة. لنظام التشغيل العديد من الواجهات.

**الصوت.** اضغط Ctrl+Option على ماك وقل ما تريد تنفيذه. يقوم CORE بالتنفيذ في الخلفية دون مقاطعة سير عملك.

**المفكرة السريعة.** افتح صفحتك اليومية واكتب `[ ] إصلاح خطأ المصادقة من القضية #47`. يلتقط CORE ذلك خلال 3 دقائق، ويحمّل السياق من مستودعك وذاكرته، ويعد خطة.

**الرسائل.** واتساب، سلاك، تيليجرام. أرسل مهمة من المطار، من هاتفك، من السرير. CORE لديه كل سياقك بغض النظر من أين جاءت الرسالة.

**الدردشة.** افتح لوحة التحكم وتحدث مباشرة، كأي مساعد. عندما تريد نقاشًا قبل التفويض.

ذكاء اصطناعي واحد، أربع واجهات، نفس الذاكرة والسياق في جميعها.

## اجعله خاصًا بك

امنحه اسمًا. اختر طريقة حديثه. حدد القواعد التي يتبعها في كل مكان.

اختر من بين خمس شخصيات مدمجة — كفاءة TARS الجافة، رسمية Alfred المخلصة، واقعية Hudson الدافئة — أو اكتب شخصيتك الخاصة. تنتقل الشخصية مع كل مهمة ينفذها، كل خطة يعدها، كل رسالة يرسلها نيابة عنك.

اختر صوته للتفاعل الصوتي. من تلك اللحظة، سيبدو ويتصرف مثل الذكاء الاصطناعي الذي قمت بضبطه، عبر كل واجهة.

## CORE قيد التنفيذ

### قلها، وارجع إلى طلب السحب.

*اضغط Ctrl+Option وتكلم:* "أصلح حالة التزامن في تدفق الدفع من القضية #312."

يقوم CORE بتحميل القضية، وسحب الالتزامات والخيوط ذات الصلة من سلاك، ويعد خطة، ويدير جلسة Claude Code. تعود إلى فرق التغييرات. لم تكن على مكتبك أبدًا.

### اكتبها ليلًا، راجعها صباحًا.

*مسودة العمل:* `[ ] معالجة الأعمال المتراكمة الليلة بدءًا من الساعة 11 مساءً`

يقوم CORE بجلب المهام من Linear وGitHub والذاكرة، ويرتبها حسب الأولوية، ويعمل عليها أثناء نومك. الجلسات الناعمة جاهزة لمراجعتك. أما الجلسات العالقة فتعود إليك بسؤال محدد، وليس بعلامة تبويب متوقفة.

### أرسل له رسالة من أي مكان.

*واتساب من المطار:* "أنهِ إعادة هيكلة المصادقة."

CORE يعرف بالفعل الفرع، والسياق، وتفضيلاتك. يعمل على Railway. يبدأ الجلسة قبل أن تصعد إلى الطائرة.

### تحقق من التنبيهات قبل أن تتحول إلى حوادث.

*تنبيه من Sentry في الساعة 2 صباحًا.* يقوم CORE بالتحقيق، وجلب الآثار المرتبطة والحوادث السابقة من الذاكرة، ويقترح حلاً، ويراسلك على Slack: "المشكلة #847، تم اقتراح حل، بانتظار مراجعتك." توافق من هاتفك.

### احصل على ملخص يعرف جدولك الأسبوعي مسبقًا.

*مهمة متكررة، كل صباح في الساعة 8.* يقوم CORE بجلب البيانات من البريد الإلكتروني وGitHub وLinear وSlack، ويبرز ما يحتاج فعلاً إلى انتباهك، ويتجاوز ما لا يحتاج، ويحول المتابعات تلقائيًا إلى مهام.

## ما يوجد داخل CORE

| | |
|---|---|
| **الذاكرة** | رسم بياني معرفي زمني يشمل كل أداة وكل محادثة. التفضيلات والقرارات والأهداف والتوجيهات، حتى يبدأ كل مهمة مع تحميل السياق. |
| **المهام** | وحدات عمل لمرة واحدة أو متكررة مع المواصفات الخاصة بك، وخطة CORE، والحالة الحية، وسلسلة دردشة مخصصة. يمكن لكل مهمة أن تبدأ جلسات ترميز أو متصفح أو طرفية. |
| **الموصلات** | أكثر من 50 تطبيقًا من خلال نقطة نهاية MCP واحدة، بالإضافة إلى محفزات webhook للأتمتة الاستباقية. GitHub وLinear وJira وSlack وGmail وCalendar وSentry وNotion وTodoist والمزيد. |
| **المهارات** | تعليمات قابلة لإعادة الاستخدام تُنفذ تلقائيًا حسب السياق. مثال: "دائمًا اجلب قضايا Linear ذات الصلة قبل التخطيط للإصلاح"، "شغّل الاختبارات قبل فتح PR"، أو "انشر ملخصًا على Slack عند اكتمال المهمة". أكثر من 100 مدمجة، أو يمكنك كتابة تعليماتك الخاصة. |
| **البوابة** | يشغّل Claude Code وCodex وعملاء المتصفح والأوامر الطرفية على جهازك أو في Docker وRailway، بحيث يواصل CORE العمل حتى عند إغلاق حاسوبك المحمول. |
| **محايد للنموذج** | يمكنك استخدام مزود الخدمة الذي تفضله: Anthropic أو OpenAI أو نماذج مفتوحة المصدر. استضف البنية الكاملة ذاتيًا لتحقيق العزل التام. |

## البدء السريع

مفتوح المصدر وقابل للاستضافة الذاتية. تبقى بياناتك ضمن بنيتك التحتية.

**اختر طريقك:**

| أرغب في... | الطريقة |
|---|---|
| تجربته على جهازي | شغل التثبيت بخطوة واحدة أدناه (يتطلب Docker) |
| النشر على خادم أو VPS | النشر بنقرة واحدة عبر Railway |
| استخدام تطبيق ماك | [انضم إلى قائمة الانتظار](https://www.getcore.me/) |

**ثبّت وابدأ CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

يطلب معالج الإعداد تحديد دليل التثبيت، مزود الذكاء الاصطناعي، مفتاح API، ونموذج الدردشة. يقوم بإنشاء بيانات سرية، ويشغل الحزمة، ويفتح `http://localhost:3033`.

تستغرق معظم التثبيتات المحلية عدة دقائق بمجرد تشغيل Docker.

**أو قم بالنشر على Railway:**

[![النشر على Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**قم بتوصيل بوابة** حتى يتمكن CORE من تشغيل وكلاء البرمجة، والتحكم في متصفحك، والوصول إلى المجلدات المحلية:

```bash
corebrain login
corebrain gateway setup
```

**المتطلبات:** Docker 20.10+، Docker Compose 2.20+، معالج رباعي النواة / 8 جيجابايت من الذاكرة العشوائية

[دليل الاستضافة الذاتية الكامل](https://docs.getcore.me/self-hosting/setup)

**مهمتك الأولى (بعد دقيقتين من الإعداد):**

1. افتح **دفتر الملاحظات** (صفحتك اليومية على `http://localhost:3033`)
2. اكتب `[ ] لخص مشكلات GitHub المفتوحة الخاصة بي` أو أي مهمة تقوم بها عادة بنفسك
3. يلتقط CORE المهمة خلال 3 دقائق، ويجمع السياق من التطبيقات المتصلة، ويضع خطة مبدئية
4. وافق على الخطة وسيقوم CORE بتنفيذها وإرجاع النتيجة

[قم بتوصيل تطبيقك الأول](https://docs.getcore.me/connectors)
