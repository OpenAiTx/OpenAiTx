## 🧠 C.O.R.E.

**محرك الملاحظة والاسترجاع السياقي**

CORE هو رسم بياني للذاكرة قابل للنقل تم إنشاؤه من تفاعلاتك مع نماذج اللغة الكبيرة وبياناتك الشخصية، مما يجعل كل السياق وتاريخ سير العمل الخاص بك متاحاً لأي أداة ذكاء اصطناعي، تماماً مثل عقل رقمي. هذا يلغي الحاجة لمشاركة السياق بشكل متكرر. الهدف هو توفير:

- **ذاكرة موحدة وقابلة للنقل**: أضف واسترجع السياق بسلاسة، ووصل ذاكرتك عبر تطبيقات مثل Claude وCursor وWindsurf وغيرها.
- **علاقات، وليست مجرد حقائق مسطحة**: ينظم CORE معرفتك، ويخزن الحقائق والعلاقات بينها لذاكرة أعمق وأغنى مثل الدماغ الحقيقي.
- **ملك المستخدم**: أنت من يقرر ما تحتفظ به أو تحدثه أو تحذفه وتشارك ذاكرتك عبر الأداة التي تريدها وتتحرر من احتكار الموردين.

## 🧩 الميزات الرئيسية

- **رسم بياني للذاكرة**: تصور كيف ترتبط حقائقك وتفضيلاتك معاً
- **الدردشة مع الذاكرة**: اطرح أسئلة حول الذاكرة لتحصل على رؤى وفهم فوري
- **قابل للتوصيل والتشغيل**: استخدم ذاكرة CORE فوراً في تطبيقات مثل Cursor وClaude
