یک چت‌بات نیست که فقط بازش کنید. هوش مصنوعی که همیشه فعال است، همیشه نظاره‌گر است.
به آن نام بدهید. آن را شکل دهید. به هرچیزی که استفاده می‌کنید متصلش کنید. به هر روشی که کار می‌کنید به آن دسترسی داشته باشید.
متن‌باز، قابل میزبانی توسط خودتان، برای همیشه متعلق به شما.

## همیشه ناظر. همیشه آماده.

بیشتر ابزارهای هوش مصنوعی منتظر دستور هستند. CORE همیشه ناظر است.

آن را به برنامه‌های خود متصل کنید تا فعالیت‌ها را در همه آن‌ها زیر نظر بگیرد. یک ایمیل از مشتری می‌رسد، یک issue در گیت‌هاب اختصاص داده می‌شود، یک هشدار Sentry صادر می‌شود، جلسه‌ای به پایان می‌رسد. CORE آن را می‌بیند، حافظه و مهارت‌های تعیین‌شده را بررسی می‌کند و یا خودش رسیدگی می‌کند یا برای تصمیم شما مطرح می‌کند. شما آن را فعال نمی‌کنید؛ خودش متوجه می‌شود.

افزونه CORE را در Claude Code، Codex یا Cursor نصب کنید تا گفتگوهای ایجنت شما هم زیر نظر قرار گیرد. زمینه‌های بحث‌شده، تصمیمات گرفته‌شده، کدهای نوشته‌شده، همه به گراف دانش حافظه تغذیه می‌شوند. وظیفه بعدی که CORE برمی‌دارد، از آنچه در جلسه قبلی اتفاق افتاده مطلع است.

هنگام اقدام، مستقیم عمل می‌کند. می‌تواند به ایمیل‌ها پاسخ دهد، مسائل Linear را به‌روزرسانی کند، PRهای گیت‌هاب ثبت کند، پیام اسلک بفرستد، دستورات ترمینال اجرا کند، مرورگر را کنترل کند و یک نشست Claude Code یا Codex را از هر رابطی راه‌اندازی کند. یک پیام واتساپ از فرودگاه بفرستید و CORE می‌تواند پیش از سوارشدن شما، یک نشست کدنویسی را اجرا و یک PR باز کند.


شما تصمیم می‌گیرید کجا به طور خودکار عمل کند و کجا منتظر دستور شما باشد. هشدارهای Sentry را به طور خودکار مدیریت کند اما همیشه قبل از ادغام از شما اجازه بگیرد. قبل از شروع جلسه کدنویسی یک طرح را تأیید کنید. قبل از ارسال هر ایمیل تأیید لازم است. سطح خودمختاری را می‌توانید برای هر وظیفه، هر اپلیکیشن، هر اقدام تنظیم کنید.

## چهار راه برای دسترسی به آن

یک چت‌بات فقط یک رابط دارد. یک سیستم عامل چندین رابط.

**صوتی.** کلید Ctrl+Option را در مک فشار دهید و آنچه نیاز است را بیان کنید. CORE آن را در پس‌زمینه اجرا می‌کند بدون اینکه روند کار شما را مختل کند.

**دفترچه یادداشت.** صفحه روزانه خود را باز کنید و بنویسید `[ ] رفع اشکال احراز هویت از issue #47`. CORE ظرف ۳ دقیقه آن را دریافت می‌کند، زمینه را از مخزن و حافظه شما بارگذاری می‌کند و یک طرح اولیه تهیه می‌کند.

**پیام‌رسانی.** واتساپ، اسلک، تلگرام. یک وظیفه را از فرودگاه، از تلفن، از رختخواب ارسال کنید. CORE فارغ از اینکه پیام از کجا می‌آید، زمینه کامل شما را دارد.

**چت.** داشبورد را باز کنید و مستقیماً صحبت کنید، مثل یک دستیار معمولی. وقتی می‌خواهید قبل از واگذاری، گفتگو داشته باشید.

یک هوش مصنوعی، چهار سطح، همان حافظه و زمینه پشت همه آن‌ها.

---

## آن را مخصوص خود کنید

برای آن یک نام انتخاب کنید. نحوه صحبت کردنش را تعیین کنید. قوانین مورد نظر را در همه جا تنظیم کنید.

از پنج شخصیت داخلی انتخاب کنید — کارآمدی خشک TARS، رسمیت وفادار Alfred، عمل‌گرایی گرم Hudson — یا شخصیت خودتان را بنویسید. شخصیت در هر وظیفه‌ای که اجرا می‌کند، هر طرحی که تهیه می‌کند، هر پیامی که به نمایندگی شما ارسال می‌کند، جاری می‌شود.

صدای آن را برای تعاملات گفتاری انتخاب کنید. از آن پس، در همه رابط‌ها مانند هوش مصنوعی که تنظیم کرده‌اید، صحبت و رفتار می‌کند.

## CORE در عمل

### بگویید، به PR برگردید.

*کلید Ctrl+Option را فشار دهید، بگویید:* "اشکال race condition در جریان پرداخت از issue #312 را رفع کن."

CORE مسئله را بارگذاری می‌کند، تعهدات و رشته‌های اسلک مرتبط را جمع‌آوری می‌کند، یک طرح اولیه تهیه می‌کند و یک جلسه Claude Code اجرا می‌کند. شما به یک تفاوت برمی‌گردید. هرگز پشت میز نبودید.

### شب بنویسید، صبح مرور کنید.

*یادداشت موقت:* `[ ] امشب از ساعت ۱۱ شب کارهای عقب‌افتاده را بررسی کن`

CORE از Linear، GitHub و حافظه اطلاعات را جمع‌آوری می‌کند، اولویت‌بندی می‌کند و آن‌ها را زمانی که شما خواب هستید انجام می‌دهد. اجراهای روان منتظر بازبینی شما هستند. نشست‌های گیر کرده با یک سوال دقیق بازمی‌گردند، نه یک تب متوقف شده.

### از هر جایی پیام بدهید.

*واتساپ از فرودگاه:* "بازنویسی احراز هویت را منتشر کن."

CORE شاخه، زمینه و ترجیحات شما را از قبل می‌داند. این سیستم در Railway اجرا می‌شود. جلسه را قبل از سوار شدن شما شروع می‌کند.

### هشدارها را قبل از تبدیل به حادثه بررسی کنید.

*سنتری ساعت ۲ صبح هشدار می‌دهد.* CORE بررسی می‌کند، ترِیس‌های مرتبط و حوادث قبلی را از حافظه می‌آورد، راه‌حل پیشنهادی ارائه می‌دهد و در Slack به شما پیام می‌دهد: "مسئله #۸۴۷، راه‌حل پیشنهاد شده، منتظر بازبینی شما." شما از روی گوشی تأیید می‌کنید.

### خلاصه‌ای بگیرید که از هفته شما خبر دارد.

*کار تکرارشونده، هر روز صبح ساعت ۸.* CORE از ایمیل، GitHub، Linear و Slack جمع‌آوری می‌کند، آنچه واقعاً نیاز به توجه دارد را نمایش می‌دهد، آنچه نیاز ندارد را رد می‌کند و پیگیری‌ها را خودکار به وظایف تبدیل می‌کند.

## درون CORE چه خبر است

| | |
|---|---|
| **حافظه** | گراف دانش زمانی بر بستر هر ابزار و مکالمه. ترجیحات، تصمیمات، اهداف و دستورات؛ بنابراین هر کار با زمینه بارگذاری شده آغاز می‌شود. |
| **وظایف** | واحدهای کاری یک‌باره یا تکرارشونده با مشخصات شما، برنامه CORE، وضعیت زنده و یک گفتگوی اختصاصی. هر وظیفه می‌تواند نشست کدنویسی، مرورگر یا ترمینال ایجاد کند. |
| **اتصالات** | بیش از ۵۰ اپلیکیشن از طریق یک نقطه انتهایی MCP، به علاوه تریگرهای webhook برای اتوماسیون پیش‌دستانه. GitHub، Linear، Jira، Slack، Gmail، Calendar، Sentry، Notion، Todoist و غیره. |
| **مهارت‌ها** | دستورالعمل‌های قابل استفاده مجدد که به‌طور خودکار بر اساس زمینه اجرا می‌شوند. مثلا: "همیشه قبل از برنامه‌ریزی رفع مشکل، مسائل مرتبط Linear را جمع‌آوری کن"، "قبل از باز کردن PR تست اجرا کن" یا "وقتی کار تمام شد خلاصه را در Slack ارسال کن." بیش از ۱۰۰ مهارت داخلی یا بنویسید. |
| **دروازه** | Claude Code، Codex، ایجنت مرورگر و دستورات ترمینال را روی دستگاه شما یا در Docker و Railway اجرا می‌کند، بنابراین CORE وقتی لپ‌تاپ بسته است کار را ادامه می‌دهد. |
| **مدل مستقل** | ارائه‌دهنده دلخواه خود را بیاورید: Anthropic، OpenAI یا مدل‌های open-weight. کل پشته را برای ایزوله‌سازی کامل به صورت محلی اجرا کنید. |

## شروع سریع

متن باز و قابل میزبانی توسط خود. داده‌های شما در زیرساخت شما باقی می‌ماند.

**مسیر خود را انتخاب کنید:**

| می‌خواهم... | چگونه |
|---|---|
| امتحان روی دستگاه خودم | اجرای نصب یک‌مرحله‌ای زیر (نیاز به Docker دارد) |
| استقرار در سرور یا VPS | استقرار یک‌کلیک با Railway |
| استفاده از اپلیکیشن مک | [به لیست انتظار بپیوندید](https://www.getcore.me/) |

**نصب و شروع CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

ویزارد راه‌اندازی از شما می‌خواهد که مسیر نصب، ارائه‌دهنده هوش مصنوعی، کلید API و مدل گفتگو را وارد کنید. این ویزارد رمزها را تولید کرده، استک را راه‌اندازی می‌کند و آدرس `http://localhost:3033` را باز می‌کند.

بیشتر نصب‌های محلی پس از اجرای داکر چند دقیقه زمان می‌برند.

**یا روی Railway مستقر کنید:**

[![استقرار روی Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**یک دروازه متصل کنید** تا CORE بتواند عوامل کدنویسی را اجرا کند، مرورگر شما را کنترل کند و به پوشه‌های محلی دسترسی پیدا کند:

```bash
corebrain login
corebrain gateway setup
```
**نیازمندی‌ها:** داکر ۲۰.۱۰+، داکر کامپوز ۲.۲۰+، ۴ vCPU / ۸ گیگابایت رم

[راهنمای کامل میزبانی شخصی](https://docs.getcore.me/self-hosting/setup)

**اولین کار شما (۲ دقیقه پس از راه‌اندازی):**

۱. **دفترچه یادداشت** را باز کنید (صفحه روزانه شما در `http://localhost:3033`)
۲. بنویسید `[ ] خلاصه کردن مسائل باز گیت‌هاب من` یا هر کاری که معمولاً خودتان انجام می‌دهید
۳. CORE آن را ظرف ۳ دقیقه دریافت می‌کند، زمینه را از برنامه‌های متصل جمع‌آوری کرده و یک برنامه پیش‌نویس می‌کند
۴. برنامه را تایید کنید و CORE آن را اجرا کرده و نتیجه را بازمی‌گرداند

[اولین برنامه خود را متصل کنید](https://docs.getcore.me/connectors)

## مستندات

- [**حافظه**](https://docs.getcore.me/memory/overview) - گراف دانش زمانی، طبقه‌بندی حقایق، بازیابی مبتنی بر هدف
- [**دفترچه یادداشت**](https://docs.getcore.me/concepts/scratchpad) - سطح روزانه‌ای که وظایف و ایده‌ها از آن آغاز می‌شوند
- [**وظایف**](https://docs.getcore.me/concepts/tasks) - برنامه‌ها، وضعیت، کارهای تکراری و زمینه محدوده وظیفه
- [**جعبه‌ابزار**](https://docs.getcore.me/concepts/toolkit) - بیش از ۱۰۰۰ اقدام در ۵۰+ برنامه از طریق MCP
- [**عامل CORE**](https://docs.getcore.me/concepts/meta-agent) - محرک‌ها، حافظه، ابزارها و اجرا
- [**درگاه**](https://docs.getcore.me/access-core/overview) - دسترسی از طریق واتساپ، اسلک، تلگرام، ایمیل، وب و API
- [**مهارت‌ها**](https://docs.getcore.me/skills/overview) - دستورالعمل‌های قابل استفاده مجدد برای گردش‌کارهای تکرارشونده
- [**میزبانی شخصی**](https://docs.getcore.me/self-hosting/setup) - راهنمای کامل استقرار
- [**تغییرات**](https://docs.getcore.me/opensource/changelog) - آنچه منتشر شده است

CORE به دقت متوسط **۸۸.۲۴٪** در [معیار LoCoMo](https://github.com/RedPlanetHQ/core-benchmark) در زمینه‌های تک‌مرحله‌ای، چندمرحله‌ای، دامنه باز و استدلال زمانی دست می‌یابد. برای مشاهده نتایج کامل و مقایسه با خط پایه به مخزن معیار مراجعه کنید.

## امنیت

- دارای گواهی CASA Tier 2
- TLS 1.3 در انتقال

- AES-256 در حالت استراحت
- داده‌های شما هرگز برای آموزش مدل استفاده نمی‌شود
- میزبانی خودکار برای ایزوله‌سازی کامل
- [سیاست امنیتی](https://raw.githubusercontent.com/RedPlanetHQ/core/main/SECURITY.md)
- آسیب‌پذیری‌ها: harshith@poozle.dev
