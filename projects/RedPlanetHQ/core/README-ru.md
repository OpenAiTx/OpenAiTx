## Всегда на страже. Всегда готов.

Большинство ИИ-инструментов ждут запроса. CORE наблюдает.

Подключите его к своим приложениям, и он будет мониторить активность во всех из них. Приходит письмо от клиента, назначается задача на GitHub, срабатывает оповещение Sentry, заканчивается встреча. CORE видит это, сверяется с вашей памятью и установленными навыками и либо решает проблему сам, либо передает её на ваше рассмотрение. Вам не нужно его запускать. Он замечает сам.

Установите плагин CORE в Claude Code, Codex или Cursor, и ваши беседы с агентами тоже будут отслеживаться. Обсуждаемый контекст, принятые решения, написанный код — всё это попадает в граф знаний памяти. Следующая задача, которую возьмёт CORE, уже будет знать, что произошло в прошлой сессии.

Когда он действует, он делает это напрямую. Может отвечать на письма, обновлять задачи в Linear, создавать PR на GitHub, отправлять сообщения в Slack, выполнять команды в терминале, управлять браузером и запускать сессию Claude Code или Codex из любого интерфейса. Отправьте сообщение в WhatsApp из аэропорта — и CORE уже запустит кодовую сессию и откроет PR до вашего посадки.

Вы решаете, где он действует самостоятельно, а где ждет вашего указания. Обрабатывайте оповещения Sentry автоматически, но всегда запрашивайте подтверждение перед слиянием. Одобряйте план перед началом сессии кодирования. Требуйте подтверждения перед отправкой любого письма. Уровень автономии вы задаете сами — для каждой задачи, для каждого приложения, для каждого действия.

## Четыре способа взаимодействия

У чат-бота один интерфейс. У операционной системы — множество.

**Голос.** Нажмите Ctrl+Option на Mac и скажите, что нужно сделать. CORE выполнит это в фоновом режиме, не отвлекая вас от работы.

**Черновик.** Откройте свою ежедневную страницу и напишите `[ ] Исправить ошибку аутентификации из issue #47`. CORE заметит это в течение 3 минут, загрузит контекст из вашего репозитория и памяти и подготовит план.

**Мессенджеры.** WhatsApp, Slack, Telegram. Отправьте задачу из аэропорта, с телефона, из кровати. CORE всегда в курсе всего, независимо от источника сообщения.

**Чат.** Откройте дашборд и общайтесь напрямую, как с любым помощником. Когда нужно обсудить детали до делегирования.

Один ИИ, четыре интерфейса, единая память и контекст для всех.

Выберите голос для голосового взаимодействия. С этого момента он будет звучать и вести себя как настроенный вами ИИ во всех интерфейсах.

## CORE в действии

### Скажите — вернитесь к PR.

*Нажмите Ctrl+Option, скажите:* "Исправь гонку в процессе оформления заказа из issue #312."

CORE загрузит issue, подтянет связанные коммиты и ветки Slack, подготовит план и запустит сессию Claude Code. Вы возвращаетесь — и видите diff. Вы так и не подходили к рабочему столу.

### Напишите ночью — проверьте утром.

*Черновик:* `[ ] Разобраться с сегодняшним бэклогом начиная с 23:00`

CORE извлекает задачи из Linear, GitHub и памяти, расставляет приоритеты и работает над ними, пока вы спите. Готовые решения ждут вашего ревью. Если сессия застопорилась, CORE возвращается с одним чётким вопросом, а не с зависшей вкладкой.

### Отправьте сообщение из любой точки.

*WhatsApp из аэропорта:* "Выпусти рефакторинг авторизации."

CORE уже знает ветку, контекст и ваши предпочтения. Работает в Railway. Запускает сессию до вашего посадки на рейс.

### Проверяйте тревоги до того, как они станут инцидентами.

*Sentry срабатывает в 2 ночи.* CORE изучает проблему, извлекает связанные трассировки и прошлые инциденты из памяти, предлагает решение и пишет вам в Slack: "Issue #847, фикс предложен, ждёт вашего ревью." Вы утверждаете прямо с телефона.

### Получайте сводку, которая учитывает вашу неделю.

*Повторяющаяся задача, каждое утро в 8:00.* CORE собирает данные из email, GitHub, Linear и Slack, выделяет то, что действительно требует внимания, пропускает неважное и автоматически превращает последующие действия в задачи.

| **Задачи** | Одноразовые или повторяющиеся рабочие единицы с вашей спецификацией, планом CORE, живым состоянием и отдельным чатом. Каждая задача может запускать код, браузер или терминальные сессии. |

| **Шлюз** | Запускает Claude Code, Codex, браузерные агенты и команды терминала на вашем компьютере или в Docker и Railway, чтобы CORE работал даже при закрытом ноутбуке. |
| **Модель-независимость** | Используйте любого провайдера: Anthropic, OpenAI или open-weight модели. Самостоятельный хостинг полного стека для полной изоляции. |

## Быстрый старт

Открытый исходный код и самостоятельный хостинг. Ваши данные остаются в вашей инфраструктуре.

**Выберите свой путь:**

| Я хочу... | Как |
|---|---|
| Попробовать на своем компьютере | Запустите установку в один шаг ниже (требуется Docker) |
| Развернуть на сервере или VPS | Однокликовое развертывание через Railway |
| Использовать приложение для Mac | [Присоединиться к листу ожидания](https://www.getcore.me/) |

**Установите и запустите CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

Мастер установки запрашивает каталог установки, провайдера ИИ, API-ключ и модель чата. Он генерирует секреты, запускает стек и открывает `http://localhost:3033`.

Большинство локальных установок занимает несколько минут после запуска Docker.

**Или разверните на Railway:**

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**Подключите шлюз**, чтобы CORE мог запускать кодирующих агентов, управлять вашим браузером и получать доступ к локальным папкам:

```bash
corebrain login
corebrain gateway setup
```

**Требования:** Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

[Полное руководство по самостоятельному размещению](https://docs.getcore.me/self-hosting/setup)

**Ваше первое задание (2 минуты после установки):**

1. Откройте **Scratchpad** (ваша ежедневная страница по адресу `http://localhost:3033`)
2. Введите `[ ] Подытожить мои открытые задачи GitHub` или любую задачу, которую обычно выполняете сами
3. CORE подхватит её в течение 3 минут, соберёт контекст из подключённых приложений и составит план
4. Одобрите план, CORE выполнит его и вернёт результат

[Подключите ваше первое приложение](https://docs.getcore.me/connectors)
