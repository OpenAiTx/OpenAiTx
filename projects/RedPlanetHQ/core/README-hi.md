## हमेशा देख रहा है। हमेशा तैयार है।

अधिकांश AI टूल पूछे जाने का इंतजार करते हैं। CORE देखता है।

इसे अपने ऐप्स से जोड़ें और यह उन सभी में गतिविधियों की निगरानी करता है। एक क्लाइंट से ईमेल आता है, कोई GitHub इश्यू असाइन होता है, Sentry अलर्ट आता है, मीटिंग खत्म होती है। CORE इसे देखता है, आपकी मेमोरी और सेट की गई स्किल्स को चेक करता है, और या तो उसे खुद संभालता है या आपके निर्णय के लिए प्रस्तुत करता है। आप इसे ट्रिगर नहीं करते। यह खुद नोटिस करता है।

Claude Code, Codex, या Cursor में CORE प्लगइन इंस्टॉल करें और आपकी एजेंट बातचीत भी देखी जाती है। चर्चा किया गया संदर्भ, लिए गए निर्णय, लिखा गया कोड, यह सब मेमोरी नॉलेज ग्राफ में जुड़ता है। अगला कार्य जो CORE उठाता है, उसे पहले से पता होता है कि आपकी पिछली सत्र में क्या हुआ था।

जब यह कार्य करता है, तो सीधे करता है। यह ईमेल का जवाब दे सकता है, Linear इश्यू अपडेट कर सकता है, GitHub PR फाइल कर सकता है, Slack संदेश भेज सकता है, टर्मिनल कमांड चला सकता है, ब्राउज़र चला सकता है, और किसी भी इंटरफेस से Claude Code या Codex सत्र शुरू कर सकता है। एयरपोर्ट से WhatsApp संदेश भेजें और CORE आपके बोर्डिंग से पहले कोडिंग सत्र चला सकता है और PR खोल सकता है।


आप तय करते हैं कि यह कब खुद कार्रवाई करे और कब आपकी कॉल का इंतजार करे। Sentry अलर्ट्स को स्वचालित रूप से संभालें, लेकिन मर्जिंग से पहले हमेशा पूछें। कोडिंग सत्र शुरू होने से पहले योजना अनुमोदित करें। किसी भी ईमेल भेजने से पहले पुष्टि आवश्यक करें। स्वायत्तता का स्तर आप सेट कर सकते हैं, हर कार्य, हर ऐप, हर क्रिया के लिए।

## इसे तक पहुँचने के चार तरीके

एक चैटबॉट का एक इंटरफ़ेस होता है। एक OS के कई।

**वॉयस।** मैक पर Ctrl+Option दबाएँ और जो करना है वह कहें। CORE इसे बैकग्राउंड में चलाता है, आपके काम में बाधा डाले बिना।

**स्क्रैचपैड।** अपनी दैनिक पेज खोलें और लिखें `[ ] ऑथ बग को ठीक करें, जो issue #47 में है।` CORE इसे 3 मिनट में पकड़ लेता है, आपके रिपो और मेमोरी से संदर्भ लोड करता है, और एक योजना ड्राफ्ट करता है।

**मैसेजिंग।** WhatsApp, Slack, Telegram। एयरपोर्ट से, फोन से, बिस्तर से टास्क भेजें। संदेश कहीं से भी आए, CORE के पास आपका पूरा संदर्भ रहता है।

**चैट।** डैशबोर्ड खोलें और सीधे बात करें, जैसे कोई सहायक। जब आप डेलीगेट करने से पहले चर्चा करना चाहते हैं।

एक AI, चार सतहें, सभी के पीछे वही मेमोरी और संदर्भ।

## इसे अपना बनाएं

इसका नाम दें। चुनें कि यह कैसे बात करता है। नियम सेट करें जो यह हर जगह फॉलो करता है।

पाँच बिल्ट-इन पर्सनैलिटी में से चुनें — TARS की सूखी दक्षता, Alfred की वफादार औपचारिकता, Hudson की गर्म व्यावहारिकता — या अपनी खुद की लिखें। पर्सनैलिटी हर कार्य में, हर योजना में, हर संदेश में झलकती है।

बोलने के लिए इसकी आवाज चुनें। इसके बाद, यह आपके द्वारा कॉन्फ़िगर किए गए AI की तरह ही आवाज़ देता है और व्यवहार करता है, हर इंटरफ़ेस पर।

## CORE का क्रियान्वयन

### कहें, PR पर वापस आएं।

*Ctrl+Option दबाएँ, बोलें:* "चेकआउट फ्लो में रेस कंडीशन को issue #312 से ठीक करें।"

CORE मुद्दा लोड करता है, संबंधित कमिट्स और Slack थ्रेड्स खींचता है, योजना ड्राफ्ट करता है, और Claude Code सत्र चलाता है। आप एक डिफ पर वापस आते हैं। आप कभी डेस्क पर नहीं थे।

### रात में लिखें, सुबह समीक्षा करें।

*Scratchpad:* `[ ] आज रात की लंबित सूची पर 11 बजे से काम करें`

CORE Linear, GitHub, और मेमोरी से डेटा लेता है, प्राथमिकता देता है, और आपके सोते समय इन पर काम करता है। सफल रन आपकी समीक्षा के लिए तैयार रहते हैं। अटकी हुई सत्र केवल एक सटीक प्रश्न के साथ लौटती है, न कि अटकी हुई टैब के साथ।

### इसे कहीं से भी संदेश भेजें।

*हवाई अड्डे से WhatsApp:* "auth refactor शिप करें।"

CORE को पहले से ही ब्रांच, संदर्भ और आपकी प्राथमिकताएँ पता हैं। यह Railway में चल रहा है। आपके बोर्डिंग से पहले ही यह सत्र शुरू कर देता है।

### अलर्ट्स की जाँच करें, इससे पहले कि वे घटनाएँ बनें।

*2am पर Sentry अलर्ट करता है।* CORE जांच करता है, संबंधित ट्रेस और पिछले घटनाओं को मेमोरी से निकालता है, समाधान प्रस्तावित करता है, और आपको Slack पर पिंग करता है: "Issue #847, समाधान प्रस्तावित, आपकी समीक्षा की प्रतीक्षा।" आप अपने फोन से अनुमोदन देते हैं।

### एक संक्षिप्त रिपोर्ट पाएं जो आपकी हफ्ते को पहले से जानती हो।

*हर सुबह 8am पर आवर्ती कार्य।* CORE ईमेल, GitHub, Linear और Slack से डेटा लेता है, जो वास्तव में ध्यान देने योग्य है उसे सामने लाता है, जो नहीं है उसे छोड़ता है, और फॉलो-अप को स्वचालित रूप से कार्यों में बदल देता है।

## त्वरित प्रारंभ

ओपन सोर्स और स्वयं-होस्टेड। आपका डेटा आपकी इन्फ्रास्ट्रक्चर में ही रहता है।

**अपना रास्ता चुनें:**

| मैं यह करना चाहता हूँ... | कैसे |
|---|---|
| इसे अपनी मशीन पर आज़माएँ | नीचे दिया गया एक-चरणीय इंस्टॉल चलाएँ (Docker आवश्यक) |
| सर्वर या VPS पर डिप्लॉय करें | एक-क्लिक रेलवे डिप्लॉय |
| मैक ऐप का उपयोग करें | [वेटलिस्ट ज्वाइन करें](https://www.getcore.me/) |

**CORE इंस्टॉल करें और प्रारंभ करें:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```
सेटअप विजार्ड इंस्टॉल डायरेक्टरी, एआई प्रोवाइडर, एपीआई की, और चैट मॉडल पूछता है। यह सीक्रेट्स जनरेट करता है, स्टैक शुरू करता है, और `http://localhost:3033` खोलता है।

अधिकांश लोकल इंस्टॉल्स कुछ मिनटों में हो जाते हैं जब डॉकर चल रहा होता है।

**या Railway पर डिप्लॉय करें:**

[![Railway पर डिप्लॉय करें](https://railway.app/button.svg)](https://railway.com/deploy/core)

**एक गेटवे कनेक्ट करें** ताकि CORE कोडिंग एजेंट्स चला सके, आपका ब्राउज़र चला सके, और लोकल फोल्डर्स एक्सेस कर सके:


```bash
corebrain login
corebrain gateway setup
```
**आवश्यकताएँ:** Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

[पूर्ण सेल्फ-होस्टिंग गाइड](https://docs.getcore.me/self-hosting/setup)

**आपका पहला कार्य (सेटअप के 2 मिनट बाद):**

1. **Scratchpad** खोलें (आपका दैनिक पृष्ठ `http://localhost:3033` पर)
2. टाइप करें `[ ] मेरे खुले GitHub मुद्दों को संक्षिप्त करें` या कोई कार्य जो आप सामान्यतः स्वयं करते हैं
3. CORE इसे 3 मिनट के भीतर उठाता है, जुड़े ऐप्स से संदर्भ एकत्र करता है, और एक योजना तैयार करता है
4. योजना को स्वीकृत करें और CORE इसे चलाता है और परिणाम वापस लाता है

[अपना पहला ऐप कनेक्ट करें](https://docs.getcore.me/connectors)
