## हमेशा देख रहा। हमेशा तैयार।

ज्यादातर AI टूल्स इंतजार करते हैं कि उनसे पूछा जाए। CORE निगरानी करता है।

इसे अपनी ऐप्स से जोड़ें और यह उन सभी में गतिविधियों की निगरानी करता है। किसी क्लाइंट से ईमेल आता है, कोई GitHub इश्यू असाइन होता है, Sentry अलर्ट आता है, कोई मीटिंग खत्म होती है। CORE इसे देखता है, आपकी मेमोरी और सेट किए गए स्किल्स चेक करता है, और या तो खुद संभालता है या आपके लिए सतह पर लाता है। आपको इसे ट्रिगर करने की जरूरत नहीं। यह खुद ही नोटिस कर लेता है।

Claude Code, Codex, या Cursor में CORE प्लगइन इंस्टॉल करें और आपकी एजेंट वार्तालापों की भी निगरानी होती है। चर्चा किया गया संदर्भ, लिए गए निर्णय, लिखा गया कोड—सब मेमोरी नॉलेज ग्राफ में जाता है। अगला कार्य जो CORE उठाता है, उसे पहले से पता होता है कि आपकी पिछली सत्र में क्या हुआ था।

जब यह कार्य करता है, तो सीधे करता है। यह ईमेल का जवाब दे सकता है, Linear इश्यू अपडेट कर सकता है, GitHub PR फाइल कर सकता है, Slack संदेश भेज सकता है, टर्मिनल कमांड चला सकता है, ब्राउज़र चला सकता है, और किसी भी इंटरफ़ेस से Claude Code या Codex सत्र शुरू कर सकता है। एयरपोर्ट से WhatsApp संदेश भेजें और CORE आपके बोर्डिंग से पहले कोडिंग सत्र चालू कर सकता है और PR खोल सकता है।
आप तय करते हैं कि यह कहाँ स्वतः कार्य करे और कहाँ आपके आदेश की प्रतीक्षा करे। Sentry अलर्ट्स को स्वचालित रूप से संभालें लेकिन मर्ज करने से पहले हमेशा पूछें। कोडिंग सत्र शुरू होने से पहले योजना को स्वीकृत करें। कोई भी ईमेल भेजने से पहले पुष्टि की आवश्यकता रखें। स्वायत्तता का स्तर आप तय करते हैं — हर कार्य, हर ऐप, हर क्रिया के लिए।

---

## इसे पहुँचने के चार तरीके

एक चैटबॉट का एक इंटरफेस होता है। एक OS के कई।

**वॉयस।** मैक पर Ctrl+Option दबाएँ और बताएँ क्या करना है। CORE इसे बैकग्राउंड में चलाता है, बिना आपके फ्लो को बाधित किए।

**स्क्रैचपैड।** अपना डेली पेज खोलें और लिखें `[ ] Fix the auth bug from issue #47`। CORE इसे 3 मिनट के भीतर पकड़ लेता है, आपके रेपो और मेमोरी से संदर्भ लोड करता है, और एक योजना बनाता है।

**मैसेजिंग।** WhatsApp, Slack, Telegram। एयरपोर्ट से, फोन से, बिस्तर से कोई कार्य भेजें। CORE आपके संदेश के स्रोत की परवाह किए बिना आपके पूरे संदर्भ को समझता है।

**चैट।** डैशबोर्ड खोलें और सीधे बात करें, जैसे किसी सहायक से। जब आप कार्य सौंपने से पहले संवाद करना चाहें।

एक AI, चार इंटरफेस, सभी के पीछे वही मेमोरी और संदर्भ।

---

## इसे अपना बनाएं

इसे एक नाम दें। चुनें यह कैसे बोले। वह नियम सेट करें जिनका यह हर जगह पालन करे।

पाँच बिल्ट-इन व्यक्तित्वों में से चुनें — TARS की सूखी दक्षता, Alfred की वफादार औपचारिकता, Hudson की गर्मजोशी और व्यावहारिकता — या खुद का व्यक्तित्व लिखें। इसका व्यक्तित्व हर कार्य, हर योजना, हर संदेश में दिखेगा।

बोली के लिए इसकी आवाज चुनें। इसके बाद से, हर इंटरफेस पर यह उसी AI की तरह बोलेगा और व्यवहार करेगा जैसा आपने बनाया।

## CORE की कार्यशैली

### बोलें, PR पर लौटें।

*Ctrl+Option दबाएँ, बोलें:* "चेकआउट फ्लो में रेस कंडीशन को issue #312 से ठीक करें।"

CORE उस मुद्दे को लोड करता है, संबंधित कमिट्स और Slack थ्रेड्स लाता है, एक योजना बनाता है, और एक Claude Code सत्र चलाता है। आप वापस आते हैं तो एक डिफ मिलता है। आप कभी अपनी डेस्क पर नहीं थे।

### रात में लिखें, सुबह समीक्षा करें।


*Scratchpad:* `[ ] आज रात के बैकलॉग को 11pm से शुरू करके निपटाएं`

CORE Linear, GitHub, और मेमोरी से डेटा लेता है, प्राथमिकता तय करता है, और जब आप सो रहे होते हैं तब काम करता है। स्मूद रन आपकी समीक्षा के लिए तैयार मिलते हैं। फंसे हुए सेशन एक सटीक सवाल के साथ लौटते हैं, न कि अटकी हुई टैब के साथ।

### कहीं से भी मैसेज भेजें।

*एयरपोर्ट से WhatsApp:* "auth refactor शिप करो।"

CORE को पहले से ब्रांच, संदर्भ, और आपकी प्राथमिकताएँ पता हैं। यह Railway में चल रहा है। आपके बोर्डिंग से पहले यह सेशन शुरू कर देता है।

### अलर्ट्स की जाँच करें, इससे पहले कि वे घटनाएँ बनें।

*2am पर Sentry अलर्ट देता है।* CORE जांच करता है, संबंधित ट्रेसेस और पूर्व घटनाएँ मेमोरी से निकालता है, समाधान सुझाता है, और आपको Slack पर पिंग करता है: "Issue #847, समाधान प्रस्तावित, आपकी समीक्षा का इंतजार।" आप अपने फोन से अप्रूव कर देते हैं।

### एक ब्रीफ पाएं, जो पहले से आपका हफ्ता जानता है।

*हर सुबह 8am पर दोहराया जाने वाला कार्य।* CORE ईमेल, GitHub, Linear, और Slack से डेटा लेता है, जो सच में ध्यान देने लायक है उसे सामने लाता है, जो जरूरी नहीं उसे छोड़ देता है, और फॉलो-अप्स को अपने-आप कार्यों में बदल देता है।

## त्वरित प्रारंभ

ओपन सोर्स और स्वयं-होस्टेड। आपका डेटा आपके इन्फ्रास्ट्रक्चर में ही रहता है।

**अपना मार्ग चुनें:**

| मैं यह करना चाहता हूँ... | कैसे |
|---|---|
| इसे अपनी मशीन पर आज़माएं | नीचे दिया गया वन-स्टेप इंस्टॉल चलाएँ (Docker आवश्यक) |
| सर्वर या VPS पर डिप्लॉय करें | वन-क्लिक रेलवे डिप्लॉय |
| मैक ऐप का उपयोग करें | [वेटलिस्ट में जुड़ें](https://www.getcore.me/) |

**CORE इंस्टॉल और शुरू करें:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```
सेटअप विजार्ड इंस्टॉल डायरेक्टरी, एआई प्रोवाइडर, एपीआई की, और चैट मॉडल पूछता है। यह सीक्रेट्स जनरेट करता है, स्टैक शुरू करता है, और `http://localhost:3033` खोलता है।

अधिकांश लोकल इंस्टॉल्स कुछ मिनटों में हो जाते हैं जब डॉकर चल रहा होता है।

**या Railway पर डिप्लॉय करें:**

[![Railway पर डिप्लॉय करें](https://railway.app/button.svg)](https://railway.com/deploy/core)

**एक गेटवे कनेक्ट करें** ताकि CORE कोडिंग एजेंट्स चला सके, आपका ब्राउज़र चला सके, और लोकल फोल्डर्स एक्सेस कर सके:


```bash
corebrain login
corebrain gateway setup
```
**आवश्यकताएँ:** Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

[पूर्ण सेल्फ-होस्टिंग गाइड](https://docs.getcore.me/self-hosting/setup)

**आपका पहला कार्य (सेटअप के 2 मिनट बाद):**

1. **Scratchpad** खोलें (आपका दैनिक पृष्ठ `http://localhost:3033` पर)
2. टाइप करें `[ ] मेरे खुले GitHub मुद्दों को संक्षिप्त करें` या कोई कार्य जो आप सामान्यतः स्वयं करते हैं
3. CORE इसे 3 मिनट के भीतर उठाता है, जुड़े ऐप्स से संदर्भ एकत्र करता है, और एक योजना तैयार करता है
4. योजना को स्वीकृत करें और CORE इसे चलाता है और परिणाम वापस लाता है
