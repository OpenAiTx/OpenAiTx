CORE 是你的數位大腦 —— 一個統一、持續運作的人工智慧，能記住你的情境、協作你的 AI 代理，並於各種工具間執行操作。

開源。可自我部署。你的資料，由你掌控。

---

## 什麼是數位大腦

### 它記住一切

每一次對話、決策、偏好與關係都會被存為結構化知識於時間知識圖譜中。不是原始文字，而是分類後的事實（偏好、決策、目標、指令），於你需要時即刻浮現。幾週前的事情也能詢問，答案連同完整上下文都在。

[深入了解記憶體 →](https://docs.getcore.me/concepts/memory/overview)

### 它可以執行操作

建立 GitHub 問題、發送 Slack 訊息、管理行事曆，全部無需離開對話。也能處理複雜的多步驟工作流程：請求你的程式代理檢查 Sentry 是否有新錯誤，為每個錯誤建立 GitHub 問題，根據 git blame 分配給合適的工程師，並將摘要發佈到團隊的 Slack 頻道，全部只需一個指令。透過單一連接點，可存取 50+ 應用程式的 200+ 項操作。

[深入了解工具包 →](https://docs.getcore.me/concepts/toolkit)

### 它能跨系統思考與運作

你的工具是隔離的。Claude Code 不能與你的 WhatsApp 通訊。Cursor 不能檢查你的行事曆。CORE 的代理打破這些壁壘——它是協調者，將記憶體、工具包與你的應用程式整合在一起。

CORE 代理能做什麼：

- **理解意圖**：分類你的需求，導向正確的工具與記憶體
- **產生其他代理**：從 WhatsApp 啟動 Claude Code 或瀏覽器會話來修復錯誤或研究問題
- **協調多步驟工作流程**：從記憶體提取情境、跨多個應用程式行動並回報
- **隨時隨地工作**：從 WhatsApp、電子郵件、Slack 或網頁儀表板發送訊息——同一個大腦、同一個情境

[深入了解 CORE 代理 →](https://docs.getcore.me/concepts/meta-agent)

### 它能主動執行

你不必事事開口要求。CORE 監控事件——新電子郵件、GitHub 警報、行事曆變更——並根據它對你的了解進行評估。客戶郵件一到；CORE 已經知道你如何處理這段關係並草擬合適回覆。Sentry 在夜間發出警報；CORE 啟動 Claude Code 會話進行調查。

[深入了解概念 →](https://docs.getcore.me/concepts/overview)

## 架構

CORE 建構於三個層級之上，實現這四大支柱：

- **記憶體**：一個時序知識圖，儲存事件、實體和分類事實。每條資訊都會被分類（偏好、決策、指令、目標等）並隨時間連結。這讓檢索基於意圖而非關鍵字。[文件 →](https://docs.getcore.me/concepts/memory/overview)

- **工具包**：一個統一的操作層，適用於所有 MCP 相容的代理。你的應用程式（GitHub、Linear、Slack、Gmail、行事曆等）僅需連接一次，每個連接的 AI 工具即可透過單一端點存取 200+ 項操作。[文件 →](https://docs.getcore.me/toolkit/overview)

- **CORE 代理**：協調記憶體與工具包的核心。它理解意圖、搜尋記憶體、選擇工具、產生其他代理，並能根據觸發條件與規則主動行動。[文件 →](https://docs.getcore.me/concepts/meta-agent)

## 您可以做什麼

### 作為您的個人助理使用

從 WhatsApp、電子郵件或網頁儀表板與 CORE 對話。它能在您的應用程式間執行操作、管理您的行程、撰寫電子郵件，並記住您的身份和工作方式。

[開始使用 →](https://docs.getcore.me/quickstart/chief-of-staff)

### 強化您的 AI 代理人

賦予 Claude Code、Cursor 及其他 AI 工具持久記憶與行動能力。您的程式代理人能記住專案架構、過去決策及偏好，跨會話無需重複解釋背景。

[開始使用 →](https://docs.getcore.me/quickstart/supercharge-ai-agents)

<details>
<summary><b>Claude Code（推薦：插件）</b></summary>

```sh
npm install -g @redplanethq/corebrain
```

然後在 Claude Code 中：

```
/plugin marketplace add redplanethq/core
/plugin install core_brain
```

重新啟動 Claude Code 並執行 `/mcp` 以進行身份驗證。

此插件會在每次會話開始時自動載入你的個人設定（偏好、規則、決策），並在你結束對話時將其內容匯入記憶體。

</details>

```sh
claude mcp add --transport http --scope user core-memory https://app.getcore.me/api/v1/mcp?source=Claude-Code
```
然後輸入 `/mcp` 並開啟 core-memory 進行驗證。

</details>

## 給代理建構者

正在打造 AI 代理人？將記憶與整合交給 CORE 處理，讓你專注於代理人的邏輯設計。

- **卸載記憶管理** — 使用 CORE 的時序知識圖作為代理人的長期記憶。儲存對話、透過意圖驅動的搜尋檢索上下文，讓你的代理人隨時間累積知識，無需自行管理向量資料庫或知識圖。
- **卸載整合管理** — 在 CORE 中一次連接應用，代理人即可獲得所有應用的 MCP 工具。無需建構 OAuth 流程，無需維護 API，無需為每個整合編寫程式碼。
- **透過 MCP 或 API 建構** — 透過 MCP（單一端點）將代理人連接至 CORE，或直接使用 REST API。

**範例專案**

- [**core-cli**](https://github.com/RedPlanetHQ/core-cli) — 具備記憶與 Linear/GitHub 同步的任務管理代理人
- [**holo**](https://github.com/RedPlanetHQ/holo) — 將你的 CORE 記憶轉換為可聊天的個人網站

[API 參考文件 →](https://docs.getcore.me/api-reference) · [SDK 文件 →](https://docs.getcore.me)

## 安全性

CASA Tier 2 認證 — 經第三方稽核，符合 Google 的 OAuth 要求。

- 加密：TLS 1.3（傳輸）+ AES-256（靜態）
- 以工作區為基礎的隔離、基於角色的權限管理
- 你的資料絕不會用於 AI 模型訓練
- 提供自我託管選項以達到完全隔離
