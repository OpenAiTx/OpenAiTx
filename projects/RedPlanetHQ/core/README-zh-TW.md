## 時刻監控。時刻準備。

大多數 AI 工具等你指示。CORE 主動監控。

將其連接至你的應用程式，它會監控所有應用的活動。當收到客戶的電子郵件、GitHub 指派 issue、Sentry 發出警報、會議結束時，CORE 都能偵測、檢查你的記憶和已設定的技能，然後自動處理或提交給你決斷。你無需手動觸發，它會自動察覺。

將 CORE 外掛安裝在 Claude Code、Codex 或 Cursor，你的智能代理對話也會受到監控。討論的內容、做出的決策、撰寫的程式碼，都會進入記憶知識圖。下一次 CORE 接手任務時，已經知道你上次會話的情況。

當它執行時，會直接採取行動。可以回覆電子郵件、更新 Linear 任務、建立 GitHub PR、發送 Slack 訊息、執行終端指令、操控瀏覽器，並從任何介面啟動 Claude Code 或 Codex 工作階段。即使你在機場發送 WhatsApp 訊息，CORE 也能在你登機前，啟動寫程式工作並開啟 PR。


你可以決定哪些操作由它自主執行，哪些則等你指示。自動處理 Sentry 警報，但合併前必須徵求你的同意。每次編碼會議開始前先批准計畫。任何電子郵件發送前都要求確認。自主權的高低，由你針對每個任務、每個應用、每個動作自行設定。

## 四種方式與它互動

聊天機器人只有一個介面。作業系統則有很多。

**語音。** 在 Mac 上按下 Ctrl+Option，然後說出需要處理的事項。CORE 於背景執行，不影響你的工作流程。

**便條紙。** 打開你的每日頁面，寫下 `[ ] 修復第 #47 號問題的授權錯誤`。CORE 在 3 分鐘內擷取，從你的程式庫和記憶體載入上下文，並擬定計畫。

**訊息傳送。** WhatsApp、Slack、Telegram。從機場、手機、床上發送任務。無論訊息來自哪裡，CORE 都能完整掌握你的上下文。

**聊天。** 打開儀表板，直接對話，就像任何助理一樣。當你想要在委派前多加討論時使用。

一個 AI，四個介面，背後同樣的記憶與脈絡。

## 打造專屬於你的 AI

給它取個名字。選擇它的說話方式。訂定它在各處都要遵守的規則。

五種內建個性任你挑選──TARS 的冷靜高效、Alfred 的忠誠禮貌、Hudson 的溫暖務實──也可以自訂。它的個性貫穿每個任務、每份計畫、每則訊息。

選擇口語互動時的語音。從此，無論在哪個介面，它都以你設定的 AI 形象發聲與行動。

## CORE 實際應用

### 說出需求，回來就是 PR。

*按下 Ctrl+Option，說：*「修復第 #312 號問題中，結帳流程的競爭狀況。」

CORE 讀取該問題，拉取相關提交和 Slack 討論串，擬定計畫，並啟動 Claude Code 會議。你回來時看到一份差異檔。你甚至不必坐在桌前。

### 晚上寫下任務，早上審查成果。

*Scratchpad:* `[ ] 從今晚 11 點開始處理積壓工作`

CORE 從 Linear、GitHub 和記憶體中提取任務、排定優先順序，並在你睡覺時自動執行。順利完成的流程等你審查，遇到卡關的情境則只返回一個明確問題，而不是停滯的分頁。

### 隨時隨地傳送指令。

*機場 WhatsApp 指令：*「部署身份驗證重構。」

CORE 已經知道分支、背景和你的偏好。它運行於 Railway，在你登機前已經啟動工作流程。

### 事件發生前先調查警報。

*Sentry 於凌晨 2 點發出警報。* CORE 進行調查，從記憶體中提取相關追蹤和過往事件，提出修正方案，並在 Slack 上通知你：「問題 #847，修正方案已提出，等待你審查。」你可直接在手機上批准。

### 提供了解你本週狀態的摘要。

*每日上午 8 點的重複任務。* CORE 從郵件、GitHub、Linear 和 Slack 取得資訊，篩選出真正需要注意的事項，自動略過不重要的，並將後續行動自動轉為任務。

## CORE 內部架構

| | |
|---|---|
| **記憶體** | 橫跨所有工具與對話的時序知識圖譜。偏好、決策、目標與指令，確保每個任務都在情境完整下展開。 |
| **任務** | 一次性或重複性工作單元，包含你的規格、CORE 的計畫、即時狀態及專屬聊天室。每個任務可衍生程式碼、瀏覽器或終端機會話。 |
| **連接器** | 50+ 應用透過一個 MCP 端點串接，並支援 webhook 觸發主動自動化。GitHub、Linear、Jira、Slack、Gmail、Calendar、Sentry、Notion、Todoist 等。 |
| **技能** | 可重用指令，根據情境自動觸發。例如：「修復前自動拉取相關 Linear 問題」、「開 PR 前自動執行測試」、「任務完成自動發送 Slack 摘要」。內建 100+ 種，也可自訂。 |
| **閘道** | 可在你的電腦、Docker 或 Railway 上運行 Claude Code、Codex、瀏覽器代理和終端指令，即使筆電關閉 CORE 仍可運作。 |
| **模型無關** | 可自選供應商：Anthropic、OpenAI 或開源模型。可自架全套系統以實現完全隔離。 |

## 快速開始

開源且自行部署。您的資料留在您的基礎設施內。

**選擇您的路徑：**

| 我想要... | 方法 |
|---|---|
| 在我的機器上試用 | 執行下方的一步安裝（需要 Docker） |
| 在伺服器或 VPS 上部署 | 一鍵 Railway 部署 |
| 使用 Mac 應用程式 | [加入候補名單](https://www.getcore.me/) |

**安裝並啟動 CORE：**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```
設定精靈會要求輸入安裝目錄、AI 服務提供者、API 金鑰和聊天模型。它會產生密鑰、啟動堆疊，並開啟 `http://localhost:3033`。

大多數本地安裝只需在 Docker 執行後幾分鐘即可完成。

**或部署到 Railway：**

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**連接閘道器**，讓 CORE 可以運行程式代理、操作您的瀏覽器並存取本機資料夾：


```bash
corebrain login
corebrain gateway setup
```

**需求:** Docker 20.10+、Docker Compose 2.20+、4 核心 CPU / 8GB 記憶體

[完整自架指南](https://docs.getcore.me/self-hosting/setup)

**您的第一個任務（設定後 2 分鐘內）：**

1. 開啟 **Scratchpad**（您每日的頁面，網址為 `http://localhost:3033`）
2. 輸入 `[ ] Summarize my open GitHub issues` 或任何您平常會自己執行的任務
3. CORE 會在 3 分鐘內自動接收，從連接的應用取得情境，並草擬計劃
4. 核准該計劃後，CORE 會執行並回傳結果

[連接您的第一個應用](https://docs.getcore.me/connectors)
