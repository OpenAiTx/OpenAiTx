Open source, zelf-gehost, voor altijd van jou.

## Altijd oplettend. Altijd klaar.

De meeste AI-tools wachten tot ze gevraagd worden. CORE kijkt mee.

Verbind het met je apps en het bewaakt de activiteit over al je applicaties. Er komt een e-mail van een klant, een GitHub issue wordt toegewezen, een Sentry-melding gaat af, een vergadering eindigt. CORE ziet het, controleert je geheugen en de vaardigheden die je hebt ingesteld, en handelt het af of legt het aan je voor. Jij triggert het niet. Het merkt het zelf op.

Installeer de CORE-plugin in Claude Code, Codex of Cursor en je agentgesprekken worden ook bekeken. Context besproken, beslissingen genomen, code geschreven, alles voedt de kennisgrafiek van het geheugen. De volgende taak die CORE oppakt, weet al wat er in je vorige sessie is gebeurd.

Wanneer het handelt, doet het dat direct. Het kan e-mails beantwoorden, Linear-issues bijwerken, GitHub PR's indienen, Slack-berichten versturen, terminalcommando's uitvoeren, een browser bedienen en een Claude Code- of Codex-sessie starten vanaf elk interface. Stuur een WhatsApp-bericht vanaf het vliegveld en CORE kan een codesessie laten draaien en een PR openen voordat je aan boord gaat.

Jij bepaalt waar het zelfstandig handelt en waar het op jouw oproep wacht. Handel Sentry-meldingen automatisch af, maar vraag altijd toestemming voordat er gemerged wordt. Keur een plan goed voordat een codingsessie begint. Vraag om bevestiging voordat er een e-mail wordt verstuurd. Het niveau van autonomie bepaal jij, per taak, per app, per actie.

## Vier manieren om het te benaderen

Een chatbot heeft één interface. Een besturingssysteem heeft er veel.

**Stem.** Druk Ctrl+Option op Mac en spreek uit wat er moet gebeuren. CORE voert het uit op de achtergrond zonder je workflow te verstoren.

**Notitieblok.** Open je dagelijkse pagina en schrijf `[ ] Los de auth-bug op uit issue #47`. CORE pakt het binnen 3 minuten op, laadt context uit je repo en geheugen, en stelt een plan op.

**Berichten.** WhatsApp, Slack, Telegram. Verstuur een taak vanaf het vliegveld, vanaf je telefoon, vanuit bed. CORE heeft altijd je volledige context, ongeacht waar het bericht vandaan komt.

**Chat.** Open het dashboard en praat direct, zoals met elke assistent. Wanneer je eerst even wilt overleggen voordat je delegeert.

Eén AI, vier oppervlakken, dezelfde geheugen en context achter elk van hen.

## Maak het van jou

Geef het een naam. Kies hoe het spreekt. Stel de regels in die het overal volgt.

Kies uit vijf ingebouwde persoonlijkheden — TARS' droge efficiëntie, Alfred's loyale formaliteit, Hudson's warme praktischheid — of schrijf je eigen. De persoonlijkheid is aanwezig bij elke taak die het uitvoert, elk plan dat het opstelt, elk bericht dat het namens jou verstuurt.

Kies zijn stem voor gesproken interacties. Vanaf dan klinkt en handelt het als de AI die jij hebt geconfigureerd, op elke interface.

## CORE in actie

### Spreek het uit, kom terug naar een PR.

*Druk Ctrl+Option, spreek:* "Los de raceconditie op in de checkout-flow uit issue #312."

CORE laadt het issue, haalt gerelateerde commits en Slack-threads op, stelt een plan op, en start een Claude Code-sessie. Je komt terug bij een diff. Je was nooit aan je bureau.

### Schrijf het 's nachts, review het 's ochtends.

*Kladblok:* `[ ] Werk de achterstand van vanavond weg vanaf 23:00`

CORE haalt gegevens uit Linear, GitHub en het geheugen, prioriteert, en werkt het door terwijl jij slaapt. Soepele runs wachten op jouw beoordeling. Vastgelopen sessies komen terug met één gerichte vraag, niet met een vastgelopen tabblad.

### Stuur er overal berichten naartoe.

*WhatsApp vanaf het vliegveld:* "Deploy de auth refactor."

CORE kent de branch, de context en jouw voorkeuren al. Het draait op Railway. Het start de sessie voordat je aan boord gaat.

### Onderzoek waarschuwingen voordat ze incidenten worden.

*Sentry slaat aan om 2 uur 's nachts.* CORE onderzoekt, haalt gerelateerde traces en eerdere incidenten uit het geheugen, stelt een oplossing voor, en stuurt je een ping op Slack: "Issue #847, oplossing voorgesteld, wacht op jouw beoordeling." Jij keurt goed vanaf je telefoon.

### Ontvang een briefing die je week al kent.

*Terugkerende taak, elke ochtend om 8 uur.* CORE haalt uit e-mail, GitHub, Linear en Slack, toont wat echt aandacht nodig heeft, slaat over wat niet nodig is, en maakt van follow-ups automatisch taken.

Bijvoorbeeld: "haal altijd gerelateerde Linear-issues op voor je een fix plant", "draai tests voor het openen van een PR" of "post een Slack-samenvatting als een taak klaar is." 100+ ingebouwd, of schrijf je eigen. |
| **Gateway** | Draait Claude Code, Codex, browseragents en terminalcommando's op jouw machine of in Docker en Railway, zodat CORE blijft werken als je laptop dicht is. |
| **Model agnostisch** | Gebruik je eigen provider: Anthropic, OpenAI of open-weight modellen. Zelf hosten voor volledige isolatie. |

## Snelstart

Open source en zelfgehost. Jouw data blijft binnen je eigen infrastructuur.

**Kies je pad:**

| Ik wil... | Hoe |
|---|---|
| Het op mijn machine proberen | Voer onderstaande eenstapsinstallatie uit (vereist Docker) |
| Deployen op een server of VPS | Eén-klik Railway deployment |
| De Mac-app gebruiken | [Meld je aan voor de wachtlijst](https://www.getcore.me/) |

**Installeer en start CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

De installatie-assistent vraagt om een installatiemap, AI-provider, API-sleutel en chatmodel. Het genereert geheimen, start de stack en opent `http://localhost:3033`.

De meeste lokale installaties duren enkele minuten zodra Docker draait.

**Of implementeer op Railway:**

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**Verbind een gateway** zodat CORE codeeragents kan uitvoeren, je browser kan aansturen en toegang heeft tot lokale mappen:

```bash
corebrain login
corebrain gateway setup
```

**Vereisten:** Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

[Volledige zelf-hosting gids](https://docs.getcore.me/self-hosting/setup)

**Je eerste taak (2 minuten na installatie):**

1. Open de **Scratchpad** (je dagelijkse pagina op `http://localhost:3033`)
2. Typ `[ ] Vat mijn open GitHub issues samen` of een taak die je normaal zelf zou doen
3. CORE pikt dit binnen 3 minuten op, verzamelt context uit gekoppelde apps, en stelt een plan op
4. Keur het plan goed en CORE voert het uit en brengt het resultaat terug

[Verbind je eerste app](https://docs.getcore.me/connectors)
