## Altijd waakzaam. Altijd klaar.

De meeste AI-tools wachten tot ze gevraagd worden. CORE kijkt mee.

Verbind het met je apps en het monitort activiteiten over al deze apps. Een e-mail komt binnen van een klant, een GitHub issue wordt toegewezen, een Sentry-melding wordt getriggerd, een vergadering eindigt. CORE ziet het, controleert je geheugen en de ingestelde vaardigheden, en handelt het af of geeft het door voor jouw oordeel. Jij activeert het niet. Het merkt het zelf op.

Installeer de CORE-plugin in Claude Code, Codex, of Cursor en je agentgesprekken worden ook bekeken. Context die besproken wordt, beslissingen die worden genomen, code die wordt geschreven—alles voedt het geheugen kennisgrafiek. De volgende taak die CORE oppakt weet al wat er in je vorige sessie gebeurde.

Wanneer het handelt, handelt het direct. Het kan e-mails beantwoorden, Linear-issues bijwerken, GitHub PR's indienen, Slack-berichten versturen, terminalcommando's uitvoeren, een browser aansturen, en een Claude Code of Codex sessie starten vanuit elke interface. Stuur een WhatsApp-bericht vanaf het vliegveld en CORE kan een codingsessie starten en een PR openen voordat je aan boord gaat.


Je bepaalt zelf waar het zelfstandig handelt en waar het wacht op jouw instructie. Handel Sentry-meldingen automatisch af, maar vraag altijd om bevestiging voordat er wordt gemerged. Keur een plan goed voordat een codeersessie begint. Vereis bevestiging voordat er een e-mail wordt verzonden. Het niveau van autonomie stel je zelf in, per taak, per app, per actie.

## Vier manieren om CORE te bereiken

Een chatbot heeft één interface. Een besturingssysteem heeft er veel.

**Stem.** Druk op Ctrl+Option op je Mac en zeg wat er moet gebeuren. CORE voert het uit op de achtergrond, zonder je workflow te onderbreken.

**Kladblok.** Open je dagpagina en schrijf `[ ] Los de authenticatiefout op uit issue #47`. CORE pikt het binnen 3 minuten op, laadt context uit je repo en geheugen en stelt een plan op.

**Berichten.** WhatsApp, Slack, Telegram. Stuur een taak vanaf het vliegveld, vanaf je telefoon, vanuit bed. CORE heeft je volledige context, ongeacht waar het bericht vandaan komt.

**Chat.** Open het dashboard en praat direct, zoals met elke assistent. Wanneer je een gesprek wil voordat je iets delegeert.

Eén AI, vier kanalen, overal dezelfde geheugen en context.

## Maak het persoonlijk

Geef het een naam. Kies hoe het spreekt. Stel de regels in die het overal volgt.

Kies uit vijf ingebouwde persoonlijkheden — de droge efficiëntie van TARS, de loyale formaliteit van Alfred, de warme praktische instelling van Hudson — of schrijf je eigen profiel. De persoonlijkheid komt terug in elke taak, elk plan, elk bericht dat het namens jou verstuurt.

Kies de stem voor gesproken interacties. Vanaf dat moment klinkt en gedraagt het zich als de AI die jij hebt ingesteld, op elk platform.

## CORE in de praktijk

### Zeg het, kom terug bij een PR.

*Druk op Ctrl+Option, spreek:* "Los de raceconditie op in de checkoutflow uit issue #312."

CORE laadt het issue, haalt gerelateerde commits en Slack-threads op, maakt een plan en start een Claude Code-sessie. Jij komt terug bij een diff. Je zat nooit aan je bureau.

### Schrijf het ‘s avonds, beoordeel het ‘s ochtends.

*Notitieblok:* `[ ] Werk het achterstallige werk van vanavond weg vanaf 23:00 uur`

CORE haalt op uit Linear, GitHub en geheugen, prioriteert, en werkt het af terwijl je slaapt. Soepele runs wachten op je review. Vastgelopen sessies komen terug met één scherpe vraag, niet een vastgelopen tabblad.

### Stuur het een bericht vanaf elke plek.

*WhatsApp vanaf het vliegveld:* "Ship de auth refactor."

CORE kent de branch, de context en jouw voorkeuren al. Het draait op Railway. Het start de sessie voordat je aan boord gaat.

### Onderzoek meldingen voordat het incidenten worden.

*Sentry meldt zich om 2 uur 's nachts.* CORE onderzoekt, haalt gerelateerde traces en eerdere incidenten uit het geheugen, stelt een oplossing voor en pingt je op Slack: "Issue #847, oplossing voorgesteld, wacht op je review." Je keurt goed vanaf je telefoon.

### Ontvang een briefing die jouw week al kent.

*Terugkerende taak, elke ochtend om 8 uur.* CORE haalt uit e-mail, GitHub, Linear en Slack, toont wat echt aandacht nodig heeft, slaat over wat niet nodig is, en zet opvolgingen automatisch om in taken.

| | |
|---|---|
| **Geheugen** | Tijdelijke kennisgrafiek over elk hulpmiddel en gesprek. Voorkeuren, beslissingen, doelen en richtlijnen, zodat elke taak begint met geladen context. |
| **Taken** | Eenmalige of terugkerende werkunits met jouw specificatie, CORE's plan, live status en een speciale chatthread. Elke taak kan coding-, browser- of terminalsessies starten. |
| **Connectors** | 50+ apps via één MCP-endpoint, plus webhook-triggers voor proactieve automatisering. GitHub, Linear, Jira, Slack, Gmail, Agenda, Sentry, Notion, Todoist en meer. |
| **Vaardigheden** | Herbruikbare instructies die automatisch afgaan op basis van context. Bijvoorbeeld: "haal altijd gerelateerde Linear-issues op voordat je een oplossing plant," "voer tests uit vóór het openen van een PR," of "plaats een Slack-samenvatting als een taak klaar is." 100+ ingebouwd, of schrijf je eigen. |
| **Gateway** | Draait Claude Code, Codex, browseragents en terminalcommando's op je machine of in Docker en Railway, zodat CORE blijft werken als je laptop dicht is. |
| **Modelagnostisch** | Gebruik je eigen provider: Anthropic, OpenAI of open-weight modellen. Host de volledige stack zelf voor volledige isolatie. |

## Snelstart

Open source en zelf gehost. Uw data blijft binnen uw infrastructuur.

**Kies uw pad:**

| Ik wil... | Hoe |
|---|---|
| Het op mijn computer proberen | Voer de onderstaande één-stap installatie uit (vereist Docker) |
| Implementeren op een server of VPS | Eén-klik Railway implementatie |
| De Mac-app gebruiken | [Word lid van de wachtlijst](https://www.getcore.me/) |

**Installeer en start CORE:**

```bash
npm install -g @redplanethq/corebrain && corebrain setup
```

De installatie-assistent vraagt om een installatiemap, AI-provider, API-sleutel en chatmodel. Het genereert geheimen, start de stack en opent `http://localhost:3033`.

De meeste lokale installaties duren enkele minuten zodra Docker draait.

**Of implementeer op Railway:**

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.com/deploy/core)

**Verbind een gateway** zodat CORE codeeragents kan uitvoeren, je browser kan aansturen en toegang heeft tot lokale mappen:

```bash
corebrain login
corebrain gateway setup
```

**Vereisten:** Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

[Volledige zelf-hosting gids](https://docs.getcore.me/self-hosting/setup)

**Je eerste taak (2 minuten na installatie):**

1. Open de **Scratchpad** (je dagelijkse pagina op `http://localhost:3033`)
2. Typ `[ ] Vat mijn open GitHub issues samen` of een taak die je normaal zelf zou doen
3. CORE pikt dit binnen 3 minuten op, verzamelt context uit gekoppelde apps, en stelt een plan op
4. Keur het plan goed en CORE voert het uit en brengt het resultaat terug

[Verbind je eerste app](https://docs.getcore.me/connectors)
