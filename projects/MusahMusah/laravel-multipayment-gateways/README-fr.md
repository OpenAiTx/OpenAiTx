Le package `laravel-multipayment-gateways` offre un moyen pratique de gérer les paiements via plusieurs passerelles de paiement dans une **application Laravel 8, 9 et 10**.  
Le package supporte actuellement plusieurs passerelles telles que **Paystack**, **Flutterwave** et **Stripe**.  
Le package propose une interface facile à utiliser qui abstrait les complexités liées à l’intégration avec ces passerelles de paiement.  
Il fournit également un moyen de gérer les webhooks provenant des passerelles de paiement.
