# یک پکیج لاراول که پیاده‌سازی درگاه‌های پرداخت چندگانه و نقاط انتهایی و وب‌هوک‌ها را بی‌دردسر می‌کند

پکیج `laravel-multipayment-gateways` راهی آسان برای مدیریت پرداخت‌ها از طریق چندین درگاه پرداخت در **برنامه‌های Laravel 11، 12 و 13** فراهم می‌کند.
این پکیج در حال حاضر از درگاه‌های مختلفی مانند **Paystack**، **Flutterwave** و **Stripe** پشتیبانی می‌کند.
پکیج رابط کاربری آسانی ارائه می‌دهد که پیچیدگی‌های یکپارچه‌سازی با این درگاه‌های پرداخت را انتزاع می‌کند.
همچنین راهکاری برای مدیریت وب‌هوک‌ها از درگاه‌های پرداخت فراهم می‌کند.

## سازگاری نسخه

| پکیج   | لاراول   | PHP  |
|--------|----------|------|
| 1.x    | 8–12     | 8.0+ |
| 2.x    | 11–13    | 8.2+ |

> **نسخه v2.x در حال حاضر بتا است** (انتشار پایدار: ۱۷ مارس ۲۰۲۶). برای نصب نسخه بتا:
> ```bash
> composer require musahmusah/laravel-multipayment-gateways:^2.0@beta
> ```

> اگر از لاراول ۱۰ یا پایین‌تر استفاده می‌کنید، نسخه قبلی اصلی را نصب کنید:
> `composer require musahmusah/laravel-multipayment-gateways:^1.0`
