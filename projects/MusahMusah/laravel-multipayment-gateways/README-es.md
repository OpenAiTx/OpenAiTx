El paquete `laravel-multipayment-gateways` proporciona una forma conveniente de manejar pagos a través de múltiples pasarelas de pago en una **aplicación Laravel 8, 9 y 10**.  
Actualmente, el paquete soporta varias pasarelas como **Paystack**, **Flutterwave** y **Stripe**.  
El paquete ofrece una interfaz fácil de usar que abstrae las complejidades de integrar estas pasarelas de pago.  
También proporciona una forma de manejar webhooks desde las pasarelas de pago.
