## Installation and Running

1. Clone the project

```bash
git clone https://github.com/Lily-404/blog.git
cd jimmy-blog
```

2. Install Dependencies

```bash
npm install
```

3. Running the Development Server

```bash
npm run dev
```

4. Building the Production Version

```bash
npm run build
```
