Permet à Android Auto d’afficher n’importe quelle application, solution emboîtée basée sur VirtualDisplay, Star! Star! Star!

Support uniquement A12+, nécessite LSPosed, certains ROM peuvent provoquer des plantages, merci d’avoir les connaissances nécessaires pour récupérer votre appareil.

Théoriquement supporte jusqu’à A10+, mais faute d’appareils et de ressources pour maintenir, ce n’est pas supporté, contributions bienvenues.

Veuillez ne pas soumettre de issues pour des problèmes sans lien avec ce projet, c’est impératif ! Par exemple : installation de Magisk/LSPosed, connexion ou utilisation d’AA, etc.

## Méthode d’utilisation
- Activez ce module dans LSPosed, cochez System Framework et Android Auto, installez votre lanceur préféré, et saisissez le nom du paquet dans les réglages du module.
- Il est conseillé de régler le DPI, cela améliore les applications utilisant le contexte d’application pour dessiner l’interface.
- Ajoutez les paramètres Properties relatifs à AA, pour modifier par hook la configuration d’AA, supporte seulement les types de base.
- Contournement automatique de la vérification de la source d’installation de l’application AA par Android Auto.
- Les permissions root ne sont utilisées que pour l’exécution des commandes Shell configurées par l’utilisateur, pas obligatoire si non nécessaire.
