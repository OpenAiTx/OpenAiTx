## Sitios
* [ModTheSpire Wiki](https://github.com/kiooeht/ModTheSpire/wiki)<br>
<b>ModTheSpire</b> (abreviado MTS) es una herramienta que permite cargar mods externos para Slay the Spire sin modificar los archivos base del juego, y permite que los mods parcheen su propio código dentro del juego.<br>
La Wiki de MTS explica cómo realizar guardados globales, patches, entre otros.

* [BaseMod Wiki](https://github.com/daviscook477/BaseMod/wiki)<br>
<b>BaseMod</b> es la API base para mods, que facilita a los creadores agregar sus propias cartas y otros contenidos al juego y gestionarlos centralizadamente.<br>
La Wiki incluye herramientas muy útiles como registro automático de todas las cartas (AutoAdd), modificadores de cartas (CardModifier), guardado dentro de la partida (CustomSavable), etc. También contiene tutoriales de creación de mods escritos por el autor de BaseMod.

## Herramientas
* [JD-GUI](http://java-decompiler.github.io/)<br>
Una herramienta de descompilación Java con interfaz gráfica.<br>
Permite ver el código fuente reestructurado del juego u otros mods para facilitar ~~copiar~~ aprender de otros códigos.<br>
También puede usarse para consultar las líneas necesarias para hacer un patch (el descompilador integrado en IDEA no es preciso).

* [sts裁图器](https://github.com/JohnnyBazooka89/StSModdingToolCardImagesCreator)<br>

Recorta la imagen al tamaño y forma que necesita la tarjeta Spire.<br>
No he usado esto, pero en el grupo hay usuarios que han creado otra herramienta con la misma función.

## Animación
* [DragonBones](https://dragonbones.github.io/cn/index.html)<br>
Software que puede exportar animaciones Spine, generalmente este es suficiente, se puede buscar versiones disponibles.

*Crear animaciones requiere cierta base, pero en realidad la mayoría de los mods solo necesitan una imagen.*
* [Spine](http://zh.esotericsoftware.com/)<br>
Software de animación 2D usado por Spire.
