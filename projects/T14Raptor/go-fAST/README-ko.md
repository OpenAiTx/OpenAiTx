# go-fAST

`go-fAST`는 자바스크립트 추상 구문 트리(Abstract Syntax Trees, AST)를 파싱, 변환 및 생성하기 위해 설계된 초고속 Golang 라이브러리입니다. 이 라이브러리는 Go에서 자바스크립트 AST를 간단하고 효율적으로 다룰 수 있게 하여, 개발자들이 코드 분석 및 (디)오브퓨케이션 등 다양한 작업을 수행할 수 있도록 합니다.

[![Discord Banner2](https://discord.com/api/guilds/1288989871498858496/widget.png?style=banner2)](https://discord.gg/wdJ4VcdrwK)

## 기능

- **파싱**: 자바스크립트 코드를 AST 표현으로 변환합니다.
- **변환**: 자바스크립트 AST에 변환을 적용하여 코드 구조를 수정합니다.
- **생성**: AST로부터 자바스크립트 코드를 생성합니다.

## 설치

`go-fAST`를 Go 프로젝트에서 사용하려면 `go get`을 사용해 설치해야 합니다:

```sh
go get github.com/t14raptor/go-fast
```
