# go-fAST

`go-fAST` は、JavaScriptの抽象構文木（AST）を解析、変換、生成するために設計された超高速のGolangライブラリです。このライブラリは、GoでJavaScriptのASTを扱うためのシンプルかつ効率的な方法を提供し、コード解析や（逆）難読化など様々なタスクを開発者が実行できるようにします。

[![Discord Banner2](https://discord.com/api/guilds/1288989871498858496/widget.png?style=banner2)](https://discord.gg/wdJ4VcdrwK)

## 特徴

- **解析**: JavaScriptコードをAST表現に変換します。
- **変換**: JavaScriptのASTに変換を適用してコード構造を修正します。
- **生成**: ASTからJavaScriptコードを生成します。

## インストール

`go-fAST` をGoプロジェクトで使用するには、`go get` を使ってインストールする必要があります:

```sh
go get github.com/t14raptor/go-fast
```
