# go-fAST

`go-fAST` es una biblioteca de Golang súper rápida diseñada para analizar, transformar y generar Árboles de Sintaxis Abstracta (AST) de JavaScript. Esta biblioteca proporciona una manera simple y eficiente de trabajar con ASTs de JavaScript en Go, permitiendo a los desarrolladores realizar una variedad de tareas que incluyen análisis de código y (de)ofuscación.

[![Discord Banner2](https://discord.com/api/guilds/1288989871498858496/widget.png?style=banner2)](https://discord.gg/wdJ4VcdrwK)

## Características

- **Análisis**: Convierte código JavaScript en su representación AST.
- **Transformación**: Aplica transformaciones a ASTs de JavaScript para modificar estructuras de código.
- **Generación**: Genera código JavaScript a partir de ASTs.

## Instalación

Para usar `go-fAST` en tu proyecto Go, necesitas instalarlo usando `go get`:

```sh
go get github.com/t14raptor/go-fast
```
