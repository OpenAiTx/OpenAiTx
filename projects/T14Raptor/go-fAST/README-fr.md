# go-fAST

`go-fAST` est une bibliothèque Golang ultra-rapide conçue pour analyser, transformer et générer des arbres de syntaxe abstraite (AST) JavaScript. Cette bibliothèque offre un moyen simple et efficace de travailler avec les AST JavaScript en Go, permettant aux développeurs d'effectuer diverses tâches incluant l'analyse de code et la (dé)obfuscation.

[![Discord Banner2](https://discord.com/api/guilds/1288989871498858496/widget.png?style=banner2)](https://discord.gg/wdJ4VcdrwK)

## Fonctionnalités

- **Analyse** : Convertir du code JavaScript en sa représentation AST.
- **Transformation** : Appliquer des transformations aux AST JavaScript pour modifier les structures de code.
- **Génération** : Générer du code JavaScript à partir des AST.

## Installation

Pour utiliser `go-fAST` dans votre projet Go, vous devez l'installer avec `go get` :

```sh
go get github.com/t14raptor/go-fast
```
