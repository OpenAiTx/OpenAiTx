# Gotify Tray


一个用于接收来自 [Gotify 服务器](https://github.com/gotify/server) 消息的托盘通知应用。


## 快速开始


- [下载最新版本。](https://github.com/seird/gotify-tray/releases/latest)

- 或者，通过 pip 安装：
    ```shell
    $ pip install gotify-tray
    ```
- 或者，从源代码运行：

    ```shell
    $ pip install -r requirements.txt
    $ python -m gotify_tray
    ```
## 功能

- 在本地通知区域接收 gotify 消息。
- 从休眠唤醒或网络连接丢失后重新连接。
- 禁用低优先级消息的通知横幅。
- 手动删除已接收的消息。
- 浏览所有以前接收过的消息历史。
- 在网络连接丢失后接收错过的消息。
