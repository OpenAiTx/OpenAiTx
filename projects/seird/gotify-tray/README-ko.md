# Gotify 트레이


[Gotify 서버](https://github.com/gotify/server)로부터 메시지를 받기 위한 트레이 알림 애플리케이션입니다.


## 시작하기


- [최신 버전을 다운로드하세요.](https://github.com/seird/gotify-tray/releases/latest)

- 또는, pip를 통해 설치하세요:
    ```shell
    $ pip install gotify-tray
    ```
- 또는 소스에서 실행:

    ```shell
    $ pip install -r requirements.txt
    $ python -m gotify_tray
    ```
## 기능

- 네이티브 알림 영역에서 gotify 메시지 수신.
- 절전 모드 해제 또는 네트워크 연결 손실 후 재연결.
- 낮은 우선순위 메시지에 대해 알림 배너 비활성화.
- 수신한 메시지를 수동으로 삭제.
- 이전에 수신한 모든 메시지의 기록 확인.
- 네트워크 연결 손실 후 놓친 메시지 수신.
