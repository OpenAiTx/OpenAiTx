# Gotify Tray


Une application de notification en barre pour recevoir des messages d'un [serveur Gotify](https://github.com/gotify/server).


## Pour commencer


- [Téléchargez la dernière version.](https://github.com/seird/gotify-tray/releases/latest)

- ou, installez via pip :
    ```shell
    $ pip install gotify-tray
    ```
- ou, exécutez depuis la source :

    ```shell
    $ pip install -r requirements.txt
    $ python -m gotify_tray
    ```
## Fonctionnalités

- Recevoir les messages Gotify dans la zone de notification native.  
- Reconnexion après une sortie de veille ou une perte de connexion réseau.  
- Désactiver les bannières de notification pour les messages à faible priorité.  
- Supprimer manuellement les messages reçus.  
- Parcourir l'historique de tous les messages précédemment reçus.  
- Recevoir les messages manqués après une perte de connexion réseau.
