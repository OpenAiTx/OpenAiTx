# Gotify Tray


Una aplicación de notificación en la bandeja para recibir mensajes de un [servidor Gotify](https://github.com/gotify/server).


## Comenzando


- [Descarga la última versión.](https://github.com/seird/gotify-tray/releases/latest)

- o, instala vía pip:
    ```shell
    $ pip install gotify-tray
    ```
- o, ejecutar desde el código fuente:

    ```shell
    $ pip install -r requirements.txt
    $ python -m gotify_tray
    ```
## Características

- Recibir mensajes de gotify en el área de notificaciones nativa.
- Reconectarse después de despertar del modo de suspensión o perder la conexión de red.
- Desactivar los banners de notificación para mensajes de baja prioridad.
- Eliminar manualmente los mensajes recibidos.
- Revisar un historial de todos los mensajes recibidos anteriormente.
- Recibir mensajes perdidos después de perder la conexión de red.
