## 🚀 脚本功能
- **物品、NPC及宝箱ESP**：穿墙查看所有珍贵资源、敌人和战利品，获得极致感知。
- **精准瞄准辅助**：按住右键自动锁定并消灭任何威胁。
- **瞬间传送**：瞬间传送至篝火、研磨机、迷失儿童、宝箱及所有主要物品。
- **完整飞行模式**：使用WASD、空格和Shift键自由飞行全地图（按Q切换）。
- **自动采树与木材收集**：通过高级自动采集系统自动完成繁琐任务。
- **可调速度**：通过自定义速度倍率更快穿越地图。
- **防死亡机制**：当生命值极低时自动传送至安全地点。
- **无雾与夜视**：关闭环境雾气，昼夜获得清晰视野。
- **自动攻击 / 杀戮光环**：自动击败靠近的敌人。
