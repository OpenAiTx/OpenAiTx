## 🚀 スクリプトの特徴
- **アイテム、NPC、チェストESP**: 壁の内側にあるすべての貴重な資源、敵、戦利品を確認して、最高の状況認識を実現。
- **精密エイムボット**: 右クリックを押し続けると自動的にターゲットをロックし、脅威を排除。
- **即時テレポート**: キャンプファイヤー、グラインダー、迷子の子供、チェスト、主要アイテムすべてに瞬時にテレポート。
- **フルフライモード**: WASD、スペース、シフトでマップ上を自由に飛行（Qで切替）。
- **自動木材伐採＆収集**: 高度な自動収集システムで最も面倒な作業を自動化。
- **速度調整可能**: カスタマイズ可能な速度倍率で、これまで以上に速く移動可能。
- **アンチデス機能**: 体力が危険なほど低下した際に自動的に安全な場所へテレポート。
- **霧なし＆ナイトビジョン**: 環境霧をオフにして、昼夜を問わずクリアな視界を獲得。
- **自動攻撃／キルオーラ**: 近づいてきた敵を自動的に撃破。

## 📖 このスクリプトの使い方
1.  まず、Robloxスクリプト実行環境が必要です。シンプルさでDecryptorを推奨します。
2.  GUIが簡単に作られているので、そこでスクリプトを簡単に実行できます。
3.  実行後、「99 Nights in the Forest」のRobloxゲームを開いてください。
4.  DecryptorのUIが表示され、すべての機能を設定できるようになります。
