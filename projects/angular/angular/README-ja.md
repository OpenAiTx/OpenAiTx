## 開発環境のセットアップ

### 前提条件

- [Node.js]（[Node Package Manager][npm] を含む）をインストールしてください

### プロジェクトのセットアップ

Angular CLIをグローバルにインストール:

```
npm install -g @angular/cli
```

ワークスペースの作成:

```
ng new [PROJECT NAME]
```

アプリケーションの実行:

```
cd [PROJECT NAME]
ng serve
```

Angularはクロスプラットフォーム、高速、スケーラブル、優れたツール群を持ち、何百万人もの開発者に愛されています。
