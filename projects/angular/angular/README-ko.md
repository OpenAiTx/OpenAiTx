## 개발 환경 설정

### 필수 조건

- [Node.js] 및 [Node Package Manager][npm] 설치

### 프로젝트 설정

Angular CLI를 전역으로 설치하세요:

```
npm install -g @angular/cli
```

워크스페이스 생성:

```
ng new [PROJECT NAME]
```

애플리케이션 실행:

```
cd [PROJECT NAME]
ng serve
```

Angular는 크로스 플랫폼, 빠름, 확장 가능, 훌륭한 도구 지원, 수백만 명이 사랑하는 프레임워크입니다.
