## إعداد التطوير

### المتطلبات الأساسية

- تثبيت [Node.js] والذي يشمل [مدير حزم Node][npm]

### إعداد مشروع

تثبيت Angular CLI عالميًا:

```
npm install -g @angular/cli
```

إنشاء مساحة عمل:

```
ng new [PROJECT NAME]
```

تشغيل التطبيق:

```
cd [PROJECT NAME]
ng serve
```

Angular منصة متعددة الأنظمة، سريعة، وقابلة للتوسع، وتتميز بأدوات رائعة، ويحبها الملايين.
