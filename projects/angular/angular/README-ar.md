## إعداد بيئة التطوير

### المتطلبات المسبقة

- تثبيت [Node.js] الذي يشمل [مدير حزم Node][npm]

### إعداد مشروع

قم بتثبيت واجهة Angular CLI بشكل عالمي:

```
npm install -g @angular/cli
```

إنشاء مساحة عمل:

```
ng new [اسم المشروع]
```

تشغيل التطبيق:

```
cd [اسم المشروع]
ng serve
```

Angular منصة متعددة الأنظمة، سريعة، وقابلة للتوسع، وتتميز بأدوات مذهلة ويحبها الملايين.
