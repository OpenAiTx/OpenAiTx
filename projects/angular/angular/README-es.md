## Configuración de Desarrollo

### Requisitos Previos

- Instala [Node.js] que incluye el [Node Package Manager][npm]

### Configuración de un Proyecto

Instala el Angular CLI globalmente:

```
npm install -g @angular/cli
```

Crea un espacio de trabajo:

```
ng new [NOMBRE_DEL_PROYECTO]
```

Ejecuta la aplicación:

```
cd [NOMBRE_DEL_PROYECTO]
ng serve
```

Angular es multiplataforma, rápido, escalable, tiene herramientas increíbles y es amado por millones.
