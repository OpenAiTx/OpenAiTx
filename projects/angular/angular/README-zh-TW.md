## 開發環境設置

### 先決條件

- 安裝 [Node.js]（包含 [Node 套件管理員][npm]）

### 建立專案

全域安裝 Angular CLI：

```
npm install -g @angular/cli
```

建立工作區：

```
ng new [PROJECT NAME]
```

運行應用程式：

```
cd [PROJECT NAME]
ng serve
```
