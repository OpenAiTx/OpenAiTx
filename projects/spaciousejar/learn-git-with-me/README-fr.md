## ⭐ Fonctionnalités

### 🐭 Frameworks

- **[Next.js](https://nextjs.org/)** - Le framework React pour le Web (avec **App Directory**)

### 🐮 Plateformes

- **[Vercel](https://vercel.com/)** – Déployez facilement votre application Next.js


### 🐒 Interface utilisateur

- **[Tailwind CSS](https://tailwindcss.com/)** – Framework CSS utilitaire pour un développement rapide de l’interface utilisateur
- **[Shadcn/ui](https://ui.shadcn.com/)** – Composants réutilisables construits avec Radix UI et Tailwind CSS
- **[next/font](https://nextjs.org/docs/basic-features/font-optimization)** – Optimisez les polices personnalisées et supprimez les requêtes réseau externes pour une meilleure performance
- **[MDX](https://mdxjs.com/docs)** – Markdown pour les composants
  
### 🐴 Qualité du code

- **[TypeScript](https://www.typescriptlang.org/)** – Vérificateur de types statique pour une sécurité de type de bout en bout
- **[Prettier](https://prettier.io/)** – Formatteur de code opinionné pour un style de code cohérent
- **[ESLint](https://eslint.org/)** – Linter extensible pour Next.js et TypeScript


### 🐑 Performance

- **[Vercel Analytics](https://vercel.com/analytics)** – Mesures de performance en temps réel pour votre application Next.js


## 📦 Applications et Packages

- `web` : L'application principale Next.js
- `UI` : Composants UI partagés
