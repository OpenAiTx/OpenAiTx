## ⭐ Características

### 🐭 Frameworks

- **[Next.js](https://nextjs.org/)** - El framework React para la web (con **Directorio App**)

### 🐮 Plataformas

- **[Vercel](https://vercel.com/)** – Despliega tu app Next.js con facilidad


### 🐒 UI

- **[Tailwind CSS](https://tailwindcss.com/)** – Framework CSS utilitario para desarrollo rápido de interfaces
- **[Shadcn/ui](https://ui.shadcn.com/)** – Componentes reutilizables construidos con Radix UI y Tailwind CSS
- **[next/font](https://nextjs.org/docs/basic-features/font-optimization)** – Optimiza fuentes personalizadas y elimina solicitudes externas para mejorar el rendimiento
- **[MDX](https://mdxjs.com/docs)** – Markdown para los componentes
  
### 🐴 Calidad de Código

- **[TypeScript](https://www.typescriptlang.org/)** – Verificador estático de tipos para seguridad de tipos de extremo a extremo
- **[Prettier](https://prettier.io/)** – Formateador de código con opiniones para estilo de código consistente
- **[ESLint](https://eslint.org/)** – Linter enchufable para Next.js y TypeScript


### 🐑 Rendimiento

- **[Vercel Analytics](https://vercel.com/analytics)** – Métricas de rendimiento en tiempo real para tu aplicación Next.js


## 📦 Aplicaciones y Paquetes

- `web`: La aplicación principal de Next.js
- `UI`: Componentes de interfaz de usuario compartidos
