## ⭐ 기능

### 🐭 프레임워크

- **[Next.js](https://nextjs.org/)** - 웹을 위한 React 프레임워크 (**앱 디렉터리** 포함)

### 🐮 플랫폼

- **[Vercel](https://vercel.com/)** – Next.js 앱을 쉽게 배포하세요


### 🐒 UI

- **[Tailwind CSS](https://tailwindcss.com/)** – 빠른 UI 개발을 위한 유틸리티 우선 CSS 프레임워크
- **[Shadcn/ui](https://ui.shadcn.com/)** – Radix UI와 Tailwind CSS를 사용하여 만든 재사용 가능한 컴포넌트
- **[next/font](https://nextjs.org/docs/basic-features/font-optimization)** – 커스텀 폰트를 최적화하고 외부 네트워크 요청을 제거하여 성능 향상
- **[MDX](https://mdxjs.com/docs)** – 컴포넌트를 위한 마크다운
  
### 🐴 코드 품질

- **[TypeScript](https://www.typescriptlang.org/)** – 종단 간 타입 안전성을 위한 정적 타입 검사기
- **[Prettier](https://prettier.io/)** – 일관된 코드 스타일을 위한 의견 기반 코드 포맷터
- **[ESLint](https://eslint.org/)** – Next.js 및 TypeScript용 플러그형 린터


### 🐑 성능

- **[Vercel Analytics](https://vercel.com/analytics)** – Next.js 앱을 위한 실시간 성능 지표



## 📦 앱 및 패키지

- `web`: 주요 Next.js 애플리케이션
- `UI`: 공유 UI 컴포넌트
