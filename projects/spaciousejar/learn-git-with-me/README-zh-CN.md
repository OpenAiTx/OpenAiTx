## ⭐ 功能

### 🐭 框架

- **[Next.js](https://nextjs.org/)** - Web 的 React 框架（带有 **App 目录**）

### 🐮 平台

- **[Vercel](https://vercel.com/)** – 轻松部署你的 Next.js 应用


### 🐒 界面

- **[Tailwind CSS](https://tailwindcss.com/)** – 实用优先的 CSS 框架，用于快速 UI 开发
- **[Shadcn/ui](https://ui.shadcn.com/)** – 基于 Radix UI 和 Tailwind CSS 构建的可复用组件
- **[next/font](https://nextjs.org/docs/basic-features/font-optimization)** – 优化自定义字体，移除外部网络请求以提升性能
- **[MDX](https://mdxjs.com/docs)** – 组件化的 Markdown
  
### 🐴 代码质量

- **[TypeScript](https://www.typescriptlang.org/)** – 静态类型检查器，实现端到端的类型安全
- **[Prettier](https://prettier.io/)** – 有明确风格的代码格式化工具，保持代码风格一致
- **[ESLint](https://eslint.org/)** – 适用于 Next.js 和 TypeScript 的可插拔代码检查工具


### 🐑 性能

- **[Vercel Analytics](https://vercel.com/analytics)** – 适用于您的 Next.js 应用的实时性能指标



## 📦 应用和包

- `web`：主要的 Next.js 应用
- `UI`：共享的 UI 组件
