# Puedes contribuir
## ¿Qué puedes contribuir?
- Se pueden aportar notas y exámenes relacionados con los __cursos del primer año__ del esquema 2022 de RVCE.
- El tamaño del archivo no debe superar los 100 MB. Si quieres agregar algún libro de texto, solo deben añadirse los capítulos relevantes a las unidades del curso.
## ¿Cómo contribuir?
- Si estás familiarizado con el uso de GitHub:
  + Si deseas agregar materiales a un curso que no está en este repositorio, crea una carpeta y envía una solicitud de push.
  + De lo contrario, solo añade archivos a la carpeta del curso y envía una solicitud de push.
  
- Si no estás familiarizado con el uso de GitHub, por favor consulta el [manual de instrucciones](https://github.com/AnanthMAthreya/HowToContribute)
- Si no quieres usar GitHub, por favor llena este formulario [Enlace al formulario](https://forms.gle/GR2N4dKWF2NwvwiK8)

  __Recibirás créditos.__

## Libro de Texto BS Grewal
Este repositorio también contiene algunos capítulos seleccionados para todos los cursos de matemáticas del primer año, del libro de texto [Higher Engineering Mathematics](https://github.com/AnanthMAthreya/1st-year-resources-2022-scheme-rvce/tree/main/BS%20Grewal%20Math%20textbook) del Dr. BS Grewal. Esos capítulos son:
1) Álgebra Lineal
2) Cálculo Diferencial
3) Diferenciación Parcial
4) Integrales Múltiples
5) Cálculo Vectorial
6) Ecuaciones Diferenciales Lineales
7) Aplicaciones de EDL
8) Ecuaciones Diferenciales Parciales
9) Aplicaciones de EDP
10) Transformadas de Laplace
11) Métodos Estadísticos
12) Métodos Numéricos
13) Diferencias finitas e interpolación
14) Diferenciación e integración numérica
15) Solución numérica de EDO
