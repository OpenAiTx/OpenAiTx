# Dosidicus electronicae
### ニューラルネットワークを備えたデジタルペット【研究プロジェクト】
* ニューラルネットワーク、ヘッブ学習、神経新生の可視化と理解のためのツールを含みます

* `PyQt5` と `numpy` が必要です



![image](https://github.com/user-attachments/assets/5a6449c8-e138-42aa-9acf-d9bd9b46d6e4)



### 自律行動:

* イカのニューラルネットワークは現在の状態（空腹、眠気など）に基づき自律的に意思決定を行います。
* 食物検出のためのビジョンコーンを実装し、現実的な採餌行動をシミュレートします。
* ヘッブ学習アルゴリズムが30秒ごとに実行されます（設定可能）
* 短期記憶と長期記憶の経験が意思決定に影響します
* イカは環境に応じて新しいニューロンを作り出すことができます（神経新生）

すべてをドキュメント化しようとしています！
[[https://github.com/ViciousSquid/Dosidicus/wiki](https://github.com/ViciousSquid/Dosidicus/wiki)]

### ニーズ管理システム:

* 空腹、眠気、幸福度、清潔さなど様々なニーズを追跡します。
* ニーズは時間とともに変化し、ペットの健康や行動に影響します。
* イカはニーズを無視すると病気になったり死んだりすることがあります。

### 性格システム

* 行動に影響を与える7つの異なる[性格タイプ](https://github.com/ViciousSquid/Dosidicus/blob/main/Docs/Personalities.md)

### 飾り付けとカスタマイズ！

* イカが触れ合う環境に装飾を選んで配置できます！
