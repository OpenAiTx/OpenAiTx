## OpenVPN

📖 **[Instrucción completa para OpenVPN](https://github.com/ryderams/OpenVPN-ru)**

### Instalación

```bash
wget -O openvpn.sh https://get.vpnsetup.net/ovpn
sudo bash openvpn.sh --auto
```

### Gestión de clientes

**Crear un nuevo cliente** (por ejemplo, `client1`):
```bash
sudo bash openvpn.sh --addclient client1
```

**Eliminar cliente** (por ejemplo, `client1`):
```bash
sudo bash openvpn.sh --revokeclient client1
```
