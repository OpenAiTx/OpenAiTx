## Software para combinar o empaquetar texturas en canales de imagen, produciendo una imagen con hasta cuatro texturas.

este programa es ideal si quieres empaquetar texturas individuales en escala de grises en una sola imagen.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img1.jpg)

## Requisitos
Para ejecutar este programa necesitas [.NET Framework 4.8](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48) (instalado por defecto en la mayoría de los ordenadores con Windows)

## Uso
1. Descarga el software desde [releases](https://github.com/Fidifis/TextureMerge/releases) \
*Instálalo usando el instalador o extrae la carpeta zip*
2. Abre TextureMerge.exe
3. Haz clic en cargar bajo el canal de color en el que deseas insertar la textura.\
**También puedes *arrastrar y soltar* archivos en las ranuras.**

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img2.jpg)

4. Aparece el diálogo de Windows. Selecciona la textura y haz clic en abrir.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img3.jpg)

5. Repite para otras texturas. Los canales vacíos serán negros o puedes cambiarlos a blanco (el canal alfa no se añadirá si no se proporciona imagen).
6. Cuando termines haz clic en Combinar.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img4.jpg)

7. Aparece el diálogo de Windows. Introduce un nombre de archivo y haz clic en guardar.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img5.jpg)
