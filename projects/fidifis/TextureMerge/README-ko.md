## 텍스처를 병합하거나 이미지 채널에 패킹하여 최대 네 개의 텍스처를 하나의 이미지로 생성하는 소프트웨어입니다.

이 프로그램은 개별 그레이스케일 텍스처를 하나의 이미지에 패킹하려는 경우 이상적입니다.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img1.jpg)

## 요구사항
이 프로그램을 실행하려면 [.NET Framework 4.8](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48) 이 필요합니다 (대부분의 윈도우 컴퓨터에 기본 설치되어 있음).

## 사용법
1. [릴리즈](https://github.com/Fidifis/TextureMerge/releases)에서 소프트웨어를 다운로드 하세요. \
*설치 프로그램을 사용하거나 zip 폴더를 추출하여 설치하세요.*
2. TextureMerge.exe를 실행하세요.
3. 텍스처를 삽입하려는 컬러 채널 아래의 로드 버튼을 클릭하세요.\
**슬롯에 파일을 *드래그 앤 드롭*할 수도 있습니다.**

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img2.jpg)

4. 윈도우 대화 상자가 나타납니다. 텍스처를 선택하고 열기를 클릭하세요.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img3.jpg)

5. 다른 텍스처에 대해서도 반복하세요. 빈 채널은 검정색이 되며 흰색으로 변경할 수도 있습니다 (이미지가 제공되지 않으면 알파 채널은 추가되지 않습니다).
6. 완료되면 병합 버튼을 클릭하세요.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img4.jpg)

7. 윈도우 대화 상자가 나타납니다. 파일 이름을 입력하고 저장을 클릭하세요.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img5.jpg)
