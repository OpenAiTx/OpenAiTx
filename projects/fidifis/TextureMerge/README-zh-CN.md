## 用于将纹理合并或打包到图像通道中的软件，生成包含最多四个纹理的单张图像。

该程序非常适合将单独的灰度纹理打包到一张图像中。

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img1.jpg)

## 要求
运行此程序需要 [.NET Framework 4.8](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48)（大多数 Windows 电脑默认已安装）

## 使用方法
1. 从[发布页面](https://github.com/Fidifis/TextureMerge/releases)下载软件 \
*使用安装程序安装或解压缩 ZIP 文件夹*
2. 打开 TextureMerge.exe
3. 点击你想插入纹理的颜色通道下的加载按钮。\
**你也可以将文件*拖放*到对应槽位。**

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img2.jpg)

4. Windows 对话框弹出。选择纹理并点击打开。

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img3.jpg)

5. 对其他纹理重复以上操作。空白通道将显示为黑色，也可以更改为白色（如果未提供图像，Alpha 通道将不会被添加）。
6. 完成后点击合并。

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img4.jpg)

7. Windows 对话框弹出。输入文件名并点击保存。

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img5.jpg)
