## Logiciel pour fusionner ou empaqueter des textures dans les canaux d'image, produisant une image avec jusqu'à quatre textures.

ce programme est idéal si vous souhaitez empaqueter des textures individuelles en niveaux de gris dans une seule image.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img1.jpg)

## Exigences
Pour exécuter ce programme, vous avez besoin de [.NET Framework 4.8](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48) (installé par défaut sur la plupart des ordinateurs Windows)

## Utilisation
1. Téléchargez le logiciel depuis les [releases](https://github.com/Fidifis/TextureMerge/releases) \
*Installez-le via le programme d'installation ou extrayez le dossier zip*
2. Ouvrez TextureMerge.exe
3. Cliquez sur charger sous le canal de couleur dans lequel vous souhaitez insérer la texture.\
**Vous pouvez également *glisser-déposer* des fichiers dans les emplacements.**

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img2.jpg)

4. La fenêtre de dialogue Windows apparaît. Sélectionnez la texture et cliquez sur ouvrir.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img3.jpg)

5. Répétez pour les autres textures. Les canaux vides seront noirs ou vous pouvez les changer en blanc (le canal alpha ne sera pas ajouté si aucune image n’est fournie).
6. Lorsque vous avez terminé, cliquez sur Fusionner.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img4.jpg)

7. La fenêtre de dialogue Windows apparaît. Entrez un nom de fichier et cliquez sur enregistrer.

![image](https://github.com/Fidifis/TextureMerge/raw/master/Tutorial-images/img5.jpg)
