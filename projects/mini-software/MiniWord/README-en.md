## Introduction

MiniWord is an easy and effective .NET Word Template library.

![image](https://user-images.githubusercontent.com/12729184/190835307-6cd87982-b5f3-4a79-9682-bdd1cc02a4ea.png)



## Getting Started

### Installation

- nuget link : https://www.nuget.org/packages/MiniWord

### Quick Start

Templates follow the "WHAT you see is what you get" design, and the template tag styles are fully preserved.

```csharp
var value = new Dictionary<string, object>(){["title"] = "Hello MiniWord"};
MiniSoftware.MiniWord.SaveAsByTemplate(outputPath, templatePath, value);
```

![image](https://user-images.githubusercontent.com/12729184/190875707-6c5639ab-9518-4dc1-85d8-81e20af465e8.png)

### Input, Output

- Input supports file path, byte[]
- Output supports file path, byte[], stream

```csharp
SaveAsByTemplate(string path, string templatePath, Dictionary<string, object> value)
SaveAsByTemplate(string path, byte[] templateBytes, Dictionary<string, object> value)
SaveAsByTemplate(this Stream stream, string templatePath, Dictionary<string, object> value)
SaveAsByTemplate(this Stream stream, byte[] templateBytes, Dictionary<string, object> value)
```



## Tags

The MiniWord template format string is similar to Vue, React `{{tag}}`. Users just need to ensure that the tag and the value parameter key are the same, then the system will automatically replace them.

### Text

```csharp
{{tag}}
```

##### Example

```csharp
var value = new Dictionary<string, object>()
{
    ["Name"] = "Jack",
    ["Department"] = "IT Department",
    ["Purpose"] = "Shanghai site needs a new system to control HR system.",
    ["StartDate"] = DateTime.Parse("2022-09-07 08:30:00"),
    ["EndDate"] = DateTime.Parse("2022-09-15 15:30:00"),
    ["Approved"] = true,
    ["Total_Amount"] = 123456,
};
MiniWord.SaveAsByTemplate(path, templatePath, value);
```

        };

        var templatePath = Path.Combine(Directory.GetCurrentDirectory(), "Templates", "template.docx");
        var outputPath = Path.Combine(Directory.GetCurrentDirectory(), "Output", "result.docx");

        MiniWord.SaveAsByTemplate(outputPath, templatePath, value);

        var stream = new FileStream(outputPath, FileMode.Open, FileAccess.Read);

## Document Download

If you want to provide document download functionality in your web application, you can use the following code:

```csharp
