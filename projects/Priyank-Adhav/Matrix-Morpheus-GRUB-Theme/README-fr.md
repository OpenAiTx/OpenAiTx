# Thème GRUB Matrix Morpheus  
**Pilule Rouge vs Pilule Bleue**  

Un thème GRUB minimaliste inspiré de Matrix avec des arrière-plans dynamiques plein écran changeant entre Linux et Windows.  

---  

**Note :**  
Actuellement, le thème inclut uniquement les icônes **Arch Linux** et **Windows**.  

De plus, bien que les icônes soient **disposées horizontalement** à l'écran,  
vous naviguez toujours avec les **flèches Haut** et **Bas** comme dans un menu GRUB normal.

## Installation  

1. Cloner le dépôt

```shell
git clone https://github.com/Priyank-Adhav/Matrix-Morpheus-GRUB-Theme
```
2. Allez dans le dossier 


```shell
cd Matrix-Morpheus-GRUB-Theme
```

3. Rendre l'installateur exécutable

```shell
chmod +x install.sh
```

4. Exécutez le script d'installation en tant qu'administrateur

```shell
sudo ./install.sh
```

5. Redémarrez pour tester votre nouveau thème

---
Optionnel : Simplifiez votre menu GRUB

J'ai conçu ce thème pour une disposition à deux entrées et je n'ai pas vraiment réfléchi à la manière de gérer visuellement les entrées supplémentaires.

Si votre menu GRUB contient actuellement des entrées supplémentaires telles que :

- « Options avancées pour Arch Linux »
- « Paramètres du firmware UEFI »

Je vous recommande de supprimer les entrées supplémentaires du menu dans la configuration de grub si vous ne les utilisez pas.
