# 매트릭스 모르피어스 GRUB 테마
**레드 필 vs 블루 필**

리눅스와 윈도우 사이를 전환하는 전체 화면 동적 배경을 특징으로 하는 미니멀리스트 매트릭스 영감 GRUB 테마입니다.

---

**참고:**  
현재 테마는 **Arch Linux**와 **Windows** 아이콘만 포함하고 있습니다.  
 
또한, 아이콘은 화면에 **수평으로 배열**되어 있지만,  
일반 GRUB 메뉴처럼 **위쪽** 및 **아래쪽 화살표 키**로 탐색할 수 있습니다.

## 설치

1. 저장소를 클론하세요

```shell
git clone https://github.com/Priyank-Adhav/Matrix-Morpheus-GRUB-Theme
```
2. 폴더로 이동하십시오  


```shell
cd Matrix-Morpheus-GRUB-Theme
```

3. 설치 프로그램을 실행 가능하게 만들기

```shell
chmod +x install.sh
```

4. 설치 스크립트를 관리자 권한으로 실행하십시오

```shell
sudo ./install.sh
```
5. 새 테마를 테스트하기 위해 재부팅하세요

---
선택 사항: GRUB 메뉴 간소화

이 테마는 두 개의 항목 레이아웃을 위해 설계되었으며 추가 항목들을 시각적으로 처리하는 방법에 대해서는 깊이 고민하지 않았습니다.

현재 GRUB 메뉴에 다음과 같은 추가 항목이 있다면:

- "Arch Linux용 고급 옵션"
- "UEFI 펌웨어 설정"

사용하지 않는다면 grub 설정에서 추가 메뉴 항목을 제거할 것을 권장합니다.
