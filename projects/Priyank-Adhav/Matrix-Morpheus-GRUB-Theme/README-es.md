# Tema Matrix Morpheus GRUB  
**Píldora Roja vs Píldora Azul**  

Un tema minimalista para GRUB inspirado en Matrix que presenta fondos dinámicos a pantalla completa que cambian entre Linux y Windows.  

---  

**Nota:**  
Actualmente, el tema solo incluye los íconos de **Arch Linux** y **Windows**.  

Además, aunque los íconos están **dispuestos horizontalmente** en pantalla,  
todavía navegas usando las **teclas de flecha Arriba y Abajo** como en un menú normal de GRUB.  

---  
![Vista previa del tema Matrix Morpheus GRUB mostrando los íconos de arranque de Arch y Windows](https://raw.githubusercontent.com/Priyank-Adhav/Matrix-Morpheus-GRUB-Theme/master/preview.gif)  
## Instalación  

1. Clona el repositorio

```shell
git clone https://github.com/Priyank-Adhav/Matrix-Morpheus-GRUB-Theme
```
2. Entra en la carpeta 


```shell
cd Matrix-Morpheus-GRUB-Theme
```

3. Hacer el instalador ejecutable

```shell
chmod +x install.sh
```
4. Ejecute el script de instalación como administrador


```shell
sudo ./install.sh
```

5. Reinicia para probar tu nuevo tema

---
Opcional: Simplifica tu Menú GRUB

Diseñé este tema para una disposición de dos entradas y realmente no he pensado en cómo manejar visualmente las entradas adicionales.

Si tu menú GRUB actualmente tiene entradas extra como:

- “Opciones avanzadas para Arch Linux”
- “Configuración de firmware UEFI”

Te recomendaría eliminar las entradas extra del menú desde la configuración de grub si no las usas.
