## 🚀 ポートフォリオ V3.0

> [!TIP]
> このポートフォリオを使いたいですか？上のフォークボタンをクリックしてください！
>
> ほぼスクラッチから、すべての面倒な設定がされた状態で始めたいなら、この[スターターテンプレート](https://github.com/jcserv/react-vite-shadcn-template)をチェックしてください。

このポートフォリオはTypeScriptで書き直され、レスポンシブ性とアクセシビリティを考慮して刷新され、新しいライブラリを試すために作成されました。

V2.0からの変更点：

- 🏙️ TypeScript
- 🔍 Fuse.jsを使った検索バー
- ⌨️ キーボードナビゲーション
- 🗞️ aboutセクションに証明書セクションを追加
- 😎 最新のトレンディなUIライブラリで更新（shadcn、tailwindcss、vite）
- 🐛 Experienceセクションのような揺れバグを修正

過去のポートフォリオのバージョン：

- V2.0: [github](https://github.com/jcserv/portfoliov2), [デプロイ](https://portfoliov2-taupe-one.vercel.app/)（Chakra-UI、フロントエンドPWA）
- V1.0: [github](https://github.com/jcserv/portfoliov1)（react-bootstrap、クライアント/サーバー）

## 💼 前提条件

- [Node.js](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)

## ⚙️ インストール方法

1. masterブランチの最新コードをクローンする
2. 依存関係をインストールする
   - `pnpm i`
3. ウェブサーバーを起動！
   - `pnpm dev`

## 🎈 使い方

- ブラウザで http://localhost:5173 にアクセスしてください

## 💫 デプロイ

プロジェクトを Vercel にインポートします。これでプッシュ時に自動デプロイが行われます。

## ⛏️ 使用技術

- [React](https://reactjs.org/)
- [shadcn](https://ui.shadcn.com/)
- [TailwindCSS](https://tailwindcss.com/)
- [Vite](https://vite.dev/)
