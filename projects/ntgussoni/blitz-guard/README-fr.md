Blitz Guard est une bibliothèque d'autorisation qui permet ou restreint l'accès à des ressources spécifiques.
Elle implémente le [contrôle d'accès basé sur les attributs](https://fr.wikipedia.org/wiki/Contr%C3%B4le_d%27acc%C3%A8s_bas%C3%A9_sur_les_attributs)

Elle vise à réduire le code répétitif et les règles d'autorisation difficiles à comprendre en n'ayant qu'une seule source de vérité dans le [fichier Ability](https://ntgussoni.github.io/blitz-guard/docs/ability-file)

# Installation

Utiliser la recette est la manière la plus simple. Elle ajoutera les paquets et créera les fichiers de configuration de base.

`blitz install ntgussoni/blitz-guard-recipe`
