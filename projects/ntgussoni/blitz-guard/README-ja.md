> Blitz.jsのための集中型権限ベース認可

<p align="center">

</p>

Blitz Guardは特定のリソースへのアクセスを許可または制限する認可ライブラリです。
[属性ベースアクセス制御](https://en.wikipedia.org/wiki/Attribute-based_access_control)を実装しています。

ボイラープレートや理解しにくい認可ルールを減らすことを目的としており、唯一の真実の情報源として[Abilityファイル](https://ntgussoni.github.io/blitz-guard/docs/ability-file)を持ちます。

# インストール

レシピを使うのが最も簡単な方法です。パッケージを追加し、基本的な設定ファイルを作成します。

`blitz install ntgussoni/blitz-guard-recipe`
