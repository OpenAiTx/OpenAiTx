> ЩҲЫҢЩҶШҜЩҲШІ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЪҜШұ ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ ЩҮШіШӘЫҢШҜШҢ Щ„Ш·ЩҒШ§ЩӢ Ш§ШІ WSL Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
>
> ШӘЩҲШөЫҢЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ… ШЁШ§ CLI ЩҲ ШӯШҜШ§ЩӮЩ„ ЩҲШёЫҢЩҒЩҮ ШҙШұЩҲШ№ Ъ©ЩҶЫҢШҜ ШӘШ§ Щ…Ш·Щ…ШҰЩҶ ШҙЩҲЫҢШҜ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ЩҲ Щ…ШӯЫҢШ· ШЁЩҮ ШҜШұШіШӘЫҢ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

### ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ

Ш§ЪҜШұ Ш§ШІ CLI Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ `pptagent onboard` Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ШөЩҲШұШӘ ШӘШ№Ш§Щ…Щ„ЫҢ ШҜШұ Ш§ЫҢШ¬Ш§ШҜ ЩҲ ШЁШұЩҲШІШұШіШ§ЩҶЫҢ Ш§ЫҢЩҶ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢвҖҢЩҮШ§ Ъ©Щ…Ъ© Ъ©ЩҶШҜ. Ш§ЪҜШұ Ш§ШІ Docker Compose ЫҢШ§ ШіШ§Ш®ШӘ Ш§ШІ ШіЩҲШұШі Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ ШЁШ§ЫҢШҜ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШҜШіШӘЫҢ ШўЩ…Ш§ШҜЩҮ Ъ©ЩҶЫҢШҜ:

```bash
cp deeppresenter/config.yaml.example deeppresenter/config.yaml
cp deeppresenter/mcp.json.example deeppresenter/mcp.json
```

#### ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш§Ш®ШӘЫҢШ§ШұЫҢ Ъ©ЩҮ Ъ©ЫҢЩҒЫҢШӘ ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШЁШ®ШҙЩҶШҜ

ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШІЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©ЫҢЩҒЫҢШӘ ШӘЩҲЩ„ЫҢШҜ ШұШ§ ШЁЩҮ Ш·ЩҲШұ Щ…ШӯШіЩҲШіЫҢ ШЁЩҮШЁЩҲШҜ ШҜЩҮЩҶШҜШҢ ШЁЩҮ ЩҲЫҢЪҳЩҮ ШЁШұШ§ЫҢ Ш№Щ…ЩӮ ШӘШӯЩӮЫҢЩӮШҢ ШӘШ¬ШІЫҢЩҮ PDF ЩҲ ШӘЩҲЩ„ЫҢШҜ ШҜШ§ШұШ§ЫҢЫҢвҖҢЩҮШ§ЫҢ ШЁШөШұЫҢ:

- **Tavily**: Ъ©ЫҢЩҒЫҢШӘ Ш¬ШіШӘШ¬ЩҲЫҢ ЩҲШЁ ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШҜЩҮШҜ. ШЁШұШ§ЫҢ ШҜШұЫҢШ§ЩҒШӘ Ъ©Щ„ЫҢШҜ API ШЁЩҮ [tavily.com](https://www.tavily.com/) Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ШіЩҫШі Щ…ЩӮШҜШ§Шұ `TAVILY_API_KEY` ШұШ§ ШҜШұ [`deeppresenter/mcp.json`](https://raw.githubusercontent.com/icip-cas/PPTAgent/main/deeppresenter/mcp.json) ШӘЩҶШёЫҢЩ… ЩҶЩ…Ш§ЫҢЫҢШҜ.
- **MinerU**: Ъ©ЫҢЩҒЫҢШӘ ШӘШ¬ШІЫҢЩҮ PDF ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШҜЩҮШҜ. Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШұШ§ЫҢ ШҜШұЫҢШ§ЩҒШӘ Ъ©Щ„ЫҢШҜ API ШЁЩҮ [mineru.net](https://mineru.net/apiManage/docs) Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ШұШҜЩҮ ЩҲ Щ…ЩӮШҜШ§Шұ `MINERU_API_KEY` ШұШ§ ШҜШұ [`deeppresenter/mcp.json`](https://raw.githubusercontent.com/icip-cas/PPTAgent/main/deeppresenter/mcp.json) ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜШҢ ЫҢШ§ MinerU ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Щ…ШӯЩ„ЫҢ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ Ъ©ШұШҜЩҮ ЩҲ ШЁЩҮ Ш¬Ш§ЫҢ ШўЩҶ Щ…ЩӮШҜШ§Шұ `MINERU_API_URL` ШұШ§ ШӘЩҶШёЫҢЩ… ЩҶЩ…Ш§ЫҢЫҢШҜ.
- **Щ…ШҜЩ„ ШӘШЁШҜЫҢЩ„ Щ…ШӘЩҶ ШЁЩҮ ШӘШөЩҲЫҢШұ**: Ъ©ЫҢЩҒЫҢШӘ ШӘЩҲЩ„ЫҢШҜ ШӘШөЩҲЫҢШұ ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШҜЩҮШҜ. Щ…ЩӮШҜШ§Шұ `t2i_model` ШұШ§ ШҜШұ [`deeppresenter/config.yaml`](https://raw.githubusercontent.com/icip-cas/PPTAgent/main/deeppresenter/config.yaml) ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Ъ©ЩҶЫҢШҜ.


Ш§ЪҜШұ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ ШЁЩҮ Ш·ЩҲШұ Ъ©Ш§Щ…Щ„ ШЁЩҮ ШөЩҲШұШӘ ШўЩҒЩ„Ш§ЫҢЩҶ Ъ©Ш§Шұ Ъ©ЩҶЫҢШҜШҢ MinerU ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Щ…ШӯЩ„ЫҢ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ Ъ©ШұШҜЩҮ ЩҲ Щ…ЩӮШҜШ§Шұ `offline_mode: true` ШұШ§ ШҜШұ `deeppresenter/config.yaml` ШӘЩҶШёЫҢЩ… ЩҶЩ…Ш§ЫҢЫҢШҜ ШӘШ§ Ш§ШІ ШЁШ§ШұЪҜШ°Ш§ШұЫҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҲШ§ШЁШіШӘЩҮ ШЁЩҮ ШҙШЁЪ©ЩҮ Щ…Ш«Щ„ Ш¬ШіШӘШ¬ЩҲЫҢ ЩҲШЁ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙЩҲШҜ.

Щ…ШӘШәЫҢШұЩҮШ§ЫҢ ЩӮШ§ШЁЩ„ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШЁЫҢШҙШӘШұ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұ [constants.py](https://raw.githubusercontent.com/icip-cas/PPTAgent/main/deeppresenter/utils/constants.py) ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ.

### 1. Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШ®ШөЫҢ / Ш§ШҜШәШ§Щ… ШЁШ§ OpenClaw: ШұШ§ШЁШ· Ш®Ш· ЩҒШұЩ…Ш§ЩҶ (CLI)

> [!ШӘЩҲШ¬ЩҮ]

>
> ШҜШұ Щ„ЫҢЩҶЩҲЪ©ШіШҢ ШЁШ§ЫҢШҜ Щ…ШӯЫҢШ· ШұШ§ Ш®ЩҲШҜШӘШ§ЩҶ ШўЩ…Ш§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

### ЫІ. ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ ШӯШҜШ§ЩӮЩ„ЫҢ / ШӘЩҲШіШ№ЩҮ: ШіШ§Ш®ШӘ Ш§ШІ ШіЩҲШұШі

Ш§ШІ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ Ш§ЪҜШұ Ъ©ЩҲЪҶЪ©ШӘШұЫҢЩҶ Щ„Ш§ЫҢЩҮ Ш§ЩҶШӘШІШ§Ш№ЫҢ ЩҲ Ъ©ЩҶШӘШұЩ„ Ъ©Ш§Щ…Щ„ ШЁШұ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШұШ§ ШҜШұ ШІЩ…Ш§ЩҶ ШӘЩҲШіШ№ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ.

```bash
uv pip install -e .
playwright install-deps
playwright install chromium
npm install --prefix deeppresenter/html2pptx

docker pull forceless/deeppresenter-host
docker tag forceless/deeppresenter-sandbox deeppresenter-sandbox
docker tag forceless/deeppresenter-host deeppresenter-host

# or pull through the 1ms.run mirror
docker pull docker.1ms.run/forceless/deeppresenter-sandbox
docker pull docker.1ms.run/forceless/deeppresenter-host
docker tag docker.1ms.run/forceless/deeppresenter-sandbox deeppresenter-sandbox
docker tag docker.1ms.run/forceless/deeppresenter-host deeppresenter-host

# or build from dockerfile
docker build -t deeppresenter-sandbox -f deeppresenter/docker/SandBox.Dockerfile .
docker build -t deeppresenter-host -f deeppresenter/docker/Host.Dockerfile .
```

### Ыі. Ш§ШіШӘЩӮШұШ§Шұ ШіШұЩҲШұ: Docker Compose

Ш§ШІ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ ШЁШұШ§ЫҢ Щ…ШӯЫҢШ· ШіШұЩҲШұ ЩҫШ§ЫҢШҜШ§Шұ ШЁШ§ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…ШҙШ®Шө Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

```bash
# Pull the public images to avoid build from source
docker pull forceless/deeppresenter-sandbox
docker pull forceless/deeppresenter-host
docker tag forceless/deeppresenter-sandbox deeppresenter-sandbox
docker tag forceless/deeppresenter-host deeppresenter-host

# Or pull through the 1ms.run mirror
docker pull docker.1ms.run/forceless/deeppresenter-sandbox
docker pull docker.1ms.run/forceless/deeppresenter-host
docker tag docker.1ms.run/forceless/deeppresenter-sandbox deeppresenter-sandbox
docker tag docker.1ms.run/forceless/deeppresenter-host deeppresenter-host

# Or build from source
docker build -t deeppresenter-sandbox -f deeppresenter/docker/SandBox.Dockerfile .
docker build -t deeppresenter-host -f deeppresenter/docker/Host.Dockerfile .

# Start the host service
docker compose up -d
```

The service exposes the web UI on `http://localhost:7861`.
