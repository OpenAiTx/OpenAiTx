### ⚡️ Anime y Manga ⚡️

- Descubre Anime de Temporada, Próximos Animes Destacados, Animes Más Populares, Listas de Clasificación de Anime, Favoritos de Todos los Tiempos y más.

- Accede a Sinopsis de Anime/Manga, contenido Relacionado y Recomendado.

- Lee reseñas detalladas de Anime y consulta estadísticas de Anime/Manga.

### ⚡️ Temas y Personalización ⚡️

- Elige entre 4 Temas Oscuros distintos: Primavera, Verano, Otoño e Invierno.

- Personaliza la barra de Navegación Inferior y ajusta la frecuencia de actualización de caché.

### ⚡️ Foros de MyAnimeList ⚡️

- Participa en foros dedicados a MyAnimeList, Anime y Manga, y Discusiones Generales.

### ⚡️ Barra de Búsqueda Avanzada ⚡️

- Realiza búsquedas avanzadas directamente desde la barra de búsqueda usando "@" y "#" con facilidad.

- Experimenta tiempos de carga de búsqueda más rápidos con mecanismos de caché.

### ⚡️ Funciones Específicas para Usuarios ⚡️

- Edita/Actualiza rápidamente tu Lista de Anime/Manga.

- Ordena tu lista por Título, Puntuación de la Lista, Fecha de Inicio y Fecha de Actualización.


### Comenzando

- Para ejecutar este proyecto, necesitas tener [Flutter](https://flutter.dev/docs/get-started/install) instalado en tu máquina.
- Clona este repositorio en tu máquina: `git clone`
- Usa `dal-code.code-workspace` para abrir el proyecto en VSCode
- Ejecuta el script `update_gradle.sh` para preparar los archivos gradle para el desarrollo
- Ejecuta `flutter pub get` para instalar las dependencias
- Lanza la aplicación usando `flutter run` o `F5` en VSCode

### Configuración Adicional

- Ejecuta la tarea de usuario - `intl generate` para generar archivos de localización
- Ejecuta la tarea de usuario - `build runner` para generar archivos necesarios
- Ejecuta la tarea de usuario - `python translate` para generar archivos de traducción
