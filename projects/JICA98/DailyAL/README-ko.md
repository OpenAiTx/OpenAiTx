### ⚡️ 애니메이션 & 만화 ⚡️

- 계절별 애니메이션, 다가오는 인기 애니메이션, 가장 인기 있는 애니메이션, 애니메이션 순위 목록, 역대 인기작 등을 발견하세요.

- 애니메이션/만화 줄거리, 관련 및 추천 콘텐츠에 접근하세요.

- 상세한 애니메이션 리뷰를 읽고 애니메이션/만화 통계도 확인하세요.

### ⚡️ 테마 및 사용자화 ⚡️

- 봄, 여름, 가을, 겨울의 4가지 독특한 다크 테마 중 선택하세요.

- 하단 내비게이션 바를 사용자화하고 캐시 업데이트 빈도를 조절하세요.

### ⚡️ MyAnimeList 포럼 ⚡️

- MyAnimeList, 애니메이션 & 만화, 일반 토론 전용 포럼에 참여하세요.

### ⚡️ 고급 검색창 ⚡️

- "@"와 "#"를 사용해 검색창에서 직접 고급 검색을 쉽게 수행하세요.

- 캐싱 메커니즘으로 더 빠른 검색 로딩 시간을 경험하세요.

### ⚡️ 사용자 맞춤 기능 ⚡️

- 애니메이션/만화 목록을 빠르게 편집/업데이트하세요.

- 제목, 목록 점수, 시작 날짜, 업데이트 날짜로 목록을 정렬하세요.


### 시작하기

- 이 프로젝트를 실행하려면 머신에 [Flutter](https://flutter.dev/docs/get-started/install)가 설치되어 있어야 합니다.
- 이 저장소를 머신에 복제하세요: `git clone`
- VSCode에서 프로젝트를 열려면 `dal-code.code-workspace`를 사용하세요
- gradle 파일을 개발 준비 상태로 만들기 위해 `update_gradle.sh` 스크립트를 실행하세요  
- `flutter pub get`을 실행하여 의존성을 설치하세요  
- `flutter run` 또는 VSCode에서 `F5`를 눌러 앱을 실행하세요  

### 추가 설정  

- 사용자 작업 실행 - `intl generate`로 로컬라이제이션 파일 생성  
- 사용자 작업 실행 - `build runner`로 필요한 파일 생성  
- 사용자 작업 실행 - `python translate`로 번역 파일 생성
