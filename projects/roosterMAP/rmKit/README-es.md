# rmKit

**rmKit** es un complemento para Blender lleno de funciones diseñado para mejorar tu flujo de trabajo de modelado 3D con una amplia variedad de herramientas y utilidades. Proporciona funcionalidad avanzada para la edición de mallas, modelado de precisión y gestión de materiales, convirtiéndolo en un conjunto de herramientas esencial tanto para profesionales como para aficionados.

---

## Características Clave

- **Herramientas Avanzadas de Edición de Mallas**: Incluye operadores para conectar bordes, crear tubos, extruir a lo largo de bordes, engrosar mallas y mucho más.
- **Flujo de Trabajo Personalizable**: Herramientas como asignación rápida de materiales, alineación radial y configuración del plano de trabajo permiten un flujo de trabajo altamente adaptable.
- **Modelado de Precisión**: Características como el visualizador de dimensiones, la herramienta de caída lineal, la herramienta mover-al-más-lejano y el cuchillo en pantalla aseguran precisión en tus diseños.
- **Normales de Vértice Basadas en Selección**: Una herramienta personalizada para gestionar normales de vértices basadas en conjuntos de selección, permitiendo un control fino sobre el sombreado.
- **Paneles Amigables para el Usuario**: Paneles de interfaz intuitivos para acceso fácil a herramientas y configuraciones, organizados para una navegación eficiente.

---

## ¿Por Qué Usar rmKit?

- **Ahorro de Tiempo**: Automatiza tareas repetitivas y simplifica operaciones complejas.
- **Versátil**: Adecuado para una amplia gama de tareas de modelado, desde modelado de superficies duras hasta formas orgánicas.
- **Integrado**: Se integra perfectamente en la interfaz de Blender para una experiencia de usuario fluida.
- **Personalizable**: Ofrece opciones para adaptar las herramientas a tus necesidades específicas.

---

## Comienza

Activa rmKit en el menú de complementos de Blender y explora su extenso conjunto de herramientas para potenciar tu flujo de trabajo de modelado. Ya sea que estés creando diseños intrincados o gestionando proyectos grandes, rmKit es tu complemento ideal para eficiencia y precisión.

Para más información, visita la [documentación oficial](https://rmkit.readthedocs.io/en/latest/).
