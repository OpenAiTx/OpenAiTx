## freeCodeCamp.org のオープンソースコードベースとカリキュラム

[freeCodeCamp.org](https://www.freecodecamp.org) は、無料でコーディングを学べるフレンドリーなコミュニティです。これは、数百万の多忙な大人がテック業界へ移行できるよう支援する [寄付支援による 501(c)(3) チャリティー](https://www.freecodecamp.org/donate) によって運営されています。私たちのコミュニティは、すでに 40,000 人以上が初めての開発者職を得るのをサポートしてきました。

私たちのフルスタックウェブ開発および機械学習のカリキュラムは完全に無料で、自分のペースで学習できます。数千のインタラクティブなコーディングチャレンジがあり、スキルを拡張できます。

### 認定証

freeCodeCamp.org では複数の無料開発者認定証を提供しています。各認定証では、5つの必須ウェブアプリプロジェクトの作成と、それらのプロジェクト準備のための数百の任意コーディングチャレンジが含まれています。各認定証の取得には、初心者プログラマーでおよそ300時間かかると見積もっています。

freeCodeCamp.org カリキュラム内のこれら50のプロジェクトそれぞれには、独自のアジャイルユーザーストーリーと自動テストが備わっています。これにより、プロジェクトを段階的に構築でき、提出前にすべてのユーザーストーリーを満たしていることを確認できます。

これらのテストスイートは [freeCodeCamp の CDN](https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js) 経由で利用できます。つまり、CodePen や Replit などのウェブサイト、またはローカルの開発環境でもプロジェクトを作成できます。

一度認定証を取得すれば、それは常に保持できます。LinkedIn や履歴書からいつでもリンクできます。また、雇用主やフリーランスのクライアントがそのリンクをクリックすると、あなた専用の検証済み認定証が表示されます。

唯一の例外は、[学術的誠実性ポリシー](https://www.freecodecamp.org/news/academic-honesty-policy/)違反が発覚した場合です。他人のコードやプロジェクトを自分のものとして無断で提出するなど、明確な剽窃が発覚した場合は、すべての厳格な教育機関が行うべきこととして、認定証の取り消しおよびアカウントの停止を行います。
