## Panoramica

Annota le tue immagini e lascia che una foto parli per mille parole.

- Carica immagini dal file system, dagli appunti, o crea uno screenshot da annotare.
- Aggiungi forme, adesivi, immagini, testo, disegni e altri elementi per evidenziare dettagli dell’immagine.
- Aggiungi ingranditori per migliorare i dettagli dell’immagine.
- Sfoca parti dell’immagine per oscurare dati.
- Ritaglia, ridimensiona e aggiungi bordi all’immagine.
- Controlla colori, spessore delle linee e dettagli dei font.
- Supporto zoom.
- Supporto selettore colori all'interno di un'immagine caricata.
- Annulla/ripeti illimitato di qualsiasi modifica.
- Trascina e rilascia copie PNG dell’immagine annotata.
- Esporta nei formati immagine JPEG, PNG, TIFF, BMP, PDF, SVG e WebP.
- Supporto per la copia dell’immagine annotata negli appunti.
- Supporto stampante.

## Installazione

### Debian (da sorgente)

Avrai bisogno delle seguenti dipendenze per compilare Annotator:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Per installare Annotator da sorgente, esegui `./app install`.

Per eseguire Annotator, avvia `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Puoi utilizzare il [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) mantenuto da @PandaJim. Il PPA supporta Ubuntu 20.04+. Inserisci i seguenti comandi uno alla volta

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Se sei un utente Arch Linux, esiste un
[pacchetto AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Inoltre, Annotator può essere installato ed eseguito tramite Flatpak.

Per compilare il Flatpak elementary dal sorgente, esegui `./app elementary`.

Per compilare il Flatpak Flathub dal sorgente, esegui `./app flathub`.

Successivamente, puoi eseguirlo tramite: `./app run-flatpak`.
