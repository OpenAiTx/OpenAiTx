## Ikhtisar

Annotasi gambar Anda dan biarkan gambar berbicara seribu kata.

- Muat gambar dari sistem file, clipboard, atau buat tangkapan layar untuk dianotasi.
- Tambahkan bentuk, stiker, gambar, teks, gambar tangan bebas, dan callout lain untuk menyorot detail gambar.
- Tambahkan pembesar untuk memperjelas detail gambar.
- Kaburkan bagian gambar untuk menyembunyikan data.
- Potong, ubah ukuran dan tambahkan bingkai gambar.
- Kontrol warna, ketebalan garis dan detail font.
- Dukungan zoom.
- Dukungan pemilih warna di dalam gambar yang dimuat.
- Undo/redo tanpa batas pada setiap perubahan.
- Seret dan lepas salinan PNG dari gambar yang telah dianotasi.
- Ekspor ke format gambar JPEG, PNG, TIFF, BMP, PDF, SVG dan WebP.
- Dukungan untuk menyalin gambar yang telah dianotasi ke clipboard.
- Dukungan printer.

## Instalasi

### Debian (dari sumber)

Anda akan membutuhkan dependensi berikut untuk membangun Annotator:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Untuk menginstal Annotator dari sumber, jalankan `./app install`.

Untuk menjalankan Annotator, jalankan `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Anda dapat menggunakan [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) yang dikelola oleh @PandaJim. PPA ini mendukung Ubuntu 20.04+. Masukkan perintah berikut satu per satu

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Jika Anda pengguna Arch Linux, tersedia
[paket AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Selain itu, Annotator dapat diinstal dan dijalankan melalui Flatpak.

Untuk membangun elementary Flatpak dari sumber, jalankan `./app elementary`.

Untuk membangun Flathub Flatpak dari sumber, jalankan `./app flathub`.

Setelah itu, Anda dapat menjalankannya melalui: `./app run-flatpak`.
