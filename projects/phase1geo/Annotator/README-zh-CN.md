## 概述

为您的图片添加注释，让图片表达千言万语。

- 从文件系统、剪贴板加载图片，或创建截图进行标注。
- 添加形状、贴纸、图片、文本、绘图和其他标注以突出图片细节。
- 添加放大镜以增强图片细节。
- 模糊图片部分以隐藏数据。
- 裁剪、调整大小并添加图片边框。
- 控制颜色、线条粗细和字体细节。
- 支持缩放。
- 支持在加载图片内的取色器。
- 对任何更改无限撤销/重做。
- 拖拽注释后的PNG图片副本。
- 导出为JPEG、PNG、TIFF、BMP、PDF、SVG和WebP图片格式。
- 支持将注释后的图片复制到剪贴板。
- 支持打印机。

## 安装

### Debian（源码安装）

构建 Annotator 需要以下依赖项：

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

要从源码安装 Annotator，请运行 `./app install`。

要运行 Annotator，请运行 `com.github.phase1geo.annotator`。

### Ubuntu (PPA)

你可以使用由 @PandaJim 维护的 [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/)。该 PPA 支持 Ubuntu 20.04+。请依次输入以下命令

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

如果你是 Arch Linux 用户，有一个
[AUR 软件包](https://aur.archlinux.org/packages/annotator/)
`annotator`：

```
% yay -S annotator
```

### Flatpak

此外，Annotator 可以通过 Flatpak 安装和运行。

要从源代码构建 elementary Flatpak，请运行 `./app elementary`。

要从源代码构建 Flathub Flatpak，请运行 `./app flathub`。

之后，你可以通过以下命令运行：`./app run-flatpak`。
