## Aperçu

Annotez vos images et laissez une image exprimer mille mots.

- Chargez une image depuis le système de fichiers, le presse-papiers ou créez une capture d'écran à annoter.
- Ajoutez des formes, autocollants, images, texte, dessins et autres éléments pour mettre en valeur des détails de l'image.
- Ajoutez des loupes pour améliorer les détails de l'image.
- Floutez des parties de l'image pour masquer des données.
- Rognez, redimensionnez et ajoutez des bordures à l'image.
- Contrôlez les couleurs, l'épaisseur des lignes et les détails de la police.
- Prise en charge du zoom.
- Pipette à couleurs disponible sur une image chargée.
- Annulation/rétablissement illimités de toute modification.
- Glisser-déposer des copies PNG de l'image annotée.
- Exportez vers les formats JPEG, PNG, TIFF, BMP, PDF, SVG et WebP.
- Prise en charge de la copie de l'image annotée dans le presse-papiers.
- Prise en charge de l'impression.

## Installation

### Debian (depuis les sources)

Vous aurez besoin des dépendances suivantes pour compiler Annotator :

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Pour installer Annotator depuis les sources, exécutez `./app install`.

Pour exécuter Annotator, lancez `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Vous pouvez utiliser le [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) maintenu par @PandaJim. Le PPA prend en charge Ubuntu 20.04+. Saisissez les commandes suivantes une par une

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Si vous êtes un utilisateur d'Arch Linux, il existe un
[paquet AUR](https://aur.archlinux.org/packages/annotator/)
`annotator` :

```
% yay -S annotator
```

### Flatpak

De plus, Annotator peut être installé et exécuté via Flatpak.

Pour construire le Flatpak elementary à partir du code source, exécutez `./app elementary`.

Pour construire le Flatpak Flathub à partir du code source, exécutez `./app flathub`.

Ensuite, vous pouvez l'exécuter via : `./app run-flatpak`.
