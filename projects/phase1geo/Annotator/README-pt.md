## Visão Geral

Anote suas imagens e deixe uma imagem valer por mil palavras.

- Carregue imagem do sistema de arquivos, área de transferência ou crie uma captura de tela para anotar.
- Adicione formas, adesivos, imagens, texto, desenhos e outros destaques para realçar detalhes da imagem.
- Adicione lupas para ampliar detalhes da imagem.
- Borra partes da imagem para ocultar dados.
- Corte, redimensione e adicione bordas à imagem.
- Controle cores, espessura da linha e detalhes da fonte.
- Suporte a zoom.
- Suporte ao seletor de cores dentro de uma imagem carregada.
- Desfazer/refazer ilimitado de qualquer alteração.
- Arraste e solte cópias PNG da imagem anotada.
- Exporte para os formatos de imagem JPEG, PNG, TIFF, BMP, PDF, SVG e WebP.
- Suporte para copiar imagem anotada para a área de transferência.
- Suporte à impressão.

## Instalação

### Debian (a partir do código fonte)

Você precisará das seguintes dependências para compilar o Annotator:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Para instalar o Annotator a partir do código fonte, execute `./app install`.

Para executar o Annotator, execute `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Você pode usar o [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) mantido por @PandaJim. O PPA suporta Ubuntu 20.04+. Digite os seguintes comandos um por um

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Se você é um usuário do Arch Linux, existe um
[Pacote AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Além disso, o Annotator pode ser instalado e executado via Flatpak.

Para compilar o Flatpak do elementary a partir do código-fonte, execute `./app elementary`.

Para compilar o Flatpak do Flathub a partir do código-fonte, execute `./app flathub`.

Em seguida, você pode executá-lo via: `./app run-flatpak`.
