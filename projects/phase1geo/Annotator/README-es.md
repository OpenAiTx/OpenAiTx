## Resumen

Anota tus imágenes y deja que una imagen diga 1000 palabras.

- Carga imágenes desde el sistema de archivos, portapapeles, o crea una captura de pantalla para anotar.
- Añade formas, pegatinas, imágenes, texto, dibujos y otros llamados para resaltar detalles de la imagen.
- Añade lupas para mejorar los detalles de la imagen.
- Difumina partes de la imagen para ocultar datos.
- Recorta, cambia tamaño y añade bordes a la imagen.
- Controla colores, grosor de línea y detalles de la fuente.
- Soporte de zoom.
- Soporte de selector de color dentro de una imagen cargada.
- Deshacer/rehacer ilimitado de cualquier cambio.
- Arrastrar y soltar copias PNG de la imagen anotada.
- Exporta a formatos de imagen JPEG, PNG, TIFF, BMP, PDF, SVG y WebP.
- Soporte para copiar la imagen anotada al portapapeles.
- Soporte para impresora.

## Instalación

### Debian (desde el código fuente)

Necesitarás las siguientes dependencias para compilar Annotador:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-dev
- libxml2-dev
- libgtk-3-dev
- libhandy-1-dev
- libwebp-dev

Para instalar Annotator desde el código fuente, ejecute `./app install`.

Para ejecutar Annotator, ejecute `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Puede usar el [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) mantenido por @PandaJim. El PPA es compatible con Ubuntu 20.04+. Ingrese los siguientes comandos uno por uno

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Si eres usuario de Arch Linux, existe un
[paquete AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Además, Annotator se puede instalar y ejecutar mediante Flatpak.

Para compilar e instalar el Flatpak desde el código fuente, ejecute `./app flatpak`.

Después, puede ejecutarlo con: `flatpak run com.github.phase1geo.annotator`.
