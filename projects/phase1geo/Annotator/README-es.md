## Descripción general

Anota tus imágenes y deja que una imagen diga más que mil palabras.

- Carga imágenes desde el sistema de archivos, el portapapeles, o crea una captura de pantalla para anotar.
- Agrega formas, pegatinas, imágenes, texto, dibujos y otros elementos para resaltar detalles de la imagen.
- Añade lupas para ampliar detalles de la imagen.
- Difumina partes de la imagen para ocultar datos.
- Recorta, cambia el tamaño y añade bordes a la imagen.
- Controla los colores, el grosor de línea y los detalles de la fuente.
- Soporte de zoom.
- Selector de color dentro de una imagen cargada.
- Deshacer/rehacer ilimitado de cualquier cambio.
- Arrastra y suelta copias PNG de la imagen anotada.
- Exporta a formatos de imagen JPEG, PNG, TIFF, BMP, PDF, SVG y WebP.
- Soporte para copiar la imagen anotada al portapapeles.
- Soporte de impresión.

## Instalación

### Debian (desde el código fuente)

Necesitarás las siguientes dependencias para compilar Annotator:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Para instalar Annotator desde el código fuente, ejecuta `./app install`.

Para ejecutar Annotator, ejecute `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Puede usar el [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) mantenido por @PandaJim. El PPA es compatible con Ubuntu 20.04+. Introduzca los siguientes comandos uno por uno

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Si eres usuario de Arch Linux, existe un
[paquete AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Además, Annotator se puede instalar y ejecutar mediante Flatpak.

Para compilar el Flatpak de elementary desde el código fuente, ejecuta `./app elementary`.

Para compilar el Flatpak de Flathub desde el código fuente, ejecuta `./app flathub`.

Luego, puedes ejecutarlo mediante: `./app run-flatpak`.
