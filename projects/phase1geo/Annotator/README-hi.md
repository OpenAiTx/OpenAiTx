## अवलोकन

अपनी छवियों को एनोटेट करें और एक तस्वीर को 1000 शब्द कहने दें।

- फ़ाइल सिस्टम, क्लिपबोर्ड से छवि लोड करें या एनोटेशन के लिए स्क्रीनशॉट बनाएं।
- छवि विवरण को उजागर करने के लिए आकार, स्टिकर, छवियां, टेक्स्ट, ड्रॉइंग और अन्य कॉलआउट जोड़ें।
- छवि विवरण को बढ़ाने के लिए मैग्निफायर जोड़ें।
- डेटा छुपाने के लिए छवि के कुछ हिस्सों को ब्लर करें।
- छवि को क्रॉप करें, आकार बदलें और बॉर्डर जोड़ें।
- रंग, लाइन मोटाई और फ़ॉन्ट विवरण नियंत्रित करें।
- ज़ूम समर्थन।
- लोड की गई छवि के भीतर रंग चयनकर्ता समर्थन।
- किसी भी परिवर्तन की अनलिमिटेड पूर्ववत/फिर से करें।
- एनोटेटेड छवि की PNG कॉपी ड्रैग-एंड-ड्रॉप करें।
- JPEG, PNG, TIFF, BMP, PDF, SVG और WebP छवि प्रारूपों में निर्यात करें।
- एनोटेटेड छवि को क्लिपबोर्ड में कॉपी करने का समर्थन।
- प्रिंटर समर्थन।

## स्थापना

### डेबियन (सोर्स से)

Annotator बनाने के लिए आपको निम्नलिखित डिपेंडेंसीज़ की आवश्यकता होगी:

- मेसन
- वलाक
- डेबहेल्पर
- गोबजेक्ट-2.0
- ग्लिब-2.0
- लिबगी-0.8-डेव
- लिबग्रेनाइट-7-डेव
- लिबGTK-4-डेव
- लिबXML2-डेव
- लिबआर्काइव-डेव
- लिबWebP-डेव

Annotator को सोर्स से स्थापित करने के लिए `./app install` चलाएँ।

Annotator चलाने के लिए, `com.github.phase1geo.annotator` चलाएँ।

### उबुन्टू (PPA)

आप @PandaJim द्वारा बनाए गए [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) का उपयोग कर सकते हैं। यह PPA उबुन्टू 20.04+ को सपोर्ट करता है। निम्नलिखित कमांड्स एक-एक करके डालें

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### आर्च लिनक्स

यदि आप आर्च लिनक्स उपयोगकर्ता हैं, तो वहाँ एक
[AUR पैकेज](https://aur.archlinux.org/packages/annotator/)
`annotator` उपलब्ध है:

```
% yay -S annotator
```

### Flatpak

अतिरिक्त रूप से, Annotator को Flatpak के माध्यम से इंस्टॉल और चलाया जा सकता है।

elementary Flatpak को स्रोत से बनाने के लिए, `./app elementary` चलाएँ।

Flathub Flatpak को स्रोत से बनाने के लिए, `./app flathub` चलाएँ।

इसके बाद, आप इसे इस तरह चला सकते हैं: `./app run-flatpak`।
