## Przegląd

Adnotuj swoje obrazy i pozwól, by jeden obraz mówił 1000 słów.

- Załaduj obraz z systemu plików, schowka lub utwórz zrzut ekranu do adnotacji.
- Dodaj kształty, naklejki, obrazy, tekst, rysunki i inne oznaczenia, aby podkreślić szczegóły obrazu.
- Dodaj lupy, aby uwydatnić szczegóły obrazu.
- Zamazuj fragmenty obrazu, aby ukryć dane.
- Przycinaj, zmieniaj rozmiar i dodawaj ramki do obrazów.
- Kontroluj kolory, grubość linii i szczegóły czcionki.
- Obsługa powiększania.
- Obsługa wybierania koloru w załadowanym obrazie.
- Nieograniczone cofanie/przywracanie dowolnych zmian.
- Przeciągnij i upuść kopie PNG adnotowanego obrazu.
- Eksport do formatów JPEG, PNG, TIFF, BMP, PDF, SVG i WebP.
- Obsługa kopiowania adnotowanego obrazu do schowka.
- Obsługa drukarki.

## Instalacja

### Debian (ze źródła)

Do zbudowania Annotator potrzebne są następujące zależności:

- meson
- valac
- debhelper
- gobject-2.0
- glib-2.0
- libgee-0.8-dev
- libgranite-7-dev
- libgtk-4-dev
- libxml2-dev
- libarchive-dev
- libwebp-dev

Aby zainstalować Annotator ze źródła, uruchom `./app install`.

Aby uruchomić Annotator, uruchom `com.github.phase1geo.annotator`.

### Ubuntu (PPA)

Możesz użyć [PPA](https://launchpad.net/~ubuntuhandbook1/+archive/ubuntu/annotator/) utrzymywanego przez @PandaJim. PPA obsługuje Ubuntu 20.04+. Wprowadź poniższe polecenia po kolei

```
sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator
```

### Arch Linux

Jeśli jesteś użytkownikiem Arch Linux, dostępny jest
[pakiet AUR](https://aur.archlinux.org/packages/annotator/)
`annotator`:

```
% yay -S annotator
```

### Flatpak

Dodatkowo, Annotator można zainstalować i uruchomić za pomocą Flatpak.

Aby zbudować Flatpak elementary ze źródła, uruchom `./app elementary`.

Aby zbudować Flatpak Flathub ze źródła, uruchom `./app flathub`.

Następnie możesz uruchomić go za pomocą: `./app run-flatpak`.
