# Drawing-with-Metapost

トビー・サーストン -- 2025年1月28日

このプロジェクトは、ジョン・ホビーのMetapost言語を使って技術図を描く方法について解説したドキュメントを提供します。  
200点以上のMetapostで作成されたイラストと、インスピレーションや例としてのソースコードが含まれています。  
対象レベルは完全な初心者ではなく、中級から上級ユーザー向けです。  
Metapostの入門、チュートリアル、その他の記事については、  
http://www.tug.org/metapost.html を参照してください。

トップディレクトリの `Drawing-with-Metapost.pdf` から始めてください。

`src` ディレクトリには  
- 本文書のTeXソース  
- Metapostソースコードのマーキングに使われるスタイルファイル  
- 本文書で使用されている各イラストのMetapostソース  
- 各MPソースファイルから作成された対応するPDFファイル  
が含まれています。

本文にはイラスト作成に用いられた技法の解説があり、ほとんどのイラストについて完全または部分的なリスティングが含まれています。  
これらのリスティングは使用されたソースコードから直接引用されているため、すべての場合においてページ上に表示されるものと出力が完全に一致します。

まず本文書を読むことをお勧めしますが、`src` ディレクトリ内のPDFをブラウズし、興味のあるものがあれば対応するMPソースファイルを参照するのも良いでしょう。  
PDF名とMPソース名は一対一で対応しているため、  
`geometry-apollonius.pdf` は `geometry-apollonius.mp` から作成されています。  
`src` ディレクトリには本文書に含まれない図もいくつかあります。

誤りを見つけた場合や改善提案がある場合は、Githubでイシューを立てるかプルリクエストを提出してください。
