# Drawing-with-Metapost

Toby Thurston -- 28 Ene 2025

Este proyecto ofrece un documento que explica cómo dibujar diagramas técnicos
con el lenguaje Metapost de John Hobby. Incluye más de 200 ilustraciones creadas
con Metapost, junto con el código fuente como inspiración y ejemplos. El
nivel previsto es para usuarios intermedios a avanzados y no para
principiantes completos. Para introducciones, tutoriales y otros artículos sobre Metapost, vea
http://www.tug.org/metapost.html

Comience con `Drawing-with-Metapost.pdf` en el directorio principal.

El directorio `src` contiene 
- el código fuente TeX para el documento principal
- el archivo de estilo usado para marcar el código fuente Metapost
- el código fuente Metapost de cada ilustración usada en el documento principal
- los archivos PDF correspondientes creados a partir de cada archivo fuente MP

El texto incluye una discusión de las técnicas usadas para dibujar las
ilustraciones, e incluye listados completos o parciales de la mayoría de ellas.
Estos listados se incluyen directamente desde el código fuente usado, por lo que en todos los casos
la salida debe coincidir exactamente con lo que se ve en la página.

Puede que quiera leer primero el documento principal, pero también puede
explorar los PDFs en el directorio `src`, y cuando encuentre uno que le
interese, echar un vistazo al archivo fuente MP correspondiente. Hay una
correspondencia uno a uno entre los nombres PDF y los nombres fuente MP, así que
`geometry-apollonius.pdf` se crea a partir de `geometry-apollonius.mp`. El directorio `src`
contiene algunos dibujos que no están incluidos en el documento principal.

Si encuentra errores o tiene sugerencias para mejorar, por favor abra un
