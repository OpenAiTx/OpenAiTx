# Drawing-with-Metapost

Toby Thurston -- 2025年1月28日

本项目提供了一份文档，讨论如何使用 John Hobby 的 Metapost 语言绘制技术图表。  
文档包含200多幅用 Metapost 创建的插图，附带源代码作为灵感和示例。  
目标读者为中级到高级用户，而非完全初学者。  
有关 Metapost 的入门、教程及其他文章，请参见  
http://www.tug.org/metapost.html

请从顶层目录下的 `Drawing-with-Metapost.pdf` 开始阅读。

`src` 目录包含  
- 主文档的 TeX 源文件  
- 用于标记 Metapost 源代码的样式文件  
- 主文档中每幅插图使用的 Metapost 源代码  
- 从每个 MP 源文件生成的对应 PDF 文件

文中讨论了绘制插图所用的技术，并包含了大部分插图的完整或部分代码清单。  
这些代码清单直接从使用的源代码中导入，因此输出应与页面上看到的完全一致。

你可以先阅读主文档，也可以浏览 `src` 目录中的 PDF 文件，  
当你发现感兴趣的文件时，查看对应的 MP 源文件。  
PDF 文件名与 MP 源文件名一一对应，  
例如 `geometry-apollonius.pdf` 是由 `geometry-apollonius.mp` 生成的。  
`src` 目录中还包含一些未收录在主文档中的绘图。
