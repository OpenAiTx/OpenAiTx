# 소개
**LOLAutomationTools** 는 **리그 오브 레전드(LOL)** 의 기능 확장 도구입니다. 사용자가 일시적으로 자리를 비우거나 손을 사용할 수 없는 상황에서 간단한 클라이언트 작업을 자동으로 처리하여 약간의 시간을 절약할 수 있도록 도와줍니다. 이 소프트웨어는 게임에 대한 이해와 플레이 방식에 과도하게 관여하지 않으므로 일부 기능은 수동 설정이 필요할 수 있습니다(예: 챔피언 룬 설정). 과도한 제약은 게임의 재미를 떨어뜨릴 수 있으니, 적극적이고 즐거운 게임 환경을 함께 만들어 갑시다.

## 영웅 선택
**영웅 선택**과 **영웅 금지**를 사용하기 전에, 설정 페이지에서 편집 작업(스위치 옆의 버튼)을 해야 합니다.

라인 설정에 추가된 영웅 항목은 일부 *사전 선택 슬롯*이 있는 게임 모드에서만 적용됩니다(예: *랭크전* 및 *극한 번개*), 랜덤은 *랜덤 영웅* 모드에서만 적용(예: *극지 대난투*, *무한 난투*, 영웅 금지 설정에는 랜덤 설정 페이지가 포함되지 않음), 일반은 그 외 모든 모드에서 적용(앞서 언급한 모드는 포함되지 않음, 사용 규칙을 위해), 자동 영웅 선택 실행 시 **영웅 선택**의 설정 항목을 위에서 아래로 순서대로 선택합니다.

## 사전 선택 영웅
이 기능의 실행 논리는 **영웅 선택**과 거의 동일하며, 차이점은 **사전 선택 영웅**은 금지 단계가 있는 모드에서만 적용된다는 점입니다. 예: *랭크전*과 일부 영웅 금지가 필요한 오락 모드.

## 영웅 금지
**영웅 금지** 설정 페이지는 **영웅 선택** 설정 페이지와 거의 동일하며, 오른쪽 상단에 모드 전환 버튼이 추가되어 있습니다. 버튼을 클릭하면 금지 모드로 전환할 수 있습니다.

*일반 금지 모드* : 이 모드의 동작 논리는 **영웅 선택** 설정 페이지와 동일하며, 자동 영웅 금지 실행 시 금지할 영웅이 설정되어 있지 않으면 Ban 단계 남은 시간 0초에 빈 Ban을 하게 됩니다.

*연합 금지 모드* : 설정 리스트의 항목이 **영웅 선택** 설정의 해당 영웅으로 교체되며, 항목을 선택하면 왼쪽 영웅 항목을 더블 클릭해 해당 영웅의 **금지 그룹**에 추가할 수 있습니다. 자동 영웅 금지 실행 시 **영웅 선택** 설정 항목 순서대로 금지하며, 모든 **영웅 선택** 설정 **금지 그룹**에 금지할 영웅이 없으면 일반 금지 모드 논리대로 금지합니다.

> *팀원 사전 선택 무시*를 활성화하면 자동 영웅 금지 실행 시 팀 사전 선택 감지를 건너뜁니다. 이 항목 활성화는 권장하지 않습니다.

## 대전 수락
매칭된 대전을 자동으로 수락합니다.

> *지연 대전 수락* 옵션을 활성화하면, **대전 수락**은 대전 수락 카운트다운의 마지막 1초에 자동으로 대전을 수락합니다.

## 대전 생성
**대전 생성** 버튼을 통해 대전을 수동으로 생성하거나 이 스위치 항목을 켜서 대전을 자동으로 생성할 수 있습니다.

> *자동 대전 생성*은 *추가 설정*에서 생성할 대전 모드를 선택해야 하며, 이 옵션은 설정에 저장되지 않습니다.

## 대전 찾기
대전을 자동으로 찾으며, 이 항목은 대전 모드 방에서 자신이 팀 소유자(방장)일 때만 적용됩니다.

> *타임아웃 감지* 옵션을 활성화하면, 매칭 대기열에서 대기 예상 시간을 지정된 간격 이상 초과할 경우 자동으로 대전을 다시 찾습니다.

## 칭찬 건너뛰기
대전 종료 시 팀원의 칭찬 페이지를 건너뜁니다.

## 결산 건너뛰기
대전 종료 시 결산 대기(검은 화면에서 로딩) 또는 경기 후 결산 페이지를 건너뛰고 대전 모드 방으로 돌아갑니다.

## 교환 기능
메인 스위치와 세 개의 독립 스위치가 있으며, 추가 설정이 필요합니다. **교환 기능**의 추가 설정에서 랜덤 영웅 모드에서 팀원과 영웅 교환, 드래프트 모드에서 팀원과 순서(영웅 선택 순서) 또는 사전 선택 자리 교환 여부를 설정할 수 있습니다.

*영웅 교환과 자리 교환 설정은 비교적 간단하므로 자세한 설명은 생략합니다.*

<img width="900" height="680" alt="image" src="https://github.com/user-attachments/assets/361b4b19-cd72-4cb3-901f-69d49284bc11" />

**순서 교환**의 **추가 설정**에서 교환 순서(즉, 순서)와 교환 모드를 설정할 수 있습니다.

<img width="900" height="680" alt="image" src="https://github.com/user-attachments/assets/697bed4a-2984-43f6-b747-36c7911ef99d" />

*교환 모드*:
- 목표 순서만: 지정한 *순서*와 일치하는 팀원의 요청만 수락합니다.
- 목표 포함 우선 순서: 지정한 *순서* 이하의 팀원 요청을 수락합니다.
- 목표 포함 후순위: 지정한 *순서* 이상의 팀원 요청을 수락합니다.
- 목표 순서 기반 범위: *순서* ≤ (*지정 순서* - *범위*)와 *순서* ≥ (*지정 순서* + *범위*)인 팀원 요청을 수락합니다.

또한, **기타 설정**에는 두 가지 교환 거부 옵션이 있습니다. 이 두 항목을 **활성화**하면 팀원이 영웅 또는 자리 교환 요청 시 조건에 부합하지 않을 경우 자동으로 해당 교환 요청을 거부하며, **비활성화**하면 아무런 조치를 취하지 않으므로 교환 수락 여부를 직접 결정할 수 있습니다.

## 금지/선택 영웅 퇴장 방지
이 항목을 활성화하면 BP 단계의 마지막 1초에 **영웅 선택** 또는 **영웅 금지**를 실행합니다.

> *기능 설정 페이지*에서 **금지/선택 영웅 퇴장 방지** 기능의 모드 설정을 변경할 수 있습니다.

## 잠시 자리 비움 응답
이 항목은 주 스위치이며, 추가 설정이 필요합니다. **더보기** 설정 페이지에서, 자리 비움 시 자동으로 친구 메시지나 게임 초대에 응답할지(각 항목 기본값은 활성화 상태), 응답할 텍스트 내용을 설정합니다.

## 부가 창
이 항목을 활성화하면 일부 클라이언트 단계에서 **부가 창**이 표시됩니다. 또한, 이 스위치 옆 버튼을 통해 **부가 창**의 **설정 페이지**에 들어갈 수 있으며, 여기서 현재 사용 가능한 엔터테인먼트 모드의 영웅 **밸런스 속성**(있을 경우)을 확인하고, 각 모드에서 사용할 수 있는 룬 구성과 클라이언트 독립 설정을 지정할 수 있습니다.

## 자동 게임 종료
간단하게, 게임 종료 후 크리스탈 파괴 애니메이션이 재생될 때 게임 프로세스를 자동으로 종료합니다.

## 자동 재접속
게임에서 재접속이 필요할 때 자동으로 재접속을 시도합니다.

## 자동 카드 선택(확장 기능)
현재 게임에서 사용하는 영웅이 트위스티드 페이트일 때, Z/X/C 키를 사용하여 파란/빨간/노란 카드를 자동으로 선택합니다.

![pickCards](https://github.com/user-attachments/assets/df5aa45a-8593-44c9-8c3f-bec315552113)

> 이 기능은 그래픽 처리와 키보드 시뮬레이션 방식으로 구현되었으며, 현재 자주 사용하는 두 해상도(1920x1080/2560x1440)만 테스트하였습니다. 별도 설정 없이 다른 해상도에서는 정상적으로 사용할 수 없습니다(특히 16:9 비율이 아닌 해상도). 탐지 시 일정한 필터 범위를 허용하지만, 기능 사용 시 필터를 켜지 않는 것을 권장합니다. 필터로 인해 탐지가 무효화될 수 있습니다.
