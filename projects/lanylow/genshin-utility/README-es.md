# Genshin Utility [![License](https://img.shields.io/badge/License-GPL3.0-green.svg)](https://github.com/lanylow/genshin-utility/blob/main/LICENSE) ![OS](https://img.shields.io/badge/OS-Windows-yellow.svg) [![Discord](https://img.shields.io/badge/chat-discord-informational)](https://discord.gg/MrtJvV5tKv)

Herramienta gratuita y de código abierto para Genshin Impact y Honkai: Star Rail que incluye algunas características de calidad de vida. Compatible solo con la versión para OS del juego.

## FAQ

### ¿Cómo cargar el mod?
1. Descarga y extrae los binarios desde la [última versión](https://github.com/lanylow/genshin-utility/releases).
2. Abre `loader.exe` **como administrador**, debería aparecer una ventana informando que está esperando el juego.
3. Ahora con eso en ejecución, abre Genshin Impact o Honkai: Star Rail.
4. El juego se abrirá y el mod se cargará.

### ¿Cómo abro y cierro el menú?
Por defecto, la tecla usada para alternar el menú es <kbd>INSERT</kbd>, sin embargo puedes cambiarla modificando el archivo de configuración.

### ¿Dónde se guarda mi configuración?
La carpeta `genshin-utility` dentro de tu directorio Documentos contiene el archivo de configuración guardado en formato `.ini`. Después de algunas actualizaciones, puede ser necesario eliminar y recrear el archivo de configuración.

### ¿Qué hago si no aparece el menú de Genshin Utility?
La causa más común de este problema son las superposiciones de terceros. Asegúrate de que programas como MSI Afterburner (RTSS) o Discord Game Overlay no estén adjuntos al juego, porque podrían interferir con el menú de esta herramienta. Otra posible causa es que no estés ejecutando el juego a través del lanzador, este programa no funcionará si abres el ejecutable del juego directamente.
