- `giscus`를 이용한 댓글 및 토론 기능 🌈
- `mailerlite`를 통한 이메일 구독 및 발송 📮
- 본 프로젝트는 `netlify`에 호스팅되어 있으며, `crawler`를 활용해 문서를 크롤링하여 `algolia` 검색에 필요한 `schema`로 저장함

### 로컬 개발

```bash

pnpm run install

pnpm run start
```
