- Trang web cá nhân phát triển bằng `Astro`
- Sử dụng `algolia` để tìm kiếm 🔍
- Sử dụng `giscus` cho bình luận, thảo luận 🌈
- Sử dụng `mailerlite` để gửi đăng ký email 📮
- Dự án này được lưu trữ trên `netlify`, dựa vào `crawler` để thu thập tài liệu và lưu trữ `schema` tìm kiếm cần thiết cho `algolia`

### Phát triển cục bộ

```bash

pnpm run install

pnpm run start
```
