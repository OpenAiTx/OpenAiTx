# GIMP 3용 포토샵 키맵

GIMP는 포토샵 대체제가 아닙니다.

이 확장 기능은 인기 있는 포토샵 키보드 단축키를 GIMP 3에 이식합니다.

확장 기능을 설치하고 GIMP를 재시작하면 포토샵에서 즐겨 사용하던 키보드 단축키를 사용할 수 있습니다.

## 수동 설치

다운로드, 복사, 붙여넣기...

리눅스: `~/.config/GIMP/3.*` 

맥: `~/Application Support/GIMP/3.*` 

윈도우: `%appdata%\GIMP\3.*\`

(*을 버전 번호로 교체, 예: gimp 3.0은 '0')


*포토샵 CC 2019 기준.

## 파일
| 명령어           | 키 바인딩                      |   | 포토샵     |
|:-----------------|:-------------------------------|---|-----------|
| 새로 만들기      | `Ctrl` + `N`                   |   |           |
| 열기             | `Ctrl` + `O`                   |   |           |
| 레이어로 열기    | `Ctrl` + `Shift` + `O`         |   |           |
| 저장             | `Ctrl` + `S`                   |   |           |
