# Kodi SponsorBlock

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/siku2/script.service.sponsorblock)](https://github.com/siku2/script.service.sponsorblock/releases/latest)
[![gitlocalized](https://gitlocalize.com/repo/4205/whole_project/badge.svg)](https://gitlocalize.com/repo/4205)

这是[ SponsorBlock ](https://sponsor.ajay.app/)浏览器扩展的**非官方**移植版本。
它作为[YouTube 插件](https://github.com/anxdpanic/plugin.video.youtube)的扩展工作。

安装后，插件会自动跳过你观看的所有 YouTube 和 Invidious 视频中的赞助片段。

有关 SponsorBlock 工作原理的详细说明，请访问[官方网站](https://sponsor.ajay.app/)。

## 安装

该插件可在[siku2 的仓库](https://siku2.io/kodi-repository)中获得。

1. 安装[siku2 的仓库](https://siku2.io/kodi-repository/install)
2. 进入“插件” > “从仓库安装” > “siku2 的仓库”
3. 在“服务”下安装“SponsorBlock”

恭喜，你现在已安装 SponsorBlock。

## 配置

根据你的[设置级别](https://kodi.wiki/view/Settings#Settings_Level)会显示不同的设置。
例如，只有当你的设置级别为“高级”或以上时，才能更改用户 ID。

### 分类

SponsorBlock 不仅能跳过赞助片段。
你可以在设置的“分类”部分调整你想跳过的片段类型。
默认设置是只跳过赞助片段。

### 导入你的账号

如果你已经在使用 SponsorBlock，可能想使用你现有的账号（用户 ID）。
有关操作，请参考[导入现有账号](https://github.com/siku2/script.service.sponsorblock/wiki/Import-an-existing-account)维基页面。
