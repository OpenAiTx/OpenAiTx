# Kodi SponsorBlock

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/siku2/script.service.sponsorblock)](https://github.com/siku2/script.service.sponsorblock/releases/latest)
[![gitlocalized](https://gitlocalize.com/repo/4205/whole_project/badge.svg)](https://gitlocalize.com/repo/4205)

これは[スポンサー区間スキップ拡張機能SponsorBlock](https://sponsor.ajay.app/)の**非公式**ポートです。  
[YouTubeプラグイン](https://github.com/anxdpanic/plugin.video.youtube)の拡張として動作します。

インストールすると、視聴するすべてのYouTubeおよびInvidious動画のスポンサー区間を自動的にスキップします。

SponsorBlockの仕組みについて詳しく知りたい方は、[公式ウェブサイト](https://sponsor.ajay.app/)をご覧ください。

## インストール

アドオンは[siku2のリポジトリ](https://siku2.io/kodi-repository)で利用可能です。

1. [siku2のリポジトリ](https://siku2.io/kodi-repository/install)をインストールする  
2. 「アドオン」 > 「リポジトリからインストール」 > 「siku2のリポジトリ」へ進む  
3. 「サービス」内の「SponsorBlock」をインストールする

おめでとうございます。SponsorBlockがインストールされました。

## 設定

[設定レベル](https://kodi.wiki/view/Settings#Settings_Level)に応じて異なる設定が表示されます。  
たとえば、ユーザーIDは設定レベルが「上級」以上でのみ変更可能です。

### カテゴリ

SponsorBlockはスポンサー区間以外の区間もスキップ可能です。  
スキップしたい区間は設定の「カテゴリ」セクションで調整できます。  
デフォルト設定ではスポンサー区間のみをスキップします。

### アカウントのインポート

既にSponsorBlockを使用している場合は既存のアカウント（ユーザーID）を使うこともできます。  
詳しくは[既存アカウントのインポート](https://github.com/siku2/script.service.sponsorblock/wiki/Import-an-existing-account)のWikiページを参照してください。
