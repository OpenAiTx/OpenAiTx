# Kodi SponsorBlock

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/siku2/script.service.sponsorblock)](https://github.com/siku2/script.service.sponsorblock/releases/latest)
[![gitlocalized](https://gitlocalize.com/repo/4205/whole_project/badge.svg)](https://gitlocalize.com/repo/4205)

Ceci est un port **non officiel** de l’extension de navigateur [SponsorBlock](https://sponsor.ajay.app/).
Il fonctionne comme une extension du [plugin YouTube](https://github.com/anxdpanic/plugin.video.youtube).

Une fois installé, le module complémentaire sautera automatiquement les segments sponsorisés dans toutes les vidéos YouTube et Invidious que vous regardez.

Pour une explication détaillée du fonctionnement de SponsorBlock, veuillez visiter le [site officiel](https://sponsor.ajay.app/).

## Installation

Le module complémentaire est disponible dans le [dépôt de siku2](https://siku2.io/kodi-repository).

1. Installez le [dépôt de siku2](https://siku2.io/kodi-repository/install)
2. Allez dans « Modules complémentaires » > « Installer depuis un dépôt » > « Dépôt de siku2 »
3. Installez « SponsorBlock » sous « Services »

Félicitations, vous avez maintenant SponsorBlock installé.

## Configuration

Différents paramètres sont affichés selon votre [Niveau de paramètres](https://kodi.wiki/view/Settings#Settings_Level).
Par exemple, vous ne pouvez modifier votre identifiant utilisateur que si votre niveau de paramètres est « Avancé » ou supérieur.

### Catégories

SponsorBlock est capable de sauter plus que des segments sponsorisés.
Vous pouvez ajuster les segments que vous souhaitez faire sauter dans la section « Catégories » des paramètres.
Le réglage par défaut est de ne sauter que les segments sponsorisés.

### Importer votre compte

Si vous utilisez déjà SponsorBlock, vous pourriez vouloir utiliser votre compte existant (identifiant utilisateur).
Pour ce faire, référez-vous à la page wiki [Importer un compte existant](https://github.com/siku2/script.service.sponsorblock/wiki/Import-an-existing-account).

## Aider au projet

Vous pouvez aider à traduire le module complémentaire dans différentes langues.  
La localisation se fait via [GitLocalize](https://gitlocalize.com/).  
Vous pouvez commencer à traduire [ici](https://gitlocalize.com/repo/4205) (vous pouvez vous connecter avec votre compte GitHub).  
Si une langue manque, n’hésitez pas à demander son ajout en ouvrant un nouveau ticket.

N’hésitez pas à [ouvrir un nouveau ticket](https://github.com/siku2/script.service.sponsorblock/issues/new) si vous avez rencontré un problème ou si vous avez une suggestion.  
Des points bonus si vous [fournissez les journaux de débogage](https://kodi.wiki/view/Log_file/Easy).
