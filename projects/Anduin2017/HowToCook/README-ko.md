최근 집에서 요리를 하면서, 프로그래머로서 가끔 인터넷에서 레시피와 조리법을 찾아보곤 합니다. 하지만 이런 레시피들은 대개 작성 방식이 제각각이고, 중간에 갑자기 재료가 추가되는 경우도 많아, 형식 언어에 익숙한 프로그래머에게는 매우 불친절합니다.

그래서 저는 직접 레시피를 찾아 실제 요리 경험과 결합해, 더 명확하고 정확한 설명으로 흔히 먹는 요리의 조리법을 정리하려 합니다. 이를 통해 프로그래머가 집에서 더 쉽게 요리할 수 있도록 돕고자 합니다.

또한, 이 프로젝트가 커뮤니티가 주도하고 유지하는 오픈소스 프로젝트가 되기를 희망합니다. 더 많은 분들이 함께 재미있는 저장소를 만들어갈 수 있도록 여러분의 기여를 환영합니다~

## 로컬 배포

레시피 웹 서비스를 로컬에 배포하려면, Docker를 설치한 후 아래 명령어를 실행하세요:

```bash
docker pull ghcr.io/anduin2017/how-to-cook:latest
docker run -d -p 5000:5000 ghcr.io/anduin2017/how-to-cook:latest
```

PDF 버전을 다운로드하려면 브라우저에서 [/document.pdf](https://cook.aiursoft.cn/document.pdf)로 접속하세요.
