最近宅在家煮飯，作為程式設計師，我偶爾會在網路上查找食譜和做法。但這些食譜往往寫得千奇百怪，經常中間莫名其妙出現一些食材。對於習慣了形式語言的程式設計師來說極不友善。

所以，我計劃自己搜集食譜並結合實際下廚經驗，準備用更清晰精確的描述來整理常見菜餚的做法，方便程式設計師在家做飯。

同時，我希望這是一個由社群驅動和維護的開源專案，讓更多人能一起打造一個有趣的倉庫。所以非常歡迎大家貢獻～

## 本機部署

如果需要在本機部署食譜 Web 服務，可於安裝 Docker 後執行下列指令：

```bash
docker pull ghcr.io/anduin2017/how-to-cook:latest
docker run -d -p 5000:5000 ghcr.io/anduin2017/how-to-cook:latest
```

如需下載 PDF 版本，可在瀏覽器中開啟 [/document.pdf](https://cook.aiursoft.cn/document.pdf)

## 如何貢獻

針對發現的問題，直接修改並提交 Pull request 即可。

撰寫新食譜時，請複製並修改現有的食譜模板: [範例菜](https://github.com/Anduin2017/HowToCook/blob/master/dishes/template/%E7%A4%BA%E4%BE%8B%E8%8F%9C/%E7%A4%BA%E4%BE%8B%E8%8F%9C.md?plain=1)。
