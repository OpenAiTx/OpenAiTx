# MusicDJ 重制版

这是对2000年代索尼爱立信设备上同名应用的重制。

![MusicDJ 图片](https://user-images.githubusercontent.com/49322676/198876697-d3a7e75d-c21b-44d5-90d2-3104591055af.jpg)


## 功能
* 使用原始采样创建歌曲
* 打开原始 MusicDJ MIDI 文件
* 导出歌曲为 .wav 格式
* 提供暗色和白色主题，未来将增加更多

## 计划新增
* 更高质量的采样
* 自定义节奏
