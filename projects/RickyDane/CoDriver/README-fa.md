  - Щ…ШіЫҢШұЩҮШ§ ШұШ§ ШЁЩҮ ШҜЩ„Ш®ЩҲШ§ЩҮ Ш®ЩҲШҜ ШҜШұ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Ъ©ЩҶЫҢШҜ

  - Ш¬ШіШӘШ¬ЩҲЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ШЁШ§ F8
  - Ъ©ЩҫЫҢ Ш№ЩҶШөШұ Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ЩҒШ№Щ„ЫҢ ШЁЩҮ ЩҫЩҶШ¬ШұЩҮ ШҜЫҢЪҜШұ ШЁШ§ F5
  - Ш§ЩҶШӘЩӮШ§Щ„ Ш№ЩҶШөШұ Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ЩҒШ№Щ„ЫҢ ШЁЩҮ ЩҫЩҶШ¬ШұЩҮ ШҜЫҢЪҜШұ ШЁШ§ LShift + F5

  - Ш§Ш¬ШұШ§ЫҢ ШӘШәЫҢЫҢШұ ЩҶШ§Щ… ЪҜШұЩҲЩҮЫҢ ШЁШ§ Ctrl / Cmd + Return

## рҹ–ҘпёҸ Ш§ШҜШәШ§Щ… FTP / SFTP

CoDriver ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ъ©Ш§Щ…Щ„Ш§ЩӢ ШЁЩҲЩ…ЫҢ ЩҲ ШўЩ…Ш§ШҜЩҮ ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„вҖҢЩҮШ§ЫҢ ШұШ§ЩҮ ШҜЩҲШұ FTP ЩҲ SFTP ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҮЫҢЪҶ ЩҲШ§ШЁШіШӘЪҜЫҢ Ш®Ш§ШұШ¬ЫҢШҢ Щ„Ш§ЫҢЩҮвҖҢЩҮШ§ЫҢ FUSE ЫҢШ§ Щ…Ш§ЩҶШӘвҖҢЩҮШ§ЫҢ `sshfs` Щ„Ш§ШІЩ… ЩҶЫҢШіШӘ! ЩҒЩӮШ· Ъ©Ш§ЩҒЫҢШіШӘ ШіШұЩҲШұЩҮШ§ЫҢ ШұШ§ЩҮ ШҜЩҲШұ Ш®ЩҲШҜ ШұШ§ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШҜШұ ЩҫЩҶЩ„ Ъ©ЩҶШ§ШұЫҢ Щ…ШӘШөЩ„ Ъ©ЩҶЫҢШҜ.

## рҹҸҙвҖҚвҳ пёҸ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШІШЁШ§ЩҶ
- Ш§ЩҶЪҜЩ„ЫҢШіЫҢ
  - ШЁЩҮ ШІЩҲШҜЫҢ ЪҜШІЫҢЩҶЩҮ Ш§ЩҶШӘШ®Ш§ШЁ ШЁЫҢЩҶ ШІШЁШ§ЩҶвҖҢЩҮШ§ Ш§Ш¶Ш§ЩҒЩҮ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ ...

## вҡ пёҸ Щ…ШҙЪ©Щ„Ш§ШӘ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ:
- Ъ©ШҙЫҢШҜЩҶ ЩҲ ШұЩҮШ§ Ъ©ШұШҜЩҶ Ш®Ш§ШұШ¬ Ш§ШІ ЩҫЩҶШ¬ШұЩҮ ШҜШұ Щ„ЫҢЩҶЩҲЪ©Ші ЩҮЩ…ЩҲШ§ШұЩҮ Ъ©Ш§Шұ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ
- ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170) ШұШ§ ЩҶШөШЁ Ъ©ЩҶЫҢШҜ
- Щ…Ш¬ЩҲШІЩҮШ§ ШҜШұ ms-windows Ъ©Щ…ЫҢ Ш№Ш¬ЫҢШЁ ЩҮШіШӘЩҶШҜ
  - Ш§ЪҜШұ ШҜШұ Ъ©ЩҫЫҢ Ъ©ШұШҜЩҶ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ЫҢШ§ Щ…ЩҲШ§ШұШҜ Щ…ШҙШ§ШЁЩҮ Щ…ШҙЪ©Щ„ ШҜШ§ШҙШӘЫҢШҜШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…ШҜЫҢШұ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ
- Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ openssl1.1 ШұШ§ ШҜШұ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Щ„ЫҢЩҶЩҲЪ©Ші ЩҶШөШЁ Ъ©ЩҶЫҢШҜ Ш§ЪҜШұ ШЁШұЩҶШ§Щ…ЩҮ Ш§Ш¬ШұШ§ ЩҶШҙШҜ

## рҹ“қ Ъ©Ш§ШұЩҮШ§ЫҢ ШЁШ§ЩӮЫҢвҖҢЩ…Ш§ЩҶШҜЩҮ:
- ЪҶЩҶШҜ ШІШЁШ§ЩҶЩҮ ШҙШҜЩҶ
- Ш№Щ„Ш§ЩӮЩҮвҖҢЩ…ЩҶШҜЫҢвҖҢЩҮШ§
- ШҜШіШӘШұШіЫҢ ШЁЩҮ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ШўЩҶЩ„Ш§ЫҢЩҶ (ЪҜЩҲЪҜЩ„ ШҜШұШ§ЫҢЩҲ ЩҲ ШәЫҢШұЩҮ)

## ЪҶЪҜЩҲЩҶЩҮ Щ…ШҙШ§ШұЪ©ШӘ Ъ©ЩҶЫҢЩ…

</br></br>
ЩҫШі Ш§ШІ Ш§ЩҶШ¬Ш§Щ… Ш§ЫҢЩҶ Ъ©Ш§Шұ ЩҒЩӮШ· Ъ©Ш§ЩҒЫҢШіШӘ ШҜШіШӘЩҲШұ ```git clone https://github.com/RickyDane/CoDriver``` ЫҢШ§ ```gh repo clone RickyDane/CoDriver``` ШұШ§ ШҜШұ Щ…ШӯЩ„ЫҢ Ш§ШІ ШіЫҢШіШӘЩ… Ш®ЩҲШҜ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.
</br></br>
ШЁШ§ЫҢШҜ ШЁШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ Ш§Ш¬ШұШ§ЫҢ ШҜШіШӘЩҲШұ ```cargo tauri dev``` ШҜШұ ЩҫЩҲШҙЩҮ Ш§ШөЩ„ЫҢ ЩҫШұЩҲЪҳЩҮШҢ CoDriver ШұШ§ Ш§Ш¬ШұШ§ ЩҲ ШӘЩҲШіШ№ЩҮ ШҜЩҮЫҢШҜ.
</br>

</br>
