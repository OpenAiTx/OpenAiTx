## ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫШ§ЫҢЩҮ
- Щ…ШұЩҲШұ ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢвҖҢЩҮШ§ ШЁЩҮ ШұЩҲШҙЫҢ Ъ©ЩҮ ШЁШ§ ШўЩҶ ШўШҙЩҶШ§ ЩҮШіШӘЫҢШҜ
- Ъ©ЩҫЫҢ ЩҲ Ш¬Ш§ЫҢвҖҢЪҜШ°Ш§ШұЫҢШҢ ШӯШ°ЩҒШҢ Ш§ЫҢШ¬Ш§ШҜ ЩҲ ШӘШәЫҢЫҢШұ ЩҶШ§Щ… ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ЩҲ ЩҫЩҲШҙЩҮвҖҢЩҮШ§
- Ш¬Ш§ШЁШ¬Ш§ЫҢЫҢ ШЁЫҢЩҶ ШӯШ§Щ„ШӘвҖҢЩҮШ§ЫҢ ШҙШЁЪ©ЩҮвҖҢШ§ЫҢШҢ Щ„ЫҢШіШӘЫҢ ЩҲ ШіШӘЩҲЩҶвҖҢЩҮШ§ЫҢ Щ…ЫҢЩ„Шұ
- ШЁШіШӘЩҶ ЩҫЩҶШ¬ШұЩҮвҖҢЩҮШ§ЫҢ ШЁШ§ШІШҙЩҲ ШЁШ§ Ъ©Щ„ЫҢШҜ esc
- ЩҫШұШҙ Щ…ШіШӘЩӮЫҢЩ… ШЁЩҮ ЫҢЪ© ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢ ШЁШ§ Ctrl / Cmd + G ШЁШ§ ЩҲШ§ШұШҜ Ъ©ШұШҜЩҶ Щ…ШіЫҢШұ
- Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ШўЫҢШӘЩ…вҖҢЩҮШ§ ШҜШұ ШӯШ§Щ„ШӘ Щ„ЫҢШіШӘЫҢ ШЁШұ Ш§ШіШ§Ші Ш§ЩҶШҜШ§ШІЩҮШҢ ЩҶШ§Щ… ЫҢШ§ ШўШ®ШұЫҢЩҶ ШӘШәЫҢЫҢШұ

## ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ
- ЩҒШҙШұШҜЩҮвҖҢШіШ§ШІЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ЩҲ ЩҫЩҲШҙЩҮвҖҢЩҮШ§
  - zip
  - zstd
  - brotli
  - density (https://github.com/g1mv/density)
- Ш§ШіШӘШ®ШұШ§Ш¬ ШўШұШҙЫҢЩҲЩҮШ§
  - rar
  - zip
  - 7zip
  - tar (.gz, .bz2)
  - density
- ШұЩҒШӘЩҶ ШЁЩҮ ЫҢЪ© ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢ ШЁШ§ Щ…ЫҢШ§ЩҶШЁШұ LAlt + 1 / 2 / 3 | (ШҜШұ Щ…Ъ©вҖҢШ§ЩҲвҖҢШ§Ші option + 1 / 2 / 3)
  - Щ…ШіЫҢШұЩҮШ§ ШұШ§ ШЁЩҮ ШҜЩ„Ш®ЩҲШ§ЩҮ Ш®ЩҲШҜ ШҜШұ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Ъ©ЩҶЫҢШҜ
- Ш§ЫҢШ¬Ш§ШҜ ЩҒШ§ЫҢЩ„ ШЁШ§ F6
- Ш§ЫҢШ¬Ш§ШҜ ЩҫЩҲШҙЩҮ ШЁШ§ F7
- ЩҶЩ…Ш§ЫҢ ШҜЩҲ ЩҫЩҶШ¬ШұЩҮвҖҢШ§ЫҢ
  - Ш¬ШіШӘШ¬ЩҲЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ШЁШ§ F8
  - Ъ©ЩҫЫҢ Ш№ЩҶШөШұ Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ЩҒШ№Щ„ЫҢ ШЁЩҮ ЩҫЩҶШ¬ШұЩҮ ШҜЫҢЪҜШұ ШЁШ§ F5
  - Ш§ЩҶШӘЩӮШ§Щ„ Ш№ЩҶШөШұ Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ЩҒШ№Щ„ЫҢ ШЁЩҮ ЩҫЩҶШ¬ШұЩҮ ШҜЫҢЪҜШұ ШЁШ§ LShift + F5
- Ъ©ШҙЫҢШҜЩҶ ЩҲ ШұЩҮШ§ Ъ©ШұШҜЩҶ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ШЁЩҮ Ш§Ъ©ШіЩҫЩ„ЩҲШұШұ ШЁШұШ§ЫҢ Ъ©ЩҫЫҢ ШўЩҶвҖҢЩҮШ§ ШЁЩҮ ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢ ЩҒШ№Щ„ЫҢ
- ШӘШәЫҢЫҢШұ ЩҶШ§Щ… ЪҜШұЩҲЩҮЫҢ Ш§ЩҶШӘШ®Ш§ШЁвҖҢЩҮШ§ ШЁШ§ Ctrl / Cmd + LShift + M
  - Ш§Ш¬ШұШ§ЫҢ ШӘШәЫҢЫҢШұ ЩҶШ§Щ… ЪҜШұЩҲЩҮЫҢ ШЁШ§ Ctrl / Cmd + Return
- ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ШіШұЫҢШ№ ЩҒШ§ЫҢЩ„ -> Ш§ЩҶШӘШ®Ш§ШЁ ЩҲШұЩҲШҜЫҢ ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢ ЩҲ ШІШҜЩҶ Ъ©Щ„ЫҢШҜ ЩҒШ§ШөЩ„ЩҮ.
  - ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢвҖҢШҙШҜЩҮ: ЩҮЩ…ЩҮ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШӘШөЩҲЫҢШұЫҢ (.jpg, .png, ...)ШҢ .pdfШҢ .mp4ШҢ .jsonШҢ .txtШҢ .html
  - ШіШ§ЫҢШұ ШўЫҢШӘЩ…вҖҢЩҮШ§ ЫҢЪ© Ъ©Ш§ШҙЫҢ Ъ©ЩҲЪҶЪ© ШЁШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ ШҜШұШЁШ§ШұЩҮ ШўЩҶ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. (Щ…ШіЫҢШұШҢ Ш§ЩҶШҜШ§ШІЩҮШҢ ШўШ®ШұЫҢЩҶ ШӘШәЫҢЫҢШұ)
- ЩҶШ§ЩҲШЁШұЫҢ ЩҒЩҲШұЫҢ -> ШҙШұЩҲШ№ ШЁЩҮ ШӘШ§ЫҢЩҫ Ъ©ЩҶЫҢШҜ ЩҲ ЩҲШұЩҲШҜЫҢвҖҢЩҮШ§ЫҢ ШҜШ§ЫҢШұЪ©ШӘЩҲШұЫҢ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ЩҒЫҢЩ„ШӘШұ Ъ©ЩҶЫҢШҜ ШӘШ§ ЪҜШ§ЩҮЫҢ <br/>
  ШЁШіЫҢШ§Шұ ШіШұЫҢШ№вҖҢШӘШұ ШЁЩҮ Щ…Ъ©Ш§ЩҶ Щ…ЩҲШұШҜ ЩҶШёШұ ШЁШұШіЫҢШҜ

## вҡ пёҸ Щ…ШҙЪ©Щ„Ш§ШӘ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ:
- ЩӮШ§ШЁЩ„ЫҢШӘ Ъ©ШҙЫҢШҜЩҶ ЩҲ ШұЩҮШ§ Ъ©ШұШҜЩҶ Ш®Ш§ШұШ¬ Ш§ШІ ЩҫЩҶШ¬ШұЩҮ ШҜШұ Щ„ЫҢЩҶЩҲЪ©Ші ЩҮЩ…ЫҢШҙЩҮ ШЁЩҮ ШҜШұШіШӘЫҢ Ъ©Ш§Шұ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ
- ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170) ШұШ§ ЩҶШөШЁ Ъ©ЩҶЫҢШҜ
- ШіШ·Шӯ ШҜШіШӘШұШіЫҢвҖҢЩҮШ§ ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ Ъ©Щ…ЫҢ Ш№Ш¬ЫҢШЁ Ш§ШіШӘ
  - Ш§ЪҜШұ ЩҮЩҶЪҜШ§Щ… Ъ©ЩҫЫҢ Ъ©ШұШҜЩҶ Ш№ЩҶШ§ШөШұ ЫҢШ§ Ъ©Ш§ШұЩҮШ§ЫҢ Щ…ШҙШ§ШЁЩҮ ШЁШ§ Щ…ШҙЪ©Щ„ Щ…ЩҲШ§Ш¬ЩҮ ШҙШҜЫҢШҜШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…ШҜЫҢШұ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ
- Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁШ§ШҙШҜ openssl1.1 ШұШ§ ШҜШұ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Щ„ЫҢЩҶЩҲЪ©Ші ЩҶШөШЁ Ъ©ЩҶЫҢШҜШҢ Ш§ЪҜШұ ШЁШұЩҶШ§Щ…ЩҮ Ш§Ш¬ШұШ§ ЩҶШҙЩҲШҜ

## рҹ“қ Ъ©Ш§ШұЩҮШ§ЫҢ ШЁШ§ЩӮЫҢвҖҢЩ…Ш§ЩҶШҜЩҮ:
- ЪҶЩҶШҜШІШЁШ§ЩҶЩҮ ШҙШҜЩҶ
