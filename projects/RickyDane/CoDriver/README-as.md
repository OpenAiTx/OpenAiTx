## মুল বৈশিষ্ট্যসমূহ
- ডাইৰেক্টৰীসমূহত নেভিগেট কৰক যি দৰে আপুনি জানে
- কপি & পেষ্ট, ডিলিট, সৃষ্টি আৰু ফাইল আৰু ফোল্ডাৰ ৰেনেম কৰক
- গ্ৰিড, তালিকা আৰু মিলাৰ-কলাম মোডৰ মাজত সলনি কৰক
- esc দিয়ে পপআপ বন্ধ কৰক
- Ctrl / Cmd + G ব্যৱহাৰ কৰি এখন পথ ইনপুট কৰি ডাইৰেক্টৰীত সোজা যাওক
- তালিকা মোডত আইটেমসমূহক আকাৰ, নাম বা শেষ সংশোধন অনুসৰি সজাওক

## উন্নত বৈশিষ্ট্যসমূহ
- ফাইল আৰু ফোল্ডাৰসমূহ কম্প্ৰেছ কৰক
  - zip
  - zstd
  - brotli
  - density (https://github.com/g1mv/density)
- আর্কাইভ আনপেক কৰক
  - rar
  - zip
  - 7zip
  - tar (.gz, .bz2)
  - density
- LAlt + 1 / 2 / 3 | (macOS option + 1 / 2 / 3) চাৰ্টকাট ব্যৱহাৰ কৰি ডাইৰেক্টৰীত নেভিগেট কৰক
  - ছেটিংসমূহত নিজে পথসমূহ কনফিগাৰ কৰক
- F6 দিয়ে ফাইল সৃষ্টি কৰক
- F7 দিয়ে ফোল্ডাৰ সৃষ্টি কৰক
- ডুয়েল-পেন ভিউ
  - F8 দিয়ে ফাইল সন্ধান কৰক
  - F5 দিয়ে বৰ্তমান নিৰ্বাচিত উপাদান অন্য পেনত কপি কৰক
  - LShift + F5 দিয়ে বৰ্তমান নিৰ্বাচিত উপাদান অন্য পেনত স্থানান্তৰ কৰক
- ফাইলসমূহ explorer-ত drag আৰু drop কৰি বৰ্তমান ডাইৰেক্টৰীত কপি কৰক
- Ctrl / Cmd + LShift + M দিয়ে আপোনাৰ নিৰ্বাচিত আইটেমসমূহ একেলগে ৰেনেম কৰক
  - Ctrl / Cmd + Return দিয়ে multi rename চলাওক
- ফাইলৰ তৎক্ষণাৎ পূৰ্বদৰ্শন -> ডাইৰেক্টৰীৰ এন্ট্রি নিৰ্বাচন কৰক আৰু space bar চাপক।
  - সমৰ্থিত ফাইল: সকলো ইমেজ ফাইল (.jpg, .png, ...), .pdf, .mp4, .json, .txt, .html
  - অন্য সকলো আইটেমত ইয়াৰ বিষয়ে কিছু তথ্যসহ সৰু টাইল দেখিব। (পথ, আকাৰ, শেষ সংশোধন)
- তৎক্ষণাৎ নেভিগেশন -> টাইপ কৰা আৰম্ভ কৰক আৰু ডাইৰেক্টৰীৰ এন্ট্ৰিসমূহ অটোমেটিকেলি ফিল্টাৰ হব <br/>
  যাৰ ফলত প্ৰায়ে আপুনি ইচ্ছা কৰা স্থানত বহুত সোনকালে নেভিগেট কৰিব পাৰে

## 🖥️ FTP / SFTP সংযোগ

CoDriver-এ FTP আৰু SFTP ৰিম'ট সংযোগৰ বাবে সম্পূৰ্ণ দেশীয়, বক্সৰ পৰা বাহিৰে আহা সমৰ্থন আছে। বাহ্যিক নিৰ্ভৰশীলতা, FUSE স্তৰ, বা `sshfs` মাউণ্টৰ প্ৰয়োজন নাই! কেৱল আপোনাৰ ৰিম'ট ছাৰ্ভাৰসমূহক চাইডবাৰ পেনেলত সোঁমাজে সংযোগ কৰক।

## ⚠️ জনা সমস্যা:
- উইণ্ডোৰ পৰা টানিবলৈ আৰু পেলাবলৈ (drag and drop) লিনাক্সত সদায়েই কাম নকৰে
- উইণ্ডোজত আপোনাক [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170) ইনষ্টল কৰিব লাগিব পাৰে
- ms-windows ত অনুমতি (permissions) অলপ আচৰিত
  - উপাদান কপি কৰাত বা এনেধৰণৰ সমস্যাত আপোনাক প্ৰশাসক হিচাপে (administrator) প্ৰ'গ্ৰামটো চলাব লাগিব পাৰে
- লিনাক্স ছিষ্টেমত প্ৰ'গ্ৰামটো আৰম্ভ নোহোৱাত openssl1.1 ইনষ্টল কৰিব লাগিব পাৰে

## কেনেকৈ সহায় কৰিব
আপোনাৰ মেচিনত tauri v1 এপ্লিকেশ্যন ডেভেলপ কৰাৰ বাবে ছেটআপ কৰক: [Tauri prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites)
</br></br>
এইটো হোৱা পিছত কেৱল ```git clone https://github.com/RickyDane/CoDriver``` বা ```gh repo clone RickyDane/CoDriver``` আপোনাৰ মেচিনৰ যিকোনো ঠাইত কৰক।
</br></br>
আপুনি এই প্ৰজেক্টৰ মূল ডিৰেক্টৰীত ```cargo tauri dev``` চলাই CoDriver নিৰ্মাণ আৰু চলাব পাৰিব।
</br>
নিশ্চিত কৰক যে tauri-cli ইনষ্টল কৰা আছে: ```cargo install tauri-cli```
</br>
