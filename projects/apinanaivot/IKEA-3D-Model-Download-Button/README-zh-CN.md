# IKEA 3D 模型下载器

这个 Tampermonkey 脚本在 IKEA 产品页面添加了一个下载按钮，允许您轻松保存 IKEA 家具和装饰的 .GLB 文件。它适用于不同语言版本的 IKEA 网站，并自动根据产品名称和颜色命名下载的文件。文件可以在 Blender 等 3D 软件中打开。这样您可以在购买前，先在自己的 3D 家居规划软件中试用家具。

更新：2025 年 11 月 28 日 - 适用于最新的 IKEA 网站

## 功能

- 在 IKEA 产品页面的“3D 查看”按钮旁添加“下载 3D”按钮
- 适用于所有语言版本的 IKEA 网站
- 自动使用产品名称和颜色命名下载的文件

## 安装

1. 为您的浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 点击[此链接](https://github.com/apinanaivot/IKEA-3D-Model-Download-Button/raw/refs/heads/main/ikea-3d-model-downloader.user.js)安装或更新此脚本，或者在 Tampermonkey 中创建新脚本并粘贴 `ikea-3d-model-downloader.user.js` 的内容。
3. 保存脚本并确保在 Tampermonkey 中启用它。

## 使用方法

1. 进入任何带有“3D 查看”按钮的 IKEA 产品页面。
2. 在“3D 查看”按钮旁会出现一个新的“下载 3D”按钮。
3. 点击“下载 3D”按钮即可下载 3D 模型的 GLB 文件。
4. 文件将以格式 `[产品名称] - [颜色].glb` 保存在本地。

## 故障排除

- 如果下载按钮未出现，请确认您在带有 3D 模型的产品页面，并刷新页面。
- 如果使用 Chrome，尝试开启开发者模式。
- 如果仍无法解决，欢迎在[此处提交问题](https://github.com/apinanaivot/IKEA-3D-Model-Download-Button/issues)，我会尽快修复。
