# IKEA 3Dモデルダウンローダー

このTampermonkeyスクリプトは、IKEAの製品ページに3Dモデルのダウンロードボタンを追加し、IKEAの家具や装飾品の.GLbファイルを簡単に保存できるようにします。IKEAの各言語版ウェブサイトで動作し、ダウンロードしたファイル名は製品名と色に基づいて自動的に命名されます。これらのファイルはBlenderなどの3Dソフトウェアで開くことができます。購入前に自分の3Dホームプランニングソフトで家具を試すことが可能です。

UPDATE: 2025年11月28日 - 最新のIKEAウェブサイトで動作確認済み

## 特徴

- IKEA製品ページの「3Dで見る」ボタンの隣に「3Dダウンロード」ボタンを追加
- IKEAの全言語版ウェブサイトで動作
- ダウンロードファイル名を製品名と色で自動命名

## インストール方法

1. ご使用のブラウザに[Tampermonkey](https://www.tampermonkey.net/)拡張機能をインストールします。
2. [こちらのリンク](https://github.com/apinanaivot/IKEA-3D-Model-Download-Button/raw/refs/heads/main/ikea-3d-model-downloader.user.js)をクリックしてスクリプトをインストールまたは更新するか、Tampermonkeyで新しいスクリプトを作成し、`ikea-3d-model-downloader.user.js`の内容を貼り付けます。
3. スクリプトを保存し、Tampermonkeyで有効になっていることを確認してください。

## 使い方

1. 「3Dで見る」ボタンがあるIKEA製品ページにアクセスします。
2. 「3Dで見る」ボタンの隣に新しい「3Dダウンロード」ボタンが表示されます。
3. 「3Dダウンロード」ボタンをクリックして、3DモデルのGLBファイルをダウンロードします。
4. ファイルは「[製品名] - [色].glb」という形式の名前で保存されます。

## トラブルシューティング

- ダウンロードボタンが表示されない場合は、3Dモデルが利用可能な製品ページにいることを確認し、ページを更新してください。
- Chromeを使用している場合は、開発者モードを有効にしてみてください。
- それでも問題が解決しない場合は、[こちら](https://github.com/apinanaivot/IKEA-3D-Model-Download-Button/issues)からバグレポートを作成してください。できるだけ早く対応します。
