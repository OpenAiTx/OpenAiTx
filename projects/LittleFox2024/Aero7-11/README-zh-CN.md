# Aero7-11
适用于 Windows 11 23H2 及以上版本的 Aero 7 主题！

</div>

> [!NOTE]
> Aero7-11 使用 Windows 11 默认的 `aero.msstyles` 制作。此主题在 Windows 11 23H2 和 24H2 上制作并测试。
> 
> Aero7-11 支持从 22H2 到 23H2。25H2 版本未测试，且 24H2 中的 DWM 修改可能存在一些小问题。

## 警告

> [!NOTE]
> 建议使用浅色应用模式和深色 Windows 模式。此主题*不支持*深色应用模式。Windows 模式无关紧要，但大多数任务栏修改在深色基础上表现更好。
>
> 24H2 版本已证实存在问题。如果你最近几个月更新过 Windows，任何 DWM 修改都有可能导致系统崩溃。请谨慎操作，应用 24H2 和 25H2 的修改时务必备份。

> [!WARNING]
> 请注意，使用此主题风险自负。我***不能***保证其与更新的兼容性，也不对任何损害负责。我仅负责视觉样式和指向第三方工具。唯一由我制作的修改是本仓库中的 Borders 修改，专为 23H2 制作。
>
> 如果你修改了任何系统文件，我***强烈建议***备份**所有**说明和文件（原始和新文件），以便系统崩溃时可恢复更改。

## 安装
本节假设你已经修补系统以允许第三方主题。如果尚未操作，请参考相关教程。你需要使用 UltraUXThemePatcher 或 SecureUxTheme 之类的工具。

- 以 ZIP 文件形式下载仓库并解压到你选择的文件夹。
- 将 [1 - Theme](./1%20-%20Theme/) 文件夹中的内容复制到 `%SYSTEMROOT%\Resources\Themes`。
- 按照 [2 - Borders](./%20-%20Borders/) 文件夹中的说明，使用 Windhawk 和 `Bring Back the Borders!` 模组设置边框。
- 按照 [3 - Glass](./3%20-%20Glass/) 文件夹中的说明，设置玻璃效果。
- （可选）打开 [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) 文件夹，探索附加主题和资源。
- 右键点击桌面，选择“个性化”，然后进入“主题”，选择所需主题以应用。
