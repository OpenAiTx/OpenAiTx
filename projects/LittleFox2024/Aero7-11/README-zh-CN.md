# Aero7-11
适用于 Windows 11 23H2+ 的 Aero 7 主题！

</div>

> [!NOTE]
> Aero7-11 是使用 Windows 11 默认的 `aero.msstyles` 制作的。该主题是在 Windows 11 23H2 和 24H2 上制作并测试的。
> 
> Aero7-11 支持 22H2 到 23H2。在 25H2 上尚未测试，并且在 24H2 上的 DWM 修改可能会有些不稳定。

## 警告

> [!NOTE]
> 我建议使用浅色应用模式和深色 Windows 模式。本主题*不*支持深色应用模式。Windows 模式无所谓，但大多数任务栏修改在深色基础上比浅色基础效果更好。
>
> 24H2 已经证明存在一些问题。如果你在最近几个月更新过 Windows，任何 DWM 修改都有可能导致系统损坏。请谨慎操作，并在 24H2 和 25H2 上应用修改时务必备份。

> [!WARNING]
> 请注意，使用本主题风险自负。我***无法***保证与后续更新兼容，也不对任何损坏负责。我只处理视觉样式，并指向第三方工具。我唯一制作的修改是本仓库中的 Borders 修改，仅适用于 23H2。
>
> 如果你修改了任何系统文件，我***强烈建议***将**所有**操作说明和文件（包括原始和新文件）备份在一个即使系统损坏也能访问的位置，以便可以撤销任何更改。

## 安装
本节假设你已经为系统打了补丁以允许第三方主题。如果尚未操作，请查阅相关指南。你需要使用如 UltraUXThemePatcher 或 SecureUxTheme 等工具。

- 下载本仓库为 ZIP 文件，并解压到你选择的文件夹。
- 将 [1 - Theme](./1%20-%20Theme/) 文件夹中的内容复制到 `%SYSTEMROOT%\Resources\Themes`。
- 按照 [2 - Borders](./%20-%20Borders/) 文件夹的说明，使用 Windhawk 和 `Bring Back the Borders!` 修改设置边框。
- 按照 [3 - Glass](./3%20-%20Glass/) 文件夹的说明设置玻璃效果。
- （可选）打开 [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) 文件夹，探索其他主题和资源。
- 右键点击桌面，选择 `个性化`，进入 `主题`，选择想要的主题以应用。
