# Aero7-11  
Thème Aero 7 pour Windows 11 23H2+ !

</div>

> [!NOTE]  
> Aero7-11 est réalisé en utilisant le style par défaut `aero.msstyles` de Windows 11. Il a été créé et testé sur Windows 11 23H2 et 24H2.  
>   
> Aero7-11 supporte de 22H2 à 23H2. Il n’a pas été testé sur 25H2, et les mods DWM peuvent être un peu bugués sur 24H2.

## Avertissements

> [!NOTE]  
> Je recommande d’utiliser le mode application clair et le mode Windows sombre. Ce thème *NE* supporte *PAS* le mode application sombre. Le mode Windows n’a pas d’importance, mais la plupart des mods de barre des tâches fonctionnent mieux sur une base sombre que claire.  
>   
> Le 24H2 a montré des problèmes. Si vous avez mis à jour Windows ces derniers mois, il y a un risque que des modifications DWM puissent casser votre système. Veuillez procéder avec prudence et garder des sauvegardes lorsque vous appliquez des mods sur 24H2 et 25H2.

> [!WARNING]  
> Veuillez noter que l’utilisation de ce thème est à vos risques et périls. Je ***ne peux pas*** garantir la compatibilité avec les mises à jour, et ne peux être tenu responsable de tout dommage. Je travaille uniquement avec le style visuel et en orientant vers des outils tiers. Le seul mod que j’ai réalisé est le mod des Bordures dans ce dépôt, qui est fait pour 23H2.  
>   
> Si vous modifiez des fichiers système, je ***recommande fortement*** de garder une sauvegarde de **TOUTES** les instructions et fichiers (originaux et nouveaux) dans un endroit accessible si votre système venait à être inutilisable afin de pouvoir annuler tout changement.

## Installation  
Cette section suppose que vous avez déjà patché votre système pour autoriser les thèmes tiers. Si ce n’est pas le cas, veuillez consulter des guides pour le faire. Vous devrez utiliser un outil tel que UltraUXThemePatcher ou SecureUxTheme.

- Téléchargez le dépôt en fichier ZIP et extrayez-le dans un dossier de votre choix.  
- Copiez le contenu du dossier [1 - Theme](./1%20-%20Theme/) dans `%SYSTEMROOT%\Resources\Themes`.  
- Suivez les instructions dans le dossier [2 - Borders](./%20-%20Borders/) pour configurer les bordures avec Windhawk et le mod `Bring Back the Borders!`.  
- Suivez les instructions dans le dossier [3 - Glass](./3%20-%20Glass/) pour configurer les effets de verre.  
- (Optionnel) Ouvrez le dossier [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) pour explorer des thèmes et ressources supplémentaires.  
- Appliquez le thème en cliquant droit sur votre bureau, en sélectionnant `Personnaliser`, puis en naviguant dans `Thèmes` et en sélectionnant le thème désiré.

N'importe qui peut créer une copie ou redistribuer ceci. Cependant, la seule condition est qu'il doit rester gratuit et que toutes les personnes suivantes doivent être créditées pour leurs contributions à la personnalisation de Windows :

Je ne peux pas revendiquer le mérite de ce thème, sauf à quelques endroits. J'ai simplement pris les designs de Windows 7, les ai fusionnés avec les aero.msstyles de Windows 11, puis tout rassemblé en un seul endroit pour l'implémenter. Je fais cela dans l'espoir que cela aidera les utilisateurs de Windows 11 cherchant un bon thème Windows 7 pour leur système.
