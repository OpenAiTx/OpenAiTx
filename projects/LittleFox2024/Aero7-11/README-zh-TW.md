# Aero7-11
Aero 7 主題，適用於 Windows 11 23H2 以上版本！

</div>

> [!NOTE]
> Aero7-11 是使用 Windows 11 預設的 `aero.msstyles` 製作而成。本主題在 Windows 11 23H2 和 24H2 上製作並測試。
> 
> Aero7-11 支援 22H2 至 23H2。尚未在 25H2 測試，且 DWM 修改在 24H2 可能會有一些錯誤。

## 警告

> [!NOTE]
> 我建議使用淺色應用程式模式和深色 Windows 模式。本主題 *不* 支援深色應用程式模式。Windows 模式本身無影響，但多數工作列修改在深色基底上比淺色基底運作更佳。
>
> 24H2 已證實存在問題。如果你在最近幾個月有更新 Windows，任何 DWM 修改都有可能導致系統損壞。請謹慎操作並在 24H2 與 25H2 上套用修改前備份。

> [!WARNING]
> 請注意，使用此主題需自行承擔風險。我 ***無法*** 保證與更新相容，也不對任何損害負責。我僅處理視覺樣式，並指向第三方工具。唯一我製作的修改是本倉庫中的 Borders 修改，僅為 23H2 製作。
>
> 若你要修改任何系統檔案，我 ***強烈建議*** 備份**所有**說明及檔案（包含原始與新檔案），並存放在系統損壞時可存取的位置，以便還原。

## 安裝
本節假設你已經修補系統以允許第三方主題。如尚未修補，請參考相關指引。你需使用如 UltraUXThemePatcher 或 SecureUxTheme 等工具。

- 將本倉庫下載為 ZIP 檔並解壓縮到你選擇的資料夾。
- 將 [1 - Theme](./1%20-%20Theme/) 資料夾內容複製到 `%SYSTEMROOT%\Resources\Themes`。
- 依照 [2 - Borders](./%20-%20Borders/) 資料夾說明，使用 Windhawk 與 `Bring Back the Borders!` 修改設定邊框。
- 依照 [3 - Glass](./3%20-%20Glass/) 資料夾說明設定玻璃特效。
- （選用）開啟 [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) 資料夾以探索其他主題與資源。
- 於桌面按右鍵，選擇 `個人化`，再進入 `主題` 並選取欲套用主題。

## 致謝

本主題為「開放原始碼」，即任何人皆可自由使用及編輯。可惜僅有二進位格式，故無法釋出原始碼。我不設任何限制或收費。

任何人皆可複製或重新分發。但唯一條件是必須保持免費，且以下所有貢獻者必須被正確標註於 Windows 客製化：

- **teknixstuff**：在 Windhawk 創建了原始「Bring Back the Borders!」修改，基於 Ittr 的 uDWM 修補。
- **Zee-Ay (Za9-118)**：貢獻了 Shellstyle dll，並創建了 Vista 主題。
- **vaporvance**：在原始 Aero10 主題中創建了 Shellstyle dll。
- **Ittr**：創建了包含 uDWM.dll 修補程式。
- **SandTechStuff**：更新了說明以提升可讀性。
- **TheDolphinSlayer**：新增了 Windows 7 Alt Tab 資源。
- **OG Nimbi**：實作了 Optional Extras 資料夾，並在多個檔案中更新說明和格式，以提升可讀性與管理便利性。
- **所有 Windows 7 Aero 主題 for Windows 10/11 的開發者**：我從各種主題中取用了一些元素。

除了極少數部分，我不能將這個主題歸功於自己。我只是將 Windows 7 的設計，與 Windows 11 的 aero.msstyles 合併，然後把一切集中到一個地方實作出來。我這麼做，是希望能幫助正在尋找優質 Windows 7 主題的 Windows 11 使用者。
