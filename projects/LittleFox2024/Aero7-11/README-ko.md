# Aero7-11
Windows 11 23H2+용 Aero 7 테마!

</div>

> [!NOTE]
> Aero7-11은 Windows 11의 기본 `aero.msstyles`를 사용하여 제작되었습니다. 이 테마는 Windows 11 23H2 및 24H2에서 제작 및 테스트되었습니다.
>
> Aero7-11은 22H2부터 23H2까지 지원합니다. 25H2에서는 테스트되지 않았으며, DWM 모드는 24H2에서 약간 버그가 있을 수 있습니다.

## 경고

> [!NOTE]
> 앱 모드는 밝게, Windows 모드는 어둡게 사용하는 것을 권장합니다. 이 테마는 *어두운 앱 모드*를 지원하지 않습니다. Windows 모드는 중요하지 않지만, 대부분의 작업표시줄 모드는 밝은 기반보다 어두운 기반에서 더 잘 작동합니다.
>
> 24H2에서는 문제가 발생하는 것으로 확인되었습니다. 최근 몇 달 내에 Windows를 업데이트했다면, 어떤 DWM 수정이라도 시스템이 손상될 가능성이 있습니다. 24H2 및 25H2에서 모드를 적용할 때 신중히 진행하고 반드시 백업을 유지하세요.

> [!WARNING]
> 이 테마의 사용은 전적으로 본인 책임입니다. 업데이트와의 호환성을 ***보장할 수 없으며***, 발생한 피해에 대해 책임을 지지 않습니다. 저는 비주얼 스타일을 다루고 제3자 도구만 안내합니다. 이 저장소에서 제가 만든 유일한 모드는 23H2용 Borders 모드입니다.
>
> 시스템 파일을 수정하는 경우, ***반드시*** **모든** 지침과 파일(원본 및 새 파일)을 백업해 두고, 시스템이 망가졌을 때 접근 가능한 위치에 저장하는 것을 강력히 권장합니다. 그래야 변경 사항을 되돌릴 수 있습니다.

## 설치 방법
이 섹션은 이미 서드파티 테마가 적용될 수 있도록 시스템을 패치했다는 가정하에 진행합니다. 그렇지 않다면 적용 방법을 안내하는 가이드를 참고하세요. UltraUXThemePatcher 또는 SecureUxTheme 같은 도구가 필요합니다.

- 저장소를 ZIP 파일로 다운로드하여 원하는 폴더에 압축을 풉니다.
- [1 - Theme](./1%20-%20Theme/) 폴더의 내용을 `%SYSTEMROOT%\Resources\Themes`로 복사합니다.
- [2 - Borders](./%20-%20Borders/) 폴더의 지침을 따라 Windhawk와 `Bring Back the Borders!` 모드를 이용해 테두리를 설정합니다.
- [3 - Glass](./3%20-%20Glass/) 폴더의 지침을 따라 글래스 효과를 설정합니다.
- (선택 사항) [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) 폴더를 열어 추가 테마 및 리소스를 확인합니다.
- 바탕화면에서 우클릭 → `개인 설정` → `테마`로 이동하여 원하는 테마를 적용합니다.

## 크레딧

이 테마는 "오픈 소스"로 누구나 자유롭게 사용 및 수정할 수 있습니다. 다만, 이진 형식에 묶여 있어 별도의 코드는 제공하지 못합니다. 어떠한 제한이나 비용도 부과하지 않습니다.

누구든 복사하거나 재배포할 수 있습니다. 단, 반드시 무료로 유지되어야 하며, Windows 사용자 지정에 기여한 다음 모든 분들을 크레딧에 포함해야 합니다:

- **teknixstuff**: Windhawk에서 Ittr의 uDWM 패치를 기반으로 원본 "Bring Back the Borders!" 모드 제작.
- **Zee-Ay (Za9-118)**: Shellstyle dll을 기여하고 Vista 테마를 제작.
- **vaporvance**: 원본 Aero10 테마에서 Shellstyle dll 제작.
- **Ittr**: 포함된 uDWM.dll 패치를 제작했습니다.
- **SandTechStuff**: 가독성을 위해 지침을 업데이트했습니다.
- **TheDolphinSlayer**: Windows 7 Alt Tab 리소스를 추가했습니다.
- **OG Nimbi**: Optional Extras 폴더를 구현하고 여러 파일의 지침 및 형식을 업데이트하여 가독성과 관리 용이성을 높였습니다.
- **Windows 10/11용 모든 Windows 7 Aero 테마 개발자들**: 다양한 테마에서 일부 요소를 가져왔습니다.

이 테마에 대해 제가 인정받을 부분은 거의 없습니다. Windows 7 디자인을 가져와 Windows 11 aero.msstyles와 병합한 뒤, 모든 것을 한곳에 모아 구현했습니다. 이것이 Windows 11 사용자가 시스템에 적합한 Windows 7 테마를 찾는 데 도움이 되길 바랍니다.
