# Aero7-11
Windows 11 23H2+용 Aero 7 테마!

</div>

> [!NOTE]
> Aero7-11은 Windows 11의 기본 `aero.msstyles`를 사용하여 제작되었습니다. Windows 11 23H2와 24H2에서 제작 및 테스트되었습니다.
> 
> Aero7-11은 22H2부터 23H2까지 지원합니다. 25H2에서는 테스트되지 않았으며, 24H2에서는 DWM 모드가 다소 불안정할 수 있습니다.

## 경고

> [!NOTE]
> 밝은 앱 모드와 어두운 Windows 모드 사용을 권장합니다. 이 테마는 어두운 앱 모드를 지원하지 않습니다. Windows 모드는 상관없지만, 대부분의 작업 표시줄 모드는 밝은 기본보다 어두운 기본에서 더 잘 작동합니다.
>
> 24H2에서는 문제가 있는 것으로 확인되었습니다. 최근 몇 개월 내에 Windows를 업데이트한 경우, DWM 수정이 시스템에 문제를 일으킬 수 있으므로 주의해서 진행하시고 24H2 및 25H2에 모드를 적용할 때는 백업을 꼭 유지하시기 바랍니다.

> [!WARNING]
> 이 테마 사용은 사용자 책임임을 명심해 주십시오. 업데이트와의 호환성을 ***보장할 수 없으며***, 어떠한 손해에 대해서도 책임지지 않습니다. 저는 비주얼 스타일 작업과 서드파티 도구 안내만 진행합니다. 제가 만든 유일한 모드는 이 저장소 내의 Borders 모드로, 23H2용입니다.
>
> 시스템 파일을 수정하는 경우, 시스템이 손상될 시 되돌릴 수 있도록 **모든** 지침과 파일(원본 및 새 파일 모두)을 백업해 둘 것을 ***강력히 권장***합니다.

## 설치
이 섹션은 서드파티 테마 적용을 위해 시스템 패치를 이미 완료했다고 가정합니다. 완료하지 않았다면 관련 가이드를 참조하십시오. UltraUXThemePatcher 또는 SecureUxTheme 같은 도구를 사용해야 합니다.

- 저장소를 ZIP 파일로 다운로드하고 원하는 폴더에 압축을 풉니다.
- [1 - Theme](./1%20-%20Theme/) 폴더의 내용을 `%SYSTEMROOT%\Resources\Themes`에 복사합니다.
- [2 - Borders](./%20-%20Borders/) 폴더의 지침에 따라 Windhawk와 `Bring Back the Borders!` 모드를 사용해 테두리를 설정합니다.
- [3 - Glass](./3%20-%20Glass/) 폴더의 지침을 따라 글래스 효과를 설정합니다.
- (선택 사항) [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) 폴더를 열어 추가 테마와 리소스를 탐색합니다.
- 바탕화면을 우클릭하고 `개인 설정`을 선택한 후 `테마`에서 원하는 테마를 적용합니다.

## 크레딧
이 테마는 누구나 사용하고 편집할 수 있다는 점에서 "오픈 소스"입니다. 불행히도 이 테마는 바이너리 형식으로 되어 있어 제가 공개할 수 있는 코드는 없습니다. 저는 이에 대해 어떠한 제한이나 비용도 부과하지 않습니다.

누구나 이 테마를 복사하거나 재배포할 수 있습니다. 다만, 유일한 조건은 무료로 유지되어야 하며, 다음 모든 분들이 Windows 커스터마이징에 기여한 점에 대해 반드시 크레딧을 표기해야 한다는 것입니다:

- **teknixstuff**: Ittr의 uDWM 패치를 기반으로 Windhawk에서 원본 "Bring Back the Borders!" 모드를 제작했습니다.
- **Zee-Ay (Za9-118)**: Shellstyle dll을 기여했고, Vista 테마를 만들었습니다.
- **vaporvance**: 원본 Aero10 테마에서 Shellstyle dll을 만들었습니다.
- **Ittr**: 포함된 uDWM.dll 패치를 제작했습니다.
- **SandTechStuff**: 가독성을 위한 지침을 업데이트했습니다.
- **TheDolphinSlayer**: Windows 7 Alt Tab 리소스를 추가했습니다.
- **OG Nimbi**: Optional Extras 폴더를 구현하고 여러 파일의 지침과 포맷을 가독성과 관리 편의를 위해 업데이트했습니다.
- **All Windows 7 Aero theme for Windows 10/11 developers**: 다양한 테마에서 일부 요소를 가져왔습니다.

이 테마에 대해 제가 자랑할 만한 점은 거의 없습니다. 저는 단지 Windows 7 디자인을 가져와 Windows 11 aero.msstyles와 합친 후 모든 것을 한 곳에 모아 구현했을 뿐입니다. 이 작업이 Windows 11 사용자들이 좋은 Windows 7 테마를 찾는 데 도움이 되길 바랍니다.
