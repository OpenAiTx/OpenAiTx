# Aero7-11
Tema Aero 7 untuk Windows 11 23H2+!

</div>

> [!CATATAN]
> Aero7-11 dibuat menggunakan `aero.msstyles` bawaan di Windows 11. Tema ini dibuat dan diuji pada Windows 11 23H2 dan 24H2.
> 
> Aero7-11 mendukung 22H2 hingga 23H2. Belum diuji di 25H2, dan mod DWM mungkin agak buggy di 24H2.

## Peringatan

> [!NOTE]
> Saya merekomendasikan menggunakan mode aplikasi terang dan mode Windows gelap. Tema ini *TIDAK* mendukung mode aplikasi gelap. Mode Windows tidak berpengaruh, tetapi sebagian besar mod taskbar bekerja lebih baik pada basis gelap daripada basis terang.
>
> 24H2 terbukti memiliki masalah. Jika Anda memperbarui Windows dalam beberapa bulan terakhir, ada kemungkinan modifikasi DWM dapat merusak sistem Anda. Harap lanjutkan dengan hati-hati dan buat cadangan sebelum menerapkan mod pada 24H2 dan 25H2.

> [!WARNING]
> Harap diperhatikan bahwa penggunaan tema ini adalah risiko Anda sendiri. Saya ***TIDAK*** dapat menjamin kompatibilitas dengan pembaruan, dan tidak bertanggung jawab atas kerusakan apapun. Saya hanya bekerja dengan Visual Style dan mengarahkan ke alat pihak ketiga saja. Satu-satunya mod yang saya buat adalah Borders mod di repository ini, yang dibuat untuk 23H2.
>
> Jika Anda memodifikasi file sistem apapun saya ***sangat merekomendasikan*** untuk menyimpan cadangan **SEMUA** instruksi dan file (baik asli maupun baru) di tempat yang dapat diakses jika sistem Anda rusak sehingga Anda dapat membatalkan perubahan.

## Instalasi
Bagian ini mengasumsikan Anda sudah menambal sistem agar dapat menggunakan tema pihak ketiga. Jika belum, silakan lihat panduan cara melakukannya. Anda perlu menggunakan alat seperti UltraUXThemePatcher atau SecureUxTheme.

- Unduh repository sebagai file ZIP dan ekstrak ke folder pilihan Anda.
- Salin isi folder [1 - Theme](./1%20-%20Theme/) ke `%SYSTEMROOT%\Resources\Themes`.
- Ikuti petunjuk di folder [2 - Borders](./%20-%20Borders/) untuk mengatur borders menggunakan Windhawk dan mod `Bring Back the Borders!`.
- Ikuti petunjuk di folder [3 - Glass](./3%20-%20Glass/) untuk mengatur efek kaca.
- (Opsional) Buka folder [4 - (Optional) Extras](./4%20-%20(Optional)%20Extras/) untuk menjelajahi tema dan sumber daya tambahan.
- Terapkan tema dengan klik kanan di desktop, pilih `Personalize`, lalu masuk ke `Themes` dan pilih tema yang diinginkan.

## Kredit

Tema ini "Open Source" artinya terbuka untuk siapapun digunakan dan diedit. Sayangnya masih dalam format biner, jadi tidak ada kode yang bisa saya rilis. Saya tidak memberikan pembatasan atau pembayaran apapun.

Siapapun dapat membuat salinan atau mendistribusikan ulang. Namun, satu-satunya syarat adalah harus tetap gratis dan semua orang berikut harus dikreditkan atas kontribusi mereka pada kustomisasi Windows:

- **teknixstuff**: Membuat mod "Bring Back the Borders!" asli di Windhawk, berdasarkan patch uDWM dari Ittr.
- **Zee-Ay (Za9-118)**: Menyumbangkan Shellstyle dll, dan membuat Tema Vista.
- **vaporvance**: Membuat Shellstyle dll dalam tema Aero10 asli.
- **Ittr**: Membuat patch uDWM.dll yang disertakan.
- **SandTechStuff**: Memperbarui instruksi agar lebih mudah dibaca.
- **TheDolphinSlayer**: Menambahkan sumber daya alt tab Windows 7.
- **OG Nimbi**: Mengimplementasikan folder Optional Extras dan memperbarui instruksi serta format di berbagai file untuk keterbacaan dan kemudahan pengelolaan.
- **Semua pengembang tema Windows 7 Aero untuk Windows 10/11**: Saya mengambil bagian dari berbagai tema tersebut.

Saya tidak dapat mengambil kredit penuh untuk tema ini, kecuali di beberapa bagian saja. Saya hanya mengambil desain Windows 7, menggabungkannya dengan Windows 11 aero.msstyles, lalu mengumpulkan semuanya menjadi satu untuk diimplementasikan. Saya melakukan ini dengan harapan dapat membantu pengguna Windows 11 yang mencari tema Windows 7 yang bagus untuk sistem mereka.
