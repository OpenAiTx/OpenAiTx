Maha는 아랍어 텍스트 처리를 위해 특별히 개발된 텍스트 처리 라이브러리입니다. 베타 버전은 스트리밍 기능 유무에 상관없이 텍스트, 파일 및 폴더를 정리하고 구문 분석하는 데 사용할 수 있습니다.

도움이 필요하거나 Maha와 관련된 주제를 논의하고 싶다면, 언제든지 저희 [Discord](https://discord.gg/6W2tRFE7k4) 서버에 접속하세요. 버그 리포트나 기능 요청을 제출하고 싶다면, 이슈를 열어주시기 바랍니다.

## 설치

Maha를 설치하려면 다음 명령어를 실행하세요:

```bash
pip install mahad # pronounced maha d
```
소스 설치에 대해서는 [문서](https://maha.readthedocs.io/en/latest/contributing/guidelines.html)를 확인하세요.
