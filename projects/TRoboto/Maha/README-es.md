<hr />

Maha es una biblioteca de procesamiento de texto desarrollada especialmente para manejar texto árabe. La versión beta puede usarse para limpiar y analizar texto, archivos y carpetas con o sin capacidad de transmisión.

Si necesitas ayuda o quieres discutir temas relacionados con Maha, no dudes en unirte a nuestro servidor de [Discord](https://discord.gg/6W2tRFE7k4). Si deseas enviar un informe de errores o una solicitud de función, por favor abre un issue.

## Instalación

Simplemente ejecuta lo siguiente para instalar Maha:

```bash
pip install mahad # pronounced maha d
```

Para la instalación desde el código fuente, consulte la [documentación](https://maha.readthedocs.io/en/latest/contributing/guidelines.html).
