<hr />

Maha 是一个专门为处理阿拉伯语文本而开发的文本处理库。测试版可用于清理和解析文本、文件及文件夹，支持或不支持流式处理。

如果您需要帮助或想讨论与 Maha 相关的话题，欢迎加入我们的 [Discord](https://discord.gg/6W2tRFE7k4) 服务器。如果您想提交错误报告或功能请求，请打开一个 issue。

## 安装

只需运行以下命令即可安装 Maha：

```bash
pip install mahad # pronounced maha d
```

对于源码安装，请查看[文档](https://maha.readthedocs.io/en/latest/contributing/guidelines.html)。
