<hr />

Mahaはアラビア語テキストを扱うために特別に開発されたテキスト処理ライブラリです。ベータ版では、ストリーミング機能の有無にかかわらず、テキスト、ファイル、フォルダのクリーンアップおよび解析に使用できます。

Mahaに関する支援が必要な場合や議論したいトピックがある場合は、ぜひ[Discord](https://discord.gg/6W2tRFE7k4)サーバーにご参加ください。バグ報告や機能リクエストを提出したい場合は、Issueを開いてください。

## インストール

Mahaをインストールするには、以下を実行してください：

```bash
pip install mahad # pronounced maha d
```
ソースからのインストールについては、[ドキュメント](https://maha.readthedocs.io/en/latest/contributing/guidelines.html)を参照してください。
