# ۳۰ ثانیه کد

> مقالات برنامه‌نویسی برای ارتقاء مهارت‌های توسعه‌ی شما

* برای مشاهده تمام مقالات، به [وب‌سایت](https://30secondsofcode.org) مراجعه کنید.
* با استفاده از نام، برچسب‌ها، زبان یا توضیحات، مقالات و مجموعه‌هایی که مناسب نیاز شما هستند را جستجو کنید.
* همه‌ی [مقالات](https://30secondsofcode.org/snippets/p/1) یا [مجموعه‌های](https://30secondsofcode.org/collections/p/1) مربوط به هر موضوع را مرور کنید.
* برای مشاهده کامل هر مقاله شامل قطعه‌کدها، توضیحات و مثال‌ها، روی هر کارت کلیک کنید.
* اگر از پروژه خوشتان آمد، به آن ستاره بدهید. این کار اهمیت زیادی دارد.

## مایل به همکاری هستید؟

* در حال حاضر مشارکت محتوای جدید پذیرفته نمی‌شود.
* با این حال می‌توانید مشکلات وب‌سایت را گزارش دهید یا پیشنهاد بهبود ارائه کنید، با [ایجاد یک ایشو](https://github.com/Chalarangelo/30-seconds-of-code/issues/new).
