# Surge
Règles, scripts et modules Surge pour usage personnel

### Fichier de configuration général
[General.conf](https://github.com/BlueGrave/Surge/blob/master/General.conf) basé sur les fichiers de configuration commentés de [@scomper](https://github.com/scomper/Surge) et [@DivineEngine](https://github.com/DivineEngine/Profiles/tree/master/Surge)

### Fichiers de règles
Règles spéciales pour usage personnel
- [AppleOS_Update.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/AppleOS_Update.list) règles OTA pour divers Apple OS compilées à partir de plusieurs experts
- [Mail.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/Mail.list) résolution des problèmes de réception avec l’application iOS Mail

### Fichiers de modules
- [MyMoneyPro.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/MyMoneyPro.sgmodule) bloque les publicités au lancement de l’application MyMoneyPro
- [WhiteList.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/WhiteList.sgmodule) évite les blocages erronés d’AdBlock et ajoute des règles spéciales personnelles
- [Cookie.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Cookie.sgmodule) et [Task.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Task.sgmodule) connexions quotidiennes personnalisées pour diverses applications
- [VIFOnly.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/VIFOnly.sgmodule) active le mode compatibilité pour éviter les alertes de proxy dans les applications
