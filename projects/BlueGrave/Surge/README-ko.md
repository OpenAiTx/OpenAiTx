# Surge
자체 사용 Surge 규칙, 스크립트 및 모듈

### 일반 구성 파일
[General.conf](https://github.com/BlueGrave/Surge/blob/master/General.conf) 참고 [@scomper](https://github.com/scomper/Surge) 및 [@DivineEngine](https://github.com/DivineEngine/Profiles/tree/master/Surge) 의 구성 파일 주석

### 규칙 파일
자체 사용 특수 규칙
- [AppleOS_Update.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/AppleOS_Update.list) 여러 고수들의 관련 규칙을 결합하여 정리한 각 Apple OS OTA 규칙
- [Mail.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/Mail.list) iOS Mail 앱 수신 문제 해결

### 모듈 파일
- [MyMoneyPro.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/MyMoneyPro.sgmodule) 수시노트 프로 앱 오프닝 광고 차단
- [WhiteList.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/WhiteList.sgmodule) AdBlock 오작동 제외, 일부 자체 사용 특수 규칙 추가
- [Cookie.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Cookie.sgmodule) 및 [Task.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Task.sgmodule) 자체 사용 각 앱 일일 출석
- [VIFOnly.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/VIFOnly.sgmodule) 호환 모드 활성화, 앱에서 프록시 사용 알림 방지
