# Surge
Personal Surge rules, scripts, and modules

### General Configuration Files
[General.conf](https://github.com/BlueGrave/Surge/blob/master/General.conf) refers to the configuration file annotations of [@scomper](https://github.com/scomper/Surge) and [@DivineEngine](https://github.com/DivineEngine/Profiles/tree/master/Surge)

### Rules Files
Special rules for personal use
- [AppleOS_Update.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/AppleOS_Update.list) Apple OS OTA rules compiled from multiple experts’ related rules
- [Mail.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/Mail.list) solves iOS Mail App receiving issues

### Module Files
- [MyMoneyPro.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/MyMoneyPro.sgmodule) blocks splash ads in the MyMoneyPro app
- [WhiteList.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/WhiteList.sgmodule) excludes AdBlock false positives and adds some personal special rules
- [Cookie.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Cookie.sgmodule) and [Task.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Task.sgmodule) daily sign-ins for various personal apps
- [VIFOnly.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/VIFOnly.sgmodule) enables compatibility mode to avoid app proxy usage warnings
