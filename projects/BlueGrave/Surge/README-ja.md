# Surge
自用の Surge ルール、スクリプト、およびモジュール

### 共通設定ファイル
[General.conf](https://github.com/BlueGrave/Surge/blob/master/General.conf) は [@scomper](https://github.com/scomper/Surge) と [@DivineEngine](https://github.com/DivineEngine/Profiles/tree/master/Surge) の設定ファイル注釈を参考にしています

### ルールファイル
自用の特殊ルール
- [AppleOS_Update.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/AppleOS_Update.list) 複数の有志の関連ルールをまとめた各 Apple OS OTA ルール
- [Mail.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/Mail.list) iOS Mail アプリの受信問題を解決

### モジュールファイル
- [MyMoneyPro.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/MyMoneyPro.sgmodule) 随手记プロ版アプリの起動画面広告をブロック
- [WhiteList.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/WhiteList.sgmodule) AdBlock の誤検知を除外し、自用の特殊ルールを追加
- [Cookie.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Cookie.sgmodule) と [Task.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Task.sgmodule) 自用の各アプリ毎日のログイン
- [VIFOnly.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/VIFOnly.sgmodule) 互換モードを有効にし、アプリのプロキシ利用警告を回避
