# Surge
Reglas, scripts y módulos de Surge para uso personal

### Archivo de configuración general
[General.conf](https://github.com/BlueGrave/Surge/blob/master/General.conf) Referencia los archivos de configuración comentados de [@scomper](https://github.com/scomper/Surge) y [@DivineEngine](https://github.com/DivineEngine/Profiles/tree/master/Surge)

### Archivos de reglas
Reglas especiales para uso personal
- [AppleOS_Update.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/AppleOS_Update.list) Reglas OTA para Apple OS combinadas y organizadas a partir de varios expertos
- [Mail.list](https://github.com/BlueGrave/Surge/blob/master/Ruleset/Mail.list) Soluciona problemas de recepción en la app iOS Mail

### Archivos de módulos
- [MyMoneyPro.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/MyMoneyPro.sgmodule) Bloquea los anuncios de apertura de la app MyMoneyPro
- [WhiteList.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/WhiteList.sgmodule) Excluye bloqueos erróneos de AdBlock y añade reglas especiales para uso personal
- [Cookie.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Cookie.sgmodule) y [Task.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/Task.sgmodule) Firmas diarias personalizadas para varias apps
- [VIFOnly.sgmodule](https://github.com/BlueGrave/Surge/blob/master/Module/VIFOnly.sgmodule) Activa modo compatible para evitar alertas de proxy en apps
