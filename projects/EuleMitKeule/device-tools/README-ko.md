# Home Assistant용 디바이스 도구

디바이스를 수정하고 상호작용할 수 있는 맞춤형 Home Assistant 통합입니다.

> [!CAUTION]
> 이 통합은 핵심 데이터를 수정하므로 사용하기 전에 Home Assistant 인스턴스를 백업할 것을 강력히 권장합니다.

## 기능

* 디바이스 속성 수정
* 엔티티를 디바이스에 할당
* 새로운 디바이스 생성
* 디바이스 병합
* 제거 시 모든 수정 사항 자동 복원

# 설치

1. [HACS](https://hacs.xyz/) 통합을 설치합니다.
2. 리드미 상단의 My Home Assistant 링크를 클릭합니다.
