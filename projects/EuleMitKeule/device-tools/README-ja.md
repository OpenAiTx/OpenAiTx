# Home Assistant用デバイスツール

デバイスの変更および操作を可能にするカスタムHome Assistantインテグレーションです。

> [!CAUTION]
> このインテグレーションはコアデータを変更するため、使用前にHome Assistantインスタンスのバックアップを強くお勧めします。

## 機能

* デバイス属性の変更
* エンティティのデバイスへの割り当て
* 新しいデバイスの作成
* デバイスのマージ
* 削除時に自動的に変更を元に戻す

# インストール

1. [HACS](https://hacs.xyz/)インテグレーションをインストールします
2. READMEの上部にあるMy Home Assistantリンクをクリックします
