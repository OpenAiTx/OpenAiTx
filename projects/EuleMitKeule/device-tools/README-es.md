# Herramientas para dispositivos en Home Assistant

Una integración personalizada de Home Assistant que permite modificar e interactuar con dispositivos.

> [!PRECAUCIÓN]
> Recomiendo encarecidamente hacer una copia de seguridad de tu instancia de Home Assistant antes de usar esta integración, ya que modifica datos centrales.

## Características

* Modificar atributos del dispositivo
* Asignar entidades a dispositivos
* Crear nuevos dispositivos
* Fusionar dispositivos
* Revertir automáticamente cualquier modificación al eliminar

# Instalación

1. Instalar la integración [HACS](https://hacs.xyz/)
2. Hacer clic en el enlace Mi Home Assistant en la parte superior del readme
