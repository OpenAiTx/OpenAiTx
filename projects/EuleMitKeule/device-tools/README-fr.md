# Outils pour appareils dans Home Assistant

Une intégration personnalisée pour Home Assistant qui vous permet de modifier et d'interagir avec les appareils.

> [!CAUTION]
> Je recommande fortement de sauvegarder votre instance Home Assistant avant d'utiliser cette intégration, car elle modifie des données essentielles.

## Fonctionnalités

* Modifier les attributs des appareils
* Assigner des entités aux appareils
* Créer de nouveaux appareils
* Fusionner des appareils
* Revenir automatiquement sur toute modification lors de la suppression

# Installation

1. Installez l'intégration [HACS](https://hacs.xyz/)
2. Cliquez sur le lien My Home Assistant en haut du readme
