# Home Assistant 设备工具

一个自定义的 Home Assistant 集成，允许您修改和操作设备。

> [!CAUTION]
> 强烈建议在使用此集成前备份您的 Home Assistant 实例，因为它会修改核心数据。

## 功能

* 修改设备属性
* 将实体分配给设备
* 创建新设备
* 合并设备
* 在移除时自动恢复任何修改

# 安装

1. 安装 [HACS](https://hacs.xyz/) 集成
2. 点击自述文件顶部的 My Home Assistant 链接
