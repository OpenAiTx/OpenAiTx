# Comment contribuer

Bienvenue aux pull requests.

1. Forkez ce dépôt.  
2. Créez une page dans le panneau wiki.  
3. Mettez à jour README.md avec le lien de votre page wiki.  
4. Créez une pull request.

Bienvenue à soumettre des PR.
1. Forkez un dépôt (Repo) dans votre compte GitHub.  
2. Créez une page dans l’onglet Wiki.  
3. Modifiez le fichier README.md en ajoutant un lien vers la page créée à l’étape 2 selon le format.  
4. Soumettez une Pull Request (PR).
