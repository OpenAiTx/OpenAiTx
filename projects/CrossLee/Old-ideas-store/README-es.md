# Cómo contribuir

Bienvenido a enviar pull requests.

1. Haz fork de este repositorio.
2. Crea una página en el panel de wiki.
3. Actualiza README.md con el enlace a tu página wiki.
4. Crea un pull request.

Bienvenido a enviar PR.
1. Haz un Fork de un Repo a tu cuenta de GitHub.  
2. Crea una página en la pestaña Wiki.  
3. Modifica el README.md para añadir un enlace al página creada en el paso 2 siguiendo el formato.  
4. Envía un PR.
