# 貢献方法

プルリクエストを歓迎します。

1. このリポジトリをフォークしてください。
2. Wikiパネルでページを作成してください。
3. README.mdを更新し、あなたのWikiページへのリンクを追加してください。
4. プルリクエストを作成してください。

PRのご提出を歓迎します。

1. Repo をご自分の GitHub アカウントに Fork してください。
2. Wikiタブでページを1つ作成してください。
3. README.mdに2で作成したページへのリンクをフォーマットに従って追加してください。
4. PRを提出してください。
