## Qu'est-ce que FastExcel

Outil Java rapide et concis pour le traitement des fichiers Excel qui résout les problèmes de débordement de mémoire avec les fichiers volumineux.

FastExcel sera toujours gratuit et open source, utilisant la licence Apache compatible avec les entreprises, ce qui le rend adapté à tous les scénarios commerciaux. Cela offre aux développeurs et aux entreprises une grande liberté et flexibilité.
Nous prévoyons d'introduire plus de nouvelles fonctionnalités à l'avenir pour améliorer continuellement l'expérience utilisateur et la convivialité de l'outil. Restez
connecté au compte public "Programmer Xiao Lan" pour les mises à jour sur le développement de FastExcel. FastExcel s'engage à
être votre meilleur choix pour la gestion des fichiers Excel.

## Fonctionnalités

- **Lecture et écriture haute performance** : FastExcel se concentre sur l'optimisation des performances, capable de gérer efficacement
  des données Excel à grande échelle. Comparé à certaines bibliothèques traditionnelles de traitement Excel, il peut réduire considérablement
  la consommation de mémoire.
- **Simplicité et facilité d'utilisation** : La bibliothèque offre une API simple et intuitive, permettant aux développeurs de l'intégrer facilement
  dans des projets, que ce soit pour des opérations Excel simples ou des traitements de données complexes.
- **Opérations en flux** : FastExcel supporte la lecture en flux, minimisant le problème de chargement de grandes quantités de données
  d'un coup. Cette conception est particulièrement importante lorsqu'on traite des centaines de milliers voire des millions de lignes de données.

### Mise à jour de la version


Pour des journaux de mise à jour détaillés, consultez [Détails des mises à jour de version](https://raw.githubusercontent.com/fast-excel/fastexcel/main/./CHANGELOG.md). Vous pouvez également trouver toutes les versions disponibles dans le [Dépôt Central Maven](https://mvnrepository.com/artifact/cn.idev.excel/fastexcel).

### Maven

Si vous utilisez Maven pour la construction du projet, ajoutez la configuration suivante dans le fichier `pom.xml` :


```xml

<dependency>
    <groupId>cn.idev.excel</groupId>
    <artifactId>fastexcel</artifactId>
    <version>version</version>
</dependency>
```

### Gradle

Si vous utilisez Gradle pour la construction du projet, ajoutez la configuration suivante dans le fichier build.gradle :

```gradle
dependencies {
    implementation 'cn.idev.excel:fastexcel:version'
}
```

## Exemple

### Lecture de fichiers Excel

Voici un exemple de lecture d'un document Excel :

```java
// Implement the ReadListener interface to set up operations for reading data
public class DemoDataListener implements ReadListener<DemoData> {
    
    @Override
    public void invoke(DemoData data, AnalysisContext context) {
        System.out.println("Parsed a data entry" + JSON.toJSONString(data));
    }
    
    @Override
    public void doAfterAllAnalysed(AnalysisContext context) {
        System.out.println("All data parsed!");
    }
}

public static void main(String[] args) {
    String fileName = "demo.xlsx";
    // Read Excel file
    FastExcel.read(fileName, DemoData.class, new DemoDataListener()).sheet().doRead();
}
```

### Création de fichiers Excel

Voici un exemple simple de création d’un document Excel :

```java
// Sample data class
public class DemoData {
    
    @ExcelProperty("String Title")
    private String string;
    
    @ExcelProperty("Date Title")
    private Date date;
    
    @ExcelProperty("Number Title")
    private Double doubleData;
    
    @ExcelIgnore
    private String ignore;
}

// Prepare data to write
private static List<DemoData> data() {
    List<DemoData> list = new ArrayList<>();
    for (int i = 0; i < 10; i++) {
        DemoData data = new DemoData();
        data.setString("String" + i);
        data.setDate(new Date());
        data.setDoubleData(0.56);
        list.add(data);
    }
    return list;
}

public static void main(String[] args) {
    String fileName = "demo.xlsx";
    // Create a "Template" sheet and write data
    FastExcel.write(fileName, DemoData.class).sheet("Template").doWrite(data());
}
```
