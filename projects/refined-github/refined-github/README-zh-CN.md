# <img src="https://raw.githubusercontent.com/refined-github/refined-github/main/source/icon.png" width="45" align="left"> Refined GitHub

> 简化 GitHub 界面并增加实用功能的浏览器扩展

我们频繁使用 GitHub，发现了许多想要修复的不便之处。所以，这里有很多“龙”。

我们希望 GitHub 能注意到并实现这些亟需的改进。如果你喜欢其中任何一项功能，欢迎在 [GitHub feedback](https://github.com/github-community/community/discussions) 发起讨论，或联系 [GitHub support](https://support.github.com/request) 反馈建议。

同样支持 GitHub Enterprise：[如何启用](https://fregante.github.io/webext-permission-toggle/?name=Refined%20GitHub&icon=https%3A%2F%2Fraw.githubusercontent.com%2Frefined-github%2Frefined-github%2Fmain%2Fsource%2Ficon.png)。<!-- icon.png renders best -->

*GITHUB 和 REFINED GITHUB 商标归 GitHub, Inc. 所有，并授权使用。*

### 曾经属于 Refined GitHub 的功能

[GitHub 借鉴了 Refined GitHub 的灵感](https://github.blog/2018-08-28-announcing-paper-cuts/)，并原生实现了数十项功能 🎉，因此这些功能已被[从本扩展移除。](https://github.com/refined-github/refined-github/pulls?q=is%3Apr+is%3Amerged+label%3A%22implemented+by+github%22) 你也可以在[此处一次性查看 Refined GitHub 的所有历史功能列表](https://github.com/refined-github/refined-github/pulls?q=is%3Apr+is%3Amerged+-label%3Ameta+in%3Atitle+drop+feature)。

## 自定义

大多数功能都可以禁用（如果它们基于 JavaScript），你还可以在扩展选项中用自己的 CSS 覆盖我们的样式。

我们欢迎建议和贡献，但请注意这是一个高度主观的项目。[新增功能的门槛很高。](https://github.com/refined-github/refined-github/wiki/%22Can-you-add-this-feature%3F%22) 用户间总会有分歧。尽管如此，我们仍乐于讨论。
