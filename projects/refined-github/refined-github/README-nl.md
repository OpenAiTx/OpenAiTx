# <img src="https://raw.githubusercontent.com/refined-github/refined-github/main/source/icon.png" width="45" align="left"> Refined GitHub

> Browserextensie die de GitHub-interface vereenvoudigt en nuttige functies toevoegt

Wij gebruiken GitHub veel en merken veel ergernissen op die we graag willen oplossen. Dus hier zijn draken.

We hopen dat GitHub sommige van deze broodnodige verbeteringen zal opmerken en implementeren. Dus als je een van deze verbeteringen waardeert, open dan een discussie op [GitHub feedback](https://github.com/github-community/community/discussions) of neem contact op met [GitHub support](https://support.github.com/request) om het voor elkaar te krijgen.

GitHub Enterprise wordt ook ondersteund: [Hoe schakel je het in](https://fregante.github.io/webext-permission-toggle/?name=Refined%20GitHub&icon=https%3A%2F%2Fraw.githubusercontent.com%2Frefined-github%2Frefined-github%2Fmain%2Fsource%2Ficon.png). <!-- icon.png wordt het beste weergegeven -->

*De handelsmerken GITHUB en REFINED GITHUB zijn eigendom van GitHub, Inc. en worden gebruikt onder licentie.*

### Voorheen onderdeel van Refined GitHub

[GitHub heeft inspiratie gehaald uit Refined GitHub](https://github.blog/2018-08-28-announcing-paper-cuts/) en tientallen functies ervan native geïmplementeerd, 🎉 dus zijn ze [uit deze extensie verwijderd.](https://github.com/refined-github/refined-github/pulls?q=is%3Apr+is%3Amerged+label%3A%22implemented+by+github%22) Je kunt ook [alle voorgaande functies](https://github.com/refined-github/refined-github/pulls?q=is%3Apr+is%3Amerged+-label%3Ameta+in%3Atitle+drop+feature) van Refined GitHub in één lijst bekijken.

## Aanpassen

De meeste functies kunnen uitgeschakeld worden als ze JavaScript-gebaseerd zijn en je kunt onze CSS overschrijven met je eigen in de extensie-opties.

We ontvangen graag suggesties en bijdragen, maar wees je ervan bewust dat dit een sterk uitgesproken project is. Er is [een hoge drempel voor het toevoegen van functies.](https://github.com/refined-github/refined-github/wiki/%22Can-you-add-this-feature%3F%22) Gebruikers zijn het altijd ergens niet mee eens. Dat gezegd hebbende, staan we open voor discussie.
