### Si vous aimez ce dépôt, ce serait génial si vous pouviez lui mettre une étoile ! Cela aide vraiment à sa visibilité. Pour toute question, nous serions ravis de vous entendre à jfuruness@gmail.com

Bienvenue dans BGPy ! L’un des principaux simulateurs de sécurité BGP en Python au monde. Vous êtes en bonne compagnie ; ce simulateur a été utilisé par des équipes du monde entier, y compris le NIST. BGPy est également une [recherche évaluée par les pairs](https://dl.acm.org/doi/fullHtml/10.1145/3607505.3607509) et a été présenté dans [plusieurs publications](https://github.com/jfuruness/bgpy/wiki/Citations-and-Use-Cases), avec beaucoup d’autres en cours.

Avec BGPy, vous pouvez simuler toute la topologie Internet sur votre ordinateur portable. Vous pouvez programmer vos propres politiques de sécurité et attaques, utiliser la topologie pour des analyses personnalisées, utiliser des données ROV réelles, etc. BGPy est également livré avec une suite de tests personnalisée qui permet de tester facilement les protocoles de sécurité. Nous supportons [plus de 20 politiques](https://github.com/jfuruness/bgpy_pkg/wiki/Routing-Policies) incluant ROV, ASPA, et ASRA.

Si vous cherchez un tutoriel approfondi sur les fonctionnalités et l’utilisation de BGPy, je recommande vivement le [tutoriel](https://github.com/jfuruness/bgpy/wiki/Tutorial).

Si vous souhaitez simplement exécuter/visualiser des scénarios d’attaque/défense sans programmer, je recommande de consulter l’interface frontale de cet outil : [BGPSimulator.com](https://www.bgpsimulator.com)

Si vous souhaitez citer BGPy ou consulter les cas d’utilisation, vous pouvez les trouver [ici](https://github.com/jfuruness/bgpy/wiki/Citations-and-Use-Cases). Si vous citez BGPy, merci de nous en informer et nous vous ajouterons à la liste :)
