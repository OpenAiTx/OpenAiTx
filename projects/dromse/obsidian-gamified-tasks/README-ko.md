## 기능

- **작업**: 저장소 내 모든 작업을 확인하세요.
- **필터**: 완료 상태, 반복 여부로 작업을 필터링하거나 내용 내 검색이 가능합니다.
- **보상**: 획득한 코인으로 보상을 구매하세요.
- **기록**: 수입 및 지출 내역을 확인할 수 있습니다.
- **반복**: 동일한 작업을 반복하는 대신, 반복 작업은 기록에 저장되고 시간이 되면 작업 목록에 다시 나타납니다.
- **카운터**: 작업을 더 쉽게 완료할 수 있도록 작은 단위로 분할하는 카운터를 활용하세요.
- **난이도**: 난이도 수준을 설정하여 이에 따른 코인을 획득할 수 있습니다.
