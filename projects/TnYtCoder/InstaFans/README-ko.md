## 설치 방법 (Termux / Nethunter)

```
git clone https://github.com/TnYtCoder/InstaFans
cd InstaFans
chmod +x *
pip3 install -r requirements.txt
```

이제, 당신은 `필요합니다` 사용자 이름과 비밀번호가

그런 다음 credentials.ini 파일에서 사용자 이름과 비밀번호를 실행하여 교체하세요
```
nano credentials.ini
```
```
[Instagram Credentials]
username
password
```
`"` 없이 자격 증명만 교체하세요

실행 방법 →
```
python3 instafans.py
```
