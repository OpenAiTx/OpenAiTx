# 🌟 Características Clave

1. **Soporte para Diversos Proveedores de LLM**:

- ☁️ Principales servicios de LLM en la nube: OpenAI, Gemini, Anthropic y más
- 🔗 Integración de servicios web de IA: Claude, Peplexity, Poe y otros
- 💻 Soporte de modelos locales con Ollama, LM Studio

2. **Asistentes de IA y Conversaciones**:

- 📚 Más de 300 asistentes de IA preconfigurados
- 🤖 Creación de Asistentes Personalizados
- 💬 Conversaciones Simultáneas Multi-modelo

3. **Procesamiento de Documentos y Datos**:

- 📄 Soporte para Texto, Imágenes, Office, PDF y más
- ☁️ Gestión y Respaldo de Archivos WebDAV
- 📊 Visualización de Diagramas Mermaid
- 💻 Resaltado de Sintaxis de Código

4. **Integración de Herramientas Prácticas**:

- 🔍 Funcionalidad de Búsqueda Global
- 📝 Sistema de Gestión de Temas
- 🔤 Traducción Potenciada por IA
- 🎯 Ordenación mediante Arrastrar y Soltar
- 🔌 Soporte para Mini Programas
- ⚙️ Servidor MCP (Protocolo de Contexto de Modelo)

5. **Experiencia de Usuario Mejorada**:

- 🖥️ Soporte Multiplataforma para Windows, Mac y Linux
- 📦 Listo para Usar, Sin Configuración de Entorno Requerida
- 🎨 Temas Claro/Oscuro y Ventana Transparente
- 📝 Renderizado Completo de Markdown
- 🤲 Compartición de Contenido Fácil

# 🏢 Edición Empresarial

Sobre la base de la Edición Comunitaria, nos enorgullece presentar la **Edición Empresarial de Cherry Studio**: una plataforma de productividad y gestión de IA desplegable de forma privada, diseñada para equipos y empresas modernas.

La Edición Empresarial aborda los desafíos clave en la colaboración de equipos centralizando la gestión de recursos de IA, conocimientos y datos. Permite a las organizaciones mejorar la eficiencia, fomentar la innovación y garantizar el cumplimiento, manteniendo el 100% del control sobre sus datos en un entorno seguro.

## Ventajas Principales

- **Gestión Unificada de Modelos**: Integra y gestiona centralizadamente varios LLMs en la nube (por ejemplo, OpenAI, Anthropic, Google Gemini) y modelos privados desplegados localmente. Los empleados pueden usarlos de inmediato sin configuración individual.
- **Base de Conocimiento de Nivel Empresarial**: Crea, gestiona y comparte bases de conocimiento a nivel de equipo. Asegura que el conocimiento se conserve y sea consistente, permitiendo que los miembros del equipo interactúen con la IA a partir de información unificada y precisa.
- **Control de Acceso Granular**: Gestiona fácilmente las cuentas de empleados y asigna permisos basados en roles para diferentes modelos, bases de conocimiento y funcionalidades a través de un backend administrativo unificado.
- **Despliegue Totalmente Privado**: Despliega todo el servicio backend en tus propios servidores o nube privada, asegurando que tus datos permanezcan 100% privados y bajo tu control para cumplir con los estándares más estrictos de seguridad y cumplimiento.
- **Servicios Backend Confiables**: Proporciona servicios API estables, mecanismos de respaldo y recuperación de datos de nivel empresarial para garantizar la continuidad del negocio.
