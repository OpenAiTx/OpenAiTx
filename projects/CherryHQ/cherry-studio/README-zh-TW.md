# 🍒 Cherry Studio

Cherry Studio 是一款支援多種 LLM 供應商的桌面客戶端，適用於 Windows、Mac 和 Linux。

👏 加入 [Telegram 群組](https://t.me/CherryStudioAI)｜[Discord](https://discord.gg/wez8HtpxqQ) | [QQ 群(575014769)](https://qm.qq.com/q/lo0D4qVZKi)

# 🌟 主要功能

1. **多元 LLM 供應商支援**：

- ☁️ 主流 LLM 雲端服務：OpenAI、Gemini、Anthropic 等
- 🔗 AI 網頁服務整合：Claude、Peplexity、Poe 及其他
- 💻 本地模型支援 Ollama、LM Studio

2. **AI 助理與對話**：

- 📚 300+ 預設配置的 AI 助理
- 🤖 自訂助理建立
- 💬 多模型同時對話

3. **文件與資料處理**：

- 📄 支援文字、圖片、Office、PDF 等多種格式
- ☁️ WebDAV 檔案管理與備份
- 📊 Mermaid 圖表視覺化
- 💻 程式碼語法高亮顯示

4. **實用工具整合**：

- 🔍 全域搜尋功能
- 📝 主題管理系統
- 🔤 AI 驅動翻譯
- 🎯 拖放排序
- 🔌 小程式支援
- ⚙️ MCP（模型上下文協議）伺服器

5. **優化使用體驗**：

- 🖥️ 跨平台支援 Windows、Mac 和 Linux
- 📦 開箱即用，無需環境配置
- 🎨 明亮/深色主題及透明視窗
- 📝 完整 Markdown 呈現
- 🤲 輕鬆內容分享

# 🤝 貢獻指南

我們歡迎你加入 Cherry Studio 的貢獻！你可以這樣參與：

1. **貢獻程式碼**：開發新功能或優化現有程式碼。
2. **修復錯誤**：提交你發現的任何錯誤修正。
3. **維護議題**：協助管理 GitHub 議題。
4. **產品設計**：參與設計討論。
5. **編寫文件**：完善使用手冊和指南。
6. **社群互動**：參與討論並協助用戶。
7. **推廣使用**：宣傳 Cherry Studio。

請參閱 [分支策略](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/branching-strategy-en.md) 以了解貢獻規範

## 開始貢獻

1. **Fork 倉庫**：Fork 並克隆到本地機器。
2. **建立分支**：為你的更改創建分支。
3. **提交更改**：Commit 並推送你的更改。
4. **開啟 Pull Request**：描述你的更改與理由。

欲知更詳細的貢獻說明，請參閱 [貢獻指南](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/./CONTRIBUTING.md)。

感謝你的支持與貢獻！

# 🔧 開發者共創計畫
我們正在啟動 Cherry Studio 開發者共創計劃，以促進開源生態系統內的健康正向反饋循環。我們相信偉大的軟體是協作打造的，每一次合併的 Pull Request 都為專案注入新生命。

我們誠摯邀請您加入我們的貢獻者行列，與我們一同塑造 Cherry Studio 的未來。

## 貢獻者獎勵計劃

為了回饋我們的核心貢獻者並創造良性循環，我們制定了以下長期激勵方案。

**本計劃的首個統計週期為 2025 年第三季（7 月、8 月、9 月），該週期的獎勵將於 10 月 1 日發放。**

在任一統計週期內（例如第一輪為 7 月 1 日至 9 月 30 日），任何開發者只要對 Cherry Studio 的任一 GitHub 開源專案貢獻超過 **30 次有意義的提交（commit）**，即可獲得以下福利：

- **Cursor 訂閱贊助**：獲得 **70 美元** [Cursor](https://cursor.sh/) 訂閱抵用金或報銷，讓 AI 成為您最高效的編程夥伴。
- **模型無限使用權**：獲得 **DeepSeek** 與 **Qwen** 模型的 **API 無限次數調用權**。
- **前沿技術體驗**：不定期獲得如 **Claude**、**Gemini**、**OpenAI** 等模型的 API 權益，持續站在技術前沿。

## 共同成長與未來規劃

充滿活力的社群是任何可持續開源專案的核心驅動力。隨著 Cherry Studio 的成長，我們的獎勵計劃也將不斷升級。我們承諾持續與業界最頂尖的工具與資源接軌，確保核心貢獻者獲得有意義的支持，推動開發者、社群與專案的良性共生。

**展望未來，專案也將持續擴大開放，回饋整個開源社群。**

## 如何開始參與？

我們期待您的第一個 Pull Request！

您可以從瀏覽我們的程式庫開始，挑選 `good first issue`，或主動提出自己的改進建議。每一個 commit 都是開源精神的體現。

感謝您的關注與貢獻。

讓我們一起共創未來。

# 🏢 企業版

在社群版的基礎上，我們自豪地推出 **Cherry Studio 企業版**——一款可私有部署的 AI 生產力與管理平台，專為現代團隊與企業設計。

企業版聚焦團隊協作的核心挑戰，集中管理 AI 資源、知識與數據，幫助組織提升效率、激發創新、保障合規，同時確保所有資料 100% 由您自主掌控並於安全環境下運作。

## 核心優勢

- **統一模型管理**：集中整合與管理各類雲端 LLM（如 OpenAI、Anthropic、Google Gemini）與本地私有部署模型，員工無需個人配置即可開箱即用。
- **企業級知識庫**：建構、管理並團隊共享知識庫，確保知識沉澱與一致，團隊成員可基於統一且準確的資訊與 AI 互動。
- **細緻權限管控**：統一後台輕鬆管理員工帳號，為不同模型、知識庫及功能分配角色權限。
- **全私有部署**：可將整套後端服務部署於本地伺服器或私有雲，確保您的數據 100% 私有並自主掌控，滿足最嚴格的安全與合規要求。
- **可靠後端服務**：提供穩定的 API 服務、企業級數據備份與災難恢復機制，確保業務持續運作。
