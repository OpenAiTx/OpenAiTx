- **Cursor Subscription Sponsorship**: আপোনাৰ [Cursor](https://cursor.sh/) সাবস্ক্ৰিপশ্যনৰ বাবে **$৭০ USD** ক্ৰেডিট বা ৰিম্বাৰ্ছমেণ্ট লাভ কৰক, AI-কে আপোনাৰ আটাইতকৈ কাৰ্যকৰী ক’ডিং অংশীদাৰ হিচাপে গঢ়ি তুলিবলৈ।
- **Unlimited Model Access**: **DeepSeek** আৰু **Qwen** মডেলসমূহৰ বাবে **অসীম** API কল লাভ কৰক।
- **Cutting-Edge Tech Access**: সময় সময়ত বিশেষ সুবিধা, যেনে **Claude**, **Gemini**, আৰু **OpenAI**-ৰ দৰে মডেলসমূহলৈ API এক্সেছ, যাতে আপুনি প্ৰযুক্তিৰ আগভাগত থাকিব পাৰে।

## একেলগে বৃদ্ধি আৰু ভৱিষ্যৎ পৰিকল্পনা

এখন জীৱন্ত কমিউনিটি যিকোনো টেকসই মুক্ত উৎস প্ৰকল্পৰ মূল শক্তি। Cherry Studio-ৰ বৃদ্ধি অনুসৰি আমাৰ পুৰস্কাৰ কাৰ্যসূচীও বৃদ্ধি পাব। আমি ইণ্ডাস্ট্ৰীৰ সৰ্বোত্তম টুল আৰু উৎসসমূহৰ সৈতে আমাৰ সুবিধাসমূহ সদায় মিলাই ৰাখিবলৈ প্ৰতিজ্ঞাবদ্ধ। এইদৰে আমাৰ মুখ্য অৱদানকাৰীসকলে অৰ্থপূৰ্ণ সহায় পায়, যাৰ দ্বাৰা ডেভেলপাৰ, কমিউনিটি, আৰু প্ৰকল্পে একেলগে বৃদ্ধি পায়।

**ভৱিষ্যতে, প্ৰকল্পটোৱে অধিক মুক্ত মনোভাব গ্ৰহণ কৰি সম্পূৰ্ণ মুক্ত উৎস কমিউনিটিলৈ সহায় আগবঢ়াব।**

# 🏢 এণ্টাৰপ্ৰাইজ সংস্কৰণ

কমিউনিটি সংস্কৰণৰ ওপৰত আধাৰ কৰি, আমি গর্বেৰে আগবঢ়াইছোঁ **Cherry Studio Enterprise Edition**—এটা ব্যক্তিগতভাৱে স্থাপনযোগ্য AI productivity আৰু management platform, আধুনিক দল আৰু উদ্যোগৰ বাবে।

Enterprise Edition-এ দলীয় সহযোগিতাৰ মূল সমস্যা সমাধান কৰে, AI resources, knowledge, আৰু data-ৰ কেন্দ্ৰীয় ব্যৱস্থাপনা কৰি। ই সংস্থাসমূহক দক্ষতা বৃদ্ধি, উদ্ভাৱন উন্নয়ন, আৰু নিয়ম অনুসৰণৰ নিশ্চয়তা দিয়ে, আৰু সকলো data ১০০% নিয়ন্ত্ৰণ আৰু নিৰাপদ পৰিৱেশত ৰাখে।

## মুখ্য সুবিধাসমূহ

- **একত্ৰিত মডেল ব্যৱস্থাপনা**: কেন্দ্ৰীয়ভাৱে বহুবিধ cloud-based LLM (OpenAI, Anthropic, Google Gemini) আৰু স্থানীয়ভাৱে স্থাপন কৰা ব্যক্তিগত মডেলসমূহ সংহত আৰু ব্যৱস্থাপনা কৰক। কৰ্মচাৰীয়ে বিশেষ সংহতি নকৰাকৈ ব্যৱহাৰ কৰিব পাৰে।
- **এণ্টাৰপ্ৰাইজ-গ্ৰেড জ্ঞানভঁৰাল**: দলব্যাপী জ্ঞানভঁৰাল গঠন, পৰিচালনা, আৰু ভাগ-বতৰা কৰক। জ্ঞান সংৰক্ষণ আৰু একতাৰ নিশ্চয়তা দিয়ে, যাতে দলৰ সদস্যসকলে AI-ৰ সৈতে একত্ৰিত আৰু সঠিক তথ্যত আধাৰিতভাৱে যোগাযোগ কৰিব পাৰে।
- **সূক্ষ্মভাৱে নিয়ন্ত্ৰিত এক্সেছ**: কৰ্মচাৰী একাউণ্ট সহজে পৰিচালনা কৰক আৰু বিভিন্ন মডেল, জ্ঞানভঁৰাল, আৰু ফিচাৰৰ বাবে ভূমিকা-ভিত্তিক অনুমতি একত্ৰিত এডমিন বেকএণ্ডৰ জৰিয়তে নিযুক্ত কৰক।
- **সম্পূৰ্ণ ব্যক্তিগত স্থাপন**: সম্পূৰ্ণ বেকএণ্ড সেৱা আপোনাৰ নিজস্ব চাৰ্ভাৰ বা ব্যক্তিগত ক্লাউডত স্থাপন কৰক, যাতে আপোনাৰ data ১০০% ব্যক্তিগত আৰু নিয়ন্ত্ৰণত থাকে, কঠোৰতম নিৰাপত্তা আৰু নিয়ম মানি চলিবলৈ।
- **বিশ্বস্ত বেকএণ্ড সেৱাসমূহ**: ব্যৱসায়িক ধাৰাবাহিকতাৰ নিশ্চয়তা দিয়াৰ বাবে স্থিতিশীল API সেৱা আৰু এণ্টাৰপ্ৰাইজ-গ্ৰেড data backup আৰু recovery ব্যৱস্থা প্ৰদান কৰে।
