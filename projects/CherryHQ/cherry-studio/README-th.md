# 🍒 Cherry Studio

Cherry Studio เป็นไคลเอนต์เดสก์ท็อปที่รองรับผู้ให้บริการ LLM หลายราย ใช้งานได้บน Windows, Mac และ Linux

# 🌟 คุณสมบัติเด่น
1. **รองรับผู้ให้บริการ LLM ที่หลากหลาย**:

- ☁️ บริการคลาวด์ LLM รายใหญ่: OpenAI, Gemini, Anthropic และอื่น ๆ
- 🔗 การเชื่อมต่อบริการ AI บนเว็บ: Claude, Perplexity, [Poe](https://poe.com/) และอื่น ๆ
- 💻 รองรับโมเดลโลคัลด้วย Ollama, LM Studio

2. **ผู้ช่วย AI & การสนทนา**:

- 📚 ผู้ช่วย AI ที่ตั้งค่าล่วงหน้ากว่า 300 รายการ
- 🤖 สร้างผู้ช่วยแบบกำหนดเอง
- 💬 สนทนาหลายโมเดลพร้อมกัน

3. **การประมวลผลเอกสาร & ข้อมูล**:

- 📄 รองรับข้อความ, รูปภาพ, Office, PDF และอื่น ๆ
- ☁️ จัดการไฟล์และสำรองข้อมูลผ่าน WebDAV
- 📊 แสดงภาพชาร์ตด้วย Mermaid
- 💻 ไฮไลท์ซินแทกซ์โค้ด

4. **เครื่องมือเสริมเพื่อการใช้งานจริง**:

- 🔍 ฟังก์ชันค้นหาทั่วทั้งระบบ
- 📝 ระบบจัดการหัวข้อ
- 🔤 แปลภาษาโดยใช้ AI
- 🎯 ลากและวางเพื่อจัดลำดับ
- 🔌 รองรับมินิโปรแกรม
- ⚙️ เซิร์ฟเวอร์ MCP (Model Context Protocol)

5. **ประสบการณ์ผู้ใช้ที่ดียิ่งขึ้น**:

- 🖥️ รองรับข้ามแพลตฟอร์มทั้ง Windows, Mac และ Linux
- 📦 พร้อมใช้งานทันที - ไม่ต้องติดตั้งสภาพแวดล้อมเพิ่มเติม
- 🎨 ธีมสว่าง/มืด และหน้าต่างโปร่งใส
- 📝 แสดงผล Markdown ได้สมบูรณ์
- 🤲 แชร์เนื้อหาได้อย่างง่ายดาย

# 📝 Roadmap

เรากำลังพัฒนาและปรับปรุงฟีเจอร์เหล่านี้อย่างต่อเนื่อง:


1. 🎯 **ฟีเจอร์หลัก**

- ผู้ช่วยการเลือกเนื้อหาด้วยการเพิ่มประสิทธิภาพการเลือกแบบอัจฉริยะ
- การวิจัยเชิงลึกด้วยความสามารถวิจัยขั้นสูง
- ระบบหน่วยความจำด้วยการรับรู้บริบทระดับโลก
- การประมวลผลเอกสารล่วงหน้าด้วยการจัดการเอกสารที่ดีขึ้น
- ตลาด MCP สำหรับระบบนิเวศ Model Context Protocol

2. 🗂 **การจัดการความรู้**

- บันทึกและคอลเลกชัน
- การแสดงภาพแบบ Dynamic Canvas
- ความสามารถด้าน OCR
- รองรับ TTS (แปลงข้อความเป็นเสียง)

3. 📱 **รองรับแพลตฟอร์ม**

- HarmonyOS Edition (PC)
- แอป Android (ระยะที่ 1)
- แอป iOS (ระยะที่ 1)
- รองรับหลายหน้าต่าง
- ฟังก์ชันปักหมุดหน้าต่าง
- รองรับ Intel AI PC (Core Ultra)

4. 🔌 **ฟีเจอร์ขั้นสูง**

- ระบบปลั๊กอิน
- ASR (รู้จำเสียงอัตโนมัติ)
- ปรับโครงสร้างปฏิสัมพันธ์ผู้ช่วยและหัวข้อ

ติดตามความคืบหน้าและมีส่วนร่วมได้ที่ [project board](https://github.com/orgs/CherryHQ/projects/7)

อยากมีอิทธิพลต่อแผนงาน? เข้าร่วม [GitHub Discussions](https://github.com/CherryHQ/cherry-studio/discussions) เพื่อแบ่งปันแนวคิดและข้อเสนอแนะ!

# 🤝 การมีส่วนร่วม

เรายินดีต้อนรับการมีส่วนร่วมใน Cherry Studio! นี่คือวิธีที่คุณสามารถช่วยได้:

1. **ร่วมพัฒนาโค้ด**: สร้างฟีเจอร์ใหม่หรือปรับปรุงโค้ดที่มีอยู่
2. **แก้ไขบั๊ก**: ส่งคำแก้ไขปัญหาที่คุณพบ
3. **ดูแล Issues**: ช่วยจัดการปัญหาใน GitHub
4. **ออกแบบผลิตภัณฑ์**: เข้าร่วมพูดคุยด้านการออกแบบ
5. **เขียนเอกสาร**: ปรับปรุงคู่มือและคำแนะนำการใช้งาน
6. **มีส่วนร่วมในชุมชน**: ร่วมพูดคุยและช่วยเหลือผู้ใช้
7. **ส่งเสริมการใช้งาน**: กระจายข่าวเกี่ยวกับ Cherry Studio

โปรดดู [กลยุทธ์การแตกสาขา](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/en/guides/branching-strategy.md) สำหรับแนวทางการมีส่วนร่วม

## เริ่มต้นใช้งาน

1. **Fork Repository**: Fork และ clone มายังเครื่องของคุณ
2. **สร้าง Branch**: สำหรับการเปลี่ยนแปลงของคุณ
3. **ส่งการเปลี่ยนแปลง**: commit และ push โค้ดของคุณ
4. **เปิด Pull Request**: อธิบายสิ่งที่เปลี่ยนแปลงและเหตุผล

สำหรับแนวทางเพิ่มเติม โปรดดู [คู่มือการมีส่วนร่วม](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/CONTRIBUTING.md) ของเรา

ขอขอบคุณสำหรับการสนับสนุนและการมีส่วนร่วมของคุณ!

# 🔧 โปรแกรมร่วมสร้างสรรค์สำหรับนักพัฒนา

เรากำลังเปิดตัวโปรแกรมร่วมสร้างสรรค์ Cherry Studio เพื่อส่งเสริมวงจรตอบรับเชิงบวกในระบบนิเวศโอเพ่นซอร์ส เราเชื่อว่าซอฟต์แวร์ที่ยอดเยี่ยมเกิดจากความร่วมมือ และทุก pull request ที่ merge คือการเติมชีวิตให้โครงการ

ขอเชิญคุณเข้าร่วมเป็นหนึ่งในผู้ร่วมสร้างอนาคตของ Cherry Studio กับเรา

## โปรแกรมรางวัลสำหรับผู้ร่วมพัฒนา

เพื่อเป็นการตอบแทนผู้ร่วมพัฒนาหลักและสร้างวงจรที่ดี เราได้จัดตั้งแผนจูงใจระยะยาวดังต่อไปนี้

**ช่วงการติดตามผลครั้งแรกของโปรแกรมนี้จะเริ่มในไตรมาสที่ 3 ปี 2025 (กรกฎาคม, สิงหาคม, กันยายน) โดยจะมีการแจกจ่ายรางวัลสำหรับรอบนี้ในวันที่ 1 ตุลาคม**

ในช่วงระยะเวลาการติดตามผลใด ๆ (เช่น 1 กรกฎาคม ถึง 30 กันยายนสำหรับรอบแรก) นักพัฒนาที่มีส่วนร่วมมากกว่า **30 คอมมิตต์ที่มีคุณค่า** กับโปรเจกต์โอเพ่นซอร์สใด ๆ ของ Cherry Studio บน GitHub จะมีสิทธิ์ได้รับสิทธิประโยชน์ดังต่อไปนี้:

- **สนับสนุนการสมัครสมาชิก Cursor**: รับเครดิตหรือคืนเงิน **$70 USD** สำหรับ [Cursor](https://cursor.sh/) เพื่อให้ AI เป็นคู่หูการเขียนโค้ดที่มีประสิทธิภาพสูงสุดของคุณ
- **เข้าถึงโมเดลไม่จำกัด**: รับสิทธิ์การเรียกใช้ API สำหรับโมเดล **DeepSeek** และ **Qwen** ได้ **ไม่จำกัด**
- **เข้าถึงเทคโนโลยีล้ำสมัย**: รับสิทธิประโยชน์เสริม เช่น การเข้าถึง API โมเดลอย่าง **Claude**, **Gemini** และ **OpenAI** เพื่อให้คุณอยู่ในแนวหน้าของเทคโนโลยี

## เติบโตไปด้วยกัน & แผนในอนาคต

ชุมชนที่แข็งแกร่งคือแรงผลักดันสำคัญเบื้องหลังโปรเจกต์โอเพ่นซอร์สที่ยั่งยืน เมื่อ Cherry Studio เติบโตขึ้น โปรแกรมรางวัลของเราก็จะขยายตามไปด้วย เรามุ่งมั่นที่จะปรับปรุงสิทธิประโยชน์ให้สอดคล้องกับเครื่องมือและทรัพยากรที่ดีที่สุดในอุตสาหกรรม เพื่อให้ผู้ร่วมพัฒนาหลักได้รับการสนับสนุนที่มีคุณค่า สร้างวงจรเชิงบวกที่นักพัฒนา, ชุมชน และโปรเจกต์เติบโตไปด้วยกัน

**ต่อไปนี้ โปรเจกต์จะเปิดกว้างมากขึ้นเพื่อส่งเสริมและคืนคุณค่าให้กับชุมชนโอเพ่นซอร์สทั้งหมด**

## จะเริ่มต้นอย่างไร?

เรารอคอย Pull Request แรกของคุณ!

คุณสามารถเริ่มต้นด้วยการสำรวจรีโพซิทอรีของเรา เลือก `good first issue` หรือเสนอแนวทางปรับปรุงของคุณเอง ทุกคอมมิตต์คือหลักฐานแห่งจิตวิญญาณโอเพ่นซอร์ส

ขอขอบคุณสำหรับความสนใจและการมีส่วนร่วมของคุณ

# 🏢 รุ่นสำหรับองค์กร (Enterprise Edition)

ต่อยอดจาก Community Edition เราภูมิใจนำเสนอ **Cherry Studio Enterprise Edition** — แพลตฟอร์ม AI สำหรับเพิ่มประสิทธิภาพและบริหารจัดการที่ติดตั้งใช้งานในองค์กรโดยเฉพาะ เหมาะสำหรับทีมและองค์กรยุคใหม่

Enterprise Edition เข้ามาแก้ไขปัญหาหลักด้านการทำงานร่วมกันของทีมโดยรวมศูนย์จัดการทรัพยากร AI, ความรู้ และข้อมูล ช่วยให้องค์กรเพิ่มประสิทธิภาพ สร้างนวัตกรรม และรักษามาตรฐานความปลอดภัยและความเป็นส่วนตัวของข้อมูลอย่างสมบูรณ์

## จุดเด่นหลัก

- **บริหารจัดการโมเดลแบบรวมศูนย์**: เชื่อมต่อและจัดการ LLMs จากคลาวด์ (เช่น OpenAI, Anthropic, Google Gemini) และโมเดลภายในองค์กรได้ในที่เดียว พนักงานสามารถใช้งานได้ทันทีโดยไม่ต้องตั้งค่าเองทีละคน
- **ฐานความรู้ระดับองค์กร**: สร้าง จัดการ และแบ่งปันฐานความรู้สำหรับทั้งทีม รักษาความรู้และความสม่ำเสมอ ให้สมาชิกทีมโต้ตอบกับ AI โดยใช้ข้อมูลที่ถูกต้องและเป็นหนึ่งเดียว
- **ควบคุมสิทธิ์การเข้าถึงอย่างละเอียด**: จัดการบัญชีพนักงานและกำหนดสิทธิ์ตามบทบาทสำหรับโมเดล ฐานความรู้ และฟีเจอร์ต่าง ๆ ผ่านระบบแอดมินรวมศูนย์
- **ติดตั้งแบบส่วนตัวสมบูรณ์**: ติดตั้งบริการ backend ทั้งหมดในเซิร์ฟเวอร์ภายในองค์กรหรือคลาวด์ส่วนตัว ข้อมูลของคุณจะปลอดภัยและเป็นส่วนตัว 100% รองรับมาตรฐานความปลอดภัยและการปฏิบัติตามข้อกำหนดสูงสุด
- **บริการ backend ที่เชื่อถือได้**: ให้บริการ API ที่เสถียร พร้อมระบบสำรองข้อมูลและกู้คืนระดับองค์กร เพื่อความต่อเนื่องของธุรกิจ
