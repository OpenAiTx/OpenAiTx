# 🍒 Cherry Studio

Cherry Studio — это десктопный клиент, поддерживающий множество LLM-провайдеров, доступный для Windows, Mac и Linux.

👏 Присоединяйтесь к [Группе Telegram](https://t.me/CherryStudioAI)｜[Discord](https://discord.gg/wez8HtpxqQ) | [Группа QQ(575014769)](https://qm.qq.com/q/lo0D4qVZKi)

# 🌟 Ключевые возможности
1. **Поддержка различных поставщиков LLM**:

- ☁️ Основные облачные сервисы LLM: OpenAI, Gemini, Anthropic и другие
- 🔗 Интеграция веб-сервисов ИИ: Claude, Perplexity, [Poe](https://poe.com/) и другие
- 💻 Поддержка локальных моделей с Ollama, LM Studio

2. **ИИ-ассистенты и диалоги**:

- 📚 Более 300 преднастроенных ИИ-ассистентов
- 🤖 Создание пользовательских ассистентов
- 💬 Многомодельные одновременные беседы

3. **Обработка документов и данных**:

- 📄 Поддержка текста, изображений, Office, PDF и других форматов
- ☁️ Управление файлами и резервное копирование через WebDAV
- 📊 Визуализация диаграмм Mermaid
- 💻 Подсветка синтаксиса кода

4. **Интеграция практических инструментов**:

- 🔍 Глобальный поиск
- 📝 Система управления темами
- 🔤 Перевод с помощью ИИ
- 🎯 Сортировка методом drag-and-drop
- 🔌 Поддержка мини-приложений
- ⚙️ MCP(Model Context Protocol) сервер

5. **Улучшенный пользовательский опыт**:

- 🖥️ Кроссплатформенная поддержка Windows, Mac и Linux
- 📦 Готово к использованию — не требует настройки окружения
- 🎨 Светлая/тёмная тема и прозрачное окно
- 📝 Полная поддержка Markdown-рендеринга
- 🤲 Простое совместное использование контента

# 🔧 Программа совместного развития для разработчиков

Мы запускаем Программу совместного развития Cherry Studio, чтобы поддерживать здоровую и позитивную обратную связь в экосистеме open-source. Мы уверены, что отличное ПО создаётся совместно, и каждый принятый pull request даёт проекту новую жизнь.

Мы искренне приглашаем вас присоединиться к числу наших участников и вместе формировать будущее Cherry Studio.

## Программа поощрения участников

Чтобы благодарить наших основных участников и поддерживать положительный цикл, мы разработали следующий долгосрочный план стимулов.

**Первый отслеживаемый период для этой программы — 3 квартал 2025 года (июль, август, сентябрь). Награды за этот цикл будут распределены 1 октября.**

В течение любого отслеживаемого периода (например, с 1 июля по 30 сентября для первого цикла) любой разработчик, который внесет более **30 значимых коммитов** в любой из open-source проектов Cherry Studio на GitHub, будет иметь право на следующие преимущества:

- **Спонсорство подписки Cursor**: Получите **$70 USD** в виде кредита или компенсации за вашу подписку на [Cursor](https://cursor.sh/), сделав ИИ вашим самым эффективным партнером по программированию.
- **Безлимитный доступ к моделям**: Получите **безлимитные** вызовы API для моделей **DeepSeek** и **Qwen**.
- **Доступ к передовым технологиям**: Наслаждайтесь дополнительными привилегиями, включая доступ к API таких моделей, как **Claude**, **Gemini** и **OpenAI**, чтобы всегда быть на передовой технологий.

# 🏢 Корпоративная версия

В развитие Community Edition мы рады представить **Cherry Studio Enterprise Edition** — платформу для повышения производительности и управления на базе ИИ, предназначенную для современных команд и предприятий с возможностью частного развертывания.

Enterprise Edition решает ключевые задачи командного взаимодействия, централизуя управление AI-ресурсами, знаниями и данными. Она позволяет организациям повысить эффективность, стимулировать инновации и обеспечить соответствие требованиям, при этом сохраняя 100% контроль над своими данными в защищенной среде.

## Основные преимущества

- **Единое управление моделями**: Централизованно интегрируйте и управляйте различными облачными LLM (например, OpenAI, Anthropic, Google Gemini), а также локально развернутыми приватными моделями. Сотрудники могут использовать их сразу, без индивидуальной настройки.
- **Корпоративная база знаний**: Создавайте, управляйте и делитесь командными базами знаний. Обеспечивает сохранение и единообразие знаний, позволяя сотрудникам работать с ИИ на основе унифицированной и точной информации.
- **Тонкая настройка доступа**: Легко управляйте учетными записями сотрудников и назначайте ролевые права на доступ к различным моделям, базам знаний и функциям через единый административный интерфейс.
- **Полностью приватное развертывание**: Разверните весь backend-сервис на собственных серверах или в частном облаке, чтобы ваши данные всегда оставались на 100% приватными и под вашим контролем, соответствуя самым строгим стандартам безопасности и соответствия.
- **Надежные серверные сервисы**: Обеспечивает стабильные API-сервисы и корпоративные механизмы резервного копирования и восстановления данных для гарантии непрерывности бизнеса.
