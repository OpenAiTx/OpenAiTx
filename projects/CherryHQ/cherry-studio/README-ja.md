# 🍒 Cherry Studio

Cherry Studioは、複数のLLMプロバイダーに対応したデスクトップクライアントで、Windows、Mac、Linuxで利用可能です。

👏 [Telegramグループ](https://t.me/CherryStudioAI)｜[Discord](https://discord.gg/wez8HtpxqQ) | [QQグループ(575014769)](https://qm.qq.com/q/lo0D4qVZKi) に参加しましょう

# 🌟 主な機能

1. **多様なLLMプロバイダー対応**:

- ☁️ 主要なLLMクラウドサービス: OpenAI、Gemini、Anthropic など
- 🔗 AI Webサービス連携: Claude、Peplexity、Poe ほか
- 💻 Ollama、LM Studioによるローカルモデルのサポート

2. **AIアシスタント＆会話機能**:

- 📚 300以上のプリセットAIアシスタント
- 🤖 カスタムアシスタント作成
- 💬 マルチモデル同時会話

3. **ドキュメント & データ処理**:

- 📄 テキスト、画像、Office、PDFなどの対応
- ☁️ WebDAV ファイル管理およびバックアップ
- 📊 Mermaid チャート可視化
- 💻 コード構文ハイライト

4. **実用ツール統合**:

- 🔍 グローバル検索機能
- 📝 トピック管理システム
- 🔤 AI翻訳機能
- 🎯 ドラッグ＆ドロップ並び替え
- 🔌 ミニプログラム対応
- ⚙️ MCP（Model Context Protocol）サーバー

5. **ユーザー体験の向上**:

- 🖥️ Windows、Mac、Linuxのクロスプラットフォーム対応
- 📦 すぐに利用可能、環境構築不要
- 🎨 ライト/ダークテーマおよび透明ウィンドウ
- 📝 完全なMarkdownレンダリング
- 🤲 簡単なコンテンツ共有

# 📝 ロードマップ

現在、以下の機能と改善に積極的に取り組んでいます:

1. 🎯 **コア機能**

- セレクションアシスタント - スマートなコンテンツ選択強化
- ディープリサーチ - 高度なリサーチ機能
- メモリーシステム - グローバルなコンテキスト認識
- ドキュメント前処理 - ドキュメント処理の改善
- MCPマーケットプレイス - Model Context Protocol エコシステム

2. 🗂 **ナレッジ管理**

- ノートとコレクション
- ダイナミックキャンバス可視化
- OCR機能
- TTS（テキスト読み上げ）対応

3. 📱 **プラットフォーム対応**

- HarmonyOS エディション（PC）
- Androidアプリ（第1フェーズ）
- iOSアプリ（第1フェーズ）
- マルチウィンドウ対応
- ウィンドウピン留め機能

4. 🔌 **高度な機能**

- プラグインシステム
- ASR（自動音声認識）
- アシスタントとトピックの対話リファクタリング

# 🤝 コントリビューション

Cherry Studioへのご貢献を歓迎します！貢献方法は以下の通りです:

1. **コードの貢献**: 新機能の開発や既存コードの最適化
2. **バグ修正**: 見つけたバグの修正
3. **イシュー管理**: GitHubイシューの管理サポート
4. **プロダクトデザイン**: ディスカッションへの参加
5. **ドキュメント執筆**: マニュアルやガイドの改善
6. **コミュニティ参加**: ディスカッションやユーザーサポート
7. **利用促進**: Cherry Studioの普及活動

貢献ガイドラインは[ブランチ戦略](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/branching-strategy-en.md)をご参照ください

## はじめに

1. **リポジトリをフォーク**: フォークし、ローカル環境にクローンします。
2. **ブランチを作成**: 変更内容ごとに作成します。
3. **変更をプッシュ**: 変更内容をコミットし、プッシュします。
4. **プルリクエストを作成**: 変更内容と理由を記載します。

より詳しいガイドラインは[コントリビューションガイド](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/./CONTRIBUTING.md)をご参照ください。

ご支援・ご貢献ありがとうございます！
