# 🍒 Cherry Studio

Cherry Studioは、複数のLLMプロバイダーに対応したデスクトップクライアントで、Windows、Mac、Linuxで利用可能です。

👏 [Telegramグループ](https://t.me/CherryStudioAI)｜[Discord](https://discord.gg/wez8HtpxqQ) | [QQグループ(575014769)](https://qm.qq.com/q/lo0D4qVZKi) に参加しましょう

# 🌟 主な機能

1. **多様なLLMプロバイダー対応**:

- ☁️ 主要なLLMクラウドサービス: OpenAI、Gemini、Anthropic など
- 🔗 AI Webサービス連携: Claude、Peplexity、Poe ほか
- 💻 Ollama、LM Studioによるローカルモデルのサポート

2. **AIアシスタント＆会話機能**:

- 📚 300以上のプリセットAIアシスタント
- 🤖 カスタムアシスタント作成  
- 💬 複数モデル同時会話

3. **ドキュメント & データ処理**:

- 📄 テキスト、画像、Office、PDFなどの対応
- ☁️ WebDAVファイル管理とバックアップ
- 📊 Mermaidチャート可視化
- 💻 コード構文ハイライト

4. **実用ツール統合**:

- 🔍 グローバル検索機能
- 📝 トピック管理システム
- 🔤 AI搭載翻訳
- 🎯 ドラッグ＆ドロップ並べ替え
- 🔌 ミニプログラム対応
- ⚙️ MCP（Model Context Protocol）サーバー

5. **ユーザー体験の向上**:

- 🖥️ Windows、Mac、Linuxのクロスプラットフォーム対応
- 📦 すぐに使える、環境構築不要
- 🎨 ライト／ダークテーマ・ウィンドウ透過
- 📝 完全なMarkdownレンダリング
- 🤲 簡単コンテンツ共有

# 🤝 コントリビューション

Cherry Studioへの貢献を歓迎します！貢献方法は以下の通りです：

1. **コードの貢献**：新機能の開発や既存コードの最適化
2. **バグ修正**：見つけたバグの修正
3. **Issue管理**：GitHubのIssue管理
4. **プロダクトデザイン**：設計ディスカッションへの参加
5. **ドキュメント作成**：ユーザーマニュアルやガイドの改善
6. **コミュニティ参加**：議論やユーザーサポート
7. **利用促進**：Cherry Studioの普及

貢献ガイドラインについては[ブランチ戦略](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/branching-strategy-en.md)を参照してください

## はじめかた

1. **リポジトリのフォーク**：フォークしてローカル環境へクローン
2. **ブランチの作成**：変更内容ごとにブランチ作成
3. **変更のコミット**：変更をコミット＆プッシュ
4. **プルリクエストの作成**：変更点と理由を記載して提出

より詳細なガイドラインは[コントリビューションガイド](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/./CONTRIBUTING.md)をご覧ください。

**このプログラムの初回トラッキング期間は2025年第3四半期（7月・8月・9月）です。このサイクルの報酬は10月1日に配布されます。**

各トラッキング期間内（例：初回サイクルは7月1日～9月30日）、Cherry StudioのGitHub上のオープンソースプロジェクトに対し**30件以上の意義あるコミット**を行った開発者は、以下の特典を受ける資格があります：

- **Cursor サブスクリプション支援**： [Cursor](https://cursor.sh/) のサブスクリプションに対し **70ドル（USD）** のクレジットまたは返金を受け取れます。AIを最も効率的なコーディングパートナーに。

- **最新技術へのアクセス**： **Claude**、**Gemini**、**OpenAI** などのモデルAPIアクセスなど、最先端技術の特典も随時提供。

# 🏢 エンタープライズエディション

コミュニティエディションを基盤に、**Cherry Studio エンタープライズエディション** をご紹介します。これは、現代のチームや企業向けに設計された、プライベート展開可能なAI生産性・管理プラットフォームです。

エンタープライズエディションは、AIリソース・ナレッジ・データの一元管理を通じて、チームコラボレーションの課題を解決します。組織が効率性を高め、イノベーションを促進し、コンプライアンスを確保しつつ、100%自社データのコントロールを安全な環境で実現します。

## 主な利点

- **統合モデル管理**： OpenAI、Anthropic、Google Geminiなどのクラウド型LLMや、ローカル展開のプライベートモデルを一元的に統合・管理。従業員は個別設定不要ですぐに利用可能。
- **企業向けナレッジベース**： チーム全体で利用できるナレッジベースを構築・管理・共有。知識の継承と一貫性を確保し、統一かつ正確な情報に基づいたAI活用を実現。
- **きめ細かなアクセス制御**： 統合管理画面から従業員アカウントを容易に管理し、各モデルやナレッジベース、機能ごとにロールベースの権限を割り当て可能。
- **完全プライベート展開**： バックエンドサービス全体を自社サーバーやプライベートクラウドに展開可能。データは100%自社管理下で、最も厳格なセキュリティ・コンプライアンス要件にも対応。
- **信頼性の高いバックエンドサービス**： 安定したAPIサービス、エンタープライズ級のデータバックアップとリカバリ機構により、事業継続性を確保。
