# 🌟 ویژگی‌های کلیدی
1. **پشتیبانی از ارائه‌دهندگان متنوع مدل‌های زبانی بزرگ (LLM)**:

- ☁️ خدمات ابری اصلی مدل‌های زبانی: OpenAI، Gemini، Anthropic و سایرین
- 🔗 یکپارچه‌سازی سرویس‌های وب هوش مصنوعی: Claude، Perplexity، Poe و دیگران
- 💻 پشتیبانی از مدل‌های محلی با Ollama، LM Studio

2. **دستیارهای هوش مصنوعی و گفتگوها**:

- 📚 بیش از ۳۰۰ دستیار هوش مصنوعی از پیش پیکربندی شده
- 🤖 ایجاد دستیار سفارشی
- 💬 گفتگوهای همزمان با چند مدل

3. **پردازش سند و داده**:

- 📄 پشتیبانی از متن، تصویر، آفیس، PDF و موارد دیگر
- ☁️ مدیریت و پشتیبان‌گیری فایل WebDAV
- 📊 مصورسازی نمودار Mermaid
- 💻 برجسته‌سازی نحوی کد

4. **ادغام ابزارهای کاربردی**:

- 🔍 قابلیت جستجوی سراسری
- 📝 سیستم مدیریت موضوعات
- 🔤 ترجمه مبتنی بر هوش مصنوعی
- 🎯 مرتب‌سازی با کشیدن و رها کردن
- 🔌 پشتیبانی از برنامه‌های کوچک
- ⚙️ سرور MCP (پروتکل زمینه مدل)

5. **تجربه کاربری ارتقاء یافته**:

- 🖥️ پشتیبانی چندسکویی برای ویندوز، مک و لینوکس
- 📦 آماده استفاده - بدون نیاز به تنظیم محیط
- 🎨 تم‌های روشن/تاریک و پنجره شفاف
- 📝 رندر کامل Markdown
- 🤲 اشتراک‌گذاری آسان محتوا

# 🤝 مشارکت

ما از مشارکت در Cherry Studio استقبال می‌کنیم! در اینجا برخی راه‌های مشارکت آورده شده است:

1. **مشارکت در کدنویسی**: توسعه ویژگی‌های جدید یا بهینه‌سازی کدهای موجود.
2. **رفع باگ‌ها**: ارائه راه‌حل برای مشکلات یافت‌شده.
3. **مدیریت مسائل**: کمک به مدیریت مسائل گیت‌هاب.
4. **طراحی محصول**: شرکت در بحث‌های طراحی.
5. **نوشتن مستندات**: بهبود راهنماها و مستندات کاربری.
6. **تعامل با جامعه**: شرکت در بحث‌ها و کمک به کاربران.
7. **ترویج استفاده**: معرفی Cherry Studio به دیگران.

برای راهنمایی مشارکت به [استراتژی شاخه‌بندی](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/branching-strategy-en.md) مراجعه کنید

## شروع کار

1. **فورک مخزن**: مخزن را فورک کرده و روی دستگاه خود کلون کنید.
2. **ایجاد یک شاخه**: برای تغییرات خود.
3. **ثبت تغییرات**: تغییرات را کامیت و پوش کنید.
4. **باز کردن Pull Request**: تغییرات و دلایل خود را شرح دهید.

برای راهنمایی‌های بیشتر لطفاً به [راهنمای مشارکت](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/CONTRIBUTING.md) ما مراجعه کنید.

سپاس از حمایت و مشارکت شما!

# 🔧 برنامه هم‌آفرینی توسعه‌دهندگان

ما برنامه هم‌آفرینی توسعه‌دهندگان Cherry Studio را راه‌اندازی کرده‌ایم تا چرخه‌ای سالم و بازخورد مثبت در اکوسیستم متن‌باز ایجاد کنیم. باور داریم نرم‌افزار عالی با همکاری ساخته می‌شود و هر Pull Request پذیرفته شده به پروژه جان تازه‌ای می‌بخشد.

ما صمیمانه شما را به جمع مشارکت‌کنندگان دعوت می‌کنیم تا آینده Cherry Studio را با هم شکل دهیم.

## برنامه پاداش مشارکت‌کنندگان

برای قدردانی از مشارکت‌کنندگان اصلی و ایجاد چرخه‌ای نیکو، برنامه مشوق بلندمدت زیر را راه‌اندازی کرده‌ایم.

**دوره نخست رهگیری این برنامه Q3 2025 (جولای، آگوست، سپتامبر) خواهد بود. پاداش‌های این دوره در تاریخ ۱ اکتبر اهدا خواهد شد.**
در هر دوره پیگیری (مثلاً از ۱ ژوئیه تا ۳۰ سپتامبر برای چرخه اول)، هر توسعه‌دهنده‌ای که بیش از **۳۰ کامیت معنادار** به هر یک از پروژه‌های متن‌باز Cherry Studio در GitHub مشارکت کند، واجد شرایط دریافت مزایای زیر خواهد بود:

- **اسپانسر اشتراک Cursor**: دریافت اعتبار یا بازپرداخت **۷۰ دلار آمریکا** برای اشتراک [Cursor](https://cursor.sh/)، تا هوش مصنوعی به موثرترین شریک کدنویسی شما تبدیل شود.
- **دسترسی نامحدود به مدل‌ها**: دریافت **تماس‌های API نامحدود** برای مدل‌های **DeepSeek** و **Qwen**.
- **دسترسی به فناوری‌های نوین**: برخورداری از امتیازات گاه‌به‌گاه، از جمله دسترسی API به مدل‌هایی مانند **Claude**، **Gemini** و **OpenAI**، تا همیشه در خط مقدم فناوری باشید.

## رشد مشترک و برنامه‌های آتی

یک جامعه پویا نیروی محرکه پشت هر پروژه متن‌باز پایدار است. با رشد Cherry Studio، برنامه پاداش‌های ما نیز توسعه خواهد یافت. ما متعهدیم که مزایای خود را همواره با بهترین ابزارها و منابع صنعت همسو کنیم. این امر باعث می‌شود که مشارکت‌کنندگان اصلی حمایت معناداری دریافت کنند و چرخه مثبتی شکل بگیرد که توسعه‌دهندگان، جامعه و پروژه همزمان رشد کنند.

**از این پس، پروژه رویکردی بازتر اتخاذ خواهد کرد تا به کل جامعه متن‌باز بازگرداند.**

# 🏢 نسخه سازمانی

بر پایه نسخه جامعه، با افتخار نسخه **Cherry Studio Enterprise Edition** را معرفی می‌کنیم—یک پلتفرم مدیریت و بهره‌وری هوش مصنوعی قابل استقرار خصوصی، ویژه تیم‌ها و سازمان‌های مدرن.

نسخه سازمانی به چالش‌های اصلی همکاری تیمی با متمرکزسازی مدیریت منابع هوش مصنوعی، دانش و داده‌ها پاسخ می‌دهد. این نسخه سازمان‌ها را قادر می‌سازد تا بهره‌وری را افزایش دهند، نوآوری را تقویت کنند و از رعایت مقررات اطمینان حاصل نمایند؛ همه این‌ها با حفظ کنترل ۱۰۰٪ داده‌ها در محیطی امن.

## مزایای اصلی

- **مدیریت یکپارچه مدل‌ها**: ادغام و مدیریت متمرکز انواع LLM ابری (مانند OpenAI، Anthropic، Google Gemini) و مدل‌های خصوصی مستقر شده به صورت محلی. کارکنان بدون پیکربندی فردی می‌توانند به راحتی از آن‌ها استفاده کنند.
- **پایگاه دانش در سطح سازمانی**: ساخت، مدیریت و اشتراک‌گذاری پایگاه‌های دانش در سطح تیم. تضمین حفظ و یکنواختی دانش، و امکان تعامل اعضای تیم با هوش مصنوعی بر اساس اطلاعات دقیق و یکپارچه.
- **کنترل دسترسی دقیق**: مدیریت آسان حساب‌های کارکنان و تخصیص مجوزهای مبتنی بر نقش برای مدل‌ها، پایگاه‌های دانش و قابلیت‌ها از طریق پنل مدیریتی یکپارچه.
- **استقرار کاملاً خصوصی**: استقرار کل سرویس بک‌اند روی سرورهای داخلی یا ابر خصوصی، تا داده‌های شما کاملاً خصوصی و تحت کنترل شما باقی بماند و استانداردهای سخت‌گیرانه امنیت و انطباق را رعایت کند.
- **سرویس‌های بک‌اند قابل اعتماد**: ارائه سرویس‌های API پایدار و مکانیسم‌های پشتیبان‌گیری و بازیابی داده‌ها در سطح سازمانی برای تضمین تداوم کسب‌وکار.
