# 🤝 مشارکت

ما از مشارکت شما در Cherry Studio استقبال می‌کنیم! راه‌های مختلفی برای مشارکت وجود دارد:

1. **مشارکت در کد**: توسعه ویژگی‌های جدید یا بهینه‌سازی کد موجود.
2. **رفع باگ‌ها**: ارسال رفع اشکال برای باگ‌هایی که پیدا می‌کنید.
3. **مدیریت مسائل**: کمک به مدیریت مسائل گیت‌هاب.
4. **طراحی محصول**: شرکت در بحث‌های طراحی.
5. **نوشتن مستندات**: بهبود راهنماها و مستندات کاربران.
6. **تعامل با جامعه**: شرکت در گفتگوها و کمک به کاربران.
7. **ترویج استفاده**: معرفی Cherry Studio به دیگران.

برای راهنمایی مشارکت به [استراتژی شاخه‌بندی](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/docs/en/guides/branching-strategy.md) مراجعه کنید

## شروع کار

1. **فورک کردن مخزن**: مخزن را فورک و روی دستگاه خود کلون کنید.
2. **ایجاد یک شاخه**: برای تغییرات خود یک شاخه بسازید.
3. **ارسال تغییرات**: تغییرات خود را کامیت و پوش کنید.
4. **باز کردن Pull Request**: تغییرات و دلایل خود را توضیح دهید.

برای راهنمایی‌های بیشتر به [راهنمای مشارکت](https://raw.githubusercontent.com/CherryHQ/cherry-studio/main/CONTRIBUTING.md) مراجعه کنید.

از حمایت و مشارکت شما سپاسگزاریم!

# 🔧 برنامه هم‌آفرینی توسعه‌دهندگان

ما برنامه هم‌آفرینی توسعه‌دهندگان Cherry Studio را برای تقویت اکوسیستم متن‌باز و ایجاد چرخه بازخورد مثبت راه‌اندازی کرده‌ایم. باور داریم نرم‌افزار عالی با همکاری ساخته می‌شود و هر Pull Request پذیرفته شده به پروژه جان تازه‌ای می‌بخشد.

با احترام از شما دعوت می‌کنیم به جمع مشارکت‌کنندگان ما بپیوندید و آینده Cherry Studio را با ما شکل دهید.

## برنامه پاداش مشارکت‌کنندگان

برای قدردانی از مشارکت‌کنندگان کلیدی و ایجاد چرخه مثبت، برنامه پاداش بلندمدتی را طراحی کرده‌ایم.

**دوره اولیه ردیابی برای این برنامه سه‌ماهه سوم سال ۲۰۲۵ خواهد بود (جولای، آگوست، سپتامبر). پاداش‌های این دوره در تاریخ ۱ اکتبر توزیع خواهند شد.**

در هر دوره ردیابی (مثلاً از ۱ جولای تا ۳۰ سپتامبر برای اولین چرخه)، هر توسعه‌دهنده‌ای که بیش از **۳۰ کامیت معنادار** به هر یک از پروژه‌های متن‌باز Cherry Studio در گیت‌هاب اضافه کند، واجد شرایط دریافت مزایای زیر خواهد بود:

- **اسپانسر اشتراک Cursor**: دریافت اعتبار یا بازپرداخت **۷۰ دلار آمریکا** برای اشتراک [Cursor](https://cursor.sh/)، تا هوش مصنوعی موثرترین شریک کدنویسی شما باشد.
- **دسترسی نامحدود به مدل‌ها**: دریافت تماس API **نامحدود** برای مدل‌های **DeepSeek** و **Qwen**.
- **دسترسی به فناوری روز**: بهره‌مندی از مزایای دوره‌ای، شامل دسترسی API به مدل‌هایی مانند **Claude**، **Gemini**، و **OpenAI**، تا همیشه در لبه فناوری باشید.

## رشد مشترک و برنامه‌های آینده

یک جامعه پویا نیروی محرکه هر پروژه متن‌باز پایدار است. با رشد Cherry Studio، برنامه پاداش‌های ما نیز گسترش خواهد یافت. ما متعهد هستیم که مزایا را با بهترین ابزارها و منابع صنعت همسو کنیم تا مشارکت‌کنندگان اصلی حمایت معناداری دریافت کنند و چرخه‌ای مثبت ایجاد شود که توسعه‌دهندگان، جامعه و پروژه با هم رشد کنند.

**در آینده، پروژه رویکرد بازتری را برای حمایت از کل جامعه متن‌باز در پیش خواهد گرفت.**

# 🏢 نسخه سازمانی

بر پایه نسخه جامعه، با افتخار **Cherry Studio Enterprise Edition** را معرفی می‌کنیم—یک پلتفرم خصوصی قابل استقرار برای بهره‌وری و مدیریت هوش مصنوعی، ویژه تیم‌ها و سازمان‌های مدرن.

نسخه سازمانی چالش‌های اصلی همکاری تیمی را با متمرکز کردن مدیریت منابع هوش مصنوعی، دانش و داده‌ها رفع می‌کند. این نسخه سازمان‌ها را توانمند می‌سازد تا کارایی را افزایش دهند، نوآوری ایجاد کنند و مطمئن شوند که الزامات انطباق را رعایت می‌کنند، در حالی که کنترل کامل داده‌ها در محیطی امن حفظ می‌شود.

## مزایای اصلی

- **مدیریت یکپارچه مدل‌ها**: یکپارچه‌سازی و مدیریت مرکزی مدل‌های ابری LLM (مانند OpenAI، Anthropic، Google Gemini) و مدل‌های خصوصی نصب‌شده محلی. کارکنان می‌توانند بدون پیکربندی فردی از آن‌ها استفاده کنند.
- **پایگاه دانش در سطح سازمانی**: ساخت، مدیریت و اشتراک پایگاه‌های دانش تیمی. حفظ و ثبات دانش را تضمین می‌کند و امکان تعامل اعضا با هوش مصنوعی بر اساس اطلاعات یکپارچه و دقیق را فراهم می‌آورد.
- **کنترل دسترسی دقیق**: مدیریت ساده حساب‌های کارکنان و تخصیص مجوزهای مبتنی بر نقش برای مدل‌ها، پایگاه‌های دانش و قابلیت‌ها از طریق پنل مدیریتی یکپارچه.
- **استقرار کاملاً خصوصی**: استقرار کل سرویس بک‌اند روی سرورهای داخلی یا فضای ابری خصوصی شما، تا داده‌هایتان ۱۰۰٪ خصوصی و تحت کنترل باقی بماند و استانداردهای امنیت و انطباق سخت را رعایت کند.
- **سرویس‌های بک‌اند قابل اعتماد**: ارائه سرویس‌های پایدار API و مکانیزم‌های پشتیبان‌گیری و بازیابی داده‌ها در سطح سازمانی برای تضمین تداوم کسب‌وکار.

# 🔗 پروژه‌های مرتبط

- [new-api](https://github.com/QuantumNous/new-api): درگاه نسل جدید LLM و سامانه مدیریت دارایی‌های هوش مصنوعی با پشتیبانی از چند زبان.

- [one-api](https://github.com/songquanpeng/one-api): سامانه مدیریت و توزیع API مدل‌های زبانی با پشتیبانی از مدل‌های رایج مانند OpenAI، Azure و Anthropic. دارای رابط API یکپارچه، مناسب برای مدیریت کلید و توزیع ثانویه.

- [Poe](https://poe.com/): Poe دسترسی به بهترین هوش‌های مصنوعی را در یک مکان فراهم می‌کند. GPT-5، Claude Opus 4.1، DeepSeek-R1، Veo 3، ElevenLabs و میلیون‌ها مدل دیگر را کاوش کنید.

- [ublacklist](https://github.com/iorate/ublacklist): جلوگیری از نمایش سایت‌های خاص در نتایج جستجوی گوگل
