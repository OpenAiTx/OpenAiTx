# üè¢ Enterprise Edition

Berdasarkan Community Edition, kami dengan bangga memperkenalkan **Cherry Studio Enterprise Edition**‚Äîplatform produktivitas dan manajemen AI yang dapat di-deploy secara privat, dirancang untuk tim dan perusahaan modern.

Enterprise Edition mengatasi tantangan utama kolaborasi tim dengan memusatkan pengelolaan sumber daya AI, pengetahuan, dan data. Ini memberdayakan organisasi untuk meningkatkan efisiensi, mendorong inovasi, dan memastikan kepatuhan, sembari menjaga kendali 100% atas data mereka dalam lingkungan yang aman.

## Keunggulan Utama

- **Manajemen Model Terpadu**: Integrasi dan kelola berbagai LLM berbasis cloud (mis. OpenAI, Anthropic, Google Gemini) dan model privat yang di-deploy secara lokal. Karyawan dapat langsung menggunakannya tanpa konfigurasi individu.
- **Basis Pengetahuan Kelas Perusahaan**: Bangun, kelola, dan bagikan basis pengetahuan tim. Pastikan pengetahuan tetap tersimpan dan konsisten, memungkinkan anggota tim berinteraksi dengan AI berdasarkan informasi yang terintegrasi dan akurat.
- **Kontrol Akses Granular**: Kelola akun karyawan dan tetapkan izin berbasis peran untuk model, basis pengetahuan, dan fitur yang berbeda melalui backend admin terpusat.
- **Deploy Sepenuhnya Privat**: Deploy seluruh layanan backend di server lokal atau cloud privat Anda, memastikan data Anda 100% privat dan berada di bawah kendali Anda untuk memenuhi standar keamanan dan kepatuhan paling ketat.
- **Layanan Backend Andal**: Menyediakan layanan API stabil, mekanisme backup dan pemulihan data kelas perusahaan untuk memastikan kelangsungan bisnis.
