# ğŸ¢ Enterprise Edition

Berdasarkan Community Edition, kami dengan bangga memperkenalkan **Cherry Studio Enterprise Edition**â€”platform AI produktivitas dan manajemen yang dapat di-deploy secara privat, dirancang untuk tim dan perusahaan modern.

Enterprise Edition menangani tantangan utama kolaborasi tim dengan memusatkan manajemen sumber daya AI, pengetahuan, dan data. Solusi ini memberdayakan organisasi untuk meningkatkan efisiensi, mendorong inovasi, dan memastikan kepatuhan, sekaligus menjaga kontrol penuh atas data Anda dalam lingkungan yang aman.

## Keunggulan Inti

- **Manajemen Model Terpadu**: Integrasi dan manajemen terpusat untuk berbagai LLM berbasis cloud (misal OpenAI, Anthropic, Google Gemini) serta model privat yang di-deploy secara lokal. Karyawan dapat langsung menggunakannya tanpa konfigurasi individu.
- **Basis Pengetahuan Kelas Perusahaan**: Bangun, kelola, dan bagikan basis pengetahuan untuk seluruh tim. Pastikan pengetahuan tetap tersimpan dan konsisten, sehingga anggota tim dapat berinteraksi dengan AI berdasarkan informasi yang terintegrasi dan akurat.
- **Kontrol Akses Granular**: Kelola akun karyawan dan tetapkan hak akses berbasis peran untuk setiap model, basis pengetahuan, dan fitur melalui backend admin yang terpusat.
- **Deploy Secara Privat Sepenuhnya**: Deploy seluruh layanan backend di server on-premises atau private cloud Anda, memastikan data Anda tetap 100% privat dan di bawah kendali Anda demi memenuhi standar keamanan dan kepatuhan yang paling ketat.
- **Layanan Backend Andal**: Menyediakan layanan API yang stabil, mekanisme backup dan pemulihan data kelas perusahaan untuk menjamin kelangsungan bisnis.
