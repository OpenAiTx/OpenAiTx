# 🍒 Cherry Studio

Cherry Studio adalah klien desktop yang mendukung berbagai penyedia LLM, tersedia untuk Windows, Mac, dan Linux.

👏 Bergabunglah dengan [Grup Telegram](https://t.me/CherryStudioAI)｜[Discord](https://discord.gg/wez8HtpxqQ) | [Grup QQ(575014769)](https://qm.qq.com/q/lo0D4qVZKi)

# 🌟 Fitur Utama
1. **Dukungan Beragam Penyedia LLM**:

- ☁️ Layanan Cloud LLM Utama: OpenAI, Gemini, Anthropic, dan lainnya
- 🔗 Integrasi Layanan Web AI: Claude, Perplexity, Poe, dan lain-lain
- 💻 Dukungan Model Lokal dengan Ollama, LM Studio

2. **Asisten AI & Percakapan**:

- 📚 300+ Asisten AI Pra-konfigurasi
- 🤖 Pembuatan Asisten Kustom
- 💬 Percakapan Simultan Multi-model

3. **Pemrosesan Dokumen & Data**:

- 📄 Mendukung Teks, Gambar, Office, PDF, dan lainnya
- ☁️ Manajemen dan Backup File WebDAV
- 📊 Visualisasi Grafik Mermaid
- 💻 Penyorotan Sintaksis Kode

4. **Integrasi Alat Praktis**:

- 🔍 Fitur Pencarian Global
- 📝 Sistem Manajemen Topik
- 🔤 Terjemahan Berbasis AI
- 🎯 Pengurutan Drag-and-drop
- 🔌 Dukungan Mini Program
- ⚙️ Server MCP(Model Context Protocol)

5. **Pengalaman Pengguna yang Ditingkatkan**:

- 🖥️ Dukungan lintas platform untuk Windows, Mac, dan Linux
- 📦 Siap Digunakan - Tanpa Penyiapan Lingkungan
- 🎨 Tema Terang/Gelap dan Jendela Transparan
- 📝 Rendering Markdown Lengkap
- 🤲 Berbagi Konten Mudah

# 🏢 Enterprise Edition

Berdasarkan Community Edition, kami dengan bangga memperkenalkan **Cherry Studio Enterprise Edition**—platform AI produktivitas dan manajemen yang dapat di-deploy secara privat, dirancang untuk tim dan perusahaan modern.

Enterprise Edition mengatasi tantangan utama kolaborasi tim dengan memusatkan pengelolaan sumber daya AI, pengetahuan, dan data. Ini memberdayakan organisasi untuk meningkatkan efisiensi, mendorong inovasi, dan memastikan kepatuhan, sekaligus menjaga kontrol penuh atas data mereka di lingkungan yang aman.

## Keunggulan Utama

- **Manajemen Model Terpadu**: Mengintegrasikan dan mengelola berbagai LLM berbasis cloud (misal OpenAI, Anthropic, Google Gemini) dan model privat yang di-deploy secara lokal secara terpusat. Karyawan dapat langsung menggunakannya tanpa konfigurasi individual.
- **Basis Pengetahuan Kelas Enterprise**: Bangun, kelola, dan bagikan basis pengetahuan tim. Menjamin retensi dan konsistensi pengetahuan, memungkinkan anggota tim berinteraksi dengan AI berdasarkan informasi yang seragam dan akurat.
- **Kontrol Akses Granular**: Kelola akun karyawan dan tetapkan izin berbasis peran untuk model, basis pengetahuan, dan fitur berbeda melalui backend admin terpadu.
- **Deploy Sepenuhnya Privat**: Deploy seluruh layanan backend di server lokal atau cloud privat Anda, memastikan data tetap 100% privat dan di bawah kendali Anda untuk memenuhi standar keamanan dan kepatuhan paling ketat.
- **Layanan Backend Andal**: Menyediakan layanan API stabil dan mekanisme backup serta pemulihan data kelas enterprise untuk menjamin kelangsungan bisnis.
