# 🌟 Funzionalità Principali
1. **Supporto per diversi fornitori LLM**:

- ☁️ Principali servizi cloud LLM: OpenAI, Gemini, Anthropic e altri
- 🔗 Integrazione con servizi web AI: Claude, Perplexity, [Poe](https://poe.com/) e altri
- 💻 Supporto per modelli locali con Ollama, LM Studio

2. **Assistenti AI & Conversazioni**:

- 📚 Oltre 300 assistenti AI preconfigurati
- 🤖 Creazione assistente personalizzato
- 💬 Conversazioni simultanee multi-modello

3. **Elaborazione documenti & dati**:

- 📄 Supporta testo, immagini, Office, PDF e altro
- ☁️ Gestione file e backup WebDAV
- 📊 Visualizzazione grafici Mermaid
- 💻 Evidenziazione sintassi codice

4. **Integrazione strumenti pratici**:

- 🔍 Funzionalità di ricerca globale
- 📝 Sistema di gestione argomenti
- 🔤 Traduzione potenziata da AI
- 🎯 Ordinamento drag-and-drop
- 🔌 Supporto mini-programmi
- ⚙️ Server MCP (Model Context Protocol)

5. **Esperienza utente migliorata**:

- 🖥️ Supporto multipiattaforma per Windows, Mac e Linux
- 📦 Pronto all’uso - Nessuna configurazione richiesta
- 🎨 Temi chiaro/scuro e finestra trasparente
- 📝 Rendering Markdown completo
- 🤲 Condivisione contenuti semplificata

# 🔧 Programma di Co-creazione per Sviluppatori

Stiamo lanciando il Programma di Co-creazione per Sviluppatori Cherry Studio per favorire un ecosistema open-source sano e con feedback positivo. Crediamo che il grande software nasca dalla collaborazione, e ogni pull request accettata dà nuova vita al progetto.

Ti invitiamo sinceramente a unirti al nostro gruppo di contributori e a plasmare insieme il futuro di Cherry Studio.

## Programma Premi per i Contributori

Per ricompensare i nostri contributori principali e creare un ciclo virtuoso, abbiamo istituito il seguente piano di incentivi a lungo termine.

**Il periodo inaugurale di monitoraggio per questo programma sarà il Q3 2025 (luglio, agosto, settembre). I premi per questo ciclo saranno distribuiti il 1° ottobre.**

Durante qualsiasi periodo di monitoraggio (es. dal 1° luglio al 30 settembre per il primo ciclo), ogni sviluppatore che contribuisce con più di **30 commit significativi** a uno qualsiasi dei progetti open-source di Cherry Studio su GitHub sarà idoneo ai seguenti vantaggi:

- **Sponsorizzazione Abbonamento Cursor**: Ricevi un credito o rimborso di **$70 USD** per il tuo abbonamento a [Cursor](https://cursor.sh/), rendendo l’IA il tuo partner di coding più efficiente.
- **Accesso Illimitato ai Modelli**: Ottieni **chiamate API illimitate** per i modelli **DeepSeek** e **Qwen**.
- **Accesso a Tecnologie all’Avanguardia**: Godi di vantaggi occasionali, inclusi accessi API a modelli come **Claude**, **Gemini** e **OpenAI**, mantenendoti all’avanguardia nella tecnologia.

# 🏢 Enterprise Edition

Basandosi sulla Community Edition, siamo orgogliosi di presentare la **Cherry Studio Enterprise Edition**—una piattaforma di produttività e gestione AI, installabile privatamente, progettata per team e aziende moderne.

L’Enterprise Edition affronta le principali sfide nella collaborazione di team centralizzando la gestione di risorse AI, conoscenze e dati. Permette alle organizzazioni di aumentare l’efficienza, favorire l’innovazione e garantire la conformità, mantenendo sempre il 100% del controllo sui dati in un ambiente sicuro.

## Vantaggi Principali

- **Gestione Unificata dei Modelli**: Integra e gestisci centralmente vari LLM cloud (es. OpenAI, Anthropic, Google Gemini) e modelli privati in locale. I dipendenti possono usarli immediatamente senza configurazione individuale.
- **Knowledge Base di Livello Enterprise**: Costruisci, gestisci e condividi knowledge base a livello di team. Garantisce conservazione e coerenza delle informazioni, consentendo ai membri di interagire con l’IA su basi unificate e accurate.
- **Controllo Accessi Granulare**: Gestisci facilmente account dipendenti e assegna permessi basati sui ruoli per modelli, knowledge base e funzionalità dal backend amministrativo unificato.
- **Deployment Completamente Privato**: Distribuisci l’intero servizio backend sui tuoi server on-premise o cloud privato, assicurando che i tuoi dati restino sempre privati e sotto il tuo controllo, per soddisfare i più severi standard di sicurezza e conformità.
- **Servizi Backend Affidabili**: Offre servizi API stabili e meccanismi di backup e ripristino dati di livello enterprise, per garantire la continuità operativa.
