# 🌟 주요 기능
1. **다양한 LLM 공급자 지원**:

- ☁️ 주요 LLM 클라우드 서비스: OpenAI, Gemini, Anthropic 등
- 🔗 AI 웹 서비스 연동: Claude, Perplexity, [Poe](https://poe.com/) 등
- 💻 Ollama, LM Studio를 통한 로컬 모델 지원

2. **AI 어시스턴트 및 대화**:

- 📚 300개 이상의 사전 구성된 AI 어시스턴트
- 🤖 사용자 정의 어시스턴트 생성
- 💬 다중 모델 동시 대화

- 📄 텍스트, 이미지, 오피스, PDF 등 지원
- ☁️ WebDAV 파일 관리 및 백업
- 📊 Mermaid 차트 시각화
- 💻 코드 구문 강조

4. **실용적 도구 통합**:

- 🔍 글로벌 검색 기능
- 📝 주제 관리 시스템
- 🔤 AI 기반 번역
- 🎯 드래그 앤 드롭 정렬
- 🔌 미니 프로그램 지원
- ⚙️ MCP(모델 컨텍스트 프로토콜) 서버

5. **향상된 사용자 경험**:

- 🖥️ Windows, Mac, Linux 크로스플랫폼 지원
- 📦 즉시 사용 가능 - 별도 환경 설정 불필요
- 🎨 라이트/다크 테마 및 투명 창 지원
- 📝 완벽한 마크다운 렌더링
- 🤲 쉬운 콘텐츠 공유

# 🔧 개발자 공동창작 프로그램

Cherry Studio 개발자 공동창작 프로그램을 시작하여 오픈소스 생태계 내에서 건강하고 긍정적인 피드백 루프를 조성하고자 합니다. 우리는 훌륭한 소프트웨어가 협업을 통해 만들어진다고 믿으며, 병합된 모든 풀 리퀘스트가 프로젝트에 새로운 생명을 불어넣는다고 생각합니다.

기여자 여러분을 진심으로 초대하며, 저희와 함께 Cherry Studio의 미래를 만들어가길 바랍니다.

## 기여자 보상 프로그램

핵심 기여자들에게 보답하고 선순환 구조를 만들기 위해 다음과 같은 장기 인센티브 플랜을 마련했습니다.

**이 프로그램의 첫 추적 기간은 2025년 3분기(7월, 8월, 9월)입니다. 이번 사이클의 보상은 10월 1일에 분배됩니다.**

각 추적 기간(예: 첫 번째 사이클의 경우 7월 1일부터 9월 30일까지) 동안, GitHub에 있는 Cherry Studio의 오픈소스 프로젝트에 **의미 있는 커밋 30회 이상** 기여한 개발자는 다음과 같은 혜택을 받을 수 있습니다:

- **Cursor 구독 지원**: [Cursor](https://cursor.sh/) 구독을 위해 **70달러(USD)** 크레딧 또는 환급을 받아 AI를 가장 효율적인 코딩 파트너로 만드세요.
- **무제한 모델 사용**: **DeepSeek** 및 **Qwen** 모델에 대해 **무제한** API 호출이 가능합니다.
- **최신 기술 접근**: **Claude**, **Gemini**, **OpenAI** 등과 같은 모델의 API 접근 등 최신 기술을 경험할 수 있는 특별 혜택이 제공됩니다.

## 함께 성장 & 미래 계획

활기찬 커뮤니티는 모든 지속가능한 오픈소스 프로젝트의 원동력입니다. Cherry Studio가 성장함에 따라 보상 프로그램도 확장됩니다. 업계 최고 수준의 도구와 리소스에 혜택을 지속적으로 맞추겠습니다. 이를 통해 핵심 기여자들에게 의미 있는 지원을 제공하고, 개발자·커뮤니티·프로젝트가 함께 성장하는 선순환을 만듭니다.

**앞으로 프로젝트는 전체 오픈소스 커뮤니티에 환원하는 점점 더 개방적인 방침을 채택할 것입니다.**

관심과 기여에 감사드립니다.

함께 만들어 갑시다.

# 🏢 엔터프라이즈 에디션

커뮤니티 에디션을 기반으로, 저희는 현대적 팀과 기업을 위한 사내 배포형 AI 생산성 및 관리 플랫폼인 **Cherry Studio 엔터프라이즈 에디션**을 소개합니다.

엔터프라이즈 에디션은 AI 리소스, 지식, 데이터를 중앙에서 관리하여 팀 협업의 핵심 문제를 해결합니다. 조직이 효율성을 높이고, 혁신을 촉진하며, 컴플라이언스를 확보하면서도 데이터를 100% 안전하게 직접 통제할 수 있게 돕습니다.

## 핵심 장점

- **통합 모델 관리**: 다양한 클라우드 기반 LLM(OpenAI, Anthropic, Google Gemini 등)과 로컬 프라이빗 모델을 중앙에서 통합·관리합니다. 직원들은 별도 설정 없이 즉시 사용할 수 있습니다.
- **엔터프라이즈급 지식베이스**: 팀 전체의 지식베이스를 구축, 관리, 공유할 수 있습니다. 지식의 보존과 일관성을 보장하며, 팀원들은 통합된 정확한 정보를 기반으로 AI와 상호작용할 수 있습니다.
- **세분화된 접근 제어**: 단일 관리자 백엔드에서 직원 계정 관리 및 모델, 지식베이스, 기능별 역할 기반 권한 할당이 쉽습니다.
- **완전 사내 배포**: 전체 백엔드 서비스를 사내 서버 또는 프라이빗 클라우드에 배포하여, 데이터가 100% 개인 소유 및 통제 하에 보관되고, 최고 수준의 보안 및 컴플라이언스 기준을 충족합니다.
- **신뢰성 높은 백엔드 서비스**: 안정적인 API 서비스와 엔터프라이즈급 데이터 백업 및 복구 메커니즘을 제공하여 비즈니스 연속성을 보장합니다.
