# 🌟 주요 기능

- ☁️ 주요 LLM 클라우드 서비스: OpenAI, Gemini, Anthropic 등

- 📄 텍스트, 이미지, 오피스, PDF 등 지원

- 🔌 미니 프로그램 지원
- ⚙️ MCP(모델 컨텍스트 프로토콜) 서버

5. **향상된 사용자 경험**:

- 🖥️ Windows, Mac, Linux 크로스 플랫폼 지원
- 📦 바로 사용 가능, 환경 설정 불필요

- 📝 완전한 마크다운 렌더링
- 🤲 손쉬운 콘텐츠 공유

# 🔧 개발자 공동창작 프로그램

Cherry Studio 개발자 공동창작 프로그램을 시작하여 오픈소스 생태계 내 건강한 선순환을 도모합니다. 위대한 소프트웨어는 협업으로 만들어지며, 모든 병합된 풀 리퀘스트가 프로젝트에 새로운 생명을 불어넣는다고 믿습니다.

진심으로 여러분을 기여자 대열에 초대하며, 함께 Cherry Studio의 미래를 만들어가고자 합니다.

## 기여자 보상 프로그램

핵심 기여자에게 보답하고 선순환을 만들기 위해 다음과 같은 장기 인센티브 플랜을 마련했습니다.

**이 프로그램의 첫 추적 기간은 2025년 3분기(7월, 8월, 9월)입니다. 해당 사이클의 보상은 10월 1일에 배포됩니다.**
임의의 트래킹 기간(예: 첫 번째 사이클의 경우 7월 1일부터 9월 30일까지) 동안, Cherry Studio의 GitHub 오픈소스 프로젝트에 **30개 이상의 의미 있는 커밋**을 기여한 개발자는 다음과 같은 혜택을 받을 수 있습니다:

- **Cursor 구독 후원**: [Cursor](https://cursor.sh/) 구독에 대해 **$70 USD**의 크레딧 또는 환급을 받아, AI를 가장 효율적인 코딩 파트너로 만드세요.
- **무제한 모델 액세스**: **DeepSeek** 및 **Qwen** 모델에 대해 **무제한** API 호출을 사용할 수 있습니다.
- **최신 기술 선도**: **Claude**, **Gemini**, **OpenAI** 등의 모델 API 액세스를 포함한 다양한 특전을 간헐적으로 제공하여, 항상 기술의 최전선에 설 수 있습니다.

## 함께 성장하기 & 향후 계획

활기찬 커뮤니티는 지속 가능한 오픈소스 프로젝트의 원동력입니다. Cherry Studio가 성장함에 따라 보상 프로그램도 확장될 것입니다. 업계 최고 수준의 도구 및 자원과 혜택을 지속적으로 연계하겠습니다. 이를 통해 핵심 기여자들에게 실질적인 지원이 제공되며, 개발자-커뮤니티-프로젝트가 함께 성장하는 선순환이 만들어집니다.

**앞으로 프로젝트는 전체 오픈소스 커뮤니티에 환원하기 위해 더욱 개방적인 방향을 추구할 것입니다.**

# 🏢 엔터프라이즈 에디션

커뮤니티 에디션을 기반으로, 저희는 **Cherry Studio 엔터프라이즈 에디션**을 자신 있게 소개합니다. 이는 현대 팀과 기업을 위한 사내 배포형 AI 생산성 및 관리 플랫폼입니다.

엔터프라이즈 에디션은 팀 협업의 핵심 과제를 해결합니다. AI 리소스, 지식, 데이터를 중앙집중적으로 관리하여, 조직이 효율성을 높이고 혁신을 촉진하며, 컴플라이언스를 준수할 수 있도록 지원합니다. 모든 데이터는 100% 안전하게 사내에 보관됩니다.

## 핵심 장점

- **통합 모델 관리**: 다양한 클라우드 기반 LLM(OpenAI, Anthropic, Google Gemini 등) 및 사내에 배포된 프라이빗 모델을 중앙에서 관리 및 통합합니다. 임직원은 별도 설정 없이 즉시 사용할 수 있습니다.
- **기업용 지식 베이스**: 팀 전체가 공유하는 지식 베이스를 구축·관리·공유하세요. 지식이 일관되고 보존되도록 하여, 팀원이 AI와 신뢰성 높은 정보로 상호작용할 수 있습니다.
- **세분화된 접근 제어**: 통합 관리 백엔드에서 임직원 계정, 각 모델·지식 베이스·기능별 역할 기반 권한을 쉽게 관리하세요.
- **완전한 프라이빗 배포**: 백엔드 전체를 온프레미스 서버 또는 프라이빗 클라우드에 배포해, 데이터의 100% 사내 보관 및 통제, 최고 수준의 보안/컴플라이언스 요구를 충족합니다.
- **신뢰할 수 있는 백엔드 서비스**: 안정적인 API 서비스, 기업용 데이터 백업·복구 메커니즘을 제공하여 비즈니스 연속성을 보장합니다.

## ✨ 온라인 데모

> 🚧 **퍼블릭 베타 안내**
>
> 엔터프라이즈 에디션은 현재 초창기 공개 베타 단계에 있으며, 기능을 지속적으로 개선하고 있습니다. 아직 완벽히 안정적이지 않을 수 있습니다. 체험 중 문제나 개선 의견이 있으면 이메일로 피드백을 주시면 감사하겠습니다.
