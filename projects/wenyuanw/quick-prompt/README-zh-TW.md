<h1 align="center">Quick Prompt</h1>

一款強大的瀏覽器擴充功能，專注於提示詞管理與快速輸入。協助使用者建立、管理和組織提示詞庫，並能在任何網頁輸入框中快速插入預設的 Prompt 內容，提高工作效率。

## ✨ 功能特性

- 📚 **提示詞管理**：便捷地建立、編輯與管理你的提示詞庫
- 🚀 **快速輸入**：在任何網頁輸入框中透過 `/p` 快速觸發提示詞選擇器
- ⌨️ 支援設定快捷鍵開啟提示詞選擇器＆將選中文本儲存為提示詞
- 📑 支援右鍵選單直接儲存選中文本為提示詞
- 🎯 支援自訂提示詞，包括分類、標題、內容、附件、標籤與變數
- 🧑‍💻 **變數支援**：提示詞中可使用 `{{變數名}}` 格式的變數，使用時可填寫具體值
- 💾 **資料備份**：支援提示詞庫匯出與匯入，方便跨裝置遷移與備份
- 🔗 **雲端同步**：提示詞庫支援 WebDAV 同步、Notion 資料庫、Gitee/GitHub Gist 同步
- 🔍 支援搜尋與篩選提示詞功能
- 🌙 自動適應系統的明暗主題

## 🚀 使用方法

1. **快速觸發**：在任何網頁的文字輸入框中輸入 `/p` 以觸發提示詞選擇器
2. **快捷鍵開啟選擇器**：使用 `Ctrl+Shift+P`（Windows/Linux）或 `Command+Shift+P`（macOS）開啟提示詞選擇器
3. **選擇提示詞**：從彈出的選擇器中點擊所需提示詞，將自動插入到當前輸入框
4. **快速儲存提示詞**：選取任意文本後，使用 `Ctrl+Shift+S`（Windows/Linux）或 `Command+Shift+S`（macOS）快速儲存為提示詞
5. **右鍵選單儲存**：選取任意文本後，右鍵點擊並選擇「儲存該提示詞」，將選取內容儲存為提示詞
6. **匯出提示詞庫**：於管理頁點擊「匯出」按鈕，將所有提示詞匯出為 JSON 檔儲存到本地
7. **匯入提示詞庫**：於管理頁點擊「匯入」按鈕，從本地 JSON 檔匯入提示詞（支援合併或覆蓋現有提示詞）

## ⚙️ 自訂配置

1. 點擊擴充圖示，然後點擊「管理提示詞」按鈕
2. 在管理頁面，你可以：
   - 新增提示詞
   - 編輯現有提示詞
   - 刪除不需要的提示詞
   - 為提示詞添加標籤分類
   - 匯出提示詞庫以備份
   - 匯入先前備份的提示詞庫

### 從 GitHub Releases 取得

1. 前往 [GitHub Releases 頁面](https://github.com/wenyuanw/quick-prompt/releases)
2. 下載最新版本已建構好的擴充套件包
3. 解壓下載的檔案
4. 按照下方說明安裝已建構的擴充功能

### 從原始碼建構

1. 複製倉庫
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. 安裝相依套件
   ```bash
   pnpm install
   ```
3. 開發與建構

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### 安裝已構建的擴充功能

#### Chrome / Edge
1. 打開擴充功能管理頁面（`chrome://extensions` 或 `edge://extensions`）
2. 啟用「開發者模式」
3. 點擊「加載已解壓的擴充功能」
4. 選擇專案的 `.output/chrome-mv3/` 目錄

#### Firefox
1. 打開 `about:debugging`
2. 點擊「此 Firefox」
3. 點擊「臨時加載附加元件」
4. 選擇專案的 `.output/firefox-mv2/` 目錄中的 `manifest.json` 檔案
