ส่วนขยายเบราว์เซอร์ที่ทรงพลัง มุ่งเน้นการจัดการและป้อนคำ Prompt อย่างรวดเร็ว ช่วยผู้ใช้สร้าง จัดการ และจัดระเบียบคลังคำ Prompt รวมถึงสามารถแทรกเนื้อหา Prompt ที่ตั้งไว้ล่วงหน้าในช่องกรอกข้อมูลบนหน้าเว็บใดก็ได้ เพิ่มประสิทธิภาพการทำงาน

## ✨ คุณสมบัติเด่น

- 📚 **การจัดการพรอมต์**: สร้าง แก้ไข และจัดการคลังพรอมต์ของคุณได้อย่างสะดวก
- 🚀 **การกรอกข้อมูลอย่างรวดเร็ว**: พิมพ์ `/p` ในกล่องกรอกข้อความของเว็บเพจใด ๆ เพื่อเรียกตัวเลือกพรอมต์อย่างรวดเร็ว
- ⌨️ รองรับการตั้งค่าปุ่มลัดเพื่อเปิดตัวเลือกพรอมต์และบันทึกข้อความที่เลือกเป็นพรอมต์
- 📑 รองรับเมนูคลิกขวาเพื่อบันทึกข้อความที่เลือกเป็นพรอมต์ได้โดยตรง
- 🎯 รองรับการกำหนดพรอมต์เอง รวมถึงหมวดหมู่ ชื่อเรื่อง เนื้อหา ไฟล์แนบ แท็ก และตัวแปร
- 🧑‍💻 **รองรับตัวแปร**: สามารถใช้ตัวแปรในรูปแบบ `{{ชื่อตัวแปร}}` ในพรอมต์ และใส่ค่าจริงขณะใช้งาน
- 💾 **สำรองข้อมูล**: รองรับการส่งออกและนำเข้าคลังพรอมต์ สะดวกต่อการย้ายและสำรองข้ามอุปกรณ์
- 🔗 **ซิงก์กับคลาวด์**: คลังพรอมต์รองรับการซิงก์ผ่าน WebDAV, ฐานข้อมูล Notion, และ Gitee/GitHub Gist
- 🔍 รองรับฟังก์ชันค้นหาและกรองพรอมต์
- 🌙 ปรับเข้ากับโหมดแสง/มืดของระบบโดยอัตโนมัติ

## 🚀 วิธีใช้งาน

1. **เรียกใช้อย่างรวดเร็ว**: พิมพ์ `/p` ในกล่องกรอกข้อความของเว็บเพจใด ๆ เพื่อเปิดตัวเลือกพรอมต์
2. **เปิดตัวเลือกด้วยปุ่มลัด**: ใช้ `Ctrl+Shift+P` (Windows/Linux) หรือ `Command+Shift+P` (macOS) เพื่อเปิดตัวเลือกพรอมต์
3. **เลือกพรอมต์**: คลิกพรอมต์ที่ต้องการจากตัวเลือกที่แสดงขึ้นมา ระบบจะใส่พรอมต์ลงในกล่องกรอกข้อความอัตโนมัติ
4. **บันทึกพรอมต์อย่างรวดเร็ว**: เลือกข้อความใด ๆ แล้วใช้ `Ctrl+Shift+S` (Windows/Linux) หรือ `Command+Shift+S` (macOS) เพื่อบันทึกเป็นพรอมต์ทันที
5. **บันทึกผ่านเมนูคลิกขวา**: เลือกข้อความใด ๆ จากนั้นคลิกขวาและเลือก "บันทึกพรอมต์นี้" เพื่อบันทึกเนื้อหาที่เลือกเป็นพรอมต์
6. **ส่งออกคลังพรอมต์**: ในหน้าจัดการคลิกปุ่ม "ส่งออก" เพื่อส่งออกพรอมต์ทั้งหมดเป็นไฟล์ JSON ลงเครื่อง
7. **นำเข้าคลังพรอมต์**: ในหน้าจัดการคลิกปุ่ม "นำเข้า" เพื่อเลือกไฟล์ JSON จากเครื่องและนำเข้าพรอมต์ (รองรับการรวม/ทับข้อมูลเดิม)

## ⚙️ การตั้งค่ากำหนดเอง

1. คลิกไอคอนส่วนขยาย จากนั้นคลิกปุ่ม "จัดการพรอมต์"
2. ในหน้าจัดการ คุณสามารถ:
   - เพิ่มพรอมต์ใหม่
   - แก้ไขพรอมต์ที่มีอยู่
   - ลบพรอมต์ที่ไม่ต้องการ
   - เพิ่มแท็กให้พรอมต์เพื่อจัดหมวดหมู่
   - ส่งออกคลังพรอมต์เพื่อสำรองข้อมูล
   - นำเข้าคลังพรอมต์ที่สำรองไว้ก่อนหน้านี้

### ดาวน์โหลดจาก GitHub Releases

1. เข้าไปที่ [หน้าดาวน์โหลด GitHub Releases](https://github.com/wenyuanw/quick-prompt/releases)
2. ดาวน์โหลดไฟล์ปลั๊กอินเวอร์ชันล่าสุดที่สร้างเสร็จแล้ว
3. แตกไฟล์ที่ดาวน์โหลด
4. ติดตั้งส่วนขยายที่สร้างเสร็จแล้วตามคำแนะนำด้านล่าง

### สร้างจากซอร์สโค้ด

1. โคลนที่เก็บโค้ด
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. ติดตั้งไลบรารีที่จำเป็น
   ```bash
   pnpm install
   ```
3. การพัฒนาและการสร้าง

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### ติดตั้งส่วนขยายที่ได้สร้างแล้ว

#### Chrome / Edge
1. เปิดหน้าจัดการส่วนขยาย (`chrome://extensions` หรือ `edge://extensions`)
2. เปิดใช้งาน "โหมดนักพัฒนา"
3. คลิก "โหลดส่วนขยายที่ถูกแตกไฟล์แล้ว"
4. เลือกไดเรกทอรี `.output/chrome-mv3/` ของโปรเจกต์

#### Firefox
1. เปิด `about:debugging`
2. คลิก "Firefox นี้"
3. คลิก "โหลดส่วนเสริมแบบชั่วคราว"
4. เลือกไฟล์ `manifest.json` ในไดเรกทอรี `.output/firefox-mv2/` ของโปรเจกต์
