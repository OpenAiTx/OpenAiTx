<h1 align="center">Quick Prompt</h1>

إضافة متصفح قوية تركز على إدارة الكلمات المفتاحية والإدخال السريع. تساعد المستخدمين على إنشاء وإدارة وتنظيم مكتبة الكلمات المفتاحية، وإدراج محتوى Prompt المُعد مسبقاً بسرعة في أي مربع إدخال على صفحات الويب، مما يعزز من كفاءة العمل.

## ✨ الميزات الرئيسية

- 📚 **إدارة الكلمات المفتاحية**: إنشاء وتحرير وإدارة مكتبة الكلمات المفتاحية الخاصة بك بسهولة
- 🚀 **إدخال سريع**: يمكنك تفعيل منتقي الكلمات المفتاحية بسرعة عبر `/p` في أي مربع إدخال نصي في صفحات الويب
- ⌨️ يدعم ضبط اختصارات لوحة المفاتيح لفتح منتقي الكلمات المفتاحية وحفظ النص المحدد ككلمة مفتاحية
- 📑 يدعم حفظ النص المحدد ككلمة مفتاحية مباشرة من قائمة النقر بزر الفأرة الأيمن
- 🎯 يدعم تخصيص الكلمات المفتاحية، بما في ذلك التصنيف، العنوان، المحتوى، المرفقات، العلامات والمتغيرات
- 🧑‍💻 **دعم المتغيرات**: يمكنك استخدام متغيرات بصيغة `{{اسم المتغير}}` ضمن الكلمات المفتاحية، ويمكنك إدخال قيم محددة عند الاستخدام
- 💾 **نسخ احتياطي للبيانات**: يدعم تصدير واستيراد مكتبة الكلمات المفتاحية، لتسهيل النقل بين الأجهزة والنسخ الاحتياطي
- 🔗 **المزامنة السحابية**: تدعم مكتبة الكلمات المفتاحية مزامنة WebDAV، قاعدة بيانات Notion، مزامنة Gitee/GitHub Gist
- 🔍 يدعم البحث وتصفية الكلمات المفتاحية
- 🌙 يتكيف تلقائياً مع وضع النظام المظلم أو الفاتح

## 🚀 طريقة الاستخدام

1. **تفعيل سريع**: أدخل `/p` في أي مربع إدخال نصي في صفحات الويب لتفعيل منتقي الكلمات المفتاحية
2. **فتح المنتقي باختصار لوحة المفاتيح**: استخدم `Ctrl+Shift+P` (Windows/Linux) أو `Command+Shift+P` (macOS) لفتح منتقي الكلمات المفتاحية
3. **اختيار كلمة مفتاحية**: انقر على الكلمة المفتاحية المطلوبة من المنتقي المنبثق، وسيتم إدراجها تلقائياً في مربع الإدخال الحالي
4. **حفظ كلمة مفتاحية بسرعة**: بعد تحديد أي نص، استخدم `Ctrl+Shift+S` (Windows/Linux) أو `Command+Shift+S` (macOS) لحفظه بسرعة ككلمة مفتاحية
5. **حفظ من قائمة النقر الأيمن**: بعد تحديد نص، انقر بزر الفأرة الأيمن واختر "حفظ هذه الكلمة المفتاحية" لحفظ المحتوى المحدد ككلمة مفتاحية
6. **تصدير مكتبة الكلمات المفتاحية**: في صفحة الإدارة، انقر على زر "تصدير" لتصدير جميع الكلمات المفتاحية كملف JSON وحفظه محلياً
7. **استيراد مكتبة الكلمات المفتاحية**: في صفحة الإدارة، انقر على زر "استيراد" لاستيراد الكلمات المفتاحية من ملف JSON محلي (يدعم الدمج أو استبدال الكلمات الحالية)

## ⚙️ الإعدادات المخصصة

1. انقر على أيقونة الإضافة، ثم انقر على زر "إدارة العبارات الترويجية"
2. في صفحة الإدارة يمكنك:
   - إضافة عبارات ترويجية جديدة
   - تحرير العبارات الترويجية الحالية
   - حذف العبارات الترويجية غير المطلوبة
   - إضافة تصنيفات للعبارات الترويجية لتنظيمها
   - تصدير مكتبة العبارات الترويجية للنسخ الاحتياطي
   - استيراد مكتبة العبارات الترويجية من نسخة احتياطية سابقة

### الحصول عليها من إصدارات GitHub

1. قم بزيارة [صفحة إصدارات GitHub](https://github.com/wenyuanw/quick-prompt/releases)
2. حمل أحدث نسخة من ملف الإضافة الجاهز
3. فك ضغط الملف الذي تم تحميله
4. وفقًا للإرشادات أدناه، قم بتثبيت الإضافة التي تم إنشاؤها بالفعل

### البناء من المصدر

1. استنساخ المستودع
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. تثبيت الاعتمادات
   ```bash
   pnpm install
   ```
3. التطوير والبناء

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### تثبيت الامتداد الذي تم إنشاؤه

#### كروم / إيدج
1. افتح صفحة إدارة الامتدادات (`chrome://extensions` أو `edge://extensions`)
2. فعّل "وضع المطور"
3. انقر على "تحميل الامتداد غير المضغوط"
4. اختر مجلد المشروع `.output/chrome-mv3/`

#### فايرفوكس
1. افتح `about:debugging`
2. انقر على "هذا الفايرفوكس"
3. انقر على "تحميل إضافة مؤقتة"
4. اختر ملف `manifest.json` داخل مجلد `.output/firefox-mv2/` الخاص بالمشروع
