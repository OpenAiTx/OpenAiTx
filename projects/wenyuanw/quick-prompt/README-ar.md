إضافة قوية للمتصفح تركز على إدارة الكلمات الدلالية والإدخال السريع. تساعد المستخدمين على إنشاء وإدارة وتنظيم مكتبة الكلمات الدلالية، كما تتيح إدراج محتوى المطالبات المعد مسبقًا بسرعة في أي حقل إدخال على صفحات الويب، مما يعزز كفاءة العمل.

> نظرًا لأن إضافات المتصفح يمكن استخدامها فقط في الويب، تم فتح مصدر إضافة [Raycast](https://github.com/wenyuanw/quick-prompt-raycast) ذات الوظائف المماثلة، وهي متوافقة مع تنسيق بيانات إضافة المتصفح وتدعم الانتقال السلس باستخدام نفس بيانات JSON.

## ✨ الميزات الوظيفية

- 📚 **إدارة العبارات المقترحة**: أنشئ، حرر وأدر مكتبة العبارات المقترحة لديك بسهولة
- 🚀 **إدخال سريع**: فعّل أداة اختيار العبارات بسرعة عبر كتابة `/p` في أي حقل إدخال نصي على صفحات الويب
- ⌨️ دعم تهيئة اختصارات لوحة المفاتيح لفتح أداة اختيار العبارات أو حفظ النص المحدد كعبارة مقترحة
- 📑 دعم حفظ النص المحدد كعبارة مقترحة مباشرة عبر قائمة الزر الأيمن
- 🎯 دعم تخصيص العبارات المقترحة، بما يشمل التصنيف، العنوان، المحتوى، العلامات والمتغيرات
- 🧑‍💻 **دعم المتغيرات**: يمكن استخدام متغيرات بصيغة `{{اسم المتغير}}` ضمن العبارات، مع إمكانية تحديد القيم عند الاستخدام
- 💾 **نسخ احتياطي للبيانات**: دعم تصدير واستيراد مكتبة العبارات لتسهيل النقل أو النسخ الاحتياطي بين الأجهزة
- 🔗 **مزامنة سحابية**: دعم مزامنة مكتبة العبارات مع قاعدة بيانات Notion أو Gitee/GitHub Gist
- 🔍 دعم البحث وتصفية العبارات المقترحة
- 🌙 التوافق التلقائي مع وضع الإضاءة أو الظلام حسب النظام

## 🚀 طريقة الاستخدام

1. **تشغيل سريع**: اكتب `/p` في أي حقل إدخال نصي على صفحة الويب لتفعيل أداة اختيار العبارات المقترحة
2. **اختصار لوحة المفاتيح**: استخدم `Ctrl+Shift+P` (ويندوز/لينكس) أو `Command+Shift+P` (ماك) لفتح أداة اختيار العبارات
3. **اختيار العبارة**: اختر العبارة المطلوبة من القائمة المنبثقة وسيتم إدراجها تلقائيًا في حقل الإدخال الحالي
4. **حفظ عبارة بسرعة**: بعد تحديد أي نص، استخدم `Ctrl+Shift+S` (ويندوز/لينكس) أو `Command+Shift+S` (ماك) لحفظه كعبارة مقترحة
5. **حفظ من قائمة الزر الأيمن**: بعد تحديد النص، انقر بزر الفأرة الأيمن واختر "حفظ هذه العبارة" لحفظ النص المحدد كعبارة مقترحة
6. **تصدير مكتبة العبارات**: في صفحة الإدارة، انقر على زر "تصدير" لتصدير جميع العبارات كملف JSON محلي
7. **استيراد مكتبة العبارات**: في صفحة الإدارة، انقر على زر "استيراد" لاستيراد العبارات من ملف JSON محلي (يدعم الدمج أو الاستبدال)

## ⚙️ إعدادات مخصصة

1. انقر على أيقونة الإضافة، ثم انقر على زر "إدارة الكلمات المفتاحية"
2. في صفحة الإدارة يمكنك:
   - إضافة كلمات مفتاحية جديدة
   - تعديل الكلمات المفتاحية الحالية
   - حذف الكلمات المفتاحية غير المرغوب فيها
   - إضافة تصنيفات للكلمات المفتاحية لتنظيمها
   - تصدير قاعدة بيانات الكلمات المفتاحية للنسخ الاحتياطي
   - استيراد قاعدة بيانات النسخ الاحتياطي للكلمات المفتاحية السابقة

## 📦 دليل التثبيت

### التثبيت من متجر التطبيقات

الإضافة متوفرة الآن على متجر Chrome! [انقر هنا للتحميل والتثبيت](https://chromewebstore.google.com/detail/quick-prompt/hnjamiaoicaepbkhdoknhhcedjdocpkd)

### الحصول عليها من GitHub Releases

1. قم بزيارة [صفحة GitHub Releases](https://github.com/wenyuanw/quick-prompt/releases)
2. قم بتنزيل أحدث إصدار من حزمة الإضافة الجاهزة
3. فك ضغط الملف الذي تم تنزيله
4. اتبع التعليمات أدناه لتثبيت الإضافة المبنية

### البناء من المصدر

1. استنساخ المستودع
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. تثبيت الاعتمادات
   ```bash
   pnpm install
   ```
3. التطوير والبناء

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### تثبيت الامتداد الذي تم إنشاؤه

#### كروم / إيدج
1. افتح صفحة إدارة الامتدادات (`chrome://extensions` أو `edge://extensions`)
2. فعّل "وضع المطور"
3. انقر على "تحميل الامتداد غير المضغوط"
4. اختر مجلد المشروع `.output/chrome-mv3/`

#### فايرفوكس
1. افتح `about:debugging`
2. انقر على "هذا الفايرفوكس"
3. انقر على "تحميل إضافة مؤقتة"
4. اختر ملف `manifest.json` داخل مجلد `.output/firefox-mv2/` الخاص بالمشروع
