<h1 align="center">Quick Prompt</h1>

강력한 브라우저 확장 프로그램으로, 프롬프트 관리와 빠른 입력에 특화되어 있습니다. 사용자가 프롬프트 라이브러리를 생성, 관리, 구성할 수 있도록 도와주며, 어떤 웹페이지 입력란에서도 사전 설정된 프롬프트 내용을 신속하게 삽입해 업무 효율을 높입니다.

## ✨ 기능 특성

- 📚 **프롬프트 관리**: 프롬프트 라이브러리를 손쉽게 생성, 편집 및 관리할 수 있습니다
- 🚀 **빠른 입력**: 모든 웹페이지 입력란에서 `/p` 입력으로 프롬프트 선택기를 빠르게 호출
- ⌨️ 단축키 설정을 통해 프롬프트 선택기 열기 및 선택한 텍스트를 프롬프트로 저장 지원
- 📑 마우스 오른쪽 버튼 메뉴에서 선택한 텍스트를 바로 프롬프트로 저장 지원
- 🎯 프롬프트 사용자 정의 지원: 분류, 제목, 내용, 첨부파일, 태그, 변수 포함
- 🧑‍💻 **변수 지원**: 프롬프트에 `{{변수명}}` 형식의 변수를 사용할 수 있으며, 사용 시 값을 입력 가능
- 💾 **데이터 백업**: 프롬프트 라이브러리 내보내기 및 가져오기 지원, 기기 간 이전 및 백업 용이
- 🔗 **클라우드 동기화**: 프롬프트 라이브러리 WebDAV, Notion 데이터베이스, Gitee/GitHub Gist 동기화 지원
- 🔍 프롬프트 검색 및 필터링 기능 지원
- 🌙 시스템의 다크/라이트 테마 자동 적용

## 🚀 사용 방법

1. **빠른 호출**: 모든 웹페이지의 텍스트 입력란에 `/p` 입력으로 프롬프트 선택기 호출
2. **단축키로 선택기 열기**: `Ctrl+Shift+P`(Windows/Linux) 또는 `Command+Shift+P`(macOS)로 프롬프트 선택기 열기
3. **프롬프트 선택**: 팝업된 선택기에서 원하는 프롬프트 클릭 시 현재 입력란에 자동 삽입
4. **프롬프트 빠른 저장**: 텍스트 선택 후 `Ctrl+Shift+S`(Windows/Linux) 또는 `Command+Shift+S`(macOS)로 빠르게 프롬프트로 저장
5. **마우스 오른쪽 버튼 메뉴 저장**: 텍스트 선택 후 우클릭, "해당 프롬프트 저장" 선택으로 내용 저장
6. **프롬프트 라이브러리 내보내기**: 관리 페이지에서 "내보내기" 버튼 클릭 시 전체 프롬프트를 JSON 파일로 로컬 저장
7. **프롬프트 라이브러리 가져오기**: 관리 페이지에서 "가져오기" 버튼 클릭, 로컬 JSON 파일에서 프롬프트 가져오기(병합 또는 기존 프롬프트 덮어쓰기 지원)

## ⚙️ 사용자 정의 설정

1. 확장 프로그램 아이콘을 클릭한 후 "프롬프트 관리" 버튼을 클릭하세요.
2. 관리 페이지에서 다음을 할 수 있습니다:
   - 새로운 프롬프트 추가
   - 기존 프롬프트 수정
   - 불필요한 프롬프트 삭제
   - 프롬프트에 태그를 추가하여 분류
   - 프롬프트 라이브러리 내보내기(백업)
   - 이전에 백업한 프롬프트 라이브러리 가져오기

### 소스 코드에서 빌드하기

1. 저장소를 클론합니다
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. 의존성 설치
   ```bash
   pnpm install
   ```
3. 개발 및 빌드

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### 이미 빌드된 확장 프로그램 설치

#### Chrome / Edge
1. 확장 프로그램 관리 페이지를 엽니다 (`chrome://extensions` 또는 `edge://extensions`)
2. "개발자 모드"를 활성화합니다
3. "압축 해제된 확장 프로그램 로드"를 클릭합니다
4. 프로젝트의 `.output/chrome-mv3/` 디렉터리를 선택합니다

#### Firefox
1. `about:debugging`을 엽니다
2. "이 Firefox"를 클릭합니다
3. "임시 부가 기능 로드"를 클릭합니다
4. 프로젝트의 `.output/firefox-mv2/` 디렉터리 내 `manifest.json` 파일을 선택합니다
