<h1 align="center">Quick Prompt</h1>

Мощное расширение для браузера, сфокусированное на управлении подсказками и быстром вводе. Помогает пользователям создавать, управлять и организовывать библиотеку подсказок, а также быстро вставлять заранее заданные Prompts в любые поля ввода на веб-страницах, повышая эффективность работы.

## ✨ Особенности

- 📚 **Управление подсказками**: Удобное создание, редактирование и управление вашей библиотекой подсказок
- 🚀 **Быстрый ввод**: В любом поле ввода на веб-странице быстро вызывайте селектор подсказок с помощью `/p`
- ⌨️ Поддержка настройки горячих клавиш для открытия селектора и сохранения выделенного текста как подсказки
- 📑 Поддержка сохранения выделенного текста как подсказки через контекстное меню правой кнопки мыши
- 🎯 Поддержка пользовательских подсказок, включая категории, заголовки, содержимое, вложения, теги и переменные
- 🧑‍💻 **Поддержка переменных**: В подсказках можно использовать переменные в формате `{{имя_переменной}}`, при использовании можно задать конкретное значение
- 💾 **Резервное копирование данных**: Поддержка экспорта и импорта библиотеки подсказок для удобства миграции и резервного копирования между устройствами
- 🔗 **Облачная синхронизация**: Библиотека подсказок поддерживает синхронизацию через WebDAV, базу данных Notion, Gitee/GitHub Gist
- 🔍 Поддержка поиска и фильтрации подсказок
- 🌙 Автоматическая адаптация к светлой и тёмной теме системы

## 🚀 Инструкция по использованию

1. **Быстрый вызов**: В любом текстовом поле на веб-странице введите `/p`, чтобы вызвать селектор подсказок
2. **Горячие клавиши для открытия селектора**: Используйте `Ctrl+Shift+P` (Windows/Linux) или `Command+Shift+P` (macOS) для открытия селектора подсказок
3. **Выбор подсказки**: В появившемся селекторе кликните по нужной подсказке, она автоматически вставится в текущее поле ввода
4. **Быстрое сохранение подсказки**: После выделения любого текста используйте `Ctrl+Shift+S` (Windows/Linux) или `Command+Shift+S` (macOS) для быстрого сохранения как подсказки
5. **Сохранение через контекстное меню**: После выделения текста кликните правой кнопкой мыши и выберите «Сохранить эту подсказку» для добавления выделенного в подсказки
6. **Экспорт библиотеки подсказок**: На странице управления нажмите кнопку «Экспорт», чтобы сохранить все подсказки в файл JSON на локальный компьютер
7. **Импорт библиотеки подсказок**: На странице управления нажмите кнопку «Импорт» для загрузки подсказок из локального файла JSON (поддерживается объединение или замена текущих подсказок)

## ⚙️ Пользовательская настройка

1. Нажмите на значок расширения, затем кнопку "Управление промптами"
2. На странице управления вы можете:
   - Добавить новый промпт
   - Редактировать существующий промпт
   - Удалить ненужный промпт
   - Добавить метки для классификации промптов
   - Экспортировать базу промптов для резервного копирования
   - Импортировать ранее сохранённую базу промптов

### Загрузка с GitHub Releases

1. Перейдите на [страницу GitHub Releases](https://github.com/wenyuanw/quick-prompt/releases)
2. Скачайте последнюю версию готового пакета расширения
3. Распакуйте скачанный файл
4. Установите собранное расширение согласно инструкциям ниже

### Сборка из исходного кода

1. Клонируйте репозиторий
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. Установка зависимостей
   ```bash
   pnpm install
   ```
3. Разработка и сборка

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### Установка собранного расширения

#### Chrome / Edge
1. Откройте страницу управления расширениями (`chrome://extensions` или `edge://extensions`)
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите каталог `.output/chrome-mv3/` вашего проекта

#### Firefox
1. Откройте `about:debugging`
2. Нажмите "Этот Firefox"
3. Нажмите "Временно загрузить дополнение"
4. Выберите файл `manifest.json` в каталоге `.output/firefox-mv2/` вашего проекта
