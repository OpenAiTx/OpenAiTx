এটা শক্তিশালী ব্ৰাউজাৰ এক্সটেনচন, যি প্ৰম্পট শব্দ ব্যৱস্থাপনা আৰু দ্ৰুত ইনপুটত কেন্দ্ৰিত। এইটো ব্যৱহাৰকাৰীক প্ৰম্পট শব্দৰ ডাটাবেইজ সৃষ্টি, পৰিচালনা আৰু সংগঠিত কৰিবলৈ সহায় কৰে আৰু যিকোনো ৱেবপেইজৰ ইনপুট বাকচত আগতীয়াকৈ সংৰক্ষিত প্ৰম্প্ট সমল দ্ৰুতভাৱে সুমুৱাব পাৰে, যাৰ ফলত কামৰ দক্ষতা বৃদ্ধি হয়।

> ব্ৰাউজাৰ প্লাগইন কেৱল ৱেব সংস্কৰণতেই ব্যৱহাৰ কৰিব পাৰি, সেয়ে একে সুবিধাসম্পন্ন এটা [Raycast প্লাগইন](https://github.com/wenyuanw/quick-prompt-raycast) মুক্ত উৎস হিচাপে উপলব্ধ, যি ব্ৰাউজাৰ প্লাগইনৰ ডাটা ফৰ্মেটৰ লগত সদৃশ আৰু একে JSON ডাটাৰ সহায়ত নির্বিঘ্নে স্থানান্তৰ কৰিব পাৰে।

## ✨ ফিচাৰ বৈশিষ্ট্যসমূহ

- 📚 **প্ৰম্পট শব্দ ব্যৱস্থাপনা**: আপোনাৰ প্ৰম্পট শব্দবোৰ সহজে সৃষ্টি, সম্পাদনা আৰু ব্যৱস্থাপনা কৰক
- 🚀 **দ্ৰুত ইনপুট**: যিকোনো ৱেবপৃষ্ঠাৰ ইনপুট বাকচত `/p` টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী দ্ৰুতকৈ আৰম্ভ কৰক
- ⌨️ প্ৰম্পট শব্দ চয়নকাৰী খোলাৰ বাবে কিবোৰ্ড চাৰ্টকাট কনফিগাৰ আৰু নিৰ্বাচিত টেক্সট প্ৰম্পট শব্দ হিচাপে সংৰক্ষণৰ সুবিধা
- 📑 সোঁ-চিকন মেনুতেও নিৰ্বাচিত টেক্সট প্ৰম্পট শব্দ হিচাপে সংৰক্ষণৰ সুবিধা
- 🎯 কাষ্টম প্ৰম্পট শব্দৰ সুবিধা, তাৰ ভিতৰত শ্ৰেণী, শিৰোনাম, সমল, টেগ আৰু ভেৰিয়েবল
- 🧑‍💻 **ভেৰিয়েবল সমৰ্থন**: প্ৰম্পট শব্দত `{{ভেৰিয়েবল নাম}}` ফৰ্মেটৰ ভেৰিয়েবল ব্যৱহাৰ কৰিব পাৰি, ব্যৱহাৰৰ সময়ত নির্দিষ্ট মান পূৰণ কৰিব পাৰি
- 💾 **ডাটা বেকআপ**: প্ৰম্পট শব্দ ডাটাবেসৰ এক্সপোৰ্ট আৰু ইম্পোৰ্ট সমৰ্থন, ক্ৰছ ডিভাইচ স্থানান্তৰ আৰু বেকআপৰ সুবিধা
- 🔗 **ক্লাউড ছিঞ্চ**: Notion ডাটাবেস বা Gitee/GitHub Gist ৰ সৈতে প্ৰম্পট শব্দ ডাটাবেস ছিঞ্চ কৰাৰ সুবিধা
- 🔍 প্ৰম্পট শব্দৰ সন্ধান আৰু ফিল্টাৰ কৰাৰ সুবিধা
- 🌙 চিস্টেমৰ লাইট-ডাৰ্ক থিমটো অটোমেটিকেলি এডাপ্ট কৰে

## 🚀 ব্যৱহাৰ পদ্ধতি

1. **দ্ৰুত আৰম্ভ**: যিকোনো ৱেবপৃষ্ঠাৰ টেক্সট ইনপুট বাকচত `/p` টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী আৰম্ভ কৰক
2. **চাৰ্টকাটে চয়নকাৰী খোলক**: `Ctrl+Shift+P` (Windows/Linux) বা `Command+Shift+P` (macOS) ব্যৱহাৰ কৰি প্ৰম্পট শব্দ চয়নকাৰী খোলক
3. **প্ৰম্পট শব্দ চয়ন**: পপ-আপ চয়নকাৰীত পৰা প্ৰয়োজনীয় প্ৰম্পট শব্দ ক্লিক কৰক, ইয়াৰ পৰা ইনপুট বাকচত স্বয়ংক্ৰিয়ভাৱে ইনচাৰ্ট হয়
4. **দ্ৰুত সংৰক্ষণ**: যিকোনো টেক্সট চয়ন কৰি `Ctrl+Shift+S` (Windows/Linux) বা `Command+Shift+S` (macOS) ব্যৱহাৰ কৰি প্ৰম্পট শব্দ হিচাপে দ্ৰুত সংৰক্ষণ কৰক
5. **সোঁ-চিকন মেনু সংৰক্ষণ**: যিকোনো টেক্সট চয়ন কৰি সোঁ-চিকন ক্লিক কৰক আৰু "এই প্ৰম্পট শব্দ সংৰক্ষণ" চয়ন কৰক, চয়ন কৰা সমল প্ৰম্পট শব্দ হিচাপে সংৰক্ষণ হ'ব
6. **প্ৰম্পট শব্দ ডাটাবেস এক্সপোৰ্ট**: ব্যৱস্থাপনা পৃষ্ঠাত "এক্সপোৰ্ট" বুটাম ক্লিক কৰি সকলো প্ৰম্পট শব্দ JSON ফাইল হিচাপে লোকেলত সংৰক্ষণ কৰক
7. **প্ৰম্পট শব্দ ডাটাবেস ইম্পোৰ্ট**: ব্যৱস্থাপনা পৃষ্ঠাত "ইম্পোৰ্ট" বুটাম ক্লিক কৰি লোকেল JSON ফাইলৰ পৰা প্ৰম্পট শব্দ ইম্পোৰ্ট কৰক (মিলোৱা বা বৰ্তমান প্ৰম্পট শব্দ আচ্ছাদিত কৰাৰ সুবিধা)

   - বিদ্যমান প্ৰম্প্ট সম্পাদনা কৰক
   - অপ্রয়োজনীয় প্ৰম্প্ট আঁতৰাওক
   - প্ৰম্প্টক শিতানৰ বাবে টেগ যোগ কৰক
   - প্ৰম্প্ট লাইব্ৰেৰী ৰপ্তানি কৰি বেকআপ কৰক

## 📦 ইনস্টল নির্দেশনা

### এপ্লিকেশ্যন ষ্ট'ৰ পৰা ইনস্টল কৰক

এতিয়া Chrome এপ্লিকেশ্যন ষ্ট'ৰত উপলব্ধ! [ইয়াত ক্লিক কৰি ডাউনলোড আৰু ইনস্টল কৰক](https://chromewebstore.google.com/detail/quick-prompt/hnjamiaoicaepbkhdoknhhcedjdocpkd)

### GitHub Releases ৰ পৰা লাভ কৰক

1. [GitHub Releases পৃষ্ঠা](https://github.com/wenyuanw/quick-prompt/releases) ভিজিট কৰক
2. শেহতীয়া সংস্কৰণৰ নিৰ্মিত এক্সটেনচন পেকেজ ডাউনলোড কৰক
3. ডাউনলোড কৰা ফাইলটো আনজিপ কৰক
4. তলত দিয়া নিৰ্দেশনা অনুসৰি নিৰ্মিত এক্সটেনচন ইনস্টল কৰক

### সোর্সকোডৰ পৰা নিৰ্মাণ কৰক

1. ৰিপ'জিট'ৰিটো ক্লোন কৰক
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. নিৰ্ভৰশীলতা ইনষ্টল কৰক
   ```bash
   pnpm install
   ```

3. বিকাশ আৰু নিৰ্মাণ
   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### নিৰ্মিত এক্সটেনশ্যন ইনষ্টল কৰক

#### Chrome / Edge
1. এক্সটেনশ্যন ব্যৱস্থাপনা পৃষ্ঠা খুলক (`chrome://extensions` অথবা `edge://extensions`)
2. "ডেভেলপাৰ মোড" সক্ষম কৰক
3. "লোড unpacked extension" ত ক্লিক কৰক
4. প্ৰকল্পটোৰ `.output/chrome-mv3/` ডাইৰেক্টৰী নিৰ্বাচন কৰক

#### Firefox
1. `about:debugging` খুলক
2. "এই Firefox" ত ক্লিক কৰক
3. "অস্থায়ীভাৱে সংলগ্ন সংযোজন লোড কৰক" ত ক্লিক কৰক
4. প্ৰকল্পটোৰ `.output/firefox-mv2/` ডাইৰেক্টৰীৰ ভিতৰৰ `manifest.json` ফাইলটো নিৰ্বাচন কৰক
