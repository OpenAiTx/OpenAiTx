এটা শক্তিশালী ব্রাউজাৰ এক্সটেনশন, যি প্ৰম্প্ট শব্দ ব্যৱস্থাপনা আৰু দ্ৰুত ইনপুটত কেন্দ্ৰিত। এইটো ব্যৱহাৰকাৰীক প্ৰম্প্ট শব্দৰ ডাটাবেছ সৃষ্টি, ব্যৱস্থাপনা আৰু সংগঠিত কৰাত সহায় কৰে আৰু যিকোনো ৱেবপেজৰ ইনপুট ফিল্ডত দ্ৰুতভাৱে সংৰক্ষিত প্ৰম্প্ট বিষয়বস্তু সন্নিৱিষ্ট কৰে, কামৰ দক্ষতা বৃদ্ধি কৰে।

## ✨ ফিচাৰ বৈশিষ্ট্য

- 📚 **প্ৰম্পট শব্দ ব্যৱস্থাপনা**: আপোনাৰ প্ৰম্পট শব্দ সংৰক্ষণাগাৰ সহজে সৃষ্টি, সম্পাদনা আৰু পৰিচালনা কৰক
- 🚀 **দ্ৰুত ইনপুট**: যিকোনো ৱেবপৃষ্ঠাৰ ইনপুট বাক্সত `/p` টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী তৎক্ষণাৎ খুলিব
- ⌨️ প্ৰম্পট শব্দ চয়নকাৰী খুলিবলৈ আৰু নিৰ্বাচিত পাঠ্য প্ৰম্পট শব্দ হিচাপে সংৰক্ষণ কৰিবলৈ কিবোৰ্ড শ্বটকাট কনফিগাৰ কৰক
- 📑 সোঁমুৰি মেনুতে নিৰ্বাচিত পাঠ্যক প্ৰম্পট শব্দ হিচাপে সোঁমুৰি সংৰক্ষণ কৰাৰ সুবিধা
- 🎯 কাষ্টম প্ৰম্পট শব্দৰ সমৰ্থন, ইয়াত শ্ৰেণী, শিৰোনাম, বিষয়বস্তু, সংলগ্ন, টেগ আৰু ভেৰিয়েবল অন্তৰ্ভুক্ত
- 🧑‍💻 **ভেৰিয়েবল সমৰ্থন**: প্ৰম্পট শব্দত `{{ভেৰিয়েবল নাম}}` আকারৰ ভেৰিয়েবল ব্যৱহাৰ কৰিব পাৰি, ব্যৱহাৰ কৰাৰ সময়ত নির্দিষ্ট মান লিখিব পাৰি
- 💾 **ডাটা ব্যাকআপ**: প্ৰম্পট শব্দ সংৰক্ষণাগাৰ ডাটাৰ এক্সপৰ্ট আৰু ইমপৰ্ট সমৰ্থন, ডিভাইচৰ মাজত স্থানান্তৰ আৰু ব্যাকআপ সহজ
- 🔗 **ক্লাউড ছিংক**: প্ৰম্পট শব্দ সংৰক্ষণাগাৰ WebDAV ছিংক, Notion ডাটাবেছ, Gitee/GitHub Gist ছিংক সমৰ্থন
- 🔍 প্ৰম্পট শব্দৰ সন্ধান আৰু ফিল্টাৰ সমৰ্থন
- 🌙 সিষ্টেমৰ লাইট/ডাৰ্ক থীম স্বয়ংক্ৰিয়ভাৱে মানি চলা

## 🚀 ব্যৱহাৰ পদ্ধতি

1. **দ্ৰুত সূচনা**: যিকোনো ৱেবপৃষ্ঠাৰ পাঠ্য ইনপুট বাক্সত `/p` টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী খুলক
2. **শ্বটকাটে চয়নকাৰী খোলা**: `Ctrl+Shift+P` (Windows/Linux) অথবা `Command+Shift+P` (macOS) ব্যৱহাৰ কৰি চয়নকাৰী খোলা
3. **প্ৰম্পট শব্দ বাছনি**: চয়নকাৰীত প্ৰয়োজনীয় প্ৰম্পট শব্দ ক্লিক কৰক, ই স্বয়ংক্ৰিয়ভাৱে ইনপুট বাক্সত সোমাই যাব
4. **দ্ৰুত সংৰক্ষণ**: যিকোনো পাঠ্য নিৰ্বাচন কৰাৰ পিছত `Ctrl+Shift+S` (Windows/Linux) অথবা `Command+Shift+S` (macOS) ব্যৱহাৰ কৰি দ্ৰুত সংৰক্ষণ কৰক
5. **সোঁমুৰি মেনু সংৰক্ষণ**: পাঠ্য নিৰ্বাচন কৰাৰ পিছত সোঁমুৰি ক্লিক কৰি "এই প্ৰম্পট শব্দ সংৰক্ষণ কৰক" বাছি সংৰক্ষণ কৰক
6. **সংৰক্ষণাগাৰ এক্সপৰ্ট**: পৰিচালনা পৃষ্ঠাত "এক্সপৰ্ট" বুটন ক্লিক কৰি সকলো প্ৰম্পট শব্দ JSON ফাইলত লোকেলত সংৰক্ষণ কৰক
7. **সংৰক্ষণাগাৰ ইমপৰ্ট**: পৰিচালনা পৃষ্ঠাত "ইমপৰ্ট" বুটন ক্লিক কৰি লোকেল JSON ফাইলৰ পৰা প্ৰম্পট শব্দ ইমপৰ্ট কৰক (মিলোৱা বা বিদ্যমান শব্দৰ ওপৰত লিখা সমৰ্থন)

## ⚙️ ব্যক্তিগতকৃত সংহতি

1. এক্সটেনচন আইকনটোত ক্লিক কৰক, তাৰ পাছত "প্ৰম্প্ট শব্দ ব্যৱস্থাপনা" বুটামত ক্লিক কৰক
2. ব্যৱস্থাপনা পৃষ্ঠাত, আপুনি কৰিব পাৰিব:
   - নতুন প্ৰম্প্ট শব্দ যোগ কৰা
   - বৰ্তমান থকা প্ৰম্প্ট শব্দ সম্পাদনা কৰা
   - অপ্ৰয়োজনীয় প্ৰম্প্ট শব্দ মচি দিয়া
   - প্ৰম্প্ট শব্দসমূহত টেগ যোগ কৰি শ্ৰেণীবদ্ধ কৰা
   - প্ৰম্প্ট শব্দ ডাটাবেজ এক্সপোৰ্ট কৰি বেকআপ লোৱা
   - আগতে সংৰক্ষিত প্ৰম্প্ট শব্দ ডাটাবেজ ইম্পোৰ্ট কৰা

### GitHub Releases ৰ পৰা লাভ কৰক

### উৎস সংকেতৰ পৰা নিৰ্মাণ কৰা

1. ৰিপ’জিট’ৰিটো ক্ল’ন কৰক
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. নিৰ্ভৰশীলতা ইনষ্টল কৰক
   ```bash
   pnpm install
   ```

3. বিকাশ আৰু নিৰ্মাণ
   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### নিৰ্মিত এক্সটেনশ্যন ইনষ্টল কৰক

#### Chrome / Edge
1. এক্সটেনশ্যন ব্যৱস্থাপনা পৃষ্ঠা খুলক (`chrome://extensions` অথবা `edge://extensions`)
2. "ডেভেলপাৰ মোড" সক্ষম কৰক
3. "লোড unpacked extension" ত ক্লিক কৰক
4. প্ৰকল্পটোৰ `.output/chrome-mv3/` ডাইৰেক্টৰী নিৰ্বাচন কৰক

#### Firefox
1. `about:debugging` খুলক
2. "এই Firefox" ত ক্লিক কৰক
3. "অস্থায়ীভাৱে সংলগ্ন সংযোজন লোড কৰক" ত ক্লিক কৰক
4. প্ৰকল্পটোৰ `.output/firefox-mv2/` ডাইৰেক্টৰীৰ ভিতৰৰ `manifest.json` ফাইলটো নিৰ্বাচন কৰক
