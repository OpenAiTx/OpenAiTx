एक शक्तिशाली ब्राउज़र एक्सटेंशन, जो प्रॉम्प्ट मैनेजमेंट और तेज़ इनपुट पर केंद्रित है। यह उपयोगकर्ताओं को प्रॉम्प्ट लाइब्रेरी बनाने, प्रबंधित करने और व्यवस्थित करने में मदद करता है, और किसी भी वेबपेज इनपुट बॉक्स में पूर्व-सेट प्रॉम्प्ट कंटेंट को तेज़ी से डालता है, जिससे कार्यक्षमता बढ़ती है।

> चूंकि ब्राउज़र प्लगइन केवल वेब संस्करण में ही काम करता है, इसलिए एक समान कार्यक्षमता वाला [Raycast प्लगइन](https://github.com/wenyuanw/quick-prompt-raycast) ओपन सोर्स किया गया है, जो ब्राउज़र प्लगइन डेटा फॉर्मेट के अनुकूल है और समान JSON डेटा के साथ सहज माइग्रेशन को सपोर्ट करता है।

## ✨ विशेषताएँ

- 📚 **प्रॉम्प्ट मैनेजमेंट**: अपनी प्रॉम्प्ट लाइब्रेरी को आसानी से बनाएं, संपादित करें और प्रबंधित करें
- 🚀 **तेज़ इनपुट**: किसी भी वेबपेज इनपुट बॉक्स में `/p` टाइप करके प्रॉम्प्ट चयनकर्ता को तुरंत खोलें
- ⌨️ प्रॉम्प्ट चयनकर्ता खोलने के लिए व शॉर्टकट कुंजी एवं चयनित टेक्स्ट को प्रॉम्प्ट के रूप में सहेजने के लिए कीबाइंडिंग का समर्थन
- 📑 राइट-क्लिक मेनू से सीधे चयनित टेक्स्ट को प्रॉम्प्ट के रूप में सहेजने का समर्थन
- 🎯 कस्टम प्रॉम्प्ट का समर्थन, जिसमें श्रेणी, शीर्षक, सामग्री, टैग और वेरिएबल शामिल हैं
- 🧑‍💻 **वेरिएबल सपोर्ट**: प्रॉम्प्ट में `{{वेरिएबल नाम}}` प्रारूप का वेरिएबल प्रयोग करें, उपयोग करते समय मान भरें
- 💾 **डेटा बैकअप**: प्रॉम्प्ट लाइब्रेरी को निर्यात और आयात करने का समर्थन, जिससे डिवाइस पर स्थानांतरण और बैकअप आसान हो
- 🔗 **क्लाउड सिंक**: प्रॉम्प्ट लाइब्रेरी को Notion डाटाबेस या Gitee/GitHub Gist के साथ सिंक्रनाइज़ करने का समर्थन
- 🔍 प्रॉम्प्ट खोजने और फ़िल्टर करने का समर्थन
- 🌙 सिस्टम के डार्क या लाइट थीम के अनुसार स्वचालित रूप से अनुकूलित

## 🚀 उपयोग विधि

1. **त्वरित ट्रिगर**: किसी भी वेबपेज के टेक्स्ट इनपुट बॉक्स में `/p` टाइप कर प्रॉम्प्ट चयनकर्ता खोलें
2. **शॉर्टकट से चयनकर्ता खोलें**: `Ctrl+Shift+P` (Windows/Linux) या `Command+Shift+P` (macOS) का उपयोग करें
3. **प्रॉम्प्ट चुनें**: पॉप-अप चयनकर्ता से आवश्यक प्रॉम्प्ट पर क्लिक करें, यह अपने आप इनपुट बॉक्स में भर जाएगा
4. **त्वरित प्रॉम्प्ट सहेजें**: कोई भी टेक्स्ट चुनने के बाद `Ctrl+Shift+S` (Windows/Linux) या `Command+Shift+S` (macOS) से तुरंत प्रॉम्प्ट के रूप में सहेजें
5. **राइट-क्लिक मेनू से सहेजें**: टेक्स्ट चुनने के बाद राइट-क्लिक कर "प्रॉम्प्ट सहेजें" चुनें, चयनित सामग्री को प्रॉम्प्ट में बदलें
6. **प्रॉम्प्ट लाइब्रेरी निर्यात**: प्रबंधन पेज पर "निर्यात" बटन क्लिक करें, सभी प्रॉम्प्ट को JSON फ़ाइल के रूप में लोकल में सहेजें
7. **प्रॉम्प्ट लाइब्रेरी आयात**: प्रबंधन पेज पर "आयात" बटन क्लिक कर लोकल JSON फ़ाइल से प्रॉम्प्ट आयात करें (मर्ज या ओवरराइट समर्थन)

### प्रॉम्प्ट प्रबंधन पृष्ठ

## ⚙️ कस्टम कॉन्फ़िगरेशन

1. एक्सटेंशन आइकन पर क्लिक करें, फिर "प्रॉम्प्ट प्रबंधन" बटन पर क्लिक करें
2. प्रबंधन पृष्ठ पर, आप कर सकते हैं:
   - नया प्रॉम्प्ट जोड़ें
   - मौजूदा प्रॉम्प्ट संपादित करें
   - अनावश्यक प्रॉम्प्ट हटाएं
   - प्रॉम्प्ट को टैग द्वारा वर्गीकृत करें
   - बैकअप के लिए प्रॉम्प्ट लाइब्रेरी एक्सपोर्ट करें
   - पहले से बैकअप की गई प्रॉम्प्ट लाइब्रेरी इम्पोर्ट करें

## 📦 इंस्टॉलेशन गाइड

### ऐप स्टोर से इंस्टॉल करें

अब Chrome वेब स्टोर पर उपलब्ध! [इंस्टॉल करने के लिए यहां क्लिक करें](https://chromewebstore.google.com/detail/quick-prompt/hnjamiaoicaepbkhdoknhhcedjdocpkd)

### GitHub Releases से प्राप्त करें

1. [GitHub Releases पेज](https://github.com/wenyuanw/quick-prompt/releases) पर जाएं
2. नवीनतम बिल्ट प्लगइन पैकेज डाउनलोड करें
3. डाउनलोड की गई फाइल को अनज़िप करें
4. नीचे दिए गए निर्देशों के अनुसार निर्मित एक्सटेंशन इंस्टॉल करें

### सोर्स कोड से बिल्ड करें

1. रिपॉजिटरी को क्लोन करें
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. निर्भरता स्थापित करें
   ```bash
   pnpm install
   ```
3. विकास और निर्माण

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### निर्मित एक्सटेंशन को इंस्टॉल करें

#### क्रोम / एज
1. एक्सटेंशन प्रबंधन पृष्ठ खोलें (`chrome://extensions` या `edge://extensions`)
2. "डेवलपर मोड" सक्षम करें
3. "अनज़िप्ड एक्सटेंशन लोड करें" पर क्लिक करें
4. प्रोजेक्ट की `.output/chrome-mv3/` डायरेक्टरी चुनें

#### फ़ायरफ़ॉक्स
1. `about:debugging` खोलें
2. "यह फ़ायरफ़ॉक्स" पर क्लिक करें
3. "अस्थायी ऐड-ऑन लोड करें" पर क्लिक करें
4. प्रोजेक्ट की `.output/firefox-mv2/` डायरेक्टरी में `manifest.json` फ़ाइल चुनें
