<h1 align="center">क्विक प्रॉम्प्ट</h1>

एक शक्तिशाली ब्राउज़र एक्सटेंशन, जो प्रॉम्प्ट प्रबंधन और त्वरित इनपुट पर केंद्रित है। यह उपयोगकर्ताओं को प्रॉम्प्ट पुस्तकालय बनाने, प्रबंधित करने और व्यवस्थित करने में मदद करता है, और किसी भी वेबपेज इनपुट बॉक्स में पूर्वनिर्धारित प्रॉम्प्ट सामग्री को जल्दी से सम्मिलित करके कार्यक्षमता बढ़ाता है।

## ✨ फीचर विशेषताएँ

- 📚 **प्रॉम्प्ट प्रबंधन**: अपनी प्रॉम्प्ट लाइब्रेरी को आसानी से बनाएं, संपादित करें और प्रबंधित करें
- 🚀 **तेज इनपुट**: किसी भी वेबपेज इनपुट बॉक्स में `/p` टाइप करके प्रॉम्प्ट चयनकर्ता तुरंत खोलें
- ⌨️ शॉर्टकट कुंजी द्वारा प्रॉम्प्ट चयनकर्ता खोलना और चयनित टेक्स्ट को प्रॉम्प्ट के रूप में सेव करना समर्थित
- 📑 राइट-क्लिक मेनू से चयनित टेक्स्ट को सीधे प्रॉम्प्ट के रूप में सेव करना समर्थित
- 🎯 कस्टम प्रॉम्प्ट का समर्थन, जिसमें श्रेणी, शीर्षक, सामग्री, अटैचमेंट, टैग और वेरिएबल्स शामिल हैं
- 🧑‍💻 **वेरिएबल्स समर्थन**: प्रॉम्प्ट में `{{वेरिएबलनाम}}` प्रारूप के वेरिएबल्स उपयोग किए जा सकते हैं, उपयोग के समय मान दर्ज करें
- 💾 **डेटा बैकअप**: प्रॉम्प्ट लाइब्रेरी को एक्सपोर्ट और इम्पोर्ट करने का समर्थन, जिससे डिवाइस बदलने या बैकअप में आसानी होती है
- 🔗 **क्लाउड सिंक**: प्रॉम्प्ट लाइब्रेरी WebDAV, Notion डेटाबेस, Gitee/GitHub Gist सिंक को सपोर्ट करती है
- 🔍 प्रॉम्प्ट खोज और फ़िल्टर करने की सुविधा उपलब्ध
- 🌙 सिस्टम की लाइट/डार्क थीम के अनुसार अपने आप अनुकूलित

## 🚀 उपयोग विधि

1. **त्वरित ट्रिगर**: किसी भी वेबपेज के टेक्स्ट इनपुट बॉक्स में `/p` टाइप करें ताकि प्रॉम्प्ट चयनकर्ता सक्रिय हो जाए
2. **शॉर्टकट से चयनकर्ता खोलें**: `Ctrl+Shift+P` (Windows/Linux) या `Command+Shift+P` (macOS) से प्रॉम्प्ट चयनकर्ता खोलें
3. **प्रॉम्प्ट चुनें**: पॉप-अप चयनकर्ता में से आवश्यक प्रॉम्प्ट पर क्लिक करें, यह अपने आप इनपुट बॉक्स में डल जाएगा
4. **प्रॉम्प्ट को जल्दी सेव करें**: किसी भी टेक्स्ट का चयन कर `Ctrl+Shift+S` (Windows/Linux) या `Command+Shift+S` (macOS) से तुरंत प्रॉम्प्ट के रूप में सेव करें
5. **राइट-क्लिक मेनू से सेव करें**: किसी भी टेक्स्ट का चयन कर राइट-क्लिक करें और "इस प्रॉम्प्ट को सेव करें" चुनें, चयनित सामग्री प्रॉम्प्ट बन जाएगी
6. **प्रॉम्प्ट लाइब्रेरी एक्सपोर्ट करें**: प्रबंधन पेज पर "एक्सपोर्ट" बटन पर क्लिक करें, सभी प्रॉम्प्ट को JSON फाइल के रूप में लोकल सेव करें
7. **प्रॉम्प्ट लाइब्रेरी इम्पोर्ट करें**: प्रबंधन पेज पर "इम्पोर्ट" बटन पर क्लिक करें, लोकल JSON फाइल से प्रॉम्प्ट इम्पोर्ट करें (मर्ज या ओवरराइट विकल्प उपलब्ध)

## 📸 इंटरफ़ेस प्रीव्यू

Quick Prompt एक सहज और यूजर-फ्रेंडली इंटरफ़ेस प्रदान करता है, जिससे आप प्रॉम्प्ट को आसानी से प्रबंधित और उपयोग कर सकते हैं।

`/p` शॉर्टकट कमांड या शॉर्टकट कुंजी से किसी भी इनपुट बॉक्स में तुरंत प्रॉम्प्ट चयनकर्ता खोलें, जिससे आवश्यक प्रॉम्प्ट चुनना और डालना आसान हो जाए।

### प्रॉम्प्ट प्रबंधन पेज

### राइट-क्लिक मेनू सेव करें

### प्रॉम्प्ट वेरिएबल इनपुट

प्रॉम्प्ट वेरिएबल्स को सपोर्ट करता है, चयन के बाद संबंधित वेरिएबल वैल्यू दर्ज करने के लिए पॉप-अप दिखेगा।

## ⚙️ कस्टम कॉन्फ़िगरेशन

1. एक्सटेंशन आइकन पर क्लिक करें, फिर "प्रॉम्प्ट प्रबंधन" बटन पर क्लिक करें
2. प्रबंधन पृष्ठ पर आप कर सकते हैं:
   - नया प्रॉम्प्ट जोड़ें
   - मौजूदा प्रॉम्प्ट संपादित करें
   - अनावश्यक प्रॉम्प्ट हटाएं
   - प्रॉम्प्ट को टैग द्वारा वर्गीकृत करें
   - बैकअप के लिए प्रॉम्प्ट लाइब्रेरी को एक्सपोर्ट करें
   - पहले से बैकअप की गई प्रॉम्प्ट लाइब्रेरी को इंपोर्ट करें

### GitHub Releases से प्राप्त करें

1. [GitHub Releases पृष्ठ](https://github.com/wenyuanw/quick-prompt/releases) पर जाएं
2. नवीनतम संस्करण का तैयार किया गया प्लगइन पैकेज डाउनलोड करें
3. डाउनलोड की गई फाइल को अनज़िप करें
4. नीचे दिए गए निर्देशों के अनुसार निर्मित एक्सटेंशन इंस्टॉल करें

### स्रोत कोड से निर्माण

1. रिपॉजिटरी क्लोन करें
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. निर्भरता स्थापित करें
   ```bash
   pnpm install
   ```
3. विकास और निर्माण

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### निर्मित एक्सटेंशन को इंस्टॉल करें

#### क्रोम / एज
1. एक्सटेंशन प्रबंधन पृष्ठ खोलें (`chrome://extensions` या `edge://extensions`)
2. "डेवलपर मोड" सक्षम करें
3. "अनज़िप्ड एक्सटेंशन लोड करें" पर क्लिक करें
4. प्रोजेक्ट की `.output/chrome-mv3/` डायरेक्टरी चुनें

#### फ़ायरफ़ॉक्स
1. `about:debugging` खोलें
2. "यह फ़ायरफ़ॉक्स" पर क्लिक करें
3. "अस्थायी ऐड-ऑन लोड करें" पर क्लिक करें
4. प्रोजेक्ट की `.output/firefox-mv2/` डायरेक्टरी में `manifest.json` फ़ाइल चुनें
