Un'estensione browser potente, focalizzata sulla gestione dei prompt e sull'inserimento rapido. Aiuta gli utenti a creare, gestire e organizzare una libreria di prompt, e a inserire rapidamente contenuti Prompt preimpostati in qualsiasi campo di input su una pagina web, migliorando l'efficienza lavorativa.

> Poiché le estensioni del browser possono essere utilizzate solo sul web, è stato reso open source un [plugin Raycast](https://github.com/wenyuanw/quick-prompt-raycast) con le stesse funzionalità, compatibile con il formato dati dell'estensione browser e che supporta la migrazione senza interruzioni tramite gli stessi dati JSON.

## ✨ Caratteristiche principali

- 📚 **Gestione dei prompt**: crea, modifica e gestisci facilmente la tua libreria di prompt
- 🚀 **Inserimento rapido**: attiva velocemente il selettore di prompt digitando `/p` in qualsiasi campo di input su una pagina web
- ⌨️ Supporta la configurazione di scorciatoie per aprire il selettore di prompt e salvare il testo selezionato come prompt
- 📑 Supporta il salvataggio diretto del testo selezionato come prompt tramite menu contestuale
- 🎯 Supporta prompt personalizzati, inclusi categorie, titolo, contenuto, etichette e variabili
- 🧑‍💻 **Supporto variabili**: puoi utilizzare variabili nel formato `{{nome_variabile}}` nei prompt, specificando il valore desiderato all'uso
- 💾 **Backup dei dati**: consente di esportare e importare la libreria di prompt, facilitando la migrazione e il backup tra dispositivi
- 🔗 **Sincronizzazione cloud**: permette di sincronizzare la libreria di prompt con un database Notion o Gitee/GitHub Gist
- 🔍 Supporta ricerca e filtraggio dei prompt
- 🌙 Adattamento automatico al tema chiaro/scuro del sistema

## 🚀 Modalità d’uso

1. **Attivazione rapida**: digita `/p` in qualsiasi campo di testo su una pagina web per attivare il selettore di prompt
2. **Apri il selettore con scorciatoia**: usa `Ctrl+Shift+P` (Windows/Linux) o `Command+Shift+P` (macOS) per aprire il selettore di prompt
3. **Seleziona un prompt**: clicca sul prompt desiderato nel selettore a comparsa, verrà inserito automaticamente nel campo di input attivo
4. **Salva rapidamente un prompt**: seleziona qualsiasi testo e usa `Ctrl+Shift+S` (Windows/Linux) o `Command+Shift+S` (macOS) per salvarlo velocemente come prompt
5. **Salvataggio tramite menu contestuale**: seleziona del testo, clicca col tasto destro e scegli "Salva questo prompt" per salvarlo
6. **Esporta la libreria di prompt**: nella pagina di gestione, clicca su "Esporta" per salvare tutti i prompt in un file JSON sul tuo dispositivo
7. **Importa la libreria di prompt**: nella pagina di gestione, clicca su "Importa" per aggiungere prompt da un file JSON locale (supporta unione o sovrascrittura)

## ⚙️ Configurazione personalizzata

1. Clicca sull’icona dell’estensione, poi clicca sul pulsante "Gestisci Prompt"
2. Nella pagina di gestione, puoi:
   - Aggiungere nuovi prompt
   - Modificare i prompt esistenti
   - Eliminare i prompt non necessari
   - Aggiungere etichette ai prompt per classificarli
   - Esportare la libreria dei prompt per il backup
   - Importare una libreria di prompt precedentemente salvata

## 📦 Guida all’installazione

### Installazione dallo store

Ora disponibile sul Chrome Web Store! [Clicca qui per scaricare e installare](https://chromewebstore.google.com/detail/quick-prompt/hnjamiaoicaepbkhdoknhhcedjdocpkd)

### Scarica dalle GitHub Releases

1. Vai alla [pagina delle GitHub Releases](https://github.com/wenyuanw/quick-prompt/releases)
2. Scarica il pacchetto dell’ultima versione del plugin già compilata
3. Estrai il file scaricato
4. Segui le istruzioni sottostanti per installare l’estensione compilata

### Compilazione dal sorgente

1. Clona il repository
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. Installazione delle dipendenze
   ```bash
   pnpm install
   ```
3. Sviluppo e costruzione

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### Installazione dell'estensione già compilata

#### Chrome / Edge
1. Apri la pagina di gestione delle estensioni (`chrome://extensions` o `edge://extensions`)
2. Abilita la "Modalità sviluppatore"
3. Clicca su "Carica estensione non pacchettizzata"
4. Seleziona la cartella `.output/chrome-mv3/` del progetto

#### Firefox
1. Apri `about:debugging`
2. Clicca su "Questo Firefox"
3. Clicca su "Carica componente aggiuntivo temporaneo"
4. Seleziona il file `manifest.json` nella cartella `.output/firefox-mv2/` del progetto
