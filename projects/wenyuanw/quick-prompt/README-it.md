Una potente estensione per browser, focalizzata sulla gestione dei prompt e sull'inserimento rapido. Aiuta gli utenti a creare, gestire e organizzare una libreria di prompt e a inserire rapidamente contenuti preimpostati nei campi di input di qualsiasi pagina web, aumentando l'efficienza lavorativa.

## ✨ Caratteristiche principali

- 📚 **Gestione dei prompt**: crea, modifica e gestisci comodamente la tua libreria di prompt
- 🚀 **Inserimento rapido**: attiva rapidamente il selettore di prompt in qualsiasi campo di testo di una pagina web digitando `/p`
- ⌨️ Supporta la configurazione di scorciatoie da tastiera per aprire il selettore di prompt e salvare il testo selezionato come prompt
- 📑 Supporta il salvataggio diretto del testo selezionato come prompt tramite il menu contestuale con il tasto destro
- 🎯 Supporta prompt personalizzati, inclusi categoria, titolo, contenuto, allegati, etichette e variabili
- 🧑‍💻 **Supporto variabili**: nei prompt puoi utilizzare variabili nel formato `{{nome_variabile}}` e inserire valori specifici all'uso
- 💾 **Backup dati**: supporta l'esportazione e l'importazione della libreria di prompt, facilitando migrazione e backup tra dispositivi
- 🔗 **Sincronizzazione cloud**: la libreria di prompt supporta la sincronizzazione tramite WebDAV, database Notion, Gitee/GitHub Gist
- 🔍 Supporta la ricerca e il filtraggio dei prompt
- 🌙 Adattamento automatico al tema chiaro/scuro del sistema

## 🚀 Modalità d'uso

1. **Attivazione rapida**: digita `/p` in qualsiasi campo di testo di una pagina web per attivare il selettore di prompt
2. **Apri il selettore con scorciatoia**: usa `Ctrl+Shift+P` (Windows/Linux) o `Command+Shift+P` (macOS) per aprire il selettore di prompt
3. **Seleziona un prompt**: fai clic sul prompt desiderato dal selettore popup, verrà inserito automaticamente nel campo di input corrente
4. **Salvataggio rapido prompt**: seleziona qualsiasi testo, poi usa `Ctrl+Shift+S` (Windows/Linux) o `Command+Shift+S` (macOS) per salvarlo rapidamente come prompt
5. **Salvataggio dal menu contestuale**: seleziona qualsiasi testo, fai clic con il tasto destro e scegli "Salva questo prompt" per salvarlo come prompt
6. **Esporta la libreria di prompt**: nella pagina di gestione clicca sul pulsante "Esporta" per salvare tutti i prompt in un file JSON locale
7. **Importa la libreria di prompt**: nella pagina di gestione clicca sul pulsante "Importa" per importare prompt da un file JSON locale (supporta unione o sovrascrittura dei prompt esistenti)

## 📸 Anteprima interfaccia

Quick Prompt offre un'interfaccia utente intuitiva e amichevole per gestire e utilizzare facilmente i prompt.

### Selettore di prompt

![Selettore di prompt](https://github.com/user-attachments/assets/41b9897c-d701-4ff0-97f7-2f1754f570a8)

![Selettore di prompt](https://github.com/user-attachments/assets/22d9d30c-b4c3-4e34-a0a0-8ef51e2cb942)

Usa il comando rapido `/p` o la scorciatoia da tastiera per richiamare velocemente il selettore di prompt in qualsiasi campo di input, così potrai scegliere e inserire facilmente il prompt desiderato.

### Pagina di gestione dei prompt

![Gestione dei prompt](https://github.com/user-attachments/assets/371ae51e-1cee-4a66-a2a5-cca017396872)

Nella pagina di gestione puoi creare nuovi prompt, modificare quelli esistenti, aggiungere etichette e gestire le categorie. L'interfaccia è chiara e le operazioni risultano semplici e veloci.

### Salvataggio dal menu contestuale

![Salva dal menu contestuale](https://github.com/user-attachments/assets/17fc3bfd-3fa4-4b0b-ae1a-5cfd0b62be2e)

Seleziona semplicemente qualsiasi testo nella pagina web, fai clic con il tasto destro e potrai salvarlo rapidamente come prompt, migliorando l’efficienza del lavoro.

### Inserimento delle variabili del prompt

![Finestra popup variabili prompt](https://github.com/user-attachments/assets/c91c1156-983a-454d-aad0-5698b0291b9b)

I prompt supportano la configurazione delle variabili; dopo la selezione apparirà una finestra per inserire i valori delle variabili corrispondenti.

## ⚙️ Configurazione personalizzata

1. Clicca sull'icona dell'estensione, poi clicca sul pulsante "Gestisci prompt"
2. Nella pagina di gestione puoi:
   - Aggiungere nuovi prompt
   - Modificare i prompt esistenti
   - Eliminare i prompt non necessari
   - Aggiungere etichette ai prompt per classificarli
   - Esportare la libreria di prompt per il backup
   - Importare una libreria di prompt precedentemente salvata

### Compilazione dal sorgente

1. Clonare il repository
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. Installazione delle dipendenze
   ```bash
   pnpm install
   ```
3. Sviluppo e costruzione

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### Installazione dell'estensione già compilata

#### Chrome / Edge
1. Apri la pagina di gestione delle estensioni (`chrome://extensions` o `edge://extensions`)
2. Abilita la "Modalità sviluppatore"
3. Clicca su "Carica estensione non pacchettizzata"
4. Seleziona la cartella `.output/chrome-mv3/` del progetto

#### Firefox
1. Apri `about:debugging`
2. Clicca su "Questo Firefox"
3. Clicca su "Carica componente aggiuntivo temporaneo"
4. Seleziona il file `manifest.json` nella cartella `.output/firefox-mv2/` del progetto
