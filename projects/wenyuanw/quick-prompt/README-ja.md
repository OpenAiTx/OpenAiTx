強力なブラウザ拡張機能で、プロンプト管理と素早い入力に特化しています。ユーザーがプロンプトライブラリを作成・管理・整理でき、あらゆるウェブページの入力欄にプリセットのプロンプト内容をすばやく挿入し、作業効率を向上させます。

## ✨ 機能特長

- 📚 **プロンプト管理**：プロンプトライブラリの作成、編集、管理が簡単に行えます
- 🚀 **高速入力**：任意のウェブページ入力欄で `/p` を入力してプロンプトセレクターを即座に呼び出せます
- ⌨️ ショートカットキーでプロンプトセレクターの起動＆選択したテキストをプロンプトとして保存可能
- 📑 右クリックメニューから選択テキストを直接プロンプトとして保存可能
- 🎯 カスタムプロンプト対応：カテゴリ、タイトル、内容、添付ファイル、タグ、変数を設定可能
- 🧑‍💻 **変数サポート**：プロンプト内で `{{変数名}}` 形式の変数を使用でき、利用時に具体的な値を入力可能
- 💾 **データバックアップ**：プロンプトライブラリのエクスポート・インポートに対応し、デバイス間移行やバックアップが容易
- 🔗 **クラウド同期**：プロンプトライブラリは WebDAV、Notionデータベース、Gitee/GitHub Gist同期に対応
- 🔍 プロンプト検索・フィルタ機能に対応
- 🌙 システムのダーク/ライトテーマに自動適応

## 🚀 使い方

1. **クイックトリガー**：任意のウェブページのテキスト入力欄で `/p` を入力してプロンプトセレクターを起動
2. **ショートカットでセレクター起動**：`Ctrl+Shift+P`（Windows/Linux）または `Command+Shift+P`（macOS）でプロンプトセレクターを開く
3. **プロンプト選択**：ポップアップしたセレクターから必要なプロンプトをクリックすると、現在の入力欄へ自動挿入
4. **プロンプトのクイック保存**：任意テキストを選択後、`Ctrl+Shift+S`（Windows/Linux）または `Command+Shift+S`（macOS）で素早くプロンプト保存
5. **右クリックで保存**：任意テキストを選択後、右クリックメニューから「このプロンプトを保存」を選び、選択内容をプロンプト保存
6. **プロンプトライブラリのエクスポート**：管理画面で「エクスポート」ボタンをクリックし、全プロンプトをJSONファイルとしてローカル保存
7. **プロンプトライブラリのインポート**：管理画面で「インポート」ボタンをクリックし、ローカルのJSONファイルからプロンプトを読み込み（既存プロンプトとのマージまたは上書き対応）

## ⚙️ カスタム設定

1. 拡張機能アイコンをクリックし、「プロンプト管理」ボタンをクリックします
2. 管理ページでは以下が可能です：
   - 新しいプロンプトの追加
   - 既存プロンプトの編集
   - 不要なプロンプトの削除
   - プロンプトにタグを追加して分類
   - プロンプトライブラリのエクスポートによるバックアップ
   - バックアップ済みプロンプトライブラリのインポート

### GitHub Releases から入手

1. [GitHub Releases ページ](https://github.com/wenyuanw/quick-prompt/releases)にアクセスします
2. 最新バージョンのビルド済みプラグインパッケージをダウンロードします
3. ダウンロードしたファイルを解凍します
4. 以下の手順に従ってビルド済みの拡張機能をインストールします

### ソースコードからビルド

1. リポジトリをクローン
   ```bash
   git clone https://github.com/wenyuanw/quick-prompt.git
   cd quick-prompt
   ```
2. 依存関係のインストール
   ```bash
   pnpm install
   ```
3. 開発と構築

   ```bash
   # 开发模式 (Chrome)
   pnpm dev
   
   # 开发模式 (Firefox)
   pnpm dev:firefox
   
   # 构建扩展 (Chrome)
   pnpm build
   
   # 构建扩展 (Firefox)
   pnpm build:firefox
   ```

### Build済み拡張機能のインストール

#### Chrome / Edge
1. 拡張機能管理ページを開く（`chrome://extensions` または `edge://extensions`）
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. プロジェクトの `.output/chrome-mv3/` ディレクトリを選択

#### Firefox
1. `about:debugging` を開く
2. 「この Firefox」をクリック
3. 「一時的にアドオンを読み込む」をクリック
4. プロジェクトの `.output/firefox-mv2/` ディレクトリ内の `manifest.json` ファイルを選択
