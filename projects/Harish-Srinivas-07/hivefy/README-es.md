## ✨ Características

### 🎨 Experiencia Moderna

- **Interfaz y animaciones** inspiradas en Spotify
- Tematización dinámica con **Material You**
- Iconos y tipografía adaptativos usando **SpotifyMix**

### 🎧 Reproductor Potente

- Reproducción en segundo plano vía `audio_service`
- Gestos de deslizamiento y controles visuales de búsqueda
- **Transición animada de minirreproductor a reproductor completo**

### 💾 Enfoque Offline

- Descarga y almacenamiento local de música (álbumes / listas de reproducción / canciones)
- Limpieza automática de archivos inválidos
- Notificaciones de descarga en segundo plano

### 🔍 Búsqueda Inteligente y Biblioteca

- Búsqueda global unificada (canciones, álbumes, artistas, listas de reproducción)
- Biblioteca persistente potenciada por **Hive DB**
- Funciona perfectamente **sin Internet**

### ⚙️ Configuración y Personalización
- Selector de servidor (Principal / Respaldo / Espejo)
- Gestión de caché y monitor de descargas
- Cambiar tema, limpiar almacenamiento y más

## 🚀 Primeros pasos  

### Requisitos previos  

- Flutter SDK **(>=3.7.2)**  
- Configuración de Android Studio / VS Code para Flutter  
- Java JDK **17+** (para firma de apps)  
- Dispositivo / emulador con **Android 7.0+**  

### Configuración y ejecución  

```bash
git clone https://github.com/Harish-Srinivas-07/hivefy.git
cd hivefy
flutter pub get
flutter run
```

## 🧩 Tecnología Utilizada

| Categoría   | Paquetes                                                                           |
| ----------- | ---------------------------------------------------------------------------------- |
| **Estado**  | `flutter_riverpod`                                                                 |
| **Audio**   | `just_audio`, `audio_service`, `just_audio_background`                             |
| **Almacenamiento** | `hive`, `shared_preferences`, `path_provider`                                  |
| **UI/UX**   | `shimmer`, `cached_network_image`, `figma_squircle`, `flashy_tab_bar2`, `readmore` |
| **Red**     | `dio`, `http`, `html_unescape`                                                     |
| **Sistema** | `flutter_local_notifications`, `disk_space_plus`, `package_info_plus`              |
| **Utilidades** | `toastification`, `permission_handler`, `internet_connection_checker_plus`         |

## 💡 Notas para Desarrolladores

- Las **compilaciones de depuración** usan el endpoint `DebugServer` (no requiere firma)
- Las **compilaciones de lanzamiento** se optimizan automáticamente con ProGuard
- Todos los modelos de datos incluyen `toJson` y `fromJson` con normalización de desescape
- Puedes modificar la lógica del backend en `services/` y los flujos de UI en `views/`
