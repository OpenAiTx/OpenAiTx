## 🎁 為什麼選擇 CueCut？  
眾所皆知，CapCut 非常強大，並且在影片剪輯領域引領了產業潮流，但幾乎所有功能都需要付費。我們的願景是完全整合開源社群技術，打造一個完全開放、真正免費、人人可用的 AI 影片剪輯生態系統。   

- [x] 完全免費，無限制。  
- [x] 無需註冊。  
- [x] 無任何附加條件。  
- [x] 內建先進 AI 技術。
