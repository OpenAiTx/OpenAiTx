## 🎁 为什么选择 CueCut？  
众所周知，CapCut 剪映非常强大，并且已经成为视频编辑领域的行业标杆，但几乎所有功能都需要付费。我们的愿景是充分整合开源社区技术，打造一个完全开源、真正免费的 AI 视频编辑生态系统，让每个人都能使用。   

- [x] 免费，无任何限制。  
- [x] 无需注册账号。  
- [x] 没有任何附加条件。  
- [x] 内置先进 AI 技术。
