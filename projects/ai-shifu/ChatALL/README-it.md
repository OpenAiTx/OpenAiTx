1. **Aggiorna** - premi <kbd>Ctrl</kbd> + <kbd>R</kbd> o <kbd>‚åò</kbd> + <kbd>R</kbd>.

4. **Crea una nuova chat** - clicca sul pulsante `Nuova Chat` e invia di nuovo il prompt.

Se nessuno dei metodi sopra funziona, puoi provare a **resettare ChatALL**. Nota che questa operazione eliminer√† tutte le tue impostazioni e la cronologia dei messaggi.

Puoi resettare ChatALL eliminando le seguenti cartelle:

Se il problema persiste, [segnala un problema](https://github.com/ai-shifu/ChatALL/issues).
