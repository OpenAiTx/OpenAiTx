## Instalación
### HACS (Recomendado)
1. Pestaña **HACS** > Menú > **Repositorio Personalizado**
4. Pega la URL de este repositorio
5. Selecciona **Integración** en el menú desplegable
6. Instala **cupertino-icons** desde HACS
7. Reinicia Home Assistant
8. Home Assistant > Ajustes > Integraciones > Añadir > Cupertino Icons > Instalar

Los iconos deberían estar disponibles para usar en Home Assistant ahora. Si no aparecen, intenta refrescar la página, usar la navegación privada o reiniciar Home Assistant.

### Manual
1. Copia `custom_components/cupertino` en tu carpeta custom_components
7. Reinicia Home Assistant
8. Home Assistant > Ajustes > Integraciones > Añadir > Cupertino Icons > Instalar

Los iconos deberían estar disponibles para usar en Home Assistant ahora. Si no aparecen, intenta refrescar la página, usar la navegación privada o reiniciar Home Assistant.

SF Symbols 3+ admite iconos multicolor. ¡Esta funcionalidad ha sido incorporada en este repositorio para soportar iconos multicolor en Home Assistant!  

¡Simplemente añade los sufijos **#color**, **#invert**, **#color-invert** o **#fullcolor** para cambiar los colores del icono! (Crédito a [thomasloven](https://github.com/thomasloven/hass-fontawesome) por la inspiración)

Ej:
 - ios:gear-badge-xmark#color
 - ios:gear-badge-xmark#color-invert
 - ios:gear-badge-xmark#fullcolor

## Referencia de Iconos

[SF Symbols | Apple](https://developer.apple.com/sf-symbols/)  

Puedes descargar la aplicación para MacOS desde Apple para una mejor referencia.

**Nota:** los guiones bajos (\_) y los puntos (.) deben ser intercambiados por guiones (-)  

## Cambios importantes
A partir de la versión v3.0+, este repositorio ha cambiado de ser un repositorio lovelace/frontend a una integración completa. Si estás actualizando desde una versión anterior:
1. HACS > Desinstalar "cupertino-icons"
2. HACS > Menú > **Repositorio Personalizado**
3. Elimina este repositorio
4. Sigue las instrucciones de **Instalación** arriba

## Aviso Legal
Los iconos con derechos de autor no han sido exportados y no están presentes en este repositorio. No serán incluidos.
