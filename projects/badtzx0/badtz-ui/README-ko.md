# BadtzUI - 프론트엔드 개발자를 위한 현대적인 UI 라이브러리 🛠️✨

![GitHub Repo stars](https://img.shields.io/github/stars/badtzx0/badtz-ui?style=social)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

프론트엔드 개발자가 아름다운 인터페이스를 더 빠르게 제공할 수 있도록 만든 세련되고 현대적인 **React UI 라이브러리**입니다. ⚡

---

## 🚀 주요 기능

- 💅 **미리 제작된 컴포넌트** — 빠른 개발을 위한 즉시 사용 가능한 UI 요소들.
- 🎨 **커스터마이징 가능한 테마** — 브랜드에 맞게 스타일을 쉽게 조정.
- ⚡ **성능 최적화** — 가볍고 매우 빠름.
- 📦 **완전 타입 지원** — 타입 안정성을 위한 TypeScript 사용.

## 🤝 기여하기

기여를 환영합니다! 🎉

1. 저장소를 포크하세요  
2. 브랜치를 만드세요: `git checkout -b feature/your-feature`  
3. 변경 사항을 커밋하세요: `git commit -m "feat: add new feature"`  
4. 브랜치에 푸시하세요: `git push origin feature/your-feature`  
5. 풀 리퀘스트를 열어주세요  

자세한 내용은 [기여 가이드](https://raw.githubusercontent.com/badtzx0/badtz-ui/main/CONTRIBUTING.md)를 참고하세요.
