## 🎯 Qu'est-ce que MemGallery ?

**MemGallery** est une **banque de mémoire** intelligente et respectueuse de la vie privée conçue pour vous aider à stocker et à retrouver toute votre vie numérique. C’est un pipeline intelligent pour vos audios, images, textes et URL, garantissant que vous ne perdiez plus jamais une pensée fugace ou une capture d’écran importante.

Contrairement aux applications traditionnelles, MemGallery est construit sur un **mandat zéro cloud** :
- **Confidentialité Totale** : Vos données restent sur votre appareil. Pas de synchronisation cloud, pas de suivi.
- **Pas d’Inscription Requise** : Commencez à indexer votre vie immédiatement.
- **Intelligence Propulsée par l’IA** : Choisissez votre moteur — utilisez des modèles locaux, **Google Gemini**, **Groq (Q)**, ou tout fournisseur **compatible OpenAI**.

### 🧠 Intelligence Proactive  
MemGallery ne se contente pas de stocker les données ; il les comprend. Notre moteur IA proactif :
1. **Indexe Tout** : Scanne vos images, transcrit vos audios et résume vos URL.
2. **Extrait les Actions** : Identifie rendez-vous, tâches et rappels cachés dans vos souvenirs.
3. **Rappel Intelligent** : Trouvez instantanément n’importe quoi avec une recherche simple en langage naturel.

## 🤔 Pourquoi MemGallery ?  

### Le Problème  
Nous recevons chaque jour des centaines d’informations—captures d’écran, conversations, articles, rappels. Les galeries traditionnelles et les applications de notes traitent ces éléments comme des fichiers déconnectés. Retrouver cette capture d’écran ou ce lien précis de la semaine dernière est souvent compliqué.  

### La Solution  
MemGallery transforme votre appareil Android en un système de mémoire intelligent qui :  

1. **N’Oublie Jamais** : Capture automatiquement les captures d’écran et organise les liens web  
2. **Comprend le Contexte** : L’IA lit et comprend le contenu de vos images et audios  
3. **Rend Tout Recherchable** : Trouvez n’importe quoi avec un langage naturel  
4. **Extrait des Actions** : Détecte automatiquement les événements et les ajoute à votre liste de tâches  
5. **Fonctionne Hors-Ligne en Priorité** : Vos données restent sur votre appareil jusqu’à ce que vous choisissiez de les traiter

## 📥 Installation

### Option 1 : Télécharger l'APK (Recommandé)
1. Allez sur la page [Releases](../../releases)
2. Téléchargez le dernier fichier `MemGallery-v*.apk`
3. Ouvrez l’APK sur votre appareil Android

**Exigences minimales :**
- Android 7.0 (API 24) ou supérieur
- Connexion Internet pour le traitement IA
- ~50 Mo d’espace de stockage

## 🔑 Configuration

### 1. Obtenir une clé API Google AI Studio

MemGallery nécessite une **clé API Google Gemini** pour le traitement IA. C’est gratuit pour la plupart des usages personnels.

1. Visitez [Google AI Studio](https://aistudio.google.com/)
2. Connectez-vous avec votre compte Google
3. Cliquez sur **« Obtenir une clé API »** -> **« Créer une clé API »**
4. Copiez la clé

### 2. Configurer MemGallery

1. Lancez MemGallery et terminez l’introduction
2. Collez votre **clé API** lorsqu’elle est demandée
3. Accordez les autorisations nécessaires (Caméra, Microphone, Notifications)
4. (Optionnel) Activez **« Indexation automatique des captures d’écran »** dans les Paramètres

## 📱 Utilisation

### Capturer des Souvenirs
- **Captures d’écran** : Prenez une capture d’écran comme d’habitude. MemGallery la détectera et la traitera automatiquement.  
- **Favoris Web** : Dans votre navigateur, appuyez sur **Partager** -> **MemGallery** pour sauvegarder et analyser le lien.  
- **Capture Rapide** : Utilisez le bouton **+** ou appuyez longuement sur l’icône de l’application pour ajouter des types spécifiques (Audio, Texte, Photo).  

### Gestion des Tâches  

- Naviguez vers l’onglet **Tâches** (si activé dans les Paramètres).  
- Visualisez les tâches et événements détectés par l’IA à partir de vos souvenirs.  
- Utilisez la vue **Calendrier** pour voir votre emploi du temps d’un coup d’œil.  

### Personnalisation  

- Allez dans **Paramètres** -> **Apparence** pour activer le **Mode AMOLED** ou changer le **Thème de l’App**.  
- Utilisez les **Paramètres Avancés** pour fournir une consigne système personnalisée pour l’IA (ex. : « Soyez concis », « Concentrez-vous sur les détails techniques »).

## 🛠️ Stack Technologique  

- **Langage** : Kotlin  
- **Interface** : Jetpack Compose (Material 3)  
- **Architecture** : MVVM + Architecture Clean  
- **DI** : Hilt  
- **IA** : Google Gemini 2.5 Flash via Google Gen AI SDK  
- **Base de données** : Room  
- **Arrière-plan** : WorkManager & Services au premier plan  
- **Analyse Web** : Jsoup  

---  

## 🔒 Confidentialité  

- **Stockage Local** : Tous les souvenirs et tâches sont stockés localement sur votre appareil.  
- **IA Sécurisée** : Le contenu est envoyé à l’API Gemini de Google uniquement pour analyse et n’est pas utilisé pour entraîner des modèles publics (confidentialité de niveau entreprise).  
- **Clés Chiffrées** : Votre clé API est stockée de manière sécurisée via EncryptedSharedPreferences d’Android.
