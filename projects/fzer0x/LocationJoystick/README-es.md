# Location Joystick

## Español

Location Joystick es un módulo Xposed diseñado para la simulación de ubicación. Proporciona una superposición de joystick y varias configuraciones relacionadas con la ubicación para mejorar la experiencia de simulación.

## Características

- **Superposición de Joystick:** Navega en tiempo real con un joystick personalizable.
- **Simulación de Ubicación:** Simula tu ubicación GPS con precisión.
- **Simulación GNSS:** Simulación completa del estado GNSS (conteo de satélites, SVID, C/N0, etc.) para simular un comportamiento de posicionamiento realista.
- **Aleatorización:** Añade un radio de desplazamiento aleatorio a tu ubicación para un movimiento más natural.
- **Favoritos:** Guarda y administra tus ubicaciones favoritas para cambiar rápidamente.
- **Interfaz Personalizable:**
    - Ajusta la transparencia (alfa) del joystick y minimapa.
    - Velocidad de joystick configurable.
    - Guardado persistente de la posición del joystick.

## Requisitos

- **Android:** 10+ (SDK mínimo 30)
- **Root:** Requerido para el uso del Framework Xposed.
- **Framework Xposed:** Se recomienda LSPosed.  
- **[NUEVO] Funcionalidad sin Root/Xposed (Ubicación simulada)**  

## Instalación y Configuración del Alcance en LSPosed  

1. Instala el APK de Location Joystick.  
2. Abre tu administrador Xposed (por ejemplo, LSPosed).  
3. Activa el módulo **Location Joystick**.  
4. **Importante:** Configura el alcance en LSPosed:  
    - **Android-System** (debe estar activado para GNSS y simulación a nivel de sistema).  
    - **Aplicación(es) objetivo** (p. ej. Pokémon GO, Maps).  
5. Reinicia tu dispositivo (o fuerza la detención de la app objetivo y de la interfaz del sistema si usas LSPosed).  
6. Abre Location Joystick y concede los permisos requeridos (Superposición, Ubicación).  

## Uso  

1. Abre la app y establece tu ubicación inicial deseada en el mapa.  
2. Configura los ajustes de simulación (velocidad, precisión, etc.).  
3. Inicia el servicio para mostrar la superposición del joystick.  
4. Abre tu app objetivo y muévete usando el joystick.

## Funcionalidades

- **Joystick virtual flotante:** Usa un joystick personalizable para moverte en tiempo real.
- **Simulación de ubicación:** Simula con precisión tu ubicación GPS.
- **Simulación GNSS:** Simulación completa del estado GNSS (número de satélites, SVID, C/N0, etc.), útil para enfrentar mecanismos avanzados de detección de ubicación.
- **Aleatorización:** Añade un rango de desviación aleatoria a la ubicación para que la trayectoria de movimiento sea más natural.
- **Favoritos:** Guarda y administra ubicaciones frecuentes para cambiar rápidamente.
- **Interfaz personalizable:**
    - Ajusta la transparencia (Alpha) del joystick y el minimapa.
    - Configura la velocidad de movimiento del joystick.
    - Guarda automáticamente la posición del joystick.

## Requisitos del sistema

- **Android:** 10+ (SDK mínimo 30)
- **Root:** Se requiere permiso Root para usar el framework Xposed.
- **Framework Xposed:** Se recomienda usar LSPosed.
- **[Nueva función] Simulación de ubicación sin Root/Xposed**

## Instalación y configuración del ámbito de LSPosed

1. Instala el APK de Location Joystick.
2. Abre el gestor Xposed (por ejemplo LSPosed).

3. Habilitar el módulo **Location Joystick**.  
4. **Importante:** Configurar el ámbito en LSPosed:  
    - **Android-System** (debe estar marcado, para GNSS y simulación de ubicación a nivel del sistema).  
    - **Aplicación objetivo** (por ejemplo, Pokémon GO, aplicaciones de mapas, etc.).  
5. Reiniciar el dispositivo (si usa LSPosed, también puede forzar la detención de la aplicación objetivo y System UI).  
6. Abrir Location Joystick y otorgar los permisos necesarios (ventana flotante, ubicación).  

## Cómo usar  

1. Abrir la aplicación y establecer la ubicación inicial en el mapa.  
2. Configurar los parámetros de simulación (velocidad, precisión, etc.).  
3. Iniciar el servicio para mostrar el joystick flotante.  
4. Abrir la aplicación objetivo y usar el joystick para moverse.
