# NotebookLM Pro Tree

GoogleのNotebookLMを単なるリストから強力なプロジェクト管理ダッシュボードに変えるChrome拡張機能。

終わりのない平坦なリストをスクロールし続けるのはやめましょう。ソースをネストされたツリーに整理し、高度なタスク、マルチウィンドウワークフロー、Zenモードで集中して複雑なプロジェクトを管理します。

## 重要な警告 ⚠️

アンインストールするとデータが削除されます：この拡張機能はプライバシーと速度のためにローカルストレージを使用しています。拡張機能を削除・アンインストールすると、フォルダとタスクはChromeによって即座に削除されます。

解決策：再インストールや新しいコンピュータへの移行前に、必ずエクスポートボタンでバックアップのJSONファイルを保存してください。

Googleの更新と信頼性：この拡張機能はNotebookLMのウェブサイトを「読み取る」ことで動作します。Googleがウェブサイトを更新すると、この拡張機能は動作しなくなる可能性があります。

免責事項：これは私の空き時間に維持されている無料のオープンソースプロジェクトであり、Googleがコードを変更した場合に即座に修正できる保証はありません。このツールは便利な拡張として使用し、重要なデータは必ずバックアップを取ってください。

## v17.9の新機能：生産性スイート 🚀
NotebookLMをマルチウィンドウワークスペースに変革。このアップデート（v17.6〜v17.9の統合）は大幅なワークフローの改善をもたらします：

ポップアウトノート：任意のノートを浮動ウィンドウで開き、執筆中に並べて参照可能。ポップアウトは書式を保持（読み取り専用）し、ライト/ダークテーマと同期します。

スマートタスク＆クイックキャプチャ：任意のノート内でテキストを選択すると即座に「タスク作成」ボタンが表示されます。これにより、そのソースノートに直接リンクされたタスクが作成されます。

高度なタスク管理：タスクをカスタムセクション（折りたたみ可能なグループ）に整理し、クイックセレクトオプション付きの期限を追加し、リッチな説明を含められます。

優雅な安定性：Googleがコードを更新しても全体がクラッシュせず、壊れた機能を個別に無効化する新しい自己修復エンジン。

## 機能
**生産性＆ワークフロー** 📋


*   **マルチウィンドウワークフロー** — Studioパネル内の任意のノートの「ポップアウト」アイコンをクリックすると、別ウィンドウで開きます。デュアルモニター環境に最適です。

*   **統合タスクマネージャー** — インターフェースに組み込まれた完全なプロジェクト管理ツール：
    *   **セクション:** タスクをフェーズごとにグループ化（例：「リサーチ」、「ドラフト作成」、「レビュー」）。
    *   **スマートリンク:** ノートから作成されたタスクには「リンク」アイコンが付き、元のソースにフィルタリングされたビューを表示。
    *   **優先度:** タスクにフラグ（赤／黄／青）を付け、優先度や期限で並べ替え。

*   **「ゼンモード」** — UIを切り替え、AIチャットやサイドバーを非表示にして気を散らさずに読書／執筆が可能なスタジオ環境。

 
 **整理** 📂

*   **ネストフォルダ** — ソース、ノート、生成アーティファクト（オーディオ概要、FAQなど）のための深い構造を作成。

*   **ピン留め** — 重要な契約書や仕様書をリストの上部に固定。

*   **カラーコーディング** — フォルダに色を割り当てて視覚的に整理。

*   **一括選択** — マスターチェックボックスでフォルダ内またはノート全体のすべてのソースを瞬時に選択／解除。


**深層検索** 🔍 

*   **コンテンツインデックス** — タイトルだけでなくノート内の実際のテキストを検索。

    > **注意:** インデックス作成はノートを開いた時に受動的に行われます。

*   **スマート圧縮** — LZ-String圧縮を使用し、5MBまでの検索インデックスデータをローカルに保存し、パフォーマンスに影響なし。


**信頼性** 🛡️ 

*   **ローカルファーストアーキテクチャ** — ネットワーク遅延ゼロ。全データはデバイス内に保存。

*   **優雅な劣化** — ウェブサイトの更新により機能が失敗しても、拡張機能の他の部分は正常に動作し続けます。

*   **スタイル付きダイアログ** — 滑らかでテーマ化された確認ダイアログ（ブラウザの不快なポップアップはなし）。

> **注意:** 各ノートブックは独立したデータを持ちます。「プロジェクトA」で作成したタスクやフォルダが「プロジェクトB」を煩雑にしません。

## インストール
### アンパックロード（開発者モード）

1.  このリポジトリをダウンロードまたはクローンします。
2.  Chromeを開き、`chrome://extensions` にアクセスします。
3.  **開発者モード** を有効にします（右上のトグル）。
4.  **アンパックロード** をクリックします。
5.  この拡張機能のファイルが入っているフォルダを選択します。
6.  NotebookLMに移動して整理を始めましょう！

## 使い方
インストール後、NotebookLMに新しい「プロジェクトダッシュボード」インターフェースが表示されます：

*   **クイックキャプチャ：** 任意のノートでテキストをハイライト -> 浮かぶ＋ボタンをクリックしてタスクに変換。

*   **ポップアウト：** Studioパネルのノートにホバー -> 新しいウィンドウアイコンをクリック。

*   **タスク詳細：** タスクの説明アイコンをクリックしてノートやコンテキストを追加。

*   **一括操作：** フォルダのチェックボックスを使って複数のソースをAIコンテキストウィンドウに切り替え。

*   **エクスポート/インポート：** 構造をバックアップしたり新しいPCに移動するためのボタン。

## ストレージとデータガイド 💾 
**データの保存場所**

v17.9は速度とプライバシーを優先しているため、データはデバイスに保存されます：

*   **ローカルストレージ：** フォルダ、タスク、設定、検索インデックス。

*   **自動同期なし：** フォルダはコンピューター間で自動的に同期されません。

*   **データ移動：** 設定をノートパソコンに移す場合は、エクスポートボタンでJSONファイルを取得し、新しいマシンでインポートしてください。

> **警告：** Chromeの「サイトデータ」や「クッキー」をクリアすると、フォルダが失われます。定期的にバックアップをエクスポートしてください。

## プライバシーとセキュリティ



**この拡張機能はローカルファーストです：**

*   **すべてのデータはブラウザのローカルストレージに保存されます。**

*   **開発者に解析や使用データは送信されません。**

*   **オープンソース：** コードを検査して、データがマシンを離れないことを確認できます。

詳細は[PRIVACY_POLICY.md](https://raw.githubusercontent.com/benju66/Notebook-Nest/main/PRIVACY_POLICY.md)をご覧ください。

## 既知の制限事項

*   **ポップアウト編集：** ポップアウトウィンドウは現在読み取り専用です。テキスト編集はメインウィンドウで行う必要があります。

*   **下線：** ノート内の下線は持続しません（これは拡張機能ではなくGoogle NotebookLMの問題です）。

*   **シークレットモード：** 拡張機能のメモリシステムはシークレットモードで無効になります。
