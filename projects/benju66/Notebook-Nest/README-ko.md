# NotebookLM Pro 트리

구글의 NotebookLM을 단순 목록에서 강력한 프로젝트 관리 대시보드로 변환하는 크롬 확장 프로그램입니다.

끝없는 평면 목록을 스크롤하지 마세요. 소스를 중첩된 트리로 구성하고, 고급 작업, 다중 창 워크플로우, 그리고 Zen 모드로 집중하며 복잡한 프로젝트를 관리하세요.



## 중요한 경고 ⚠️

제거 시 데이터 삭제: 이 확장 프로그램은 개인 정보 보호와 속도를 위해 로컬 스토리지를 사용합니다. 확장 프로그램을 제거/삭제하면 폴더와 작업이 크롬에 의해 즉시 삭제됩니다.

해결책: 항상 내보내기 버튼을 사용하여 재설치하거나 새 컴퓨터로 이동하기 전에 백업 JSON 파일을 저장하세요.

구글 업데이트 및 신뢰성: 이 확장 프로그램은 NotebookLM 웹사이트를 "읽는" 방식으로 작동합니다. 구글이 웹사이트를 업데이트하면 이 확장 프로그램이 작동하지 않을 수 있습니다.

면책 조항: 이 도구는 제가 여가 시간에 유지하는 무료 오픈 소스 프로젝트이므로 구글이 코드를 변경하면 즉각적인 수정이 보장되지 않습니다. 이 도구를 유용한 보조 수단으로 사용하되 중요한 데이터는 반드시 백업해 두세요.

## 프로젝트 상태 및 면책 조항 🚧
읽어주세요: 저는 전문 소프트웨어 개발자가 아닌 건설 프로젝트 관리자입니다. 제 일상 워크플로우에 필요해서 이 도구를 만들었습니다.

"내 컴퓨터에서는 잘 작동": 이 업데이트(v17.9)는 개인적으로 안정적이지만, 모든 가능한 브라우저 환경에서 광범위한 테스트를 거치지 않았습니다.

실험적 기능: 다중 창 시스템과 DOM 관찰자는 복잡합니다. 제가 발견하지 못한 버그가 있을 수 있습니다.

피드백: 버그를 발견하면 GitHub에 이슈를 열어주세요!

## v17.9의 새로운 기능: 생산성 스위트 🚀
NotebookLM을 다중 창 작업 공간으로 변환했습니다. 이 업데이트(v17.6–v17.9 통합)는 주요 워크플로우 개선을 제공합니다:

팝아웃 노트: 어떤 노트든 떠다니는 창으로 열어 옆에 두고 참조하며 작성할 수 있습니다. 팝아웃은 서식을 유지하며(읽기 전용) 라이트/다크 테마와 동기화됩니다.

스마트 작업 및 빠른 캡처: 어떤 노트에서든 텍스트를 선택하면 즉시 "작업 생성" 버튼이 나타납니다. 이 버튼은 해당 소스 노트에 직접 연결된 작업을 만듭니다.

고급 작업 관리: 작업을 사용자 정의 섹션(접을 수 있는 그룹)으로 구성하고, 빠른 선택 옵션이 있는 기한 날짜를 추가하며, 풍부한 설명을 포함할 수 있습니다.

우아한 안정성: 구글이 코드를 업데이트해도 전체 확장 프로그램이 충돌하지 않고 특정 고장 기능만 개별적으로 비활성화하는 새로운 자가 치유 엔진.

## 기능
**생산성 및 워크플로우** 📋
*   **다중 창 워크플로우** — Studio 패널의 노트에서 "팝아웃" 아이콘을 클릭하면 별도의 창으로 엽니다. 듀얼 모니터 환경에 최적입니다.

*   **통합 작업 관리자** — 인터페이스에 내장된 완전한 프로젝트 관리 도구:
    *   **섹션:** 단계별로 작업 그룹화(예: "연구," "초안 작성," "검토").
    *   **스마트 링크:** 노트에서 생성된 작업에는 원본 소스로 필터링하는 "링크" 아이콘이 포함됩니다.
    *   **우선순위:** 작업에 플래그 지정(빨강/노랑/파랑)하고 우선순위 또는 마감일로 정렬합니다.

*   **"젠 모드"** — AI 채팅 및 사이드바를 숨겨 방해 없는 읽기/쓰기 스튜디오로 UI 전환.


 **조직화** 📂

*   **중첩 폴더** — 소스, 노트, 생성된 산출물(오디오 개요, FAQ 등)을 위한 깊은 구조 생성.

*   **고정 기능** — 중요한 계약서나 사양을 리스트 상단에 고정 유지.

*   **색상 코딩** — 시각적 정리를 위해 폴더에 색상 지정.

*   **대량 선택** — 마스터 체크박스로 폴더 내 모든 소스 또는 전체 노트북을 즉시 선택/해제.


**정밀 검색** 🔍 

*   **콘텐츠 인덱싱** — 제목뿐만 아니라 노트 내부의 실제 텍스트 검색.

    > **참고:** 인덱싱은 노트를 열 때 수동으로 발생합니다.

*   **스마트 압축** — LZ-String 압축을 사용해 최대 5MB 검색 인덱스 데이터를 성능 저하 없이 로컬에 저장.


**신뢰성** 🛡️ 

*   **로컬 우선 아키텍처** — 네트워크 지연 없음. 모든 데이터가 기기에 저장됨.

*   **점진적 기능 저하** — 웹사이트 업데이트로 기능이 실패해도 확장 프로그램의 나머지 부분은 계속 작동.

*   **스타일링된 대화상자** — 부드럽고 테마가 적용된 확인 대화상자(더 이상 갑작스러운 브라우저 팝업 없음).

> **참고:** 각 노트북은 독립적인 데이터를 가집니다. "프로젝트 A"에서 생성된 작업과 폴더는 "프로젝트 B"를 혼잡하게 하지 않습니다.

## 설치
### 압축 해제된 확장 프로그램 불러오기 (개발자 모드)

1.  이 저장소를 다운로드하거나 클론합니다.
2.  크롬을 열고 `chrome://extensions`로 이동합니다.
3.  **개발자 모드**를 활성화합니다 (오른쪽 상단 토글).
4.  **압축 해제된 확장 프로그램 불러오기**를 클릭합니다.
5.  이 확장 프로그램 파일이 들어있는 폴더를 선택합니다.
6.  NotebookLM으로 이동하여 정리를 시작하세요!

## 사용법
설치가 완료되면 NotebookLM에 새 "프로젝트 대시보드" 인터페이스가 주입됩니다:

*   **빠른 캡처:** 어떤 노트에서든 텍스트를 강조 표시 -> 떠 있는 + 버튼을 클릭하여 작업으로 전환.

*   **팝아웃:** Studio 패널에서 노트 위에 마우스를 올린 후 새 창 아이콘 클릭.

*   **작업 세부사항:** 작업의 설명 아이콘을 클릭하여 메모나 컨텍스트 추가.

*   **대량 작업:** 폴더 체크박스를 사용하여 AI 컨텍스트 창에 여러 소스 토글.

*   **내보내기/가져오기:** 구조를 백업하거나 새 컴퓨터로 옮길 때 이 버튼을 사용.

## 저장소 및 데이터 안내 💾 
**데이터가 저장되는 위치**

v17.9은 속도와 개인 정보 보호를 우선시하므로 데이터는 기기에 저장됩니다:

*   **로컬 저장소:** 폴더, 작업, 설정, 검색 인덱스.

*   **자동 동기화 없음:** 폴더는 컴퓨터 간 자동 동기화되지 않음.

*   **데이터 이동:** 설정을 노트북으로 옮기려면 내보내기 버튼을 눌러 JSON 파일을 얻고, 새 기기에서 가져오기 사용.

> **경고:** 크롬의 "사이트 데이터" 또는 "쿠키"를 삭제하면 폴더가 사라집니다. 백업을 정기적으로 내보내세요.

## 알려진 제한 사항

*   **팝아웃 편집:** 팝아웃 창은 현재 읽기 전용입니다. 텍스트 편집은 메인 창에서 해야 합니다.

*   **밑줄 표시:** 노트 내 밑줄은 유지되지 않습니다(이는 확장 프로그램이 아닌 Google NotebookLM의 문제입니다).

*   **시크릿 모드:** 확장 프로그램의 메모리 시스템은 시크릿 모드에서 비활성화됩니다.
