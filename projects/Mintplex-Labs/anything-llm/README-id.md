### Gambaran Produk

AnythingLLM adalah aplikasi full-stack di mana Anda dapat menggunakan LLM komersial siap pakai atau LLM open source populer serta solusi vectorDB untuk membangun ChatGPT privat tanpa kompromi yang dapat dijalankan secara lokal maupun di-host secara remote dan dapat chat secara cerdas dengan dokumen apa pun yang Anda sediakan.

AnythingLLM membagi dokumen Anda ke dalam objek yang disebut `workspaces`. Workspace berfungsi seperti thread, namun dengan tambahan pengelompokan dokumen Anda. Workspace dapat berbagi dokumen, tapi tidak saling terhubung sehingga Anda dapat menjaga konteks tetap bersih di setiap workspace.

### Gambaran Teknis

Monorepo ini terdiri dari enam bagian utama:

- `frontend`: Frontend viteJS + React yang bisa Anda jalankan untuk membuat dan mengelola semua konten yang dapat digunakan LLM.
- `server`: Server NodeJS express untuk menangani semua interaksi dan mengelola vectorDB serta interaksi LLM.
- `collector`: Server NodeJS express yang memproses dan mengurai dokumen dari UI.
- `docker`: Instruksi Docker dan proses build + informasi untuk membangun dari source.
- `embed`: Submodul untuk generasi & pembuatan [widget embed web](https://github.com/Mintplex-Labs/anythingllm-embed).
- `browser-extension`: Submodul untuk [ekstensi browser chrome](https://github.com/Mintplex-Labs/anythingllm-extension).

## Cara setup untuk pengembangan

- `yarn setup` Untuk mengisi file `.env` yang diperlukan di setiap bagian aplikasi (dari root repo).
  - Isi file tersebut sebelum melanjutkan. Pastikan `server/.env.development` sudah terisi atau aplikasi tidak akan berjalan dengan benar.
- `yarn dev:server` Untuk menjalankan server secara lokal (dari root repo).
- `yarn dev:frontend` Untuk menjalankan frontend secara lokal (dari root repo).
- `yarn dev:collector` Untuk menjalankan document collector (dari root repo).

## Telemetri & Privasi

AnythingLLM oleh Mintplex Labs Inc memiliki fitur telemetri yang mengumpulkan informasi penggunaan anonim.

<details>
<summary><kbd>Selengkapnya tentang Telemetri & Privasi AnythingLLM</kbd></summary>

### Mengapa?

Kami menggunakan informasi ini untuk membantu memahami bagaimana AnythingLLM digunakan, menentukan prioritas pengembangan fitur dan perbaikan bug, serta meningkatkan performa dan stabilitas AnythingLLM.

### Cara keluar
Setel `DISABLE_TELEMETRY` di pengaturan .env server atau docker Anda menjadi "true" untuk menonaktifkan telemetri. Anda juga dapat melakukannya di dalam aplikasi dengan masuk ke sidebar > `Privacy` dan menonaktifkan telemetri.

### Apa saja yang secara eksplisit kami lacak?

Kami hanya akan melacak detail penggunaan yang membantu kami membuat keputusan produk dan roadmap, secara spesifik:

- Tipe instalasi Anda (Docker atau Desktop)

- Ketika sebuah dokumen ditambahkan atau dihapus. Tidak ada informasi _tentang_ dokumen tersebut. Hanya bahwa peristiwa itu terjadi. Ini memberi kami gambaran penggunaan.

- Tipe database vektor yang digunakan. Ini membantu kami memprioritaskan perubahan saat pembaruan tiba untuk penyedia tersebut.

- Tipe penyedia LLM & tag model yang digunakan. Ini membantu kami memprioritaskan perubahan saat pembaruan tiba untuk penyedia atau model tersebut, atau kombinasinya. Contoh: reasoning vs regular, model multi-modal, dll.

- Ketika sebuah chat dikirim. Ini adalah "peristiwa" yang paling rutin dan memberi kami gambaran tentang aktivitas harian proyek ini di semua instalasi. Sekali lagi, hanya **peristiwa** yang dikirim - kami tidak memiliki informasi mengenai sifat atau isi chat itu sendiri.

Anda dapat memverifikasi klaim ini dengan mencari semua lokasi di mana `Telemetry.sendTelemetry` dipanggil. Selain itu, peristiwa ini juga ditulis ke log output sehingga Anda dapat melihat data spesifik yang dikirim - jika diaktifkan. **Tidak ada IP atau informasi identifikasi lain yang dikumpulkan**. Penyedia Telemetri adalah [PostHog](https://posthog.com/) - layanan pengumpulan telemetri open-source.

Kami sangat serius dalam menjaga privasi, dan kami berharap Anda memahami bahwa kami ingin belajar bagaimana alat kami digunakan, tanpa menggunakan survei pop-up yang mengganggu, sehingga kami dapat membangun sesuatu yang layak digunakan. Data anonim _tidak pernah_ dibagikan kepada pihak ketiga, kapan pun.

[Lihat semua peristiwa telemetri di source code](https://github.com/search?q=repo%3AMintplex-Labs%2Fanything-llm%20.sendTelemetry\(&type=code)

</details>
