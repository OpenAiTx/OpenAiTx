### 製品概要

AnythingLLMは、商用の市販LLMや人気のオープンソースLLM、ベクターデータベースソリューションを利用して、妥協のないプライベートなChatGPTをローカルで動かしたり、リモートでホストしたりできるフルスタックアプリケーションです。あなたが提供する任意のドキュメントとインテリジェントにチャットできます。

AnythingLLMは、あなたのドキュメントを「ワークスペース」と呼ばれるオブジェクトに分割します。ワークスペースはスレッドによく似ていますが、ドキュメントのコンテナ化という特徴が加わっています。ワークスペース間でドキュメントを共有することはできますが、ワークスペース同士は通信しないため、それぞれのワークスペースのコンテキストをクリーンに保つことができます。

### 技術概要

このモノレポジトリは、主に6つのセクションで構成されています：

- `frontend`: ViteJS + React製フロントエンド。LLMが利用できるすべてのコンテンツの作成・管理が容易に行えます。
- `server`: NodeJS Expressサーバー。すべてのやり取りやベクターデータベース管理、LLMとの連携を行います。
- `collector`: NodeJS Expressサーバー。UIからドキュメントを処理・解析します。
- `docker`: Docker用の手順・ビルドプロセス、およびソースからのビルド情報。
- `embed`: [ウェブ埋め込みウィジェット](https://github.com/Mintplex-Labs/anything-llm-embed)の生成・作成用サブモジュール。
- `browser-extension`: [Chromeブラウザ拡張](https://github.com/Mintplex-Labs/anything-llm-extension)用サブモジュール。

## 開発用セットアップ方法

- `yarn setup` 必要な`.env`ファイルを各アプリケーションセクション用に作成（リポジトリのルートから実行）。
  - 先にこれらのファイルを入力してください。`server/.env.development`が入力されていないと正しく動作しません。
- `yarn dev:server` サーバーをローカル起動（リポジトリのルートから）。
- `yarn dev:frontend` フロントエンドをローカル起動（リポジトリのルートから）。
- `yarn dev:collector` ドキュメントコレクターを起動（リポジトリのルートから）。

## テレメトリーとプライバシー

AnythingLLM（Mintplex Labs Inc製）には匿名の利用情報を収集するテレメトリー機能があります。

<details>
<summary><kbd>AnythingLLMのテレメトリーとプライバシーについて詳細</kbd></summary>

### なぜ？

この情報は、AnythingLLMがどのように利用されているかを理解し、新機能やバグ修正の優先順位付け、パフォーマンスや安定性の向上のために活用されます。

### オプトアウト方法
サーバーまたは Docker の .env 設定で `DISABLE_TELEMETRY` を "true" に設定すると、テレメトリーをオプトアウトできます。アプリ内でもサイドバー > `プライバシー` からテレメトリーを無効化できます。

### 明示的に追跡する内容は？

私たちは、プロダクトやロードマップの決定に役立つ使用状況の詳細のみを追跡します。具体的には以下の通りです。

- インストールの種類（Docker または Desktop）

- ドキュメントが追加または削除されたとき。そのドキュメント _自体に関する_ 情報は一切ありません。イベントが発生したという事実のみです。これにより利用状況を把握できます。

- 使用中のベクターデータベースの種類。これにより、プロバイダーのアップデート時の優先対応が判断できます。

- 使用中の LLM プロバイダーおよびモデルタグの種類。これにより、プロバイダーまたはモデル、またはその組み合わせ（例：推論型 vs 通常型、マルチモーダルモデルなど）のアップデート時に優先対応が判断できます。

- チャットが送信されたとき。これは最も頻繁に発生する「イベント」であり、全インストールにおける本プロジェクトの日次アクティビティの把握に役立ちます。繰り返しますが、**イベント** のみが送信され、チャット自体の性質や内容に関する情報は一切ありません。

これらの主張は、`Telemetry.sendTelemetry` が呼ばれている全箇所を探すことで検証できます。さらに、これらのイベントは出力ログにも記録されるため、送信された具体的なデータも確認できます（有効な場合）。**IP やその他の識別情報は一切収集されません**。テレメトリープロバイダーは [PostHog](https://posthog.com/) です。これはオープンソースのテレメトリー収集サービスです。

私たちはプライバシーを非常に重視しています。面倒なポップアップアンケートを使わず、ツールの利用状況を把握し、より良いものを作りたいと考えています。匿名データが第三者と共有されることは _絶対に_ ありません。

[ソースコード内の全テレメトリーイベントを見る](https://github.com/search?q=repo%3AMintplex-Labs%2Fanything-llm%20.sendTelemetry\(&type=code)

</details>
