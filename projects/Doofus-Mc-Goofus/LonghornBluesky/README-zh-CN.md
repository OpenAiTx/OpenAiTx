## 即将推出的功能（无特定顺序）
- 视频播放
- 适当的高 DPI 支持（无错误，仅图标质量较低）
- 双因素认证支持
- 附加替代文本
- 个人资料编辑
- 可更改的设置
- 通知、聊天、信息流和列表
- 完成“探索”功能
- 举报帖子和用户
- 账户创建和删除
- Tenor GIF 支持
- 以访客身份查看
- 完成帮助与支持
- 客户端主题
- 个人资料主题？
- ……以及剩余的 Bluesky 功能

## 兼容性
✔ Windows 11 / Server 2025  
✔ Windows 10 / Server (2016/2019/2022)  
✔ Windows 8.x / Server 2012  
✔ Windows 7 / Server 2008 R2  
🟡* Windows Vista / Windows Server 2008 (含 SP2)（未测试）（.NET 4.7.2 通常不支持 Windows Vista，但我相信可能有办法安装。）  
❓ Windows 测试版 / 内部预览版（可能可用，但不保证兼容性）  
❌ Windows Longhorn（预重置版）（抱歉）  
❌ Windows XP 及更早版本
