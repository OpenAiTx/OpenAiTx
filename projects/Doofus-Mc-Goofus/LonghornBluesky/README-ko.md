## 예정 기능 (순서 없음)
- 비디오 재생  
- 적절한 고해상도 DPI 지원 (버그 없음, 단 아이콘 해상도 낮음)  
- 2단계 인증 지원  
- 대체 텍스트 첨부  
- 프로필 편집  
- 설정 변경 가능  
- 알림, 채팅, 피드, 리스트  
- 탐색 기능 완성  
- 게시물 및 사용자 신고  
- 계정 생성 및 삭제  
- 테너 GIF 지원  
- 게스트로 보기  
- 도움말 및 지원 완성  
- 클라이언트 테마  
- 프로필 테마?  
- ...그리고 나머지 블루스카이 기능들
