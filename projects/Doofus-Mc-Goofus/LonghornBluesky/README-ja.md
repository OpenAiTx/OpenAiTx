## 今後の機能（順不同）
- 動画再生
- 適切な高DPI対応（バグなし、ただしアイコンの画質が低い）
- 2段階認証サポート
- 代替テキストの添付
- プロフィール編集
- 設定の変更可能化
- 通知、チャット、フィード、リスト
- Exploreの完成
- 投稿とユーザーの報告機能
- アカウント作成と削除
- Tenor GIF対応
- ゲスト閲覧
- ヘルプとサポートの完成
- クライアントテーマ
- プロフィールテーマ？
- …その他残りのBluesky機能

## 対応環境
✔ Windows 11 / Server 2025  
✔ Windows 10 / Server (2016/2019/2022)  
✔ Windows 8.x / Server 2012  
✔ Windows 7 / Server 2008 R2  
🟡* Windows Vista / Windows Server 2008 (SP2適用済み)（未検証）（通常Windows Vistaでは.NET 4.7.2はサポートされていませんが、インストール方法があるかもしれません）  
❓ Windowsベータ版 / インサイダービルド（動作するかもしれませんが互換性は保証されません）  
❌ Windows Longhorn（リセット前）（申し訳ありません）  
❌ Windows XP以前
