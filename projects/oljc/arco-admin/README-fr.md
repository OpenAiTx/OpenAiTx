## Caractéristiques
+ ⚡️ Toujours à jour avec les dernières versions de Vue et Vite.
+ 👍🏻 [Vue](https://cn.vuejs.org) + le [Vue Router](https://router.vuejs.org/zh) officiel recommandé + gestionnaire [Pinia](https://pinia.vuejs.org/zh).
+ 🔩 Bibliothèque de composants d'écosystème de design d'entreprise [ArcoDesign](https://arco.design/) développée par ByteDance.
+ 🎨 Préprocesseur CSS [Less](https://less.bootcss.com/).
+ 📚 Utilisation de [i18n](https://github.com/intlify/vue-i18n-next/tree/master/packages/vue-i18n#readme) + [i18n-ally](https://github.com/lokalise/i18n-ally) pour une solution d’internationalisation automatisée plus simple et standardisée.
+ ⌨️ Intégration de nombreuses pages de scénarios métier, composants encapsulés, support du cadre pour une personnalisation facile.
+ 🧚🏻 Le projet utilise entièrement la syntaxe modulaire standard `ESM`, suivant la grande tendance unifiée du développement front-end moderne.
+ 📦 Optimisation continue des configurations pour suivre les mises à jour, suppressions, et fusions de fonctionnalités des `dépendances` et de `Vite` afin d’obtenir une optimisation de bundling optimale.

## Utilisation
1. Cloner ce dépôt
```bash
git clone https://github.com/oljc/arco-admin
```
2. Installation des dépendances
```bash
pnpm i
```
3. 运行
```bash
pnpm run dev
```
