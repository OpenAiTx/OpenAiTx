## 기능

### **자동 복구**

다음 작업을 포함하여 자동으로 시스템 정리 및 복구를 수행합니다:

- 시스템 복원 지점 생성
- 악성 시스템 정책 삭제
- 레지스트리 및 시스템 최적화 실행
- 취약한 Windows 기능 제거
- 최적의 Windows Defender 설정 적용
- 찾기 어려운 임시 파일 정리
- 사전 설치된 블로트웨어 제거
- 디스크 문제 및 손상된 Windows 파일 복구
- 구버전 설치 프로그램 업데이트

복구의 모든 기능은 완전히 사용자 지정할 수 있으며, 패널의 체크박스 또는 **config** 폴더의 설정 파일을 이용해
원하는 복구만 활성화하거나 비활성화할 수 있습니다. **이 파일들은 충분히 숙지한 경우에만 수정하십시오.**

> **참고:** 이전에 제공되던 **악성코드 제거** 옵션은 이제 삭제되었습니다. **Windows Defender 최적화 실행**이
> 이제는 최적의 Windows Defender 설정을 적용합니다. 수동 검사 옵션은 유용한 프로그램 패널의 Emsisoft 또는 Sophos
> 검사 옵션을 통해 사용할 수 있습니다.

### **유용한 프로그램**

시스템 유지 관리를 위한 필수 소프트웨어 도구에 접근하세요:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: 시스템 하드웨어 정보를 확인합니다.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: 하드웨어 온도와 사양을 모니터링합니다.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Emsisoft Emergency Kit로 악성코어를 검사합니다.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Sophos Scan & Clean으로 악성코드를 검사합니다.
- **[TreeSize](https://jam-software.com/treesize_free)**: 디스크 용량 분석 및 관리 도구입니다.
- **[Everything](https://voidtools.com)**: 초고속 파일 검색 엔진입니다.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: 디스크 드라이브의 상태와 건강을 모니터링합니다.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: 디스크 드라이브의 읽기/쓰기 속도를
  벤치마크합니다.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: 윈도우 시작 관리자 대체 도구입니다.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: 윈도우 작업 관리자
  대체 도구입니다.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: 시스템 활동과 프로세스를
  모니터링합니다.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: 윈도우 BSOD(블루스크린) 덤프 분석 도구입니다.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: 경량화된 NVIDIA 그래픽 드라이버
  업데이트 도구입니다.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: 디스플레이
  드라이버 및 패키지 제거 도구입니다.
- **[Recuva](https://ccleaner.com/recuva)**: 삭제된 파일 복구용 데이터 복구 프로그램입니다.
- **[DiskGenius](https://diskgenius.com)**: 디스크 파티션 관리와 데이터 복구 도구입니다.

## 다운로드 및 실행

RepairKit을 다운로드하고 실행하는 방법을 모르겠다면 다음 단계를 따라주세요:

1. [Releases 페이지](https://github.com/Foulest/RepairKit/releases/latest)에서
   최신 버전의 RepairKit을 다운로드하세요 (`RepairKit-X.X.X.zip` 파일을 클릭).
2. ZIP 파일을 컴퓨터의 폴더에 압축 해제하세요. [7-Zip](https://7-zip.org) 같은 프로그램을 사용해
   내용을 압축 해제할 수 있습니다.
3. `RepairKit-X.X.X.exe` 파일을 더블 클릭하여 프로그램을 실행하세요.
4. Windows Defender SmartScreen이 앱을 차단하면 `추가 정보`를 클릭한 후 `어쨌든 실행`을 선택하세요.
5. 사용자 계정 컨트롤 프롬프트에서 `예`를 클릭하세요.

이제 RepairKit이 열리고 기능을 사용할 수 있습니다.

## 컴파일 방법

1. 저장소를 클론하세요.
2. 저장소 디렉터리에서 명령 프롬프트/터미널을 엽니다.
3. Windows에서는 `gradlew createExe`, macOS 또는 Linux에서는 `./gradlew createExe`를 실행하세요.
4. 빌드된 `RepairKit-X.X.X.zip` 파일은 `build` 폴더에 있습니다.
