## 기능

### **자동 수리**

포괄적인 시스템 정리 및 수리를 자동으로 수행하며, 다음을 포함합니다:

- 시스템 복원 지점 생성
- 악성 시스템 정책 삭제
- 레지스트리 및 시스템 조정 실행
- 취약한 윈도우 기능 제거
- 최적의 윈도우 디펜더 설정 적용
- 찾기 힘든 정크 파일 정리
- 사전 설치된 불필요한 프로그램 제거
- 디스크 문제 및 손상된 윈도우 파일 수리
- 구식 설치 프로그램 업데이트
수리의 모든 기능은 완전히 사용자 정의가 가능하며, 패널의 체크박스나 **config** 폴더에 위치한 구성 파일을 사용하여 필요한 특정 수리를 활성화하거나 비활성화할 수 있습니다. **이 파일들은 자신이 무엇을 하는지 알고 있을 때만 수정하세요.**

> **참고:** 이전에 제공되던 **악성코드 제거(Remove Malware)** 옵션은 이제 제거되었습니다. 대신 **Windows Defender 조정 실행(Run Windows Defender Tweaks)** 이 최적의 Windows Defender 설정을 적용합니다. 수동 검사 옵션은 유용한 프로그램 패널 내의 Emsisoft 또는 Sophos 검사 옵션을 통해 사용할 수 있습니다.

### **유용한 프로그램**

시스템 유지 관리를 위한 필수 소프트웨어 도구에 접근할 수 있습니다:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: 시스템 하드웨어 세부 정보 확인.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: 하드웨어 온도 및 사양 모니터링.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Emsisoft 긴급 키트를 사용한 악성코드 검사.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Sophos 스캔 및 클린을 이용한 악성코드 검사.
- **[TreeSize](https://jam-software.com/treesize_free)**: 디스크 내용 분석 및 관리.
- **[Everything](https://voidtools.com)**: 초고속 파일 검색 엔진.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: 디스크 드라이브의 상태 및 건강 모니터링.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: 디스크 드라이브의 읽기/쓰기 속도 벤치마크.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: Windows 시작 관리자 대체 도구.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: Windows 작업 관리자 대체 도구.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: 시스템 활동 및 프로세스 모니터링.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: Windows 블루스크린(BSOD) 크래시 덤프 보기 및 분석.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: 경량 NVIDIA 그래픽 카드 드라이버 업데이트 도구.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: 디스플레이 드라이버 및 패키지 제거 도구.
- **[Recuva](https://ccleaner.com/recuva)**: 삭제된 파일 복구 도구.
- **[DiskGenius](https://diskgenius.com)**: 디스크 파티션 관리자 및 데이터 복구 도구.

## 다운로드 및 실행

RepairKit을 어떻게 다운로드하고 실행하는지 모르시겠나요? 다음 단계를 따라주세요:

1. 최신 버전의 RepairKit을 [릴리즈 페이지](https://github.com/Foulest/RepairKit/releases/latest)에서 다운로드하세요 (`RepairKit-X.X.X.zip` 파일을 클릭).
2. ZIP 파일을 컴퓨터의 폴더에 압축 해제하세요. [7-Zip](https://7-zip.org) 같은 프로그램을 사용해 압축을 풀 수 있습니다.
3. `RepairKit-X.X.X.exe` 파일을 더블 클릭하여 프로그램을 실행하세요.
4. Windows Defender SmartScreen이 앱을 차단하면 `자세히`를 클릭한 후 `그래도 실행`을 선택하세요.
5. 사용자 계정 컨트롤 프롬프트에서 `예`를 클릭하세요.

이제 RepairKit이 열리며, 기능을 사용하기 시작할 수 있습니다.



## 컴파일

1. 저장소를 클론합니다.  
2. 저장소 디렉토리에서 명령 프롬프트/터미널을 엽니다.  
3. Windows에서는 `gradlew createExe`를, macOS 또는 Linux에서는 `./gradlew createExe`를 실행합니다.  
4. 생성된 `RepairKit-X.X.X.zip` 파일은 `build` 폴더에 있습니다.
