### **স্বয়ংক্ৰিয় মেরামতি**

স্বয়ংক্ৰিয়ভাৱে এটা বিস্তৃত চিষ্টেম পৰিষ্কাৰ আৰু মেরামতি সম্পাদন কৰে, অন্তর্ভুক্ত:

- এখন চিষ্টেম Restore Point সৃষ্টি কৰে
- ক্ষতিকাৰক চিষ্টেম নীতি বিলোপ কৰে
- ৰেজিস্ট্ৰি আৰু চিষ্টেম টুইক চলায়
- দুৰ্বল Windows বৈশিষ্ট্যসমূহ আঁতৰায়
- Windows Defender-ৰ উৎকৃষ্ট ছেটিংসমূহ প্ৰয়োগ কৰে
- সন্ধান কৰিবলৈ অসুবিধাজনক জাংক ফাইলসমূহ পৰিষ্কাৰ কৰে
- আগত-সংস্থাপন কৰা বেলাবেয়াৰ আঁতৰায়
- ডিস্ক সমস্যাবোৰ আৰু বিকৃত Windows ফাইলসমূহ মেরামতি কৰে
- পুৰণি সংস্থাপন কৰা প্ৰোগ্ৰামসমূহ আপডেট কৰে

মেরামতিৰ প্ৰতিটো ফাংশন সম্পূৰ্ণ ৰূপে কাষ্টমাইজ কৰিব পৰা যায়, আপোনাৰ প্ৰয়োজন অনুসৰি বিশেষ মেরামতি সক্ৰিয় বা নিষ্ক্ৰিয় কৰিবলৈ
পেনেলত থকা চেকবক্স অথবা **config** ফোল্ডাৰত থকা কনফিগাৰেচন ফাইলসমূহ ব্যৱহাৰ কৰি। **এই ফাইলসমূহ কেৱল আপুনি
জানি থাকিলে সলনি কৰক।**

> **টোকা:** পূৰ্বৰ উপলব্ধ **Remove Malware** বিকল্প এতিয়া আঁতৰ কৰা হৈছে। **Run Windows Defender Tweaks** এতিয়া
> উৎকৃষ্ট Windows Defender ছেটিংসমূহ প্ৰয়োগ কৰে। ম্যানুয়েল স্কেনিং বিকল্পসমূহ Useful Programs পেনেলত Emsisoft অথবা Sophos

## Download আৰু Run

RepairKit কেনেকৈ ডাউনলোড আৰু চলাব লাগে জানো নে? এই পদক্ষেপসমূহ অনুসৰণ কৰক:

1. [Releases page](https://github.com/Foulest/RepairKit/releases/latest) ৰ পৰা RepairKit ৰ নতুন সংস্কৰণ ডাউনলোড কৰক ( `RepairKit-X.X.X.zip` ফাইলত ক্লিক কৰক)।
2. ZIP ফাইলটো আপোনাৰ কম্পিউটাৰৰ এখন ফোল্ডাৰত Extract কৰক। [7-Zip](https://7-zip.org) যেনে এটা প্ৰোগ্ৰাম ব্যৱহাৰ কৰি
   বিষয়বস্তু Extract কৰিব পাৰে।
3. `RepairKit-X.X.X.exe` ফাইলটোত ডাবল-ক্লিক কৰি প্ৰোগ্ৰামটো চলাওক।
4. যদি Windows Defender SmartScreen এ এপ্পটো ব্লক কৰে, `More info` ত ক্লিক কৰক আৰু তাৰপিছত `Run anyway` ত ক্লিক কৰক।
5. User Account Control prompt ত `Yes` ত ক্লিক কৰক।

RepairKit এতিয়া খুলিব, আৰু আপুনি ইয়াৰ সুবিধাসমূহ ব্যৱহাৰ আৰম্ভ কৰিব পাৰে।

## Compiling

1. ৰিপজিটৰীটো ক্লোন কৰক।
2. ৰিপজিটৰীৰ ডাইৰেক্টৰীত কমাণ্ড প্ৰম্পট/টার্মিনাল খুলক।
3. Windows ত `gradlew createExe` অথবা macOS বা Linux ত `./gradlew createExe` চলাওক।
4. তৈয়াৰ হোৱা `RepairKit-X.X.X.zip` ফাইলটো `build` ফোল্ডাৰত থাকিব।
