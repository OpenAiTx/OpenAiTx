## Fonctionnalités

### **Réparations Automatiques**

Effectue automatiquement un nettoyage complet du système et une réparation, incluant :

- Création d’un point de restauration système
- Suppression des politiques système malveillantes
- Application de réglages sur le registre et le système
- Suppression des fonctionnalités Windows vulnérables
- Application des paramètres optimaux de Windows Defender
- Nettoyage des fichiers indésirables difficiles à trouver
- Suppression des logiciels préinstallés indésirables
- Réparation des problèmes de disque et des fichiers Windows corrompus
- Mise à jour des programmes installés obsolètes

Chaque fonction de la réparation est entièrement personnalisable, vous permettant d’activer ou de désactiver les réparations spécifiques selon vos besoins via
les cases à cocher sur le panneau, ou les fichiers de configuration situés dans le dossier **config**. **Ne modifiez ces fichiers
que si vous savez ce que vous faites.**

> **Remarque :** L’option **Supprimer les malwares** précédemment disponible a été supprimée. **Exécuter les réglages de Windows Defender**
> applique désormais les paramètres optimaux de Windows Defender. Des options d’analyse manuelle sont disponibles via les options Emsisoft ou Sophos
> dans le panneau des Programmes Utiles.

### **Programmes Utiles**

Accédez aux outils logiciels essentiels pour la maintenance du système, y compris :

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)** : Identifiez les détails matériels de votre système.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)** : Surveillez les températures et les spécifications du matériel.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)** : Analysez les malwares avec Emsisoft Emergency Kit.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)** : Analysez les malwares avec Sophos Scan & Clean.
- **[TreeSize](https://jam-software.com/treesize_free)** : Analysez et gérez le contenu des disques.
- **[Everything](https://voidtools.com)** : Un moteur de recherche de fichiers ultra-rapide.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)** : Surveillez l'état et la santé de vos disques.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)** : Testez les vitesses de lecture/écriture de vos disques.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)** : Une alternative au gestionnaire de démarrage Windows.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)** : Une alternative au gestionnaire de tâches Windows.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)** : Surveillez l'activité système et les processus.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)** : Affichez et analysez les rapports de crash BSOD Windows.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)** : Un utilitaire léger de mise à jour des pilotes NVIDIA.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)** : Supprimez les pilotes et paquets d’affichage.
- **[Recuva](https://ccleaner.com/recuva)** : Un outil de récupération de données pour fichiers supprimés.
- **[DiskGenius](https://diskgenius.com)** : Un gestionnaire de partitions de disque et outil de récupération de données.

## Télécharger et exécuter

Vous ne savez pas comment télécharger et lancer RepairKit ? Suivez ces étapes :

1. Téléchargez la dernière version de RepairKit depuis
   la [page des Releases](https://github.com/Foulest/RepairKit/releases/latest) (cliquez sur le fichier `RepairKit-X.X.X.zip`).
2. Extrayez le fichier ZIP dans un dossier sur votre ordinateur. Vous pouvez utiliser un programme comme [7-Zip](https://7-zip.org) pour extraire
   le contenu.
3. Double-cliquez sur le fichier `RepairKit-X.X.X.exe` pour lancer le programme.
4. Si Windows Defender SmartScreen bloque l’application, cliquez sur `Informations complémentaires` puis sur `Exécuter quand même`.
5. Cliquez sur `Oui` dans l’invite du contrôle de compte utilisateur.

RepairKit va maintenant s’ouvrir, et vous pouvez commencer à utiliser ses fonctionnalités.

## Compilation

1. Clonez le dépôt.
2. Ouvrez un invite de commandes/terminal dans le dossier du dépôt.
3. Exécutez `gradlew createExe` sous Windows, ou `./gradlew createExe` sous macOS ou Linux.
4. Le fichier `RepairKit-X.X.X.zip` généré se trouvera dans le dossier `build`.
