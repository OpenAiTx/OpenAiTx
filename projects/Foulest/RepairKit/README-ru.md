## Возможности

### **Автоматический ремонт**

Автоматически выполняет комплексную очистку и восстановление системы, включая:

- Создание точки восстановления системы
- Удаление вредоносных политик системы
- Запуск твиков реестра и системы
- Удаление уязвимых компонентов Windows
- Применение оптимальных настроек Windows Defender
- Очистка труднонаходимых временных файлов
- Удаление предустановленного мусора
- Исправление ошибок диска и поврежденных файлов Windows
- Обновление устаревших установленных программ

Каждая функция ремонта полностью настраиваема, позволяя включать или отключать конкретные операции с помощью
чекбоксов на панели или конфигурационных файлов в папке **config**. **Изменяйте эти
файлы только если точно знаете, что делаете.**

> **Примечание:** Ранее доступная опция **Удалить вредоносное ПО** была удалена. Теперь **Твики Windows Defender**
> применяют оптимальные параметры Windows Defender. Опции ручного сканирования доступны через Emsisoft или Sophos
> в панели "Полезные программы".

### **Полезные программы**

Получите доступ к необходимым программам для обслуживания системы, включая:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: Определите характеристики аппаратного обеспечения вашей системы.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: Следите за температурой и характеристиками железа.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Сканируйте систему на вредоносное ПО с помощью Emsisoft Emergency Kit.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Сканируйте систему на вредоносное ПО с помощью Sophos Scan & Clean.
- **[TreeSize](https://jam-software.com/treesize_free)**: Анализируйте и управляйте содержимым диска.
- **[Everything](https://voidtools.com)**: Сверхбыстрый поисковик файлов.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: Контролируйте состояние и здоровье накопителя.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: Тестируйте скорость чтения/записи
  вашего диска.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: Альтернатива Диспетчеру автозагрузки Windows.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: Альтернатива
  Диспетчеру задач Windows.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: Мониторинг активности системы и
  процессов.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: Просмотр и анализ дампов аварийных синих экранов Windows.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: Легковесная утилита обновления драйверов NVIDIA.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: Удаление
  драйверов и пакетов видеокарт.
- **[Recuva](https://ccleaner.com/recuva)**: Программа для восстановления удалённых файлов.
- **[DiskGenius](https://diskgenius.com)**: Менеджер разделов и инструмент восстановления данных.

## Загрузка и запуск

Не знаете, как скачать и запустить RepairKit? Следуйте этим шагам:

1. Скачайте последнюю версию RepairKit с
   [страницы релизов](https://github.com/Foulest/RepairKit/releases/latest) (нажмите на файл `RepairKit-X.X.X.zip`).
2. Извлеките ZIP-файл в папку на вашем компьютере. Для извлечения можно использовать программу [7-Zip](https://7-zip.org).
3. Дважды кликните на файл `RepairKit-X.X.X.exe`, чтобы запустить программу.
4. Если Windows Defender SmartScreen блокирует приложение, нажмите `Подробнее`, затем `Выполнить в любом случае`.
5. Нажмите `Да` в запросе контроля учетных записей.

RepairKit теперь откроется, и вы можете начать использовать его функции.

## Сборка

1. Клонируйте репозиторий.
2. Откройте командную строку/терминал в каталоге репозитория.
3. Выполните `gradlew createExe` в Windows или `./gradlew createExe` в macOS или Linux.
4. Готовый файл `RepairKit-X.X.X.zip` будет в папке `build`.
