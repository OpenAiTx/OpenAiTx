## 功能

### **自动修复**

自动执行全面的系统清理和修复，包括：

- 创建系统还原点
- 删除恶意系统策略
- 运行注册表和系统调整
- 移除易受攻击的Windows功能
- 应用最佳的Windows Defender设置
- 清理难以发现的垃圾文件
- 移除预装的臃肿软件
- 修复磁盘问题和损坏的Windows文件
- 更新过时的已安装程序
修复中的每个功能都可完全自定义，允许您根据需要启用或禁用特定的修复，方法是使用面板上的复选框，或位于 **config** 文件夹中的配置文件。**只有在知道自己在做什么时才修改这些文件。**

> **注意：** 之前提供的 **移除恶意软件** 选项现已移除。**运行 Windows Defender 调优** 现在改为应用最佳的 Windows Defender 设置。手动扫描选项可通过“实用程序”面板中的 Emsisoft 或 Sophos 扫描选项使用。

## 下载与运行

不确定如何下载和运行 RepairKit？请按照以下步骤操作：

1. 从[发布页面](https://github.com/Foulest/RepairKit/releases/latest)下载最新版本的 RepairKit（点击 `RepairKit-X.X.X.zip` 文件）。
2. 将 ZIP 文件解压到电脑上的一个文件夹。你可以使用类似 [7-Zip](https://7-zip.org) 的程序进行解压。
3. 双击 `RepairKit-X.X.X.exe` 文件运行程序。
4. 如果 Windows Defender SmartScreen 阻止应用，点击“更多信息”，然后选择“仍然运行”。
5. 在用户账户控制提示中点击“是”。

RepairKit 现在将打开，你可以开始使用其功能。



## 编译

1. 克隆仓库。
2. 打开命令提示符/终端，进入仓库目录。
3. 在 Windows 上运行 `gradlew createExe`，在 macOS 或 Linux 上运行 `./gradlew createExe`。
4. 生成的 `RepairKit-X.X.X.zip` 文件将在 `build` 文件夹中。
