## 功能特色

### **自动修复**

自动执行全面的系统清理与修复操作，包括：

- 创建系统还原点
- 删除恶意系统策略
- 运行注册表和系统优化
- 移除易受攻击的Windows功能
- 应用最佳的Windows Defender设置
- 清理难以发现的垃圾文件
- 移除预装的臃肿软件
- 修复磁盘问题及损坏的Windows文件
- 更新过时的已安装程序

修复中的每一项功能都可以完全自定义，您可以通过面板上的复选框或**config**文件夹中的配置文件
选择启用或禁用特定修复项。**仅当您了解相关内容时再修改这些文件。**

> **注意：** 先前提供的**移除恶意软件**选项已被移除。**运行Windows Defender优化**现在
> 应用最佳的Windows Defender设置。您也可以在实用程序面板中通过Emsisoft或Sophos
> 扫描选项手动进行查杀。

## 下载与运行

不知道如何下载和运行 RepairKit？请按照以下步骤操作：

1. 从 [Releases 页面](https://github.com/Foulest/RepairKit/releases/latest) 下载最新版本的 RepairKit（点击 `RepairKit-X.X.X.zip` 文件）。
2. 将 ZIP 文件解压到电脑上的某个文件夹。你可以使用 [7-Zip](https://7-zip.org) 等程序来解压内容。
3. 双击 `RepairKit-X.X.X.exe` 文件运行程序。
4. 如果 Windows Defender SmartScreen 阻止了该应用，请点击 `更多信息`，然后点击 `仍然运行`。
5. 在用户账户控制提示中点击 `是`。

RepairKit 现在会打开，你可以开始使用它的功能了。

## 编译

1. 克隆该仓库。
2. 打开命令提示符/终端，定位到仓库目录。
3. 在 Windows 上运行 `gradlew createExe`，或在 macOS 或 Linux 上运行 `./gradlew createExe`。
4. 构建好的 `RepairKit-X.X.X.zip` 文件将在 `build` 文件夹中。
