## 功能介紹

### **自動修復**

自動執行全面的系統清理與修復，包括：

- 建立系統還原點
- 刪除惡意系統政策
- 執行登錄檔與系統優化
- 移除高風險的 Windows 功能
- 套用最佳化的 Windows Defender 設定
- 清理難以發現的垃圾檔案
- 移除預載的臃腫軟體
- 修復磁碟問題與損壞的 Windows 檔案
- 更新過時的已安裝程式

修復中的每一個功能都可以完全自訂，您可以透過面板上的勾選框或位於 **config** 資料夾中的設定檔
啟用或停用特定修復項目。**僅在您確定操作時才修改這些檔案。**

> **注意：** 先前提供的 **移除惡意軟體** 選項已經移除。現在的 **執行 Windows Defender 優化** 
> 會套用最佳的 Windows Defender 設定。您可於實用程式面板中，使用 Emsisoft 或 Sophos 
> 掃描選項執行手動掃描。

## 下載與執行

不確定如何下載並執行 RepairKit？請依照以下步驟：

1. 從[發行頁面](https://github.com/Foulest/RepairKit/releases/latest)下載最新版本的 RepairKit（點擊 `RepairKit-X.X.X.zip` 檔案）。
2. 將 ZIP 壓縮檔解壓縮到電腦上的資料夾。你可以使用 [7-Zip](https://7-zip.org) 等程式來解壓縮內容。
3. 雙擊 `RepairKit-X.X.X.exe` 檔案以執行程式。
4. 若 Windows Defender SmartScreen 阻擋此應用程式，請點擊「更多資訊」再選擇「仍要執行」。
5. 在使用者帳戶控制提示時點選「是」。

RepairKit 現已開啟，您可以開始使用其功能。

## 編譯說明

1. 複製（clone）此儲存庫。
2. 開啟命令提示字元或終端機，切換到儲存庫目錄。
3. 在 Windows 執行 `gradlew createExe`，或在 macOS/Linux 執行 `./gradlew createExe`。
4. 編譯後的 `RepairKit-X.X.X.zip` 檔案將會在 `build` 資料夾中。
