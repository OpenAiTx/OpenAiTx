## 機能

### **自動修復**

包括的なシステムクリーニングと修復を自動的に実行します。内容は以下の通りです：

- システムの復元ポイントの作成
- 悪意のあるシステムポリシーの削除
- レジストリやシステムの調整実行
- 脆弱なWindows機能の削除
- 最適なWindows Defender設定の適用
- 見つけにくい不要ファイルのクリーンアップ
- プレインストールされた不要ソフトの削除
- ディスク問題や破損したWindowsファイルの修復
- インストール済みプログラムの更新

修復の各機能は完全にカスタマイズ可能で、パネル上のチェックボックスや**config**フォルダ内の
設定ファイルから必要に応じて個別に有効／無効化できます。**これらのファイルは内容を理解している場合のみ編集してください。**

> **注意：** 以前の**マルウェアの削除**オプションは削除されました。**Windows Defender Tweaksの実行**は
> 現在、最適なWindows Defender設定を適用します。EmsisoftまたはSophosスキャンオプションを使えば
> Useful Programsパネルで手動スキャンが可能です。

- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: ディスクドライブの読み書き速度を
  ベンチマークします。

  代替ツールです。

  代替ツールです。
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: システムの動作やプロセスを
  監視します。
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: WindowsのBSODクラッシュダンプを閲覧・解析します。

## ダウンロードと実行

RepairKitのダウンロードと実行方法が分からない場合は、以下の手順に従ってください：

1. [リリースページ](https://github.com/Foulest/RepairKit/releases/latest)からRepairKitの最新版をダウンロードします（`RepairKit-X.X.X.zip`ファイルをクリック）。
2. ZIPファイルをPC上のフォルダに展開します。[7-Zip](https://7-zip.org)などのプログラムで内容を解凍できます。
3. `RepairKit-X.X.X.exe`ファイルをダブルクリックしてプログラムを実行します。
4. Windows Defender SmartScreenがアプリをブロックした場合は、「詳細情報」をクリックし、「実行」を選択してください。
5. ユーザーアカウント制御のプロンプトで「はい」をクリックします。

RepairKitが起動し、機能の利用を開始できます。

## コンパイル方法

1. リポジトリをクローンします。
2. コマンドプロンプト/ターミナルでリポジトリディレクトリを開きます。
3. Windowsでは`gradlew createExe`、macOSやLinuxでは`./gradlew createExe`を実行します。
4. 作成された`RepairKit-X.X.X.zip`ファイルは`build`フォルダ内にあります。
