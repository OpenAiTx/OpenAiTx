## Funzionalità

### **Riparazioni Automatiche**

Esegui automaticamente una pulizia e riparazione completa del sistema, tra cui:

- Creazione di un Punto di Ripristino del sistema
- Eliminazione di politiche di sistema dannose
- Applicazione di modifiche al registro e al sistema
- Rimozione di funzionalità vulnerabili di Windows
- Applicazione delle impostazioni ottimali di Windows Defender
- Pulizia di file inutili difficili da trovare
- Rimozione di software preinstallati indesiderati
- Riparazione di problemi del disco e file Windows corrotti
- Aggiornamento dei programmi installati obsoleti

Ogni funzione della riparazione è completamente personalizzabile, permettendo di abilitare o disabilitare riparazioni specifiche secondo necessità tramite
i checkbox sul pannello, oppure i file di configurazione situati nella cartella **config**. **Modifica questi file
solo se sai cosa stai facendo.**

> **Nota:** L’opzione **Rimuovi Malware** precedentemente disponibile è stata rimossa. **Applica Modifiche a Windows Defender** ora
> applica le impostazioni ottimali di Windows Defender. Opzioni di scansione manuale sono disponibili tramite le opzioni di scansione Emsisoft o Sophos
> nel pannello Programmi Utili.

### **Programmi Utili**

Accedi agli strumenti software essenziali per la manutenzione del sistema, tra cui:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: Identifica i dettagli hardware del tuo sistema.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: Monitora temperature e specifiche hardware.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Scansiona il malware con Emsisoft Emergency Kit.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Scansiona il malware con Sophos Scan & Clean.
- **[TreeSize](https://jam-software.com/treesize_free)**: Analizza e gestisci il contenuto del disco.
- **[Everything](https://voidtools.com)**: Motore di ricerca file ultra-veloce.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: Monitora lo stato e la salute del disco.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: Valuta le velocità di lettura/scrittura
  del tuo disco.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: Alternativa al Windows Startup
  Manager.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: Alternativa al
  Task Manager di Windows.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: Monitora attività e processi
  del sistema.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: Visualizza e analizza i crash dump BSOD di Windows.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: Aggiornatore leggero dei driver NVIDIA.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: Rimuove
  driver e pacchetti display.
- **[Recuva](https://ccleaner.com/recuva)**: Strumento di recupero dati per file eliminati.
- **[DiskGenius](https://diskgenius.com)**: Gestore di partizioni e strumento di recupero dati.

## Download e Avvio

Non sai come scaricare ed eseguire RepairKit? Segui questi passaggi:

1. Scarica l’ultima versione di RepairKit dalla
   [pagina Releases](https://github.com/Foulest/RepairKit/releases/latest) (clicca sul file `RepairKit-X.X.X.zip`).
2. Estrai il file ZIP in una cartella sul tuo computer. Puoi usare un programma come [7-Zip](https://7-zip.org) per estrarre
   i contenuti.
3. Fai doppio clic sul file `RepairKit-X.X.X.exe` per avviare il programma.
4. Se Windows Defender SmartScreen blocca l’app, clicca su `Ulteriori informazioni` e poi su `Esegui comunque`.
5. Clicca su `Sì` nella richiesta del Controllo Account Utente.

RepairKit ora si aprirà e potrai iniziare a usare le sue funzioni.

## Compilazione

1. Clona il repository.
2. Apri un prompt dei comandi/terminale nella directory del repository.
3. Esegui `gradlew createExe` su Windows, oppure `./gradlew createExe` su macOS o Linux.
4. Il file `RepairKit-X.X.X.zip` generato sarà nella cartella `build`.
