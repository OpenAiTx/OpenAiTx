## Funcionalidades

### **Reparos Automáticos**

Executa automaticamente uma limpeza e reparo abrangentes do sistema, incluindo:

- Criação de um Ponto de Restauração do sistema
- Exclusão de políticas maliciosas do sistema
- Execução de ajustes no registro e sistema
- Remoção de recursos vulneráveis do Windows
- Aplicação de configurações otimizadas do Windows Defender
- Limpeza de arquivos inúteis de difícil acesso
- Remoção de programas pré-instalados indesejados
- Reparo de problemas de disco e arquivos corrompidos do Windows
- Atualização de programas instalados desatualizados

Cada função do reparo é totalmente personalizável, permitindo habilitar ou desabilitar reparos específicos conforme necessário usando
as caixas de seleção no painel ou os arquivos de configuração localizados na pasta **config**. **Só modifique estes
arquivos se souber o que está fazendo.**

> **Nota:** A opção **Remover Malware** anteriormente disponível foi removida. **Executar Ajustes do Windows Defender** agora
> aplica as configurações otimizadas do Windows Defender. Opções de verificação manual estão disponíveis usando as opções de Verificação Emsisoft ou Sophos
> no painel de Programas Úteis.

## Download e Execução

Não sabe como baixar e executar o RepairKit? Siga estes passos:

1. Baixe a versão mais recente do RepairKit na
   [página de Releases](https://github.com/Foulest/RepairKit/releases/latest) (clique no arquivo `RepairKit-X.X.X.zip`).
2. Extraia o arquivo ZIP para uma pasta em seu computador. Você pode usar um programa como o [7-Zip](https://7-zip.org) para extrair
   o conteúdo.
3. Dê um duplo clique no arquivo `RepairKit-X.X.X.exe` para executar o programa.
4. Se o Windows Defender SmartScreen bloquear o aplicativo, clique em `Mais informações` e depois em `Executar mesmo assim`.
5. Clique em `Sim` na solicitação de Controle de Conta de Usuário.

O RepairKit será aberto agora, e você pode começar a usar seus recursos.

## Compilando

1. Clone o repositório.
2. Abra um prompt de comando/terminal no diretório do repositório.
3. Execute `gradlew createExe` no Windows, ou `./gradlew createExe` no macOS ou Linux.
4. O arquivo `RepairKit-X.X.X.zip` compilado estará na pasta `build`.
