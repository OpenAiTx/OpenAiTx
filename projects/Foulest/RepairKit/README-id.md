## Fitur

### **Perbaikan Otomatis**

Secara otomatis melakukan pembersihan dan perbaikan sistem secara menyeluruh, termasuk:

- Membuat Restore Point sistem
- Menghapus kebijakan sistem berbahaya
- Menjalankan tweak registry dan sistem
- Menghapus fitur Windows yang rentan
- Menerapkan pengaturan optimal Windows Defender
- Membersihkan file sampah yang sulit ditemukan
- Menghapus bloatware yang sudah terpasang
- Memperbaiki masalah disk dan file Windows yang rusak
- Memperbarui program terpasang yang sudah usang

Setiap fungsi dalam perbaikan sepenuhnya dapat disesuaikan, memungkinkan Anda untuk mengaktifkan atau menonaktifkan perbaikan tertentu sesuai kebutuhan menggunakan
baik kotak centang pada panel, maupun file konfigurasi yang terletak di folder **config**. **Hanya ubah file ini jika Anda tahu apa yang Anda lakukan.**

> **Catatan:** Opsi **Remove Malware** yang sebelumnya tersedia kini telah dihapus. **Jalankan Windows Defender Tweaks** sekarang
> menerapkan pengaturan optimal Windows Defender. Opsi pemindaian manual tersedia melalui opsi Emsisoft atau Sophos
> Scan di panel Program Berguna.


![Perbaikan Otomatis](https://i.imgur.com/mnkUQV1.png)

### **Program Berguna**

Akses alat perangkat lunak penting untuk pemeliharaan sistem, termasuk:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: Identifikasi detail perangkat keras sistem Anda.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: Pantau suhu dan spesifikasi perangkat keras.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Pindai malware menggunakan Emsisoft Emergency Kit.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Pindai malware menggunakan Sophos Scan & Clean.
- **[TreeSize](https://jam-software.com/treesize_free)**: Analisis dan kelola isi disk.
- **[Everything](https://voidtools.com)**: Mesin pencari file yang sangat cepat.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: Pantau kesehatan dan status drive disk Anda.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: Uji kecepatan baca/tulis drive disk Anda.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: Alternatif untuk Windows Startup Manager.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: Alternatif untuk Windows Task Manager.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: Pantau aktivitas dan proses sistem.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: Lihat dan analisis dump crash BSOD Windows.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: Pembaru driver kartu grafis NVIDIA yang ringan.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: Hapus driver dan paket display.
- **[Recuva](https://ccleaner.com/recuva)**: Alat pemulihan data untuk file yang terhapus.
- **[DiskGenius](https://diskgenius.com)**: Manajer partisi disk dan alat pemulihan data.

## Unduh dan Jalankan

Tidak yakin cara mengunduh dan menjalankan RepairKit? Ikuti langkah-langkah berikut:

1. Unduh versi terbaru RepairKit dari
   [halaman Rilis](https://github.com/Foulest/RepairKit/releases/latest) (klik file `RepairKit-X.X.X.zip`).
2. Ekstrak file ZIP ke folder di komputer Anda. Anda dapat menggunakan program seperti [7-Zip](https://7-zip.org) untuk mengekstrak
   isinya.
3. Klik dua kali file `RepairKit-X.X.X.exe` untuk menjalankan program.
4. Jika Windows Defender SmartScreen memblokir aplikasi, klik `More info` lalu `Run anyway`.
5. Klik `Yes` pada prompt User Account Control.

RepairKit akan terbuka, dan Anda dapat mulai menggunakan fitur-fiturnya.

## Kompilasi

1. Clone repository.
2. Buka command prompt/terminal ke direktori repository.
3. Jalankan `gradlew createExe` di Windows, atau `./gradlew createExe` di macOS atau Linux.
4. File `RepairKit-X.X.X.zip` hasil build akan berada di folder `build`.
