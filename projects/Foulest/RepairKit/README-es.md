## Características

### **Reparaciones Automáticas**

Realiza automáticamente una limpieza y reparación exhaustiva del sistema, incluyendo:

- Creación de un Punto de Restauración del sistema
- Eliminación de políticas maliciosas del sistema
- Ejecución de ajustes en el registro y el sistema
- Eliminación de características vulnerables de Windows
- Aplicación de configuraciones óptimas para Windows Defender
- Limpieza de archivos basura difíciles de encontrar
- Eliminación de bloatware preinstalado
- Reparación de problemas de disco y archivos corruptos de Windows
- Actualización de programas instalados desactualizados

Cada función en la reparación es totalmente personalizable, permitiéndote activar o desactivar reparaciones específicas según lo requieras utilizando
ya sea las casillas de verificación en el panel, o los archivos de configuración ubicados en la carpeta **config**. **Solo modifica estos
archivos si sabes lo que estás haciendo.**

> **Nota:** La opción **Eliminar Malware** que estaba disponible anteriormente ha sido eliminada. **Ejecutar ajustes de Windows Defender** ahora
> aplica configuraciones óptimas de Windows Defender en su lugar. Opciones de escaneo manual están disponibles usando las opciones de escaneo Emsisoft o Sophos
> en el panel de Programas Útiles.

### **Programas Útiles**

Accede a herramientas esenciales de software para el mantenimiento del sistema, incluyendo:

- **[CPU-Z](https://cpuid.com/softwares/cpu-z.html)**: Identifica los detalles de hardware de tu sistema.
- **[HWMonitor](https://cpuid.com/softwares/hwmonitor.html)**: Monitorea temperaturas y especificaciones del hardware.
- **[Emsisoft Scan](https://emsisoft.com/en/home/emergency-kit)**: Escanea en busca de malware con Emsisoft Emergency Kit.
- **[Sophos Scan](https://sophos.com/en-us/free-tools/virus-removal-tool)**: Escanea en busca de malware con Sophos Scan & Clean.
- **[TreeSize](https://jam-software.com/treesize_free)**: Analiza y gestiona el contenido del disco.
- **[Everything](https://voidtools.com)**: Un motor de búsqueda de archivos ultra-rápido.
- **[CrystalDiskInfo](https://sourceforge.net/projects/crystaldiskinfo)**: Monitorea el estado y la salud de tu disco.
- **[CrystalDiskMark](https://crystalmark.info/en/software/crystaldiskmark)**: Realiza pruebas de velocidad de lectura/escritura
  de tu disco.
- **[Autoruns](https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns)**: Una alternativa al Administrador de inicio de Windows.
- **[Process Explorer](https://learn.microsoft.com/en-us/sysinternals/downloads/process-explorer)**: Una alternativa al
  Administrador de tareas de Windows.
- **[Process Monitor](https://learn.microsoft.com/en-us/sysinternals/downloads/procmon)**: Monitorea la actividad y los procesos del sistema.
- **[BlueScreenView](https://nirsoft.net/utils/blue_screen_view.html)**: Visualiza y analiza volcados de errores de Windows BSOD.
- **[NVCleanstall](https://techpowerup.com/download/techpowerup-nvcleanstall)**: Actualizador liviano de drivers de tarjetas NVIDIA.
- **[DisplayDriverUninstaller](https://guru3d.com/files-details/display-driver-uninstaller-download.html)**: Elimina
  controladores y paquetes de pantalla.
- **[Recuva](https://ccleaner.com/recuva)**: Una herramienta de recuperación de datos para archivos eliminados.
- **[DiskGenius](https://diskgenius.com)**: Gestor de particiones de disco y herramienta de recuperación de datos.

## Descargar y ejecutar

¿No sabes cómo descargar y ejecutar RepairKit? Sigue estos pasos:

1. Descarga la última versión de RepairKit desde
   la [página de lanzamientos](https://github.com/Foulest/RepairKit/releases/latest) (haz clic en el archivo `RepairKit-X.X.X.zip`).
2. Extrae el archivo ZIP en una carpeta de tu computadora. Puedes usar un programa como [7-Zip](https://7-zip.org) para extraer
   el contenido.
3. Haz doble clic en el archivo `RepairKit-X.X.X.exe` para ejecutar el programa.
4. Si Windows Defender SmartScreen bloquea la app, haz clic en `Más información` y luego en `Ejecutar de todas formas`.
5. Haz clic en `Sí` en el aviso de Control de cuentas de usuario.

RepairKit se abrirá y podrás comenzar a usar sus funciones.

## Compilación

1. Clona el repositorio.
2. Abre una terminal o símbolo del sistema en el directorio del repositorio.
3. Ejecuta `gradlew createExe` en Windows o `./gradlew createExe` en macOS o Linux.
4. El archivo `RepairKit-X.X.X.zip` generado estará en la carpeta `build`.
