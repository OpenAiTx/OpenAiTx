# Un repositorio con miles de presets de color predefinidos para ReShade

## **¿Qué es un LUT?**

Los LUTs (Tabla de Consulta de Color) son presets de color predeterminados que puedes usar en cualquier juego donde puedas instalar ReShade.

## **¿Qué necesito para que funcione?**

+ [ReShade](https://reshade.me)
+ El archivo **_BaseLUT.fxh** que viene con este repositorio
+ Archivos que vienen con la instalación por defecto de reshade.
  + **ReShade.fxh**
  + **ReShadeUI.fxh**
+ Los archivos **.fx** y **.png** con nombres coincidentes.

## **Instalación**

1. [Descarga](https://github.com/TheGordinho/MLUT/archive/refs/heads/master.zip) el repositorio.
2. Mueve los archivos **.fx** que quieras (no necesitas todos, solo los que desees) a la carpeta Shaders dentro de tu carpeta de instalación de reshade.
3. Mueve los archivos **.png** correspondientes a la carpeta Textures dentro de tu carpeta de instalación de reshade.
4. ¡Listo!

https://user-images.githubusercontent.com/23023051/118344702-16177780-b506-11eb-8b80-907017128b5b.mp4

## **Consejos y Trucos**

Puedes combinar y mezclar tantos LUTs como quieras, pero ten en cuenta que comenzarás a ver bandas de color intensas con 2 o más LUTs activos al mismo tiempo.

> Cambia el orden de carga del shader para obtener resultados diferentes.

https://user-images.githubusercontent.com/23023051/118404928-719f4d80-b64b-11eb-964d-09fa9b60fe8b.mp4

> Coloca los LUTs entre los 2 shaders PCGI para cambiar los colores del GI.

https://user-images.githubusercontent.com/23023051/118405513-9cd76c00-b64e-11eb-915f-edffe52f1dae.mp4

> Ahora puedes usar MLUT con profundidad y pintar el cielo con todos los colores del arcoíris \o/
