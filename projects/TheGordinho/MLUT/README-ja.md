# ReShade用に何千ものプリセットカラープリセットを収録したリポジトリ

## **LUTとは何ですか？**

LUT（カラー・ルックアップ・テーブル）は、ReShadeをインストールできる任意のゲームで使える焼き込み済みのカラープリセットです。

## **動作させるために何が必要ですか？**

+ [ReShade](https://reshade.me)
+ このリポジトリに付属する**_BaseLUT.fxh**ファイル
+ デフォルトのReShadeインストールに含まれるファイル
  + **ReShade.fxh**
  + **ReShadeUI.fxh**
+ ファイル名が一致する**.fx**および**.png**ファイル

## **インストール方法**

1. [リポジトリをダウンロード](https://github.com/TheGordinho/MLUT/archive/refs/heads/master.zip)します。
2. 使用したい**.fx**ファイル（全部である必要はなく、使いたいものだけ）をReShadeのインストールフォルダ内のShadersフォルダに移動します。
3. 対応する**.png**ファイルをReShadeのインストールフォルダ内のTexturesフォルダに移動します。
4. 完了です！

https://user-images.githubusercontent.com/23023051/118344702-16177780-b506-11eb-8b80-907017128b5b.mp4

## **ヒントとコツ**


好きなだけ多くのLUTを組み合わせることができますが、2つ以上のLUTを同時に有効にすると強いカラーバンディングが発生し始めることに注意してください。

> シェーダーの読み込み順を変更して、異なる結果を得てみてください。

https://user-images.githubusercontent.com/23023051/118404928-719f4d80-b64b-11eb-964d-09fa9b60fe8b.mp4

> 2つのPCGIシェーダーの間にLUTを挟んで配置し、GIの色を変えてみてください。

https://user-images.githubusercontent.com/23023051/118405513-9cd76c00-b64e-11eb-915f-edffe52f1dae.mp4

> これでMLUTを深度と組み合わせて使い、空を虹色に塗ることができます \o/

https://user-images.githubusercontent.com/23023051/146495932-cdbec80d-e532-4300-9a88-323bf675bc10.mp4
