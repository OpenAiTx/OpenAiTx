# ReShade용 수천 개의 미리 정의된 색상 프리셋 저장소

## **LUT란 무엇인가요?**

LUT(컬러 룩업 테이블)는 ReShade를 설치할 수 있는 모든 게임에서 사용할 수 있는 미리 구워진 색상 프리셋입니다.

## **작동시키려면 무엇이 필요한가요?**

+ [ReShade](https://reshade.me)
+ 이 저장소에 포함된 **_BaseLUT.fxh** 파일
+ 기본 ReShade 설치 시 포함된 파일들
  + **ReShade.fxh**
  + **ReShadeUI.fxh**
+ 파일 이름이 일치하는 **.fx** 및 **.png** 파일

## **설치 방법**

1. 저장소를 [다운로드](https://github.com/TheGordinho/MLUT/archive/refs/heads/master.zip)합니다.
2. 원하는 **.fx** 파일들(모두가 아니라 원하는 것들만) 을 ReShade 설치 폴더 내 Shaders 폴더로 이동합니다.
3. 일치하는 **.png** 파일들을 ReShade 설치 폴더 내 Textures 폴더로 이동합니다.
4. 완료!

https://user-images.githubusercontent.com/23023051/118344702-16177780-b506-11eb-8b80-907017128b5b.mp4

## **팁과 요령**

여러 개의 LUT를 원하는 만큼 혼합하여 사용할 수 있지만, 2개 이상의 LUT가 동시에 활성화되면 심한 색상 밴딩 현상이 나타날 수 있습니다.

> 셰이더의 로드 순서를 변경하여 다른 결과를 얻어보세요.

> GI 색상을 변경하려면 LUT를 2개의 PCGI 셰이더 사이에 배치하세요.
