# 一个拥有数千预设色彩方案的ReShade资源库

## **什么是LUT？**

LUTs（颜色查找表）是预先烘焙的色彩预设，你可以在任何安装了ReShade的游戏中使用它们。

## **我需要什么来使其工作？**

+ [ReShade](https://reshade.me)
+ 本仓库附带的 **_BaseLUT.fxh** 文件
+ 默认ReShade安装包中附带的文件。
  + **ReShade.fxh**
  + **ReShadeUI.fxh**
+ 文件名匹配的 **.fx** 和 **.png** 文件。

## **安装**

1. [下载](https://github.com/TheGordinho/MLUT/archive/refs/heads/master.zip)该资源库。
2. 将你想要的 **.fx** 文件（不必全部，只需你想用的）移动到ReShade安装文件夹内的Shaders文件夹。
3. 将匹配的 **.png** 文件移动到ReShade安装文件夹内的Textures文件夹。
4. 完成！

## **技巧与窍门**

你可以随意混合搭配任意数量的LUT，但请注意，当同时激活两个或更多LUT时，会开始出现严重的色带现象。
