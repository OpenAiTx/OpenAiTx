## Snel starten

**Opmerking:** onderstaande instructies zijn voor het installeren van het Excalidraw [npm package](https://www.npmjs.com/package/@excalidraw/excalidraw) wanneer je Excalidraw in je eigen app wilt integreren. Om de repository lokaal te draaien voor ontwikkeling, zie onze [Development Guide](https://docs.excalidraw.com/docs/introduction/development).

Gebruik `npm` of `yarn` om het pakket te installeren.

```bash
npm install react react-dom @excalidraw/excalidraw
# of
yarn add react react-dom @excalidraw/excalidraw
```

Bekijk onze [documentatie](https://docs.excalidraw.com/docs/@excalidraw/excalidraw/installation) voor meer details!
