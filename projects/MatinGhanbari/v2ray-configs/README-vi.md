🧩 Kho lưu trữ do cộng đồng duy trì cung cấp các cấu hình V2ray miễn phí giúp bạn duyệt web an toàn và riêng tư.

## ✨ Tính Năng Nổi Bật
- Tương thích đa giao thức (VMess, VLESS, Trojan, Shadowsocks, SSR, TUIC, Hysteria2, Warp)
- Tự động cập nhật cấu hình mỗi **15 phút**
- Được tổ chức theo loại giao thức
- Mã hóa Base64 để tiện sử dụng
- Nguồn liên tục làm mới

## 🚀 Bắt Đầu Sử Dụng
Chọn một trong hai cách:
1. Sử dụng các liên kết đăng ký được cung cấp
2. Tải cấu hình thủ công từ thư mục `subscriptions`

### 🔗 Tất Cả Đăng Ký
- Tất cả giao thức [+7500 cấu hình]:
```
https://raw.githubusercontent.com/MatinGhanbari/v2ray-configs/main/subscriptions/v2ray/all_sub.txt
```

### ♾️ Đăng Ký Siêu Cấp
- Những cấu hình tốt nhất được chọn lọc qua kiểm tra từ tất cả cấu hình:
```
https://raw.githubusercontent.com/MatinGhanbari/v2ray-configs/main/subscriptions/v2ray/super-sub.txt
```

### 👨‍🚀 Đăng Ký Chia Nhỏ

- Đăng ký chia nhỏ [200 cấu hình mỗi đăng ký]:

    **Bạn có thể thay đổi mã số đăng ký từ 1 đến 39.** Ví dụ:
```
https://raw.githubusercontent.com/MatinGhanbari/v2ray-configs/main/subscriptions/v2ray/subs/sub1.txt
https://raw.githubusercontent.com/MatinGhanbari/v2ray-configs/main/subscriptions/v2ray/subs/sub2.txt
```
