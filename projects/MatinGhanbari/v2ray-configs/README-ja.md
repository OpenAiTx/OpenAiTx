🧩 コミュニティによって管理されているリポジトリで、無料のV2ray構成を提供し、安全かつプライベートに閲覧できるようにサポートします。

## ✨ 主な特徴
- マルチプロトコル対応（VMess、VLESS、Trojan、Shadowsocks、SSR、TUIC、Hysteria2、Warp）
- **15分ごと**に自動で構成を更新
- プロトコルタイプごとに整理
- 便利なBase64エンコード
- 頻繁にリフレッシュされるソース

## 🚀 はじめに
次のいずれかの方法を選択してください：
1. 提供されているサブスクリプションURLを使用する
2. `subscriptions` フォルダから手動で構成ファイルをダウンロードする

### 🔗 すべてのサブスクリプション
- すべてのプロトコル [+7500構成]:
```
https://raw.githubusercontent.com/MatinGhanbari/v2ray-configs/main/subscriptions/v2ray/all_sub.txt
```
