# Serenity Stock Choke · A-share “bottleneck” aandelen-selectiekader

> "**Volg de toeleveringsketen omhoog om dat ene knooppunt te vinden—het knooppunt waarvan, als de levering stopt, een triljoenindustrie op zijn grondvesten zal schudden. Het kleine aandeel op dat knooppunt is de volgende grote kans.**"

Deze Skill is gebaseerd op de supply chain-bottlenecktheorie van de legendarische Reddit-gebruiker **Serenity (@aleabitoreddit)** en is aangepast voor de A-aandelenmarkt, waardoor een universeel analysemodel voor sectoren ontstaat.

---

## Kernmethodologie

### Straat van Hormuz-analogie

> "De Straat van Hormuz is de levensader van de wereldolie. Zodra deze wordt geblokkeerd, worden alle betrokkenen beïnvloed. Maar als je aandelen hebt in de Straat van Hormuz, heb je de prijszettingsmacht."

**Toegepast op A-aandelen**: Welke schakel is de 'Straat van Hormuz' van een bepaalde grote sector? Welk A-aandeelbedrijf beheerst deze?

### Drie kernlogische stappen

```
① AI大爆发 / 政策驱动 / 技术跃迁
    ↓
② 上游硬件/原材料需求暴增，供应链某些环节产能跟不上
    ↓
③ 瓶颈环节掌握定价权 → 其中小盘股弹性最大
```

## Zes Stappen Analyseketen

| Stap | Kernvraag | Output |
|------|-----------|--------|
| 1️⃣ Bepaal sectorcyclus | Vraagexplosie / technologische sprong / aanbodbeperking? | Fase van de sector |
| 2️⃣ Herleid toeleveringsketen | Waar zit de bottleneck? | Overzicht toeleveringsketen |
| 3️⃣ Zoek A-aandelen | Welke bedrijven zitten op deze knelpunten? | Vierdimensionale signaalkaart |
| 4️⃣ Selectie echt/onecht | Welke zijn ‘echte bottlenecks’, welke liften mee? | Selectieresultaat |
| 5️⃣ Bevestig long/short | Long-signalen vs short-signalen? | Signaalvergelijkingstabel |
| 6️⃣ Rapportage | Inclusief actieadvies + risicowaarschuwing | Gestructureerd rapport |

Elektriciteit, optische modules/CPO, halfgeleiderapparatuur, defensie-industrie, nieuwe energievoertuigen, medische apparatuur, innovatieve geneesmiddelen, pesticiden, OLED... **Elke A-aandelen sector is te analyseren**.

## Risicowaarschuwing

1. Deze Skill is alleen ter referentie en vormt geen beleggingsadvies
2. Smallcaps zijn zeer volatiel, dagelijkse schommelingen van 20-30% zijn niet ongebruikelijk
3. Aandelen op de A-markt worden sterk beïnvloed door beleid, houd toezicht op regelgeving
4. De echtheid van informatie uit de toeleveringsketen is moeilijk te bepalen, onafhankelijk verifiëren is noodzakelijk
5. Logische bevestiging kan 1-3 jaar duren, geduld is vereist
