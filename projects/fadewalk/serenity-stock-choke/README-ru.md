# Serenity Stock Choke · Каркас для выбора акций "узких мест" A-акций

> "**Продвигаясь вверх по цепочке создания стоимости, найдите ту ключевую точку, где 'если поставки прекратятся, триллионная индустрия содрогнется' — именно небольшая компания на этом участке станет следующей взрывной возможностью.**"

Данный навык основан на теории "бутылочных горлышек" цепочки поставок от легендарной личности Reddit **Serenity (@aleabitoreddit)**, адаптированной для рынка A-акций, и формирует универсальную структуру анализа секторов.

---

## Основная методология

### Аналогия с Ормузским проливом

> "Ормузский пролив — это горло мировой нефти. Как только он блокируется, все оказываются под воздействием. Но если у тебя есть доля в Ормузском проливе, ты обладаешь правом устанавливать цену."

**Перевод на A-акции**: Какой этап является "Ормузским проливом" для крупного сектора? Какая компания на рынке A контролирует его?

### Трёхшаговая основная логика

```
① AI大爆发 / 政策驱动 / 技术跃迁
    ↓
② 上游硬件/原材料需求暴增，供应链某些环节产能跟不上
    ↓
③ 瓶颈环节掌握定价权 → 其中小盘股弹性最大
```


### Инструменты данных

- **neodata-financial-search**: котировки секторов, исследовательские отчеты, анализ движения капитала
- **westock-data**: распределение акций, крупные сделки, маржинальное кредитование и краткосрочные займы, рейтинги институтов

## Предупреждение о рисках

1. Данный Skill предоставлен только для ознакомления и не является инвестиционной рекомендацией
2. Волатильность малых акций крайне высока, рост или падение на 20-30% за день не редкость
3. A-акции значительно зависят от политики, необходимо внимательно следить за регуляторными изменениями
4. Информацию о цепочке поставок трудно проверить на достоверность, требуется независимая верификация
5. Проверка логики может занять 1-3 года, требуется терпение
