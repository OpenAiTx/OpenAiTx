# Serenity Stock Choke · A股卡脖子選股框架

> 「**沿著產業鏈向上游追溯，找到那個『一旦斷貨，萬億產業就要地震』的關鍵節點——那個節點上的小盤股，就是下一個暴擊機會。**」

本 Skill 基於 Reddit 傳奇人物 **Serenity（@aleabitoreddit）** 的供應鏈瓶頸理論，將其適配到 A 股市場，形成一套通用的板塊分析框架。

---

## 核心方法論

### 霍爾木茲海峽類比

> 「霍爾木茲海峽是全球石油的咽喉。一旦封鎖，所有人受影響。但如果你在霍爾木茲海峽擁有股份，你就擁有了定價權。」

**翻譯到 A 股**：哪個環節是某個大賽道的「霍爾木茲」？哪個 A 股公司控制它？

### 三步核心邏輯

```
① AI大爆发 / 政策驱动 / 技术跃迁
    ↓
② 上游硬件/原材料需求暴增，供应链某些环节产能跟不上
    ↓
③ 瓶颈环节掌握定价权 → 其中小盘股弹性最大
```

### Serenity 原版戰績（美股）

## 六步分析鏈路

| 步驟 | 核心問題 | 產出 |
|------|---------|------|
| 1️⃣ 定位板塊週期 | 需求爆發 / 技術躍遷 / 供給受限？ | 板塊所處階段 |
| 2️⃣ 溯源供應鏈 | 瓶頸在哪一層？ | 供應鏈層級圖 |
| 3️⃣ 找 A 股標的 | 哪些公司卡在這個節點？ | 四維信號卡 |
| 4️⃣ 真偽篩選 | 哪些是「真瓶頸」，哪些是蹭熱點？ | 篩選結論 |
| 5️⃣ 多空確認 | 做多信號 vs 做空信號？ | 信號對照表 |
| 6️⃣ 輸出報告 | 含操作建議 + 風險提示 | 結構化報告 |

電力、光模組/CPO、半導體設備、軍工、新能源車、醫療器械、創新藥、農藥、OLED……**任意 A 股板塊均可分析**。

| 板塊 | 卡脖子節點 | 代表標的 |

## 快速開始

### 觸發方式

在 WorkBuddy 中直接說：

```
用 serenity-stock-choke 分析 [板块名]
例：用 serenity-stock-choke 分析军工板块
```

或者：

```
分析 [产业链] 的卡脖子机会
找 [赛道] 的瓶颈环节
帮我用 Serenity 框架分析 [板块]
```

### 數據工具

- **neodata-financial-search**：板塊行情、研報、資金流向查詢
- **westock-data**：籌碼分布、大宗交易、融資融券、機構評級

## 風險提示

1. 本 Skill 僅供參考，不構成投資建議
2. 小型股波動極大，單日漲跌 20-30% 並不罕見
3. A 股受政策影響顯著，需密切關注監管動態
4. 供應鏈資訊真偽難辨，需獨立驗證
5. 邏輯驗證可能需要 1-3 年，需有耐心
