# Ex&#46;Co&#46; <img src="https://github.com/matkuki/Ex-Co/blob/master/resources/exco-icon.png" align="top" width="48" height="48">
### Deskripsi: ###
Ex&#46;Co&#46; (Extensible Coder) adalah editor teks GUI yang ditulis dalam Python dan menggunakan pustaka PyQt/QScintilla dengan beberapa lexer yang tersedia dalam Cython untuk penyorotan teks yang lebih efisien.
Saya mengembangkannya dengan pemikiran kesederhanaan penggunaan dan kemudahan untuk diperluas.<br>
Untuk paket prekompilasi Windows, lihat halaman rilis (https://github.com/matkuki/ExCo/releases/latest).
<br><br>

### Beberapa fitur: ###
- **Sistem tata letak multi-jendela baru**
- Semua fungsi standar editor teks (salin, potong, tempel, ...)
- 'Roda fungsi' untuk akses cepat ke sebagian besar fungsi Ex&#46;Co&#46;
- REPL (Read-Eval-Print Loop) Python terintegrasi satu/berbaris untuk akses/manipulasi langsung teks setiap jendela editor dan semua fungsi lainnya
- Pencocokan teks (termasuk antar jendela editor)
- Eksekusi perintah Terminal/Shell langsung dari REPL (Windows atau Linux)
- Kemampuan menambahkan fungsi Python kustom Anda
- Penyorotan sintaks bahasa: AVS, Ada, Bash, Batch, CMake, CPP, CSS, CSharp, CoffeeScript, Cython, D, Diff, Fortran, Fortran77, HTML, IDL, Java, JavaScript, Lua, Makefile, Matlab, Nim, Oberon, Octave, PO, POV, Pascal, Perl, PostScript, Properties, Python, RouterOS (MikroTik), Ruby, SQL, Spice, TCL, TeX, Text, VHDL, Verilog, XML dan YAML
- Penampilan pohon kode untuk: 
    - Python 3
    - Nim
    - C
    - Banyak lagi akan datang ...
<br><br>

### Platform yang didukung: ###
- __Windows__ ([biner pra-kompilasi tersedia](https://github.com/matkuki/ExCo/releases))
- __Linux__ (telah diuji pada Ubuntu, Lubuntu, Mint, Fedora, Raspbian (Raspberry pi, pi2, pi3 dan pi4))
- __Mac OS__ (Terima kasih banyak kepada __zenlc2000__ yang telah menguji ini di Mac OS 10.11)

### Persyaratan: ###
- __Python__ 3.6 atau lebih tinggi
- __Packages__ instal dengan: `pip install -r requirements.txt` (periksa file untuk detail lebih lanjut)
