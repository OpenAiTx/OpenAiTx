# サイバーインテリジェンスツールキット

## 🎯 概要

**サイバーインテリジェンスツールキット**は、幅広い調査および分析タスクを支援するために構築された、マニュアル、プレイブック、チェックリスト、付録の厳選されたコレクションです。調査官、アナリスト、および実務者に対し、迅速な検証やOSINTワークフローから深層フォレンジック分析に至るまで、構造化され信頼性が高く実用的なデジタル調査のリファレンスを提供します。

このリポジトリは以下を強調しています：

* **OSINT（オープンソースインテリジェンス）：** オープンソースから情報を発見、検証、分析する方法。
* **デジタルフォレンジックス：** メディアやデジタル痕跡の真正性を検証するワークフローとツール。
* **AIおよび合成メディア検出：** テキスト、画像、音声、動画におけるAI生成コンテンツを識別する実践的アプローチ。
* **運用セキュリティ（OPSEC）：** 敏感な操作中に調査官を保護するためのガイドラインとベストプラクティス。
* **調査フレームワーク：** 一貫性と再現性を確保するための標準化された方法論。

このツールキットは理論と実践的手順を組み合わせており、迅速な現場使用、構造化分析、長期的な調査プロジェクトに適しています。
