# 网络情报工具包

## 🎯 概述

**网络情报工具包** 是一套精心整理的手册、操作手册、检查清单和附录，旨在支持广泛的调查和分析任务。它为调查人员、分析师和从业者提供结构化、可靠且可操作的数字调查参考——从快速验证和开源情报工作流程到深入的法医分析。

本库重点包括：

* **开源情报（OSINT）：** 发现、验证和分析开源信息的方法。
* **数字取证：** 验证媒体和数字痕迹真实性的工作流程和工具。
* **人工智能与合成媒体检测：** 识别文本、图像、音频和视频中AI生成内容的实用方法。
* **行动安全（OPSEC）：** 保护调查人员在敏感操作期间的指导方针和最佳实践。
* **调查框架：** 确保一致性和可重复性的标准化方法论。

该工具包结合理论与实操程序，适合快速现场使用、结构化分析及长期调查项目。
