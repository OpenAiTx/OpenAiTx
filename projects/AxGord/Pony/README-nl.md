Pony is een set tools voor het ontwikkelen, voorbereiden, bouwen, testen en publiceren van projecten.
Het kan worden gebruikt voor het maken van Heaps, PixiJS, NodeJS, Electron, Cordova, PHP, OpenFL, Unity3D, Flash en Cocos Creator applicaties.

Het werkt op servers of aan de clientzijde.

De bibliotheek heeft een op XML gebaseerd systeem voor het bouwen van UI en componenten, een krachtig eventsysteem en veel hulpprogrammafuncties.

Installatie
===========
Stabiele versie

    haxelib install pony

Onstabiele versie

    haxelib git pony https://github.com/AxGord/Pony

Installeer Pony Tools

    haxelib run pony

Stil installatievoorbeeld

    haxelib run pony install +code -code-insiders +npm +userpath -nodepath +ponypath

`+` - optie inschakelen

`-` - optie uitschakelen

`code` - Installeer aanbevolen plugins voor Visual Studio Code

`code-insiders` - Installeer aanbevolen plugins voor Visual Studio Code Insiders

`npm` - Installeer NPM (Node Package Manager) modules

`userpath` - Stel gebruikerspaden in

`nodepath` - Stel gebruikerspad in naar node_modules, alleen voor Windows
`ponypath` - Stel gebruikerspad in naar pony.exe, alleen voor Windows

CI installatievoorbeeld

    haxelib run pony install -code -code-insiders +npm +userpath
