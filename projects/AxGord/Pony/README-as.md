Pony হৈছে প্ৰকল্পসমূহ বিকাশ, প্রস্তুত, নিৰ্মাণ, পৰীক্ষা আৰু প্ৰকাশ কৰাৰ বাবে এটা টুলসৰ সমষ্টি।
এইটো Heaps, PixiJS, NodeJS, Electron, Cordova, PHP, OpenFL, Unity3D, Flash আৰু Cocos Creator এপ্লিকেশ্যন সৃষ্টি কৰিবলৈ ব্যৱহাৰ কৰিব পাৰি।

এইটো ছাৰ্ভাৰ বা ক্লায়েন্ট-ছাইডত কাম কৰে।

লাইব্ৰেৰীটোৰ এটা XML-ভিত্তিক UI নিৰ্মাণ প্ৰণালী আৰু উপাদান, এটা শক্তিশালী ইভেন্ট প্ৰণালী, আৰু বহুসংখ্যক ইউটিলিটি ফাংচন আছে।

ইন্সটলেশ্যন
============
স্থিতিশীল সংস্কৰণ

    haxelib install pony

অস্থিতিশীল সংস্কৰণ

    haxelib git pony https://github.com/AxGord/Pony

Pony Tools ইনস্টল কৰক

    haxelib run pony

নীৰৱ ইন্সটল উদাহৰণ

    haxelib run pony install +code -code-insiders +npm +userpath -nodepath +ponypath

`+` - অপশ্যন সক্ৰিয় কৰক

`-` - অপশ্যন নিষ্ক্ৰিয় কৰক

`code` - Visual Studio Code ৰ প্ৰস্তাৱিত প্লাগিনসমূহ ইনস্টল কৰক

`code-insiders` - Visual Studio Code Insiders ৰ প্ৰস্তাৱিত প্লাগিনসমূহ ইনস্টল কৰক

`npm` - NPM (Node Package Manager) মডিউলসমূহ ইনস্টল কৰক

`userpath` - ইউজাৰ পাথসমূহ সংহতি কৰক

`nodepath` - ইউজাৰ পাথ node_modules লৈ সংহতি কৰক, কেৱল Windows ৰ বাবে

`ponypath` - pony.exe-লৈ ব্যৱহাৰকাৰীৰ পথ স্থাপন কৰক, কেৱল Windows-ৰ বাবে

CI স্থাপন উদাহৰণ

    haxelib run pony install -code -code-insiders +npm +userpath
