# TidalSwift

Swiftで書かれたTidal音楽ストリーミングクライアント＆ライブラリ

公式Tidalアプリの主要な機能をすべてサポートしつつ、新作リリース、歌詞、自動ダークモード、ダウンロード＆オフライン再生などの追加機能も備えています。しかも公式アプリの約1/10のサイズです。

> [!IMPORTANT]
>
> TidalSwiftの開発を継続しています 🎉



## ダウンロード

最新バージョンは[こちら](https://github.com/melgu/TidalSwift/releases)からダウンロードできます。
TidalSwift.zipをダウンロードして解凍した後、アプリをアプリケーションフォルダに移動してください。アプリは署名されていないため（現在Appleに金を払うのを拒否しているため）、初回起動後にプライバシーのシステム設定で明示的に許可する必要があります。もしくは、以下を実行してください：

```sh
xattr -d com.apple.quarantine /Applications/TidalSwift.app
```
## 印象

### 新着リリース

公式のTidalアプリとは異なり、TidalSwiftはお気に入りのアーティストの新着リリースを表示できます。

![New Releases](https://raw.githubusercontent.com/melgu/TidalSwift/main/README.assets/NewReleases.png)

### 歌詞

また、公式アプリとは違い、現在再生中の曲の歌詞を表示できます。

<img src="https://raw.githubusercontent.com/melgu/TidalSwift/main/README.assets/Lyrics.png" alt="Lyrics" width="400">

### オフライン

これは大きなポイントです。公式のデスクトップアプリはまだオフライン再生に対応しておらず、おそらく今後も対応しないでしょう。このアプリは対応しています！

![My Mixes](https://raw.githubusercontent.com/melgu/TidalSwift/main/README.assets/OfflineTracks.png)

### ダウンロード

さらに一歩進んでいます。音楽をハードドライブにダウンロードして、好きなように利用できます。
