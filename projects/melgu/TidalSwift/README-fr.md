# TidalSwift

Client et bibliothèque de streaming musical Tidal écrits en Swift

Il prend en charge toutes les fonctionnalités majeures de l'application officielle Tidal, tout en ajoutant des fonctionnalités supplémentaires, telles que les Nouveautés, les Paroles, le Mode Sombre automatique, les Téléchargements et la Lecture Hors Ligne – tout cela en étant seulement 1/10ème de la taille de l'application officielle.

> [!IMPORTANT]
>
> J'ai repris le développement de TidalSwift 🎉



## Téléchargement

Vous pouvez télécharger la dernière version [ici](https://github.com/melgu/TidalSwift/releases).
Après avoir téléchargé et décompressé le fichier TidalSwift.zip, déplacez l'application dans le dossier Applications. Comme l'application n'est pas signée (je refuse de donner de l'argent à Apple actuellement), vous devez explicitement autoriser l'application après le premier lancement dans les paramètres de confidentialité du système. Alternativement, vous pouvez exécuter ceci :

```sh
xattr -d com.apple.quarantine /Applications/TidalSwift.app
```

### Hors ligne

C'est un point important. L'application de bureau officielle ne prend toujours pas en charge la lecture hors ligne – et ne le fera probablement jamais. Cette application le fait !
