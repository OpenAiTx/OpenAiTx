# TidalSwift

Swift로 작성된 Tidal 음악 스트리밍 클라이언트 및 라이브러리

공식 Tidal 앱의 모든 주요 기능을 지원하면서, 신곡 출시, 가사, 자동 다크 모드, 다운로드 및 오프라인 재생과 같은 추가 기능을 제공하며, 공식 앱 크기의 1/10에 불과합니다.

> [!IMPORTANT]
>
> 저는 TidalSwift 작업을 계속하고 있습니다 🎉



## 다운로드

최신 버전은 [여기](https://github.com/melgu/TidalSwift/releases)에서 다운로드할 수 있습니다.
TidalSwift.zip을 다운로드하고 압축을 푼 후, 앱을 응용 프로그램 폴더로 이동하세요. 앱이 서명되지 않았기 때문에(저는 현재 애플에 돈을 주는 것을 거부합니다), 첫 실행 후 개인정보 보호 시스템 설정에서 앱을 명시적으로 허용해야 합니다. 또는 다음 명령을 실행할 수 있습니다:

```sh
xattr -d com.apple.quarantine /Applications/TidalSwift.app
```
