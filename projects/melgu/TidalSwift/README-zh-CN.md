# TidalSwift

用 Swift 编写的 Tidal 音乐流媒体客户端和库

它支持官方 Tidal 应用的所有主要功能，同时增加了新功能，如新发行、歌词、自动暗模式、下载和离线播放——而且体积仅为官方应用的十分之一。

> [!IMPORTANT]
>
> 我已经继续开发 TidalSwift 🎉



## 下载

您可以在[这里](https://github.com/melgu/TidalSwift/releases)下载最新版本。
下载并解压 TidalSwift.zip 后，将应用程序移动到“应用程序”文件夹。由于该应用未签名（我拒绝向苹果支付费用），首次启动后需要在隐私系统设置中明确允许该应用。或者，您可以执行以下操作：

```sh
xattr -d com.apple.quarantine /Applications/TidalSwift.app
```
