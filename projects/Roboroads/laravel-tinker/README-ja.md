<!-- Plugin description -->
PhpStorm/IDEA内でまるで`laravel artisan tinker`を使用しているかのように、PHPコードを簡単に書いて実行できます。Laravelのクラスに完全にアクセスできるコードスニペットの実行を、IDEのオートコンプリート、構文強調表示、エラー検査によって強化された便利さとともにお楽しみください。

すべてのプラグインアクションには`ツール -> Laravel Tinker`からアクセスするか、デフォルトショートカットの`ctrl + Shift + T`を使って、Tinkerがあなたの意図（開く、実行など）を賢く判断します。

- [💡 GitHub Issues](https://github.com/Roboroads/laravel-tinker/issues): 機能リクエストやバグ報告の提出
- [🙏 Ko-Fi](https://ko-fi.com/roboroads) / [GitHub Sponsors](https://github.com/sponsors/Roboroads): 役立つと思ったらプロジェクトをサポートしてください ❤️
- [🔌 プラグインページ](https://plugins.jetbrains.com/plugin/14957-laravel-tinker/): PHPStormまたはIDEA用のダウンロード
- [🔗 シェア](https://linktr.ee/laraveltinkerplugin): プラグインをネットワークで共有

このプラグインはLaravelプロジェクト外でも動作します - 基本機能のために`psy/psysh`がプロジェクトに必須です（一部制限あり）。
