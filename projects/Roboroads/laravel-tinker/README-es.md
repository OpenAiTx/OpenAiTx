<!-- Descripción del plugin -->
Escribe y ejecuta código PHP fácilmente directamente en PhpStorm/IDEA como si usaras `laravel artisan tinker`. Disfruta de la comodidad de ejecutar fragmentos de código con acceso completo a tus clases de Laravel, mejorado por la autocompletación, resaltado de sintaxis e inspecciones de errores de tu IDE.

Accede a todas las acciones del plugin vía `Herramientas -> Laravel Tinker` o usa el atajo por defecto `ctrl + Shift + T` para que Tinker determine inteligentemente tu intención (abrir, ejecutar, etc.).

- [💡 Problemas en GitHub](https://github.com/Roboroads/laravel-tinker/issues): Envía solicitudes de funciones y reportes de errores
- [🙏 Ko-Fi](https://ko-fi.com/roboroads) / [Patrocinadores de GitHub](https://github.com/sponsors/Roboroads): Apoya el proyecto si te resulta útil ❤️
- [🔌 Página del Plugin](https://plugins.jetbrains.com/plugin/14957-laravel-tinker/): Descarga para PHPStorm o IDEA
- [🔗 Compartir](https://linktr.ee/laraveltinkerplugin): Comparte el plugin con tu red

El plugin también funciona fuera de proyectos Laravel - solo asegúrate de que `psy/psysh` esté requerido en tu proyecto para funcionalidad básica (con algunas limitaciones).
