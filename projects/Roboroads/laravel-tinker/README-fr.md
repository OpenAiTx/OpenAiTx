Écrivez et exécutez facilement du code PHP directement dans PhpStorm/IDEA comme si vous utilisiez `laravel artisan tinker`. Profitez de la commodité d’exécuter des extraits de code avec un accès complet à vos classes Laravel, enrichi par l’autocomplétion, la coloration syntaxique et les inspections d’erreurs de votre IDE.

Accédez à toutes les actions du plugin via `Outils -> Laravel Tinker` ou utilisez le raccourci par défaut `ctrl + Maj + T` pour que Tinker détermine intelligemment votre intention (ouvrir, exécuter, etc.).

- [💡 Problèmes GitHub](https://github.com/Roboroads/laravel-tinker/issues) : Soumettez des demandes de fonctionnalités et des rapports de bugs
- [🙏 Ko-Fi](https://ko-fi.com/roboroads) / [GitHub Sponsors](https://github.com/sponsors/Roboroads) : Soutenez le projet si vous le trouvez utile ❤️
- [🔌 Page du plugin](https://plugins.jetbrains.com/plugin/14957-laravel-tinker/) : Téléchargez pour PHPStorm ou IDEA
- [🔗 Partager](https://linktr.ee/laraveltinkerplugin) : Partagez le plugin avec votre réseau

Le plugin fonctionne également en dehors des projets Laravel – assurez-vous simplement que `psy/psysh` est requis dans votre projet pour une fonctionnalité basique (avec quelques limitations).
