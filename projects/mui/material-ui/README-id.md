[Material¬†UI](https://mui.com/material-ui/) adalah perpustakaan komponen React yang komprehensif dengan implementasi independen dari sistem [Material Design](https://m2.material.io/design/introduction/) milik Google.
Material UI dipercaya oleh beberapa tim produk terbaik di dunia karena telah diuji secara intensif selama lebih dari satu dekade pengembangan oleh ribuan kontributor open-source.

Fungsi inti Material¬†UI diperluas oleh [MUI¬†X](https://github.com/mui/mui-x), rangkaian komponen kompleks untuk kebutuhan lanjutan.
[Toolpad](https://github.com/mui/toolpad) dibangun di atas Material¬†UI untuk menyediakan komponen full-stack dan pembuat alat internal low-code.

## Joy UI

Repositori ini juga berisi Joy UI, perpustakaan komponen eksperimental yang mengimplementasikan Joy Design buatan kami sendiri.
Joy UI masih dalam versi beta dan _pengembangannya saat ini sedang dihentikan sementara_.
Ketika memulai proyek baru dari awal, kami merekomendasikan Material¬†UI daripada Joy UI karena kami dapat menjamin dukungan berkelanjutan.

Perlu diingat bahwa para maintainer saat ini lebih fokus pada proyek lain dan mungkin tidak dapat merespons masalah atau pull request terkait Joy UI secara tepat waktu.

Lihat [dokumentasi Joy UI](https://mui.com/joy-ui/getting-started/).
