[Material UI](https://mui.com/material-ui/) 是一個全面的 React 元件庫，採用我們獨立實現的 Google [Material Design](https://m2.material.io/design/introduction/) 系統。
它受到全球頂尖產品團隊的信任，因為它經過了超過十年、數千名開源貢獻者嚴格實戰測試。

Material UI 的核心功能由 [MUI X](https://github.com/mui/mui-x) 擴展，這是一套針對進階使用情境的複雜元件組合。
[Toolpad](https://github.com/mui/toolpad) 在 Material UI 基礎上構建，提供全端元件與低程式碼內部工具建構器。

## Joy UI

本倉庫同時包含 Joy UI，一個實驗性元件庫，實現了我們自家設計的 Joy Design。
Joy UI 目前處於測試階段，_開發暫時停止_。
若是從零開始新專案，我們建議優先選用 Material UI，因為我們能保證持續支援。

請注意，維護者目前主要專注於其他專案，可能無法即時回應有關 Joy UI 的問題或合併請求。

請參考 [Joy UI 文件](https://mui.com/joy-ui/getting-started/)。
