[Material¬†UI](https://mui.com/material-ui/) is een uitgebreide bibliotheek van React-componenten die onze onafhankelijke implementatie van Google's [Material Design](https://m2.material.io/design/introduction/) systeem bevat.
Het wordt vertrouwd door enkele van 's werelds beste productteams omdat het grondig getest is in de praktijk door meer dan een decennium aan ontwikkeling door duizenden open-source bijdragers.

De kernfunctionaliteit van Material¬†UI wordt uitgebreid door [MUI¬†X](https://github.com/mui/mui-x), een suite van complexe componenten voor geavanceerde gebruikssituaties.
[Toolpad](https://github.com/mui/toolpad) bouwt voort op Material¬†UI om full-stack componenten en een low-code interne toolbouwer te bieden.

## Joy UI

Deze repository bevat ook Joy UI, een experimentele componentenbibliotheek die ons eigen Joy Design implementeert.
Joy UI is in b√®ta en _de ontwikkeling is momenteel gepauzeerd_.
Wanneer je een nieuw project helemaal opnieuw start, raden wij Material¬†UI aan boven Joy UI omdat we voortdurende ondersteuning kunnen garanderen.

Houd er rekening mee dat de beheerders zich vooral op andere projecten richten en mogelijk niet tijdig kunnen reageren op issues of pull requests met betrekking tot Joy UI.
