# सॉफ्टवेयर परिचय

zyfun एक निःशुल्क, अत्यंत सरल, सर्व-समर्थ क्रॉस-प्लेटफॉर्म मीडिया मैनेजर है, जो एक ही स्थान पर T1-T4 सभी प्रकार के संसाधनों का प्रबंधन करता है। इसमें मल्टी-कोर प्लेयर इनबिल्ट है जिसे आप मनचाहे तरीके से स्विच कर सकते हैं। अनूठा बॉस की (Boss Key) — एक क्लिक में छुपाएं, बेफिक्र काम करें। फिल्मों के माध्यम से दुनिया को जानें!

## 🌴 ध्यान देने योग्य बातें


- केवल Github पर जारी, कृपया धोखा न खाएं; सभी व्यवस्थापकों से अनुरोध है कि इस सॉफ़्टवेयर का प्रचार या ट्रैफ़िक न बढ़ाएं।
- वैध रूप से फिल्में देखने का दृढ़ समर्थन, यह सॉफ़्टवेयर केवल प्लेयर टूल है, संसाधन संग्रहण या वितरण में शामिल नहीं है।
- केवल व्यक्तिगत अध्ययन और चर्चा हेतु, कृपया 24 घंटे के भीतर स्वयं ही अनइंस्टॉल करें, व्यावसायिक उपयोग न करें।
- पूर्व में [ZY-Player](https://github.com/Hunlongyu/ZY-Player) से लिया गया, आइकन @fourbeauty द्वारा डिज़ाइन किया गया।
- उपयोग प्रारंभ करने से पूर्व कृपया उपयोगकर्ता अनुबंध को ध्यानपूर्वक पढ़ें और सहमत हों, तथा संबंधित नियमों का पालन करें।

#### MacOS इंस्टॉल के बाद खोलने पर दिखाता है "फाइल खराब है"

```bash
sudo spctl --master-disable
sudo xattr -cr /Applications/zyfun.app
```

#### Linux Appimage चलाने में विफल

> रूट के रूप में --no-sandbox के बिना चलाना समर्थित नहीं है

```bash
./zyfun.AppImage --no-sandbox
```

> dlopen() : libfuse.so.2 लोड करने में त्रुटि

```bash
sudo apt-get install libfuse2
```

> प्रारंभिकरण के दौरान त्रुटियों के कारण GPU प्रक्रिया समाप्त हो रही है

```bash
xhost +
```

#### Linux deb पैकेज इंस्टॉलेशन विफल

> पैकेज libnss3-1d इंस्टॉल नहीं है

```bash
sudo apt-get install libnss3-1d
```

> पैकेज libxss1 इंस्टॉल नहीं है

```bash
sudo apt-get install libxss1
```

#### Linux rpm पैकेज इंस्टॉल विफल

> निर्भरता जांच विफल हुई, libXScrnSaver आवश्यक है

```bash
sudo yum install epel-release
sudo yum install libXScrnSaver
```

> निर्भरता जांच विफल हुई xdg-utils आवश्यक है

```bash
sudo yum install xdg-utils
```
