<summary>展開查看安裝問題</summary>

#### MacOS 安裝後開啟提示「檔案已損壞」

```bash
sudo spctl --master-disable
sudo xattr -cr /Applications/zyfun.app
```

#### Linux Appimage 執行失敗

> 以 root 身份執行且未加上 --no-sandbox 參數是不被支援的

```bash
./zyfun.AppImage --no-sandbox
```

> dlopen()：載入 libfuse.so.2 時發生錯誤

```bash
sudo apt-get install libfuse2
```

> 因初始化期間發生錯誤，正在結束 GPU 程序

```bash
xhost +
```

#### Linux deb包安裝失敗

> 未安裝套件 libnss3-1d

```bash
sudo apt-get install libnss3-1d
```

> 套件 libxss1 未安裝

```bash
sudo apt-get install libxss1
```

#### Linux rpm套件安裝失敗

> 依賴性檢查失敗，需要 libXScrnSaver

```bash
sudo yum install epel-release
sudo yum install libXScrnSaver
```

> 依賴性檢查失敗，需要 xdg-utils

```bash
sudo yum install xdg-utils
```
