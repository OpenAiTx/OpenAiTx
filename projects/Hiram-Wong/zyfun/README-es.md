# Introducción al software

zyfun es un gestor multimedia gratuito, minimalista y versátil multiplataforma, que permite gestionar todo tipo de recursos T1-T4 en un solo lugar. Incluye un reproductor multinúcleo integrado que permite cambiar de motor fácilmente. Tecla exclusiva de “modo jefe” para ocultarte con un solo clic sin preocupaciones. ¡Descubre el mundo viendo películas!

#### MacOS: Al abrir después de instalar, aparece el mensaje "El archivo está dañado"

```bash
sudo spctl --master-disable
sudo xattr -cr /Applications/zyfun.app
```

#### Error al ejecutar Appimage en Linux

> Ejecutar como root sin --no-sandbox no es compatible

```bash
./zyfun.AppImage --no-sandbox
```

> dlopen(): error al cargar libfuse.so.2

```bash
sudo apt-get install libfuse2
```

> Saliendo del proceso de GPU debido a errores durante la inicialización

```bash
xhost +
```

#### Fallo en la instalación del paquete deb de Linux

> El paquete libnss3-1d no está instalado

```bash
sudo apt-get install libnss3-1d
```

> El paquete libxss1 no está instalado

```bash
sudo apt-get install libxss1
```

#### Error de instalación del paquete rpm en Linux

> Falló la verificación de dependencias, se necesita libXS

```bash
sudo yum install epel-release
sudo yum install libXScrnSaver
```

> Falló la verificación de dependencias, se necesita xdg-utils

```bash
sudo yum install xdg-utils
```
