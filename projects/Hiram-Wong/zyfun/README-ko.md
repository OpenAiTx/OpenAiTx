<summary>설치 문제 펼치기</summary>

#### MacOS 설치 후 실행 시 "파일이 손상되었습니다" 안내

```bash
sudo spctl --master-disable
sudo xattr -cr /Applications/zyfun.app
```

#### Linux Appimage 실행 실패

> --no-sandbox 없이 root로 실행하는 것은 지원되지 않습니다.

```bash
./zyfun.AppImage --no-sandbox
```

> dlopen()：libfuse.so.2 로딩 오류

```bash
sudo apt-get install libfuse2
```

> 초기화 중 오류로 인해 GPU 프로세스를 종료합니다

```bash
xhost +
```

#### Linux deb 패키지 설치 실패

> 패키지 libnss3-1d가 설치되어 있지 않습니다.

```bash
sudo apt-get install libnss3-1d
```

> 패키지 libxss1가 설치되어 있지 않습니다

```bash
sudo apt-get install libxss1
```

#### Linux rpm 패키지 설치 실패

> 의존성 검사 실패 libXScrnSaver가 필요함

```bash
sudo yum install epel-release
sudo yum install libXScrnSaver
```

> 종속성 확인 실패 xdg-utils가 필요합니다

```bash
sudo yum install xdg-utils
```

</details>
