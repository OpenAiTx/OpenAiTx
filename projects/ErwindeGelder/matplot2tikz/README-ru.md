```python

plt.xlabel("time (s)")
plt.ylabel("Voltage (mV)")

```

## matplot2tikz vs. tikzplotlib

–≠—Ç–∞ –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ matplot2tikz –≤–æ–∑–Ω–∏–∫–ª–∞ –∏–∑ –ø—Ä–æ–µ–∫—Ç–∞ [tikzplotlib](https://github.com/nschloe/tikzplotlib).
–ü—Ä–∏—á–∏–Ω–æ–π —Å–æ–∑–¥–∞–Ω–∏—è –Ω–æ–≤–æ–π –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ —Å—Ç–∞–ª–æ —Ç–æ, —á—Ç–æ tikzplotlib –±–æ–ª—å—à–µ –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç—Å—è, –∞
–ø–æ–¥–¥–µ—Ä–∂–∫–∞ –º–æ–≥–ª–∞ –æ—Å—É—â–µ—Å—Ç–≤–ª—è—Ç—å—Å—è —Ç–æ–ª—å–∫–æ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–º –≤–ª–∞–¥–µ–ª—å—Ü–µ–º –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ tikzplotlib.
–ï—Å–ª–∏ –≤–∞–º –Ω—É–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å—Ç–æ—Ä–æ–Ω–Ω–∏–π –∫–æ–¥, –∫–æ—Ç–æ—Ä—ã–π —É–∂–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç tikzplotlib, —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –∑–∞–º–µ–Ω–∏—Ç—å
–∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—å tikzplotlib –Ω–∞ matplot2tikz.
–ï—Å–ª–∏ —ç—Ç–æ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ, –º–æ–∂–Ω–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ö–æ–¥–Ω—ã–º —Ä–µ—à–µ–Ω–∏–µ–º ‚Äî —Ä–∞–∑–º–µ—Å—Ç–∏—Ç—å —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥ *–¥–æ* –∏–º–ø–æ—Ä—Ç–∞

