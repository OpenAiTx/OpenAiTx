## 什麼是 MCP？

[MCP](https://modelcontextprotocol.io/) 是一個開放協議，使 AI 模型能透過標準化伺服器實作，安全地存取本地與遠端資源。本清單聚焦於可投入生產與實驗性的 MCP 客戶端，這些客戶端可通過檔案存取、資料庫連接、API 整合以及其他情境服務來擴展 AI 能力。
