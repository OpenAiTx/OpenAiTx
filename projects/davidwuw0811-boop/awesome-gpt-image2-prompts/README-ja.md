## 🚀 使い方

1. [オンラインページ](https://davidwuw0811-boop.github.io/awesome-gpt-image2-prompts/)にアクセス
2. 検索ボックスにキーワードを入力（中国語と英語対応）
3. カテゴリタグをクリックして絞り込み
4. 必要なプロンプトを見つけたら、「コピー」ボタンをクリック
5. GPT Image 2 に貼り付けて使用

## 📝 プロンプト形式

各プロンプトは以下の情報を含みます：

- **中国語タグ** - カテゴリと説明、中国語ユーザーの検索に便利
- **英語プロンプト** - 完全な英語プロンプトテキスト
- **作者** - プロンプトの原作者
- **出典** - データ元リポジトリ
- **参考画像** - 一部プロンプトは参考画像のアップロードが必要
