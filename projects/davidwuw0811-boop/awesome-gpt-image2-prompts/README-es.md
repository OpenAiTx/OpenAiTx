## 🚀 Cómo usar

1. Visita [página web en línea](https://davidwuw0811-boop.github.io/awesome-gpt-image2-prompts/)
2. Escribe palabras clave en el cuadro de búsqueda (compatible con chino e inglés)
3. Haz clic en las etiquetas de categoría para filtrar
4. Después de encontrar el prompt deseado, haz clic en el botón "Copiar"
5. Pégalo en GPT Image 2 para usarlo

## 📝 Formato del prompt

Cada prompt contiene la siguiente información:

- **Etiqueta en chino** - Clasificación y descripción para facilitar la búsqueda a usuarios chinos
- **Prompt en inglés** - Texto completo del prompt en inglés
- **Autor** - Autor original del prompt
- **Fuente** - Repositorio de datos
- **Imagen de referencia** - Algunos prompts requieren subir imágenes de referencia
