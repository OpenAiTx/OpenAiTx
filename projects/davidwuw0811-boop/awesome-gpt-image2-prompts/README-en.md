## 🚀 How to Use

1. Visit [Online Webpage](https://davidwuw0811-boop.github.io/awesome-gpt-image2-prompts/)
2. Enter keywords in the search box (supports Chinese and English)
3. Click category tags to filter
4. After finding the desired prompt, click the "Copy" button
5. Paste it into GPT Image 2 for use

## 📝 Prompt Format

Each prompt contains the following information:

- **Chinese Tag** - Category and description, convenient for Chinese users to search
- **English Prompt** - Complete English prompt text
- **Author** - Original author of the prompt
- **Source** - Data source repository
- **Reference Image** - Some prompts require uploading reference images
