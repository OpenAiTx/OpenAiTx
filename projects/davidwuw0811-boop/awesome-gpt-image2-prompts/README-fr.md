## 🚀 Mode d'emploi

1. Visitez la [page en ligne](https://davidwuw0811-boop.github.io/awesome-gpt-image2-prompts/)
2. Entrez un mot-clé dans la barre de recherche (supporte le chinois et l'anglais)
3. Cliquez sur une étiquette de catégorie pour filtrer
4. Une fois le prompt souhaité trouvé, cliquez sur le bouton "Copier"
5. Collez-le dans GPT Image 2 pour l'utiliser

## 📝 Format des prompts

Chaque prompt contient les informations suivantes :

- **Étiquette en chinois** - Catégorie et description, pour faciliter la recherche des utilisateurs chinois
- **Prompt en anglais** - Texte complet du prompt en anglais
- **Auteur** - Auteur original du prompt
- **Source** - Dépôt de données source
- **Image de référence** - Certains prompts nécessitent le téléchargement d'une image de référence
