## Installation

- Téléchargez la [dernière version](https://github.com/kristianpennacchia/zzz-readermenuredesign.koplugin/releases/latest).
- Décompressez `zzz-readermenuredesign.koplugin.zip`.
- Copiez le dossier `zzz-readermenuredesign.koplugin` dans le répertoire `plugins` de KOReader sur l’appareil.
- Redémarrez KOReader.

#### Dépannage

- **Toutes les icônes sont des symboles d’avertissement (⚠️).**
    - Depuis la version v1.0.5, les icônes devraient s’installer automatiquement, mais si vous rencontrez toujours ce problème, essayez de copier manuellement les icônes du dossier `zzz-readermenuredesign.koplugin` et de les placer dans le dossier des icônes de KOReader `/koreader/resources/icons`.

- **Les icônes ne s’affichent pas du tout.**
    - Cela peut arriver sur certains appareils avec des versions plus anciennes de ce plugin. Essayez de supprimer les icônes SVG de ce plugin depuis `/koreader/resources/icons/mdlight` et remplacez-les par les icônes PNG de la [dernière version](https://github.com/kristianpennacchia/zzz-readermenuredesign.koplugin/releases/latest), puis **redémarrez KOReader**.

## Paramètres

- Menu de surlignage du lecteur :
    - Ouvrir le menu → Outils 🛠️ → Plus d'outils → Refonte du menu du lecteur → Afficher les boutons inconnus dans le menu de surlignage du lecteur.
    - Cela permettra d'afficher/masquer les boutons « inconnus » (boutons sans gestion spéciale dans ce plugin) dans le menu de surlignage du lecteur.
- Recherche rapide dans le dictionnaire :
    - Ouvrir le menu → Outils 🛠️ → Plus d'outils → Refonte du menu du lecteur → Afficher les boutons de navigation dans la recherche rapide du dictionnaire.
    - Cela permettra d'afficher/masquer les boutons « nav » dans la recherche rapide du dictionnaire pour naviguer entre les dictionnaires, les entrées Wikipédia, etc.
    - Alternativement : Vous pouvez simplement glisser de gauche à droite pour naviguer, mais ces boutons sont plus utiles sur PC/Mac.
