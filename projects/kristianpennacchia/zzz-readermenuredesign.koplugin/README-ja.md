## インストール

- [最新リリース](https://github.com/kristianpennacchia/zzz-readermenuredesign.koplugin/releases/latest)をダウンロードしてください。
- `zzz-readermenuredesign.koplugin.zip`を解凍してください。
- 解凍した`zzz-readermenuredesign.koplugin`フォルダをデバイスのKOReaderの`plugins`ディレクトリにコピーしてください。
- KOReaderを再起動してください。

#### トラブルシューティング

- **すべてのアイコンが警告（⚠️）シンボルになっている場合。**
    - バージョンv1.0.5以降ではアイコンは自動的にインストールされるはずですが、もし問題が続く場合は、`zzz-readermenuredesign.koplugin`フォルダからアイコンを手動でコピーしてKOReaderのアイコンフォルダ`/koreader/resources/icons`に入れてみてください。

- **アイコンがまったく表示されない場合。**
    - 古いバージョンのこのプラグインを使っている一部のデバイスで発生します。`/koreader/resources/icons/mdlight`からこのプラグインのSVGアイコンを削除し、[最新リリース](https://github.com/kristianpennacchia/zzz-readermenuredesign.koplugin/releases/latest)のPNGアイコンに置き換え、**KOReaderを再起動してください**。

## 設定

- リーダーハイライトメニュー：
    - メニューを開く → ツール 🛠️ → その他のツール → リーダーメニューデザイン → リーダーハイライトメニューに不明なボタンを表示。
    - これにより、リーダーハイライトメニューで「不明」ボタン（このプラグインで特別な処理がないボタン）の表示/非表示を切り替えます。
- 辞書クイックルックアップ：
    - メニューを開く → ツール 🛠️ → その他のツール → リーダーメニューデザイン → 辞書クイックルックアップにナビボタンを表示。
    - これにより、辞書クイックルックアップで辞書間やウィキペディアの項目間を移動する「ナビ」ボタンの表示/非表示を切り替えます。
    - または、単に左から右へのスワイプで移動することもできますが、これらのボタンはPC/Macでより便利です。
