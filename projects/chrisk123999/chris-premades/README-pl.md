# Readme

Kolekcja zautomatyzowanych elementów, w tym zaklęć, cech klasowych, cech potworów itp., mechanik umożliwiających ich działanie oraz rozszerzeń poprawiających komfort użytkowania dla środowiska D&D5e z wysokim poziomem automatyzacji opartym na Midi-QOL. Kompendia zawarte w tym module nie zawierają opisów przedmiotów. Chociaż ten moduł ma kilka zależności od modułów różnych autorów, nie kontaktuj się z tposneyem, Waspem ani żadnym innym autorem modułów w sprawie błędów lub problemów związanych z tym modułem. Zgłoszenia błędów i prośby o implementację dużych funkcji modułu można składać na GitHubie. Szybsze wsparcie oraz prośby o funkcje dostępne są na [serwerze Discord](https://discord.gg/BumxBcQDrT).

# Główne funkcje:
- Kompendia zautomatyzowanych zaklęć, przedmiotów, cech klasowych i innych.
- Przycisk na pasku tytułu zapewniający interfejs do stosowania i konfiguracji dostępnych automatyzacji.
- Efektowne animacje stworzone przez ***Eskiemoh*** dla niektórych automatyzacji, wykorzystujące *Patreon JB2A* oraz *Animated Spell Effects: Cartoon*.
- Niestandardowy rozstrzygacz rzutów do rzutów ręcznych, przeznaczony do gier na żywo.
- Własne API rozszerzające workflow Midi-QOL, umożliwiające precyzyjne określanie czasu zdarzeń i automatyzację nakładających się efektów zaklęć.
- Publiczne API do uruchamiania własnych makr.
- Różne opcjonalne rozszerzenia interfejsu poprawiające komfort użytkowania.
- Zmiany efektów, w tym automatyczne opisy, zmiany ikon efektów statusu oraz stosowanie odpowiednich flag Midi-QOL do efektów statusu.
- Niestandardowy interfejs efektów do przechowywania i stosowania własnych efektów.
- Opcjonalne i domowe mechaniki, w tym DMG Cleave, Eksplodujące Leczenia i Akcje Broni z BG3.
- Automatyczne kopie zapasowe postaci.
