# Readme

Коллекция автоматизированных элементов, включая заклинания, черты классов, черты монстров и др., а также механики для их реализации и расширения для повышения удобства использования в высокоавтоматизированной среде D&D5e на базе Midi-QOL. Справочники, включенные в этот модуль, не содержат описаний предметов. Несмотря на то, что данный модуль зависит от нескольких других модулей разных авторов, пожалуйста, не беспокойте tposney, Wasp или других авторов модулей по поводу ошибок или проблем, связанных с этим модулем. Сообщения об ошибках и крупные запросы на внедрение новых функций можно оставлять на GitHub. Более быстрая поддержка и возможность предложить функции доступны на [Discord сервере](https://discord.gg/BumxBcQDrT).

# Основные возможности:
- Справочники автоматизированных заклинаний, предметов, черт классов и прочего.
- Кнопка в заголовке, предоставляющая интерфейс для применения и настройки встроенных автоматизаций.
- Красочные анимации, созданные ***Eskiemoh*** для некоторых автоматизаций, с использованием *Patreon JB2A* и *Animated Spell Effects: Cartoon*.
- Пользовательский обработчик бросков для ручных бросков, предназначенный для игр вживую.
- Пользовательский API, расширяющий рабочий процесс Midi-QOL, позволяющий точно настраивать время событий и автоматизировать перекрывающиеся эффекты заклинаний.
- Публичный API для запуска пользовательских макросов.
- Различные дополнительные расширения пользовательского интерфейса для повышения удобства.
- Изменения эффектов, включая автоматическое описание, изменение иконок эффектов статуса и применение соответствующих флагов Midi-QOL к эффектам статуса.
- Пользовательский интерфейс эффектов для хранения и применения пользовательских эффектов.
- Дополнительные и домашние механики, включая DMG Cleave, Exploding Heals и BG3 Weapon Actions.
- Автоматическое создание резервных копий персонажей.
