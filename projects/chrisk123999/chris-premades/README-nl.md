# Readme

Een verzameling geautomatiseerde items, waaronder spreuken, klassefuncties, monsterfuncties, enz., mechanismen om deze mogelijk te maken, en kwaliteitsverbeterende uitbreidingen voor een sterk geautomatiseerde Midi-QOL-gebaseerde D&D5e-omgeving. Compendiums die in deze module zijn opgenomen, bevatten geen itembeschrijvingen. Hoewel deze module afhankelijk is van verschillende modules van andere auteurs, verzoeken wij u vriendelijk om tposney, Wasp of andere moduleauteurs niet lastig te vallen met bugs of problemen met betrekking tot deze module. Bugrapporten en grote module-implementatieverzoeken kunnen worden ingediend op GitHub. Snellere ondersteuning en functieverzoeken zijn toegankelijk via de [Discord-server](https://discord.gg/BumxBcQDrT).

# Belangrijkste Functies:
- Compendiums van geautomatiseerde spreuken, items, klassefuncties en meer.
- Titelbalkknop die een interface biedt om de bijgeleverde automatiseringen toe te passen en te configureren.
- Mooie animaties gemaakt door ***Eskiemoh*** voor sommige automatiseringen, met gebruik van *Patreon JB2A* en *Animated Spell Effects: Cartoon*.
- Aangepaste roll resolver voor handmatige worpen, ontworpen voor spellen aan tafel.
- Aangepaste API die voortbouwt op de Midi-QOL workflow en zorgt voor nauwkeurige eventtiming en automatisering van overlappende spelleffecten.
- Publieke API voor het uitvoeren van aangepaste macro's.
- Diverse optionele gebruikersinterface-uitbreidingen voor gebruiksgemak.
- Effectwijzigingen waaronder automatische beschrijvingen, statusicoonwijzigingen en toepassing van relevante Midi-QOL-flags op statuseffecten.
- Aangepaste effectinterface om aangepaste effecten op te slaan en toe te passen.
- Optionele en homebrew-mechanieken zoals DMG Cleave, Exploding Heals en BG3 Weapon Actions.
- Automatische back-ups van personages.
