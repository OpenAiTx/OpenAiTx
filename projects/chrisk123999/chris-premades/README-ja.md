# Readme

呪文、クラス特徴、モンスター特徴などを含む自動化アイテムのコレクション、高度な自動化を可能にするためのメカニクス、およびMidi-QOLを基盤としたD&D5e環境向けのQOL（クオリティ・オブ・ライフ）拡張機能集です。本モジュールに含まれるコンペンディウムにはアイテムの説明文は含まれていません。本モジュールは他作者による複数のモジュールに依存していますが、tposney氏、Wasp氏、その他のモジュール作者に本モジュールに関するバグや問題で問い合わせないでください。バグ報告や大規模な実装要望はGitHubで受け付けています。より迅速なサポートや機能要望は[Discordサーバー](https://discord.gg/BumxBcQDrT)で対応しています。

# 主な特徴:
- 自動化された呪文、アイテム、クラス特徴などのコンペンディウム。
- 自動化の適用と設定を行うインターフェースを持つタイトルバーのボタン。
- 一部の自動化において、***Eskiemoh***制作の華やかなアニメーション（*Patreon JB2A* および *Animated Spell Effects: Cartoon* を使用）。
- 対面プレイ向けに設計された手動ロール用のカスタムロールリゾルバー。
- Midi-QOLのワークフローを拡張したカスタムAPIによる、正確なイベントタイミングや重複する呪文効果の自動化。
- カスタムマクロを実行するためのパブリックAPI。
- 各種QOL向上のためのユーザーインターフェース拡張（任意）。
- 効果の自動説明、ステータス効果アイコンの変更、Midi-QOLフラグの自動付与等の効果変更。
- カスタム効果の保存・適用用のカスタム効果インターフェース。
- DMG Cleave、Exploding Heals、BG3 Weapon Actionsなどのオプション・自作（Homebrew）メカニクス。
- キャラクターの自動バックアップ。
