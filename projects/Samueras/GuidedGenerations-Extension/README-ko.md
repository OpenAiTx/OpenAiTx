# SillyTavern용 Guided Generations 확장

이 확장은 원본 "Guided Generations" 빠른 응답 세트의 모든 기능을 SillyTavern에 네이티브 확장으로 제공합니다. 역할극, 스토리, 캐릭터 중심 대화에 이상적인 모듈식, 상황 인식 도구로 AI 응답을 형성, 다듬고 안내할 수 있습니다. 모든 기능은 SillyTavern UI에 통합된 직관적인 버튼과 메뉴를 통해 접근할 수 있습니다.

코드 수준 문서는 [`JSDoc.md`](https://raw.githubusercontent.com/Samueras/GuidedGenerations-Extension/main/./JSDoc.md)를 참조하세요.

## 기능

### 🐕 안내된 응답
*AI가 답변하기 전에 지침을 주입합니다.*
- 지침을 입력하고 🐕 버튼을 누르세요.
- 입력한 지침이 다음 AI 응답을 안내합니다.
- [영상 예시](https://youtube.com/shorts/yxdtbF3NxW4?feature=share)

### 👈 안내된 스와이프
*새로운 지침으로 마지막 AI 메시지를 재생성합니다.*
- 새 지침을 입력하고 👈 버튼을 눌러 새 스와이프를 생성하세요.
- 마지막 메시지가 AI의 메시지인 경우에만 사용 가능합니다.
- [영상 예시](https://youtube.com/shorts/GRQ9l_8K6-Y?feature=share)

### 역할 대행 (1인칭: 👤, 2인칭: 👥, 3인칭: 🗣️)
*개요를 풍부하고 캐릭터에 맞는 내러티브로 확장합니다.*
- 간단한 개요를 입력하고, 관점을 선택한 후(설정에서 전환 가능) 해당 버튼(👤/👥/🗣️)을 누르세요.
- 선택한 시점에서 전체 메시지로 개요가 확장됩니다.
- 설정에 따라 개별적으로 숨기거나 표시할 수 있습니다. 1인칭은 기본 표시됩니다.
- 영상 예시:
  - [1인칭](https://youtube.com/shorts/FT5gv3d2kE4?feature=share)
  - [2인칭](https://youtube.com/shorts/80l12LrtBpQ?feature=share)
  - [3인칭](https://youtube.com/shorts/wWka-1URLPg?feature=share)

### 📖 지속 가이드 메뉴
*지속적인 시나리오 컨텍스트 관리.*
- 📖 버튼을 클릭하여 지속 가이드 메뉴를 엽니다.
- 가이드 유형(아래 참조)을 선택하여 컨텍스트를 생성하거나 관리합니다.

**가이드 유형:**
  - 🗺️ 상황별: 최근 채팅 또는 사용자 집중에서 컨텍스트 생성.
  - 🧠 생각: 캐릭터 생각 생성 (자동 트리거 선택 가능).
  - 👕 옷차림: 캐릭터 복장 설명 (자동 트리거 선택 가능).
  - 🧍 상태: 캐릭터 위치/상태 상세 설명 (자동 트리거 선택 가능).
  - 📜 규칙: 이야기 내 규칙 정의 또는 업데이트.
  - ➕ 사용자 정의: 사용자 지정 컨텍스트 주입.

**관리 작업:**
  - ✏️ 가이드 편집: 팝업을 통해 기존 가이드 주입 수정.
  - 👁️ 가이드 표시: 활성 가이드 모두 표시.
  - 🗑️ 가이드 삭제: 선택한 가이드 또는 모두 제거.
- 생각, 옷차림, 상태의 자동 트리거는 설정에서 전환 가능.

### 🔖 도구 메뉴
*추가 유틸리티 접근*
  - **🔧 수정:** 마지막 AI 메시지를 목표 지침으로 편집.
  - **✅ 맞춤법 검사:** 문법, 구두점, 흐름 교정.
  - **✈️ 간단 전송:** 모델 응답 없이 사용자 메시지로 입력 전송.
  - **🖋️ 소개 편집:** 요청 시 소개 메시지 재작성 또는 변환.
  - **↩️ 입력 복구:** 이전에 지운 입력 복원.

## ⚙️ 설정

모든 확장 설정은 SillyTavern의 확장 설정 패널에서 관리됩니다:

- **자동 실행**: 다음의 자동 실행을 토글합니다:
  - 사고 안내
  - 상태 안내
  - 의상 안내

- **버튼 표시**: 액션 버튼 표시 또는 숨기기:
  - 1인칭 흉내내기 (👤)
  - 2인칭 흉내내기 (👥)
  - 3인칭 흉내내기 (🗣️)
  - 안내 응답 (🐕)
  - 안내 스와이프 (👈)
  - 지속 안내 메뉴 (📖)

- **주입 역할**: 지침 주입 시 사용할 역할 (`system`, `assistant`, 또는 `user`) 선택.

- **디버그 모드**: 활성화 시 브라우저 콘솔에 상세 디버그 정보를 표시. 문제 해결에 유용하지만 일반 사용 시 콘솔이 복잡해질 수 있음.

- **프리셋**: 각 안내/도구(의상, 상태, 사고, 상황, 규칙, 커스텀, 수정, 맞춤법 검사, 소개 편집, 1/2/3인칭 흉내내기)에 대해 원하는 SillyTavern 프리셋 선택. 안내/도구 실행 전 해당 프리셋(및 구성된 API/모델)으로 전환 후 작업 수행, 이후 이전 프리셋 복원—안내별로 다른 모델 사용 가능.

- **프롬프트 재정의**: 각 안내/도구의 원본 프롬프트 템플릿을 맞춤 설정. `{{input}}`은 입력 텍스트, 기타 지원되는 플레이스홀더 사용 가능. 재정의는 다음에 적용:
  - 의상 안내 프롬프트
  - 상태 안내 프롬프트
  - 사고 안내 프롬프트
  - 상황 안내 프롬프트
  - 규칙 안내 프롬프트
  - 수정 프롬프트
  - 맞춤법 검사 프롬프트
  - 1인칭/2인칭/3인칭 프롬프트 흉내내기
  - 가이드 응답 프롬프트
  - 가이드 스와이프 프롬프트

## 문제 해결

- **버튼이 보이지 않을 때:** SillyTavern이 최신 버전(v1.12.9 이상)인지, LALib가 설치되고 활성화되어 있는지 확인하세요.
- **컨텍스트 메뉴가 나타나지 않을 때:** 채팅을 전환하거나 빠른 응답 메뉴에서 확장 기능을 다시 추가해 보세요.
- **기타 문제:** SillyTavern을 재시작하고, 업데이트가 있는지 확인하며, [SillyTavern 문서](https://github.com/SillyTavern/SillyTavern)를 참고하세요.
