## Konfigurasi API

Versi web ini saat ini mendukung layanan TTS gratis, API TTS88, dan API OpenAI:

### Konfigurasi Layanan TTS Gratis

1. Temukan opsi "Layanan TTS Gratis" di halaman pengaturan
2. Sistem telah mengatur alamat server TTS gratis secara default
3. Anda dapat melihat sisa kuota gratis harian dan waktu reset
4. Setiap klien browser memiliki kuota gratis yang terpisah

### Konfigurasi API TTS88

1. Temukan opsi "URL API Pihak Ketiga" di halaman pengaturan
2. Masukkan alamat API TTS88 Anda
3. Jika memiliki API key, masukkan ke kolom "API Key TTS88"

### Konfigurasi API OpenAI

1. Temukan opsi pengaturan OpenAI di halaman pengaturan
2. Masukkan API key OpenAI Anda
3. Pilih model yang ingin digunakan (default: gpt-3.5-turbo)
4. Jika menggunakan self-hosted atau proxy, Anda dapat mengatur URL dasar API khusus

## Catatan Penting

- Data hanya disimpan di penyimpanan lokal browser, refresh atau hapus cache tidak mempengaruhi pengguna lain
- File audio hasil konversi dapat langsung diputar atau diunduh di browser
- Layanan TTS gratis memiliki batas kuota harian, jika melebihi harus menunggu reset hari berikutnya
