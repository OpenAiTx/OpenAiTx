## API-configuratie

Deze webversie ondersteunt momenteel gratis TTS-diensten, de TTS88 API en de OpenAI API:

### Configuratie van gratis TTS-dienst

1. Zoek de optie "Gratis TTS-dienst" op de instellingenpagina
2. Standaard is er een gratis TTS-serveradres geconfigureerd
3. U kunt het resterende gratis tegoed van de dag en de reset-tijd bekijken
4. Elke browserclient heeft een eigen gratis tegoed

### TTS88 API-configuratie

1. Zoek de instelling "Derde partij API URL" op de instellingenpagina
2. Voer uw TTS88 API-adres in
3. Indien u een API-sleutel heeft, vul deze in bij het veld "TTS88 API-sleutel"

### OpenAI API-configuratie

1. Zoek de OpenAI-instellingen op de instellingenpagina
2. Voer uw OpenAI API-sleutel in
3. Kies het model dat u wilt gebruiken (standaard is gpt-3.5-turbo)
4. Bij gebruik van zelfgehoste of proxy-oplossing kunt u een aangepaste API Base URL instellen
