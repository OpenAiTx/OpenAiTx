## 機能特徴

- 🌐 完全Web化、デスクトップアプリのインストール不要
- 🔊 マイクロソフトの多様な音声合成に対応
- 🚀 TTS88 API統合に対応
- 🆓 無料TTS呼び出し対応、毎日の無料枠あり
- 🧠 OpenAIのテキスト生成対応
- 🌍 多言語対応：中文、英語、スペイン語
- 🔐 ブラウザーフィンガープリント認識で、より安全なユーザー体験

## API設定

本Web版は現在、無料TTSサービス、TTS88 API、OpenAI APIをサポートしています：

### 無料TTSサービス設定

1. 設定ページで「無料TTSサービス」オプションを見つける
2. システムはデフォルトで無料TTSサーバーのアドレスを設定済み
3. 当日の残り無料利用枠とリセット時間を確認可能
4. 各ブラウザクライアントごとに独立した無料利用枠がある

### TTS88 API設定

1. 設定ページで「第三者API URL」設定オプションを見つける
2. ご自身のTTS88 APIアドレスを入力する
3. APIキーがある場合は「TTS88 APIキー」欄に入力する

### OpenAI API設定

1. 設定ページでOpenAI設定オプションを見つける
2. ご自身のOpenAI APIキーを入力する
3. 使用するモデルを選択（デフォルトはgpt-3.5-turbo）
4. 自己ホスティングやプロキシを利用する場合はカスタムAPI Base URLを設定可能

## 注意事項

- データはブラウザのローカルストレージにのみ保存され、リフレッシュやキャッシュクリアは他のユーザーに影響しません
- 変換後の音声ファイルはブラウザ内で直接再生またはローカルにダウンロード可能
- 無料TTSサービスは1日あたりの利用上限があり、超過すると翌日のリセットまで待つ必要があります
