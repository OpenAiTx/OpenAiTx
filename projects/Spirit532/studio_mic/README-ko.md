# $100 미만의 간단하고 고품질 콘덴서 마이크

저는 좋은 품질의 마이크가 필요했지만, 시중에 나온 옵션들은 가격이 다소 높거나 성능이 만족스럽지 않아 직접 만들기로 했습니다.

이 마이크 제작에는 클론 RK12/CK12 캡슐(Aliexpress/Ebay 등에서 $40-50, "edge terminated capsule" 검색), U87 도너 바디($15-30, 역시 Aliexpress, "U87 mic" 검색어), 그리고 맞춤형 프리앰프 보드($20-30, 이 프로젝트)가 사용됩니다.

이 보드는 매우 간단하면서도 고성능입니다 - 듀얼 FET 입력 opamp를 단일 패키지로 사용해 캡슐의 임피던스를 변환하고 차동 신호를 제공하며, 바이어스는 약간 독특한 헥스 슈미트 트리거 인버터 차지 펌프로 공급됩니다.

아래 사진과 같은 구성으로, 비전문가의 귀에는 결과가 완벽합니다. 회로의 단순성과 선택한 opamp의 성능 덕분에 자체 노이즈가 사실상 없고, 충분한 다이내믹 레인지와 신호 왜곡/색채화가 없습니다. XLR 커넥터에서 캡슐이 받는 신호를 그대로 받을 수 있습니다.

#### 부품 선택 관련 참고사항
  1. 사실상 모든 FET 입력 opamp를 사용할 수 있지만, **정지 전류(quiescent current)에 특히 주의해야 합니다.** 이 회로는 상당히 높은 임피던스 전력 경로를 사용하며, 팬텀 파워 표준은 애초에 많은 전류를 공급하지 못합니다. 두 개의 opamp 전체 소비 전류는 최대 약 7-8mA(각각 3.5-4mA) 이내여야 합니다.

  2. 전력 소비 면에서는 헥스 인버터도 마찬가지지만, 기능이 비슷한 40106 클론 부품으로 대체 가능합니다.
  
  3. 지정된 곳에는 고전압 등급의 커패시터를 구매하세요 - 이것이 중요합니다! 세라믹 커패시터는 높은 전압에서 정전용량이 감소하므로 100V 이상 등급의 X7R 클래스 커패시터를 권장합니다(이것이 완전한 디레이팅 해결책은 아니지만 도움이 됩니다). 모든 값은 0805 사이즈로 쉽게 구할 수 있어 문제 없습니다.

  4. 예산이 허락된다면 세라믹 커패시터는 NP0/C0G 타입을 선택하세요. 성능이 약간 향상되지만 크게 차이나진 않습니다. (이것은 디레이팅 문제를 해결하지만 비용이 꽤 듭니다)
  
이 설계는 MIT 라이선스 하에 배포됩니다. 직접 제작하셔도 되고, 판매하셔도 되지만 이 페이지나 출처를 언급해 주세요. 사실 이 회로는 완전히 새로운 것이 아니며 여러 번 반복된 것들입니다(예: Instructables의 DJJules/Sound Sleuth 회로에서 영감을 받았습니다). 저는 여기에 제 스타일을 더했을 뿐입니다.

#### 저가 도너 바디에 대한 중요한 참고사항
대부분의 저가 도너 바디는 분리 지점(회로 커버, 헤드바스켓, 너트, 스페이서 등)의 접촉면까지 아노다이징 또는 도장이 되어 있으니, __반드시 사포로 갈아서 금속을 노출시키세요__. 그렇지 않으면 많은 메인 전원 험과 외부 전기 노이즈가 유입됩니다!
