## 📱📺 دو اپلیکیشن فوق‌العاده، یک مخزن

این مخزن شامل **دو اپلیکیشن کامل استریم موسیقی** می‌باشد:

### 🎵 **OpenSpot Mobile** (ری‌اکت نیتیو + اکسپو)
- **پلتفرم‌ها**: اندروید، iOS(در حال توسعه)
- **مکان**: `openspot-music-mobile/`
- **امکانات**: تجربه بومی موبایل با دانلود آفلاین، پخش در پس‌زمینه، و حفظ حالت پایدار

### 🖥️ **OpenSpot Desktop** (الکترون + ری‌اکت)
- **پلتفرم‌ها**: macOS، ویندوز(در حال توسعه)، لینوکس(در حال توسعه)
- **مکان**: `openspot-music-electron/`
- **امکانات**: رابط کاربری بهینه‌شده برای دسکتاپ با کنترل صوتی جهانی، مدیریت پنجره، و ذخیره‌سازی محلی

## ✨ ویژگی‌های کلیدی (هر دو اپلیکیشن)

- **🎵 پخش با کیفیت بالا**: آهنگ‌های مورد علاقه‌تان را با بهترین کیفیت گوش دهید
- **💾 دانلود آفلاین**: موسیقی را مستقیماً برای گوش دادن آفلاین روی دستگاه خود ذخیره کنید
- **❤️ پسندیدن و جمع‌آوری**: با پسندیدن آهنگ‌ها مجموعه شخصی خود را بسازید
- **🔄 پخش پس‌زمینه**: هنگام استفاده از برنامه‌های دیگر به گوش دادن ادامه دهید
- **🚫 بدون نیاز به ورود**: بلافاصله شروع کنید! بدون حساب کاربری یا ثبت‌نام
- **💾 ذخیره وضعیت پایدار**: آهنگ‌های پسندیده و آهنگ‌های پخش‌شده اخیر شما پس از راه‌اندازی مجدد برنامه ذخیره می‌شوند
- **🎨 رابط کاربری زیبا**: رابطی تمیز و مدرن که برای هر پلتفرم طراحی شده است
- **🆓 کاملاً رایگان و بدون تبلیغات**: از موسیقی بدون وقفه و بدون هیچ هزینه یا تبلیغی لذت ببرید

## 💻 فناوری‌های استفاده‌شده

### اپلیکیشن موبایل
- **فریمورک**: [React Native](https://reactnative.dev/) با [Expo](https://expo.dev/)
- **ناوبری**: [Expo Router](https://expo.github.io/router/)
- **صوتی**: [Expo AV](https://docs.expo.dev/versions/latest/sdk/av/)
- **ساخت**: [EAS Build](https://docs.expo.dev/build/introduction/)
- **دریافت موقعیت**: [IPinfo API](https://github.com/ipinfo)

### اپلیکیشن دسکتاپ
- **فریمورک**: [Electron](https://www.electronjs.org/) + [React](https://reactjs.org/)
- **رابط کاربری**: [Material-UI](https://mui.com/)
- **مدیریت وضعیت**: React Context + useReducer
- **ذخیره‌سازی**: [electron-store](https://github.com/sindresorhus/electron-store)
- **ساخت**: [electron-builder](https://www.electron.build/)

### مشترک
- **زبان**: [TypeScript](https://www.typescriptlang.org/)
- **مدیر بسته**: npm/yarn

این پروژه برای عملکرد صحیح به یک متغیر محیطی نیاز دارد.
