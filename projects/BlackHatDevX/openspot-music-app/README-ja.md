## 📖 概要

このモノレポは**最先端技術で構築された２つの完全独立音楽ストリーミングアプリ**を提供します：

| アプリ | プラットフォーム | フレームワーク | ロケーション |
|--------|------------------|---------------|--------------|
| 📱 **モバイル** | Android, iOS | React Native + Expo | `openspot-mobile/` |
| 🖥️ **デスクトップ** | macOS, Windows/Linux | Tauri + React | `openspot-desktop/` |

両アプリは同じ強力なストリーミングコアを共有し、すべてのデバイスでシームレスな音楽体験を実現します。
