## 📱📺 두 가지 멋진 앱, 하나의 저장소

이 저장소에는 **두 개의 완전한 음악 스트리밍 애플리케이션**이 포함되어 있습니다:

### 🎵 **OpenSpot Mobile** (React Native + Expo)
- **플랫폼**: Android, iOS
- **위치**: `openspot-music-mobile/`
- **기능**: 오프라인 다운로드, 백그라운드 재생, 지속 상태를 지원하는 네이티브 모바일 경험

### 🖥️ **OpenSpot Desktop** (Electron + React)
- **플랫폼**: macOS, Windows, Linux
- **위치**: `openspot-music-electron/`
- **기능**: 글로벌 오디오 컨트롤, 창 관리, 로컬 저장소를 갖춘 데스크톱 최적화 인터페이스

### 다운로드 링크

- **Android APK**: [OpenSpot-2.0.3-release.apk](https://github.com/BlackHatDevX/openspot-music-app/releases/download/v2.0.3/OpenSpot-2.0.3-release.apk)

```bash

```
### 프로덕션 빌드


```bash
cd openspot-music-app/

# Development Build
eas build --platform android --profile development

# Preview Build (APK)
eas build --platform android --profile preview

# Production Build
eas build --platform android --profile production
```

## ✨ 주요 기능 (양쪽 앱)

- **🎵 고음질 스트리밍**: 좋아하는 트랙을 최고의 음질로 감상하세요
- **💾 오프라인 다운로드**: 음악을 기기에 저장해 오프라인으로 감상하세요
- **❤️ 좋아요 & 컬렉션**: 노래에 '좋아요'를 눌러 개인 컬렉션을 만드세요
- **🔄 백그라운드 재생**: 다른 앱을 사용하면서도 음악 감상 지속
- **🚫 로그인 불필요**: 바로 시작! 계정이나 가입 없이 이용 가능
- **💾 지속적인 상태 저장**: 좋아요 표시한 곡과 최근 재생 목록이 앱 재시작 후에도 유지
- **🎨 아름다운 UI**: 각 플랫폼에 맞게 설계된 깔끔하고 현대적인 인터페이스
- **🆓 완전 무료 & 광고 없음**: 비용과 광고 없이 끊김 없는 음악 감상

## 💻 기술 스택

### 모바일 앱
- **프레임워크**: [React Native](https://reactnative.dev/) 및 [Expo](https://expo.dev/)
- **네비게이션**: [Expo Router](https://expo.github.io/router/)
- **오디오**: [Expo AV](https://docs.expo.dev/versions/latest/sdk/av/)
- **빌드**: [EAS Build](https://docs.expo.dev/build/introduction/)
- **위치 조회**: [IPinfo API](https://github.com/ipinfo)

### 데스크탑 앱
- **프레임워크**: [Electron](https://www.electronjs.org/) + [React](https://reactjs.org/)
- **UI**: [Material-UI](https://mui.com/)
- **상태 관리**: React Context + useReducer
- **저장소**: [electron-store](https://github.com/sindresorhus/electron-store)
- **빌드**: [electron-builder](https://www.electron.build/)

### 공통
- **언어**: [TypeScript](https://www.typescriptlang.org/)
- **패키지 매니저**: npm/yarn
