사용 중 문제 발생 시 저에게 개인 메시지 [@哦哟喔嚯](https://space.bilibili.com/39307302) 또는 [이슈 페이지](https://github.com/ChaozhongLiu/DyberPet_GenshinImpact/issues)에 문의해 주세요.

## 빠른 체험
### Windows 사용자
  **최초 사용 시**: 저장소를 [다운로드](https://github.com/ChaozhongLiu/DyberPet_GenshinImpact/archive/refs/heads/main.zip)하여 로컬에 압축 해제 후, **``다이보 펫-원신.exe``** 파일을 더블 클릭하면 됩니다.  
  (.exe 프로그램은 폴더 밖에서 실행할 수 없지만, 바탕화면에 바로가기를 만들 수 있습니다)
  
  
  **버전 업데이트**: 이미 다이보 데스크탑 펫을 사용 중이라면, [다운로드](https://github.com/ChaozhongLiu/DyberPet_GenshinImpact/archive/refs/heads/main.zip)한 최신 파일로 기존 구버전을 덮어쓰면 됩니다.  
  (전체 덮어쓰기, ``data`` 폴더는 보존되며 데이터는 손실되지 않습니다)
  
  **바이두 클라우드**：[링크](https://pan.baidu.com/s/1dLY0ZfFcTRySuU4tXv1-pA?pwd=bs1d) | 추출 코드：bs1d  
  **쿼크 클라우드**：[링크](https://pan.quark.cn/s/a070df462217) | 추출 코드：7RD4
