## クイックスタート
### Windows ユーザー向け
  **初めての使用**：リポジトリを[ダウンロード](https://github.com/ChaozhongLiu/DyberPet_GenshinImpact/archive/refs/heads/main.zip)し、解凍後に **``呆啵宠物-原神.exe``** をダブルクリックで起動してください。  
  （.exeファイルはフォルダ外に移動できませんが、デスクトップにショートカットを作成可能です）
  
  
  **バージョン更新**：すでに呆啵デスクトップペットを使用中の場合は、[ダウンロード](https://github.com/ChaozhongLiu/DyberPet_GenshinImpact/archive/refs/heads/main.zip)した最新ファイルで旧バージョンを上書きしてください。
