# Ш§ШіЩҫШ§ШұЪ©ЫҢвҖҢЩҒЫҢШӘЩҶШі

ЫҢЪ© Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Ш®ЩҲШҜЩ…ЫҢШІШЁШ§ЩҶ ЩҲ ШӯШұЫҢЩ…вҖҢШ®ШөЩҲШөЫҢ Щ…ШӯЩҲШұ ШЁШұШ§ЫҢ MyFitnessPal. ШӘШәШ°ЫҢЩҮШҢ ЩҲШұШІШҙШҢ Щ…Ш№ЫҢШ§ШұЩҮШ§ЫҢ ШЁШҜЩҶ ЩҲ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШіЩ„Ш§Щ…ШӘЫҢ ШұШ§ ЩҫЫҢЪҜЫҢШұЫҢ Ъ©ЩҶЫҢШҜ ЩҲ Ъ©ЩҶШӘШұЩ„ Ъ©Ш§Щ…Щ„ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ш®ЩҲШҜ ШұШ§ ШӯЩҒШё ЩҶЩ…Ш§ЫҢЫҢШҜ.

![Ш§ШіЪ©ШұЫҢЩҶвҖҢШҙШ§ШӘ](https://raw.githubusercontent.com/CodeWithCJ/SparkyFitness/main/docs/public/web_screenshot.png)

Ш§ШіЩҫШ§ШұЪ©ЫҢвҖҢЩҒЫҢШӘЩҶШі ЫҢЪ© ЩҫЩ„ШӘЩҒШұЩ… ЩҫЫҢЪҜЫҢШұЫҢ ШӘЩҶШ§ШіШЁ Ш§ЩҶШҜШ§Щ… Ш®ЩҲШҜЩ…ЫҢШІШЁШ§ЩҶ Ш§ШіШӘ Ъ©ЩҮ Ш§ШІ Щ…ЩҲШ§ШұШҜ ШІЫҢШұ ШӘШҙЪ©ЫҢЩ„ ШҙШҜЩҮ Ш§ШіШӘ:

- ЫҢЪ© ШіШұЩҲШұ ЩҫШҙШӘЫҢШЁШ§ЩҶ (API + Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ШҜШ§ШҜЩҮ)
- ЫҢЪ© ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ЩҲШЁ
- Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ЫҢ Щ…ЩҲШЁШ§ЫҢЩ„ ШЁЩҲЩ…ЫҢ ШЁШұШ§ЫҢ iOS ЩҲ Ш§ЩҶШҜШұЩҲЫҢШҜ

ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШіЩ„Ш§Щ…ШӘ ШұШ§ ШҜШұ ШІЫҢШұШіШ§Ш®ШӘЫҢ Ъ©ЩҮ Ш®ЩҲШҜШӘШ§ЩҶ Ъ©ЩҶШӘШұЩ„ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ Ш°Ш®ЫҢШұЩҮ ЩҲ Щ…ШҜЫҢШұЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁШҜЩҲЩҶ ЩҲШ§ШЁШіШӘЪҜЫҢ ШЁЩҮ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШҙШ®Шө Ш«Ш§Щ„Ш«.

## ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ

- ШұШҜЫҢШ§ШЁЫҢ ШӘШәШ°ЫҢЩҮШҢ ЩҲШұШІШҙШҢ ЩҮЫҢШҜШұШ§ШіЫҢЩҲЩҶШҢ Ш®ЩҲШ§ШЁШҢ ШұЩҲШІЩҮвҖҢШҜШ§ШұЫҢШҢ Ш®Щ„ЩӮвҖҢЩҲШ®ЩҲ ЩҲ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢвҖҢЩҮШ§ЫҢ ШЁШҜЩҶ
- ШӘШ№ЫҢЫҢЩҶ ЩҮШҜЩҒ ЩҲ ШЁШұШұШіЫҢвҖҢЩҮШ§ЫҢ ШұЩҲШІШ§ЩҶЩҮ
- ЩҶЩ…ЩҲШҜШ§ШұЩҮШ§ЫҢ ШӘШ№Ш§Щ…Щ„ЫҢ ЩҲ ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ ШЁЩ„ЩҶШҜЩ…ШҜШӘ
- ЩҫШұЩҲЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЪҶЩҶШҜ Ъ©Ш§ШұШЁШұЩҮ ЩҲ ШҜШіШӘШұШіЫҢ Ш®Ш§ЩҶЩҲШ§ШҜЪҜЫҢ
- ШӘЩ…вҖҢЩҮШ§ЫҢ ШұЩҲШҙЩҶ ЩҲ ШӘЫҢШұЩҮ
- OIDCШҢ TOTPШҢ PasskeyШҢ MFA ЩҲ ШәЫҢШұЩҮ

ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢвҖҢЩҮШ§ ШЁЩҮ Ш·ЩҲШұ Ш®ЩҲШҜЪ©Ш§Шұ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҒШ№Ш§Щ„ЫҢШӘ Щ…Ш§ЩҶЩҶШҜ ЩӮШҜЩ…вҖҢЩҮШ§ШҢ ШӘЩ…ШұЫҢЩҶШ§ШӘ ЩҲ Ш®ЩҲШ§ШЁ ШұШ§ ЩҮЩ…ШұШ§ЩҮ ШЁШ§ ШҙШ§Ш®ШөвҖҢЩҮШ§ЫҢ ШіЩ„Ш§Щ…ШӘ ЩҶШёЫҢШұ ЩҲШІЩҶ ЩҲ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢвҖҢЩҮШ§ЫҢ ШЁШҜЩҶШҢ ШЁЩҮ ШіШұЩҲШұ SparkyFitness ШҙЩ…Ш§ ЩҮЩ…ЪҜШ§Щ…вҖҢШіШ§ШІЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.

## ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш§Ш®ШӘЫҢШ§ШұЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ (ШЁШӘШ§)

SparkyAI ЫҢЪ© ШұШ§ШЁШ· Щ…Ъ©Ш§Щ„Щ…ЩҮвҖҢШ§ЫҢ ШЁШұШ§ЫҢ Ш«ШЁШӘ ШҜШ§ШҜЩҮ ЩҲ ШЁШұШұШіЫҢ ЩҫЫҢШҙШұЩҒШӘ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

- Ш«ШЁШӘ ШәШ°Ш§ШҢ ЩҲШұШІШҙШҢ ШўЩ…Ш§Шұ ШЁШҜЩҶ ЩҲ ЩӮШҜЩ…вҖҢЩҮШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЪҶШӘ
- ШЁШ§ШұЪҜШ°Ш§ШұЫҢ ШӘШөШ§ЩҲЫҢШұ ШәШ°Ш§ ШЁШұШ§ЫҢ Ш«ШЁШӘ Ш®ЩҲШҜЪ©Ш§Шұ ЩҲШ№ШҜЩҮ ШәШ°Ш§ЫҢЫҢ
- ШӘШ§ШұЫҢШ®ЪҶЩҮ Щ…Ъ©Ш§Щ„Щ…Ш§ШӘ ШұШ§ ШЁШұШ§ЫҢ ЩҫЫҢЪҜЫҢШұЫҢвҖҢЩҮШ§ЫҢ ШЁШ№ШҜЫҢ ШӯЩҒШё Щ…ЫҢвҖҢЪ©ЩҶШҜ

ШӘЩҲШ¬ЩҮ: ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ ШҜШұ ЩҶШіШ®ЩҮ ШЁШӘШ§ ЩҮШіШӘЩҶШҜ.

## ШҙШұЩҲШ№ ШіШұЫҢШ№ (ШіШұЩҲШұ)

ЫҢЪ© ШіШұЩҲШұ SparkyFitness ШұШ§ ШҜШұ Ш№ШұШ¶ ЪҶЩҶШҜ ШҜЩӮЫҢЩӮЩҮ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Docker Compose Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.

```bash
# 1. Create a new folder
mkdir sparkyfitness && cd sparkyfitness

# 2. Download Docker files only
curl -L -o docker-compose.yml https://github.com/CodeWithCJ/SparkyFitness/releases/latest/download/docker-compose.prod.yml
curl -L -o .env https://github.com/CodeWithCJ/SparkyFitness/releases/latest/download/default.env.example

# 3. (Optional) Edit .env to customize database credentials, ports, etc.

# 4. Start the app
docker compose pull && docker compose up -d

# Access application at http://localhost:8080
```
