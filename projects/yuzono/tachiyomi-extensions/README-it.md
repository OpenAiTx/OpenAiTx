# Estensioni Komikku / Mihon / Tachiyomi

Questo repository contiene cataloghi di estensioni compatibili con [Komikku](https://github.com/komikku-app/komikku) e Mihon / Tachiyomi o altri fork.

Questo repository unisce automaticamente ogni 6 ore tutti gli aggiornamenti da [Keiyoushi](https://github.com/keiyoushi/extensions-source) per offrire il meglio dei contributi della community. Oltre a ciò, include alcune estensioni sviluppate da me o miglioramenti vari. Buon divertimento!

Alcune estensioni di questo repository offrono un supporto migliore per la funzionalità `Suggerimenti` di Komikku.

## App consigliate

### [Komikku](https://github.com/komikku-app/komikku)

### [Mihon](https://github.com/mihonapp/mihon)

## Come aggiungere il repository

**Se sei nuovo ai repository/estensioni, leggi prima la [Guida introduttiva di Yūzōnō](https://yuzono.github.io/docs/guides/getting-started#adding-the-extension-repo).**

* Puoi aggiungere il nostro repository visitando il [sito Yūzōnō](https://yuzono.github.io/add-repo)
* In alternativa, copia e incolla il seguente URL:

```html
https://raw.githubusercontent.com/yuzono/manga-repo/repo/index.min.json
```

# Richieste

Per richiedere una nuova fonte o una correzione di bug, [crea una issue](https://github.com/yuzono/tachiyomi-extensions/issues/new/choose).

Tieni presente che creare una issue non garantisce che la fonte venga aggiunta o corretta in tempi brevi, poiché il lavoro è svolto su base volontaria. Alcune fonti potrebbero anche essere impossibili da implementare o troppo difficili da mantenere.

Se desideri vedere una richiesta soddisfatta e hai le competenze necessarie, considera di contribuire! Le issue sono a disposizione di qualsiasi sviluppatore se non sono già assegnate a qualcuno.

# Contribuire

I contributi sono benvenuti!
